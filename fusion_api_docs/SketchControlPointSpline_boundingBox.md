# SketchControlPointSpline.boundingBox Property

Parent Object: [SketchControlPointSpline](SketchControlPointSpline.md)  

## Description

Returns the bounding box of the entity in sketch space.

## Syntax

"sketchControlPointSpline_var" is a variable referencing a SketchControlPointSpline object.  

```python
# Get the value of the property.
propertyValue = sketchControlPointSpline_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version July 2022
