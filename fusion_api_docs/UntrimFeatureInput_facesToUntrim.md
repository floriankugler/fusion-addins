# UntrimFeatureInput.facesToUntrim Property

Parent Object: [UntrimFeatureInput](UntrimFeatureInput.md)  

## Description

Gets the face objects to untrim. Returns null/None in the case where loops are specified instead of faces.

## Syntax

"untrimFeatureInput_var" is a variable referencing a UntrimFeatureInput object.  

```python
# Get the value of the property.
propertyValue = untrimFeatureInput_var.facesToUntrim
```

## Property Value

This is a read only property whose value is an array of type [BRepFace](BRepFace.md).

## Version

Introduced in version January 2021
