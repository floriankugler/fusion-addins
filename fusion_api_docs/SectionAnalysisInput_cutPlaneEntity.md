# SectionAnalysisInput.cutPlaneEntity Property

Parent Object: [SectionAnalysisInput](SectionAnalysisInput.md)  

## Description

A property that gets and sets the planar entity used to define the cut plane and can be either a planar BRepFace or a ConstructionPlane object.

## Syntax

"sectionAnalysisInput_var" is a variable referencing a SectionAnalysisInput object.  

```python
# Get the value of the property.
propertyValue = sectionAnalysisInput_var.cutPlaneEntity

# Set the value of the property.
sectionAnalysisInput_var.cutPlaneEntity = propertyValue
```

## Property Value

This is a read/write property whose value is a [Base](Base.md).

## Version

Introduced in version January 2023
