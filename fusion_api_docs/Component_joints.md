# Component.joints Property

Parent Object: [Component](Component.md)  

## Description

Returns the collection of joints associated with this component.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.joints
```

## Property Value

This is a read only property whose value is a [Joints](Joints.md).

## Version

Introduced in version July 2015
