# ConstructionPlaneTangentDefinition.tangentFace Property

Parent Object: [ConstructionPlaneTangentDefinition](ConstructionPlaneTangentDefinition.md)  

## Description

Gets the cylindrical or conical face that the construction plane is tangent to.

## Syntax

"constructionPlaneTangentDefinition_var" is a variable referencing a ConstructionPlaneTangentDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionPlaneTangentDefinition_var.tangentFace
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version August 2014
