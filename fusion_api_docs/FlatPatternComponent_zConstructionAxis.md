# FlatPatternComponent.zConstructionAxis Property

Parent Object: [FlatPatternComponent](FlatPatternComponent.md)  

## Description

Returns the Z origin construction axis.

## Syntax

"flatPatternComponent_var" is a variable referencing a FlatPatternComponent object.  

```python
# Get the value of the property.
propertyValue = flatPatternComponent_var.zConstructionAxis
```

## Property Value

This is a read only property whose value is a [ConstructionAxis](ConstructionAxis.md).

## Version

Introduced in version October 2022
