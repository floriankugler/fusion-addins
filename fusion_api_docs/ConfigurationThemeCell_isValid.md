# ConfigurationThemeCell.isValid Property

Parent Object: [ConfigurationThemeCell](ConfigurationThemeCell.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"configurationThemeCell_var" is a variable referencing a ConfigurationThemeCell object.  

```python
# Get the value of the property.
propertyValue = configurationThemeCell_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2024
