# SketchEllipse.isFullyConstrained Property

Parent Object: [SketchEllipse](SketchEllipse.md)  

## Description

Indicates if this sketch entity is fully constrained.

## Syntax

"sketchEllipse_var" is a variable referencing a SketchEllipse object.  

```python
# Get the value of the property.
propertyValue = sketchEllipse_var.isFullyConstrained
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2016
