# CAMHoleRecognition.name Property

Parent Object: [CAMHoleRecognition](CAMHoleRecognition.md)  

## Description

Gets and sets the name of the operation as seen in the browser.Â This name is unique as compared to the names of all other operations in the document.

## Syntax

"cAMHoleRecognition_var" is a variable referencing a CAMHoleRecognition object.  

```python
# Get the value of the property.
propertyValue = cAMHoleRecognition_var.name

# Set the value of the property.
cAMHoleRecognition_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version April 2023
