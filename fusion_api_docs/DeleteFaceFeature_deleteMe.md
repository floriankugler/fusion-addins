# DeleteFaceFeature.deleteMe Method

Parent Object: [DeleteFaceFeature](DeleteFaceFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"deleteFaceFeature_var" is a variable referencing a [DeleteFaceFeature](DeleteFaceFeature.md) object.

```python
returnValue = deleteFaceFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version August 2016
