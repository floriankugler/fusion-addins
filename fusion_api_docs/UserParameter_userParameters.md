# UserParameter.userParameters Property

Parent Object: [UserParameter](UserParameter.md)  

## Description

Returns the Collection containing the UserParameter.

## Syntax

"userParameter_var" is a variable referencing a UserParameter object.  

```python
# Get the value of the property.
propertyValue = userParameter_var.userParameters
```

## Property Value

This is a read only property whose value is a [UserParameters](UserParameters.md).

## Version

Introduced in version August 2014
