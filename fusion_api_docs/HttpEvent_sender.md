# HttpEvent.sender Property

Parent Object: [HttpEvent](HttpEvent.md)  

## Description

The object that is firing the event. For example, in the case of a command input event this will return the command.

## Syntax

"httpEvent_var" is a variable referencing a HttpEvent object.  

```python
# Get the value of the property.
propertyValue = httpEvent_var.sender
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version January 2024
