# ExtendFeature.isChainingEnabled Property

Parent Object: [ExtendFeature](ExtendFeature.md)  

## Description

Gets if all edges that are tangent or curvature continuous, and end point connected, will be found automatically and extended.

## Syntax

"extendFeature_var" is a variable referencing an ExtendFeature object.  

```python
# Get the value of the property.
propertyValue = extendFeature_var.isChainingEnabled
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version June 2015
