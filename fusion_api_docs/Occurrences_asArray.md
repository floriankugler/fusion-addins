# Occurrences.asArray Method

Parent Object: [Occurrences](Occurrences.md)  

## Description

Get the current list of all occurrences. The occurrences are returned in the same order as they appear in the browser.

## Syntax

"occurrences_var" is a variable referencing an [Occurrences](Occurrences.md) object.

```python
returnValue = occurrences_var.asArray()
```

## Return Value

| Type | Description |
|----|----|
| [Occurrence](Occurrence.md)\[\] | Returns the current list of all occurrences. |

## Version

Introduced in version November 2021
