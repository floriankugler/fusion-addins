# ConcentricConstraint.deleteMe Method

Parent Object: [ConcentricConstraint](ConcentricConstraint.md)  

## Description

Deletes this constraint. The IsDeletable property can be used to determine if this constraint can be deleted.

## Syntax

"concentricConstraint_var" is a variable referencing a [ConcentricConstraint](ConcentricConstraint.md) object.

```python
returnValue = concentricConstraint_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
