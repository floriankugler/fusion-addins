# ConfigurationPlasticRuleTable.name Property

Parent Object: [ConfigurationPlasticRuleTable](ConfigurationPlasticRuleTable.md)  

## Description

Returns the name of the table as seen in the user interface.

## Syntax

"configurationPlasticRuleTable_var" is a variable referencing a ConfigurationPlasticRuleTable object.  

```python
# Get the value of the property.
propertyValue = configurationPlasticRuleTable_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version March 2024
