# OffsetFeature.deleteMe Method

Parent Object: [OffsetFeature](OffsetFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"offsetFeature_var" is a variable referencing an [OffsetFeature](OffsetFeature.md) object.

```python
returnValue = offsetFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version June 2015
