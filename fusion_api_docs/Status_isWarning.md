# Status.isWarning Property

Parent Object: [Status](Status.md)  

## Description

If true, the operation has succeeded but with an unusual result. This takes into account all of the child status messages.

## Syntax

"status_var" is a variable referencing a Status object.  

```python
# Get the value of the property.
propertyValue = status_var.isWarning
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2021
