# Circle3D.center Property

Parent Object: [Circle3D](Circle3D.md)  

## Description

Gets and sets the center position of the circle.

## Syntax

"circle3D_var" is a variable referencing a Circle3D object.  

```python
# Get the value of the property.
propertyValue = circle3D_var.center

# Set the value of the property.
circle3D_var.center = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
