# ToolPreset.name Property

Parent Object: [ToolPreset](ToolPreset.md)  

## Description

Gets and sets the name of that Preset.

## Syntax

"toolPreset_var" is a variable referencing a ToolPreset object.  

```python
# Get the value of the property.
propertyValue = toolPreset_var.name

# Set the value of the property.
toolPreset_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.
