# OffsetFacesFeature.isParametric Property

Parent Object: [OffsetFacesFeature](OffsetFacesFeature.md)  

## Description

Indicates if this feature is parametric or not.

## Syntax

"offsetFacesFeature_var" is a variable referencing an OffsetFacesFeature object.  

```python
# Get the value of the property.
propertyValue = offsetFacesFeature_var.isParametric
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version June 2017
