# CornerClosureFeature.definition Property

Parent Object: [CornerClosureFeature](CornerClosureFeature.md)  

## Description

Returns the CornerClosureFeatureDefinition object which provides access to the information defining this CornerClosureFeature and the ability to edit it.

## Syntax

"cornerClosureFeature_var" is a variable referencing a CornerClosureFeature object.  

```python
# Get the value of the property.
propertyValue = cornerClosureFeature_var.definition
```

## Property Value

This is a read only property whose value is a [CornerClosureFeatureDefinition](CornerClosureFeatureDefinition.md).

## Version

Introduced in version January 2026
