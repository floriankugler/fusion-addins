# SpunProfileInput.tolerance Property

Parent Object: [SpunProfileInput](SpunProfileInput.md)  

## Description

Gets and sets the linear tolerance in cm used for creating the spun profile. Defaults to 0.001cm.

## Syntax

"spunProfileInput_var" is a variable referencing a SpunProfileInput object.  

```python
# Get the value of the property.
propertyValue = spunProfileInput_var.tolerance

# Set the value of the property.
spunProfileInput_var.tolerance = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version November 2024
