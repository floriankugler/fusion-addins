# SketchFittedSpline.geometricConstraints Property

Parent Object: [SketchFittedSpline](SketchFittedSpline.md)  

## Description

Returns the sketch constraints that are attached to this curve.

## Syntax

"sketchFittedSpline_var" is a variable referencing a SketchFittedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFittedSpline_var.geometricConstraints
```

## Property Value

This is a read only property whose value is a [GeometricConstraintList](GeometricConstraintList.md).

## Version

Introduced in version January 2015
