# RenderEventArgs.firingEvent Property

Parent Object: [RenderEventArgs](RenderEventArgs.md)  

## Description

The event that the firing is in response to.

## Syntax

"renderEventArgs_var" is a variable referencing a RenderEventArgs object.  

```python
# Get the value of the property.
propertyValue = renderEventArgs_var.firingEvent
```

## Property Value

This is a read only property whose value is an [Event](Event.md).

## Version

Introduced in version September 2023
