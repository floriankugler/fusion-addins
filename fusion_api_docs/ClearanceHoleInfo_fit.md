# ClearanceHoleInfo.fit Property

Parent Object: [ClearanceHoleInfo](ClearanceHoleInfo.md)  

## Description

Returns the enum value that defines the amount of clearance between the hole and the fastener.

## Syntax

"clearanceHoleInfo_var" is a variable referencing a ClearanceHoleInfo object.  

```python
# Get the value of the property.
propertyValue = clearanceHoleInfo_var.fit
```

## Property Value

This is a read only property whose value is a [ClearanceHoleFits](ClearanceHoleFits.md).

## Version

Introduced in version September 2025
