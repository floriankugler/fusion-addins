# DrawingDocument.isActive Property

Parent Object: [DrawingDocument](DrawingDocument.md)  

## Description

Gets if this document is the active document in the user interface.

## Syntax

"drawingDocument_var" is a variable referencing a DrawingDocument object.  

```python
# Get the value of the property.
propertyValue = drawingDocument_var.isActive
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version December 2020
