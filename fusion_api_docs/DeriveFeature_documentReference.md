# DeriveFeature.documentReference Property

Parent Object: [DeriveFeature](DeriveFeature.md)  

## Description

Returns the DocumentReference for this derived design. Through the DocumentReference object you can modify the version and get other information associated with the reference.

## Syntax

"deriveFeature_var" is a variable referencing a DeriveFeature object.  

```python
# Get the value of the property.
propertyValue = deriveFeature_var.documentReference
```

## Property Value

This is a read only property whose value is a [DocumentReference](DocumentReference.md).

## Version

Introduced in version January 2026
