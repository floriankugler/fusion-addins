# UserInterface.activeToolbarTab Property

Parent Object: [UserInterface](UserInterface.md)  

## Description

Retrieve the active ToolbarTab being displayed in the user interface. This may be null.

## Syntax

"userInterface_var" is a variable referencing a UserInterface object.  

```python
# Get the value of the property.
propertyValue = userInterface_var.activeToolbarTab
```

## Property Value

This is a read only property whose value is a [ToolbarTab](ToolbarTab.md).

## Version

Introduced in version May 2024
