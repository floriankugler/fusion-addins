# SketchLinearDimension.deleteMe Method

Parent Object: [SketchLinearDimension](SketchLinearDimension.md)  

## Description

Deletes this dimension. The IsDeletable property indicates if this dimension can be deleted.

## Syntax

"sketchLinearDimension_var" is a variable referencing a [SketchLinearDimension](SketchLinearDimension.md) object.

```python
returnValue = sketchLinearDimension_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
