# OffsetFeature.operation Property

Parent Object: [OffsetFeature](OffsetFeature.md)  

## Description

Gets the feature operation that was performed when the feature was created, (either 'NewBodyFeatureOperation' or 'NewComponentFeatureOperation'.

## Syntax

"offsetFeature_var" is a variable referencing an OffsetFeature object.  

```python
# Get the value of the property.
propertyValue = offsetFeature_var.operation
```

## Property Value

This is a read only property whose value is a [FeatureOperations](FeatureOperations.md).

## Version

Introduced in version June 2015
