# GeometrySelection.hasError Property

Parent Object: [GeometrySelection](GeometrySelection.md)  

## Description

Gets if errors were encountered when applying the selection to a a parent.

## Syntax

"geometrySelection_var" is a variable referencing a GeometrySelection object.  

```python
# Get the value of the property.
propertyValue = geometrySelection_var.hasError
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2023
