# CustomGraphicsEntity.deleteMe Method

Parent Object: [CustomGraphicsEntity](CustomGraphicsEntity.md)  

## Description

Deletes the entity from the custom graphics group.

## Syntax

"customGraphicsEntity_var" is a variable referencing a [CustomGraphicsEntity](CustomGraphicsEntity.md) object.

```python
returnValue = customGraphicsEntity_var.deleteMe()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns true if the deletion was successful. |

## Version

Introduced in version September 2017
