# NCProgram.attributes Property

Parent Object: [NCProgram](NCProgram.md)  

## Description

Returns the collection of attributes associated with this object.

## Syntax

"nCProgram_var" is a variable referencing a NCProgram object.  

```python
# Get the value of the property.
propertyValue = nCProgram_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version April 2023
