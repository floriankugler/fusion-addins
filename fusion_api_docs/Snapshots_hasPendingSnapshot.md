# Snapshots.hasPendingSnapshot Property

Parent Object: [Snapshots](Snapshots.md)  

## Description

Indicates if there are any changes that have been made than can be snapshot.

## Syntax

"snapshots_var" is a variable referencing a Snapshots object.  

```python
# Get the value of the property.
propertyValue = snapshots_var.hasPendingSnapshot
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
