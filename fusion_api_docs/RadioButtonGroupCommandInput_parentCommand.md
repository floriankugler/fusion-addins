# RadioButtonGroupCommandInput.parentCommand Property

Parent Object: [RadioButtonGroupCommandInput](RadioButtonGroupCommandInput.md)  

## Description

Gets the parent Command.

## Syntax

"radioButtonGroupCommandInput_var" is a variable referencing a RadioButtonGroupCommandInput object.  

```python
# Get the value of the property.
propertyValue = radioButtonGroupCommandInput_var.parentCommand
```

## Property Value

This is a read only property whose value is a [Command](Command.md).

## Version

Introduced in version July 2015
