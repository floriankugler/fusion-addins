# BallJointMotion.rollLimits Property

Parent Object: [BallJointMotion](BallJointMotion.md)  

## Description

Returns a JointLimits object that defines the limits of rotation for the roll. Use the functionality of the returned JointLimits object to get, set, and modify the joint limits.

## Syntax

"ballJointMotion_var" is a variable referencing a BallJointMotion object.  

```python
# Get the value of the property.
propertyValue = ballJointMotion_var.rollLimits
```

## Property Value

This is a read only property whose value is a [JointLimits](JointLimits.md).

## Version

Introduced in version July 2015
