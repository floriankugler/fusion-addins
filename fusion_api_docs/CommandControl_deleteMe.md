# CommandControl.deleteMe Method

Parent Object: [CommandControl](CommandControl.md)  

## Description

Deletes the ToolbarControl

## Syntax

"commandControl_var" is a variable referencing a [CommandControl](CommandControl.md) object.

```python
returnValue = commandControl_var.deleteMe()
```

## Return Value

| Type    | Description                                                  |
|---------|--------------------------------------------------------------|
| boolean | Returns a boolean indicating if the deletion was successful. |

## Version

Introduced in version August 2014
