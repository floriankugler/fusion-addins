# GroupCommandInput.name Property

Parent Object: [GroupCommandInput](GroupCommandInput.md)  

## Description

Gets the user visible name of this input.

## Syntax

"groupCommandInput_var" is a variable referencing a GroupCommandInput object.  

```python
# Get the value of the property.
propertyValue = groupCommandInput_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version July 2015
