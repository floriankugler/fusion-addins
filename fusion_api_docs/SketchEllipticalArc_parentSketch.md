# SketchEllipticalArc.parentSketch Property

Parent Object: [SketchEllipticalArc](SketchEllipticalArc.md)  

## Description

Returns the parent sketch.

## Syntax

"sketchEllipticalArc_var" is a variable referencing a SketchEllipticalArc object.  

```python
# Get the value of the property.
propertyValue = sketchEllipticalArc_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
