# SketchDimension.textPosition Property

Parent Object: [SketchDimension](SketchDimension.md)  

## Description

Gets and sets position of the dimension text.

## Syntax

"sketchDimension_var" is a variable referencing a SketchDimension object.  

```python
# Get the value of the property.
propertyValue = sketchDimension_var.textPosition

# Set the value of the property.
sketchDimension_var.textPosition = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
