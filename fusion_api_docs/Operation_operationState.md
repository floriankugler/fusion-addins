# Operation.operationState Property

Parent Object: [Operation](Operation.md)  

## Description

Gets the current state of this operation.

## Syntax

"operation_var" is a variable referencing an Operation object.  

```python
# Get the value of the property.
propertyValue = operation_var.operationState
```

## Property Value

This is a read only property whose value is an [OperationStates](OperationStates.md).

## Version

Introduced in version January 2016
