# BaseComponent.decals Property

Parent Object: [BaseComponent](BaseComponent.md)  

## Description

Returns the decals collection associated with this component. This provides access to the existing decals and supports the creation of new decals.

## Syntax

"baseComponent_var" is a variable referencing a BaseComponent object.  

```python
# Get the value of the property.
propertyValue = baseComponent_var.decals
```

## Property Value

This is a read only property whose value is a [Decals](Decals.md).

## Version

Introduced in version September 2024
