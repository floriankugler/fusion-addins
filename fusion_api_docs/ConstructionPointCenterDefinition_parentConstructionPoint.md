# ConstructionPointCenterDefinition.parentConstructionPoint Property

Parent Object: [ConstructionPointCenterDefinition](ConstructionPointCenterDefinition.md)  

## Description

Returns the ConstructionPoint object

## Syntax

"constructionPointCenterDefinition_var" is a variable referencing a ConstructionPointCenterDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionPointCenterDefinition_var.parentConstructionPoint
```

## Property Value

This is a read only property whose value is a [ConstructionPoint](ConstructionPoint.md).

## Version

Introduced in version August 2014
