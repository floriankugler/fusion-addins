# CornerClosureFeature.timelineObject Property

Parent Object: [CornerClosureFeature](CornerClosureFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"cornerClosureFeature_var" is a variable referencing a CornerClosureFeature object.  

```python
# Get the value of the property.
propertyValue = cornerClosureFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version January 2026
