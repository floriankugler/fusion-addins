# SketchAngularDimension.parameter Property

Parent Object: [SketchAngularDimension](SketchAngularDimension.md)  

## Description

Returns the associated parameter or null if there is no associated parameter.

## Syntax

"sketchAngularDimension_var" is a variable referencing a SketchAngularDimension object.  

```python
# Get the value of the property.
propertyValue = sketchAngularDimension_var.parameter
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version August 2014
