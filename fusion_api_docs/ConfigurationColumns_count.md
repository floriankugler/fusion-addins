# ConfigurationColumns.count Property

Parent Object: [ConfigurationColumns](ConfigurationColumns.md)  

## Description

Returns the number of columns in the table. The name column is not included.

## Syntax

"configurationColumns_var" is a variable referencing a ConfigurationColumns object.  

```python
# Get the value of the property.
propertyValue = configurationColumns_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version January 2024
