# SketchLine.geometry Property

Parent Object: [SketchLine](SketchLine.md)  

## Description

Returns the transient geometry of the line which provides geometry information about the line. The returned geometry is always in sketch space.

## Syntax

"sketchLine_var" is a variable referencing a SketchLine object.  

```python
# Get the value of the property.
propertyValue = sketchLine_var.geometry
```

## Property Value

This is a read only property whose value is a [Line3D](Line3D.md).

## Version

Introduced in version August 2014
