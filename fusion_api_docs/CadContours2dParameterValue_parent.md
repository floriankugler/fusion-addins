# CadContours2dParameterValue.parent Property

Parent Object: [CadContours2dParameterValue](CadContours2dParameterValue.md)  

## Description

Get the parameter object that the value is associated with.

## Syntax

"cadContours2dParameterValue_var" is a variable referencing a CadContours2dParameterValue object.  

```python
# Get the value of the property.
propertyValue = cadContours2dParameterValue_var.parent
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version April 2023
