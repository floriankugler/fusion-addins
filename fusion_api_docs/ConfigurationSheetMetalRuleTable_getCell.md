# ConfigurationSheetMetalRuleTable.getCell Method

Parent Object: [ConfigurationSheetMetalRuleTable](ConfigurationSheetMetalRuleTable.md)  

## Description

Returns the cell at the specified row and column.

## Syntax

"configurationSheetMetalRuleTable_var" is a variable referencing a [ConfigurationSheetMetalRuleTable](ConfigurationSheetMetalRuleTable.md) object.

```python
returnValue = configurationSheetMetalRuleTable_var.getCell(column, row)
```

## Parameters

| Name | Type | Description |
|----|----|----|
| column | uinteger | The index of the column the cell is in. An index of 0 is the first column and does not include the name column. |
| row | uinteger | The index of the row the cell is in. An index of 0 is the first row and does not include the header row. |

## Version

Introduced in version January 2024
