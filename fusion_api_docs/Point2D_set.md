# Point2D.set Method

Parent Object: [Point2D](Point2D.md)  

## Description

Sets the coordinates of the point by specifying the x, y coordinates.

## Syntax

"point2D_var" is a variable referencing a [Point2D](Point2D.md) object.

```python
returnValue = point2D_var.set(x, y)
```

## Return Value

| Type    | Description                |
|---------|----------------------------|
| boolean | Returns true if successful |

## Parameters

| Name | Type   | Description                    |
|------|--------|--------------------------------|
| x    | double | The x coordinate of the point. |
| y    | double | The y coordinate of the point. |

## Version

Introduced in version August 2014
