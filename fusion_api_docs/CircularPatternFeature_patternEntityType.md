# CircularPatternFeature.patternEntityType Property

Parent Object: [CircularPatternFeature](CircularPatternFeature.md)  

## Description

Returns the type of entities the pattern consists of. This can be used to help determine the type of results that will be found in the pattern elements.

## Syntax

"circularPatternFeature_var" is a variable referencing a CircularPatternFeature object.  

```python
# Get the value of the property.
propertyValue = circularPatternFeature_var.patternEntityType
```

## Property Value

This is a read only property whose value is a [PatternEntityTypes](PatternEntityTypes.md).

## Version

Introduced in version July 2015
