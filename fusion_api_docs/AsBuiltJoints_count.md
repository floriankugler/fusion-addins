# AsBuiltJoints.count Property

Parent Object: [AsBuiltJoints](AsBuiltJoints.md)  

## Description

Returns number of joint origins in the collection.

## Syntax

"asBuiltJoints_var" is a variable referencing an AsBuiltJoints object.  

```python
# Get the value of the property.
propertyValue = asBuiltJoints_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version September 2015
