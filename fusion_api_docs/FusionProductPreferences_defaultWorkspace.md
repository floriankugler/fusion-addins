# FusionProductPreferences.defaultWorkspace Property

Parent Object: [FusionProductPreferences](FusionProductPreferences.md)  

## Description

Gets and sets the Default workspace setting. (Model, Sculpt or Patch)

## Syntax

"fusionProductPreferences_var" is a variable referencing a FusionProductPreferences object.  

```python
# Get the value of the property.
propertyValue = fusionProductPreferences_var.defaultWorkspace

# Set the value of the property.
fusionProductPreferences_var.defaultWorkspace = propertyValue
```

## Property Value

This is a read/write property whose value is a [DefaultWorkspaces](DefaultWorkspaces.md).

## Version

Introduced in version August 2014
