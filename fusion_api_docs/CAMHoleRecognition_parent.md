# CAMHoleRecognition.parent Property

Parent Object: [CAMHoleRecognition](CAMHoleRecognition.md)  

## Description

Returns the parent Setup, Folder or Pattern for this Folder.

## Syntax

"cAMHoleRecognition_var" is a variable referencing a CAMHoleRecognition object.  

```python
# Get the value of the property.
propertyValue = cAMHoleRecognition_var.parent
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version April 2023
