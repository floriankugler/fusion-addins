# CombineFeature.timelineObject Property

Parent Object: [CombineFeature](CombineFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"combineFeature_var" is a variable referencing a CombineFeature object.  

```python
# Get the value of the property.
propertyValue = combineFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version November 2014
