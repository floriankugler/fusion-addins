# PathPatternFeature.timelineObject Property

Parent Object: [PathPatternFeature](PathPatternFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"pathPatternFeature_var" is a variable referencing a PathPatternFeature object.  

```python
# Get the value of the property.
propertyValue = pathPatternFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version November 2014
