# SketchConicCurve.isFixed Property

Parent Object: [SketchConicCurve](SketchConicCurve.md)  

## Description

Indicates if this geometry is "fixed".

## Syntax

"sketchConicCurve_var" is a variable referencing a SketchConicCurve object.  

```python
# Get the value of the property.
propertyValue = sketchConicCurve_var.isFixed

# Set the value of the property.
sketchConicCurve_var.isFixed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2015
