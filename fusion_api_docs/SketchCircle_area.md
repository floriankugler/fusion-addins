# SketchCircle.area Property

Parent Object: [SketchCircle](SketchCircle.md)  

## Description

Returns the area of the circle in square centimeters.

## Syntax

"sketchCircle_var" is a variable referencing a SketchCircle object.  

```python
# Get the value of the property.
propertyValue = sketchCircle_var.area
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version August 2014
