# Sketch.name Property

Parent Object: [Sketch](Sketch.md)  

## Description

Gets and sets the name of this sketch as seen in the browser and timeline.

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.name

# Set the value of the property.
sketch_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version August 2014
