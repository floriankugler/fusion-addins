# ExtrudeFeature.extentType Property

Parent Object: [ExtrudeFeature](ExtrudeFeature.md)  

## Description

Returns a value indicating how the extent is defined for this extrude.

## Syntax

"extrudeFeature_var" is a variable referencing an ExtrudeFeature object.  

```python
# Get the value of the property.
propertyValue = extrudeFeature_var.extentType
```

## Property Value

This is a read only property whose value is a [FeatureExtentTypes](FeatureExtentTypes.md).
