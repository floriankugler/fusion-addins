# SharedLink.linkURL Property

Parent Object: [SharedLink](SharedLink.md)  

## Description

Returns the URL of the shared link. Returns an empty string in the case where isShared is False.

## Syntax

"sharedLink_var" is a variable referencing a SharedLink object.  

```python
# Get the value of the property.
propertyValue = sharedLink_var.linkURL
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version May 2024
