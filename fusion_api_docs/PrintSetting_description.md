# PrintSetting.description Property

Parent Object: [PrintSetting](PrintSetting.md)  

## Description

Gets and sets the description of the PrintSetting.

## Syntax

"printSetting_var" is a variable referencing a PrintSetting object.  

```python
# Get the value of the property.
propertyValue = printSetting_var.description

# Set the value of the property.
printSetting_var.description = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version April 2023
