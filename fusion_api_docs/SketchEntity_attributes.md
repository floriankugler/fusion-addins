# SketchEntity.attributes Property

Parent Object: [SketchEntity](SketchEntity.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"sketchEntity_var" is a variable referencing a SketchEntity object.  

```python
# Get the value of the property.
propertyValue = sketchEntity_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
