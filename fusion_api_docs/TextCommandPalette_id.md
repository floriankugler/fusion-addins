# TextCommandPalette.id Property

Parent Object: [TextCommandPalette](TextCommandPalette.md)  

## Description

Gets The unique, language independent, ID of this palette.

## Syntax

"textCommandPalette_var" is a variable referencing a TextCommandPalette object.  

```python
# Get the value of the property.
propertyValue = textCommandPalette_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2017
