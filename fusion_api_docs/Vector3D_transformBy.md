# Vector3D.transformBy Method

Parent Object: [Vector3D](Vector3D.md)  

## Description

Transform this vector by the specified matrix.

## Syntax

"vector3D_var" is a variable referencing a [Vector3D](Vector3D.md) object.

```python
returnValue = vector3D_var.transformBy(matrix)
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name   | Type                     | Description                |
|--------|--------------------------|----------------------------|
| matrix | [Matrix3D](Matrix3D.md) | The transformation matrix. |

## Version

Introduced in version August 2014
