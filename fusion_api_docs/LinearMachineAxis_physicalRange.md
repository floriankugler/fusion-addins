# LinearMachineAxis.physicalRange Property

Parent Object: [LinearMachineAxis](LinearMachineAxis.md)  

## Description

Range of possible values for this axis. Units are cm for linear axes or radians for rotary axes.

## Syntax

"linearMachineAxis_var" is a variable referencing a LinearMachineAxis object.  

```python
# Get the value of the property.
propertyValue = linearMachineAxis_var.physicalRange

# Set the value of the property.
linearMachineAxis_var.physicalRange = propertyValue
```

## Property Value

This is a read/write property whose value is a [MachineAxisRange](MachineAxisRange.md).

## Version

Introduced in version April 2023
