# DocumentReferences.isValid Property

Parent Object: [DocumentReferences](DocumentReferences.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"documentReferences_var" is a variable referencing a DocumentReferences object.  

```python
# Get the value of the property.
propertyValue = documentReferences_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version June 2017
