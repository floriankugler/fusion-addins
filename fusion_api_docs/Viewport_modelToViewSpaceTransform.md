# Viewport.modelToViewSpaceTransform Property

Parent Object: [Viewport](Viewport.md)  

## Description

Returns a transformation matrix that defines the transform from model to viewport space.

## Syntax

"viewport_var" is a variable referencing a Viewport object.  

```python
# Get the value of the property.
propertyValue = viewport_var.modelToViewSpaceTransform
```

## Property Value

This is a read only property whose value is a [Matrix3D](Matrix3D.md).

## Version

Introduced in version October 2022
