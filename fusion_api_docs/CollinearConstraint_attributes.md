# CollinearConstraint.attributes Property

Parent Object: [CollinearConstraint](CollinearConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"collinearConstraint_var" is a variable referencing a CollinearConstraint object.  

```python
# Get the value of the property.
propertyValue = collinearConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
