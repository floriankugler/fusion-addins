# MachineToolStation.maxToolLength Property

Parent Object: [MachineToolStation](MachineToolStation.md)  

## Description

The maximum length in cm of the tool that can be held by this tool station.

## Syntax

"machineToolStation_var" is a variable referencing a MachineToolStation object.  

```python
# Get the value of the property.
propertyValue = machineToolStation_var.maxToolLength

# Set the value of the property.
machineToolStation_var.maxToolLength = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version July 2025
