# CommandInput.isVisible Property

Parent Object: [CommandInput](CommandInput.md)  

## Description

Gets or sets if this input will be visible to the user.  

Setting a SelectionCommandInput to be invisible will clear any selections it currently has.

## Syntax

"commandInput_var" is a variable referencing a CommandInput object.  

```python
# Get the value of the property.
propertyValue = commandInput_var.isVisible

# Set the value of the property.
commandInput_var.isVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
