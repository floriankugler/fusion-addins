# OffsetFeatureInput.distance Property

Parent Object: [OffsetFeatureInput](OffsetFeatureInput.md)  

## Description

Gets and sets the ValueInput object that defines the offset distance. A positive distance value results in an offset in the positive normal direction of the faces.

## Syntax

"offsetFeatureInput_var" is a variable referencing an OffsetFeatureInput object.  

```python
# Get the value of the property.
propertyValue = offsetFeatureInput_var.distance

# Set the value of the property.
offsetFeatureInput_var.distance = propertyValue
```

## Property Value

This is a read/write property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version June 2015
