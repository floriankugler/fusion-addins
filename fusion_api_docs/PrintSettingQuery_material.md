# PrintSettingQuery.material Property

Parent Object: [PrintSettingQuery](PrintSettingQuery.md)  

## Description

The case-insensitive material specifies material of the MPBF PrintSetting.

## Syntax

"printSettingQuery_var" is a variable referencing a PrintSettingQuery object.  

```python
# Get the value of the property.
propertyValue = printSettingQuery_var.material

# Set the value of the property.
printSettingQuery_var.material = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version April 2023
