# BossFeature.isDirectionFlipped Property

Parent Object: [BossFeature](BossFeature.md)  

## Description

Gets and sets if the direction of the boss (or boss connection) is flipped.

## Syntax

"bossFeature_var" is a variable referencing a BossFeature object.  

```python
# Get the value of the property.
propertyValue = bossFeature_var.isDirectionFlipped

# Set the value of the property.
bossFeature_var.isDirectionFlipped = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version October 2022
