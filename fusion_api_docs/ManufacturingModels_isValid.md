# ManufacturingModels.isValid Property

Parent Object: [ManufacturingModels](ManufacturingModels.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"manufacturingModels_var" is a variable referencing a ManufacturingModels object.  

```python
# Get the value of the property.
propertyValue = manufacturingModels_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version April 2023
