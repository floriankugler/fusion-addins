# DrawingDocument.drawing Property

Parent Object: [DrawingDocument](DrawingDocument.md)  

## Description

Returns the Drawing product object associated with this drawing document.

## Syntax

"drawingDocument_var" is a variable referencing a DrawingDocument object.  

```python
# Get the value of the property.
propertyValue = drawingDocument_var.drawing
```

## Property Value

This is a read only property whose value is a [Drawing](Drawing.md).

## Version

Introduced in version December 2020
