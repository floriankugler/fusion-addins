# RefoldFeature.isSuppressed Property

Parent Object: [RefoldFeature](RefoldFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"refoldFeature_var" is a variable referencing a RefoldFeature object.  

```python
# Get the value of the property.
propertyValue = refoldFeature_var.isSuppressed

# Set the value of the property.
refoldFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2020
