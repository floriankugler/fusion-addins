# Analyses.zebraAnalyses Property

Parent Object: [Analyses](Analyses.md)  

## Description

Returns the ZebraAnalyses object, which provides access to any existing ZebraAnalysis objects in the design.

## Syntax

"analyses_var" is a variable referencing an Analyses object.  

```python
# Get the value of the property.
propertyValue = analyses_var.zebraAnalyses
```

## Property Value

This is a read only property whose value is a [ZebraAnalyses](ZebraAnalyses.md).

## Version

Introduced in version January 2023
