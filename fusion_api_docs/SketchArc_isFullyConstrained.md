# SketchArc.isFullyConstrained Property

Parent Object: [SketchArc](SketchArc.md)  

## Description

Indicates if this sketch entity is fully constrained.

## Syntax

"sketchArc_var" is a variable referencing a SketchArc object.  

```python
# Get the value of the property.
propertyValue = sketchArc_var.isFullyConstrained
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2016
