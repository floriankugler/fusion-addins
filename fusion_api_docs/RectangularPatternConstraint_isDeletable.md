# RectangularPatternConstraint.isDeletable Property

Parent Object: [RectangularPatternConstraint](RectangularPatternConstraint.md)  

## Description

Indicates if this constraint is deletable.

## Syntax

"rectangularPatternConstraint_var" is a variable referencing a RectangularPatternConstraint object.  

```python
# Get the value of the property.
propertyValue = rectangularPatternConstraint_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version March 2016
