# AccessibilityAnalysis.isValid Property

Parent Object: [AccessibilityAnalysis](AccessibilityAnalysis.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"accessibilityAnalysis_var" is a variable referencing an AccessibilityAnalysis object.  

```python
# Get the value of the property.
propertyValue = accessibilityAnalysis_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2023
