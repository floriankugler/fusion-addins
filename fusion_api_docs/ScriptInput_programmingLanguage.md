# ScriptInput.programmingLanguage Property

Parent Object: [ScriptInput](ScriptInput.md)  

## Description

Gets and sets which programming language the new script or add-in will use.

## Syntax

"scriptInput_var" is a variable referencing a ScriptInput object.  

```python
# Get the value of the property.
propertyValue = scriptInput_var.programmingLanguage

# Set the value of the property.
scriptInput_var.programmingLanguage = propertyValue
```

## Property Value

This is a read/write property whose value is a [ProgrammingLanguages](ProgrammingLanguages.md).

## Version

Introduced in version October 2023
