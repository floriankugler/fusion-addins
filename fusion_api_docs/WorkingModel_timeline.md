# WorkingModel.timeline Property

Parent Object: [WorkingModel](WorkingModel.md)  

## Description

Returns the timeline associated with this design.

## Syntax

"workingModel_var" is a variable referencing a WorkingModel object.  

```python
# Get the value of the property.
propertyValue = workingModel_var.timeline
```

## Property Value

This is a read only property whose value is a [Timeline](Timeline.md).

## Version

Introduced in version January 2024
