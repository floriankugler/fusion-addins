# CutPasteBody.isSuppressed Property

Parent Object: [CutPasteBody](CutPasteBody.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"cutPasteBody_var" is a variable referencing a CutPasteBody object.  

```python
# Get the value of the property.
propertyValue = cutPasteBody_var.isSuppressed

# Set the value of the property.
cutPasteBody_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version June 2017
