# CoincidentConstraint.parentSketch Property

Parent Object: [CoincidentConstraint](CoincidentConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"coincidentConstraint_var" is a variable referencing a CoincidentConstraint object.  

```python
# Get the value of the property.
propertyValue = coincidentConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
