# CoincidentToSurfaceConstraint.point Property

Parent Object: [CoincidentToSurfaceConstraint](CoincidentToSurfaceConstraint.md)  

## Description

Returns the sketch point that is constrained.

## Syntax

"coincidentToSurfaceConstraint_var" is a variable referencing a CoincidentToSurfaceConstraint object.  

```python
# Get the value of the property.
propertyValue = coincidentToSurfaceConstraint_var.point
```

## Property Value

This is a read only property whose value is a [SketchPoint](SketchPoint.md).

## Version

Introduced in version September 2023
