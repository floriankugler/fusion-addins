# FusionDocument.products Property

Parent Object: [FusionDocument](FusionDocument.md)  

## Description

Returns the products associated with this document.

## Syntax

"fusionDocument_var" is a variable referencing a FusionDocument object.  

```python
# Get the value of the property.
propertyValue = fusionDocument_var.products
```

## Property Value

This is a read only property whose value is a [Products](Products.md).
