# RadioButtonGroupCommandInput.isVisible Property

Parent Object: [RadioButtonGroupCommandInput](RadioButtonGroupCommandInput.md)  

## Description

Gets or sets if this input will be visible to the user.  

Setting a SelectionCommandInput to be invisible will clear any selections it currently has.

## Syntax

"radioButtonGroupCommandInput_var" is a variable referencing a RadioButtonGroupCommandInput object.  

```python
# Get the value of the property.
propertyValue = radioButtonGroupCommandInput_var.isVisible

# Set the value of the property.
radioButtonGroupCommandInput_var.isVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version July 2015
