# ConstructionPlane.isVisible Property

Parent Object: [ConstructionPlane](ConstructionPlane.md)  

## Description

Indicates if the construction plane is visible. This property is affected by the AssemblyContext of the construction plane.

## Syntax

"constructionPlane_var" is a variable referencing a ConstructionPlane object.  

```python
# Get the value of the property.
propertyValue = constructionPlane_var.isVisible
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
