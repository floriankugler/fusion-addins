# FlatPatternProduct.librarySheetMetalRules Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Gets the collection of sheet metal rules in the sheet metal rule library.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.librarySheetMetalRules
```

## Property Value

This is a read only property whose value is a [SheetMetalRules](SheetMetalRules.md).

## Version

Introduced in version November 2022
