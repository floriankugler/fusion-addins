# ThickenFeature.parentComponent Property

Parent Object: [ThickenFeature](ThickenFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"thickenFeature_var" is a variable referencing a ThickenFeature object.  

```python
# Get the value of the property.
propertyValue = thickenFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version June 2015
