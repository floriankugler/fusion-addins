# AreaProperties.rotationToPrincipal Property

Parent Object: [AreaProperties](AreaProperties.md)  

## Description

Gets the angle of rotation of the principal axes.

## Syntax

"areaProperties_var" is a variable referencing an AreaProperties object.  

```python
# Get the value of the property.
propertyValue = areaProperties_var.rotationToPrincipal
```

## Property Value

This is a read only property whose value is a double.
