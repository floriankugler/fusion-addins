# BooleanParameterValue.parent Property

Parent Object: [BooleanParameterValue](BooleanParameterValue.md)  

## Description

Get the parameter object that the value is associated with.

## Syntax

"booleanParameterValue_var" is a variable referencing a BooleanParameterValue object.  

```python
# Get the value of the property.
propertyValue = booleanParameterValue_var.parent
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version September 2021
