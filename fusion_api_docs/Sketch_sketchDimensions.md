# Sketch.sketchDimensions Property

Parent Object: [Sketch](Sketch.md)  

## Description

Returns the sketch dimensions collection associated with this sketch. This provides access to the existing sketch dimensions and supports the creation of new sketch dimensions.

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.sketchDimensions
```

## Property Value

This is a read only property whose value is a [SketchDimensions](SketchDimensions.md).
