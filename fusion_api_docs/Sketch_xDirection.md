# Sketch.xDirection Property

Parent Object: [Sketch](Sketch.md)  

## Description

Returns the X direction of the sketch as defined in model space.

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.xDirection
```

## Property Value

This is a read only property whose value is a [Vector3D](Vector3D.md).

## Version

Introduced in version August 2014
