# Line3D.evaluator Property

Parent Object: [Line3D](Line3D.md)  

## Description

Returns an evaluator object that lets you perform additional evaluations on the curve.

## Syntax

"line3D_var" is a variable referencing a Line3D object.  

```python
# Get the value of the property.
propertyValue = line3D_var.evaluator
```

## Property Value

This is a read only property whose value is a [CurveEvaluator3D](CurveEvaluator3D.md).

## Version

Introduced in version August 2014
