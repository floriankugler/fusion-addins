# DerivedParameter.deriveFeature Property

Parent Object: [DerivedParameter](DerivedParameter.md)  

## Description

Returns the DeriveFeature that derives this parameter from another design.

## Syntax

"derivedParameter_var" is a variable referencing a DerivedParameter object.  

```python
# Get the value of the property.
propertyValue = derivedParameter_var.deriveFeature
```

## Property Value

This is a read only property whose value is a [DeriveFeature](DeriveFeature.md).

## Version

Introduced in version January 2026
