# HoleFeature.parentComponent Property

Parent Object: [HoleFeature](HoleFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"holeFeature_var" is a variable referencing a HoleFeature object.  

```python
# Get the value of the property.
propertyValue = holeFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version August 2014
