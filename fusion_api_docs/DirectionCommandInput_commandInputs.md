# DirectionCommandInput.commandInputs Property

Parent Object: [DirectionCommandInput](DirectionCommandInput.md)  

## Description

Gets the CommandInputs class of the parent, which can be a Command, GroupCommandInput or TabCommandInput.

## Syntax

"directionCommandInput_var" is a variable referencing a DirectionCommandInput object.  

```python
# Get the value of the property.
propertyValue = directionCommandInput_var.commandInputs
```

## Property Value

This is a read only property whose value is a [CommandInputs](CommandInputs.md).

## Version

Introduced in version January 2016
