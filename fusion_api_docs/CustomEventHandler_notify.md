# CustomEventHandler.notify Method

Parent Object: [CustomEventHandler](CustomEventHandler.md)  

## Description

The function called by Fusion when the associated event is fired.

## Syntax

"customEventHandler_var" is a variable referencing a [CustomEventHandler](CustomEventHandler.md) object.

```python
returnValue = customEventHandler_var.notify(eventArgs)
```

## Parameters

| Name | Type | Description |
|----|----|----|
| eventArgs | [CustomEventArgs](CustomEventArgs.md) | Returns an object that provides access to additional information associated with the event. |

## Version

Introduced in version January 2017
