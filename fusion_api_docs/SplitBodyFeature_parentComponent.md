# SplitBodyFeature.parentComponent Property

Parent Object: [SplitBodyFeature](SplitBodyFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"splitBodyFeature_var" is a variable referencing a SplitBodyFeature object.  

```python
# Get the value of the property.
propertyValue = splitBodyFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version June 2015
