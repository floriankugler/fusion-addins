# SketchFixedSpline.startSketchPoint Property

Parent Object: [SketchFixedSpline](SketchFixedSpline.md)  

## Description

The sketch point at the start of the spline.

## Syntax

"sketchFixedSpline_var" is a variable referencing a SketchFixedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFixedSpline_var.startSketchPoint
```

## Property Value

This is a read only property whose value is a [SketchPoint](SketchPoint.md).

## Version

Introduced in version August 2014
