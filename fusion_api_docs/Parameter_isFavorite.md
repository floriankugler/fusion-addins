# Parameter.isFavorite Property

Parent Object: [Parameter](Parameter.md)  

## Description

Gets and sets whether this parameter is included in the Favorites list in the parameters dialog

## Syntax

"parameter_var" is a variable referencing a Parameter object.  

```python
# Get the value of the property.
propertyValue = parameter_var.isFavorite

# Set the value of the property.
parameter_var.isFavorite = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
