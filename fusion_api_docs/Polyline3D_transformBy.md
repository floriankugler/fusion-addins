# Polyline3D.transformBy Method

Parent Object: [Polyline3D](Polyline3D.md)  

## Description

Transforms this curve in 3D space.

## Syntax

"polyline3D_var" is a variable referencing a [Polyline3D](Polyline3D.md) object.

```python
returnValue = polyline3D_var.transformBy(matrix)
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Return true if the transform was successful. |

## Parameters

| Name | Type | Description |
|----|----|----|
| matrix | [Matrix3D](Matrix3D.md) | A 3D matrix that defines the transform to apply to the curve. |

## Version

Introduced in version September 2024
