# PlasticRule.draftAngle Property

Parent Object: [PlasticRule](PlasticRule.md)  

## Description

The draft angle used for plastic features. When using a float to set the value, it is defined in radians. When using a string to set the expression it uses degrees.

## Syntax

"plasticRule_var" is a variable referencing a PlasticRule object.  

```python
# Get the value of the property.
propertyValue = plasticRule_var.draftAngle
```

## Property Value

This is a read only property whose value is a [PlasticRuleValue](PlasticRuleValue.md).

## Version

Introduced in version January 2024
