# MachineAxisConfiguration.deleteMe Method

Parent Object: [MachineAxisConfiguration](MachineAxisConfiguration.md)  

## Description

Delete this axis configuration from the controller configuration.

## Syntax

"machineAxisConfiguration_var" is a variable referencing a [MachineAxisConfiguration](MachineAxisConfiguration.md) object.

```python
returnValue = machineAxisConfiguration_var.deleteMe()
```

## Version

Introduced in version April 2023
