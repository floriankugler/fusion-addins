# NurbsCurve3D.controlPointCount Property

Parent Object: [NurbsCurve3D](NurbsCurve3D.md)  

## Description

Gets the number of control points that define the curve.

## Syntax

"nurbsCurve3D_var" is a variable referencing a NurbsCurve3D object.  

```python
# Get the value of the property.
propertyValue = nurbsCurve3D_var.controlPointCount
```

## Property Value

This is a read only property whose value is an integer.

## Version

Introduced in version August 2014
