# Point2D.transformBy Method

Parent Object: [Point2D](Point2D.md)  

## Description

Transforms the point using the provided matrix.

## Syntax

"point2D_var" is a variable referencing a [Point2D](Point2D.md) object.

```python
returnValue = point2D_var.transformBy(matrix)
```

## Return Value

| Type    | Description                |
|---------|----------------------------|
| boolean | Returns true if successful |

## Parameters

| Name | Type | Description |
|----|----|----|
| matrix | [Matrix2D](Matrix2D.md) | The Matrix2D object that defines the transformation |

## Version

Introduced in version August 2014
