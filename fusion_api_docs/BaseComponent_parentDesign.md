# BaseComponent.parentDesign Property

Parent Object: [BaseComponent](BaseComponent.md)  

## Description

Returns the parent product this component is owned by.

## Syntax

"baseComponent_var" is a variable referencing a BaseComponent object.  

```python
# Get the value of the property.
propertyValue = baseComponent_var.parentDesign
```

## Property Value

This is a read only property whose value is a [Design](Design.md).

## Version

Introduced in version August 2014
