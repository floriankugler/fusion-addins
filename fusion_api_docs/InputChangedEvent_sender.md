# InputChangedEvent.sender Property

Parent Object: [InputChangedEvent](InputChangedEvent.md)  

## Description

The object that is firing the event. For example, in the case of a command input event this will return the command.

## Syntax

"inputChangedEvent_var" is a variable referencing an InputChangedEvent object.  

```python
# Get the value of the property.
propertyValue = inputChangedEvent_var.sender
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version August 2014
