# Features.untrimFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the Untrim features within the component and supports the creation of new Untrim features.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.untrimFeatures
```

## Property Value

This is a read only property whose value is a [UntrimFeatures](UntrimFeatures.md).

## Version

Introduced in version January 2021
