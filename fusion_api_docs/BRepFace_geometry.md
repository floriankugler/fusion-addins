# BRepFace.geometry Property

Parent Object: [BRepFace](BRepFace.md)  

## Description

Returns the underlying surface geometry of this face

## Syntax

"bRepFace_var" is a variable referencing a BRepFace object.  

```python
# Get the value of the property.
propertyValue = bRepFace_var.geometry
```

## Property Value

This is a read only property whose value is a [Surface](Surface.md).

## Version

Introduced in version August 2014
