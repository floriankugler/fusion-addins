# SketchFixedSpline.isConstruction Property

Parent Object: [SketchFixedSpline](SketchFixedSpline.md)  

## Description

Gets and sets whether this curve is construction geometry.

## Syntax

"sketchFixedSpline_var" is a variable referencing a SketchFixedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFixedSpline_var.isConstruction

# Set the value of the property.
sketchFixedSpline_var.isConstruction = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
