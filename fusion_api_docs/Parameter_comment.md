# Parameter.comment Property

Parent Object: [Parameter](Parameter.md)  

## Description

The comment associated with this parameter

## Syntax

"parameter_var" is a variable referencing a Parameter object.  

```python
# Get the value of the property.
propertyValue = parameter_var.comment

# Set the value of the property.
parameter_var.comment = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version August 2014
