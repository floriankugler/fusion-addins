# ImageCommandInput.parentCommand Property

Parent Object: [ImageCommandInput](ImageCommandInput.md)  

## Description

Gets the parent Command.

## Syntax

"imageCommandInput_var" is a variable referencing an ImageCommandInput object.  

```python
# Get the value of the property.
propertyValue = imageCommandInput_var.parentCommand
```

## Property Value

This is a read only property whose value is a [Command](Command.md).

## Version

Introduced in version July 2015
