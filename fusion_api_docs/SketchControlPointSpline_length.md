# SketchControlPointSpline.length Property

Parent Object: [SketchControlPointSpline](SketchControlPointSpline.md)  

## Description

Returns the length of the curve in centimeters.

## Syntax

"sketchControlPointSpline_var" is a variable referencing a SketchControlPointSpline object.  

```python
# Get the value of the property.
propertyValue = sketchControlPointSpline_var.length
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version July 2022
