# SplitFaceFeature.timelineObject Property

Parent Object: [SplitFaceFeature](SplitFaceFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"splitFaceFeature_var" is a variable referencing a SplitFaceFeature object.  

```python
# Get the value of the property.
propertyValue = splitFaceFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version June 2015
