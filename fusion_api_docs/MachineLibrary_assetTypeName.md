# MachineLibrary.assetTypeName Property

Parent Object: [MachineLibrary](MachineLibrary.md)  

## Description

Get the name of the asset type which can be accessed by the library.

## Syntax

"machineLibrary_var" is a variable referencing a MachineLibrary object.  

```python
# Get the value of the property.
propertyValue = machineLibrary_var.assetTypeName
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
