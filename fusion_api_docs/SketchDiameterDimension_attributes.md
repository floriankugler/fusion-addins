# SketchDiameterDimension.attributes Property

Parent Object: [SketchDiameterDimension](SketchDiameterDimension.md)  

## Description

Returns the collection of attributes associated with this sketch dimension.

## Syntax

"sketchDiameterDimension_var" is a variable referencing a SketchDiameterDimension object.  

```python
# Get the value of the property.
propertyValue = sketchDiameterDimension_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
