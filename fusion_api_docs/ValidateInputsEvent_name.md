# ValidateInputsEvent.name Property

Parent Object: [ValidateInputsEvent](ValidateInputsEvent.md)  

## Description

The name of the event - e.g. "DocumentOpening"

## Syntax

"validateInputsEvent_var" is a variable referencing a ValidateInputsEvent object.  

```python
# Get the value of the property.
propertyValue = validateInputsEvent_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2014
