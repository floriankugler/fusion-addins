# HemFeature.attributes Property

Parent Object: [HemFeature](HemFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"hemFeature_var" is a variable referencing a HemFeature object.  

```python
# Get the value of the property.
propertyValue = hemFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version September 2025
