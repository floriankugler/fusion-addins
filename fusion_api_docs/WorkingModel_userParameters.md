# WorkingModel.userParameters Property

Parent Object: [WorkingModel](WorkingModel.md)  

## Description

Returns the collection of User Parameters in a design

## Syntax

"workingModel_var" is a variable referencing a WorkingModel object.  

```python
# Get the value of the property.
propertyValue = workingModel_var.userParameters
```

## Property Value

This is a read only property whose value is a [UserParameters](UserParameters.md).

## Version

Introduced in version January 2024
