# Cylinder.copy Method

Parent Object: [Cylinder](Cylinder.md)  

## Description

Creates and returns an independent copy of this Cylinder object.

## Syntax

"cylinder_var" is a variable referencing a [Cylinder](Cylinder.md) object.

```python
returnValue = cylinder_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [Cylinder](Cylinder.md) | Returns a new Cylinder object that is a copy of this Cylinder object. |

## Version

Introduced in version August 2014
