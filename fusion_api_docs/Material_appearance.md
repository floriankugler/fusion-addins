# Material.appearance Property

Parent Object: [Material](Material.md)  

## Description

Gets the Appearance of this material.

## Syntax

"material_var" is a variable referencing a Material object.  

```python
# Get the value of the property.
propertyValue = material_var.appearance
```

## Property Value

This is a read only property whose value is an [Appearance](Appearance.md).

## Version

Introduced in version August 2014
