# SketchEntity.isFullyConstrained Property

Parent Object: [SketchEntity](SketchEntity.md)  

## Description

Indicates if this sketch entity is fully constrained.

## Syntax

"sketchEntity_var" is a variable referencing a SketchEntity object.  

```python
# Get the value of the property.
propertyValue = sketchEntity_var.isFullyConstrained
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2016
