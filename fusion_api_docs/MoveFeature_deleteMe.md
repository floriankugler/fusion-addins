# MoveFeature.deleteMe Method

Parent Object: [MoveFeature](MoveFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"moveFeature_var" is a variable referencing a [MoveFeature](MoveFeature.md) object.

```python
returnValue = moveFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version March 2015
