# EllipticalArc2D.startPoint Property

Parent Object: [EllipticalArc2D](EllipticalArc2D.md)  

## Description

Gets the position of the start point of the elliptical arc.

## Syntax

"ellipticalArc2D_var" is a variable referencing an EllipticalArc2D object.  

```python
# Get the value of the property.
propertyValue = ellipticalArc2D_var.startPoint
```

## Property Value

This is a read only property whose value is a [Point2D](Point2D.md).

## Version

Introduced in version August 2014
