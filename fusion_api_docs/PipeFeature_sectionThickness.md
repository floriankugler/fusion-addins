# PipeFeature.sectionThickness Property

Parent Object: [PipeFeature](PipeFeature.md)  

## Description

Gets the section thickness of the Pipe.  

If the pipe is not hollow, this will return null.

## Syntax

"pipeFeature_var" is a variable referencing a PipeFeature object.  

```python
# Get the value of the property.
propertyValue = pipeFeature_var.sectionThickness
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version October 2023
