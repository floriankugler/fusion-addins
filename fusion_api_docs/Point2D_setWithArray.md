# Point2D.setWithArray Method

Parent Object: [Point2D](Point2D.md)  

## Description

Sets the coordinates of the point using an array as input.

## Syntax

"point2D_var" is a variable referencing a [Point2D](Point2D.md) object.

```python
returnValue = point2D_var.setWithArray(coordinates)
```

## Return Value

| Type    | Description                |
|---------|----------------------------|
| boolean | Returns true if successful |

## Parameters

| Name        | Type       | Description                                        |
|-------------|------------|----------------------------------------------------|
| coordinates | double\[\] | An array that defines the coordinates of the point |

## Version

Introduced in version August 2014
