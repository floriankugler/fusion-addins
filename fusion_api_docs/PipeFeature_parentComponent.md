# PipeFeature.parentComponent Property

Parent Object: [PipeFeature](PipeFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"pipeFeature_var" is a variable referencing a PipeFeature object.  

```python
# Get the value of the property.
propertyValue = pipeFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version September 2015
