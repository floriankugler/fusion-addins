# MoveFeature.attributes Property

Parent Object: [MoveFeature](MoveFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"moveFeature_var" is a variable referencing a MoveFeature object.  

```python
# Get the value of the property.
propertyValue = moveFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
