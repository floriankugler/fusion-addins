# CoilFeature.isSuppressed Property

Parent Object: [CoilFeature](CoilFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"coilFeature_var" is a variable referencing a CoilFeature object.  

```python
# Get the value of the property.
propertyValue = coilFeature_var.isSuppressed

# Set the value of the property.
coilFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version March 2015
