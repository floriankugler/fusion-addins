# RuledSurfaceFeature.healthState Property

Parent Object: [RuledSurfaceFeature](RuledSurfaceFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"ruledSurfaceFeature_var" is a variable referencing a RuledSurfaceFeature object.  

```python
# Get the value of the property.
propertyValue = ruledSurfaceFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version September 2020
