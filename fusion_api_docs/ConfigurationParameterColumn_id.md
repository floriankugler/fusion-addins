# ConfigurationParameterColumn.id Property

Parent Object: [ConfigurationParameterColumn](ConfigurationParameterColumn.md)  

## Description

The id of the column. This is constant and cannot be changed by the user.

## Syntax

"configurationParameterColumn_var" is a variable referencing a ConfigurationParameterColumn object.  

```python
# Get the value of the property.
propertyValue = configurationParameterColumn_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version January 2024
