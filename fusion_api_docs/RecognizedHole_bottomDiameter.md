# RecognizedHole.bottomDiameter Property

Parent Object: [RecognizedHole](RecognizedHole.md)  

## Description

Returns the diameter of the bottom of this hole (bottom diameter of the last segment contained in this hole).

## Syntax

"recognizedHole_var" is a variable referencing a RecognizedHole object.  

```python
# Get the value of the property.
propertyValue = recognizedHole_var.bottomDiameter
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version May 2023
