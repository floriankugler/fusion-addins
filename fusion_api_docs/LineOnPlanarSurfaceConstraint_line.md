# LineOnPlanarSurfaceConstraint.line Property

Parent Object: [LineOnPlanarSurfaceConstraint](LineOnPlanarSurfaceConstraint.md)  

## Description

Returns the SketchLine object that is constrained.

## Syntax

"lineOnPlanarSurfaceConstraint_var" is a variable referencing a LineOnPlanarSurfaceConstraint object.  

```python
# Get the value of the property.
propertyValue = lineOnPlanarSurfaceConstraint_var.line
```

## Property Value

This is a read only property whose value is a [SketchLine](SketchLine.md).

## Version

Introduced in version September 2023
