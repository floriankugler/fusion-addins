# TrimFeature.healthState Property

Parent Object: [TrimFeature](TrimFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"trimFeature_var" is a variable referencing a TrimFeature object.  

```python
# Get the value of the property.
propertyValue = trimFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
