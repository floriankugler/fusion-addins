# RecognizedHole.totalLength Property

Parent Object: [RecognizedHole](RecognizedHole.md)  

## Description

Returns the total length of all segments contained in this hole.

## Syntax

"recognizedHole_var" is a variable referencing a RecognizedHole object.  

```python
# Get the value of the property.
propertyValue = recognizedHole_var.totalLength
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version May 2023
