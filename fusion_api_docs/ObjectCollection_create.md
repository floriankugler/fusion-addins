# ObjectCollection.create Method

Parent Object: [ObjectCollection](ObjectCollection.md)  

## Description

Creates a new ObjectCollection object.

## Syntax

This is a static method.  

```python

returnValue = adsk.core.ObjectCollection.create()
```

## Return Value

| Type | Description |
|----|----|
| [ObjectCollection](ObjectCollection.md) | Returns the newly created ObjectCollection. |

## Version

Introduced in version August 2014
