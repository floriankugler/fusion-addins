# FusionUnitsManager.distanceDisplayUnits Property

Parent Object: [FusionUnitsManager](FusionUnitsManager.md)  

## Description

Gets and sets the default distance units for this design. Setting this property has the side effect of changing the unitSystem property to custom.

## Syntax

"fusionUnitsManager_var" is a variable referencing a FusionUnitsManager object.  

```python
# Get the value of the property.
propertyValue = fusionUnitsManager_var.distanceDisplayUnits

# Set the value of the property.
fusionUnitsManager_var.distanceDisplayUnits = propertyValue
```

## Property Value

This is a read/write property whose value is a [DistanceUnits](DistanceUnits.md).

## Version

Introduced in version August 2014
