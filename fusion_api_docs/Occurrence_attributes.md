# Occurrence.attributes Property

Parent Object: [Occurrence](Occurrence.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"occurrence_var" is a variable referencing an Occurrence object.  

```python
# Get the value of the property.
propertyValue = occurrence_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
