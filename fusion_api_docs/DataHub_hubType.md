# DataHub.hubType Property

Parent Object: [DataHub](DataHub.md)  

## Description

Gets if this hub is a Personal (PersonalHubType) or Team (TeamHubType) type hub.

## Syntax

"dataHub_var" is a variable referencing a DataHub object.  

```python
# Get the value of the property.
propertyValue = dataHub_var.hubType
```

## Property Value

This is a read only property whose value is a [HubTypes](HubTypes.md).

## Version

Introduced in version September 2016
