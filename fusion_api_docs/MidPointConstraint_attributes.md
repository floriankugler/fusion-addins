# MidPointConstraint.attributes Property

Parent Object: [MidPointConstraint](MidPointConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"midPointConstraint_var" is a variable referencing a MidPointConstraint object.  

```python
# Get the value of the property.
propertyValue = midPointConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
