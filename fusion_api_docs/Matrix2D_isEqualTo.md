# Matrix2D.isEqualTo Method

Parent Object: [Matrix2D](Matrix2D.md)  

## Description

Compares this matrix with another matrix and returns True if they're identical.

## Syntax

"matrix2D_var" is a variable referencing a [Matrix2D](Matrix2D.md) object.

```python
returnValue = matrix2D_var.isEqualTo(matrix)
```

## Return Value

| Type    | Description                                         |
|---------|-----------------------------------------------------|
| boolean | Returns true if the matrix is equal to this matrix. |

## Parameters

| Name   | Type                     | Description               |
|--------|--------------------------|---------------------------|
| matrix | [Matrix2D](Matrix2D.md) | The matrix to compare to. |

## Version

Introduced in version August 2014
