# SheetMetalRule.twoBendReliefSize Property

Parent Object: [SheetMetalRule](SheetMetalRule.md)  

## Description

The relief size used when two bends meet in the flat pattern and the relief shape is round or square. Use the returned SheetMetalRuleValue object to get and set the current value of the relief size.

## Syntax

"sheetMetalRule_var" is a variable referencing a SheetMetalRule object.  

```python
# Get the value of the property.
propertyValue = sheetMetalRule_var.twoBendReliefSize
```

## Property Value

This is a read only property whose value is a [SheetMetalRuleValue](SheetMetalRuleValue.md).

## Version

Introduced in version November 2022
