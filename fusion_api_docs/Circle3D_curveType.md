# Circle3D.curveType Property

Parent Object: [Circle3D](Circle3D.md)  

## Description

Returns the type of geometry this curve represents.

## Syntax

"circle3D_var" is a variable referencing a Circle3D object.  

```python
# Get the value of the property.
propertyValue = circle3D_var.curveType
```

## Property Value

This is a read only property whose value is a [Curve3DTypes](Curve3DTypes.md).

## Version

Introduced in version August 2014
