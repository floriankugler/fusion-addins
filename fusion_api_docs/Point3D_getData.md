# Point3D.getData Method

Parent Object: [Point3D](Point3D.md)  

## Description

Gets the data defining the point.

## Syntax

"point3D_var" is a variable referencing a [Point3D](Point3D.md) object.  

```python
(returnValue, x, y, z) = point3D_var.getData()
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name | Type   | Description                           |
|------|--------|---------------------------------------|
| x    | double | The output x coordinate of the point. |
| y    | double | The output y coordinate of the point. |
| z    | double | The output z coordinate of the point. |

## Version

Introduced in version August 2014
