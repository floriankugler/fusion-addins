# OffsetFeatureInput.operation Property

Parent Object: [OffsetFeatureInput](OffsetFeatureInput.md)  

## Description

Gets and sets the feature operation to perform. Can be 'NewBodyFeatureOperation' or 'NewComponentFeatureOperation'.

## Syntax

"offsetFeatureInput_var" is a variable referencing an OffsetFeatureInput object.  

```python
# Get the value of the property.
propertyValue = offsetFeatureInput_var.operation

# Set the value of the property.
offsetFeatureInput_var.operation = propertyValue
```

## Property Value

This is a read/write property whose value is a [FeatureOperations](FeatureOperations.md).

## Version

Introduced in version June 2015
