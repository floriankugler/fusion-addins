# Document.isModified Property

Parent Object: [Document](Document.md)  

## Description

Property that indicates if the document has been modified since it was last saved.

## Syntax

"document_var" is a variable referencing a Document object.  

```python
# Get the value of the property.
propertyValue = document_var.isModified
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
