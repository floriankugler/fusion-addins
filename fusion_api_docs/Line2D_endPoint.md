# Line2D.endPoint Property

Parent Object: [Line2D](Line2D.md)  

## Description

Gets and sets the end point of the line.

## Syntax

"line2D_var" is a variable referencing a Line2D object.  

```python
# Get the value of the property.
propertyValue = line2D_var.endPoint

# Set the value of the property.
line2D_var.endPoint = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point2D](Point2D.md).

## Version

Introduced in version August 2014
