# ApplicationFolders.appLogFilePath Property

Parent Object: [ApplicationFolders](ApplicationFolders.md)  

## Description

Returns the full filename for the current application log file.

## Syntax

"applicationFolders_var" is a variable referencing an ApplicationFolders object.  

```python
# Get the value of the property.
propertyValue = applicationFolders_var.appLogFilePath
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version September 2024
