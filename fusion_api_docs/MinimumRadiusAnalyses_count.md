# MinimumRadiusAnalyses.count Property

Parent Object: [MinimumRadiusAnalyses](MinimumRadiusAnalyses.md)  

## Description

Returns the number of MinimumRadiusAnalysis objects in the collection.

## Syntax

"minimumRadiusAnalyses_var" is a variable referencing a MinimumRadiusAnalyses object.  

```python
# Get the value of the property.
propertyValue = minimumRadiusAnalyses_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version January 2023
