# IntegerSpinnerCommandInput.deleteMe Method

Parent Object: [IntegerSpinnerCommandInput](IntegerSpinnerCommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"integerSpinnerCommandInput_var" is a variable referencing an [IntegerSpinnerCommandInput](IntegerSpinnerCommandInput.md) object.

```python
returnValue = integerSpinnerCommandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2020
