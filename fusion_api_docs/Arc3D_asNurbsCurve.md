# Arc3D.asNurbsCurve Property

Parent Object: [Arc3D](Arc3D.md)  

## Description

Returns a NURBS curve that is geometrically identical to the arc.

## Syntax

"arc3D_var" is a variable referencing an Arc3D object.  

```python
# Get the value of the property.
propertyValue = arc3D_var.asNurbsCurve
```

## Property Value

This is a read only property whose value is a [NurbsCurve3D](NurbsCurve3D.md).
