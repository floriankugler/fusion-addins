# CAMExportFuture.exportOptions Property

Parent Object: [CAMExportFuture](CAMExportFuture.md)  

## Description

Returns the export option used to define the export associated with this future object.

## Syntax

"cAMExportFuture_var" is a variable referencing a CAMExportFuture object.  

```python
# Get the value of the property.
propertyValue = cAMExportFuture_var.exportOptions
```

## Property Value

This is a read only property whose value is a [CAMExportOptions](CAMExportOptions.md).

## Version

Introduced in version September 2024
