# OffsetFacesFeature.attributes Property

Parent Object: [OffsetFacesFeature](OffsetFacesFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"offsetFacesFeature_var" is a variable referencing an OffsetFacesFeature object.  

```python
# Get the value of the property.
propertyValue = offsetFacesFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version June 2017
