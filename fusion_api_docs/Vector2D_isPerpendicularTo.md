# Vector2D.isPerpendicularTo Method

Parent Object: [Vector2D](Vector2D.md)  

## Description

Compare this vector with another to check for perpendicularity.

## Syntax

"vector2D_var" is a variable referencing a [Vector2D](Vector2D.md) object.

```python
returnValue = vector2D_var.isPerpendicularTo(vector)
```

## Return Value

| Type    | Description                                    |
|---------|------------------------------------------------|
| boolean | Returns true if the vectors are perpendicular. |

## Parameters

| Name | Type | Description |
|----|----|----|
| vector | [Vector2D](Vector2D.md) | The vector to compare with for perpendicularity. |

## Version

Introduced in version August 2014
