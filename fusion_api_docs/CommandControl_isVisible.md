# CommandControl.isVisible Property

Parent Object: [CommandControl](CommandControl.md)  

## Description

Gets or sets if this control is currently visible.

## Syntax

"commandControl_var" is a variable referencing a CommandControl object.  

```python
# Get the value of the property.
propertyValue = commandControl_var.isVisible

# Set the value of the property.
commandControl_var.isVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
