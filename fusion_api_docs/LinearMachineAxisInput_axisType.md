# LinearMachineAxisInput.axisType Property

Parent Object: [LinearMachineAxisInput](LinearMachineAxisInput.md)  

## Description

The type of axis. This axis type determines which parameters of this object are valid to be accessed or modified.

## Syntax

"linearMachineAxisInput_var" is a variable referencing a LinearMachineAxisInput object.  

```python
# Get the value of the property.
propertyValue = linearMachineAxisInput_var.axisType
```

## Property Value

This is a read only property whose value is a [MachineAxisTypes](MachineAxisTypes.md).

## Version

Introduced in version April 2023
