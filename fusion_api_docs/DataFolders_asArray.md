# DataFolders.asArray Method

Parent Object: [DataFolders](DataFolders.md)  

## Description

Get the current list of all folders.

## Syntax

"dataFolders_var" is a variable referencing a [DataFolders](DataFolders.md) object.

```python
returnValue = dataFolders_var.asArray()
```

## Return Value

| Type                             | Description                              |
|----------------------------------|------------------------------------------|
| [DataFolder](DataFolder.md)\[\] | Returns the current list of all folders. |

## Version

Introduced in version April 2021
