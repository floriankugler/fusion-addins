# MidPointConstraint.isDeletable Property

Parent Object: [MidPointConstraint](MidPointConstraint.md)  

## Description

Indicates if this constraint is deletable.

## Syntax

"midPointConstraint_var" is a variable referencing a MidPointConstraint object.  

```python
# Get the value of the property.
propertyValue = midPointConstraint_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
