# SketchEllipticalArc.boundingBox Property

Parent Object: [SketchEllipticalArc](SketchEllipticalArc.md)  

## Description

Returns the bounding box of the entity in sketch space.

## Syntax

"sketchEllipticalArc_var" is a variable referencing a SketchEllipticalArc object.  

```python
# Get the value of the property.
propertyValue = sketchEllipticalArc_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version August 2014
