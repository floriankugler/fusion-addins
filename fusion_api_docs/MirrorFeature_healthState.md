# MirrorFeature.healthState Property

Parent Object: [MirrorFeature](MirrorFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"mirrorFeature_var" is a variable referencing a MirrorFeature object.  

```python
# Get the value of the property.
propertyValue = mirrorFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
