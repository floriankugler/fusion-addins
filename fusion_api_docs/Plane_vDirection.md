# Plane.vDirection Property

Parent Object: [Plane](Plane.md)  

## Description

Gets the V Direction of the plane.

## Syntax

"plane_var" is a variable referencing a Plane object.  

```python
# Get the value of the property.
propertyValue = plane_var.vDirection
```

## Property Value

This is a read only property whose value is a [Vector3D](Vector3D.md).

## Version

Introduced in version August 2014
