# CustomGraphicsMesh.deleteMe Method

Parent Object: [CustomGraphicsMesh](CustomGraphicsMesh.md)  

## Description

Deletes the entity from the custom graphics group.

## Syntax

"customGraphicsMesh_var" is a variable referencing a [CustomGraphicsMesh](CustomGraphicsMesh.md) object.

```python
returnValue = customGraphicsMesh_var.deleteMe()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns true if the deletion was successful. |

## Version

Introduced in version September 2017
