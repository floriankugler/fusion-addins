# SketchArc.attributes Property

Parent Object: [SketchArc](SketchArc.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"sketchArc_var" is a variable referencing a SketchArc object.  

```python
# Get the value of the property.
propertyValue = sketchArc_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
