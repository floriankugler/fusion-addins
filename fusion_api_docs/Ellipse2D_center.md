# Ellipse2D.center Property

Parent Object: [Ellipse2D](Ellipse2D.md)  

## Description

Gets and sets the center position of the ellipse.

## Syntax

"ellipse2D_var" is a variable referencing an Ellipse2D object.  

```python
# Get the value of the property.
propertyValue = ellipse2D_var.center

# Set the value of the property.
ellipse2D_var.center = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point2D](Point2D.md).

## Version

Introduced in version August 2014
