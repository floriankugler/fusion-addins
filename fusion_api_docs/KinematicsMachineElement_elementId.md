# KinematicsMachineElement.elementId Property

Parent Object: [KinematicsMachineElement](KinematicsMachineElement.md)  

## Description

Identifier for this element. Unique within an element type.

## Syntax

"kinematicsMachineElement_var" is a variable referencing a KinematicsMachineElement object.  

```python
# Get the value of the property.
propertyValue = kinematicsMachineElement_var.elementId
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
