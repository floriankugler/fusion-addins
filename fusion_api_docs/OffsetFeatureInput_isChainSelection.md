# OffsetFeatureInput.isChainSelection Property

Parent Object: [OffsetFeatureInput](OffsetFeatureInput.md)  

## Description

Get and sets whether faces that are tangentially connected to the input faces will be included in the offset.

## Syntax

"offsetFeatureInput_var" is a variable referencing an OffsetFeatureInput object.  

```python
# Get the value of the property.
propertyValue = offsetFeatureInput_var.isChainSelection

# Set the value of the property.
offsetFeatureInput_var.isChainSelection = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version June 2015
