# MachineSpindle.peakTorqueSpeed Property

Parent Object: [MachineSpindle](MachineSpindle.md)  

## Description

Specifies the speed (rpm) at which this spindle reaches peak torque (Nm).

## Syntax

"machineSpindle_var" is a variable referencing a MachineSpindle object.  

```python
# Get the value of the property.
propertyValue = machineSpindle_var.peakTorqueSpeed

# Set the value of the property.
machineSpindle_var.peakTorqueSpeed = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version July 2025
