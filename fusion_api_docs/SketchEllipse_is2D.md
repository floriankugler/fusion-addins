# SketchEllipse.is2D Property

Parent Object: [SketchEllipse](SketchEllipse.md)  

## Description

Indicates if this curve lies entirely on the sketch x-y plane.

## Syntax

"sketchEllipse_var" is a variable referencing a SketchEllipse object.  

```python
# Get the value of the property.
propertyValue = sketchEllipse_var.is2D
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
