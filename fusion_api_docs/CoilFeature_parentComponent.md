# CoilFeature.parentComponent Property

Parent Object: [CoilFeature](CoilFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"coilFeature_var" is a variable referencing a CoilFeature object.  

```python
# Get the value of the property.
propertyValue = coilFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version March 2015
