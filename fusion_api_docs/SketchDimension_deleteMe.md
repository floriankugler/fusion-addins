# SketchDimension.deleteMe Method

Parent Object: [SketchDimension](SketchDimension.md)  

## Description

Deletes this dimension. The IsDeletable property indicates if this dimension can be deleted.

## Syntax

"sketchDimension_var" is a variable referencing a [SketchDimension](SketchDimension.md) object.

```python
returnValue = sketchDimension_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
