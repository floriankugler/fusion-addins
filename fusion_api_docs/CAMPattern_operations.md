# CAMPattern.operations Property

Parent Object: [CAMPattern](CAMPattern.md)  

## Description

Returns the Operations collection that provides access to existing individual operations in this folder.

## Syntax

"cAMPattern_var" is a variable referencing a CAMPattern object.  

```python
# Get the value of the property.
propertyValue = cAMPattern_var.operations
```

## Property Value

This is a read only property whose value is an [Operations](Operations.md).

## Version

Introduced in version January 2016
