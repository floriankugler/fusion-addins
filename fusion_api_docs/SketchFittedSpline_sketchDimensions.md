# SketchFittedSpline.sketchDimensions Property

Parent Object: [SketchFittedSpline](SketchFittedSpline.md)  

## Description

Returns the sketch dimensions that are attached to this curve.

## Syntax

"sketchFittedSpline_var" is a variable referencing a SketchFittedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFittedSpline_var.sketchDimensions
```

## Property Value

This is a read only property whose value is a [SketchDimensionList](SketchDimensionList.md).

## Version

Introduced in version August 2014
