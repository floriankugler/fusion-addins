# BoundaryFillFeature.timelineObject Property

Parent Object: [BoundaryFillFeature](BoundaryFillFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"boundaryFillFeature_var" is a variable referencing a BoundaryFillFeature object.  

```python
# Get the value of the property.
propertyValue = boundaryFillFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version June 2015
