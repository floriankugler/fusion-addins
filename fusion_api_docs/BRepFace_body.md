# BRepFace.body Property

Parent Object: [BRepFace](BRepFace.md)  

## Description

Returns the parent body of the face.

## Syntax

"bRepFace_var" is a variable referencing a BRepFace object.  

```python
# Get the value of the property.
propertyValue = bRepFace_var.body
```

## Property Value

This is a read only property whose value is a [BRepBody](BRepBody.md).

## Version

Introduced in version August 2014
