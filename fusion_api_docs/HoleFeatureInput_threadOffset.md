# HoleFeatureInput.threadOffset Property

Parent Object: [HoleFeatureInput](HoleFeatureInput.md)  

## Description

Gets the thread offset when the isFullLength property is False. Returns null when the isFullLength property is true.

## Syntax

"holeFeatureInput_var" is a variable referencing a HoleFeatureInput object.  

```python
# Get the value of the property.
propertyValue = holeFeatureInput_var.threadOffset
```

## Property Value

This is a read only property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version September 2025
