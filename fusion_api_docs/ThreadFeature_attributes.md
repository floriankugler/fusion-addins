# ThreadFeature.attributes Property

Parent Object: [ThreadFeature](ThreadFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"threadFeature_var" is a variable referencing a ThreadFeature object.  

```python
# Get the value of the property.
propertyValue = threadFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
