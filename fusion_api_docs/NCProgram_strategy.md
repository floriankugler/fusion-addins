# NCProgram.strategy Property

Parent Object: [NCProgram](NCProgram.md)  

## Description

Gets the name of the strategy associated with this operation.

## Syntax

"nCProgram_var" is a variable referencing a NCProgram object.  

```python
# Get the value of the property.
propertyValue = nCProgram_var.strategy
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
