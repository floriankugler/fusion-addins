# ConfigurationThemeColumn.rowCount Property

Parent Object: [ConfigurationThemeColumn](ConfigurationThemeColumn.md)  

## Description

Returns the number of rows in this column.

## Syntax

"configurationThemeColumn_var" is a variable referencing a ConfigurationThemeColumn object.  

```python
# Get the value of the property.
propertyValue = configurationThemeColumn_var.rowCount
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version March 2024
