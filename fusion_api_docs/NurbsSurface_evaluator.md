# NurbsSurface.evaluator Property

Parent Object: [NurbsSurface](NurbsSurface.md)  

## Description

Returns the surface evaluator.

## Syntax

"nurbsSurface_var" is a variable referencing a NurbsSurface object.  

```python
# Get the value of the property.
propertyValue = nurbsSurface_var.evaluator
```

## Property Value

This is a read only property whose value is a [SurfaceEvaluator](SurfaceEvaluator.md).

## Version

Introduced in version August 2014
