# RuleFilletFeature.name Property

Parent Object: [RuleFilletFeature](RuleFilletFeature.md)  

## Description

Returns the name of the feature as seen in the browser (non-parametric) or in the timeline (parametric).

## Syntax

"ruleFilletFeature_var" is a variable referencing a RuleFilletFeature object.  

```python
# Get the value of the property.
propertyValue = ruleFilletFeature_var.name

# Set the value of the property.
ruleFilletFeature_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version September 2015
