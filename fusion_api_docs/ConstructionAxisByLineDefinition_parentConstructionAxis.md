# ConstructionAxisByLineDefinition.parentConstructionAxis Property

Parent Object: [ConstructionAxisByLineDefinition](ConstructionAxisByLineDefinition.md)  

## Description

Returns the ConstructionAxis object

## Syntax

"constructionAxisByLineDefinition_var" is a variable referencing a ConstructionAxisByLineDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionAxisByLineDefinition_var.parentConstructionAxis
```

## Property Value

This is a read only property whose value is a [ConstructionAxis](ConstructionAxis.md).

## Version

Introduced in version August 2014
