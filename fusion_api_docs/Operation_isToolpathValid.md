# Operation.isToolpathValid Property

Parent Object: [Operation](Operation.md)  

## Description

Gets if the toolpath for this operation is currently valid. (has not become invalidated by model changes).

## Syntax

"operation_var" is a variable referencing an Operation object.  

```python
# Get the value of the property.
propertyValue = operation_var.isToolpathValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2016
