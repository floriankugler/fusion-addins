# Arrange3DDefinition.isValid Property

Parent Object: [Arrange3DDefinition](Arrange3DDefinition.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"arrange3DDefinition_var" is a variable referencing an Arrange3DDefinition object.  

```python
# Get the value of the property.
propertyValue = arrange3DDefinition_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2025
