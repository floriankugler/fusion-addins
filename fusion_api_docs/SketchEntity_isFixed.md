# SketchEntity.isFixed Property

Parent Object: [SketchEntity](SketchEntity.md)  

## Description

Indicates if this geometry is "fixed".

## Syntax

"sketchEntity_var" is a variable referencing a SketchEntity object.  

```python
# Get the value of the property.
propertyValue = sketchEntity_var.isFixed

# Set the value of the property.
sketchEntity_var.isFixed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
