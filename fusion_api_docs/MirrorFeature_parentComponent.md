# MirrorFeature.parentComponent Property

Parent Object: [MirrorFeature](MirrorFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"mirrorFeature_var" is a variable referencing a MirrorFeature object.  

```python
# Get the value of the property.
propertyValue = mirrorFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version November 2014
