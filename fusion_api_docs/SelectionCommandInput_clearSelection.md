# SelectionCommandInput.clearSelection Method

Parent Object: [SelectionCommandInput](SelectionCommandInput.md)  

## Description

Clears the current selection so no entities are in the selection.

## Syntax

"selectionCommandInput_var" is a variable referencing a [SelectionCommandInput](SelectionCommandInput.md) object.

```python
returnValue = selectionCommandInput_var.clearSelection()
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Version

Introduced in version June 2015
