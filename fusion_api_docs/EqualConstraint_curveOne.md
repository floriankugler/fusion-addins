# EqualConstraint.curveOne Property

Parent Object: [EqualConstraint](EqualConstraint.md)  

## Description

Returns the first curve.

## Syntax

"equalConstraint_var" is a variable referencing an EqualConstraint object.  

```python
# Get the value of the property.
propertyValue = equalConstraint_var.curveOne
```

## Property Value

This is a read only property whose value is a [SketchCurve](SketchCurve.md).

## Version

Introduced in version August 2014
