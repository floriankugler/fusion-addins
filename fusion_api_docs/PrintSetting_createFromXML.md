# PrintSetting.createFromXML Method

Parent Object: [PrintSetting](PrintSetting.md)  

## Description

Creates a PrintSetting from a xml content.

## Syntax

This is a static method.  

```python

returnValue = adsk.cam.PrintSetting.createFromXML(xmlContent)
```

## Return Value

| Type                             | Description                             |
|----------------------------------|-----------------------------------------|
| [PrintSetting](PrintSetting.md) | Returns the newly created PrintSetting. |

## Parameters

| Name | Type | Description |
|----|----|----|
| xmlContent | string | The xml content string of a PrintSetting file to act as a base for creating a PrintSetting from. |

## Version

Introduced in version April 2023
