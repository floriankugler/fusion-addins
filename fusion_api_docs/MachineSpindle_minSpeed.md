# MachineSpindle.minSpeed Property

Parent Object: [MachineSpindle](MachineSpindle.md)  

## Description

Specifies the minimum speed (rpm) for this spindle.

## Syntax

"machineSpindle_var" is a variable referencing a MachineSpindle object.  

```python
# Get the value of the property.
propertyValue = machineSpindle_var.minSpeed

# Set the value of the property.
machineSpindle_var.minSpeed = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version April 2023
