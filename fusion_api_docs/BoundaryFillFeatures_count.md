# BoundaryFillFeatures.count Property

Parent Object: [BoundaryFillFeatures](BoundaryFillFeatures.md)  

## Description

The number of boundary fill features in the collection.

## Syntax

"boundaryFillFeatures_var" is a variable referencing a BoundaryFillFeatures object.  

```python
# Get the value of the property.
propertyValue = boundaryFillFeatures_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version June 2015
