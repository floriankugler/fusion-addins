# Setup.notes Property

Parent Object: [Setup](Setup.md)  

## Description

Gets and sets the notes of the operation.

## Syntax

"setup_var" is a variable referencing a Setup object.  

```python
# Get the value of the property.
propertyValue = setup_var.notes

# Set the value of the property.
setup_var.notes = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version May 2020
