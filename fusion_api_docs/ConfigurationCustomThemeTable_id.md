# ConfigurationCustomThemeTable.id Property

Parent Object: [ConfigurationCustomThemeTable](ConfigurationCustomThemeTable.md)  

## Description

Returns the unique ID of this table.

## Syntax

"configurationCustomThemeTable_var" is a variable referencing a ConfigurationCustomThemeTable object.  

```python
# Get the value of the property.
propertyValue = configurationCustomThemeTable_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version January 2024
