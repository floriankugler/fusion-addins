# Matrix3D.asArray Method

Parent Object: [Matrix3D](Matrix3D.md)  

## Description

Returns the contents of the matrix as a 16 element array.

## Syntax

"matrix3D_var" is a variable referencing a [Matrix3D](Matrix3D.md) object.

```python
returnValue = matrix3D_var.asArray()
```

## Return Value

| Type       | Description                       |
|------------|-----------------------------------|
| double\[\] | Returns the array of cell values. |

## Version

Introduced in version August 2014
