# FlatPatternProduct.materials Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Returns the materials contained in this document.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.materials
```

## Property Value

This is a read only property whose value is a [Materials](Materials.md).

## Version

Introduced in version October 2022
