# OperationBase.error Property

Parent Object: [OperationBase](OperationBase.md)  

## Description

Returns a message corresponding to any active error associated with the value of this parameter.

## Syntax

"operationBase_var" is a variable referencing an OperationBase object.  

```python
# Get the value of the property.
propertyValue = operationBase_var.error
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
