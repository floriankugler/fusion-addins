# Line2D.set Method

Parent Object: [Line2D](Line2D.md)  

## Description

Sets all of the data defining the line segment.

## Syntax

"line2D_var" is a variable referencing a [Line2D](Line2D.md) object.

```python
returnValue = line2D_var.set(startPoint, endPoint)
```

## Return Value

| Type    | Description                                       |
|---------|---------------------------------------------------|
| boolean | Returns true if redefining the line is successful |

## Parameters

| Name       | Type                   | Description                 |
|------------|------------------------|-----------------------------|
| startPoint | [Point2D](Point2D.md) | The start point of the line |
| endPoint   | [Point2D](Point2D.md) | The end point of the line   |

## Version

Introduced in version August 2014
