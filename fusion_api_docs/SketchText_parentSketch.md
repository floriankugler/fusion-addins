# SketchText.parentSketch Property

Parent Object: [SketchText](SketchText.md)  

## Description

Returns the parent sketch.

## Syntax

"sketchText_var" is a variable referencing a SketchText object.  

```python
# Get the value of the property.
propertyValue = sketchText_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version March 2015
