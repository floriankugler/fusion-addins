# LineParallelToPlanarSurfaceConstraint.parentSketch Property

Parent Object: [LineParallelToPlanarSurfaceConstraint](LineParallelToPlanarSurfaceConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"lineParallelToPlanarSurfaceConstraint_var" is a variable referencing a LineParallelToPlanarSurfaceConstraint object.  

```python
# Get the value of the property.
propertyValue = lineParallelToPlanarSurfaceConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version September 2023
