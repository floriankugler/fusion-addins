# Point3D.set Method

Parent Object: [Point3D](Point3D.md)  

## Description

Sets the data defining the point.

## Syntax

"point3D_var" is a variable referencing a [Point3D](Point3D.md) object.

```python
returnValue = point3D_var.set(x, y, z)
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name | Type   | Description                    |
|------|--------|--------------------------------|
| x    | double | The x coordinate of the point. |
| y    | double | The y coordinate of the point. |
| z    | double | The z coordinate of the point. |

## Version

Introduced in version August 2014
