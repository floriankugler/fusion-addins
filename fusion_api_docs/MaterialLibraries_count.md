# MaterialLibraries.count Property

Parent Object: [MaterialLibraries](MaterialLibraries.md)  

## Description

The number of Material Libraries in the collection.

## Syntax

"materialLibraries_var" is a variable referencing a MaterialLibraries object.  

```python
# Get the value of the property.
propertyValue = materialLibraries_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version August 2014
