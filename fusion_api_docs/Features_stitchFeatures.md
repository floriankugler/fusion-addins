# Features.stitchFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the Stitch features within the component and supports the creation of new Stitch features.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.stitchFeatures
```

## Property Value

This is a read only property whose value is a [StitchFeatures](StitchFeatures.md).
