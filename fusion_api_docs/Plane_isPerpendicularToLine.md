# Plane.isPerpendicularToLine Method

Parent Object: [Plane](Plane.md)  

## Description

Checks if this plane is perpendicular to a line.

## Syntax

"plane_var" is a variable referencing a [Plane](Plane.md) object.

```python
returnValue = plane_var.isPerpendicularToLine(line)
```

## Return Value

| Type    | Description                                           |
|---------|-------------------------------------------------------|
| boolean | Returns true if the plane and line are perpendicular. |

## Parameters

| Name | Type                 | Description                                    |
|------|----------------------|------------------------------------------------|
| line | [Line3D](Line3D.md) | The line to compare with for perpendicularity. |

## Version

Introduced in version August 2014
