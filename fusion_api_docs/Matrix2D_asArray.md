# Matrix2D.asArray Method

Parent Object: [Matrix2D](Matrix2D.md)  

## Description

Returns the contents of the matrix as a 9 element array.

## Syntax

"matrix2D_var" is a variable referencing a [Matrix2D](Matrix2D.md) object.

```python
returnValue = matrix2D_var.asArray()
```

## Return Value

| Type       | Description                         |
|------------|-------------------------------------|
| double\[\] | Returns the array of matrix values. |

## Version

Introduced in version August 2014
