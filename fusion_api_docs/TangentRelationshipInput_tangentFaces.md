# TangentRelationshipInput.tangentFaces Property

Parent Object: [TangentRelationshipInput](TangentRelationshipInput.md)  

## Description

Gets and sets a single BRepFace object that is part of the body that faceOne will remain tangent to. All of the faces of the body will be used when computing the tangent relationship.

## Syntax

"tangentRelationshipInput_var" is a variable referencing a TangentRelationshipInput object.  

```python
# Get the value of the property.
propertyValue = tangentRelationshipInput_var.tangentFaces

# Set the value of the property.
tangentRelationshipInput_var.tangentFaces = propertyValue
```

## Property Value

This is a read/write property whose value is a [Base](Base.md).

## Version

Introduced in version May 2022
