# KeyboardEventArgs.modifierMask Property

Parent Object: [KeyboardEventArgs](KeyboardEventArgs.md)  

## Description

Gets the set of keyboard modifiers that were active. The value is the Boolean combination of KeyboardModifiers values.

## Syntax

"keyboardEventArgs_var" is a variable referencing a KeyboardEventArgs object.  

```python
# Get the value of the property.
propertyValue = keyboardEventArgs_var.modifierMask
```

## Property Value

This is a read only property whose value is an integer.

## Version

Introduced in version August 2014
