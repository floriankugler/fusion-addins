# CAM.allMachines Property

Parent Object: [CAM](CAM.md)  

## Description

Gets an array containing all the machines in the document.

## Syntax

"cAM_var" is a variable referencing a CAM object.  

```python
# Get the value of the property.
propertyValue = cAM_var.allMachines
```

## Property Value

This is a read only property whose value is an array of type [Machine](Machine.md).

## Version

Introduced in version April 2023
