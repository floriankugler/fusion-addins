# DataFile.description Property

Parent Object: [DataFile](DataFile.md)  

## Description

Gets and sets the description information associated with this item.

## Syntax

"dataFile_var" is a variable referencing a DataFile object.  

```python
# Get the value of the property.
propertyValue = dataFile_var.description
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version January 2015
