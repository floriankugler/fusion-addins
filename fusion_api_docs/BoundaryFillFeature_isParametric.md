# BoundaryFillFeature.isParametric Property

Parent Object: [BoundaryFillFeature](BoundaryFillFeature.md)  

## Description

Indicates if this feature is parametric or not.

## Syntax

"boundaryFillFeature_var" is a variable referencing a BoundaryFillFeature object.  

```python
# Get the value of the property.
propertyValue = boundaryFillFeature_var.isParametric
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version June 2015
