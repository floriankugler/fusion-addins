# GeometricConstraint.parentSketch Property

Parent Object: [GeometricConstraint](GeometricConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"geometricConstraint_var" is a variable referencing a GeometricConstraint object.  

```python
# Get the value of the property.
propertyValue = geometricConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
