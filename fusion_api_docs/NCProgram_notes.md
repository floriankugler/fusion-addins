# NCProgram.notes Property

Parent Object: [NCProgram](NCProgram.md)  

## Description

Gets and sets the notes of the operation.

## Syntax

"nCProgram_var" is a variable referencing a NCProgram object.  

```python
# Get the value of the property.
propertyValue = nCProgram_var.notes

# Set the value of the property.
nCProgram_var.notes = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version April 2023
