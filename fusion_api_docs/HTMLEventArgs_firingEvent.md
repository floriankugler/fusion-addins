# HTMLEventArgs.firingEvent Property

Parent Object: [HTMLEventArgs](HTMLEventArgs.md)  

## Description

The event that the firing is in response to.

## Syntax

"hTMLEventArgs_var" is a variable referencing a HTMLEventArgs object.  

```python
# Get the value of the property.
propertyValue = hTMLEventArgs_var.firingEvent
```

## Property Value

This is a read only property whose value is an [Event](Event.md).

## Version

Introduced in version March 2017
