# RigidGroup.name Property

Parent Object: [RigidGroup](RigidGroup.md)  

## Description

Gets and sets the name of the rigid group as seen in the timeline.

## Syntax

"rigidGroup_var" is a variable referencing a RigidGroup object.  

```python
# Get the value of the property.
propertyValue = rigidGroup_var.name

# Set the value of the property.
rigidGroup_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version September 2015
