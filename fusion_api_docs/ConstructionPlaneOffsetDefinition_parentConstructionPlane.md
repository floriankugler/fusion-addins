# ConstructionPlaneOffsetDefinition.parentConstructionPlane Property

Parent Object: [ConstructionPlaneOffsetDefinition](ConstructionPlaneOffsetDefinition.md)  

## Description

Returns the ConstructionPlane object

## Syntax

"constructionPlaneOffsetDefinition_var" is a variable referencing a ConstructionPlaneOffsetDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionPlaneOffsetDefinition_var.parentConstructionPlane
```

## Property Value

This is a read only property whose value is a [ConstructionPlane](ConstructionPlane.md).

## Version

Introduced in version August 2014
