# Design.workspaces Property

Parent Object: [Design](Design.md)  

## Description

Returns the workspaces associated with this product.

## Syntax

"design_var" is a variable referencing a Design object.  

```python
# Get the value of the property.
propertyValue = design_var.workspaces
```

## Property Value

This is a read only property whose value is a [WorkspaceList](WorkspaceList.md).

## Version

Introduced in version August 2014
