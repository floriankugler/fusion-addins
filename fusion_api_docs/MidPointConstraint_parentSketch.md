# MidPointConstraint.parentSketch Property

Parent Object: [MidPointConstraint](MidPointConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"midPointConstraint_var" is a variable referencing a MidPointConstraint object.  

```python
# Get the value of the property.
propertyValue = midPointConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
