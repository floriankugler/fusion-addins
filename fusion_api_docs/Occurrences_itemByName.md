# Occurrences.itemByName Method

Parent Object: [Occurrences](Occurrences.md)  

## Description

Returns the specified occurrence using the name of the occurrence.

## Syntax

"occurrences_var" is a variable referencing an [Occurrences](Occurrences.md) object.

```python
returnValue = occurrences_var.itemByName(name)
```

## Return Value

| Type | Description |
|----|----|
| [Occurrence](Occurrence.md) | Returns the occurrence or null if an invalid name was specified |

## Parameters

| Name | Type   | Description                           |
|------|--------|---------------------------------------|
| name | string | The name of the occurrence to return. |

## Version

Introduced in version August 2014
