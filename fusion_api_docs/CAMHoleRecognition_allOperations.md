# CAMHoleRecognition.allOperations Property

Parent Object: [CAMHoleRecognition](CAMHoleRecognition.md)  

## Description

Returns an array containing all of the operations in this hole recognition. This includes all operations nested in folders and patterns.

## Syntax

"cAMHoleRecognition_var" is a variable referencing a CAMHoleRecognition object.  

```python
# Get the value of the property.
propertyValue = cAMHoleRecognition_var.allOperations
```

## Property Value

This is a read only property whose value is an array of type [OperationBase](OperationBase.md).

## Version

Introduced in version April 2023
