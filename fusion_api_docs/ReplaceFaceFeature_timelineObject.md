# ReplaceFaceFeature.timelineObject Property

Parent Object: [ReplaceFaceFeature](ReplaceFaceFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"replaceFaceFeature_var" is a variable referencing a ReplaceFaceFeature object.  

```python
# Get the value of the property.
propertyValue = replaceFaceFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version March 2015
