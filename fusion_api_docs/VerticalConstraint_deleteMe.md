# VerticalConstraint.deleteMe Method

Parent Object: [VerticalConstraint](VerticalConstraint.md)  

## Description

Deletes this constraint. The IsDeletable property can be used to determine if this constraint can be deleted.

## Syntax

"verticalConstraint_var" is a variable referencing a [VerticalConstraint](VerticalConstraint.md) object.

```python
returnValue = verticalConstraint_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
