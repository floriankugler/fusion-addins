# ThreadFeature.healthState Property

Parent Object: [ThreadFeature](ThreadFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"threadFeature_var" is a variable referencing a ThreadFeature object.  

```python
# Get the value of the property.
propertyValue = threadFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
