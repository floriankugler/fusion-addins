# ToolbarTab.isActive Property

Parent Object: [ToolbarTab](ToolbarTab.md)  

## Description

Gets if this toolbar tab is currently active - i.e. displayed.

## Syntax

"toolbarTab_var" is a variable referencing a ToolbarTab object.  

```python
# Get the value of the property.
propertyValue = toolbarTab_var.isActive
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version October 2019
