# DistanceExtentDefinition.distance Property

Parent Object: [DistanceExtentDefinition](DistanceExtentDefinition.md)  

## Description

Returns the parameter controlling the distance. You can edit the distance by editing the value of the parameter object.

## Syntax

"distanceExtentDefinition_var" is a variable referencing a DistanceExtentDefinition object.  

```python
# Get the value of the property.
propertyValue = distanceExtentDefinition_var.distance
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version August 2014
