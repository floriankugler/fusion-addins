# SplitFaceFeature.parentComponent Property

Parent Object: [SplitFaceFeature](SplitFaceFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"splitFaceFeature_var" is a variable referencing a SplitFaceFeature object.  

```python
# Get the value of the property.
propertyValue = splitFaceFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version June 2015
