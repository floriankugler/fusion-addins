# TangentRelationshipInput.isValid Property

Parent Object: [TangentRelationshipInput](TangentRelationshipInput.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"tangentRelationshipInput_var" is a variable referencing a TangentRelationshipInput object.  

```python
# Get the value of the property.
propertyValue = tangentRelationshipInput_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version May 2022
