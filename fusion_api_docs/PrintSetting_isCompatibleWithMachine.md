# PrintSetting.isCompatibleWithMachine Method

Parent Object: [PrintSetting](PrintSetting.md)  

## Description

Checks whether the print setting is usable with the given machine.

## Syntax

"printSetting_var" is a variable referencing a [PrintSetting](PrintSetting.md) object.

```python
returnValue = printSetting_var.isCompatibleWithMachine(machine)
```

## Parameters

| Name    | Type                   | Description |
|---------|------------------------|-------------|
| machine | [Machine](Machine.md) |             |

## Version

Introduced in version January 2025
