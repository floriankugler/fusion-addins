# SketchCircle.boundingBox Property

Parent Object: [SketchCircle](SketchCircle.md)  

## Description

Returns the bounding box of the entity in sketch space.

## Syntax

"sketchCircle_var" is a variable referencing a SketchCircle object.  

```python
# Get the value of the property.
propertyValue = sketchCircle_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version August 2014
