# ConfigurationJointSnapCell.parentColumn Property

Parent Object: [ConfigurationJointSnapCell](ConfigurationJointSnapCell.md)  

## Description

Returns the column this cell is in.

## Syntax

"configurationJointSnapCell_var" is a variable referencing a ConfigurationJointSnapCell object.  

```python
# Get the value of the property.
propertyValue = configurationJointSnapCell_var.parentColumn
```

## Property Value

This is a read only property whose value is a [ConfigurationJointSnapColumn](ConfigurationJointSnapColumn.md).

## Version

Introduced in version September 2024
