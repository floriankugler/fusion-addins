# ModelParameter.attributes Property

Parent Object: [ModelParameter](ModelParameter.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"modelParameter_var" is a variable referencing a ModelParameter object.  

```python
# Get the value of the property.
propertyValue = modelParameter_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
