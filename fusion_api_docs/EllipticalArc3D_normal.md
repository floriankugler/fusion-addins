# EllipticalArc3D.normal Property

Parent Object: [EllipticalArc3D](EllipticalArc3D.md)  

## Description

Gets the normal of the elliptical arc.

## Syntax

"ellipticalArc3D_var" is a variable referencing an EllipticalArc3D object.  

```python
# Get the value of the property.
propertyValue = ellipticalArc3D_var.normal
```

## Property Value

This is a read only property whose value is a [Vector3D](Vector3D.md).

## Version

Introduced in version August 2014
