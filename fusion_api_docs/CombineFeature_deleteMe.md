# CombineFeature.deleteMe Method

Parent Object: [CombineFeature](CombineFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"combineFeature_var" is a variable referencing a [CombineFeature](CombineFeature.md) object.

```python
returnValue = combineFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version November 2014
