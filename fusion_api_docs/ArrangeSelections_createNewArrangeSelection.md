# ArrangeSelections.createNewArrangeSelection Method

Parent Object: [ArrangeSelections](ArrangeSelections.md)  

## Description

Creates a new occurrence selection meant for arrange operations and adds it to the end of the collection.

## Syntax

"arrangeSelections_var" is a variable referencing an [ArrangeSelections](ArrangeSelections.md) object.

```python
returnValue = arrangeSelections_var.createNewArrangeSelection()
```

## Return Value

| Type | Description |
|----|----|
| [ArrangeSelection](ArrangeSelection.md) | Returns newly created ArrangeSelection. |

## Version

Introduced in version March 2024
