# SymmetryConstraint.deleteMe Method

Parent Object: [SymmetryConstraint](SymmetryConstraint.md)  

## Description

Deletes this constraint. The IsDeletable property can be used to determine if this constraint can be deleted.

## Syntax

"symmetryConstraint_var" is a variable referencing a [SymmetryConstraint](SymmetryConstraint.md) object.

```python
returnValue = symmetryConstraint_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
