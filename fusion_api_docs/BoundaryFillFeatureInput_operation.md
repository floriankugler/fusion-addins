# BoundaryFillFeatureInput.operation Property

Parent Object: [BoundaryFillFeatureInput](BoundaryFillFeatureInput.md)  

## Description

Gets and sets the type of operation performed by the boundary fill feature.

## Syntax

"boundaryFillFeatureInput_var" is a variable referencing a BoundaryFillFeatureInput object.  

```python
# Get the value of the property.
propertyValue = boundaryFillFeatureInput_var.operation

# Set the value of the property.
boundaryFillFeatureInput_var.operation = propertyValue
```

## Property Value

This is a read/write property whose value is a [FeatureOperations](FeatureOperations.md).

## Version

Introduced in version June 2015
