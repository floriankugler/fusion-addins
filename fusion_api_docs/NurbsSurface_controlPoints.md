# NurbsSurface.controlPoints Property

Parent Object: [NurbsSurface](NurbsSurface.md)  

## Description

Gets an array of control points from the surface.

## Syntax

"nurbsSurface_var" is a variable referencing a NurbsSurface object.  

```python
# Get the value of the property.
propertyValue = nurbsSurface_var.controlPoints
```

## Property Value

This is a read only property whose value is an array of type [Point3D](Point3D.md).

## Version

Introduced in version August 2014
