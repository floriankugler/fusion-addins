# DataFileFuture.uploadState Property

Parent Object: [DataFileFuture](DataFileFuture.md)  

## Description

Returns the current state of the upload.

## Syntax

"dataFileFuture_var" is a variable referencing a DataFileFuture object.  

```python
# Get the value of the property.
propertyValue = dataFileFuture_var.uploadState
```

## Property Value

This is a read only property whose value is a [UploadStates](UploadStates.md).

## Version

Introduced in version March 2015
