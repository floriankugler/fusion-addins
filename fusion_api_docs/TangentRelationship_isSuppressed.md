# TangentRelationship.isSuppressed Property

Parent Object: [TangentRelationship](TangentRelationship.md)  

## Description

Gets and sets if this tangent relationship is suppressed.

## Syntax

"tangentRelationship_var" is a variable referencing a TangentRelationship object.  

```python
# Get the value of the property.
propertyValue = tangentRelationship_var.isSuppressed

# Set the value of the property.
tangentRelationship_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version May 2022
