# CustomGraphicsCoordinates.coordinateCount Property

Parent Object: [CustomGraphicsCoordinates](CustomGraphicsCoordinates.md)  

## Description

Returns the number of coordinates defined in the CustomGraphicsCoordinates object.

## Syntax

"customGraphicsCoordinates_var" is a variable referencing a CustomGraphicsCoordinates object.  

```python
# Get the value of the property.
propertyValue = customGraphicsCoordinates_var.coordinateCount
```

## Property Value

This is a read only property whose value is an integer.

## Version

Introduced in version September 2017
