# SketchArc.is2D Property

Parent Object: [SketchArc](SketchArc.md)  

## Description

Indicates if this curve lies entirely on the sketch x-y plane.

## Syntax

"sketchArc_var" is a variable referencing a SketchArc object.  

```python
# Get the value of the property.
propertyValue = sketchArc_var.is2D
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
