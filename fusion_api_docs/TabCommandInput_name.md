# TabCommandInput.name Property

Parent Object: [TabCommandInput](TabCommandInput.md)  

## Description

Gets the user visible name of this input.

## Syntax

"tabCommandInput_var" is a variable referencing a TabCommandInput object.  

```python
# Get the value of the property.
propertyValue = tabCommandInput_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version July 2015
