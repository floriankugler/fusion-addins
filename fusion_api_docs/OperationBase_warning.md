# OperationBase.warning Property

Parent Object: [OperationBase](OperationBase.md)  

## Description

Returns a message corresponding to any active warning associated with the value of this parameter.

## Syntax

"operationBase_var" is a variable referencing an OperationBase object.  

```python
# Get the value of the property.
propertyValue = operationBase_var.warning
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
