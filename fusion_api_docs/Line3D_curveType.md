# Line3D.curveType Property

Parent Object: [Line3D](Line3D.md)  

## Description

Returns the type of geometry this curve represents.

## Syntax

"line3D_var" is a variable referencing a Line3D object.  

```python
# Get the value of the property.
propertyValue = line3D_var.curveType
```

## Property Value

This is a read only property whose value is a [Curve3DTypes](Curve3DTypes.md).

## Version

Introduced in version August 2014
