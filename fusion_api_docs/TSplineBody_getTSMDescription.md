# TSplineBody.getTSMDescription Method

Parent Object: [TSplineBody](TSplineBody.md)  

## Description

Returns the T-Spline body as a string in the form of a TSM description.

## Syntax

"tSplineBody_var" is a variable referencing a [TSplineBody](TSplineBody.md) object.

```python
returnValue = tSplineBody_var.getTSMDescription()
```

## Return Value

| Type   | Description                                        |
|--------|----------------------------------------------------|
| string | Returns the string formatted using the TSM format. |

## Version

Introduced in version April 2019
