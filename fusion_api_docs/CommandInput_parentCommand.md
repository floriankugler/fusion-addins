# CommandInput.parentCommand Property

Parent Object: [CommandInput](CommandInput.md)  

## Description

Gets the parent Command.

## Syntax

"commandInput_var" is a variable referencing a CommandInput object.  

```python
# Get the value of the property.
propertyValue = commandInput_var.parentCommand
```

## Property Value

This is a read only property whose value is a [Command](Command.md).

## Version

Introduced in version June 2015
