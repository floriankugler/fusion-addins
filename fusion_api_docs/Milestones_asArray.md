# Milestones.asArray Method

Parent Object: [Milestones](Milestones.md)  

## Description

Get the current list of all milestones.

## Syntax

"milestones_var" is a variable referencing a [Milestones](Milestones.md) object.

```python
returnValue = milestones_var.asArray()
```

## Return Value

| Type | Description |
|----|----|
| [Milestone](Milestone.md)\[\] | Returns the current list of all milestones. |

## Version

Introduced in version March 2024
