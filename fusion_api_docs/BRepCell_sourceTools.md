# BRepCell.sourceTools Property

Parent Object: [BRepCell](BRepCell.md)  

## Description

Returns the tools that we're using in the definition of this cell.

## Syntax

"bRepCell_var" is a variable referencing a BRepCell object.  

```python
# Get the value of the property.
propertyValue = bRepCell_var.sourceTools
```

## Property Value

This is a read only property whose value is an [ObjectCollection](ObjectCollection.md).

## Version

Introduced in version June 2015
