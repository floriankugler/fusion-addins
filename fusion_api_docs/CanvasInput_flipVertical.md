# CanvasInput.flipVertical Method

Parent Object: [CanvasInput](CanvasInput.md)  

## Description

Flips the image along the vertical axis. This is a convenience method that flips the direction of the Y axis of the transform.

## Syntax

"canvasInput_var" is a variable referencing a [CanvasInput](CanvasInput.md) object.

```python
returnValue = canvasInput_var.flipVertical()
```

## Return Value

| Type    | Description                              |
|---------|------------------------------------------|
| boolean | Returns true if the flip was successful. |

## Version

Introduced in version May 2023
