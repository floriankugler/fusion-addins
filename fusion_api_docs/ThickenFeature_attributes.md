# ThickenFeature.attributes Property

Parent Object: [ThickenFeature](ThickenFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"thickenFeature_var" is a variable referencing a ThickenFeature object.  

```python
# Get the value of the property.
propertyValue = thickenFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
