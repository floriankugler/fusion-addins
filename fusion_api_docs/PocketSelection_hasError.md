# PocketSelection.hasError Property

Parent Object: [PocketSelection](PocketSelection.md)  

## Description

Gets if errors were encountered when applying the selection to a a parent.

## Syntax

"pocketSelection_var" is a variable referencing a PocketSelection object.  

```python
# Get the value of the property.
propertyValue = pocketSelection_var.hasError
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2023
