# SketchConicCurve.sketchDimensions Property

Parent Object: [SketchConicCurve](SketchConicCurve.md)  

## Description

Returns the sketch dimensions that are attached to this curve.

## Syntax

"sketchConicCurve_var" is a variable referencing a SketchConicCurve object.  

```python
# Get the value of the property.
propertyValue = sketchConicCurve_var.sketchDimensions
```

## Property Value

This is a read only property whose value is a [SketchDimensionList](SketchDimensionList.md).

## Version

Introduced in version January 2015
