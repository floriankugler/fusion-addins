# HttpRequest.url Property

Parent Object: [HttpRequest](HttpRequest.md)  

## Description

Gets and sets the URL to make the request to.

## Syntax

"httpRequest_var" is a variable referencing a HttpRequest object.  

```python
# Get the value of the property.
propertyValue = httpRequest_var.url

# Set the value of the property.
httpRequest_var.url = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version January 2024
