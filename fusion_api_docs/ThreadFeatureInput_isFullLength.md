# ThreadFeatureInput.isFullLength Property

Parent Object: [ThreadFeatureInput](ThreadFeatureInput.md)  

## Description

Gets and sets if this thread is the full length of the cylinder. It defaults to true.

## Syntax

"threadFeatureInput_var" is a variable referencing a ThreadFeatureInput object.  

```python
# Get the value of the property.
propertyValue = threadFeatureInput_var.isFullLength

# Set the value of the property.
threadFeatureInput_var.isFullLength = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.
