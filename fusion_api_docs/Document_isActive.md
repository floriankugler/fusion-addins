# Document.isActive Property

Parent Object: [Document](Document.md)  

## Description

Gets if this document is the active document in the user interface.

## Syntax

"document_var" is a variable referencing a Document object.  

```python
# Get the value of the property.
propertyValue = document_var.isActive
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2016
