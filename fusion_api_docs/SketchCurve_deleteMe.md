# SketchCurve.deleteMe Method

Parent Object: [SketchCurve](SketchCurve.md)  

## Description

Deletes the entity from the sketch.

## Syntax

"sketchCurve_var" is a variable referencing a [SketchCurve](SketchCurve.md) object.

```python
returnValue = sketchCurve_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true is the delete was successful. |

## Version

Introduced in version August 2014
