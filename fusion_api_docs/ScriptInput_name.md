# ScriptInput.name Property

Parent Object: [ScriptInput](ScriptInput.md)  

## Description

Gets and sets the name of the script or add-in to create. This name must be unique with respect to the other scripts and add-ins in the folder specified by the targetFolder property.

## Syntax

"scriptInput_var" is a variable referencing a ScriptInput object.  

```python
# Get the value of the property.
propertyValue = scriptInput_var.name

# Set the value of the property.
scriptInput_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version October 2023
