# LineParallelToPlanarSurfaceConstraint.isDeletable Property

Parent Object: [LineParallelToPlanarSurfaceConstraint](LineParallelToPlanarSurfaceConstraint.md)  

## Description

Indicates if this constraint is deletable.

## Syntax

"lineParallelToPlanarSurfaceConstraint_var" is a variable referencing a LineParallelToPlanarSurfaceConstraint object.  

```python
# Get the value of the property.
propertyValue = lineParallelToPlanarSurfaceConstraint_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2023
