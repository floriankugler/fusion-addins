# BRepShell.lump Property

Parent Object: [BRepShell](BRepShell.md)  

## Description

Returns the parent lump of this shell.

## Syntax

"bRepShell_var" is a variable referencing a BRepShell object.  

```python
# Get the value of the property.
propertyValue = bRepShell_var.lump
```

## Property Value

This is a read only property whose value is a [BRepLump](BRepLump.md).

## Version

Introduced in version August 2014
