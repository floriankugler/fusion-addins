# WebRequestEventHandler Object

Derived from: [EventHandler](EventHandler.md) Object  

## Description

The WebRequestEventHandler is a client implemented class that can be added as a handler to an WebRequestEvent.

## Methods

| Name | Description |
|----|----|
| [notify](WebRequestEventHandler_notify.md) | The function called by Fusion when the associated event is fired. |

## Version

Introduced in version May 2016
