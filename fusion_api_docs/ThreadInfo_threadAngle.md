# ThreadInfo.threadAngle Property

Parent Object: [ThreadInfo](ThreadInfo.md)  

## Description

Returns the value that defines the thread angle. The units are degrees.

## Syntax

"threadInfo_var" is a variable referencing a ThreadInfo object.  

```python
# Get the value of the property.
propertyValue = threadInfo_var.threadAngle
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version January 2015
