# Vector3D.subtract Method

Parent Object: [Vector3D](Vector3D.md)  

## Description

Subtract a vector from this vector.

## Syntax

"vector3D_var" is a variable referencing a [Vector3D](Vector3D.md) object.

```python
returnValue = vector3D_var.subtract(vector)
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name   | Type                     | Description             |
|--------|--------------------------|-------------------------|
| vector | [Vector3D](Vector3D.md) | The vector to subtract. |

## Version

Introduced in version August 2014
