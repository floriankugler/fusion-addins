# FlatPattern.attributes Property

Parent Object: [FlatPattern](FlatPattern.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"flatPattern_var" is a variable referencing a FlatPattern object.  

```python
# Get the value of the property.
propertyValue = flatPattern_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version October 2022
