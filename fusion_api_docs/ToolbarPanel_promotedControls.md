# ToolbarPanel.promotedControls Property

Parent Object: [ToolbarPanel](ToolbarPanel.md)  

## Description

Gets the controls in the panel that have been promoted. Promoted controls are the controls that are displayed within the panel.

## Syntax

"toolbarPanel_var" is a variable referencing a ToolbarPanel object.  

```python
# Get the value of the property.
propertyValue = toolbarPanel_var.promotedControls
```

## Property Value

This is a read only property whose value is a [ToolbarControlList](ToolbarControlList.md).

## Version

Introduced in version August 2014
