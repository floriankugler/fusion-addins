# Arrange3DDefinition.solverType Property

Parent Object: [Arrange3DDefinition](Arrange3DDefinition.md)  

## Description

Gets the type of arrange feature defined by this definition.

## Syntax

"arrange3DDefinition_var" is a variable referencing an Arrange3DDefinition object.  

```python
# Get the value of the property.
propertyValue = arrange3DDefinition_var.solverType
```

## Property Value

This is a read only property whose value is an [ArrangeSolverTypes](ArrangeSolverTypes.md).

## Version

Introduced in version January 2025
