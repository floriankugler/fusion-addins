# Script.isAddIn Property

Parent Object: [Script](Script.md)  

## Description

Gets if this Script object represents a script or an add-in. Returns true if it is an add-in.

## Syntax

"script_var" is a variable referencing a Script object.  

```python
# Get the value of the property.
propertyValue = script_var.isAddIn
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version October 2023
