# SketchEllipticalArc.is2D Property

Parent Object: [SketchEllipticalArc](SketchEllipticalArc.md)  

## Description

Indicates if this curve lies entirely on the sketch x-y plane.

## Syntax

"sketchEllipticalArc_var" is a variable referencing a SketchEllipticalArc object.  

```python
# Get the value of the property.
propertyValue = sketchEllipticalArc_var.is2D
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
