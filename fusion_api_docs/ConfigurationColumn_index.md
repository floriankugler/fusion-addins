# ConfigurationColumn.index Property

Parent Object: [ConfigurationColumn](ConfigurationColumn.md)  

## Description

The index position of this column within the table. The first column is at index 0 and does not include the "Name" column.

## Syntax

"configurationColumn_var" is a variable referencing a ConfigurationColumn object.  

```python
# Get the value of the property.
propertyValue = configurationColumn_var.index
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version January 2024
