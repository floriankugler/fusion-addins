# SymmetricExtentDefinition.parentFeature Property

Parent Object: [SymmetricExtentDefinition](SymmetricExtentDefinition.md)  

## Description

Returns the parent feature that this definition is associated with. If this definition has been created statically and is not associated with a feature this property will return null.

## Syntax

"symmetricExtentDefinition_var" is a variable referencing a SymmetricExtentDefinition object.  

```python
# Get the value of the property.
propertyValue = symmetricExtentDefinition_var.parentFeature
```

## Property Value

This is a read only property whose value is a [Feature](Feature.md).

## Version

Introduced in version November 2016
