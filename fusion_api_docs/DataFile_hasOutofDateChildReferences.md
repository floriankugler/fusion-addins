# DataFile.hasOutofDateChildReferences Property

Parent Object: [DataFile](DataFile.md)  

## Description

Gets if this datafile has Children (referenced components) that are out of date (not the latest version).

## Syntax

"dataFile_var" is a variable referencing a DataFile object.  

```python
# Get the value of the property.
propertyValue = dataFile_var.hasOutofDateChildReferences
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2016
