# ScaleFeature.isUniform Property

Parent Object: [ScaleFeature](ScaleFeature.md)  

## Description

Gets if it's uniform scale.

## Syntax

"scaleFeature_var" is a variable referencing a ScaleFeature object.  

```python
# Get the value of the property.
propertyValue = scaleFeature_var.isUniform
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2015
