# PatternElement.faces Property

Parent Object: [PatternElement](PatternElement.md)  

## Description

Gets the faces generated as a result of this particular element.

## Syntax

"patternElement_var" is a variable referencing a PatternElement object.  

```python
# Get the value of the property.
propertyValue = patternElement_var.faces
```

## Property Value

This is a read only property whose value is an array of type [BRepFace](BRepFace.md).

## Version

Introduced in version November 2014
