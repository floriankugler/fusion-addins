# OffsetFacesFeature.timelineObject Property

Parent Object: [OffsetFacesFeature](OffsetFacesFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"offsetFacesFeature_var" is a variable referencing an OffsetFacesFeature object.  

```python
# Get the value of the property.
propertyValue = offsetFacesFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version June 2017
