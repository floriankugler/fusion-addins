# CustomGraphicsLines.isSelectable Property

Parent Object: [CustomGraphicsLines](CustomGraphicsLines.md)  

## Description

Gets and sets if the graphics entity is selectable within the graphics window. By default, when a new entity is created it is selectable.

## Syntax

"customGraphicsLines_var" is a variable referencing a CustomGraphicsLines object.  

```python
# Get the value of the property.
propertyValue = customGraphicsLines_var.isSelectable

# Set the value of the property.
customGraphicsLines_var.isSelectable = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version September 2017
