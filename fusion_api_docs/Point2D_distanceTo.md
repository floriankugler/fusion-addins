# Point2D.distanceTo Method

Parent Object: [Point2D](Point2D.md)  

## Description

Returns the distance from this point to another point.

## Syntax

"point2D_var" is a variable referencing a [Point2D](Point2D.md) object.

```python
returnValue = point2D_var.distanceTo(point)
```

## Return Value

| Type   | Description                        |
|--------|------------------------------------|
| double | Returns the distance to the point. |

## Parameters

| Name  | Type                   | Description                          |
|-------|------------------------|--------------------------------------|
| point | [Point2D](Point2D.md) | The point to measure the distance to |

## Version

Introduced in version August 2014
