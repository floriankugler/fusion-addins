# WorkingModel.allComponents Property

Parent Object: [WorkingModel](WorkingModel.md)  

## Description

Returns the Components collection that provides access to existing components in a design.

## Syntax

"workingModel_var" is a variable referencing a WorkingModel object.  

```python
# Get the value of the property.
propertyValue = workingModel_var.allComponents
```

## Property Value

This is a read only property whose value is a [Components](Components.md).

## Version

Introduced in version January 2024
