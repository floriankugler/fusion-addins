# CollinearConstraint.parentSketch Property

Parent Object: [CollinearConstraint](CollinearConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"collinearConstraint_var" is a variable referencing a CollinearConstraint object.  

```python
# Get the value of the property.
propertyValue = collinearConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
