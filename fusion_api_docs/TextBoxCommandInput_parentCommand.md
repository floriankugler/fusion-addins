# TextBoxCommandInput.parentCommand Property

Parent Object: [TextBoxCommandInput](TextBoxCommandInput.md)  

## Description

Gets the parent Command.

## Syntax

"textBoxCommandInput_var" is a variable referencing a TextBoxCommandInput object.  

```python
# Get the value of the property.
propertyValue = textBoxCommandInput_var.parentCommand
```

## Property Value

This is a read only property whose value is a [Command](Command.md).

## Version

Introduced in version June 2015
