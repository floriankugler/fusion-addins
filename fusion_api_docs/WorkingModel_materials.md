# WorkingModel.materials Property

Parent Object: [WorkingModel](WorkingModel.md)  

## Description

Returns the materials contained in this document.

## Syntax

"workingModel_var" is a variable referencing a WorkingModel object.  

```python
# Get the value of the property.
propertyValue = workingModel_var.materials
```

## Property Value

This is a read only property whose value is a [Materials](Materials.md).

## Version

Introduced in version January 2024
