# Operation.hasWarning Property

Parent Object: [Operation](Operation.md)  

## Description

Gets if problems were encountered when generating the operation.

## Syntax

"operation_var" is a variable referencing an Operation object.  

```python
# Get the value of the property.
propertyValue = operation_var.hasWarning
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version April 2023
