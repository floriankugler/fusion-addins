# MachineToolStation.coolants Property

Parent Object: [MachineToolStation](MachineToolStation.md)  

## Description

The coolants that this tool station can use. See MachineToolStationCoolant for possible values.

## Syntax

"machineToolStation_var" is a variable referencing a MachineToolStation object.  

```python
# Get the value of the property.
propertyValue = machineToolStation_var.coolants

# Set the value of the property.
machineToolStation_var.coolants = propertyValue
```

## Property Value

This is a read/write property whose value is an array of type integer.

## Version

Introduced in version July 2025
