# FlatPatternProduct.userParameters Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Returns the collection of User Parameters in a design

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.userParameters
```

## Property Value

This is a read only property whose value is a [UserParameters](UserParameters.md).

## Version

Introduced in version October 2022
