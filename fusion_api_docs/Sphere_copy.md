# Sphere.copy Method

Parent Object: [Sphere](Sphere.md)  

## Description

Creates and returns an independent copy of this Sphere object.

## Syntax

"sphere_var" is a variable referencing a [Sphere](Sphere.md) object.

```python
returnValue = sphere_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [Sphere](Sphere.md) | Returns a new Sphere object that is a copy of this Sphere object. |

## Version

Introduced in version August 2014
