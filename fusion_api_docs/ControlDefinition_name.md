# ControlDefinition.name Property

Parent Object: [ControlDefinition](ControlDefinition.md)  

## Description

Gets or sets the name for this control. This is the visible name displayed in the user interface.

## Syntax

"controlDefinition_var" is a variable referencing a ControlDefinition object.  

```python
# Get the value of the property.
propertyValue = controlDefinition_var.name

# Set the value of the property.
controlDefinition_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version August 2014
