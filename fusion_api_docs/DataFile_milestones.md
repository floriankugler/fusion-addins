# DataFile.milestones Property

Parent Object: [DataFile](DataFile.md)  

## Description

Returns a collection of Milestones associated with all versions of this DataFile.

## Syntax

"dataFile_var" is a variable referencing a DataFile object.  

```python
# Get the value of the property.
propertyValue = dataFile_var.milestones
```

## Property Value

This is a read only property whose value is a [Milestones](Milestones.md).

## Version

Introduced in version March 2024
