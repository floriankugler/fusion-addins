# ThreadInfo.threadSize Property

Parent Object: [ThreadInfo](ThreadInfo.md)  

## Description

Returns the string that defines the thread size.

## Syntax

"threadInfo_var" is a variable referencing a ThreadInfo object.  

```python
# Get the value of the property.
propertyValue = threadInfo_var.threadSize
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version January 2015
