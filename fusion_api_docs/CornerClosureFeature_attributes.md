# CornerClosureFeature.attributes Property

Parent Object: [CornerClosureFeature](CornerClosureFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"cornerClosureFeature_var" is a variable referencing a CornerClosureFeature object.  

```python
# Get the value of the property.
propertyValue = cornerClosureFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version January 2026
