# ConfigurationInsertStandardDesignCell.parentRow Property

Parent Object: [ConfigurationInsertStandardDesignCell](ConfigurationInsertStandardDesignCell.md)  

## Description

Returns the row this cell is in.

## Syntax

"configurationInsertStandardDesignCell_var" is a variable referencing a ConfigurationInsertStandardDesignCell object.  

```python
# Get the value of the property.
propertyValue = configurationInsertStandardDesignCell_var.parentRow
```

## Property Value

This is a read only property whose value is a [ConfigurationRow](ConfigurationRow.md).

## Version

Introduced in version September 2025
