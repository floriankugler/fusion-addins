# DataFolders.count Property

Parent Object: [DataFolders](DataFolders.md)  

## Description

The number of folders in this collection.

## Syntax

"dataFolders_var" is a variable referencing a DataFolders object.  

```python
# Get the value of the property.
propertyValue = dataFolders_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version January 2015
