# SketchEllipticalArc.sketchDimensions Property

Parent Object: [SketchEllipticalArc](SketchEllipticalArc.md)  

## Description

Returns the sketch dimensions that are attached to this curve.

## Syntax

"sketchEllipticalArc_var" is a variable referencing a SketchEllipticalArc object.  

```python
# Get the value of the property.
propertyValue = sketchEllipticalArc_var.sketchDimensions
```

## Property Value

This is a read only property whose value is a [SketchDimensionList](SketchDimensionList.md).

## Version

Introduced in version August 2014
