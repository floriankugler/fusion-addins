# Arrange2DDefinition.solverType Property

Parent Object: [Arrange2DDefinition](Arrange2DDefinition.md)  

## Description

Gets the type of arrange feature defined by this definition.

## Syntax

"arrange2DDefinition_var" is a variable referencing an Arrange2DDefinition object.  

```python
# Get the value of the property.
propertyValue = arrange2DDefinition_var.solverType
```

## Property Value

This is a read only property whose value is an [ArrangeSolverTypes](ArrangeSolverTypes.md).

## Version

Introduced in version January 2025
