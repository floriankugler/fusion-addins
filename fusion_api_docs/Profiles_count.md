# Profiles.count Property

Parent Object: [Profiles](Profiles.md)  

## Description

Returns the number of closed profiles in the sketch. Open and text based profiles are not included.

## Syntax

"profiles_var" is a variable referencing a Profiles object.  

```python
# Get the value of the property.
propertyValue = profiles_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version August 2014
