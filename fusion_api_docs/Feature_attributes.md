# Feature.attributes Property

Parent Object: [Feature](Feature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"feature_var" is a variable referencing a Feature object.  

```python
# Get the value of the property.
propertyValue = feature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
