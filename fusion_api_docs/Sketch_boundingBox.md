# Sketch.boundingBox Property

Parent Object: [Sketch](Sketch.md)  

## Description

Returns the 3D bounding box of the sketch

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version August 2014
