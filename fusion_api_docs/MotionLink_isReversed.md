# MotionLink.isReversed Property

Parent Object: [MotionLink](MotionLink.md)  

## Description

Gets and sets whether the motion is reversed or not.

## Syntax

"motionLink_var" is a variable referencing a MotionLink object.  

```python
# Get the value of the property.
propertyValue = motionLink_var.isReversed

# Set the value of the property.
motionLink_var.isReversed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version November 2025
