# Setup.attributes Property

Parent Object: [Setup](Setup.md)  

## Description

Returns the collection of attributes associated with this object.

## Syntax

"setup_var" is a variable referencing a Setup object.  

```python
# Get the value of the property.
propertyValue = setup_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version August 2021
