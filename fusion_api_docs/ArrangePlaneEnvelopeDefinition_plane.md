# ArrangePlaneEnvelopeDefinition.plane Property

Parent Object: [ArrangePlaneEnvelopeDefinition](ArrangePlaneEnvelopeDefinition.md)  

## Description

Gets and sets the ConstructionPlane the envelope is defined on.

## Syntax

"arrangePlaneEnvelopeDefinition_var" is a variable referencing an ArrangePlaneEnvelopeDefinition object.  

```python
# Get the value of the property.
propertyValue = arrangePlaneEnvelopeDefinition_var.plane

# Set the value of the property.
arrangePlaneEnvelopeDefinition_var.plane = propertyValue
```

## Property Value

This is a read/write property whose value is a [ConstructionPlane](ConstructionPlane.md).

## Version

Introduced in version January 2025
