# LoftFeature.isClosed Property

Parent Object: [LoftFeature](LoftFeature.md)  

## Description

Specifies if the loft closes back on itself. In other words, the first section is also used as the last section.

## Syntax

"loftFeature_var" is a variable referencing a LoftFeature object.  

```python
# Get the value of the property.
propertyValue = loftFeature_var.isClosed

# Set the value of the property.
loftFeature_var.isClosed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2022
