# BoxFeature.deleteMe Method

Parent Object: [BoxFeature](BoxFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"boxFeature_var" is a variable referencing a [BoxFeature](BoxFeature.md) object.

```python
returnValue = boxFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version September 2015
