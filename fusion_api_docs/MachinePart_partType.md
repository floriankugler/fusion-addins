# MachinePart.partType Property

Parent Object: [MachinePart](MachinePart.md)  

## Description

Get the type of this part.

## Syntax

"machinePart_var" is a variable referencing a MachinePart object.  

```python
# Get the value of the property.
propertyValue = machinePart_var.partType
```

## Property Value

This is a read only property whose value is a [MachinePartTypes](MachinePartTypes.md).

## Version

Introduced in version April 2023
