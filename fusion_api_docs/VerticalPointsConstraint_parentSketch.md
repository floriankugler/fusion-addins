# VerticalPointsConstraint.parentSketch Property

Parent Object: [VerticalPointsConstraint](VerticalPointsConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"verticalPointsConstraint_var" is a variable referencing a VerticalPointsConstraint object.  

```python
# Get the value of the property.
propertyValue = verticalPointsConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
