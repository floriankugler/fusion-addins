# BoolValueCommandInput.value Property

Parent Object: [BoolValueCommandInput](BoolValueCommandInput.md)  

## Description

Gets or sets the state of this input. If it's being displayed as a check box a value of true indicates the input is checked. If it's being displayed as a button, a value of true indicates the button is currently depressed.

## Syntax

"boolValueCommandInput_var" is a variable referencing a BoolValueCommandInput object.  

```python
# Get the value of the property.
propertyValue = boolValueCommandInput_var.value

# Set the value of the property.
boolValueCommandInput_var.value = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
