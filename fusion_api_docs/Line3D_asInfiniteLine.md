# Line3D.asInfiniteLine Method

Parent Object: [Line3D](Line3D.md)  

## Description

Creates an equivalent InfiniteLine3D.

## Syntax

"line3D_var" is a variable referencing a [Line3D](Line3D.md) object.

```python
returnValue = line3D_var.asInfiniteLine()
```

## Return Value

| Type                                 | Description                          |
|--------------------------------------|--------------------------------------|
| [InfiniteLine3D](InfiniteLine3D.md) | Returns an equivalent InfiniteLine3D |

## Version

Introduced in version August 2014
