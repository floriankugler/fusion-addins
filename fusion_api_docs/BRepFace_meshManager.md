# BRepFace.meshManager Property

Parent Object: [BRepFace](BRepFace.md)  

## Description

Returns a MeshManager object that allows access to existing and new meshes of this face.

## Syntax

"bRepFace_var" is a variable referencing a BRepFace object.  

```python
# Get the value of the property.
propertyValue = bRepFace_var.meshManager
```

## Property Value

This is a read only property whose value is a [MeshManager](MeshManager.md).

## Version

Introduced in version August 2014
