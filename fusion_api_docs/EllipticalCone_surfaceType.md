# EllipticalCone.surfaceType Property

Parent Object: [EllipticalCone](EllipticalCone.md)  

## Description

Returns the surface type.

## Syntax

"ellipticalCone_var" is a variable referencing an EllipticalCone object.  

```python
# Get the value of the property.
propertyValue = ellipticalCone_var.surfaceType
```

## Property Value

This is a read only property whose value is a [SurfaceTypes](SurfaceTypes.md).

## Version

Introduced in version August 2014
