# Sketch.areProfilesShown Property

Parent Object: [Sketch](Sketch.md)  

## Description

Indicates if the profiles of the sketch are displayed

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.areProfilesShown

# Set the value of the property.
sketch_var.areProfilesShown = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
