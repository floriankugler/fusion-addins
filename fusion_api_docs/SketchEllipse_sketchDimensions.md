# SketchEllipse.sketchDimensions Property

Parent Object: [SketchEllipse](SketchEllipse.md)  

## Description

Returns the sketch dimensions that are attached to this curve.

## Syntax

"sketchEllipse_var" is a variable referencing a SketchEllipse object.  

```python
# Get the value of the property.
propertyValue = sketchEllipse_var.sketchDimensions
```

## Property Value

This is a read only property whose value is a [SketchDimensionList](SketchDimensionList.md).

## Version

Introduced in version August 2014
