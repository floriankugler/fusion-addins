# ReverseNormalFeature.timelineObject Property

Parent Object: [ReverseNormalFeature](ReverseNormalFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"reverseNormalFeature_var" is a variable referencing a ReverseNormalFeature object.  

```python
# Get the value of the property.
propertyValue = reverseNormalFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version September 2015
