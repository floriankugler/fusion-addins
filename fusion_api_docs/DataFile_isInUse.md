# DataFile.isInUse Property

Parent Object: [DataFile](DataFile.md)  

## Description

Gets if this DataFile is currently in use (opened for edit) by any other user.

## Syntax

"dataFile_var" is a variable referencing a DataFile object.  

```python
# Get the value of the property.
propertyValue = dataFile_var.isInUse
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2016
