# ConstructionPlane.definition Property

Parent Object: [ConstructionPlane](ConstructionPlane.md)  

## Description

Returns the ConstructionPlaneDefinition object which provides access to the information defining this ConstructionPlane.

## Syntax

"constructionPlane_var" is a variable referencing a ConstructionPlane object.  

```python
# Get the value of the property.
propertyValue = constructionPlane_var.definition
```

## Property Value

This is a read only property whose value is a [ConstructionPlaneDefinition](ConstructionPlaneDefinition.md).

## Version

Introduced in version August 2014
