# TextCommandPalette.isTransparent Property

Parent Object: [TextCommandPalette](TextCommandPalette.md)  

## Description

Returns if this palette was created as a transparent palette.

## Syntax

"textCommandPalette_var" is a variable referencing a TextCommandPalette object.  

```python
# Get the value of the property.
propertyValue = textCommandPalette_var.isTransparent
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version March 2024
