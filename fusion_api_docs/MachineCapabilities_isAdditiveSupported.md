# MachineCapabilities.isAdditiveSupported Property

Parent Object: [MachineCapabilities](MachineCapabilities.md)  

## Description

Gets and sets if the machine is capable of additive operations.

## Syntax

"machineCapabilities_var" is a variable referencing a MachineCapabilities object.  

```python
# Get the value of the property.
propertyValue = machineCapabilities_var.isAdditiveSupported

# Set the value of the property.
machineCapabilities_var.isAdditiveSupported = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version April 2023
