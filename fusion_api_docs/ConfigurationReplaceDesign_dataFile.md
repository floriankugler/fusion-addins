# ConfigurationReplaceDesign.dataFile Property

Parent Object: [ConfigurationReplaceDesign](ConfigurationReplaceDesign.md)  

## Description

Gets the Design object associated with this ConfigurationReplaceDesign object. This must be a DataFile object that represents a standard design, not a configured design.

## Syntax

"configurationReplaceDesign_var" is a variable referencing a ConfigurationReplaceDesign object.  

```python
# Get the value of the property.
propertyValue = configurationReplaceDesign_var.dataFile
```

## Property Value

This is a read only property whose value is a [DataFile](DataFile.md).

## Version

Introduced in version September 2025
