# SketchTangentDistanceDimension.deleteMe Method

Parent Object: [SketchTangentDistanceDimension](SketchTangentDistanceDimension.md)  

## Description

Deletes this dimension. The IsDeletable property indicates if this dimension can be deleted.

## Syntax

"sketchTangentDistanceDimension_var" is a variable referencing a [SketchTangentDistanceDimension](SketchTangentDistanceDimension.md) object.

```python
returnValue = sketchTangentDistanceDimension_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version July 2022
