# FlatPattern.extentLinesBody Property

Parent Object: [FlatPattern](FlatPattern.md)  

## Description

Returns the wire B-Rep body that represents the extent lines of the flattened sheet metal part.

## Syntax

"flatPattern_var" is a variable referencing a FlatPattern object.  

```python
# Get the value of the property.
propertyValue = flatPattern_var.extentLinesBody
```

## Property Value

This is a read only property whose value is a [BRepBody](BRepBody.md).

## Version

Introduced in version October 2022
