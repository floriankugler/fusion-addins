# CAMPattern.hasError Property

Parent Object: [CAMPattern](CAMPattern.md)  

## Description

Gets if errors were encountered when generating the operation.

## Syntax

"cAMPattern_var" is a variable referencing a CAMPattern object.  

```python
# Get the value of the property.
propertyValue = cAMPattern_var.hasError
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version April 2023
