# JointOrigin.offsetX Property

Parent Object: [JointOrigin](JointOrigin.md)  

## Description

Gets the parameter that controls the X offset direction. The value can be changed using the functionality of the returned ModelParameter object.

## Syntax

"jointOrigin_var" is a variable referencing a JointOrigin object.  

```python
# Get the value of the property.
propertyValue = jointOrigin_var.offsetX
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version September 2015
