# ValueCommandInput.isValidExpression Property

Parent Object: [ValueCommandInput](ValueCommandInput.md)  

## Description

Returns true if the current expression is valid and can be evaluated. If this is false, the value returned should be ignored because there currently is not a valid value.

## Syntax

"valueCommandInput_var" is a variable referencing a ValueCommandInput object.  

```python
# Get the value of the property.
propertyValue = valueCommandInput_var.isValidExpression
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version November 2016
