# SplitBodyFeature.healthState Property

Parent Object: [SplitBodyFeature](SplitBodyFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"splitBodyFeature_var" is a variable referencing a SplitBodyFeature object.  

```python
# Get the value of the property.
propertyValue = splitBodyFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
