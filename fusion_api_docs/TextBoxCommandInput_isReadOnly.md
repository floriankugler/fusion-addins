# TextBoxCommandInput.isReadOnly Property

Parent Object: [TextBoxCommandInput](TextBoxCommandInput.md)  

## Description

Gets and sets if the text box is read-only or not. If it is read-only the user cannot edit the text.

## Syntax

"textBoxCommandInput_var" is a variable referencing a TextBoxCommandInput object.  

```python
# Get the value of the property.
propertyValue = textBoxCommandInput_var.isReadOnly

# Set the value of the property.
textBoxCommandInput_var.isReadOnly = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version June 2015
