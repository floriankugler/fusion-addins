# BRepEdge.isDegenerate Property

Parent Object: [BRepEdge](BRepEdge.md)  

## Description

Returns if the edge's geometry is degenerate. For example, the apex of a cone is a degenerate edge.

## Syntax

"bRepEdge_var" is a variable referencing a BRepEdge object.  

```python
# Get the value of the property.
propertyValue = bRepEdge_var.isDegenerate
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
