# MarkingMenuEventHandler Object

Derived from: [EventHandler](EventHandler.md) Object  

## Description

The MarkingMenuEventHandler is a client implemented class that can be added as a handler to a MarkingMenuEvent.

## Methods

| Name | Description |
|----|----|
| [notify](MarkingMenuEventHandler_notify.md) | The function called by Fusion when the associated event is fired. |

## Version

Introduced in version January 2017
