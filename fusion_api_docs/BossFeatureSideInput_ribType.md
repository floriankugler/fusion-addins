# BossFeatureSideInput.ribType Property

Parent Object: [BossFeatureSideInput](BossFeatureSideInput.md)  

## Description

Type of boss ribs this feature represents.

## Syntax

"bossFeatureSideInput_var" is a variable referencing a BossFeatureSideInput object.  

```python
# Get the value of the property.
propertyValue = bossFeatureSideInput_var.ribType

# Set the value of the property.
bossFeatureSideInput_var.ribType = propertyValue
```

## Property Value

This is a read/write property whose value is a [BossRibShapeTypes](BossRibShapeTypes.md).
