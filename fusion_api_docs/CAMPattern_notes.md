# CAMPattern.notes Property

Parent Object: [CAMPattern](CAMPattern.md)  

## Description

Gets and sets the notes of the operation.

## Syntax

"cAMPattern_var" is a variable referencing a CAMPattern object.  

```python
# Get the value of the property.
propertyValue = cAMPattern_var.notes

# Set the value of the property.
cAMPattern_var.notes = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version May 2020
