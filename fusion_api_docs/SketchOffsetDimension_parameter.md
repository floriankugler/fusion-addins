# SketchOffsetDimension.parameter Property

Parent Object: [SketchOffsetDimension](SketchOffsetDimension.md)  

## Description

Returns the associated parameter or null if there is no associated parameter.

## Syntax

"sketchOffsetDimension_var" is a variable referencing a SketchOffsetDimension object.  

```python
# Get the value of the property.
propertyValue = sketchOffsetDimension_var.parameter
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version August 2014
