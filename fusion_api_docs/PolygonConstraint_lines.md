# PolygonConstraint.lines Property

Parent Object: [PolygonConstraint](PolygonConstraint.md)  

## Description

This function is retired. See more information in the 'Remarks' section below.  

This has been retired and you should use the points property instead.

## Syntax

"polygonConstraint_var" is a variable referencing a PolygonConstraint object.  

```python
# Get the value of the property.
propertyValue = polygonConstraint_var.lines
```

## Property Value

This is a read only property whose value is an array of type [SketchLine](SketchLine.md).

## Version

Introduced in version August 2016  
Retired in version July 2024
