# DataFile.parentReferences Property

Parent Object: [DataFile](DataFile.md)  

## Description

Returns a collection DataFiles collection that are the parents (designs that reference) this datafile.

## Syntax

"dataFile_var" is a variable referencing a DataFile object.  

```python
# Get the value of the property.
propertyValue = dataFile_var.parentReferences
```

## Property Value

This is a read only property whose value is a [DataFiles](DataFiles.md).

## Version

Introduced in version September 2016
