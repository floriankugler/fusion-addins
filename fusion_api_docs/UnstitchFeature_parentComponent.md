# UnstitchFeature.parentComponent Property

Parent Object: [UnstitchFeature](UnstitchFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"unstitchFeature_var" is a variable referencing a UnstitchFeature object.  

```python
# Get the value of the property.
propertyValue = unstitchFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version July 2015
