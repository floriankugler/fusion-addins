# SketchControlPointSpline.is2D Property

Parent Object: [SketchControlPointSpline](SketchControlPointSpline.md)  

## Description

Indicates if this curve lies entirely on the sketch x-y plane.

## Syntax

"sketchControlPointSpline_var" is a variable referencing a SketchControlPointSpline object.  

```python
# Get the value of the property.
propertyValue = sketchControlPointSpline_var.is2D
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2022
