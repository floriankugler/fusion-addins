# ColorProperty.isReadOnly Property

Parent Object: [ColorProperty](ColorProperty.md)  

## Description

Indicates if this property is read-only. If True any attempted edits will fail.

## Syntax

"colorProperty_var" is a variable referencing a ColorProperty object.  

```python
# Get the value of the property.
propertyValue = colorProperty_var.isReadOnly
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
