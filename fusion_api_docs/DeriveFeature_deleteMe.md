# DeriveFeature.deleteMe Method

Parent Object: [DeriveFeature](DeriveFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"deriveFeature_var" is a variable referencing a [DeriveFeature](DeriveFeature.md) object.

```python
returnValue = deriveFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version January 2026
