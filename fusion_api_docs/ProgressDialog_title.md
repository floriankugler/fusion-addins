# ProgressDialog.title Property

Parent Object: [ProgressDialog](ProgressDialog.md)  

## Description

Gets and sets the title of the progress dialog

## Syntax

"progressDialog_var" is a variable referencing a ProgressDialog object.  

```python
# Get the value of the property.
propertyValue = progressDialog_var.title

# Set the value of the property.
progressDialog_var.title = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version January 2016
