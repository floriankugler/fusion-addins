# Preferences.networkPreferences Property

Parent Object: [Preferences](Preferences.md)  

## Description

Gets the NetworkPreferences object.

## Syntax

"preferences_var" is a variable referencing a Preferences object.  

```python
# Get the value of the property.
propertyValue = preferences_var.networkPreferences
```

## Property Value

This is a read only property whose value is a [NetworkPreferences](NetworkPreferences.md).

## Version

Introduced in version August 2014
