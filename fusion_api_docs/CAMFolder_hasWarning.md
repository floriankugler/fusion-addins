# CAMFolder.hasWarning Property

Parent Object: [CAMFolder](CAMFolder.md)  

## Description

Gets if problems were encountered when generating the operation.

## Syntax

"cAMFolder_var" is a variable referencing a CAMFolder object.  

```python
# Get the value of the property.
propertyValue = cAMFolder_var.hasWarning
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version April 2023
