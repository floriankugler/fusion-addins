# CAMFolders.addFolder Method

Parent Object: [CAMFolders](CAMFolders.md)  

## Description

Creates a folder with the specified name and returns it as CAMFolder object.

## Syntax

"cAMFolders_var" is a variable referencing a [CAMFolders](CAMFolders.md) object.

```python
returnValue = cAMFolders_var.addFolder(name)
```

## Return Value

| Type | Description |
|----|----|
| [CAMFolder](CAMFolder.md) | Returns the newly created folder or null if folder can't be created. |

## Parameters

| Name | Type   | Description                                            |
|------|--------|--------------------------------------------------------|
| name | string | The name (as it appears in the browser) of the folder. |

## Version

Introduced in version January 2024
