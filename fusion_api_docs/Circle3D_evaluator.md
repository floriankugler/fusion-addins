# Circle3D.evaluator Property

Parent Object: [Circle3D](Circle3D.md)  

## Description

Returns an evaluator object that lets you perform additional evaluations on the curve.

## Syntax

"circle3D_var" is a variable referencing a Circle3D object.  

```python
# Get the value of the property.
propertyValue = circle3D_var.evaluator
```

## Property Value

This is a read only property whose value is a [CurveEvaluator3D](CurveEvaluator3D.md).

## Version

Introduced in version August 2014
