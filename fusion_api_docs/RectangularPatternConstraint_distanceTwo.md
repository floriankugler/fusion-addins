# RectangularPatternConstraint.distanceTwo Property

Parent Object: [RectangularPatternConstraint](RectangularPatternConstraint.md)  

## Description

Returns the parameter that controls the distance in second direction. To change the value, use the properties on the returned ModelParameter object.

## Syntax

"rectangularPatternConstraint_var" is a variable referencing a RectangularPatternConstraint object.  

```python
# Get the value of the property.
propertyValue = rectangularPatternConstraint_var.distanceTwo
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version September 2022
