# Component.xYConstructionPlane Property

Parent Object: [Component](Component.md)  

## Description

Returns the XY origin construction plane.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.xYConstructionPlane
```

## Property Value

This is a read only property whose value is a [ConstructionPlane](ConstructionPlane.md).
