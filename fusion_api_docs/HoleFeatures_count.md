# HoleFeatures.count Property

Parent Object: [HoleFeatures](HoleFeatures.md)  

## Description

The number of hole features in the collection.

## Syntax

"holeFeatures_var" is a variable referencing a HoleFeatures object.  

```python
# Get the value of the property.
propertyValue = holeFeatures_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version August 2014
