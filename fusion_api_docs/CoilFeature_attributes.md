# CoilFeature.attributes Property

Parent Object: [CoilFeature](CoilFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"coilFeature_var" is a variable referencing a CoilFeature object.  

```python
# Get the value of the property.
propertyValue = coilFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
