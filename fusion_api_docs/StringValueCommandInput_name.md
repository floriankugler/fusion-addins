# StringValueCommandInput.name Property

Parent Object: [StringValueCommandInput](StringValueCommandInput.md)  

## Description

Gets the user visible name of this input.

## Syntax

"stringValueCommandInput_var" is a variable referencing a StringValueCommandInput object.  

```python
# Get the value of the property.
propertyValue = stringValueCommandInput_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2014
