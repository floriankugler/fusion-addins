# SketchTextInput.isVerticalFlip Property

Parent Object: [SketchTextInput](SketchTextInput.md)  

## Description

Gets and sets if the text is flipped vertically.

## Syntax

"sketchTextInput_var" is a variable referencing a SketchTextInput object.  

```python
# Get the value of the property.
propertyValue = sketchTextInput_var.isVerticalFlip

# Set the value of the property.
sketchTextInput_var.isVerticalFlip = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.
