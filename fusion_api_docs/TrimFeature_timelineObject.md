# TrimFeature.timelineObject Property

Parent Object: [TrimFeature](TrimFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"trimFeature_var" is a variable referencing a TrimFeature object.  

```python
# Get the value of the property.
propertyValue = trimFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version July 2015
