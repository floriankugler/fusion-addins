# BossFeature.parentComponent Property

Parent Object: [BossFeature](BossFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"bossFeature_var" is a variable referencing a BossFeature object.  

```python
# Get the value of the property.
propertyValue = bossFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version October 2022
