# Features.webFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the existing web features.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.webFeatures
```

## Property Value

This is a read only property whose value is a [WebFeatures](WebFeatures.md).

## Version

Introduced in version September 2015
