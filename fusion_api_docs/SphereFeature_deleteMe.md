# SphereFeature.deleteMe Method

Parent Object: [SphereFeature](SphereFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"sphereFeature_var" is a variable referencing a [SphereFeature](SphereFeature.md) object.

```python
returnValue = sphereFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version September 2015
