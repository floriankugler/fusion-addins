# ArrangeResultEnvelopes.item Method

Parent Object: [ArrangeResultEnvelopes](ArrangeResultEnvelopes.md)  

## Description

Returns the specified Arrange envelope result using an index into the collection.

## Syntax

"arrangeResultEnvelopes_var" is a variable referencing an [ArrangeResultEnvelopes](ArrangeResultEnvelopes.md) object.

```python
returnValue = arrangeResultEnvelopes_var.item(index)
```

## Return Value

| Type | Description |
|----|----|
| [ArrangeResultEnvelope](ArrangeResultEnvelope.md) | Returns the specified item or null if an invalid index was specified. |

## Parameters

| Name | Type | Description |
|----|----|----|
| index | uinteger | The index of the item within the collection to return. The first item in the collection has an index of 0. |

## Version

Introduced in version January 2025
