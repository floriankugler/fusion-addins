# DropDownControl.controls Property

Parent Object: [DropDownControl](DropDownControl.md)  

## Description

Gets the associated ToolbarControls collection. Through this you can add additional controls to the drop-down.

## Syntax

"dropDownControl_var" is a variable referencing a DropDownControl object.  

```python
# Get the value of the property.
propertyValue = dropDownControl_var.controls
```

## Property Value

This is a read only property whose value is a [ToolbarControls](ToolbarControls.md).
