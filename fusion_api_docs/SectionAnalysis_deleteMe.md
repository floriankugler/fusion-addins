# SectionAnalysis.deleteMe Method

Parent Object: [SectionAnalysis](SectionAnalysis.md)  

## Description

A method that deletes this Analysis.

## Syntax

"sectionAnalysis_var" is a variable referencing a [SectionAnalysis](SectionAnalysis.md) object.

```python
returnValue = sectionAnalysis_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version January 2023
