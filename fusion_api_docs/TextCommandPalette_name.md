# TextCommandPalette.name Property

Parent Object: [TextCommandPalette](TextCommandPalette.md)  

## Description

Gets and set the name of the palette as seen in the user interface. The name of native palettes cannot be set.

## Syntax

"textCommandPalette_var" is a variable referencing a TextCommandPalette object.  

```python
# Get the value of the property.
propertyValue = textCommandPalette_var.name

# Set the value of the property.
textCommandPalette_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version August 2017
