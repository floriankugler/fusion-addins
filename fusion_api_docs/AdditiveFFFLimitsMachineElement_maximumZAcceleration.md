# AdditiveFFFLimitsMachineElement.maximumZAcceleration Property

Parent Object: [AdditiveFFFLimitsMachineElement](AdditiveFFFLimitsMachineElement.md)  

## Description

Maximum supported acceleration for motion in the Z axis in cm/s^2.

## Syntax

"additiveFFFLimitsMachineElement_var" is a variable referencing an AdditiveFFFLimitsMachineElement object.  

```python
# Get the value of the property.
propertyValue = additiveFFFLimitsMachineElement_var.maximumZAcceleration

# Set the value of the property.
additiveFFFLimitsMachineElement_var.maximumZAcceleration = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version July 2023
