# TorusFeature.timelineObject Property

Parent Object: [TorusFeature](TorusFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"torusFeature_var" is a variable referencing a TorusFeature object.  

```python
# Get the value of the property.
propertyValue = torusFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version September 2015
