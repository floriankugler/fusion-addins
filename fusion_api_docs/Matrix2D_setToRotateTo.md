# Matrix2D.setToRotateTo Method

Parent Object: [Matrix2D](Matrix2D.md)  

## Description

Sets to the matrix of rotation that would align the 'from' vector with the 'to' vector.

## Syntax

"matrix2D_var" is a variable referencing a [Matrix2D](Matrix2D.md) object.

```python
returnValue = matrix2D_var.setToRotateTo(from, to)
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name | Type                     | Description      |
|------|--------------------------|------------------|
| from | [Vector2D](Vector2D.md) | The from vector. |
| to   | [Vector2D](Vector2D.md) | The to vector.   |

## Version

Introduced in version August 2014
