# TextBoxCommandInput.deleteMe Method

Parent Object: [TextBoxCommandInput](TextBoxCommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"textBoxCommandInput_var" is a variable referencing a [TextBoxCommandInput](TextBoxCommandInput.md) object.

```python
returnValue = textBoxCommandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2020
