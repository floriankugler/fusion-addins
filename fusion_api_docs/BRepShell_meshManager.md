# BRepShell.meshManager Property

Parent Object: [BRepShell](BRepShell.md)  

## Description

Returns the mesh manager object for this shell.

## Syntax

"bRepShell_var" is a variable referencing a BRepShell object.  

```python
# Get the value of the property.
propertyValue = bRepShell_var.meshManager
```

## Property Value

This is a read only property whose value is a [MeshManager](MeshManager.md).

## Version

Introduced in version August 2014
