# SketchControlPointSpline.isFixed Property

Parent Object: [SketchControlPointSpline](SketchControlPointSpline.md)  

## Description

Indicates if this geometry is "fixed".

## Syntax

"sketchControlPointSpline_var" is a variable referencing a SketchControlPointSpline object.  

```python
# Get the value of the property.
propertyValue = sketchControlPointSpline_var.isFixed

# Set the value of the property.
sketchControlPointSpline_var.isFixed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version July 2022
