# ImageCommandInput.isVisible Property

Parent Object: [ImageCommandInput](ImageCommandInput.md)  

## Description

Gets or sets if this input will be visible to the user.  

Setting a SelectionCommandInput to be invisible will clear any selections it currently has.

## Syntax

"imageCommandInput_var" is a variable referencing an ImageCommandInput object.  

```python
# Get the value of the property.
propertyValue = imageCommandInput_var.isVisible

# Set the value of the property.
imageCommandInput_var.isVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version July 2015
