# StitchFeatureInput.operation Property

Parent Object: [StitchFeatureInput](StitchFeatureInput.md)  

## Description

Gets and sets the feature operation to perform. This property value is only valid if the isSolid property returns true. Otherwise the value of this property is ignored.

## Syntax

"stitchFeatureInput_var" is a variable referencing a StitchFeatureInput object.  

```python
# Get the value of the property.
propertyValue = stitchFeatureInput_var.operation

# Set the value of the property.
stitchFeatureInput_var.operation = propertyValue
```

## Property Value

This is a read/write property whose value is a [FeatureOperations](FeatureOperations.md).

## Version

Introduced in version June 2015
