# ExtrudeFeature.healthState Property

Parent Object: [ExtrudeFeature](ExtrudeFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"extrudeFeature_var" is a variable referencing an ExtrudeFeature object.  

```python
# Get the value of the property.
propertyValue = extrudeFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).
