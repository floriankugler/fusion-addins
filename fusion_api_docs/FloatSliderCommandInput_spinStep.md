# FloatSliderCommandInput.spinStep Property

Parent Object: [FloatSliderCommandInput](FloatSliderCommandInput.md)  

## Description

Gets and sets the spin step value in the unit type set by the unitType argument. The value should be more than zero. This is the amount the slider will advance when the user clicks the spin button beside the value.

## Syntax

"floatSliderCommandInput_var" is a variable referencing a FloatSliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = floatSliderCommandInput_var.spinStep

# Set the value of the property.
floatSliderCommandInput_var.spinStep = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version June 2015
