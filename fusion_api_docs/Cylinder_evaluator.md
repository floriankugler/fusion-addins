# Cylinder.evaluator Property

Parent Object: [Cylinder](Cylinder.md)  

## Description

Returns the surface evaluator.

## Syntax

"cylinder_var" is a variable referencing a Cylinder object.  

```python
# Get the value of the property.
propertyValue = cylinder_var.evaluator
```

## Property Value

This is a read only property whose value is a [SurfaceEvaluator](SurfaceEvaluator.md).

## Version

Introduced in version August 2014
