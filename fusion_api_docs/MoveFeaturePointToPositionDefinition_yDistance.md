# MoveFeaturePointToPositionDefinition.yDistance Property

Parent Object: [MoveFeaturePointToPositionDefinition](MoveFeaturePointToPositionDefinition.md)  

## Description

Gets the model parameter that controls the offset in the Y direction. You can use properties

## Syntax

"moveFeaturePointToPositionDefinition_var" is a variable referencing a MoveFeaturePointToPositionDefinition object.  

```python
# Get the value of the property.
propertyValue = moveFeaturePointToPositionDefinition_var.yDistance
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version January 2023
