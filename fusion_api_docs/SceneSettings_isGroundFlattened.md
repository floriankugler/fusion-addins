# SceneSettings.isGroundFlattened Property

Parent Object: [SceneSettings](SceneSettings.md)  

## Description

Gets and sets if the ground plane is "textured" where the environment image is mapped as a texture.

## Syntax

"sceneSettings_var" is a variable referencing a SceneSettings object.  

```python
# Get the value of the property.
propertyValue = sceneSettings_var.isGroundFlattened

# Set the value of the property.
sceneSettings_var.isGroundFlattened = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version May 2023
