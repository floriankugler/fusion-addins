# SketchArc.deleteMe Method

Parent Object: [SketchArc](SketchArc.md)  

## Description

Deletes the entity from the sketch.

## Syntax

"sketchArc_var" is a variable referencing a [SketchArc](SketchArc.md) object.

```python
returnValue = sketchArc_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true is the delete was successful. |

## Version

Introduced in version August 2014
