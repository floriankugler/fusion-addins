# CAMHoleRecognition.notes Property

Parent Object: [CAMHoleRecognition](CAMHoleRecognition.md)  

## Description

Gets and sets the notes of the operation.

## Syntax

"cAMHoleRecognition_var" is a variable referencing a CAMHoleRecognition object.  

```python
# Get the value of the property.
propertyValue = cAMHoleRecognition_var.notes

# Set the value of the property.
cAMHoleRecognition_var.notes = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version April 2023
