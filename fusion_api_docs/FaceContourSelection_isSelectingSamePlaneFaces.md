# FaceContourSelection.isSelectingSamePlaneFaces Property

Parent Object: [FaceContourSelection](FaceContourSelection.md)  

## Description

Property to get and set if all planar faces lying in the same plane as the selected face should be automatically selected as well.

## Syntax

"faceContourSelection_var" is a variable referencing a FaceContourSelection object.  

```python
# Get the value of the property.
propertyValue = faceContourSelection_var.isSelectingSamePlaneFaces

# Set the value of the property.
faceContourSelection_var.isSelectingSamePlaneFaces = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version April 2023
