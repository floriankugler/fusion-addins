# SketchFixedSpline.parentSketch Property

Parent Object: [SketchFixedSpline](SketchFixedSpline.md)  

## Description

Returns the parent sketch.

## Syntax

"sketchFixedSpline_var" is a variable referencing a SketchFixedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFixedSpline_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
