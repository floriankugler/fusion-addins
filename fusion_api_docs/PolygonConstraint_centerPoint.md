# PolygonConstraint.centerPoint Property

Parent Object: [PolygonConstraint](PolygonConstraint.md)  

## Description

Returns the SketchPoint that acts as the center point of the polygon.

## Syntax

"polygonConstraint_var" is a variable referencing a PolygonConstraint object.  

```python
# Get the value of the property.
propertyValue = polygonConstraint_var.centerPoint
```

## Property Value

This is a read only property whose value is a [SketchPoint](SketchPoint.md).

## Version

Introduced in version July 2024
