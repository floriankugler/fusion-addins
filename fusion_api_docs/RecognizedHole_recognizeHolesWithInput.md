# RecognizedHole.recognizeHolesWithInput Method

Parent Object: [RecognizedHole](RecognizedHole.md)  

## Description

Returns a collection of all recognized holes.

## Syntax

This is a static method.  

```python

returnValue = adsk.cam.RecognizedHole.recognizeHolesWithInput(bodies, input)
```

## Parameters

| Name | Type | Description |
|----|----|----|
| bodies | Base\[\] | Model bodies on which to recognize holes. |
| input | [RecognizedHolesInput](RecognizedHolesInput.md) | Input object that contains filtering settings |

## Version

Introduced in version January 2024
