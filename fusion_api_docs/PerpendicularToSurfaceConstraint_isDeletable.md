# PerpendicularToSurfaceConstraint.isDeletable Property

Parent Object: [PerpendicularToSurfaceConstraint](PerpendicularToSurfaceConstraint.md)  

## Description

Indicates if this constraint is deletable.

## Syntax

"perpendicularToSurfaceConstraint_var" is a variable referencing a PerpendicularToSurfaceConstraint object.  

```python
# Get the value of the property.
propertyValue = perpendicularToSurfaceConstraint_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2023
