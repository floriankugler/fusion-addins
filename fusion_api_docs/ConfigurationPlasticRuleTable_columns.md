# ConfigurationPlasticRuleTable.columns Property

Parent Object: [ConfigurationPlasticRuleTable](ConfigurationPlasticRuleTable.md)  

## Description

Returns the collection that provides access to the columns in this table.

## Syntax

"configurationPlasticRuleTable_var" is a variable referencing a ConfigurationPlasticRuleTable object.  

```python
# Get the value of the property.
propertyValue = configurationPlasticRuleTable_var.columns
```

## Property Value

This is a read only property whose value is a [ConfigurationPlasticRuleColumns](ConfigurationPlasticRuleColumns.md).

## Version

Introduced in version January 2024
