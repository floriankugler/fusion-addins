# FlatPattern.healthState Property

Parent Object: [FlatPattern](FlatPattern.md)  

## Description

Returns the current health state of the feature.

## Syntax

"flatPattern_var" is a variable referencing a FlatPattern object.  

```python
# Get the value of the property.
propertyValue = flatPattern_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version October 2022
