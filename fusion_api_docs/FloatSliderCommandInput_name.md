# FloatSliderCommandInput.name Property

Parent Object: [FloatSliderCommandInput](FloatSliderCommandInput.md)  

## Description

Gets the user visible name of this input.

## Syntax

"floatSliderCommandInput_var" is a variable referencing a FloatSliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = floatSliderCommandInput_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version June 2015
