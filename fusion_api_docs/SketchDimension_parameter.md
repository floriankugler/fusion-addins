# SketchDimension.parameter Property

Parent Object: [SketchDimension](SketchDimension.md)  

## Description

Returns the associated parameter or null if there is no associated parameter.

## Syntax

"sketchDimension_var" is a variable referencing a SketchDimension object.  

```python
# Get the value of the property.
propertyValue = sketchDimension_var.parameter
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version August 2014
