# Sketch.parentComponent Property

Parent Object: [Sketch](Sketch.md)  

## Description

Returns the parent Component.

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version August 2014
