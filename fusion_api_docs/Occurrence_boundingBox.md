# Occurrence.boundingBox Property

Parent Object: [Occurrence](Occurrence.md)  

## Description

Returns the bounding box of this occurrence.

## Syntax

"occurrence_var" is a variable referencing an Occurrence object.  

```python
# Get the value of the property.
propertyValue = occurrence_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version November 2015
