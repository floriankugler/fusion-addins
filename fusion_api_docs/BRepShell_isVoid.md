# BRepShell.isVoid Property

Parent Object: [BRepShell](BRepShell.md)  

## Description

Returns true if the faces of this shell bound a void or an empty space within an outer shell.

## Syntax

"bRepShell_var" is a variable referencing a BRepShell object.  

```python
# Get the value of the property.
propertyValue = bRepShell_var.isVoid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
