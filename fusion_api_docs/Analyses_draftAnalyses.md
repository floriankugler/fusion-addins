# Analyses.draftAnalyses Property

Parent Object: [Analyses](Analyses.md)  

## Description

Returns the DraftAnalyses object, which provides access to any existing DraftAnalysis objects in the design.

## Syntax

"analyses_var" is a variable referencing an Analyses object.  

```python
# Get the value of the property.
propertyValue = analyses_var.draftAnalyses
```

## Property Value

This is a read only property whose value is a [DraftAnalyses](DraftAnalyses.md).

## Version

Introduced in version January 2023
