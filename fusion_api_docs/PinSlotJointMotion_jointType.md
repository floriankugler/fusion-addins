# PinSlotJointMotion.jointType Property

Parent Object: [PinSlotJointMotion](PinSlotJointMotion.md)  

## Description

Returns an enum value indicating the type of joint this joint represents.

## Syntax

"pinSlotJointMotion_var" is a variable referencing a PinSlotJointMotion object.  

```python
# Get the value of the property.
propertyValue = pinSlotJointMotion_var.jointType
```

## Property Value

This is a read only property whose value is a [JointTypes](JointTypes.md).

## Version

Introduced in version July 2015
