# Viewport.refresh Method

Parent Object: [Viewport](Viewport.md)  

## Description

Forces the view to refresh. It is sometimes useful to force a refresh to be able to see edits that have been made using the API.

## Syntax

"viewport_var" is a variable referencing a [Viewport](Viewport.md) object.

```python
returnValue = viewport_var.refresh()
```

## Return Value

| Type    | Description                                   |
|---------|-----------------------------------------------|
| boolean | Returns true if the operation was successful. |

## Version

Introduced in version August 2014
