# Polyline2D.asNurbsCurve Property

Parent Object: [Polyline2D](Polyline2D.md)  

## Description

Returns a NURBS curve that is geometrically identical to the polyline.

## Syntax

"polyline2D_var" is a variable referencing a Polyline2D object.  

```python
# Get the value of the property.
propertyValue = polyline2D_var.asNurbsCurve
```

## Property Value

This is a read only property whose value is a [NurbsCurve2D](NurbsCurve2D.md).

## Version

Introduced in version September 2024
