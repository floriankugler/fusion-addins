# SketchLinearDiameterDimension.line Property

Parent Object: [SketchLinearDiameterDimension](SketchLinearDiameterDimension.md)  

## Description

The first line being constrained.

## Syntax

"sketchLinearDiameterDimension_var" is a variable referencing a SketchLinearDiameterDimension object.  

```python
# Get the value of the property.
propertyValue = sketchLinearDiameterDimension_var.line
```

## Property Value

This is a read only property whose value is a [SketchLine](SketchLine.md).

## Version

Introduced in version September 2022
