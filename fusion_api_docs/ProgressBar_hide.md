# ProgressBar.hide Method

Parent Object: [ProgressBar](ProgressBar.md)  

## Description

Hides the progress or busy bar. This should be used when the process has completed.

## Syntax

"progressBar_var" is a variable referencing a [ProgressBar](ProgressBar.md) object.

```python
returnValue = progressBar_var.hide()
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Version

Introduced in version May 2024
