# AccessibilityAnalyses.count Property

Parent Object: [AccessibilityAnalyses](AccessibilityAnalyses.md)  

## Description

Returns the number of AccessibilityAnalysis objects in the collection.

## Syntax

"accessibilityAnalyses_var" is a variable referencing an AccessibilityAnalyses object.  

```python
# Get the value of the property.
propertyValue = accessibilityAnalyses_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version January 2023
