# SketchText.isFixed Property

Parent Object: [SketchText](SketchText.md)  

## Description

Indicates if this geometry is "fixed".

## Syntax

"sketchText_var" is a variable referencing a SketchText object.  

```python
# Get the value of the property.
propertyValue = sketchText_var.isFixed

# Set the value of the property.
sketchText_var.isFixed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version March 2015
