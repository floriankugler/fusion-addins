# GeneralPreferences.offlineCachePeriod Property

Parent Object: [GeneralPreferences](GeneralPreferences.md)  

## Description

Gets and sets the length of time, in days, that the offline cache of a document will remain.

## Syntax

"generalPreferences_var" is a variable referencing a GeneralPreferences object.  

```python
# Get the value of the property.
propertyValue = generalPreferences_var.offlineCachePeriod

# Set the value of the property.
generalPreferences_var.offlineCachePeriod = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version August 2014
