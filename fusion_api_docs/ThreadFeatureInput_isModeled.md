# ThreadFeatureInput.isModeled Property

Parent Object: [ThreadFeatureInput](ThreadFeatureInput.md)  

## Description

Gets and sets if the thread is physical or cosmetic thread. A value of true indicates a physical thread. It defaults to false.

## Syntax

"threadFeatureInput_var" is a variable referencing a ThreadFeatureInput object.  

```python
# Get the value of the property.
propertyValue = threadFeatureInput_var.isModeled

# Set the value of the property.
threadFeatureInput_var.isModeled = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2015
