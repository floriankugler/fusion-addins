# BaseFeature.finishEdit Method

Parent Object: [BaseFeature](BaseFeature.md)  

## Description

Exits from edit mode in the user-interface. If this base feature in not in edit mode, then nothing happens.

## Syntax

"baseFeature_var" is a variable referencing a [BaseFeature](BaseFeature.md) object.

```python
returnValue = baseFeature_var.finishEdit()
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |
