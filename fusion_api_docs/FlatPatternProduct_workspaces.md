# FlatPatternProduct.workspaces Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Returns the workspaces associated with this product.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.workspaces
```

## Property Value

This is a read only property whose value is a [WorkspaceList](WorkspaceList.md).

## Version

Introduced in version October 2022
