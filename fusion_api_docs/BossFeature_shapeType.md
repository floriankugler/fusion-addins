# BossFeature.shapeType Property

Parent Object: [BossFeature](BossFeature.md)  

## Description

Returns the current boss shape this feature represents.

## Syntax

"bossFeature_var" is a variable referencing a BossFeature object.  

```python
# Get the value of the property.
propertyValue = bossFeature_var.shapeType
```

## Property Value

This is a read only property whose value is a [BossShapeTypes](BossShapeTypes.md).

## Version

Introduced in version October 2022
