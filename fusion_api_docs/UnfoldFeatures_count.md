# UnfoldFeatures.count Property

Parent Object: [UnfoldFeatures](UnfoldFeatures.md)  

## Description

The number of unfold features in the collection.

## Syntax

"unfoldFeatures_var" is a variable referencing a UnfoldFeatures object.  

```python
# Get the value of the property.
propertyValue = unfoldFeatures_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version August 2020
