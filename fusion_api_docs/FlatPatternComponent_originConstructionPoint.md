# FlatPatternComponent.originConstructionPoint Property

Parent Object: [FlatPatternComponent](FlatPatternComponent.md)  

## Description

Returns the origin construction point.

## Syntax

"flatPatternComponent_var" is a variable referencing a FlatPatternComponent object.  

```python
# Get the value of the property.
propertyValue = flatPatternComponent_var.originConstructionPoint
```

## Property Value

This is a read only property whose value is a [ConstructionPoint](ConstructionPoint.md).

## Version

Introduced in version October 2022
