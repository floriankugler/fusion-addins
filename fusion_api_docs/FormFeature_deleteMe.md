# FormFeature.deleteMe Method

Parent Object: [FormFeature](FormFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"formFeature_var" is a variable referencing a [FormFeature](FormFeature.md) object.

```python
returnValue = formFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version September 2015
