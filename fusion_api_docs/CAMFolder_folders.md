# CAMFolder.folders Property

Parent Object: [CAMFolder](CAMFolder.md)  

## Description

Returns the Folders collection that provides access to existing folders in this folder.

## Syntax

"cAMFolder_var" is a variable referencing a CAMFolder object.  

```python
# Get the value of the property.
propertyValue = cAMFolder_var.folders
```

## Property Value

This is a read only property whose value is a [CAMFolders](CAMFolders.md).

## Version

Introduced in version January 2016
