# CornerClosureFeature.isSuppressed Property

Parent Object: [CornerClosureFeature](CornerClosureFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"cornerClosureFeature_var" is a variable referencing a CornerClosureFeature object.  

```python
# Get the value of the property.
propertyValue = cornerClosureFeature_var.isSuppressed

# Set the value of the property.
cornerClosureFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2026
