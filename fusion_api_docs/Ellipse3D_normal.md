# Ellipse3D.normal Property

Parent Object: [Ellipse3D](Ellipse3D.md)  

## Description

Gets the normal of the ellipse.

## Syntax

"ellipse3D_var" is a variable referencing an Ellipse3D object.  

```python
# Get the value of the property.
propertyValue = ellipse3D_var.normal
```

## Property Value

This is a read only property whose value is a [Vector3D](Vector3D.md).

## Version

Introduced in version August 2014
