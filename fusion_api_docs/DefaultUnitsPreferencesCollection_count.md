# DefaultUnitsPreferencesCollection.count Property

Parent Object: [DefaultUnitsPreferencesCollection](DefaultUnitsPreferencesCollection.md)  

## Description

Returns the number of DefaultUnitsPreference objects.

## Syntax

"defaultUnitsPreferencesCollection_var" is a variable referencing a DefaultUnitsPreferencesCollection object.  

```python
# Get the value of the property.
propertyValue = defaultUnitsPreferencesCollection_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version August 2014
