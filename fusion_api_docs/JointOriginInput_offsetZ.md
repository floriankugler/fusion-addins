# JointOriginInput.offsetZ Property

Parent Object: [JointOriginInput](JointOriginInput.md)  

## Description

Gets and sets the value that defines the Z offset direction. This defaults to zero if it's not specified. The value defines a distance and if the ValueInput is defined using the createByReal method the value is assumed to be centimeters.

## Syntax

"jointOriginInput_var" is a variable referencing a JointOriginInput object.  

```python
# Get the value of the property.
propertyValue = jointOriginInput_var.offsetZ

# Set the value of the property.
jointOriginInput_var.offsetZ = propertyValue
```

## Property Value

This is a read/write property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version September 2015
