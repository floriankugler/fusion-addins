# Sketch.arePointsShown Property

Parent Object: [Sketch](Sketch.md)  

## Description

Indicates if the sketch points in the sketch are displayed. Points that are not connected to any other geometry will continue to be shown.

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.arePointsShown

# Set the value of the property.
sketch_var.arePointsShown = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version April 2019
