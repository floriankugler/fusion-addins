# Line3D.copy Method

Parent Object: [Line3D](Line3D.md)  

## Description

Creates and returns a copy of this line object.

## Syntax

"line3D_var" is a variable referencing a [Line3D](Line3D.md) object.

```python
returnValue = line3D_var.copy()
```

## Return Value

| Type                 | Description                                      |
|----------------------|--------------------------------------------------|
| [Line3D](Line3D.md) | Returns an independent copy of this line object. |

## Version

Introduced in version August 2014
