# CAMExportFuture.progress Property

Parent Object: [CAMExportFuture](CAMExportFuture.md)  

## Description

Returns the progress as a percentage value between 0.0% and 100.0%.

## Syntax

"cAMExportFuture_var" is a variable referencing a CAMExportFuture object.  

```python
# Get the value of the property.
propertyValue = cAMExportFuture_var.progress
```

## Property Value

This is a read only property whose value is a float.

## Version

Introduced in version September 2024
