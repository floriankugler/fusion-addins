# RuleFilletFeature.parentComponent Property

Parent Object: [RuleFilletFeature](RuleFilletFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"ruleFilletFeature_var" is a variable referencing a RuleFilletFeature object.  

```python
# Get the value of the property.
propertyValue = ruleFilletFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version September 2015
