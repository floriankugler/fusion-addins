# FusionDocument.documentReferences Property

Parent Object: [FusionDocument](FusionDocument.md)  

## Description

Returns a collection containing the documents directly referenced by this document.

## Syntax

"fusionDocument_var" is a variable referencing a FusionDocument object.  

```python
# Get the value of the property.
propertyValue = fusionDocument_var.documentReferences
```

## Property Value

This is a read only property whose value is a [DocumentReferences](DocumentReferences.md).

## Version

Introduced in version June 2017
