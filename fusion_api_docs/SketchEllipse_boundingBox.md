# SketchEllipse.boundingBox Property

Parent Object: [SketchEllipse](SketchEllipse.md)  

## Description

Returns the bounding box of the entity in sketch space.

## Syntax

"sketchEllipse_var" is a variable referencing a SketchEllipse object.  

```python
# Get the value of the property.
propertyValue = sketchEllipse_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version August 2014
