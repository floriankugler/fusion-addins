# ProgressDialog.cancelButtonText Property

Parent Object: [ProgressDialog](ProgressDialog.md)  

## Description

Sets the text label on the Cancel button. The default text label is "Cancel".

## Syntax

"progressDialog_var" is a variable referencing a ProgressDialog object.  

```python
# Get the value of the property.
propertyValue = progressDialog_var.cancelButtonText

# Set the value of the property.
progressDialog_var.cancelButtonText = propertyValue
```

## Property Value

This is a read/write property whose value is a string.
