# BRepBody.deleteMe Method

Parent Object: [BRepBody](BRepBody.md)  

## Description

Deletes the body.  

This property is only valid if the IsTransient property is false.

## Syntax

"bRepBody_var" is a variable referencing a [BRepBody](BRepBody.md) object.

```python
returnValue = bRepBody_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
