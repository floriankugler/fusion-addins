# PipeFeature.healthState Property

Parent Object: [PipeFeature](PipeFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"pipeFeature_var" is a variable referencing a PipeFeature object.  

```python
# Get the value of the property.
propertyValue = pipeFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
