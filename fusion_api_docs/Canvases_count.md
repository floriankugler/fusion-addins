# Canvases.count Property

Parent Object: [Canvases](Canvases.md)  

## Description

Returns the number of canvases in the component.

## Syntax

"canvases_var" is a variable referencing a Canvases object.  

```python
# Get the value of the property.
propertyValue = canvases_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version May 2023
