# Features.unfoldFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the existing unfold features.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.unfoldFeatures
```

## Property Value

This is a read only property whose value is a [UnfoldFeatures](UnfoldFeatures.md).

## Version

Introduced in version August 2020
