# ConfigurationMaterialColumn.index Property

Parent Object: [ConfigurationMaterialColumn](ConfigurationMaterialColumn.md)  

## Description

The index position of this column within the table. The first column is at index 0 and does not include the "Name" column.

## Syntax

"configurationMaterialColumn_var" is a variable referencing a ConfigurationMaterialColumn object.  

```python
# Get the value of the property.
propertyValue = configurationMaterialColumn_var.index
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version January 2024
