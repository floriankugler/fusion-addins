# EllipticalArc3D.curveType Property

Parent Object: [EllipticalArc3D](EllipticalArc3D.md)  

## Description

Returns the type of geometry this curve represents.

## Syntax

"ellipticalArc3D_var" is a variable referencing an EllipticalArc3D object.  

```python
# Get the value of the property.
propertyValue = ellipticalArc3D_var.curveType
```

## Property Value

This is a read only property whose value is a [Curve3DTypes](Curve3DTypes.md).

## Version

Introduced in version August 2014
