# ReverseNormalFeatures.count Property

Parent Object: [ReverseNormalFeatures](ReverseNormalFeatures.md)  

## Description

The number of Reverse Normal features in the collection.

## Syntax

"reverseNormalFeatures_var" is a variable referencing a ReverseNormalFeatures object.  

```python
# Get the value of the property.
propertyValue = reverseNormalFeatures_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version September 2015
