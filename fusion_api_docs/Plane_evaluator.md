# Plane.evaluator Property

Parent Object: [Plane](Plane.md)  

## Description

Returns the surface evaluator.

## Syntax

"plane_var" is a variable referencing a Plane object.  

```python
# Get the value of the property.
propertyValue = plane_var.evaluator
```

## Property Value

This is a read only property whose value is a [SurfaceEvaluator](SurfaceEvaluator.md).

## Version

Introduced in version August 2014
