# BRepShell.volume Property

Parent Object: [BRepShell](BRepShell.md)  

## Description

Returns the volume in cm ^ 3. Returns 0 in the case the shell is not solid.

## Syntax

"bRepShell_var" is a variable referencing a BRepShell object.  

```python
# Get the value of the property.
propertyValue = bRepShell_var.volume
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version August 2014
