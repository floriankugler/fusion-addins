# PathPatternFeature.patternElements Property

Parent Object: [PathPatternFeature](PathPatternFeature.md)  

## Description

Gets the PatternElements collection that contains the elements created by this pattern.

## Syntax

"pathPatternFeature_var" is a variable referencing a PathPatternFeature object.  

```python
# Get the value of the property.
propertyValue = pathPatternFeature_var.patternElements
```

## Property Value

This is a read only property whose value is a [PatternElements](PatternElements.md).

## Version

Introduced in version November 2014
