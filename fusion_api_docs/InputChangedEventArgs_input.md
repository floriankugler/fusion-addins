# InputChangedEventArgs.input Property

Parent Object: [InputChangedEventArgs](InputChangedEventArgs.md)  

## Description

Returns the command input that has just changed.

## Syntax

"inputChangedEventArgs_var" is a variable referencing an InputChangedEventArgs object.  

```python
# Get the value of the property.
propertyValue = inputChangedEventArgs_var.input
```

## Property Value

This is a read only property whose value is a [CommandInput](CommandInput.md).

## Version

Introduced in version August 2014
