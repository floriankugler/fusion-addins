# ShellFeature.attributes Property

Parent Object: [ShellFeature](ShellFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"shellFeature_var" is a variable referencing a ShellFeature object.  

```python
# Get the value of the property.
propertyValue = shellFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
