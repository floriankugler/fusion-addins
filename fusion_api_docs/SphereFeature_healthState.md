# SphereFeature.healthState Property

Parent Object: [SphereFeature](SphereFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"sphereFeature_var" is a variable referencing a SphereFeature object.  

```python
# Get the value of the property.
propertyValue = sphereFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
