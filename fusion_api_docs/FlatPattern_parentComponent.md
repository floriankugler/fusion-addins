# FlatPattern.parentComponent Property

Parent Object: [FlatPattern](FlatPattern.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"flatPattern_var" is a variable referencing a FlatPattern object.  

```python
# Get the value of the property.
propertyValue = flatPattern_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version October 2022
