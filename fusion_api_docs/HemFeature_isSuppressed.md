# HemFeature.isSuppressed Property

Parent Object: [HemFeature](HemFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"hemFeature_var" is a variable referencing a HemFeature object.  

```python
# Get the value of the property.
propertyValue = hemFeature_var.isSuppressed

# Set the value of the property.
hemFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version September 2025
