# NCProgram.isSelected Property

Parent Object: [NCProgram](NCProgram.md)  

## Description

Gets if this operation is selected in the 'Setups' browser.

## Syntax

"nCProgram_var" is a variable referencing a NCProgram object.  

```python
# Get the value of the property.
propertyValue = nCProgram_var.isSelected
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version April 2023
