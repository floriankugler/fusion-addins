# JointOrigins.count Property

Parent Object: [JointOrigins](JointOrigins.md)  

## Description

Returns number of joint origins in the collection.

## Syntax

"jointOrigins_var" is a variable referencing a JointOrigins object.  

```python
# Get the value of the property.
propertyValue = jointOrigins_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version September 2015
