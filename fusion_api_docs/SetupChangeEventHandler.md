# SetupChangeEventHandler Object

Derived from: [EventHandler](EventHandler.md) Object  

## Description

The SetupChangeEventHandler is a client implemented class that can be added as a handler to a SetupChangeEvent.

## Methods

| Name | Description |
|----|----|
| [notify](SetupChangeEventHandler_notify.md) | The function called by CAM when the associated event is fired. |

## Version

Introduced in version April 2023
