# PrintSettingQuery.name Property

Parent Object: [PrintSettingQuery](PrintSettingQuery.md)  

## Description

The case-insensitive name specifies the name of the PrintSetting.

## Syntax

"printSettingQuery_var" is a variable referencing a PrintSettingQuery object.  

```python
# Get the value of the property.
propertyValue = printSettingQuery_var.name

# Set the value of the property.
printSettingQuery_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version April 2023
