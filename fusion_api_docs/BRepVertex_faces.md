# BRepVertex.faces Property

Parent Object: [BRepVertex](BRepVertex.md)  

## Description

Returns the BRepFaces that uses this vertex through BRepEdge

## Syntax

"bRepVertex_var" is a variable referencing a BRepVertex object.  

```python
# Get the value of the property.
propertyValue = bRepVertex_var.faces
```

## Property Value

This is a read only property whose value is a [BRepFaces](BRepFaces.md).

## Version

Introduced in version August 2014
