# ThickenFeature.deleteMe Method

Parent Object: [ThickenFeature](ThickenFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"thickenFeature_var" is a variable referencing a [ThickenFeature](ThickenFeature.md) object.

```python
returnValue = thickenFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version June 2015
