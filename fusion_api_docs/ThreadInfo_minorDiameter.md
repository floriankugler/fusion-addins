# ThreadInfo.minorDiameter Property

Parent Object: [ThreadInfo](ThreadInfo.md)  

## Description

Returns the value that defines the minor diameter. The units are centimeters.

## Syntax

"threadInfo_var" is a variable referencing a ThreadInfo object.  

```python
# Get the value of the property.
propertyValue = threadInfo_var.minorDiameter
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version January 2015
