# ToolbarPanel.parentUserInterface Property

Parent Object: [ToolbarPanel](ToolbarPanel.md)  

## Description

Gets the parent UserInterface object.

## Syntax

"toolbarPanel_var" is a variable referencing a ToolbarPanel object.  

```python
# Get the value of the property.
propertyValue = toolbarPanel_var.parentUserInterface
```

## Property Value

This is a read only property whose value is a [UserInterface](UserInterface.md).

## Version

Introduced in version August 2014
