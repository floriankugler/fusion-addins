# ArrangeEnvelopeDefinition.placementClearance Property

Parent Object: [ArrangeEnvelopeDefinition](ArrangeEnvelopeDefinition.md)  

## Description

Returns the parameter that controls the offset of the objects from the base plane of the arrangement (the "up" direction). You can modify the value by using the properties on the returned ModelParameter object.

## Syntax

"arrangeEnvelopeDefinition_var" is a variable referencing an ArrangeEnvelopeDefinition object.  

```python
# Get the value of the property.
propertyValue = arrangeEnvelopeDefinition_var.placementClearance
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version January 2025
