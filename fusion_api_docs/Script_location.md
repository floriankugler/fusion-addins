# Script.location Property

Parent Object: [Script](Script.md)  

## Description

Gets which standard location this script is located.

## Syntax

"script_var" is a variable referencing a Script object.  

```python
# Get the value of the property.
propertyValue = script_var.location
```

## Property Value

This is a read only property whose value is a [ScriptSourceLocations](ScriptSourceLocations.md).

## Version

Introduced in version January 2025
