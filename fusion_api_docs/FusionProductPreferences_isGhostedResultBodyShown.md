# FusionProductPreferences.isGhostedResultBodyShown Property

Parent Object: [FusionProductPreferences](FusionProductPreferences.md)  

## Description

Gets and sets the Show ghosted result body option

## Syntax

"fusionProductPreferences_var" is a variable referencing a FusionProductPreferences object.  

```python
# Get the value of the property.
propertyValue = fusionProductPreferences_var.isGhostedResultBodyShown

# Set the value of the property.
fusionProductPreferences_var.isGhostedResultBodyShown = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version June 2015
