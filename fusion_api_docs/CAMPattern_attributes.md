# CAMPattern.attributes Property

Parent Object: [CAMPattern](CAMPattern.md)  

## Description

Returns the collection of attributes associated with this object.

## Syntax

"cAMPattern_var" is a variable referencing a CAMPattern object.  

```python
# Get the value of the property.
propertyValue = cAMPattern_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version August 2021
