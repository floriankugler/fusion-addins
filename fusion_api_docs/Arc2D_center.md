# Arc2D.center Property

Parent Object: [Arc2D](Arc2D.md)  

## Description

Gets and sets the center position of the arc.

## Syntax

"arc2D_var" is a variable referencing an Arc2D object.  

```python
# Get the value of the property.
propertyValue = arc2D_var.center

# Set the value of the property.
arc2D_var.center = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point2D](Point2D.md).

## Version

Introduced in version August 2014
