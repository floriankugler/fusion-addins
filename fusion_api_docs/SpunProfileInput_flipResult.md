# SpunProfileInput.flipResult Property

Parent Object: [SpunProfileInput](SpunProfileInput.md)  

## Description

Gets and sets whether the profile will be created on the opposite side of the axis.

## Syntax

"spunProfileInput_var" is a variable referencing a SpunProfileInput object.  

```python
# Get the value of the property.
propertyValue = spunProfileInput_var.flipResult

# Set the value of the property.
spunProfileInput_var.flipResult = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version November 2024
