# MachineElement.typeId Property

Parent Object: [MachineElement](MachineElement.md)  

## Description

Identifier for this type of machine element.

## Syntax

"machineElement_var" is a variable referencing a MachineElement object.  

```python
# Get the value of the property.
propertyValue = machineElement_var.typeId
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
