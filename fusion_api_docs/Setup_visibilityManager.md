# Setup.visibilityManager Property

Parent Object: [Setup](Setup.md)  

## Description

Visibility manager for this setup.

## Syntax

"setup_var" is a variable referencing a Setup object.  

```python
# Get the value of the property.
propertyValue = setup_var.visibilityManager
```

## Property Value

This is a read only property whose value is a [SetupVisibilityManager](SetupVisibilityManager.md).

## Version

Introduced in version January 2024
