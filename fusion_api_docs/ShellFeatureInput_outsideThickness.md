# ShellFeatureInput.outsideThickness Property

Parent Object: [ShellFeatureInput](ShellFeatureInput.md)  

## Description

Gets and sets the outside thickness.

## Syntax

"shellFeatureInput_var" is a variable referencing a ShellFeatureInput object.  

```python
# Get the value of the property.
propertyValue = shellFeatureInput_var.outsideThickness

# Set the value of the property.
shellFeatureInput_var.outsideThickness = propertyValue
```

## Property Value

This is a read/write property whose value is a [ValueInput](ValueInput.md).
