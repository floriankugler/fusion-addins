# SmoothConstraint.isDeletable Property

Parent Object: [SmoothConstraint](SmoothConstraint.md)  

## Description

Indicates if this constraint is deletable.

## Syntax

"smoothConstraint_var" is a variable referencing a SmoothConstraint object.  

```python
# Get the value of the property.
propertyValue = smoothConstraint_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
