# RevolveFeatureInput.operation Property

Parent Object: [RevolveFeatureInput](RevolveFeatureInput.md)  

## Description

Gets and sets the type of operation performed by the revolve.

## Syntax

"revolveFeatureInput_var" is a variable referencing a RevolveFeatureInput object.  

```python
# Get the value of the property.
propertyValue = revolveFeatureInput_var.operation

# Set the value of the property.
revolveFeatureInput_var.operation = propertyValue
```

## Property Value

This is a read/write property whose value is a [FeatureOperations](FeatureOperations.md).

## Version

Introduced in version August 2014
