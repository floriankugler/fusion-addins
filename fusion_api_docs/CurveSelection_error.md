# CurveSelection.error Property

Parent Object: [CurveSelection](CurveSelection.md)  

## Description

Gets the last warning string encountered after the selection was applied to a parent.

## Syntax

"curveSelection_var" is a variable referencing a CurveSelection object.  

```python
# Get the value of the property.
propertyValue = curveSelection_var.error
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version July 2023
