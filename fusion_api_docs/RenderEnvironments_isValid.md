# RenderEnvironments.isValid Property

Parent Object: [RenderEnvironments](RenderEnvironments.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"renderEnvironments_var" is a variable referencing a RenderEnvironments object.  

```python
# Get the value of the property.
propertyValue = renderEnvironments_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version May 2023
