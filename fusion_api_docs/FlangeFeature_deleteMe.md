# FlangeFeature.deleteMe Method

Parent Object: [FlangeFeature](FlangeFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"flangeFeature_var" is a variable referencing a [FlangeFeature](FlangeFeature.md) object.

```python
returnValue = flangeFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version August 2020
