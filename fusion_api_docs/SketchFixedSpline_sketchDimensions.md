# SketchFixedSpline.sketchDimensions Property

Parent Object: [SketchFixedSpline](SketchFixedSpline.md)  

## Description

Returns the sketch dimensions that are attached to this curve.

## Syntax

"sketchFixedSpline_var" is a variable referencing a SketchFixedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFixedSpline_var.sketchDimensions
```

## Property Value

This is a read only property whose value is a [SketchDimensionList](SketchDimensionList.md).

## Version

Introduced in version August 2014
