# CAMPattern.strategy Property

Parent Object: [CAMPattern](CAMPattern.md)  

## Description

Gets the name of the strategy associated with this operation.

## Syntax

"cAMPattern_var" is a variable referencing a CAMPattern object.  

```python
# Get the value of the property.
propertyValue = cAMPattern_var.strategy
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version May 2022
