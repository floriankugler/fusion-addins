# OneSideToExtentDefinition.parentFeature Property

Parent Object: [OneSideToExtentDefinition](OneSideToExtentDefinition.md)  

## Description

Returns the parent feature that this definition is associated with. If this definition has been created statically and is not associated with a feature this property will return null.

## Syntax

"oneSideToExtentDefinition_var" is a variable referencing an OneSideToExtentDefinition object.  

```python
# Get the value of the property.
propertyValue = oneSideToExtentDefinition_var.parentFeature
```

## Property Value

This is a read only property whose value is a [Feature](Feature.md).

## Version

Introduced in version March 2022
