# HTMLEvent.remove Method

Parent Object: [HTMLEvent](HTMLEvent.md)  

## Description

Removes a handler from the event.

## Syntax

"hTMLEvent_var" is a variable referencing a [HTMLEvent](HTMLEvent.md) object.

```python
returnValue = hTMLEvent_var.remove(handler)
```

## Return Value

| Type    | Description                                            |
|---------|--------------------------------------------------------|
| boolean | Returns true if removal of the handler was successful. |

## Parameters

| Name | Type | Description |
|----|----|----|
| handler | [HTMLEventHandler](HTMLEventHandler.md) | The handler object to be removed from the event. |

## Version

Introduced in version March 2017
