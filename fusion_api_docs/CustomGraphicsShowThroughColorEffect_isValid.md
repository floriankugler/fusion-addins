# CustomGraphicsShowThroughColorEffect.isValid Property

Parent Object: [CustomGraphicsShowThroughColorEffect](CustomGraphicsShowThroughColorEffect.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"customGraphicsShowThroughColorEffect_var" is a variable referencing a CustomGraphicsShowThroughColorEffect object.  

```python
# Get the value of the property.
propertyValue = customGraphicsShowThroughColorEffect_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2020
