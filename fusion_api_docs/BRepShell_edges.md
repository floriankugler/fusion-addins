# BRepShell.edges Property

Parent Object: [BRepShell](BRepShell.md)  

## Description

returns the BRepEdges owned by this shell

## Syntax

"bRepShell_var" is a variable referencing a BRepShell object.  

```python
# Get the value of the property.
propertyValue = bRepShell_var.edges
```

## Property Value

This is a read only property whose value is a [BRepEdges](BRepEdges.md).

## Version

Introduced in version August 2014
