# FolderDialog.title Property

Parent Object: [FolderDialog](FolderDialog.md)  

## Description

Gets or sets the title displayed on the dialog.

## Syntax

"folderDialog_var" is a variable referencing a FolderDialog object.  

```python
# Get the value of the property.
propertyValue = folderDialog_var.title

# Set the value of the property.
folderDialog_var.title = propertyValue
```

## Property Value

This is a read/write property whose value is a string.
