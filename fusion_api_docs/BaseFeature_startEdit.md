# BaseFeature.startEdit Method

Parent Object: [BaseFeature](BaseFeature.md)  

## Description

Set the user-interface so that the base body is in edit mode.

## Syntax

"baseFeature_var" is a variable referencing a [BaseFeature](BaseFeature.md) object.

```python
returnValue = baseFeature_var.startEdit()
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |
