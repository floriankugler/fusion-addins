# Application.userInterface Property

Parent Object: [Application](Application.md)  

## Description

Provides access to functionality specific to the user interface.

## Syntax

"application_var" is a variable referencing an Application object.  

```python
# Get the value of the property.
propertyValue = application_var.userInterface
```

## Property Value

This is a read only property whose value is a [UserInterface](UserInterface.md).
