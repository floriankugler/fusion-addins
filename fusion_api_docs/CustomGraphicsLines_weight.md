# CustomGraphicsLines.weight Property

Parent Object: [CustomGraphicsLines](CustomGraphicsLines.md)  

## Description

Defines the thickness of the line in pixels.

## Syntax

"customGraphicsLines_var" is a variable referencing a CustomGraphicsLines object.  

```python
# Get the value of the property.
propertyValue = customGraphicsLines_var.weight

# Set the value of the property.
customGraphicsLines_var.weight = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version September 2017
