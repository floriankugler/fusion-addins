# Vector2D.asArray Method

Parent Object: [Vector2D](Vector2D.md)  

## Description

Returns the vector values as an array \[x, y\].

## Syntax

"vector2D_var" is a variable referencing a [Vector2D](Vector2D.md) object.

```python
returnValue = vector2D_var.asArray()
```

## Return Value

| Type       | Description                                       |
|------------|---------------------------------------------------|
| double\[\] | Returns an array of the vector's values \[x, y\]. |

## Version

Introduced in version August 2014
