# ArrangeFeature.timelineObject Property

Parent Object: [ArrangeFeature](ArrangeFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"arrangeFeature_var" is a variable referencing an ArrangeFeature object.  

```python
# Get the value of the property.
propertyValue = arrangeFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version January 2025
