# CAMAdditiveContainer.parent Property

Parent Object: [CAMAdditiveContainer](CAMAdditiveContainer.md)  

## Description

Returns the parent Setup.

## Syntax

"cAMAdditiveContainer_var" is a variable referencing a CAMAdditiveContainer object.  

```python
# Get the value of the property.
propertyValue = cAMAdditiveContainer_var.parent
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version July 2024
