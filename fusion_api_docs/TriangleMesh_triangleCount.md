# TriangleMesh.triangleCount Property

Parent Object: [TriangleMesh](TriangleMesh.md)  

## Description

Returns the number of triangles in the mesh.

## Syntax

"triangleMesh_var" is a variable referencing a TriangleMesh object.  

```python
# Get the value of the property.
propertyValue = triangleMesh_var.triangleCount
```

## Property Value

This is a read only property whose value is an integer.

## Version

Introduced in version August 2014
