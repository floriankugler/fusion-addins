# Matrix2D.setToRotation Method

Parent Object: [Matrix2D](Matrix2D.md)  

## Description

Sets this matrix to the matrix of rotation by the specified angle, through the specified origin.

## Syntax

"matrix2D_var" is a variable referencing a [Matrix2D](Matrix2D.md) object.

```python
returnValue = matrix2D_var.setToRotation(angle, origin)
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name   | Type                   | Description                       |
|--------|------------------------|-----------------------------------|
| angle  | double                 | The rotation angle in radians.    |
| origin | [Point2D](Point2D.md) | The origin point of the rotation. |

## Version

Introduced in version August 2014
