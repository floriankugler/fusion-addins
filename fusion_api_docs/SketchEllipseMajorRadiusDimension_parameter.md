# SketchEllipseMajorRadiusDimension.parameter Property

Parent Object: [SketchEllipseMajorRadiusDimension](SketchEllipseMajorRadiusDimension.md)  

## Description

Returns the associated parameter or null if there is no associated parameter.

## Syntax

"sketchEllipseMajorRadiusDimension_var" is a variable referencing a SketchEllipseMajorRadiusDimension object.  

```python
# Get the value of the property.
propertyValue = sketchEllipseMajorRadiusDimension_var.parameter
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version August 2014
