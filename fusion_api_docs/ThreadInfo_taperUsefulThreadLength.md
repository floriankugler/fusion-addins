# ThreadInfo.taperUsefulThreadLength Property

Parent Object: [ThreadInfo](ThreadInfo.md)  

## Description

Returns the useful length of threads for a tapered thread in centimeters.  

This is only valid when isTapered is true.

## Syntax

"threadInfo_var" is a variable referencing a ThreadInfo object.  

```python
# Get the value of the property.
propertyValue = threadInfo_var.taperUsefulThreadLength
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version September 2025
