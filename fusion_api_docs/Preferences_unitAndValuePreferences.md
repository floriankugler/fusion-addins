# Preferences.unitAndValuePreferences Property

Parent Object: [Preferences](Preferences.md)  

## Description

Gets the UnitAndValuePreferences object.

## Syntax

"preferences_var" is a variable referencing a Preferences object.  

```python
# Get the value of the property.
propertyValue = preferences_var.unitAndValuePreferences
```

## Property Value

This is a read only property whose value is a [UnitAndValuePreferences](UnitAndValuePreferences.md).

## Version

Introduced in version August 2014
