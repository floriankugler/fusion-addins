# ConfigurationJointSnaps.count Property

Parent Object: [ConfigurationJointSnaps](ConfigurationJointSnaps.md)  

## Description

Returns the number of snaps for the column.

## Syntax

"configurationJointSnaps_var" is a variable referencing a ConfigurationJointSnaps object.  

```python
# Get the value of the property.
propertyValue = configurationJointSnaps_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version September 2024
