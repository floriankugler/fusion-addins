# ImageCommandInput.imageFile Property

Parent Object: [ImageCommandInput](ImageCommandInput.md)  

## Description

Gets and sets the full path and file name of the image file. Supported image format is .png Images are displayed in the command dialog using their actual size.

## Syntax

"imageCommandInput_var" is a variable referencing an ImageCommandInput object.  

```python
# Get the value of the property.
propertyValue = imageCommandInput_var.imageFile

# Set the value of the property.
imageCommandInput_var.imageFile = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version July 2015
