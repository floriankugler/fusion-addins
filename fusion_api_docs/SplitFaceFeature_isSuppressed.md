# SplitFaceFeature.isSuppressed Property

Parent Object: [SplitFaceFeature](SplitFaceFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"splitFaceFeature_var" is a variable referencing a SplitFaceFeature object.  

```python
# Get the value of the property.
propertyValue = splitFaceFeature_var.isSuppressed

# Set the value of the property.
splitFaceFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version June 2015
