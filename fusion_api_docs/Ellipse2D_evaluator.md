# Ellipse2D.evaluator Property

Parent Object: [Ellipse2D](Ellipse2D.md)  

## Description

Returns an evaluator object that lets you perform additional evaluations on the curve.

## Syntax

"ellipse2D_var" is a variable referencing an Ellipse2D object.  

```python
# Get the value of the property.
propertyValue = ellipse2D_var.evaluator
```

## Property Value

This is a read only property whose value is a [CurveEvaluator2D](CurveEvaluator2D.md).

## Version

Introduced in version August 2014
