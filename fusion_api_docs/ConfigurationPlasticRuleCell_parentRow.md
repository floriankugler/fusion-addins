# ConfigurationPlasticRuleCell.parentRow Property

Parent Object: [ConfigurationPlasticRuleCell](ConfigurationPlasticRuleCell.md)  

## Description

Returns the row this cell is in.

## Syntax

"configurationPlasticRuleCell_var" is a variable referencing a ConfigurationPlasticRuleCell object.  

```python
# Get the value of the property.
propertyValue = configurationPlasticRuleCell_var.parentRow
```

## Property Value

This is a read only property whose value is a [ConfigurationRow](ConfigurationRow.md).

## Version

Introduced in version January 2024
