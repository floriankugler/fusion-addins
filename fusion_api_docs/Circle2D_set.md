# Circle2D.set Method

Parent Object: [Circle2D](Circle2D.md)  

## Description

Sets all of the data defining the circle.

## Syntax

"circle2D_var" is a variable referencing a [Circle2D](Circle2D.md) object.

```python
returnValue = circle2D_var.set(center, radius)
```

## Return Value

| Type    | Description                                         |
|---------|-----------------------------------------------------|
| boolean | Returns true if redefining the circle is successful |

## Parameters

| Name | Type | Description |
|----|----|----|
| center | [Point2D](Point2D.md) | A point that defines the center position of the circle. |
| radius | double | The radius of the circle. |

## Version

Introduced in version August 2014
