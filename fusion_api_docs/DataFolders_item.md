# DataFolders.item Method

Parent Object: [DataFolders](DataFolders.md)  

## Description

Returns the specified folder.

## Syntax

"dataFolders_var" is a variable referencing a [DataFolders](DataFolders.md) object.

```python
returnValue = dataFolders_var.item(index)
```

## Return Value

| Type | Description |
|----|----|
| [DataFolder](DataFolder.md) | Returns the item or null if an invalid index was specified. |

## Parameters

| Name | Type | Description |
|----|----|----|
| index | uinteger | The index of the folder to return. The first folder in the list has an index of 0. |

## Version

Introduced in version January 2015
