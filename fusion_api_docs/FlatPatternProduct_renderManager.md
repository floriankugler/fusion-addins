# FlatPatternProduct.renderManager Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Returns the RenderManager object associated with this design. Using the RenderManager you can access the same functionality that is available in the Render workspace.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.renderManager
```

## Property Value

This is a read only property whose value is a [RenderManager](RenderManager.md).

## Version

Introduced in version May 2023
