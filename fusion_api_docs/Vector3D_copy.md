# Vector3D.copy Method

Parent Object: [Vector3D](Vector3D.md)  

## Description

Creates a copy of this vector.

## Syntax

"vector3D_var" is a variable referencing a [Vector3D](Vector3D.md) object.

```python
returnValue = vector3D_var.copy()
```

## Return Value

| Type                     | Description                  |
|--------------------------|------------------------------|
| [Vector3D](Vector3D.md) | Returns the new vector copy. |

## Version

Introduced in version August 2014
