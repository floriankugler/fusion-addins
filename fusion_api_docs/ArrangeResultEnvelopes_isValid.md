# ArrangeResultEnvelopes.isValid Property

Parent Object: [ArrangeResultEnvelopes](ArrangeResultEnvelopes.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"arrangeResultEnvelopes_var" is a variable referencing an ArrangeResultEnvelopes object.  

```python
# Get the value of the property.
propertyValue = arrangeResultEnvelopes_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2025
