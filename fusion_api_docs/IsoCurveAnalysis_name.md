# IsoCurveAnalysis.name Property

Parent Object: [IsoCurveAnalysis](IsoCurveAnalysis.md)  

## Description

A property that gets and sets the name of the analysis. If you use a name that is not unique, Fusion will automatically append a number to the name to make it unique.

## Syntax

"isoCurveAnalysis_var" is a variable referencing an IsoCurveAnalysis object.  

```python
# Get the value of the property.
propertyValue = isoCurveAnalysis_var.name

# Set the value of the property.
isoCurveAnalysis_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version January 2023
