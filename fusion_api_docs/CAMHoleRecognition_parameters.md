# CAMHoleRecognition.parameters Property

Parent Object: [CAMHoleRecognition](CAMHoleRecognition.md)  

## Description

Gets the CAMParameters collection for this operation.

## Syntax

"cAMHoleRecognition_var" is a variable referencing a CAMHoleRecognition object.  

```python
# Get the value of the property.
propertyValue = cAMHoleRecognition_var.parameters
```

## Property Value

This is a read only property whose value is a [CAMParameters](CAMParameters.md).

## Version

Introduced in version April 2023
