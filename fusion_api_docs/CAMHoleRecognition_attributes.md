# CAMHoleRecognition.attributes Property

Parent Object: [CAMHoleRecognition](CAMHoleRecognition.md)  

## Description

Returns the collection of attributes associated with this object.

## Syntax

"cAMHoleRecognition_var" is a variable referencing a CAMHoleRecognition object.  

```python
# Get the value of the property.
propertyValue = cAMHoleRecognition_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version April 2023
