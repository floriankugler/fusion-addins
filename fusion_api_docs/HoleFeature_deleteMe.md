# HoleFeature.deleteMe Method

Parent Object: [HoleFeature](HoleFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"holeFeature_var" is a variable referencing a [HoleFeature](HoleFeature.md) object.

```python
returnValue = holeFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version August 2014
