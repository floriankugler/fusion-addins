# PinSlotJointMotion.rotationValue Property

Parent Object: [PinSlotJointMotion](PinSlotJointMotion.md)  

## Description

Gets and sets the rotation value. This is in radians. Setting this value is the equivalent of using the Drive Joints command.

## Syntax

"pinSlotJointMotion_var" is a variable referencing a PinSlotJointMotion object.  

```python
# Get the value of the property.
propertyValue = pinSlotJointMotion_var.rotationValue

# Set the value of the property.
pinSlotJointMotion_var.rotationValue = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version July 2015
