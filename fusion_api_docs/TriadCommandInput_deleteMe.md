# TriadCommandInput.deleteMe Method

Parent Object: [TriadCommandInput](TriadCommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"triadCommandInput_var" is a variable referencing a [TriadCommandInput](TriadCommandInput.md) object.

```python
returnValue = triadCommandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version May 2022
