# ToolbarControl.deleteMe Method

Parent Object: [ToolbarControl](ToolbarControl.md)  

## Description

Deletes the ToolbarControl

## Syntax

"toolbarControl_var" is a variable referencing a [ToolbarControl](ToolbarControl.md) object.

```python
returnValue = toolbarControl_var.deleteMe()
```

## Return Value

| Type    | Description                                                  |
|---------|--------------------------------------------------------------|
| boolean | Returns a boolean indicating if the deletion was successful. |
