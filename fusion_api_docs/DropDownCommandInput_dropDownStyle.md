# DropDownCommandInput.dropDownStyle Property

Parent Object: [DropDownCommandInput](DropDownCommandInput.md)  

## Description

Returns the style of drop down this was created as.

## Syntax

"dropDownCommandInput_var" is a variable referencing a DropDownCommandInput object.  

```python
# Get the value of the property.
propertyValue = dropDownCommandInput_var.dropDownStyle
```

## Property Value

This is a read only property whose value is a [DropDownStyles](DropDownStyles.md).

## Version

Introduced in version January 2015
