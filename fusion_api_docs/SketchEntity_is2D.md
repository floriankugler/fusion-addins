# SketchEntity.is2D Property

Parent Object: [SketchEntity](SketchEntity.md)  

## Description

Indicates if this curve lies entirely on the sketch x-y plane.

## Syntax

"sketchEntity_var" is a variable referencing a SketchEntity object.  

```python
# Get the value of the property.
propertyValue = sketchEntity_var.is2D
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
