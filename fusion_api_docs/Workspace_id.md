# Workspace.id Property

Parent Object: [Workspace](Workspace.md)  

## Description

Gets the unique Id of the workspace that can be used programmatically to find a specific workspace. It is not affected by the current language.

## Syntax

"workspace_var" is a variable referencing a Workspace object.  

```python
# Get the value of the property.
propertyValue = workspace_var.id
```

## Property Value

This is a read only property whose value is a string.
