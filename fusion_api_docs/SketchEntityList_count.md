# SketchEntityList.count Property

Parent Object: [SketchEntityList](SketchEntityList.md)  

## Description

Returns the number of sketch entities in the list.

## Syntax

"sketchEntityList_var" is a variable referencing a SketchEntityList object.  

```python
# Get the value of the property.
propertyValue = sketchEntityList_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version June 2015
