# StitchFeature.name Property

Parent Object: [StitchFeature](StitchFeature.md)  

## Description

Returns the name of the feature as seen in the browser (non-parametric) or in the timeline (parametric).

## Syntax

"stitchFeature_var" is a variable referencing a StitchFeature object.  

```python
# Get the value of the property.
propertyValue = stitchFeature_var.name

# Set the value of the property.
stitchFeature_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version June 2015
