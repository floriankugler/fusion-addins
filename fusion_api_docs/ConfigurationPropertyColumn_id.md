# ConfigurationPropertyColumn.id Property

Parent Object: [ConfigurationPropertyColumn](ConfigurationPropertyColumn.md)  

## Description

The id of the column. This is constant and cannot be changed by the user.

## Syntax

"configurationPropertyColumn_var" is a variable referencing a ConfigurationPropertyColumn object.  

```python
# Get the value of the property.
propertyValue = configurationPropertyColumn_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version January 2024
