# Line2D.curveType Property

Parent Object: [Line2D](Line2D.md)  

## Description

Returns the type of geometry this curve represents.

## Syntax

"line2D_var" is a variable referencing a Line2D object.  

```python
# Get the value of the property.
propertyValue = line2D_var.curveType
```

## Property Value

This is a read only property whose value is a [Curve2DTypes](Curve2DTypes.md).

## Version

Introduced in version August 2014
