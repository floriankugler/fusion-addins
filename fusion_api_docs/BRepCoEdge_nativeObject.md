# BRepCoEdge.nativeObject Property

Parent Object: [BRepCoEdge](BRepCoEdge.md)  

## Description

The NativeObject is the object outside the context of an assembly. Returns null in the case where this object is not in the context of an assembly but is already the native object.

## Syntax

"bRepCoEdge_var" is a variable referencing a BRepCoEdge object.  

```python
# Get the value of the property.
propertyValue = bRepCoEdge_var.nativeObject
```

## Property Value

This is a read only property whose value is a [BRepCoEdge](BRepCoEdge.md).

## Version

Introduced in version August 2014
