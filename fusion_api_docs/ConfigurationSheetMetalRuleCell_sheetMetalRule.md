# ConfigurationSheetMetalRuleCell.sheetMetalRule Property

Parent Object: [ConfigurationSheetMetalRuleCell](ConfigurationSheetMetalRuleCell.md)  

## Description

Gets and sets the sheet metal rule defined for this cell.

## Syntax

"configurationSheetMetalRuleCell_var" is a variable referencing a ConfigurationSheetMetalRuleCell object.  

```python
# Get the value of the property.
propertyValue = configurationSheetMetalRuleCell_var.sheetMetalRule

# Set the value of the property.
configurationSheetMetalRuleCell_var.sheetMetalRule = propertyValue
```

## Property Value

This is a read/write property whose value is a [SheetMetalRule](SheetMetalRule.md).

## Version

Introduced in version January 2024
