# CustomGraphicsBRepBody.parent Property

Parent Object: [CustomGraphicsBRepBody](CustomGraphicsBRepBody.md)  

## Description

Returns the parent Component for a top-level group or the CustomGraphicsGroup object for graphics entities and child groups.

## Syntax

"customGraphicsBRepBody_var" is a variable referencing a CustomGraphicsBRepBody object.  

```python
# Get the value of the property.
propertyValue = customGraphicsBRepBody_var.parent
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version September 2017
