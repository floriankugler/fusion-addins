# DropDownControl.name Property

Parent Object: [DropDownControl](DropDownControl.md)  

## Description

Gets or sets the Name displayed for this drop down. This isn't used when the drop-down is in a toolbar because an icon is used in that case.

## Syntax

"dropDownControl_var" is a variable referencing a DropDownControl object.  

```python
# Get the value of the property.
propertyValue = dropDownControl_var.name

# Set the value of the property.
dropDownControl_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version August 2014
