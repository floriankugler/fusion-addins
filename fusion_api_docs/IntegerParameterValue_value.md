# IntegerParameterValue.value Property

Parent Object: [IntegerParameterValue](IntegerParameterValue.md)  

## Description

Get or set the value of the parameter.

## Syntax

"integerParameterValue_var" is a variable referencing an IntegerParameterValue object.  

```python
# Get the value of the property.
propertyValue = integerParameterValue_var.value

# Set the value of the property.
integerParameterValue_var.value = propertyValue
```

## Property Value

This is a read/write property whose value is an integer.

## Version

Introduced in version September 2021
