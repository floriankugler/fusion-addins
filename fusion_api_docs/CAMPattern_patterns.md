# CAMPattern.patterns Property

Parent Object: [CAMPattern](CAMPattern.md)  

## Description

Returns the Patterns collection that provides access to existing patterns in this folder.

## Syntax

"cAMPattern_var" is a variable referencing a CAMPattern object.  

```python
# Get the value of the property.
propertyValue = cAMPattern_var.patterns
```

## Property Value

This is a read only property whose value is a [CAMPatterns](CAMPatterns.md).

## Version

Introduced in version January 2016
