# DraftFeatureInput.isTangentChain Property

Parent Object: [DraftFeatureInput](DraftFeatureInput.md)  

## Description

Gets and sets if any faces that are tangentially connected to any of the input faces will also be included in setting InputEntities. It defaults to true.

## Syntax

"draftFeatureInput_var" is a variable referencing a DraftFeatureInput object.  

```python
# Get the value of the property.
propertyValue = draftFeatureInput_var.isTangentChain

# Set the value of the property.
draftFeatureInput_var.isTangentChain = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2015
