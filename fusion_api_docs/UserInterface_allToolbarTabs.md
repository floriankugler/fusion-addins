# UserInterface.allToolbarTabs Property

Parent Object: [UserInterface](UserInterface.md)  

## Description

Gets all of the toolbar tabs. This returns all of the tabs available, regardless of which workspace or product they're associated with.

## Syntax

"userInterface_var" is a variable referencing a UserInterface object.  

```python
# Get the value of the property.
propertyValue = userInterface_var.allToolbarTabs
```

## Property Value

This is a read only property whose value is a [ToolbarTabList](ToolbarTabList.md).

## Version

Introduced in version August 2019
