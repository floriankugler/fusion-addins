# RecognizedHole.isThrough Property

Parent Object: [RecognizedHole](RecognizedHole.md)  

## Description

Returns true if this is a through hole, i.e. if the bottom is open.

## Syntax

"recognizedHole_var" is a variable referencing a RecognizedHole object.  

```python
# Get the value of the property.
propertyValue = recognizedHole_var.isThrough
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version May 2023
