# CombineFeature.name Property

Parent Object: [CombineFeature](CombineFeature.md)  

## Description

Returns the name of the feature as seen in the browser (non-parametric) or in the timeline (parametric).

## Syntax

"combineFeature_var" is a variable referencing a CombineFeature object.  

```python
# Get the value of the property.
propertyValue = combineFeature_var.name

# Set the value of the property.
combineFeature_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version November 2014
