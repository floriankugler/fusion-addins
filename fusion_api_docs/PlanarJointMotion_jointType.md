# PlanarJointMotion.jointType Property

Parent Object: [PlanarJointMotion](PlanarJointMotion.md)  

## Description

Returns an enum value indicating the type of joint this joint represents.

## Syntax

"planarJointMotion_var" is a variable referencing a PlanarJointMotion object.  

```python
# Get the value of the property.
propertyValue = planarJointMotion_var.jointType
```

## Property Value

This is a read only property whose value is a [JointTypes](JointTypes.md).

## Version

Introduced in version July 2015
