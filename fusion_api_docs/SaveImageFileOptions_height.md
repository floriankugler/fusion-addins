# SaveImageFileOptions.height Property

Parent Object: [SaveImageFileOptions](SaveImageFileOptions.md)  

## Description

Gets and set the height of the image to be created in pixels. A value of zero is valid and indicates the current height of the viewport is to be used. When the SaveImageFileOptions object is initially created, this is initialized to 0.

## Syntax

"saveImageFileOptions_var" is a variable referencing a SaveImageFileOptions object.  

```python
# Get the value of the property.
propertyValue = saveImageFileOptions_var.height

# Set the value of the property.
saveImageFileOptions_var.height = propertyValue
```

## Property Value

This is a read/write property whose value is an integer.

## Version

Introduced in version May 2022
