# LoftPointSharpEndCondition.parentLoftSection Property

Parent Object: [LoftPointSharpEndCondition](LoftPointSharpEndCondition.md)  

## Description

Returns the parent loft section.

## Syntax

"loftPointSharpEndCondition_var" is a variable referencing a LoftPointSharpEndCondition object.  

```python
# Get the value of the property.
propertyValue = loftPointSharpEndCondition_var.parentLoftSection
```

## Property Value

This is a read only property whose value is a [LoftSection](LoftSection.md).

## Version

Introduced in version August 2016
