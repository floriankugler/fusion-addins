# MotionLink.motionTwo Property

Parent Object: [MotionLink](MotionLink.md)  

## Description

Gets the second motion type.

## Syntax

"motionLink_var" is a variable referencing a MotionLink object.  

```python
# Get the value of the property.
propertyValue = motionLink_var.motionTwo
```

## Property Value

This is a read only property whose value is a [JointMotionTypes](JointMotionTypes.md).

## Version

Introduced in version November 2025
