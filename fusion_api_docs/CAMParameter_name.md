# CAMParameter.name Property

Parent Object: [CAMParameter](CAMParameter.md)  

## Description

Gets the name (internal name) of the parameter.

## Syntax

"cAMParameter_var" is a variable referencing a CAMParameter object.  

```python
# Get the value of the property.
propertyValue = cAMParameter_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version September 2020
