# TorusFeature.parentComponent Property

Parent Object: [TorusFeature](TorusFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"torusFeature_var" is a variable referencing a TorusFeature object.  

```python
# Get the value of the property.
propertyValue = torusFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version September 2015
