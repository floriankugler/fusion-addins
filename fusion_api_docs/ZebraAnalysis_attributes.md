# ZebraAnalysis.attributes Property

Parent Object: [ZebraAnalysis](ZebraAnalysis.md)  

## Description

A property that returns the collection of attributes associated with this Analysis.

## Syntax

"zebraAnalysis_var" is a variable referencing a ZebraAnalysis object.  

```python
# Get the value of the property.
propertyValue = zebraAnalysis_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version January 2023
