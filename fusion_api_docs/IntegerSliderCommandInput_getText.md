# IntegerSliderCommandInput.getText Method

Parent Object: [IntegerSliderCommandInput](IntegerSliderCommandInput.md)  

## Description

Gets the texts of the slider if text has been defined.

## Syntax

"integerSliderCommandInput_var" is a variable referencing an [IntegerSliderCommandInput](IntegerSliderCommandInput.md) object.

```python
returnValue = integerSliderCommandInput_var.getText(isLeft)
```

## Return Value

| Type   | Description                                   |
|--------|-----------------------------------------------|
| string | Returns the left or right text of the slider. |

## Parameters

| Name   | Type    | Description                              |
|--------|---------|------------------------------------------|
| isLeft | boolean | Indicates to get the left or right text. |

## Version

Introduced in version June 2015
