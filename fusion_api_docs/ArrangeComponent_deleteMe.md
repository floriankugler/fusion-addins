# ArrangeComponent.deleteMe Method

Parent Object: [ArrangeComponent](ArrangeComponent.md)  

## Description

Deletes this ArrangeComponent from the arrangement.

## Syntax

"arrangeComponent_var" is a variable referencing an [ArrangeComponent](ArrangeComponent.md) object.

```python
returnValue = arrangeComponent_var.deleteMe()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns true if the deletion was successful. |

## Version

Introduced in version January 2025
