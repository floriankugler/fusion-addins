# ConfigurationFeatureAspectColumn.id Property

Parent Object: [ConfigurationFeatureAspectColumn](ConfigurationFeatureAspectColumn.md)  

## Description

The id of the column. This is constant and cannot be changed by the user.

## Syntax

"configurationFeatureAspectColumn_var" is a variable referencing a ConfigurationFeatureAspectColumn object.  

```python
# Get the value of the property.
propertyValue = configurationFeatureAspectColumn_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version September 2024
