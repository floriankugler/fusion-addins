# SketchText.attributes Property

Parent Object: [SketchText](SketchText.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"sketchText_var" is a variable referencing a SketchText object.  

```python
# Get the value of the property.
propertyValue = sketchText_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
