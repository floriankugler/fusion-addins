# Component.flatPattern Property

Parent Object: [Component](Component.md)  

## Description

Gets the existing flat pattern or returns null in the case where a flat pattern doesn't exist in this component.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.flatPattern
```

## Property Value

This is a read only property whose value is a [FlatPattern](FlatPattern.md).

## Version

Introduced in version October 2022
