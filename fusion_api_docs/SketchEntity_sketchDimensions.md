# SketchEntity.sketchDimensions Property

Parent Object: [SketchEntity](SketchEntity.md)  

## Description

Returns the sketch dimensions that are attached to this curve.

## Syntax

"sketchEntity_var" is a variable referencing a SketchEntity object.  

```python
# Get the value of the property.
propertyValue = sketchEntity_var.sketchDimensions
```

## Property Value

This is a read only property whose value is a [SketchDimensionList](SketchDimensionList.md).

## Version

Introduced in version August 2014
