# Line3D.asNurbsCurve Property

Parent Object: [Line3D](Line3D.md)  

## Description

Returns a NURBS curve that is geometrically identical to the line.

## Syntax

"line3D_var" is a variable referencing a Line3D object.  

```python
# Get the value of the property.
propertyValue = line3D_var.asNurbsCurve
```

## Property Value

This is a read only property whose value is a [NurbsCurve3D](NurbsCurve3D.md).

## Version

Introduced in version August 2014
