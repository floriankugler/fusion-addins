# ConstructionAxis.isParametric Property

Parent Object: [ConstructionAxis](ConstructionAxis.md)  

## Description

Indicates if this construction axis is parametric or not.

## Syntax

"constructionAxis_var" is a variable referencing a ConstructionAxis object.  

```python
# Get the value of the property.
propertyValue = constructionAxis_var.isParametric
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
