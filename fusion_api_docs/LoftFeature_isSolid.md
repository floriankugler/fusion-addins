# LoftFeature.isSolid Property

Parent Object: [LoftFeature](LoftFeature.md)  

## Description

Indicates if this feature was initially created as a solid or a surface.

## Syntax

"loftFeature_var" is a variable referencing a LoftFeature object.  

```python
# Get the value of the property.
propertyValue = loftFeature_var.isSolid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2016
