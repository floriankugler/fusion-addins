# ToolbarTab.parentUserInterface Property

Parent Object: [ToolbarTab](ToolbarTab.md)  

## Description

Gets the parent UserInterface object.

## Syntax

"toolbarTab_var" is a variable referencing a ToolbarTab object.  

```python
# Get the value of the property.
propertyValue = toolbarTab_var.parentUserInterface
```

## Property Value

This is a read only property whose value is a [UserInterface](UserInterface.md).

## Version

Introduced in version August 2019
