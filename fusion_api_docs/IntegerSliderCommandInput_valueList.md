# IntegerSliderCommandInput.valueList Property

Parent Object: [IntegerSliderCommandInput](IntegerSliderCommandInput.md)  

## Description

Gets the value list of the slider. The list is empty if this command input was not added by value list.

## Syntax

"integerSliderCommandInput_var" is a variable referencing an IntegerSliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = integerSliderCommandInput_var.valueList
```

## Property Value

This is a read only property whose value is an array of type integer.

## Version

Introduced in version June 2015
