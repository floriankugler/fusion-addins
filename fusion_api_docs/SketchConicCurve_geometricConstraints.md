# SketchConicCurve.geometricConstraints Property

Parent Object: [SketchConicCurve](SketchConicCurve.md)  

## Description

Returns the sketch constraints that are attached to this curve.

## Syntax

"sketchConicCurve_var" is a variable referencing a SketchConicCurve object.  

```python
# Get the value of the property.
propertyValue = sketchConicCurve_var.geometricConstraints
```

## Property Value

This is a read only property whose value is a [GeometricConstraintList](GeometricConstraintList.md).

## Version

Introduced in version January 2015
