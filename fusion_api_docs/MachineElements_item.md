# MachineElements.item Method

Parent Object: [MachineElements](MachineElements.md)  

## Description

Get the element at a particular index in the collection.

## Syntax

"machineElements_var" is a variable referencing a [MachineElements](MachineElements.md) object.

```python
returnValue = machineElements_var.item(index)
```

## Return Value

| Type | Description |
|----|----|
| [MachineElement](MachineElement.md) | Returns the element at the given index. |

## Parameters

| Name  | Type    | Description       |
|-------|---------|-------------------|
| index | integer | Index of element. |

## Version

Introduced in version April 2023
