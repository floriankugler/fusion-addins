# SceneSettings.brightness Property

Parent Object: [SceneSettings](SceneSettings.md)  

## Description

Gets and sets the brightness or luminance of the scene. This must be a value between 0 and 100,000 and is in lux units.

## Syntax

"sceneSettings_var" is a variable referencing a SceneSettings object.  

```python
# Get the value of the property.
propertyValue = sceneSettings_var.brightness

# Set the value of the property.
sceneSettings_var.brightness = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version May 2023
