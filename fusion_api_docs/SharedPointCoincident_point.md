# SharedPointCoincident.point Property

Parent Object: [SharedPointCoincident](SharedPointCoincident.md)  

## Description

Returns the sketch point that the sketch curves are connected to.

## Syntax

"sharedPointCoincident_var" is a variable referencing a SharedPointCoincident object.  

```python
# Get the value of the property.
propertyValue = sharedPointCoincident_var.point
```

## Property Value

This is a read only property whose value is a [SketchPoint](SketchPoint.md).

## Version

Introduced in version March 2025
