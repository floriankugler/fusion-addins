# BRepVertex.attributes Property

Parent Object: [BRepVertex](BRepVertex.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"bRepVertex_var" is a variable referencing a BRepVertex object.  

```python
# Get the value of the property.
propertyValue = bRepVertex_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
