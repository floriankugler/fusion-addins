# SketchDimension.parentSketch Property

Parent Object: [SketchDimension](SketchDimension.md)  

## Description

Returns the parent sketch object.

## Syntax

"sketchDimension_var" is a variable referencing a SketchDimension object.  

```python
# Get the value of the property.
propertyValue = sketchDimension_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
