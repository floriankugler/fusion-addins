# SplitFaceFeature.isParametric Property

Parent Object: [SplitFaceFeature](SplitFaceFeature.md)  

## Description

Indicates if this feature is parametric or not.

## Syntax

"splitFaceFeature_var" is a variable referencing a SplitFaceFeature object.  

```python
# Get the value of the property.
propertyValue = splitFaceFeature_var.isParametric
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version June 2015
