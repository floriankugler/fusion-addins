# ToolLibrary.item Method

Parent Object: [ToolLibrary](ToolLibrary.md)  

## Description

Get Tool by index in ToolLibrary.

## Syntax

"toolLibrary_var" is a variable referencing a [ToolLibrary](ToolLibrary.md) object.

```python
returnValue = toolLibrary_var.item(index)
```

## Return Value

| Type             | Description                                         |
|------------------|-----------------------------------------------------|
| [Tool](Tool.md) | Returns the Tool in the ToolLibrary by given index. |

## Parameters

| Name  | Type     | Description                           |
|-------|----------|---------------------------------------|
| index | uinteger | Index of the Tool in the ToolLibrary. |

## Version

Introduced in version April 2023
