# OrientedBoundingBox3D.copy Method

Parent Object: [OrientedBoundingBox3D](OrientedBoundingBox3D.md)  

## Description

Create a copy of this oriented bounding box.

## Syntax

"orientedBoundingBox3D_var" is a variable referencing an [OrientedBoundingBox3D](OrientedBoundingBox3D.md) object.

```python
returnValue = orientedBoundingBox3D_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [OrientedBoundingBox3D](OrientedBoundingBox3D.md) | Returns the new oriented bounding box copy. |
