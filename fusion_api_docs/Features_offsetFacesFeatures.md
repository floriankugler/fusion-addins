# Features.offsetFacesFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the existing Offset Face features.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.offsetFacesFeatures
```

## Property Value

This is a read only property whose value is an [OffsetFacesFeatures](OffsetFacesFeatures.md).

## Version

Introduced in version June 2017
