# SketchEllipticalArc.attributes Property

Parent Object: [SketchEllipticalArc](SketchEllipticalArc.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"sketchEllipticalArc_var" is a variable referencing a SketchEllipticalArc object.  

```python
# Get the value of the property.
propertyValue = sketchEllipticalArc_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
