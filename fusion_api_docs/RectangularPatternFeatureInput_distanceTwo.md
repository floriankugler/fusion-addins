# RectangularPatternFeatureInput.distanceTwo Property

Parent Object: [RectangularPatternFeatureInput](RectangularPatternFeatureInput.md)  

## Description

Gets and sets the distance in the second direction.

## Syntax

"rectangularPatternFeatureInput_var" is a variable referencing a RectangularPatternFeatureInput object.  

```python
# Get the value of the property.
propertyValue = rectangularPatternFeatureInput_var.distanceTwo

# Set the value of the property.
rectangularPatternFeatureInput_var.distanceTwo = propertyValue
```

## Property Value

This is a read/write property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version November 2014
