# ExtendFeature.attributes Property

Parent Object: [ExtendFeature](ExtendFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"extendFeature_var" is a variable referencing an ExtendFeature object.  

```python
# Get the value of the property.
propertyValue = extendFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
