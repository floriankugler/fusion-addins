# SketchCircle.geometricConstraints Property

Parent Object: [SketchCircle](SketchCircle.md)  

## Description

Returns the sketch constraints that are attached to this curve.

## Syntax

"sketchCircle_var" is a variable referencing a SketchCircle object.  

```python
# Get the value of the property.
propertyValue = sketchCircle_var.geometricConstraints
```

## Property Value

This is a read only property whose value is a [GeometricConstraintList](GeometricConstraintList.md).

## Version

Introduced in version January 2015
