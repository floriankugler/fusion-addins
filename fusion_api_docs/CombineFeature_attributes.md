# CombineFeature.attributes Property

Parent Object: [CombineFeature](CombineFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"combineFeature_var" is a variable referencing a CombineFeature object.  

```python
# Get the value of the property.
propertyValue = combineFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
