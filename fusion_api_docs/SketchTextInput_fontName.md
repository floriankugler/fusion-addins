# SketchTextInput.fontName Property

Parent Object: [SketchTextInput](SketchTextInput.md)  

## Description

Gets and sets the name of the font to use.

## Syntax

"sketchTextInput_var" is a variable referencing a SketchTextInput object.  

```python
# Get the value of the property.
propertyValue = sketchTextInput_var.fontName

# Set the value of the property.
sketchTextInput_var.fontName = propertyValue
```

## Property Value

This is a read/write property whose value is a string.
