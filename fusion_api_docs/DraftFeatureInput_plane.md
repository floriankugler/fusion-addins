# DraftFeatureInput.plane Property

Parent Object: [DraftFeatureInput](DraftFeatureInput.md)  

## Description

Gets and sets the plane that defines the direction in which the draft is applied. This is also referred to as the pull direction. This can be defined using either a planar BRepFace or a ConstructionPlane.

## Syntax

"draftFeatureInput_var" is a variable referencing a DraftFeatureInput object.  

```python
# Get the value of the property.
propertyValue = draftFeatureInput_var.plane

# Set the value of the property.
draftFeatureInput_var.plane = propertyValue
```

## Property Value

This is a read/write property whose value is a [Base](Base.md).

## Version

Introduced in version January 2015
