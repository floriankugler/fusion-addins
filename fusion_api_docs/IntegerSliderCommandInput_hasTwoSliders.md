# IntegerSliderCommandInput.hasTwoSliders Property

Parent Object: [IntegerSliderCommandInput](IntegerSliderCommandInput.md)  

## Description

Gets if the command input has two sliders.

## Syntax

"integerSliderCommandInput_var" is a variable referencing an IntegerSliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = integerSliderCommandInput_var.hasTwoSliders
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version June 2015
