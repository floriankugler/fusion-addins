# Point3D.translateBy Method

Parent Object: [Point3D](Point3D.md)  

## Description

Translates the point using the provided vector.

## Syntax

"point3D_var" is a variable referencing a [Point3D](Point3D.md) object.

```python
returnValue = point3D_var.translateBy(vector)
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name   | Type                     | Description                               |
|--------|--------------------------|-------------------------------------------|
| vector | [Vector3D](Vector3D.md) | The vector to use to translate the point. |

## Version

Introduced in version August 2014
