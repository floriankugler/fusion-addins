# ScriptInput.description Property

Parent Object: [ScriptInput](ScriptInput.md)  

## Description

The description of the add-in that is displayed in the "Scripts and Add-Ins" dialog. This defaults to an empty string.

## Syntax

"scriptInput_var" is a variable referencing a ScriptInput object.  

```python
# Get the value of the property.
propertyValue = scriptInput_var.description

# Set the value of the property.
scriptInput_var.description = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version October 2023
