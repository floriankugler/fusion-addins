# DataHub.dataProjects Property

Parent Object: [DataHub](DataHub.md)  

## Description

Returns the projects within this hub.

## Syntax

"dataHub_var" is a variable referencing a DataHub object.  

```python
# Get the value of the property.
propertyValue = dataHub_var.dataProjects
```

## Property Value

This is a read only property whose value is a [DataProjects](DataProjects.md).

## Version

Introduced in version September 2016
