# SketchEllipseMinorRadiusDimension.deleteMe Method

Parent Object: [SketchEllipseMinorRadiusDimension](SketchEllipseMinorRadiusDimension.md)  

## Description

Deletes this dimension. The IsDeletable property indicates if this dimension can be deleted.

## Syntax

"sketchEllipseMinorRadiusDimension_var" is a variable referencing a [SketchEllipseMinorRadiusDimension](SketchEllipseMinorRadiusDimension.md) object.

```python
returnValue = sketchEllipseMinorRadiusDimension_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
