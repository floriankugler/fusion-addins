# SketchControlPointSpline.parentSketch Property

Parent Object: [SketchControlPointSpline](SketchControlPointSpline.md)  

## Description

Returns the parent sketch.

## Syntax

"sketchControlPointSpline_var" is a variable referencing a SketchControlPointSpline object.  

```python
# Get the value of the property.
propertyValue = sketchControlPointSpline_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version July 2022
