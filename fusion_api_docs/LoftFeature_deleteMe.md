# LoftFeature.deleteMe Method

Parent Object: [LoftFeature](LoftFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"loftFeature_var" is a variable referencing a [LoftFeature](LoftFeature.md) object.

```python
returnValue = loftFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version September 2015
