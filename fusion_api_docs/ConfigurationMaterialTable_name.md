# ConfigurationMaterialTable.name Property

Parent Object: [ConfigurationMaterialTable](ConfigurationMaterialTable.md)  

## Description

Returns the name of the table as seen in the user interface.

## Syntax

"configurationMaterialTable_var" is a variable referencing a ConfigurationMaterialTable object.  

```python
# Get the value of the property.
propertyValue = configurationMaterialTable_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version March 2024
