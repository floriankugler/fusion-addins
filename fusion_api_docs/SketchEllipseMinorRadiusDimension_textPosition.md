# SketchEllipseMinorRadiusDimension.textPosition Property

Parent Object: [SketchEllipseMinorRadiusDimension](SketchEllipseMinorRadiusDimension.md)  

## Description

Gets and sets position of the dimension text.

## Syntax

"sketchEllipseMinorRadiusDimension_var" is a variable referencing a SketchEllipseMinorRadiusDimension object.  

```python
# Get the value of the property.
propertyValue = sketchEllipseMinorRadiusDimension_var.textPosition

# Set the value of the property.
sketchEllipseMinorRadiusDimension_var.textPosition = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
