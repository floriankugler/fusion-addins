# ToolPresets.item Method

Parent Object: [ToolPresets](ToolPresets.md)  

## Description

Get Preset by index.

## Syntax

"toolPresets_var" is a variable referencing a [ToolPresets](ToolPresets.md) object.

```python
returnValue = toolPresets_var.item(index)
```

## Return Value

| Type | Description |
|----|----|
| [ToolPreset](ToolPreset.md) | Returns Preset at by given index, null otherwise |

## Parameters

| Name | Type | Description |
|----|----|----|
| index | uinteger | Index of the Preset in the owning Tool that should be returned. |

## Version

Introduced in version April 2023
