# FavoriteMaterials.isValid Property

Parent Object: [FavoriteMaterials](FavoriteMaterials.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"favoriteMaterials_var" is a variable referencing a FavoriteMaterials object.  

```python
# Get the value of the property.
propertyValue = favoriteMaterials_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
