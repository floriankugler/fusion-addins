# ToolbarControl.isVisible Property

Parent Object: [ToolbarControl](ToolbarControl.md)  

## Description

Gets or sets if this control is currently visible.

## Syntax

"toolbarControl_var" is a variable referencing a ToolbarControl object.  

```python
# Get the value of the property.
propertyValue = toolbarControl_var.isVisible

# Set the value of the property.
toolbarControl_var.isVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
