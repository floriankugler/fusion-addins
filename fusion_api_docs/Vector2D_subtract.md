# Vector2D.subtract Method

Parent Object: [Vector2D](Vector2D.md)  

## Description

Subtract a vector from this vector.

## Syntax

"vector2D_var" is a variable referencing a [Vector2D](Vector2D.md) object.

```python
returnValue = vector2D_var.subtract(vector)
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name   | Type                     | Description                              |
|--------|--------------------------|------------------------------------------|
| vector | [Vector2D](Vector2D.md) | The vector to subtract from this vector. |

## Version

Introduced in version August 2014
