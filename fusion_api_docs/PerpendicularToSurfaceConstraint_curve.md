# PerpendicularToSurfaceConstraint.curve Property

Parent Object: [PerpendicularToSurfaceConstraint](PerpendicularToSurfaceConstraint.md)  

## Description

Returns the SketchCurve object that is constrained.

## Syntax

"perpendicularToSurfaceConstraint_var" is a variable referencing a PerpendicularToSurfaceConstraint object.  

```python
# Get the value of the property.
propertyValue = perpendicularToSurfaceConstraint_var.curve
```

## Property Value

This is a read only property whose value is a [SketchCurve](SketchCurve.md).

## Version

Introduced in version September 2023
