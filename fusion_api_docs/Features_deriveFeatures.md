# Features.deriveFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the Derive features within the component and supports the creation of new Derive features.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.deriveFeatures
```

## Property Value

This is a read only property whose value is a [DeriveFeatures](DeriveFeatures.md).

## Version

Introduced in version January 2026
