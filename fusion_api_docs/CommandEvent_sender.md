# CommandEvent.sender Property

Parent Object: [CommandEvent](CommandEvent.md)  

## Description

The object that is firing the event. For example, in the case of a command input event this will return the command.

## Syntax

"commandEvent_var" is a variable referencing a CommandEvent object.  

```python
# Get the value of the property.
propertyValue = commandEvent_var.sender
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version August 2014
