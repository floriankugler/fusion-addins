# ThickenFeatureInput.thickness Property

Parent Object: [ThickenFeatureInput](ThickenFeatureInput.md)  

## Description

Gets and sets the ValueInput object that defines the thickness distance.

## Syntax

"thickenFeatureInput_var" is a variable referencing a ThickenFeatureInput object.  

```python
# Get the value of the property.
propertyValue = thickenFeatureInput_var.thickness

# Set the value of the property.
thickenFeatureInput_var.thickness = propertyValue
```

## Property Value

This is a read/write property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version June 2015
