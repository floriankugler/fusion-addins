# DraftFeature.timelineObject Property

Parent Object: [DraftFeature](DraftFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"draftFeature_var" is a variable referencing a DraftFeature object.  

```python
# Get the value of the property.
propertyValue = draftFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version January 2015
