# ProfileCurve.geometryType Property

Parent Object: [ProfileCurve](ProfileCurve.md)  

## Description

Return the geometry type that the Geometry property will return.

## Syntax

"profileCurve_var" is a variable referencing a ProfileCurve object.  

```python
# Get the value of the property.
propertyValue = profileCurve_var.geometryType
```

## Property Value

This is a read only property whose value is a [Curve3DTypes](Curve3DTypes.md).

## Version

Introduced in version August 2014
