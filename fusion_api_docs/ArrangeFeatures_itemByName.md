# ArrangeFeatures.itemByName Method

Parent Object: [ArrangeFeatures](ArrangeFeatures.md)  

## Description

Returns the specified Arrange feature using the name of the feature.

## Syntax

"arrangeFeatures_var" is a variable referencing an [ArrangeFeatures](ArrangeFeatures.md) object.

```python
returnValue = arrangeFeatures_var.itemByName(name)
```

## Return Value

| Type | Description |
|----|----|
| [ArrangeFeature](ArrangeFeature.md) | Returns the specified Arrange feature, if it exists. Otherwise it returns null. |

## Parameters

| Name | Type   | Description                                              |
|------|--------|----------------------------------------------------------|
| name | string | The name of the Arrange feature as seen in the timeline. |

## Version

Introduced in version January 2025
