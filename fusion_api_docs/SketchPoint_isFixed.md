# SketchPoint.isFixed Property

Parent Object: [SketchPoint](SketchPoint.md)  

## Description

Indicates if this geometry is "fixed".

## Syntax

"sketchPoint_var" is a variable referencing a SketchPoint object.  

```python
# Get the value of the property.
propertyValue = sketchPoint_var.isFixed

# Set the value of the property.
sketchPoint_var.isFixed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
