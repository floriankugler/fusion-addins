# SketchConicCurve.worldGeometry Property

Parent Object: [SketchConicCurve](SketchConicCurve.md)  

## Description

Returns a NurbsCurve3D object that is the equivalent of this sketch curve but is in the space of the parent component rather than in sketch space.

## Syntax

"sketchConicCurve_var" is a variable referencing a SketchConicCurve object.  

```python
# Get the value of the property.
propertyValue = sketchConicCurve_var.worldGeometry
```

## Property Value

This is a read only property whose value is a [NurbsCurve3D](NurbsCurve3D.md).

## Version

Introduced in version January 2015
