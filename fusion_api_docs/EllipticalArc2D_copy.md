# EllipticalArc2D.copy Method

Parent Object: [EllipticalArc2D](EllipticalArc2D.md)  

## Description

Creates and returns a copy of this EllipticalArc2D object.

## Syntax

"ellipticalArc2D_var" is a variable referencing an [EllipticalArc2D](EllipticalArc2D.md) object.

```python
returnValue = ellipticalArc2D_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [EllipticalArc2D](EllipticalArc2D.md) | Returns a new EllipticalArc2D object that is a copy of this Arc2D object. |

## Version

Introduced in version August 2014
