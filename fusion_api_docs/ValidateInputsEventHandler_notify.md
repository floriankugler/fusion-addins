# ValidateInputsEventHandler.notify Method

Parent Object: [ValidateInputsEventHandler](ValidateInputsEventHandler.md)  

## Description

This notify member is called when an event is triggered from any event that this handler has been added to.

## Syntax

"validateInputsEventHandler_var" is a variable referencing a [ValidateInputsEventHandler](ValidateInputsEventHandler.md) object.

```python
returnValue = validateInputsEventHandler_var.notify(eventArgs)
```

## Parameters

| Name | Type | Description |
|----|----|----|
| eventArgs | [ValidateInputsEventArgs](ValidateInputsEventArgs.md) | The arguments object with details about this event and the firing ValidateInputsEvent. |

## Version

Introduced in version August 2014
