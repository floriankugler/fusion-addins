# ChamferFeature.parentComponent Property

Parent Object: [ChamferFeature](ChamferFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"chamferFeature_var" is a variable referencing a ChamferFeature object.  

```python
# Get the value of the property.
propertyValue = chamferFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version November 2014
