# SketchAngularDimension.attributes Property

Parent Object: [SketchAngularDimension](SketchAngularDimension.md)  

## Description

Returns the collection of attributes associated with this sketch dimension.

## Syntax

"sketchAngularDimension_var" is a variable referencing a SketchAngularDimension object.  

```python
# Get the value of the property.
propertyValue = sketchAngularDimension_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
