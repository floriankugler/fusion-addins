# HttpEventArgs.firingEvent Property

Parent Object: [HttpEventArgs](HttpEventArgs.md)  

## Description

The event that the firing is in response to.

## Syntax

"httpEventArgs_var" is a variable referencing a HttpEventArgs object.  

```python
# Get the value of the property.
propertyValue = httpEventArgs_var.firingEvent
```

## Property Value

This is a read only property whose value is an [Event](Event.md).

## Version

Introduced in version January 2024
