# UnstitchFeature.timelineObject Property

Parent Object: [UnstitchFeature](UnstitchFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"unstitchFeature_var" is a variable referencing a UnstitchFeature object.  

```python
# Get the value of the property.
propertyValue = unstitchFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version July 2015
