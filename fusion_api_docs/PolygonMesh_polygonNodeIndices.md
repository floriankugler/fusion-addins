# PolygonMesh.polygonNodeIndices Property

Parent Object: [PolygonMesh](PolygonMesh.md)  

## Description

Returns the index values that index into the NodeCoordinates and NormalVectors arrays to define the coordinates of each polygon and the corresponding normal.

## Syntax

"polygonMesh_var" is a variable referencing a PolygonMesh object.  

```python
# Get the value of the property.
propertyValue = polygonMesh_var.polygonNodeIndices
```

## Property Value

This is a read only property whose value is an array of type integer.

## Version

Introduced in version August 2014
