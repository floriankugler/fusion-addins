# ConfigurationAppearanceCell.parentColumn Property

Parent Object: [ConfigurationAppearanceCell](ConfigurationAppearanceCell.md)  

## Description

Returns the column this cell is in.

## Syntax

"configurationAppearanceCell_var" is a variable referencing a ConfigurationAppearanceCell object.  

```python
# Get the value of the property.
propertyValue = configurationAppearanceCell_var.parentColumn
```

## Property Value

This is a read only property whose value is a [ConfigurationAppearanceColumn](ConfigurationAppearanceColumn.md).

## Version

Introduced in version January 2024
