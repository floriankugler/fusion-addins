# PrintSetting.getDefaultPrintSettingItem Method

Parent Object: [PrintSetting](PrintSetting.md)  

## Description

Gets the default PrintSetting item of the specified body preset. Throws exception when name not found.

## Syntax

"printSetting_var" is a variable referencing a [PrintSetting](PrintSetting.md) object.

```python
returnValue = printSetting_var.getDefaultPrintSettingItem()
```

## Return Value

| Type                                     | Description                     |
|------------------------------------------|---------------------------------|
| [PrintSettingItem](PrintSettingItem.md) | Returns the default parameters. |

## Version

Introduced in version April 2023
