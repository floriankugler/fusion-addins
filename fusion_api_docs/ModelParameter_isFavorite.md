# ModelParameter.isFavorite Property

Parent Object: [ModelParameter](ModelParameter.md)  

## Description

Gets and sets whether this parameter is included in the Favorites list in the parameters dialog

## Syntax

"modelParameter_var" is a variable referencing a ModelParameter object.  

```python
# Get the value of the property.
propertyValue = modelParameter_var.isFavorite

# Set the value of the property.
modelParameter_var.isFavorite = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
