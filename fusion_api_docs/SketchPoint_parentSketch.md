# SketchPoint.parentSketch Property

Parent Object: [SketchPoint](SketchPoint.md)  

## Description

Returns the parent sketch.

## Syntax

"sketchPoint_var" is a variable referencing a SketchPoint object.  

```python
# Get the value of the property.
propertyValue = sketchPoint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
