# MachineToolStationInput.maxToolLength Property

Parent Object: [MachineToolStationInput](MachineToolStationInput.md)  

## Description

The maximum length in cm of the tool that can be held by this tool station.

## Syntax

"machineToolStationInput_var" is a variable referencing a MachineToolStationInput object.  

```python
# Get the value of the property.
propertyValue = machineToolStationInput_var.maxToolLength

# Set the value of the property.
machineToolStationInput_var.maxToolLength = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version July 2025
