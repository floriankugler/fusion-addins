# ValueCommandInput.deleteMe Method

Parent Object: [ValueCommandInput](ValueCommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"valueCommandInput_var" is a variable referencing a [ValueCommandInput](ValueCommandInput.md) object.

```python
returnValue = valueCommandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2020
