# BRepVertex.edges Property

Parent Object: [BRepVertex](BRepVertex.md)  

## Description

Returns the BRepEdges bounded by this vertex

## Syntax

"bRepVertex_var" is a variable referencing a BRepVertex object.  

```python
# Get the value of the property.
propertyValue = bRepVertex_var.edges
```

## Property Value

This is a read only property whose value is a [BRepEdges](BRepEdges.md).

## Version

Introduced in version August 2014
