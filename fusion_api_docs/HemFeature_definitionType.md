# HemFeature.definitionType Property

Parent Object: [HemFeature](HemFeature.md)  

## Description

Gets the type of hem defined.

## Syntax

"hemFeature_var" is a variable referencing a HemFeature object.  

```python
# Get the value of the property.
propertyValue = hemFeature_var.definitionType
```

## Property Value

This is a read only property whose value is a [HemFeatureDefinitionTypes](HemFeatureDefinitionTypes.md).

## Version

Introduced in version September 2025
