# AdditiveFFFLimitsMachineElement.maximumXYSpeed Property

Parent Object: [AdditiveFFFLimitsMachineElement](AdditiveFFFLimitsMachineElement.md)  

## Description

Maximum supported speed for motion in the X or Y axes in cm/s.

## Syntax

"additiveFFFLimitsMachineElement_var" is a variable referencing an AdditiveFFFLimitsMachineElement object.  

```python
# Get the value of the property.
propertyValue = additiveFFFLimitsMachineElement_var.maximumXYSpeed

# Set the value of the property.
additiveFFFLimitsMachineElement_var.maximumXYSpeed = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version July 2023
