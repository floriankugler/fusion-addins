# TangentConstraint.deleteMe Method

Parent Object: [TangentConstraint](TangentConstraint.md)  

## Description

Deletes this constraint. The IsDeletable property can be used to determine if this constraint can be deleted.

## Syntax

"tangentConstraint_var" is a variable referencing a [TangentConstraint](TangentConstraint.md) object.

```python
returnValue = tangentConstraint_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
