# ArrangeFeature.attributes Property

Parent Object: [ArrangeFeature](ArrangeFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"arrangeFeature_var" is a variable referencing an ArrangeFeature object.  

```python
# Get the value of the property.
propertyValue = arrangeFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version January 2025
