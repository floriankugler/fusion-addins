# BoundingBox3D.copy Method

Parent Object: [BoundingBox3D](BoundingBox3D.md)  

## Description

Creates an independent copy of this bounding box.

## Syntax

"boundingBox3D_var" is a variable referencing a [BoundingBox3D](BoundingBox3D.md) object.

```python
returnValue = boundingBox3D_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [BoundingBox3D](BoundingBox3D.md) | Returns the new bounding box or null if the copy failed. |

## Version

Introduced in version August 2014
