# NavigationEvent.remove Method

Parent Object: [NavigationEvent](NavigationEvent.md)  

## Description

Removes a handler from the event.

## Syntax

"navigationEvent_var" is a variable referencing a [NavigationEvent](NavigationEvent.md) object.

```python
returnValue = navigationEvent_var.remove(handler)
```

## Return Value

| Type    | Description                                            |
|---------|--------------------------------------------------------|
| boolean | Returns true if removal of the handler was successful. |

## Parameters

| Name | Type | Description |
|----|----|----|
| handler | [NavigationEventHandler](NavigationEventHandler.md) | The handler object to be removed from the event. |

## Version

Introduced in version March 2021
