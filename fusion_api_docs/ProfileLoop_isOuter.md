# ProfileLoop.isOuter Property

Parent Object: [ProfileLoop](ProfileLoop.md)  

## Description

Indicates if this is an outer or inner loop. Profiles always have one outer loop and have an zero to many inner loops defining voids.

## Syntax

"profileLoop_var" is a variable referencing a ProfileLoop object.  

```python
# Get the value of the property.
propertyValue = profileLoop_var.isOuter
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
