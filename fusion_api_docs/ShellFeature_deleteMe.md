# ShellFeature.deleteMe Method

Parent Object: [ShellFeature](ShellFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"shellFeature_var" is a variable referencing a [ShellFeature](ShellFeature.md) object.

```python
returnValue = shellFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version November 2014
