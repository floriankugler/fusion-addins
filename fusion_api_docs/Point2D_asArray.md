# Point2D.asArray Method

Parent Object: [Point2D](Point2D.md)  

## Description

Get coordinate data of the point

## Syntax

"point2D_var" is a variable referencing a [Point2D](Point2D.md) object.

```python
returnValue = point2D_var.asArray()
```

## Return Value

| Type       | Description                                          |
|------------|------------------------------------------------------|
| double\[\] | Returns the coordinate data of the point as an array |

## Version

Introduced in version August 2014
