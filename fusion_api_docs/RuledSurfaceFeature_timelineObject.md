# RuledSurfaceFeature.timelineObject Property

Parent Object: [RuledSurfaceFeature](RuledSurfaceFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"ruledSurfaceFeature_var" is a variable referencing a RuledSurfaceFeature object.  

```python
# Get the value of the property.
propertyValue = ruledSurfaceFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version September 2020
