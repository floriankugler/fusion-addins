# Component.asBuiltJoints Property

Parent Object: [Component](Component.md)  

## Description

Returns the collection of as-built joints associated with this component.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.asBuiltJoints
```

## Property Value

This is a read only property whose value is an [AsBuiltJoints](AsBuiltJoints.md).

## Version

Introduced in version September 2015
