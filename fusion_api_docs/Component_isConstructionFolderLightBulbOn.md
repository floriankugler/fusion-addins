# Component.isConstructionFolderLightBulbOn Property

Parent Object: [Component](Component.md)  

## Description

Gets and sets if the light bulb of the construction folder as seen in the browser is on or off. This controls the visibility of the (non-origin) construction geometry (i.e. planes, points, axes).

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.isConstructionFolderLightBulbOn

# Set the value of the property.
component_var.isConstructionFolderLightBulbOn = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
