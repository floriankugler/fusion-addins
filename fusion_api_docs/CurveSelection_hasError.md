# CurveSelection.hasError Property

Parent Object: [CurveSelection](CurveSelection.md)  

## Description

Gets if errors were encountered when applying the selection to a a parent.

## Syntax

"curveSelection_var" is a variable referencing a CurveSelection object.  

```python
# Get the value of the property.
propertyValue = curveSelection_var.hasError
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2023
