# MachineAvoidGroups.createNewMachineAvoidDirectSelectionGroup Method

Parent Object: [MachineAvoidGroups](MachineAvoidGroups.md)  

## Description

Creates a new machine avoid group and adds it to the end of the collection.

## Syntax

"machineAvoidGroups_var" is a variable referencing a [MachineAvoidGroups](MachineAvoidGroups.md) object.

```python
returnValue = machineAvoidGroups_var.createNewMachineAvoidDirectSelectionGroup()
```

## Return Value

| Type | Description |
|----|----|
| [MachineAvoidDirectSelection](MachineAvoidDirectSelection.md) | Returns newly created MachineAvoidDirectSelection. |

## Version

Introduced in version September 2024
