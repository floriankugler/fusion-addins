# ParallelConstraint.isDeletable Property

Parent Object: [ParallelConstraint](ParallelConstraint.md)  

## Description

Indicates if this constraint is deletable.

## Syntax

"parallelConstraint_var" is a variable referencing a ParallelConstraint object.  

```python
# Get the value of the property.
propertyValue = parallelConstraint_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
