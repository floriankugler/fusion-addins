# PathPatternFeatures.isValid Property

Parent Object: [PathPatternFeatures](PathPatternFeatures.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"pathPatternFeatures_var" is a variable referencing a PathPatternFeatures object.  

```python
# Get the value of the property.
propertyValue = pathPatternFeatures_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version November 2014
