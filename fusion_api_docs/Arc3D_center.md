# Arc3D.center Property

Parent Object: [Arc3D](Arc3D.md)  

## Description

Gets and sets the center position of the arc.

## Syntax

"arc3D_var" is a variable referencing an Arc3D object.  

```python
# Get the value of the property.
propertyValue = arc3D_var.center

# Set the value of the property.
arc3D_var.center = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
