Parent Object: [CombineFeatureInput](CombineFeatureInput.md)  

## Description

## Syntax

"combineFeatureInput_var" is a variable referencing a CombineFeatureInput object.  

```python
# Get the value of the property.
propertyValue = combineFeatureInput_var.toolBodies

# Set the value of the property.
combineFeatureInput_var.toolBodies = propertyValue
```

## Property Value

This is a read/write property whose value is an [ObjectCollection](ObjectCollection.md).
