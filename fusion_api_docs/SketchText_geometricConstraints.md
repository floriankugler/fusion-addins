# SketchText.geometricConstraints Property

Parent Object: [SketchText](SketchText.md)  

## Description

Returns the sketch constraints that are attached to this curve.

## Syntax

"sketchText_var" is a variable referencing a SketchText object.  

```python
# Get the value of the property.
propertyValue = sketchText_var.geometricConstraints
```

## Property Value

This is a read only property whose value is a [GeometricConstraintList](GeometricConstraintList.md).

## Version

Introduced in version March 2015
