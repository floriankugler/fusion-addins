# PathPatternFeatureInput.isSymmetric Property

Parent Object: [PathPatternFeatureInput](PathPatternFeatureInput.md)  

## Description

Gets and sets if the pattern is in one direction or symmetric.

## Syntax

"pathPatternFeatureInput_var" is a variable referencing a PathPatternFeatureInput object.  

```python
# Get the value of the property.
propertyValue = pathPatternFeatureInput_var.isSymmetric

# Set the value of the property.
pathPatternFeatureInput_var.isSymmetric = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version November 2014
