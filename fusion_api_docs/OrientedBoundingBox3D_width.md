# OrientedBoundingBox3D.width Property

Parent Object: [OrientedBoundingBox3D](OrientedBoundingBox3D.md)  

## Description

Gets and sets the width of the oriented bounding box in centimeters.

## Syntax

"orientedBoundingBox3D_var" is a variable referencing an OrientedBoundingBox3D object.  

```python
# Get the value of the property.
propertyValue = orientedBoundingBox3D_var.width

# Set the value of the property.
orientedBoundingBox3D_var.width = propertyValue
```

## Property Value

This is a read/write property whose value is a double.
