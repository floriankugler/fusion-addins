# SketchEllipticalArc.isFullyConstrained Property

Parent Object: [SketchEllipticalArc](SketchEllipticalArc.md)  

## Description

Indicates if this sketch entity is fully constrained.

## Syntax

"sketchEllipticalArc_var" is a variable referencing a SketchEllipticalArc object.  

```python
# Get the value of the property.
propertyValue = sketchEllipticalArc_var.isFullyConstrained
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2016
