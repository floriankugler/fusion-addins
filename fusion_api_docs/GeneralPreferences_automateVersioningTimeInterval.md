# GeneralPreferences.automateVersioningTimeInterval Property

Parent Object: [GeneralPreferences](GeneralPreferences.md)  

## Description

Gets and sets the interval, in minutes, for automatic versioning.

## Syntax

"generalPreferences_var" is a variable referencing a GeneralPreferences object.  

```python
# Get the value of the property.
propertyValue = generalPreferences_var.automateVersioningTimeInterval

# Set the value of the property.
generalPreferences_var.automateVersioningTimeInterval = propertyValue
```

## Property Value

This is a read/write property whose value is an integer.

## Version

Introduced in version August 2014
