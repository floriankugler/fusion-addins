# BRepCoEdge.body Property

Parent Object: [BRepCoEdge](BRepCoEdge.md)  

## Description

Returns the body this co-edge is part of.

## Syntax

"bRepCoEdge_var" is a variable referencing a BRepCoEdge object.  

```python
# Get the value of the property.
propertyValue = bRepCoEdge_var.body
```

## Property Value

This is a read only property whose value is a [BRepBody](BRepBody.md).

## Version

Introduced in version August 2014
