# SketchConicCurve.rhoValue Property

Parent Object: [SketchConicCurve](SketchConicCurve.md)  

## Description

Get and sets the rho value for the curve. The value must be greater than zero and less than one.

## Syntax

"sketchConicCurve_var" is a variable referencing a SketchConicCurve object.  

```python
# Get the value of the property.
propertyValue = sketchConicCurve_var.rhoValue

# Set the value of the property.
sketchConicCurve_var.rhoValue = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version May 2022
