# SketchFittedSpline.length Property

Parent Object: [SketchFittedSpline](SketchFittedSpline.md)  

## Description

Returns the length of the curve in centimeters.

## Syntax

"sketchFittedSpline_var" is a variable referencing a SketchFittedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFittedSpline_var.length
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version August 2014
