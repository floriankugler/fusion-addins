# Arrange3DEnvelopeDefinition.isPartialArrangeAllowed Property

Parent Object: [Arrange3DEnvelopeDefinition](Arrange3DEnvelopeDefinition.md)  

## Description

Gets and sets if a partial arrange is allowed for this envelope.

## Syntax

"arrange3DEnvelopeDefinition_var" is a variable referencing an Arrange3DEnvelopeDefinition object.  

```python
# Get the value of the property.
propertyValue = arrange3DEnvelopeDefinition_var.isPartialArrangeAllowed

# Set the value of the property.
arrange3DEnvelopeDefinition_var.isPartialArrangeAllowed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2025
