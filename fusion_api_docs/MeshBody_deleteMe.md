# MeshBody.deleteMe Method

Parent Object: [MeshBody](MeshBody.md)  

## Description

Deletes the mesh body.

## Syntax

"meshBody_var" is a variable referencing a [MeshBody](MeshBody.md) object.

```python
returnValue = meshBody_var.deleteMe()
```

## Return Value

| Type    | Description                                                  |
|---------|--------------------------------------------------------------|
| boolean | Returns true in the case where the selection was successful. |

## Version

Introduced in version August 2014
