# CustomGraphicsCurve.deleteMe Method

Parent Object: [CustomGraphicsCurve](CustomGraphicsCurve.md)  

## Description

Deletes the entity from the custom graphics group.

## Syntax

"customGraphicsCurve_var" is a variable referencing a [CustomGraphicsCurve](CustomGraphicsCurve.md) object.

```python
returnValue = customGraphicsCurve_var.deleteMe()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns true if the deletion was successful. |

## Version

Introduced in version September 2017
