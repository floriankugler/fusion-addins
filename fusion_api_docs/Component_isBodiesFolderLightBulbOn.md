# Component.isBodiesFolderLightBulbOn Property

Parent Object: [Component](Component.md)  

## Description

Gets and sets if the light bulb of the bodies folder as seen in the browser is on or off. This controls the visibility of the solid/surface bodies and the mesh bodies in this component.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.isBodiesFolderLightBulbOn

# Set the value of the property.
component_var.isBodiesFolderLightBulbOn = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version May 2016
