# FilletFeature.healthState Property

Parent Object: [FilletFeature](FilletFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"filletFeature_var" is a variable referencing a FilletFeature object.  

```python
# Get the value of the property.
propertyValue = filletFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
