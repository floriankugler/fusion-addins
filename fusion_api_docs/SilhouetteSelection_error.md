# SilhouetteSelection.error Property

Parent Object: [SilhouetteSelection](SilhouetteSelection.md)  

## Description

Gets the last warning string encountered after the selection was applied to a parent.

## Syntax

"silhouetteSelection_var" is a variable referencing a SilhouetteSelection object.  

```python
# Get the value of the property.
propertyValue = silhouetteSelection_var.error
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version July 2023
