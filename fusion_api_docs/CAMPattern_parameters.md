# CAMPattern.parameters Property

Parent Object: [CAMPattern](CAMPattern.md)  

## Description

Gets the CAMParameters collection for this operation.

## Syntax

"cAMPattern_var" is a variable referencing a CAMPattern object.  

```python
# Get the value of the property.
propertyValue = cAMPattern_var.parameters
```

## Property Value

This is a read only property whose value is a [CAMParameters](CAMParameters.md).

## Version

Introduced in version September 2020
