# GeneralPreferences.isAutomaticVersioningEnabled Property

Parent Object: [GeneralPreferences](GeneralPreferences.md)  

## Description

Gets and sets if a version of the file is automatically saved using a background thread.

## Syntax

"generalPreferences_var" is a variable referencing a GeneralPreferences object.  

```python
# Get the value of the property.
propertyValue = generalPreferences_var.isAutomaticVersioningEnabled

# Set the value of the property.
generalPreferences_var.isAutomaticVersioningEnabled = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
