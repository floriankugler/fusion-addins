# ConstructionAxisEdgeDefinition.parentConstructionAxis Property

Parent Object: [ConstructionAxisEdgeDefinition](ConstructionAxisEdgeDefinition.md)  

## Description

Returns the ConstructionAxis object

## Syntax

"constructionAxisEdgeDefinition_var" is a variable referencing a ConstructionAxisEdgeDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionAxisEdgeDefinition_var.parentConstructionAxis
```

## Property Value

This is a read only property whose value is a [ConstructionAxis](ConstructionAxis.md).

## Version

Introduced in version August 2014
