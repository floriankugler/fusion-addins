# MirrorFeature.deleteMe Method

Parent Object: [MirrorFeature](MirrorFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"mirrorFeature_var" is a variable referencing a [MirrorFeature](MirrorFeature.md) object.

```python
returnValue = mirrorFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version November 2014
