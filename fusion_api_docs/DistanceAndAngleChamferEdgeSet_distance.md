# DistanceAndAngleChamferEdgeSet.distance Property

Parent Object: [DistanceAndAngleChamferEdgeSet](DistanceAndAngleChamferEdgeSet.md)  

## Description

Returns the model parameter that controls the offset distance of the chamfer. You can edit the distance by using the properties on the returned ModelParameter object.

## Syntax

"distanceAndAngleChamferEdgeSet_var" is a variable referencing a DistanceAndAngleChamferEdgeSet object.  

```python
# Get the value of the property.
propertyValue = distanceAndAngleChamferEdgeSet_var.distance
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version December 2020
