# CAMAdditiveContainer.isOptional Property

Parent Object: [CAMAdditiveContainer](CAMAdditiveContainer.md)  

## Description

Gets and sets the "Optional" property value of the operation. Gets/sets true if the operation is optional.

## Syntax

"cAMAdditiveContainer_var" is a variable referencing a CAMAdditiveContainer object.  

```python
# Get the value of the property.
propertyValue = cAMAdditiveContainer_var.isOptional

# Set the value of the property.
cAMAdditiveContainer_var.isOptional = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version July 2024
