# CollinearConstraint.deleteMe Method

Parent Object: [CollinearConstraint](CollinearConstraint.md)  

## Description

Deletes this constraint. The IsDeletable property can be used to determine if this constraint can be deleted.

## Syntax

"collinearConstraint_var" is a variable referencing a [CollinearConstraint](CollinearConstraint.md) object.

```python
returnValue = collinearConstraint_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
