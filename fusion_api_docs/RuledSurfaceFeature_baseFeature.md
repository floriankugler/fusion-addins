# RuledSurfaceFeature.baseFeature Property

Parent Object: [RuledSurfaceFeature](RuledSurfaceFeature.md)  

## Description

If this feature is associated with a base feature, this property will return that base feature. If it's not associated with a base feature, this property will return null.

## Syntax

"ruledSurfaceFeature_var" is a variable referencing a RuledSurfaceFeature object.  

```python
# Get the value of the property.
propertyValue = ruledSurfaceFeature_var.baseFeature
```

## Property Value

This is a read only property whose value is a [BaseFeature](BaseFeature.md).

## Version

Introduced in version September 2020
