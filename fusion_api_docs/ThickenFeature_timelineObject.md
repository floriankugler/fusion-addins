# ThickenFeature.timelineObject Property

Parent Object: [ThickenFeature](ThickenFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"thickenFeature_var" is a variable referencing a ThickenFeature object.  

```python
# Get the value of the property.
propertyValue = thickenFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version June 2015
