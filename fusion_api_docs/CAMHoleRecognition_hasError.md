# CAMHoleRecognition.hasError Property

Parent Object: [CAMHoleRecognition](CAMHoleRecognition.md)  

## Description

Gets if errors were encountered when generating the operation.

## Syntax

"cAMHoleRecognition_var" is a variable referencing a CAMHoleRecognition object.  

```python
# Get the value of the property.
propertyValue = cAMHoleRecognition_var.hasError
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version April 2023
