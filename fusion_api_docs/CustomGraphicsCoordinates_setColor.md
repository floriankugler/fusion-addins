# CustomGraphicsCoordinates.setColor Method

Parent Object: [CustomGraphicsCoordinates](CustomGraphicsCoordinates.md)  

## Description

Sets the color of the coordinate at the specified index.

## Syntax

"customGraphicsCoordinates_var" is a variable referencing a [CustomGraphicsCoordinates](CustomGraphicsCoordinates.md) object.

```python
returnValue = customGraphicsCoordinates_var.setColor(index, color)
```

## Return Value

| Type    | Description                                       |
|---------|---------------------------------------------------|
| boolean | Returns true if setting the color was successful. |

## Parameters

| Name | Type | Description |
|----|----|----|
| index | integer | The index of the coordinate to set. The first coordinate has an index of 0. |
| color | [Color](Color.md) | The color value as a Color object. |

## Version

Introduced in version September 2017
