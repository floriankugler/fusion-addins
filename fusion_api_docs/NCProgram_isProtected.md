# NCProgram.isProtected Property

Parent Object: [NCProgram](NCProgram.md)  

## Description

Gets and sets the "protected" property value of the operation. Gets/sets true if the operation is protected.

## Syntax

"nCProgram_var" is a variable referencing a NCProgram object.  

```python
# Get the value of the property.
propertyValue = nCProgram_var.isProtected

# Set the value of the property.
nCProgram_var.isProtected = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version April 2023
