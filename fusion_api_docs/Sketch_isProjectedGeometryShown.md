# Sketch.isProjectedGeometryShown Property

Parent Object: [Sketch](Sketch.md)  

## Description

Gets and sets whether projected geometry in the sketch is displayed. This provides access to the "Projected Geometries" setting in the "SKETCH PALETTE".

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.isProjectedGeometryShown

# Set the value of the property.
sketch_var.isProjectedGeometryShown = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version March 2024
