# SketchControlPointSpline.attributes Property

Parent Object: [SketchControlPointSpline](SketchControlPointSpline.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"sketchControlPointSpline_var" is a variable referencing a SketchControlPointSpline object.  

```python
# Get the value of the property.
propertyValue = sketchControlPointSpline_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version July 2022
