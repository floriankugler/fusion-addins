# CAM.unitsManager Property

Parent Object: [CAM](CAM.md)  

## Description

Returns the UnitsManager object associated with this product.

## Syntax

"cAM_var" is a variable referencing a CAM object.  

```python
# Get the value of the property.
propertyValue = cAM_var.unitsManager
```

## Property Value

This is a read only property whose value is a [UnitsManager](UnitsManager.md).

## Version

Introduced in version January 2016
