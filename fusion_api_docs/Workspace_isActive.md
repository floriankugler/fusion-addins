# Workspace.isActive Property

Parent Object: [Workspace](Workspace.md)  

## Description

Gets if the workspace is currently active - i.e. displayed

## Syntax

"workspace_var" is a variable referencing a Workspace object.  

```python
# Get the value of the property.
propertyValue = workspace_var.isActive
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
