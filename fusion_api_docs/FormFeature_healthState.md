# FormFeature.healthState Property

Parent Object: [FormFeature](FormFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"formFeature_var" is a variable referencing a FormFeature object.  

```python
# Get the value of the property.
propertyValue = formFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
