# BRepLoop.coEdges Property

Parent Object: [BRepLoop](BRepLoop.md)  

## Description

Returns the BRepCoEdges consisting this loop

## Syntax

"bRepLoop_var" is a variable referencing a BRepLoop object.  

```python
# Get the value of the property.
propertyValue = bRepLoop_var.coEdges
```

## Property Value

This is a read only property whose value is a [BRepCoEdges](BRepCoEdges.md).

## Version

Introduced in version August 2014
