# ObjectVisibility.isJointsVisible Property

Parent Object: [ObjectVisibility](ObjectVisibility.md)  

## Description

Gets and sets if the glyphs for joints in all components, local or referenced by this design, are visible.

## Syntax

"objectVisibility_var" is a variable referencing an ObjectVisibility object.  

```python
# Get the value of the property.
propertyValue = objectVisibility_var.isJointsVisible

# Set the value of the property.
objectVisibility_var.isJointsVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version November 2025
