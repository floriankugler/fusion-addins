# CAMAdditiveContainer.parentSetup Property

Parent Object: [CAMAdditiveContainer](CAMAdditiveContainer.md)  

## Description

Gets the Setup this operation belongs to.

## Syntax

"cAMAdditiveContainer_var" is a variable referencing a CAMAdditiveContainer object.  

```python
# Get the value of the property.
propertyValue = cAMAdditiveContainer_var.parentSetup
```

## Property Value

This is a read only property whose value is a [Setup](Setup.md).

## Version

Introduced in version July 2024
