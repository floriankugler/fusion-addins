# HoleFeature.attributes Property

Parent Object: [HoleFeature](HoleFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"holeFeature_var" is a variable referencing a HoleFeature object.  

```python
# Get the value of the property.
propertyValue = holeFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
