# ObjectCollection.contains Method

Parent Object: [ObjectCollection](ObjectCollection.md)  

## Description

Returns whether the specified object exists within the collection.

## Syntax

"objectCollection_var" is a variable referencing an [ObjectCollection](ObjectCollection.md) object.

```python
returnValue = objectCollection_var.contains(item)
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns true if the specified item is found in the collection. |

## Parameters

| Name | Type             | Description                             |
|------|------------------|-----------------------------------------|
| item | [Base](Base.md) | The item to look for in the collection. |

## Version

Introduced in version August 2014
