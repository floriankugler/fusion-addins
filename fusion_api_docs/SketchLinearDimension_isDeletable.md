# SketchLinearDimension.isDeletable Property

Parent Object: [SketchLinearDimension](SketchLinearDimension.md)  

## Description

Indicates if this dimension is deletable.

## Syntax

"sketchLinearDimension_var" is a variable referencing a SketchLinearDimension object.  

```python
# Get the value of the property.
propertyValue = sketchLinearDimension_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
