# BossFeatureSideInput.holeCountersinkAngle Property

Parent Object: [BossFeatureSideInput](BossFeatureSideInput.md)  

## Description

Get or set countersink angle for countersink hole. This input is used only for countersink hole.

## Syntax

"bossFeatureSideInput_var" is a variable referencing a BossFeatureSideInput object.  

```python
# Get the value of the property.
propertyValue = bossFeatureSideInput_var.holeCountersinkAngle

# Set the value of the property.
bossFeatureSideInput_var.holeCountersinkAngle = propertyValue
```

## Property Value

This is a read/write property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version October 2022
