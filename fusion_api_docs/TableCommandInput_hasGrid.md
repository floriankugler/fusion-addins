# TableCommandInput.hasGrid Property

Parent Object: [TableCommandInput](TableCommandInput.md)  

## Description

Gets and sets whether a grid is displayed for the table. For a newly created table, this property defaults to false.

## Syntax

"tableCommandInput_var" is a variable referencing a TableCommandInput object.  

```python
# Get the value of the property.
propertyValue = tableCommandInput_var.hasGrid

# Set the value of the property.
tableCommandInput_var.hasGrid = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version September 2016
