# MouseEventArgs.firingEvent Property

Parent Object: [MouseEventArgs](MouseEventArgs.md)  

## Description

The event that the firing is in response to.

## Syntax

"mouseEventArgs_var" is a variable referencing a MouseEventArgs object.  

```python
# Get the value of the property.
propertyValue = mouseEventArgs_var.firingEvent
```

## Property Value

This is a read only property whose value is an [Event](Event.md).

## Version

Introduced in version August 2014
