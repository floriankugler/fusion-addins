# SetupEventArgs.firingEvent Property

Parent Object: [SetupEventArgs](SetupEventArgs.md)  

## Description

The event that the firing is in response to.

## Syntax

"setupEventArgs_var" is a variable referencing a SetupEventArgs object.  

```python
# Get the value of the property.
propertyValue = setupEventArgs_var.firingEvent
```

## Property Value

This is a read only property whose value is an [Event](Event.md).

## Version

Introduced in version April 2023
