# ChamferEdgeSet.isTangentChain Property

Parent Object: [ChamferEdgeSet](ChamferEdgeSet.md)  

## Description

Gets and sets the Tangent chain for chamfer. This enables tangent chain option for chamfer.

## Syntax

"chamferEdgeSet_var" is a variable referencing a ChamferEdgeSet object.  

```python
# Get the value of the property.
propertyValue = chamferEdgeSet_var.isTangentChain

# Set the value of the property.
chamferEdgeSet_var.isTangentChain = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version March 2021
