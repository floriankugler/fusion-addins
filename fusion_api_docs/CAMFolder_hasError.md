# CAMFolder.hasError Property

Parent Object: [CAMFolder](CAMFolder.md)  

## Description

Gets if errors were encountered when generating the operation.

## Syntax

"cAMFolder_var" is a variable referencing a CAMFolder object.  

```python
# Get the value of the property.
propertyValue = cAMFolder_var.hasError
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version April 2023
