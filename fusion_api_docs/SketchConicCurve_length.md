# SketchConicCurve.length Property

Parent Object: [SketchConicCurve](SketchConicCurve.md)  

## Description

Returns the length of the curve in centimeters.

## Syntax

"sketchConicCurve_var" is a variable referencing a SketchConicCurve object.  

```python
# Get the value of the property.
propertyValue = sketchConicCurve_var.length
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version January 2015
