# RibFeature.attributes Property

Parent Object: [RibFeature](RibFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"ribFeature_var" is a variable referencing a RibFeature object.  

```python
# Get the value of the property.
propertyValue = ribFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
