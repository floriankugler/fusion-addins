# Point2D.vectorTo Method

Parent Object: [Point2D](Point2D.md)  

## Description

Returns a vector from this point to another point.

## Syntax

"point2D_var" is a variable referencing a [Point2D](Point2D.md) object.

```python
returnValue = point2D_var.vectorTo(point)
```

## Return Value

| Type                     | Description               |
|--------------------------|---------------------------|
| [Vector2D](Vector2D.md) | Returns a Vector2D object |

## Parameters

| Name  | Type                   | Description                                 |
|-------|------------------------|---------------------------------------------|
| point | [Point2D](Point2D.md) | The other point to use to create the vector |

## Version

Introduced in version August 2014
