# OptimizedOrientationResults.count Property

Parent Object: [OptimizedOrientationResults](OptimizedOrientationResults.md)  

## Description

The number of items in the collection.

## Syntax

"optimizedOrientationResults_var" is a variable referencing an OptimizedOrientationResults object.  

```python
# Get the value of the property.
propertyValue = optimizedOrientationResults_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version July 2023
