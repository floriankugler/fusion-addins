# ApplicationFolders.appStoreInstallPath Property

Parent Object: [ApplicationFolders](ApplicationFolders.md)  

## Description

Returns the path where apps from the Autodesk App Store are installed.

## Syntax

"applicationFolders_var" is a variable referencing an ApplicationFolders object.  

```python
# Get the value of the property.
propertyValue = applicationFolders_var.appStoreInstallPath
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version September 2024
