# FlatPatternComponent.activeSheetMetalRule Property

Parent Object: [FlatPatternComponent](FlatPatternComponent.md)  

## Description

Gets and sets the active sheet metal rule. This can return null in the case where the component has never contained any sheet metal related data.

## Syntax

"flatPatternComponent_var" is a variable referencing a FlatPatternComponent object.  

```python
# Get the value of the property.
propertyValue = flatPatternComponent_var.activeSheetMetalRule

# Set the value of the property.
flatPatternComponent_var.activeSheetMetalRule = propertyValue
```

## Property Value

This is a read/write property whose value is a [SheetMetalRule](SheetMetalRule.md).

## Version

Introduced in version November 2022
