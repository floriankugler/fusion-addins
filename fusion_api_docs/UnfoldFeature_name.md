# UnfoldFeature.name Property

Parent Object: [UnfoldFeature](UnfoldFeature.md)  

## Description

Returns the name of the feature as seen in the browser (non-parametric) or in the timeline (parametric).

## Syntax

"unfoldFeature_var" is a variable referencing a UnfoldFeature object.  

```python
# Get the value of the property.
propertyValue = unfoldFeature_var.name

# Set the value of the property.
unfoldFeature_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version August 2020
