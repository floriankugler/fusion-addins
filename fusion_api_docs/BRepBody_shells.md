# BRepBody.shells Property

Parent Object: [BRepBody](BRepBody.md)  

## Description

Returns a collection of all of the shells in the body.

## Syntax

"bRepBody_var" is a variable referencing a BRepBody object.  

```python
# Get the value of the property.
propertyValue = bRepBody_var.shells
```

## Property Value

This is a read only property whose value is a [BRepShells](BRepShells.md).

## Version

Introduced in version August 2014
