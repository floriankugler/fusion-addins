# PostConfiguration.vendor Property

Parent Object: [PostConfiguration](PostConfiguration.md)  

## Description

Gets the name of the vendor of the machine tool or controller this post configuration supports.

## Syntax

"postConfiguration_var" is a variable referencing a PostConfiguration object.  

```python
# Get the value of the property.
propertyValue = postConfiguration_var.vendor
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
