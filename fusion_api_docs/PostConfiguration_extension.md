# PostConfiguration.extension Property

Parent Object: [PostConfiguration](PostConfiguration.md)  

## Description

Gets the extension of the output file created by the post.

## Syntax

"postConfiguration_var" is a variable referencing a PostConfiguration object.  

```python
# Get the value of the property.
propertyValue = postConfiguration_var.extension
```

## Property Value

This is a read only property whose value is a string.
