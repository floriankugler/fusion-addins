# SketchCurve.isFullyConstrained Property

Parent Object: [SketchCurve](SketchCurve.md)  

## Description

Indicates if this sketch entity is fully constrained.

## Syntax

"sketchCurve_var" is a variable referencing a SketchCurve object.  

```python
# Get the value of the property.
propertyValue = sketchCurve_var.isFullyConstrained
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2016
