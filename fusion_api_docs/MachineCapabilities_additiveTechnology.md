# MachineCapabilities.additiveTechnology Property

Parent Object: [MachineCapabilities](MachineCapabilities.md)  

## Description

Gets which additive technology the machine supports. Return "NA" if the machine does not support Additive

## Syntax

"machineCapabilities_var" is a variable referencing a MachineCapabilities object.  

```python
# Get the value of the property.
propertyValue = machineCapabilities_var.additiveTechnology
```

## Property Value

This is a read only property whose value is an [AdditiveTechnologies](AdditiveTechnologies.md).

## Version

Introduced in version April 2023
