# LoftFreeEndCondition.isValid Property

Parent Object: [LoftFreeEndCondition](LoftFreeEndCondition.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"loftFreeEndCondition_var" is a variable referencing a LoftFreeEndCondition object.  

```python
# Get the value of the property.
propertyValue = loftFreeEndCondition_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2016
