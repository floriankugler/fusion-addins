# AreaProperties.perimeter Property

Parent Object: [AreaProperties](AreaProperties.md)  

## Description

Gets the perimeter in centimeters. The perimeter is the sum of the length of all the curves or edges of the profile or planar surface

## Syntax

"areaProperties_var" is a variable referencing an AreaProperties object.  

```python
# Get the value of the property.
propertyValue = areaProperties_var.perimeter
```

## Property Value

This is a read only property whose value is a double.
