# SketchConicCurve.deleteMe Method

Parent Object: [SketchConicCurve](SketchConicCurve.md)  

## Description

Deletes the entity from the sketch.

## Syntax

"sketchConicCurve_var" is a variable referencing a [SketchConicCurve](SketchConicCurve.md) object.

```python
returnValue = sketchConicCurve_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true is the delete was successful. |

## Version

Introduced in version January 2015
