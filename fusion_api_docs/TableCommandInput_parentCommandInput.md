# TableCommandInput.parentCommandInput Property

Parent Object: [TableCommandInput](TableCommandInput.md)  

## Description

Gets the parent CommandInput if this commandInput is the child of a TabCommandInput or GroupCommandInput. Returns null if there is no parent.

## Syntax

"tableCommandInput_var" is a variable referencing a TableCommandInput object.  

```python
# Get the value of the property.
propertyValue = tableCommandInput_var.parentCommandInput
```

## Property Value

This is a read only property whose value is a [CommandInput](CommandInput.md).

## Version

Introduced in version September 2016
