# SketchControlPointSpline.deleteMe Method

Parent Object: [SketchControlPointSpline](SketchControlPointSpline.md)  

## Description

Deletes the entity from the sketch.

## Syntax

"sketchControlPointSpline_var" is a variable referencing a [SketchControlPointSpline](SketchControlPointSpline.md) object.

```python
returnValue = sketchControlPointSpline_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true is the delete was successful. |

## Version

Introduced in version July 2022
