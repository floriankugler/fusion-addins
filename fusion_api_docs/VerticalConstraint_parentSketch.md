# VerticalConstraint.parentSketch Property

Parent Object: [VerticalConstraint](VerticalConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"verticalConstraint_var" is a variable referencing a VerticalConstraint object.  

```python
# Get the value of the property.
propertyValue = verticalConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
