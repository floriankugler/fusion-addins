# MeshBody.attributes Property

Parent Object: [MeshBody](MeshBody.md)  

## Description

Returns the collection of attributes associated with this mesh body.

## Syntax

"meshBody_var" is a variable referencing a MeshBody object.  

```python
# Get the value of the property.
propertyValue = meshBody_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
