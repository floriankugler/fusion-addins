# DrawingDocument.products Property

Parent Object: [DrawingDocument](DrawingDocument.md)  

## Description

Returns the products associated with this document.

## Syntax

"drawingDocument_var" is a variable referencing a DrawingDocument object.  

```python
# Get the value of the property.
propertyValue = drawingDocument_var.products
```

## Property Value

This is a read only property whose value is a [Products](Products.md).

## Version

Introduced in version December 2020
