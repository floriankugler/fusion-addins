# MachiningTime.machiningTime Property

Parent Object: [MachiningTime](MachiningTime.md)  

## Description

Gets the machining time in seconds.

## Syntax

"machiningTime_var" is a variable referencing a MachiningTime object.  

```python
# Get the value of the property.
propertyValue = machiningTime_var.machiningTime
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version September 2017
