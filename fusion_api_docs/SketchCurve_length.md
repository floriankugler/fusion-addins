# SketchCurve.length Property

Parent Object: [SketchCurve](SketchCurve.md)  

## Description

Returns the length of the curve in centimeters.

## Syntax

"sketchCurve_var" is a variable referencing a SketchCurve object.  

```python
# Get the value of the property.
propertyValue = sketchCurve_var.length
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version August 2014
