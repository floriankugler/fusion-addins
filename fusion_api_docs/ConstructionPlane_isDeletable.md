# ConstructionPlane.isDeletable Property

Parent Object: [ConstructionPlane](ConstructionPlane.md)  

## Description

Indicates if this construction plane can be deleted. Base construction planes can not be deleted.

## Syntax

"constructionPlane_var" is a variable referencing a ConstructionPlane object.  

```python
# Get the value of the property.
propertyValue = constructionPlane_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
