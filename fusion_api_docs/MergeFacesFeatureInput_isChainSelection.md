# MergeFacesFeatureInput.isChainSelection Property

Parent Object: [MergeFacesFeatureInput](MergeFacesFeatureInput.md)  

## Description

Get and sets whether or not faces that are tangentially connected and from the same body (solid or surface) will be included in the faces to merge

## Syntax

"mergeFacesFeatureInput_var" is a variable referencing a MergeFacesFeatureInput object.  

```python
# Get the value of the property.
propertyValue = mergeFacesFeatureInput_var.isChainSelection

# Set the value of the property.
mergeFacesFeatureInput_var.isChainSelection = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version September 2024
