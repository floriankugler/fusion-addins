# ArrangeDefinition3DInput.isCreateCopies Property

Parent Object: [ArrangeDefinition3DInput](ArrangeDefinition3DInput.md)  

## Description

Gets and set if the original components will be moved or copied to create the arrangement. This defaults to true.

## Syntax

"arrangeDefinition3DInput_var" is a variable referencing an ArrangeDefinition3DInput object.  

```python
# Get the value of the property.
propertyValue = arrangeDefinition3DInput_var.isCreateCopies

# Set the value of the property.
arrangeDefinition3DInput_var.isCreateCopies = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2025
