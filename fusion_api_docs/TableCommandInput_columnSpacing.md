# TableCommandInput.columnSpacing Property

Parent Object: [TableCommandInput](TableCommandInput.md)  

## Description

Gets and sets the spacing between columns. This is defined in pixels. For a newly created table, this property defaults to 1.

## Syntax

"tableCommandInput_var" is a variable referencing a TableCommandInput object.  

```python
# Get the value of the property.
propertyValue = tableCommandInput_var.columnSpacing

# Set the value of the property.
tableCommandInput_var.columnSpacing = propertyValue
```

## Property Value

This is a read/write property whose value is an integer.

## Version

Introduced in version September 2016
