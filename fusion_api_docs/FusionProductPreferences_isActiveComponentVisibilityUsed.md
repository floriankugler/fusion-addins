# FusionProductPreferences.isActiveComponentVisibilityUsed Property

Parent Object: [FusionProductPreferences](FusionProductPreferences.md)  

## Description

Gets and sets the Active Component Visibility option

## Syntax

"fusionProductPreferences_var" is a variable referencing a FusionProductPreferences object.  

```python
# Get the value of the property.
propertyValue = fusionProductPreferences_var.isActiveComponentVisibilityUsed

# Set the value of the property.
fusionProductPreferences_var.isActiveComponentVisibilityUsed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
