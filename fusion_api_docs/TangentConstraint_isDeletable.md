# TangentConstraint.isDeletable Property

Parent Object: [TangentConstraint](TangentConstraint.md)  

## Description

Indicates if this constraint is deletable.

## Syntax

"tangentConstraint_var" is a variable referencing a TangentConstraint object.  

```python
# Get the value of the property.
propertyValue = tangentConstraint_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
