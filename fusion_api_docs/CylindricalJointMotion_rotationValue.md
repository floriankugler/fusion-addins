# CylindricalJointMotion.rotationValue Property

Parent Object: [CylindricalJointMotion](CylindricalJointMotion.md)  

## Description

Gets and sets the rotation value. This is in radians. Setting this value is the equivalent of using the Drive Joints command.

## Syntax

"cylindricalJointMotion_var" is a variable referencing a CylindricalJointMotion object.  

```python
# Get the value of the property.
propertyValue = cylindricalJointMotion_var.rotationValue

# Set the value of the property.
cylindricalJointMotion_var.rotationValue = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version July 2015
