# ConfigurationPlasticRuleColumn.id Property

Parent Object: [ConfigurationPlasticRuleColumn](ConfigurationPlasticRuleColumn.md)  

## Description

The id of the column. This is constant and cannot be changed by the user.

## Syntax

"configurationPlasticRuleColumn_var" is a variable referencing a ConfigurationPlasticRuleColumn object.  

```python
# Get the value of the property.
propertyValue = configurationPlasticRuleColumn_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version January 2024
