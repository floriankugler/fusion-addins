# TrimFeature.parentComponent Property

Parent Object: [TrimFeature](TrimFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"trimFeature_var" is a variable referencing a TrimFeature object.  

```python
# Get the value of the property.
propertyValue = trimFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version July 2015
