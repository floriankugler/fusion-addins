# EllipticalArc2D.endAngle Property

Parent Object: [EllipticalArc2D](EllipticalArc2D.md)  

## Description

Gets and sets the end angle of the elliptical arc in radians, where 0 is along the major axis.

## Syntax

"ellipticalArc2D_var" is a variable referencing an EllipticalArc2D object.  

```python
# Get the value of the property.
propertyValue = ellipticalArc2D_var.endAngle

# Set the value of the property.
ellipticalArc2D_var.endAngle = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version August 2014
