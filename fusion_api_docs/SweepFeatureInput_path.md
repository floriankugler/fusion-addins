# SweepFeatureInput.path Property

Parent Object: [SweepFeatureInput](SweepFeatureInput.md)  

## Description

Gets and sets the path to create the sweep.

## Syntax

"sweepFeatureInput_var" is a variable referencing a SweepFeatureInput object.  

```python
# Get the value of the property.
propertyValue = sweepFeatureInput_var.path

# Set the value of the property.
sweepFeatureInput_var.path = propertyValue
```

## Property Value

This is a read/write property whose value is a [Path](Path.md).
