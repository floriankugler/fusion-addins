# MachineQuery.url Property

Parent Object: [MachineQuery](MachineQuery.md)  

## Description

The URL specifies the location and folder to search for in the machine library. Setting the URL updates the location.

## Syntax

"machineQuery_var" is a variable referencing a MachineQuery object.  

```python
# Get the value of the property.
propertyValue = machineQuery_var.url

# Set the value of the property.
machineQuery_var.url = propertyValue
```

## Property Value

This is a read/write property whose value is a [URL](URL.md).

## Version

Introduced in version April 2023
