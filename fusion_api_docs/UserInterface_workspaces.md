# UserInterface.workspaces Property

Parent Object: [UserInterface](UserInterface.md)  

## Description

Gets all of the workspaces currently available.

## Syntax

"userInterface_var" is a variable referencing a UserInterface object.  

```python
# Get the value of the property.
propertyValue = userInterface_var.workspaces
```

## Property Value

This is a read only property whose value is a [Workspaces](Workspaces.md).
