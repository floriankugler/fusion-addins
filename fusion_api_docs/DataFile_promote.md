# DataFile.promote Method

Parent Object: [DataFile](DataFile.md)  

## Description

Promotes this version to be the latest version. If this is the latest version, nothing happens.

## Syntax

"dataFile_var" is a variable referencing a [DataFile](DataFile.md) object.

```python
returnValue = dataFile_var.promote()
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Version

Introduced in version September 2016
