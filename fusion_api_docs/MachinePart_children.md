# MachinePart.children Property

Parent Object: [MachinePart](MachinePart.md)  

## Description

Get the collection of child parts.

## Syntax

"machinePart_var" is a variable referencing a MachinePart object.  

```python
# Get the value of the property.
propertyValue = machinePart_var.children
```

## Property Value

This is a read only property whose value is a [MachineParts](MachineParts.md).

## Version

Introduced in version April 2023
