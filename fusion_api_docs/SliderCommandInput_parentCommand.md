# SliderCommandInput.parentCommand Property

Parent Object: [SliderCommandInput](SliderCommandInput.md)  

## Description

Gets the parent Command.

## Syntax

"sliderCommandInput_var" is a variable referencing a SliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = sliderCommandInput_var.parentCommand
```

## Property Value

This is a read only property whose value is a [Command](Command.md).

## Version

Introduced in version June 2015
