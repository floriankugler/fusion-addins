# NurbsCurve2D.knots Property

Parent Object: [NurbsCurve2D](NurbsCurve2D.md)  

## Description

Returns an array of numbers that define the Knots of the curve.

## Syntax

"nurbsCurve2D_var" is a variable referencing a NurbsCurve2D object.  

```python
# Get the value of the property.
propertyValue = nurbsCurve2D_var.knots
```

## Property Value

This is a read only property whose value is an array of type double.

## Version

Introduced in version August 2014
