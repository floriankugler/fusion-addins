# MoveFeatureDefinition.parentMoveFeature Property

Parent Object: [MoveFeatureDefinition](MoveFeatureDefinition.md)  

## Description

Returns the parent MoveFeature object

## Syntax

"moveFeatureDefinition_var" is a variable referencing a MoveFeatureDefinition object.  

```python
# Get the value of the property.
propertyValue = moveFeatureDefinition_var.parentMoveFeature
```

## Property Value

This is a read only property whose value is a [MoveFeature](MoveFeature.md).

## Version

Introduced in version January 2023
