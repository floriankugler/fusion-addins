# Matrix3D.transformBy Method

Parent Object: [Matrix3D](Matrix3D.md)  

## Description

Transforms this matrix using the input matrix.

## Syntax

"matrix3D_var" is a variable referencing a [Matrix3D](Matrix3D.md) object.

```python
returnValue = matrix3D_var.transformBy(matrix)
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name   | Type                     | Description                |
|--------|--------------------------|----------------------------|
| matrix | [Matrix3D](Matrix3D.md) | The transformation matrix. |

## Version

Introduced in version August 2014
