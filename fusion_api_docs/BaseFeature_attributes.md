# BaseFeature.attributes Property

Parent Object: [BaseFeature](BaseFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"baseFeature_var" is a variable referencing a BaseFeature object.  

```python
# Get the value of the property.
propertyValue = baseFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
