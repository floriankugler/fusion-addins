# SelectionSet.deleteMe Method

Parent Object: [SelectionSet](SelectionSet.md)  

## Description

Deletes this SelectionSet object.

## Syntax

"selectionSet_var" is a variable referencing a [SelectionSet](SelectionSet.md) object.

```python
returnValue = selectionSet_var.deleteMe()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns true if the deletion was successful. |

## Version

Introduced in version May 2022
