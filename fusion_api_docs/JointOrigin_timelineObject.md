# JointOrigin.timelineObject Property

Parent Object: [JointOrigin](JointOrigin.md)  

## Description

Returns the timeline object associated with this joint origin.

## Syntax

"jointOrigin_var" is a variable referencing a JointOrigin object.  

```python
# Get the value of the property.
propertyValue = jointOrigin_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version September 2015
