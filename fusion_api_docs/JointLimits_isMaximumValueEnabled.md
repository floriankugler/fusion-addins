# JointLimits.isMaximumValueEnabled Property

Parent Object: [JointLimits](JointLimits.md)  

## Description

Gets and sets whether the maximum joint limit is enabled or not.

## Syntax

"jointLimits_var" is a variable referencing a JointLimits object.  

```python
# Get the value of the property.
propertyValue = jointLimits_var.isMaximumValueEnabled

# Set the value of the property.
jointLimits_var.isMaximumValueEnabled = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.
