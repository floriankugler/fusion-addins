# Polyline2D.pointCount Property

Parent Object: [Polyline2D](Polyline2D.md)  

## Description

Returns the number of points defining the polyline.

## Syntax

"polyline2D_var" is a variable referencing a Polyline2D object.  

```python
# Get the value of the property.
propertyValue = polyline2D_var.pointCount
```

## Property Value

This is a read only property whose value is an integer.

## Version

Introduced in version September 2024
