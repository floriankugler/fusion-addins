# InfiniteLine3D.origin Property

Parent Object: [InfiniteLine3D](InfiniteLine3D.md)  

## Description

Gets and sets the origin point of the line.

## Syntax

"infiniteLine3D_var" is a variable referencing an InfiniteLine3D object.  

```python
# Get the value of the property.
propertyValue = infiniteLine3D_var.origin

# Set the value of the property.
infiniteLine3D_var.origin = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
