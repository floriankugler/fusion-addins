# SketchFixedSpline.is2D Property

Parent Object: [SketchFixedSpline](SketchFixedSpline.md)  

## Description

Indicates if this curve lies entirely on the sketch x-y plane.

## Syntax

"sketchFixedSpline_var" is a variable referencing a SketchFixedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFixedSpline_var.is2D
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
