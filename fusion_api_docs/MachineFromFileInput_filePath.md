# MachineFromFileInput.filePath Property

Parent Object: [MachineFromFileInput](MachineFromFileInput.md)  

## Description

The path to a file to act as a base for creating a machine from. The filePath is expected to be an absolute path to the local machine file.

## Syntax

"machineFromFileInput_var" is a variable referencing a MachineFromFileInput object.  

```python
# Get the value of the property.
propertyValue = machineFromFileInput_var.filePath
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
