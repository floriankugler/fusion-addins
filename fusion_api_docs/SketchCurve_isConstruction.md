# SketchCurve.isConstruction Property

Parent Object: [SketchCurve](SketchCurve.md)  

## Description

Gets and sets whether this curve is construction geometry.

## Syntax

"sketchCurve_var" is a variable referencing a SketchCurve object.  

```python
# Get the value of the property.
propertyValue = sketchCurve_var.isConstruction

# Set the value of the property.
sketchCurve_var.isConstruction = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
