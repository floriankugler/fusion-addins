# EllipticalArc2D.isCircular Property

Parent Object: [EllipticalArc2D](EllipticalArc2D.md)  

## Description

Gets if the elliptical arc is the geometric equivalent of a circular arc

## Syntax

"ellipticalArc2D_var" is a variable referencing an EllipticalArc2D object.  

```python
# Get the value of the property.
propertyValue = ellipticalArc2D_var.isCircular
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
