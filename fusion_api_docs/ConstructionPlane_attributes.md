# ConstructionPlane.attributes Property

Parent Object: [ConstructionPlane](ConstructionPlane.md)  

## Description

Returns the collection of attributes associated with this construction plane.

## Syntax

"constructionPlane_var" is a variable referencing a ConstructionPlane object.  

```python
# Get the value of the property.
propertyValue = constructionPlane_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
