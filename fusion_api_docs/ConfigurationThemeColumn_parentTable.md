# ConfigurationThemeColumn.parentTable Property

Parent Object: [ConfigurationThemeColumn](ConfigurationThemeColumn.md)  

## Description

Returns the parent table this column is in.

## Syntax

"configurationThemeColumn_var" is a variable referencing a ConfigurationThemeColumn object.  

```python
# Get the value of the property.
propertyValue = configurationThemeColumn_var.parentTable
```

## Property Value

This is a read only property whose value is a [ConfigurationTable](ConfigurationTable.md).

## Version

Introduced in version March 2024
