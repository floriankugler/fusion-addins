# CAMTemplate.description Property

Parent Object: [CAMTemplate](CAMTemplate.md)  

## Description

Gets and sets the description of the template.

## Syntax

"cAMTemplate_var" is a variable referencing a CAMTemplate object.  

```python
# Get the value of the property.
propertyValue = cAMTemplate_var.description

# Set the value of the property.
cAMTemplate_var.description = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version April 2023
