# BoolValueCommandInput.name Property

Parent Object: [BoolValueCommandInput](BoolValueCommandInput.md)  

## Description

Gets the user visible name of this input.

## Syntax

"boolValueCommandInput_var" is a variable referencing a BoolValueCommandInput object.  

```python
# Get the value of the property.
propertyValue = boolValueCommandInput_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2014
