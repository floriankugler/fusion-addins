# SketchFittedSpline.isConstruction Property

Parent Object: [SketchFittedSpline](SketchFittedSpline.md)  

## Description

Gets and sets whether this curve is construction geometry.

## Syntax

"sketchFittedSpline_var" is a variable referencing a SketchFittedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFittedSpline_var.isConstruction

# Set the value of the property.
sketchFittedSpline_var.isConstruction = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
