# HTMLEventHandler.notify Method

Parent Object: [HTMLEventHandler](HTMLEventHandler.md)  

## Description

The function called by Fusion when the associated event is fired.

## Syntax

"hTMLEventHandler_var" is a variable referencing a [HTMLEventHandler](HTMLEventHandler.md) object.

```python
returnValue = hTMLEventHandler_var.notify(eventArgs)
```

## Parameters

| Name | Type | Description |
|----|----|----|
| eventArgs | [HTMLEventArgs](HTMLEventArgs.md) | Returns an object that provides access to additional information associated with the event. |

## Version

Introduced in version March 2017
