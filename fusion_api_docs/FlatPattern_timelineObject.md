# FlatPattern.timelineObject Property

Parent Object: [FlatPattern](FlatPattern.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"flatPattern_var" is a variable referencing a FlatPattern object.  

```python
# Get the value of the property.
propertyValue = flatPattern_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version October 2022
