# PipeFeature.sideFaces Property

Parent Object: [PipeFeature](PipeFeature.md)  

## Description

Property that returns an object that provides access to all of the faces created around the perimeter of the feature.

## Syntax

"pipeFeature_var" is a variable referencing a PipeFeature object.  

```python
# Get the value of the property.
propertyValue = pipeFeature_var.sideFaces
```

## Property Value

This is a read only property whose value is a [BRepFaces](BRepFaces.md).

## Version

Introduced in version October 2023
