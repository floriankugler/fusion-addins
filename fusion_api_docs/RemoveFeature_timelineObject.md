# RemoveFeature.timelineObject Property

Parent Object: [RemoveFeature](RemoveFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"removeFeature_var" is a variable referencing a RemoveFeature object.  

```python
# Get the value of the property.
propertyValue = removeFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version September 2015
