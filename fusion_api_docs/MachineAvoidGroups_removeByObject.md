# MachineAvoidGroups.removeByObject Method

Parent Object: [MachineAvoidGroups](MachineAvoidGroups.md)  

## Description

Function that removes the specified group object from the collection.

## Syntax

"machineAvoidGroups_var" is a variable referencing a [MachineAvoidGroups](MachineAvoidGroups.md) object.

```python
returnValue = machineAvoidGroups_var.removeByObject(group)
```

## Parameters

| Name | Type | Description |
|----|----|----|
| group | [MachineAvoidSelectionBase](MachineAvoidSelectionBase.md) |  |

## Version

Introduced in version September 2024
