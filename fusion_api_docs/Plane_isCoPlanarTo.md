# Plane.isCoPlanarTo Method

Parent Object: [Plane](Plane.md)  

## Description

Checks if this plane is coplanar with another plane.

## Syntax

"plane_var" is a variable referencing a [Plane](Plane.md) object.

```python
returnValue = plane_var.isCoPlanarTo(plane)
```

## Return Value

| Type    | Description                              |
|---------|------------------------------------------|
| boolean | Returns true if the planes are coplanar. |

## Parameters

| Name  | Type               | Description                                 |
|-------|--------------------|---------------------------------------------|
| plane | [Plane](Plane.md) | The plane to compare with for co-planarity. |

## Version

Introduced in version August 2014
