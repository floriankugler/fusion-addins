# UserParameter.isFavorite Property

Parent Object: [UserParameter](UserParameter.md)  

## Description

Gets and sets whether this parameter is included in the Favorites list in the parameters dialog

## Syntax

"userParameter_var" is a variable referencing a UserParameter object.  

```python
# Get the value of the property.
propertyValue = userParameter_var.isFavorite

# Set the value of the property.
userParameter_var.isFavorite = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
