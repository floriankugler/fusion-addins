# Document.isVisible Property

Parent Object: [Document](Document.md)  

## Description

Gets if a currently open document is open as visible.

## Syntax

"document_var" is a variable referencing a Document object.  

```python
# Get the value of the property.
propertyValue = document_var.isVisible
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version May 2016
