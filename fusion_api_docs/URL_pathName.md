# URL.pathName Property

Parent Object: [URL](URL.md)  

## Description

Get the path name of the URL, including the last '/' of the protocol followed by the path of the URL.

## Syntax

"uRL_var" is a variable referencing a URL object.  

```python
# Get the value of the property.
propertyValue = uRL_var.pathName
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
