# CustomGraphicsShowThroughColorEffect.opacity Property

Parent Object: [CustomGraphicsShowThroughColorEffect](CustomGraphicsShowThroughColorEffect.md)  

## Description

Gets and sets the opacity value associated with this CustomGraphicsShowThroughColorEffect object. The opacity is used when rendering the portion of the entity that is covered by other objects in the scene. This can be a value between 0 and 1, where 1 is fully opaque and will completely cover any other entities.

## Syntax

"customGraphicsShowThroughColorEffect_var" is a variable referencing a CustomGraphicsShowThroughColorEffect object.  

```python
# Get the value of the property.
propertyValue = customGraphicsShowThroughColorEffect_var.opacity

# Set the value of the property.
customGraphicsShowThroughColorEffect_var.opacity = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version August 2020
