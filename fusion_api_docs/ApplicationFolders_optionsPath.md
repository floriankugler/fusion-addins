# ApplicationFolders.optionsPath Property

Parent Object: [ApplicationFolders](ApplicationFolders.md)  

## Description

Returns the path to the user-specific folder where Fusion saves various options.

## Syntax

"applicationFolders_var" is a variable referencing an ApplicationFolders object.  

```python
# Get the value of the property.
propertyValue = applicationFolders_var.optionsPath
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version September 2024
