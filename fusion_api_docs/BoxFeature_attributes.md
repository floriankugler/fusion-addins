# BoxFeature.attributes Property

Parent Object: [BoxFeature](BoxFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"boxFeature_var" is a variable referencing a BoxFeature object.  

```python
# Get the value of the property.
propertyValue = boxFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
