# SweepFeature.isSolid Property

Parent Object: [SweepFeature](SweepFeature.md)  

## Description

Indicates if this feature was initially created as a solid or a surface.

## Syntax

"sweepFeature_var" is a variable referencing a SweepFeature object.  

```python
# Get the value of the property.
propertyValue = sweepFeature_var.isSolid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version March 2015
