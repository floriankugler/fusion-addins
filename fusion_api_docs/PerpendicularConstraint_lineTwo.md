# PerpendicularConstraint.lineTwo Property

Parent Object: [PerpendicularConstraint](PerpendicularConstraint.md)  

## Description

Returns the second line.

## Syntax

"perpendicularConstraint_var" is a variable referencing a PerpendicularConstraint object.  

```python
# Get the value of the property.
propertyValue = perpendicularConstraint_var.lineTwo
```

## Property Value

This is a read only property whose value is a [SketchLine](SketchLine.md).

## Version

Introduced in version August 2014
