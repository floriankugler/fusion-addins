# Component.constructionPlanes Property

Parent Object: [Component](Component.md)  

## Description

Returns the construction planes collection associated with this component. This provides access to the existing construction planes and supports the creation of new construction planes.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.constructionPlanes
```

## Property Value

This is a read only property whose value is a [ConstructionPlanes](ConstructionPlanes.md).
