# FlatPatternComponent.constructionPoints Property

Parent Object: [FlatPatternComponent](FlatPatternComponent.md)  

## Description

Returns the construction points collection associated with this component. This provides access to the existing construction points and supports the creation of new construction points.

## Syntax

"flatPatternComponent_var" is a variable referencing a FlatPatternComponent object.  

```python
# Get the value of the property.
propertyValue = flatPatternComponent_var.constructionPoints
```

## Property Value

This is a read only property whose value is a [ConstructionPoints](ConstructionPoints.md).

## Version

Introduced in version October 2022
