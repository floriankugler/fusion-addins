# TriangleMeshCalculator.calculate Method

Parent Object: [TriangleMeshCalculator](TriangleMeshCalculator.md)  

## Description

Calculates a new triangle mesh based on the current settings.

## Syntax

"triangleMeshCalculator_var" is a variable referencing a [TriangleMeshCalculator](TriangleMeshCalculator.md) object.

```python
returnValue = triangleMeshCalculator_var.calculate()
```

## Return Value

| Type | Description |
|----|----|
| [TriangleMesh](TriangleMesh.md) | Returns the new TriangleMesh object or null in the case where the calculation failed. |

## Version

Introduced in version August 2014
