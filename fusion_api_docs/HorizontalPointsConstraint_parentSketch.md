# HorizontalPointsConstraint.parentSketch Property

Parent Object: [HorizontalPointsConstraint](HorizontalPointsConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"horizontalPointsConstraint_var" is a variable referencing a HorizontalPointsConstraint object.  

```python
# Get the value of the property.
propertyValue = horizontalPointsConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
