# ConcentricConstraint.isValid Property

Parent Object: [ConcentricConstraint](ConcentricConstraint.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"concentricConstraint_var" is a variable referencing a ConcentricConstraint object.  

```python
# Get the value of the property.
propertyValue = concentricConstraint_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
