# BooleanParameterValue.isValid Property

Parent Object: [BooleanParameterValue](BooleanParameterValue.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"booleanParameterValue_var" is a variable referencing a BooleanParameterValue object.  

```python
# Get the value of the property.
propertyValue = booleanParameterValue_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2021
