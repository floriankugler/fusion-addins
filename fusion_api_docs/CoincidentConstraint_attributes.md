# CoincidentConstraint.attributes Property

Parent Object: [CoincidentConstraint](CoincidentConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"coincidentConstraint_var" is a variable referencing a CoincidentConstraint object.  

```python
# Get the value of the property.
propertyValue = coincidentConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
