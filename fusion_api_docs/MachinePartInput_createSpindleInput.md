# MachinePartInput.createSpindleInput Method

Parent Object: [MachinePartInput](MachinePartInput.md)  

## Description

Creates a new MachineSpindleInput object to be used to create a new MachineSpindle.

## Syntax

"machinePartInput_var" is a variable referencing a [MachinePartInput](MachinePartInput.md) object.

```python
returnValue = machinePartInput_var.createSpindleInput()
```

## Return Value

| Type | Description |
|----|----|
| [MachineSpindleInput](MachineSpindleInput.md) | Returns new MachineSpindleInput object. |

## Version

Introduced in version April 2023
