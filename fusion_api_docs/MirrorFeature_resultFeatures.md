# MirrorFeature.resultFeatures Property

Parent Object: [MirrorFeature](MirrorFeature.md)  

## Description

Get the features that were created for this mirror. Returns null in the case where the feature is not parametric.

## Syntax

"mirrorFeature_var" is a variable referencing a MirrorFeature object.  

```python
# Get the value of the property.
propertyValue = mirrorFeature_var.resultFeatures
```

## Property Value

This is a read only property whose value is an [ObjectCollection](ObjectCollection.md).

## Version

Introduced in version November 2014
