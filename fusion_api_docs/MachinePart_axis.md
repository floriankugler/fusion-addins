# MachinePart.axis Property

Parent Object: [MachinePart](MachinePart.md)  

## Description

Get the axis object for this part used to reference this part for other operations. Only valid when partType is Axis, otherwise returns null

## Syntax

"machinePart_var" is a variable referencing a MachinePart object.  

```python
# Get the value of the property.
propertyValue = machinePart_var.axis
```

## Property Value

This is a read only property whose value is a [MachineAxis](MachineAxis.md).

## Version

Introduced in version April 2023
