# TorusFeature.deleteMe Method

Parent Object: [TorusFeature](TorusFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"torusFeature_var" is a variable referencing a [TorusFeature](TorusFeature.md) object.

```python
returnValue = torusFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version September 2015
