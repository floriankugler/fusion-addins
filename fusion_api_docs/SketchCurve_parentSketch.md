# SketchCurve.parentSketch Property

Parent Object: [SketchCurve](SketchCurve.md)  

## Description

Returns the parent sketch.

## Syntax

"sketchCurve_var" is a variable referencing a SketchCurve object.  

```python
# Get the value of the property.
propertyValue = sketchCurve_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
