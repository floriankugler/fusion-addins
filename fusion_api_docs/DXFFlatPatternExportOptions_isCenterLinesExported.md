# DXFFlatPatternExportOptions.isCenterLinesExported Property

Parent Object: [DXFFlatPatternExportOptions](DXFFlatPatternExportOptions.md)  

## Description

Specifies if the center lines (bend line) of the flat pattern are exported in the DXF. Defaults to true.

## Syntax

"dXFFlatPatternExportOptions_var" is a variable referencing a DXFFlatPatternExportOptions object.  

```python
# Get the value of the property.
propertyValue = dXFFlatPatternExportOptions_var.isCenterLinesExported

# Set the value of the property.
dXFFlatPatternExportOptions_var.isCenterLinesExported = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2025
