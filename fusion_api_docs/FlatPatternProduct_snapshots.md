# FlatPatternProduct.snapshots Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Returns the Snapshots object associated with this design which provides access to the existing snapshots and the creation of new snapshots.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.snapshots
```

## Property Value

This is a read only property whose value is a [Snapshots](Snapshots.md).

## Version

Introduced in version October 2022
