# Operation.isGenerating Property

Parent Object: [Operation](Operation.md)  

## Description

Gets if the operation is being generated.

## Syntax

"operation_var" is a variable referencing an Operation object.  

```python
# Get the value of the property.
propertyValue = operation_var.isGenerating
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2016
