# Script.folder Property

Parent Object: [Script](Script.md)  

## Description

Gets the full path of the folder that contains this script or add-in.

## Syntax

"script_var" is a variable referencing a Script object.  

```python
# Get the value of the property.
propertyValue = script_var.folder
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version October 2023
