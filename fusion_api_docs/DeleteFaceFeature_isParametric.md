# DeleteFaceFeature.isParametric Property

Parent Object: [DeleteFaceFeature](DeleteFaceFeature.md)  

## Description

Indicates if this feature is parametric or not.

## Syntax

"deleteFaceFeature_var" is a variable referencing a DeleteFaceFeature object.  

```python
# Get the value of the property.
propertyValue = deleteFaceFeature_var.isParametric
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2016
