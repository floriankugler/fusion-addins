# DXF2DImportOptions.filename Property

Parent Object: [DXF2DImportOptions](DXF2DImportOptions.md)  

## Description

Gets and sets the filename or URL of the file to be imported.

## Syntax

"dXF2DImportOptions_var" is a variable referencing a DXF2DImportOptions object.  

```python
# Get the value of the property.
propertyValue = dXF2DImportOptions_var.filename

# Set the value of the property.
dXF2DImportOptions_var.filename = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version November 2015
