# DataFile.isMilestone Property

Parent Object: [DataFile](DataFile.md)  

## Description

Returns if the version this Datafile represents is a milestone. Returns true if it is a milestone.

## Syntax

"dataFile_var" is a variable referencing a DataFile object.  

```python
# Get the value of the property.
propertyValue = dataFile_var.isMilestone
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version March 2024
