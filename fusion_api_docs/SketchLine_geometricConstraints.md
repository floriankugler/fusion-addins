# SketchLine.geometricConstraints Property

Parent Object: [SketchLine](SketchLine.md)  

## Description

Returns the sketch constraints that are attached to this curve.

## Syntax

"sketchLine_var" is a variable referencing a SketchLine object.  

```python
# Get the value of the property.
propertyValue = sketchLine_var.geometricConstraints
```

## Property Value

This is a read only property whose value is a [GeometricConstraintList](GeometricConstraintList.md).

## Version

Introduced in version January 2015
