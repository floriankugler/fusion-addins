# Sketch.originPoint Property

Parent Object: [Sketch](Sketch.md)  

## Description

Returns the sketch point that was automatically created by projecting the origin construction point into the sketch.

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.originPoint
```

## Property Value

This is a read only property whose value is a [SketchPoint](SketchPoint.md).

## Version

Introduced in version June 2015
