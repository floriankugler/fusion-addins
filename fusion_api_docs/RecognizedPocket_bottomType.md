# RecognizedPocket.bottomType Property

Parent Object: [RecognizedPocket](RecognizedPocket.md)  

## Description

Returns the type of bottom edge this pocket has.

## Syntax

"recognizedPocket_var" is a variable referencing a RecognizedPocket object.  

```python
# Get the value of the property.
propertyValue = recognizedPocket_var.bottomType
```

## Property Value

This is a read only property whose value is a [RecognizedPocketBottomType](RecognizedPocketBottomType.md).

## Version

Introduced in version July 2023
