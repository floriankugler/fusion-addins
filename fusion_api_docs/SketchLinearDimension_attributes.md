# SketchLinearDimension.attributes Property

Parent Object: [SketchLinearDimension](SketchLinearDimension.md)  

## Description

Returns the collection of attributes associated with this sketch dimension.

## Syntax

"sketchLinearDimension_var" is a variable referencing a SketchLinearDimension object.  

```python
# Get the value of the property.
propertyValue = sketchLinearDimension_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
