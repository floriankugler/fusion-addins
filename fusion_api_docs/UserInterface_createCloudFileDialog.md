# UserInterface.createCloudFileDialog Method

Parent Object: [UserInterface](UserInterface.md)  

## Description

Creates a new CloudFileDialog object which provides the ability to show a file selection dialog to the user that allows them to choose a file from Fusion web client.

## Syntax

"userInterface_var" is a variable referencing a [UserInterface](UserInterface.md) object.

```python
returnValue = userInterface_var.createCloudFileDialog()
```

## Return Value

| Type | Description |
|----|----|
| [CloudFileDialog](CloudFileDialog.md) | Returns the created CloudFileDialog object that you can use to define the contents of and display a standard file dialog. |

## Version

Introduced in version October 2022
