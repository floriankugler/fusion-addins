# DataFile.name Property

Parent Object: [DataFile](DataFile.md)  

## Description

Gets and sets the displayed name of this item.

## Syntax

"dataFile_var" is a variable referencing a DataFile object.  

```python
# Get the value of the property.
propertyValue = dataFile_var.name

# Set the value of the property.
dataFile_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version January 2015
