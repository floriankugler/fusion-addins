# CAMExportManager.create3MFOptions Method

Parent Object: [CAMExportManager](CAMExportManager.md)  

## Description

Creates a new 3MF export option.

## Syntax

"cAMExportManager_var" is a variable referencing a [CAMExportManager](CAMExportManager.md) object.

```python
returnValue = cAMExportManager_var.create3MFOptions()
```

## Return Value

| Type | Description |
|----|----|
| [CAM3MFExportOptions](CAM3MFExportOptions.md) | Returns new CAM3MFExportOptions. |

## Version

Introduced in version November 2021
