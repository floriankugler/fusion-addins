# ConfigurationSheetMetalRuleTable.rows Property

Parent Object: [ConfigurationSheetMetalRuleTable](ConfigurationSheetMetalRuleTable.md)  

## Description

Returns the rows (configurations) defined for this table and provides the functionality to create new rows.

## Syntax

"configurationSheetMetalRuleTable_var" is a variable referencing a ConfigurationSheetMetalRuleTable object.  

```python
# Get the value of the property.
propertyValue = configurationSheetMetalRuleTable_var.rows
```

## Property Value

This is a read only property whose value is a [ConfigurationRows](ConfigurationRows.md).

## Version

Introduced in version January 2024
