# CAMHoleRecognition.parentSetup Property

Parent Object: [CAMHoleRecognition](CAMHoleRecognition.md)  

## Description

Gets the Setup this operation belongs to.

## Syntax

"cAMHoleRecognition_var" is a variable referencing a CAMHoleRecognition object.  

```python
# Get the value of the property.
propertyValue = cAMHoleRecognition_var.parentSetup
```

## Property Value

This is a read only property whose value is a [Setup](Setup.md).

## Version

Introduced in version April 2023
