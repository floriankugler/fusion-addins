# SketchOffsetDimension.textPosition Property

Parent Object: [SketchOffsetDimension](SketchOffsetDimension.md)  

## Description

Gets and sets position of the dimension text.

## Syntax

"sketchOffsetDimension_var" is a variable referencing a SketchOffsetDimension object.  

```python
# Get the value of the property.
propertyValue = sketchOffsetDimension_var.textPosition

# Set the value of the property.
sketchOffsetDimension_var.textPosition = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
