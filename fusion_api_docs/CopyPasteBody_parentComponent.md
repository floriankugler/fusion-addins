# CopyPasteBody.parentComponent Property

Parent Object: [CopyPasteBody](CopyPasteBody.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"copyPasteBody_var" is a variable referencing a CopyPasteBody object.  

```python
# Get the value of the property.
propertyValue = copyPasteBody_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version June 2017
