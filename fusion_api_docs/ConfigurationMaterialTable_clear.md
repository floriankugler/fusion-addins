# ConfigurationMaterialTable.clear Method

Parent Object: [ConfigurationMaterialTable](ConfigurationMaterialTable.md)  

## Description

Clears the content of the material table, removes the reference from the top table, and hides it in the user interface.

## Syntax

"configurationMaterialTable_var" is a variable referencing a [ConfigurationMaterialTable](ConfigurationMaterialTable.md) object.

```python
returnValue = configurationMaterialTable_var.clear()
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Version

Introduced in version March 2024
