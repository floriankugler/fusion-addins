# DataProjects.asArray Method

Parent Object: [DataProjects](DataProjects.md)  

## Description

Get the current list of all projects.

## Syntax

"dataProjects_var" is a variable referencing a [DataProjects](DataProjects.md) object.

```python
returnValue = dataProjects_var.asArray()
```

## Return Value

| Type | Description |
|----|----|
| [DataProject](DataProject.md)\[\] | Returns the current list of all projects. |

## Version

Introduced in version April 2021
