# ModelParameter.modelParameters Property

Parent Object: [ModelParameter](ModelParameter.md)  

## Description

Returns the Collection containing the ModelParameter.

## Syntax

"modelParameter_var" is a variable referencing a ModelParameter object.  

```python
# Get the value of the property.
propertyValue = modelParameter_var.modelParameters
```

## Property Value

This is a read only property whose value is a [ModelParameters](ModelParameters.md).

## Version

Introduced in version August 2014
