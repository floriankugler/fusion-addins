# DropDownControl.index Property

Parent Object: [DropDownControl](DropDownControl.md)  

## Description

Gets the position of this control within the list of controls within the panel, toolbar, or drop-down control.

## Syntax

"dropDownControl_var" is a variable referencing a DropDownControl object.  

```python
# Get the value of the property.
propertyValue = dropDownControl_var.index
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version August 2014
