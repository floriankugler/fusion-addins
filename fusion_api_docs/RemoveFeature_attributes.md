# RemoveFeature.attributes Property

Parent Object: [RemoveFeature](RemoveFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"removeFeature_var" is a variable referencing a RemoveFeature object.  

```python
# Get the value of the property.
propertyValue = removeFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
