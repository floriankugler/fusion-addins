# SweepFeature.operation Property

Parent Object: [SweepFeature](SweepFeature.md)  

## Description

Gets and sets the type of operation performed by the sweep.

## Syntax

"sweepFeature_var" is a variable referencing a SweepFeature object.  

```python
# Get the value of the property.
propertyValue = sweepFeature_var.operation

# Set the value of the property.
sweepFeature_var.operation = propertyValue
```

## Property Value

This is a read/write property whose value is a [FeatureOperations](FeatureOperations.md).

## Version

Introduced in version November 2014
