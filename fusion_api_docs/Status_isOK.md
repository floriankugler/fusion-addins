# Status.isOK Property

Parent Object: [Status](Status.md)  

## Description

If true, the operation was successful without any warnings or errors. This takes into account all of the child status messages.

## Syntax

"status_var" is a variable referencing a Status object.  

```python
# Get the value of the property.
propertyValue = status_var.isOK
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2021
