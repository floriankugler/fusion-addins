# MidPointConstraint.midPointCurve Property

Parent Object: [MidPointConstraint](MidPointConstraint.md)  

## Description

Returns the curve defining the midpoint.

## Syntax

"midPointConstraint_var" is a variable referencing a MidPointConstraint object.  

```python
# Get the value of the property.
propertyValue = midPointConstraint_var.midPointCurve
```

## Property Value

This is a read only property whose value is a [SketchCurve](SketchCurve.md).

## Version

Introduced in version August 2014
