# DataFile.createDataVersion Method

Parent Object: [DataFile](DataFile.md)  

## Description

Creates a version of this DataFile at tip.

## Syntax

"dataFile_var" is a variable referencing a [DataFile](DataFile.md) object.

```python
returnValue = dataFile_var.createDataVersion(versionDescription)
```

## Return Value

| Type    | Description                                        |
|---------|----------------------------------------------------|
| boolean | Returns true when Create Version Job is triggered. |

## Parameters

| Name | Type | Description |
|----|----|----|
| versionDescription | string | The versionDescription is visible in the History Panel. |

## Version

Introduced in version November 2025
