# CAM.attributes Property

Parent Object: [CAM](CAM.md)  

## Description

Returns the collection of attributes associated with this product.

## Syntax

"cAM_var" is a variable referencing a CAM object.  

```python
# Get the value of the property.
propertyValue = cAM_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
