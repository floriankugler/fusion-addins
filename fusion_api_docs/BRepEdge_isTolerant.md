# BRepEdge.isTolerant Property

Parent Object: [BRepEdge](BRepEdge.md)  

## Description

Returns if the edge is tolerant. The tolerance used is available from the tolerance property.

## Syntax

"bRepEdge_var" is a variable referencing a BRepEdge object.  

```python
# Get the value of the property.
propertyValue = bRepEdge_var.isTolerant
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
