# MFGDMDataEventHandler Object

Derived from: [EventHandler](EventHandler.md) Object  

## Description

The MFGDMDataEventHandler is a client implemented class that can be added as a handler to a MFGDMDataEvent.

## Methods

| Name | Description |
|----|----|
| [notify](MFGDMDataEventHandler_notify.md) | The function called by Fusion when the associated event is fired. |
