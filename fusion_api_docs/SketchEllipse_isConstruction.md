# SketchEllipse.isConstruction Property

Parent Object: [SketchEllipse](SketchEllipse.md)  

## Description

Gets and sets whether this curve is construction geometry.

## Syntax

"sketchEllipse_var" is a variable referencing a SketchEllipse object.  

```python
# Get the value of the property.
propertyValue = sketchEllipse_var.isConstruction

# Set the value of the property.
sketchEllipse_var.isConstruction = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
