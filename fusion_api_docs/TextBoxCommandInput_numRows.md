# TextBoxCommandInput.numRows Property

Parent Object: [TextBoxCommandInput](TextBoxCommandInput.md)  

## Description

Gets and sets the height of the text box as defined by the number of rows of text that can be displayed. If the text is larger than will fit in the box a scroll bar will automatically be displayed.

## Syntax

"textBoxCommandInput_var" is a variable referencing a TextBoxCommandInput object.  

```python
# Get the value of the property.
propertyValue = textBoxCommandInput_var.numRows

# Set the value of the property.
textBoxCommandInput_var.numRows = propertyValue
```

## Property Value

This is a read/write property whose value is an integer.

## Version

Introduced in version June 2015
