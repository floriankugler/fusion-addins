# BossFeatureSideInput.holeMajorDiameter Property

Parent Object: [BossFeatureSideInput](BossFeatureSideInput.md)  

## Description

Get or set major hole diameter for counterbore or countersink hole. This input is ignored for blank boss or boss with simple hole.

## Syntax

"bossFeatureSideInput_var" is a variable referencing a BossFeatureSideInput object.  

```python
# Get the value of the property.
propertyValue = bossFeatureSideInput_var.holeMajorDiameter

# Set the value of the property.
bossFeatureSideInput_var.holeMajorDiameter = propertyValue
```

## Property Value

This is a read/write property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version October 2022
