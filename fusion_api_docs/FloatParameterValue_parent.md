# FloatParameterValue.parent Property

Parent Object: [FloatParameterValue](FloatParameterValue.md)  

## Description

Get the parameter object that the value is associated with.

## Syntax

"floatParameterValue_var" is a variable referencing a FloatParameterValue object.  

```python
# Get the value of the property.
propertyValue = floatParameterValue_var.parent
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version September 2021
