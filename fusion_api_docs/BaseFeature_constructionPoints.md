# BaseFeature.constructionPoints Property

Parent Object: [BaseFeature](BaseFeature.md)  

## Description

Returns an array of the construction points associated with this base feature.

## Syntax

"baseFeature_var" is a variable referencing a BaseFeature object.  

```python
# Get the value of the property.
propertyValue = baseFeature_var.constructionPoints
```

## Property Value

This is a read only property whose value is an array of type [ConstructionPoint](ConstructionPoint.md).

## Version

Introduced in version May 2016
