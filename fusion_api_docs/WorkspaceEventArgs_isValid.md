# WorkspaceEventArgs.isValid Property

Parent Object: [WorkspaceEventArgs](WorkspaceEventArgs.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"workspaceEventArgs_var" is a variable referencing a WorkspaceEventArgs object.  

```python
# Get the value of the property.
propertyValue = workspaceEventArgs_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version March 2015
