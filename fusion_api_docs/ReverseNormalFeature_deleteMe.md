# ReverseNormalFeature.deleteMe Method

Parent Object: [ReverseNormalFeature](ReverseNormalFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"reverseNormalFeature_var" is a variable referencing a [ReverseNormalFeature](ReverseNormalFeature.md) object.

```python
returnValue = reverseNormalFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version September 2015
