# SketchEllipticalArc.geometricConstraints Property

Parent Object: [SketchEllipticalArc](SketchEllipticalArc.md)  

## Description

Returns the sketch constraints that are attached to this curve.

## Syntax

"sketchEllipticalArc_var" is a variable referencing a SketchEllipticalArc object.  

```python
# Get the value of the property.
propertyValue = sketchEllipticalArc_var.geometricConstraints
```

## Property Value

This is a read only property whose value is a [GeometricConstraintList](GeometricConstraintList.md).

## Version

Introduced in version January 2015
