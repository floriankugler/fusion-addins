# RotaryMachineAxisConfiguration.maxRapidSpeed Property

Parent Object: [RotaryMachineAxisConfiguration](RotaryMachineAxisConfiguration.md)  

## Description

Specifies the maximum rapid speed for this axis. This would be called feedrate for a linear axis or rotary speed for a rotary axis. Units are cm/s for linear axes or rad/s for rotary axes.

## Syntax

"rotaryMachineAxisConfiguration_var" is a variable referencing a RotaryMachineAxisConfiguration object.  

```python
# Get the value of the property.
propertyValue = rotaryMachineAxisConfiguration_var.maxRapidSpeed

# Set the value of the property.
rotaryMachineAxisConfiguration_var.maxRapidSpeed = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version April 2023
