# Component.attributes Property

Parent Object: [Component](Component.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
