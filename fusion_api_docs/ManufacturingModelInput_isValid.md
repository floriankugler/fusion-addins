# ManufacturingModelInput.isValid Property

Parent Object: [ManufacturingModelInput](ManufacturingModelInput.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"manufacturingModelInput_var" is a variable referencing a ManufacturingModelInput object.  

```python
# Get the value of the property.
propertyValue = manufacturingModelInput_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version April 2023
