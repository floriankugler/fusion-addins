# SketchText.sketchDimensions Property

Parent Object: [SketchText](SketchText.md)  

## Description

Returns the sketch dimensions that are attached to this curve.

## Syntax

"sketchText_var" is a variable referencing a SketchText object.  

```python
# Get the value of the property.
propertyValue = sketchText_var.sketchDimensions
```

## Property Value

This is a read only property whose value is a [SketchDimensionList](SketchDimensionList.md).

## Version

Introduced in version March 2015
