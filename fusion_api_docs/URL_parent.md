# URL.parent Property

Parent Object: [URL](URL.md)  

## Description

Get the parent URL, represented by the section before the last '/'.

## Syntax

"uRL_var" is a variable referencing a URL object.  

```python
# Get the value of the property.
propertyValue = uRL_var.parent
```

## Property Value

This is a read only property whose value is a [URL](URL.md).

## Version

Introduced in version April 2023
