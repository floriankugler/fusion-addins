# ConstructionPlane.isParametric Property

Parent Object: [ConstructionPlane](ConstructionPlane.md)  

## Description

Indicates if this construction plane is parametric or not.

## Syntax

"constructionPlane_var" is a variable referencing a ConstructionPlane object.  

```python
# Get the value of the property.
propertyValue = constructionPlane_var.isParametric
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
