# HoleFeature.holePositionDefinition Property

Parent Object: [HoleFeature](HoleFeature.md)  

## Description

Returns a HolePositionDefinition object that provides access to the information used to define the position of the hole. This returns null in the case where IsParametric is false.

## Syntax

"holeFeature_var" is a variable referencing a HoleFeature object.  

```python
# Get the value of the property.
propertyValue = holeFeature_var.holePositionDefinition
```

## Property Value

This is a read only property whose value is a [HolePositionDefinition](HolePositionDefinition.md).

## Version

Introduced in version August 2014
