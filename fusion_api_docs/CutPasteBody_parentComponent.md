# CutPasteBody.parentComponent Property

Parent Object: [CutPasteBody](CutPasteBody.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"cutPasteBody_var" is a variable referencing a CutPasteBody object.  

```python
# Get the value of the property.
propertyValue = cutPasteBody_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version June 2017
