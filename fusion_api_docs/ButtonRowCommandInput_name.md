# ButtonRowCommandInput.name Property

Parent Object: [ButtonRowCommandInput](ButtonRowCommandInput.md)  

## Description

Gets the user visible name of this input.

## Syntax

"buttonRowCommandInput_var" is a variable referencing a ButtonRowCommandInput object.  

```python
# Get the value of the property.
propertyValue = buttonRowCommandInput_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version June 2015
