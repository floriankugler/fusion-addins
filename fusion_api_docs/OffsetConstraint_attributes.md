# OffsetConstraint.attributes Property

Parent Object: [OffsetConstraint](OffsetConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"offsetConstraint_var" is a variable referencing an OffsetConstraint object.  

```python
# Get the value of the property.
propertyValue = offsetConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
