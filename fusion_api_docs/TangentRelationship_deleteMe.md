# TangentRelationship.deleteMe Method

Parent Object: [TangentRelationship](TangentRelationship.md)  

## Description

Deletes this tangent relationship.

## Syntax

"tangentRelationship_var" is a variable referencing a [TangentRelationship](TangentRelationship.md) object.

```python
returnValue = tangentRelationship_var.deleteMe()
```

## Return Value

| Type    | Description                               |
|---------|-------------------------------------------|
| boolean | Returns true if the delete is successful. |

## Version

Introduced in version May 2022
