# ModelParameter.comment Property

Parent Object: [ModelParameter](ModelParameter.md)  

## Description

The comment associated with this parameter

## Syntax

"modelParameter_var" is a variable referencing a ModelParameter object.  

```python
# Get the value of the property.
propertyValue = modelParameter_var.comment

# Set the value of the property.
modelParameter_var.comment = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version August 2014
