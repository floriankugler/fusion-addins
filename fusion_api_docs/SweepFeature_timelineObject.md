# SweepFeature.timelineObject Property

Parent Object: [SweepFeature](SweepFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"sweepFeature_var" is a variable referencing a SweepFeature object.  

```python
# Get the value of the property.
propertyValue = sweepFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version November 2014
