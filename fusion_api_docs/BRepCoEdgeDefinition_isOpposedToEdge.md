# BRepCoEdgeDefinition.isOpposedToEdge Property

Parent Object: [BRepCoEdgeDefinition](BRepCoEdgeDefinition.md)  

## Description

Gets and sets if the orientation of this BRepCoEdgeDefinition object is reversed with respect to the associated BRepEdgeDefinition object.

## Syntax

"bRepCoEdgeDefinition_var" is a variable referencing a BRepCoEdgeDefinition object.  

```python
# Get the value of the property.
propertyValue = bRepCoEdgeDefinition_var.isOpposedToEdge

# Set the value of the property.
bRepCoEdgeDefinition_var.isOpposedToEdge = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version September 2020
