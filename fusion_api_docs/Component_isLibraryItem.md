# Component.isLibraryItem Property

Parent Object: [Component](Component.md)  

## Description

Gets whether the component was created from a fasteners library item."

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.isLibraryItem
```

## Property Value

This is a read only property whose value is a boolean.
