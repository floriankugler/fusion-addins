# DraftAnalyses.count Property

Parent Object: [DraftAnalyses](DraftAnalyses.md)  

## Description

Returns the number of DraftAnalysis objects in the collection.

## Syntax

"draftAnalyses_var" is a variable referencing a DraftAnalyses object.  

```python
# Get the value of the property.
propertyValue = draftAnalyses_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version January 2023
