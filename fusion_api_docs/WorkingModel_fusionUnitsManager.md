# WorkingModel.fusionUnitsManager Property

Parent Object: [WorkingModel](WorkingModel.md)  

## Description

Returns a specialized UnitsManager that can set the default length units and work with parameters.

## Syntax

"workingModel_var" is a variable referencing a WorkingModel object.  

```python
# Get the value of the property.
propertyValue = workingModel_var.fusionUnitsManager
```

## Property Value

This is a read only property whose value is a [FusionUnitsManager](FusionUnitsManager.md).

## Version

Introduced in version January 2024
