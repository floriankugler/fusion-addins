# SketchConicCurve.boundingBox Property

Parent Object: [SketchConicCurve](SketchConicCurve.md)  

## Description

Returns the bounding box of the entity in sketch space.

## Syntax

"sketchConicCurve_var" is a variable referencing a SketchConicCurve object.  

```python
# Get the value of the property.
propertyValue = sketchConicCurve_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version January 2015
