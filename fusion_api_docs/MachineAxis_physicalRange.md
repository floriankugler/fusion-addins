# MachineAxis.physicalRange Property

Parent Object: [MachineAxis](MachineAxis.md)  

## Description

Range of possible values for this axis. Units are cm for linear axes or radians for rotary axes.

## Syntax

"machineAxis_var" is a variable referencing a MachineAxis object.  

```python
# Get the value of the property.
propertyValue = machineAxis_var.physicalRange

# Set the value of the property.
machineAxis_var.physicalRange = propertyValue
```

## Property Value

This is a read/write property whose value is a [MachineAxisRange](MachineAxisRange.md).

## Version

Introduced in version April 2023
