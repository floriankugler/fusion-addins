# Features.filletFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the fillet features within the component and supports the creation of new fillet features.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.filletFeatures
```

## Property Value

This is a read only property whose value is a [FilletFeatures](FilletFeatures.md).
