# RipFeature.timelineObject Property

Parent Object: [RipFeature](RipFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"ripFeature_var" is a variable referencing a RipFeature object.  

```python
# Get the value of the property.
propertyValue = ripFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version September 2023
