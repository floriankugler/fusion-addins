# ArrangeResultEnvelope.occurrences Property

Parent Object: [ArrangeResultEnvelope](ArrangeResultEnvelope.md)  

## Description

Returns a collection object of the occurrences in this envelope.

## Syntax

"arrangeResultEnvelope_var" is a variable referencing an ArrangeResultEnvelope object.  

```python
# Get the value of the property.
propertyValue = arrangeResultEnvelope_var.occurrences
```

## Property Value

This is a read only property whose value is an [ArrangeOccurrenceResults](ArrangeOccurrenceResults.md).

## Version

Introduced in version January 2025
