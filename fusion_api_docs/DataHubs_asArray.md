# DataHubs.asArray Method

Parent Object: [DataHubs](DataHubs.md)  

## Description

Get the current list of all hubs.

## Syntax

"dataHubs_var" is a variable referencing a [DataHubs](DataHubs.md) object.

```python
returnValue = dataHubs_var.asArray()
```

## Return Value

| Type                       | Description                           |
|----------------------------|---------------------------------------|
| [DataHub](DataHub.md)\[\] | Returns the current list of all hubs. |

## Version

Introduced in version April 2021
