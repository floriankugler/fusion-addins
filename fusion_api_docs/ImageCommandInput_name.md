# ImageCommandInput.name Property

Parent Object: [ImageCommandInput](ImageCommandInput.md)  

## Description

Gets the user visible name of this input.

## Syntax

"imageCommandInput_var" is a variable referencing an ImageCommandInput object.  

```python
# Get the value of the property.
propertyValue = imageCommandInput_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version July 2015
