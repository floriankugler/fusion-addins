# BoundaryFillFeatureInput.isRemoveTools Property

Parent Object: [BoundaryFillFeatureInput](BoundaryFillFeatureInput.md)  

## Description

Gets and sets whether any BRepBodys that were used as tools should be removed as part of the feature creation.

## Syntax

"boundaryFillFeatureInput_var" is a variable referencing a BoundaryFillFeatureInput object.  

```python
# Get the value of the property.
propertyValue = boundaryFillFeatureInput_var.isRemoveTools

# Set the value of the property.
boundaryFillFeatureInput_var.isRemoveTools = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version May 2022
