# DrawingDocument.dataFile Property

Parent Object: [DrawingDocument](DrawingDocument.md)  

## Description

Gets the DataFile that represents this document in A360.

## Syntax

"drawingDocument_var" is a variable referencing a DrawingDocument object.  

```python
# Get the value of the property.
propertyValue = drawingDocument_var.dataFile
```

## Property Value

This is a read only property whose value is a [DataFile](DataFile.md).

## Version

Introduced in version December 2020
