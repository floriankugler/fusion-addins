# RefoldFeature.deleteMe Method

Parent Object: [RefoldFeature](RefoldFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"refoldFeature_var" is a variable referencing a [RefoldFeature](RefoldFeature.md) object.

```python
returnValue = refoldFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version August 2020
