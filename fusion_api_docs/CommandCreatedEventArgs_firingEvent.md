# CommandCreatedEventArgs.firingEvent Property

Parent Object: [CommandCreatedEventArgs](CommandCreatedEventArgs.md)  

## Description

The event that the firing is in response to.

## Syntax

"commandCreatedEventArgs_var" is a variable referencing a CommandCreatedEventArgs object.  

```python
# Get the value of the property.
propertyValue = commandCreatedEventArgs_var.firingEvent
```

## Property Value

This is a read only property whose value is an [Event](Event.md).

## Version

Introduced in version August 2014
