# IntegerSliderCommandInput.name Property

Parent Object: [IntegerSliderCommandInput](IntegerSliderCommandInput.md)  

## Description

Gets the user visible name of this input.

## Syntax

"integerSliderCommandInput_var" is a variable referencing an IntegerSliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = integerSliderCommandInput_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version June 2015
