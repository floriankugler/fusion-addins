# ArrangeOccurrenceResult.parentEnvelope Property

Parent Object: [ArrangeOccurrenceResult](ArrangeOccurrenceResult.md)  

## Description

The Arrange envelope this occurrence is within.

## Syntax

"arrangeOccurrenceResult_var" is a variable referencing an ArrangeOccurrenceResult object.  

```python
# Get the value of the property.
propertyValue = arrangeOccurrenceResult_var.parentEnvelope
```

## Property Value

This is a read only property whose value is an [ArrangeResultEnvelope](ArrangeResultEnvelope.md).

## Version

Introduced in version January 2025
