# JointGeometry.origin Property

Parent Object: [JointGeometry](JointGeometry.md)  

## Description

Returns the origin point that's been calculated for this joint geometry.

## Syntax

"jointGeometry_var" is a variable referencing a JointGeometry object.  

```python
# Get the value of the property.
propertyValue = jointGeometry_var.origin
```

## Property Value

This is a read only property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version July 2015
