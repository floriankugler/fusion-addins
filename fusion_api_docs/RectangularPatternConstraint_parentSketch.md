# RectangularPatternConstraint.parentSketch Property

Parent Object: [RectangularPatternConstraint](RectangularPatternConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"rectangularPatternConstraint_var" is a variable referencing a RectangularPatternConstraint object.  

```python
# Get the value of the property.
propertyValue = rectangularPatternConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version March 2016
