# FlatPatternProduct.unitsManager Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Returns the UnitsManager object associated with this product.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.unitsManager
```

## Property Value

This is a read only property whose value is a [UnitsManager](UnitsManager.md).

## Version

Introduced in version October 2022
