# GraphicsPreferences Object

Derived from: [Base](Base.md) Object  

## Description

The GraphicsPreferences object provides access to the various graphics related preferences.

## Methods

| Name | Description |
|----|----|
| [classType](GraphicsPreferences_classType.md) | Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType(). |
