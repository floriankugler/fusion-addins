# CurvatureCombAnalysis.name Property

Parent Object: [CurvatureCombAnalysis](CurvatureCombAnalysis.md)  

## Description

A property that gets and sets the name of the analysis. If you use a name that is not unique, Fusion will automatically append a number to the name to make it unique.

## Syntax

"curvatureCombAnalysis_var" is a variable referencing a CurvatureCombAnalysis object.  

```python
# Get the value of the property.
propertyValue = curvatureCombAnalysis_var.name

# Set the value of the property.
curvatureCombAnalysis_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version January 2023
