# Occurrence.isVisible Property

Parent Object: [Occurrence](Occurrence.md)  

## Description

Gets whether the occurrence is visible. This property is affected by the assembly context.

## Syntax

"occurrence_var" is a variable referencing an Occurrence object.  

```python
# Get the value of the property.
propertyValue = occurrence_var.isVisible
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
