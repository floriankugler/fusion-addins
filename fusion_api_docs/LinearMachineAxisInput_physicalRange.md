# LinearMachineAxisInput.physicalRange Property

Parent Object: [LinearMachineAxisInput](LinearMachineAxisInput.md)  

## Description

Range of possible values for this axis. Units are cm for linear axes or radians for rotary axes.

## Syntax

"linearMachineAxisInput_var" is a variable referencing a LinearMachineAxisInput object.  

```python
# Get the value of the property.
propertyValue = linearMachineAxisInput_var.physicalRange

# Set the value of the property.
linearMachineAxisInput_var.physicalRange = propertyValue
```

## Property Value

This is a read/write property whose value is a [MachineAxisRange](MachineAxisRange.md).

## Version

Introduced in version April 2023
