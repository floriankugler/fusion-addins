# SmoothConstraint.curveTwo Property

Parent Object: [SmoothConstraint](SmoothConstraint.md)  

## Description

Returns the second curve.

## Syntax

"smoothConstraint_var" is a variable referencing a SmoothConstraint object.  

```python
# Get the value of the property.
propertyValue = smoothConstraint_var.curveTwo
```

## Property Value

This is a read only property whose value is a [SketchCurve](SketchCurve.md).

## Version

Introduced in version August 2014
