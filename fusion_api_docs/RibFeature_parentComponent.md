# RibFeature.parentComponent Property

Parent Object: [RibFeature](RibFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"ribFeature_var" is a variable referencing a RibFeature object.  

```python
# Get the value of the property.
propertyValue = ribFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version September 2015
