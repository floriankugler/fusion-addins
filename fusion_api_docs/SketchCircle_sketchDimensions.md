# SketchCircle.sketchDimensions Property

Parent Object: [SketchCircle](SketchCircle.md)  

## Description

Returns the sketch dimensions that are attached to this curve.

## Syntax

"sketchCircle_var" is a variable referencing a SketchCircle object.  

```python
# Get the value of the property.
propertyValue = sketchCircle_var.sketchDimensions
```

## Property Value

This is a read only property whose value is a [SketchDimensionList](SketchDimensionList.md).

## Version

Introduced in version August 2014
