# SketchConicCurve.isConstruction Property

Parent Object: [SketchConicCurve](SketchConicCurve.md)  

## Description

Gets and sets whether this curve is construction geometry.

## Syntax

"sketchConicCurve_var" is a variable referencing a SketchConicCurve object.  

```python
# Get the value of the property.
propertyValue = sketchConicCurve_var.isConstruction

# Set the value of the property.
sketchConicCurve_var.isConstruction = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2015
