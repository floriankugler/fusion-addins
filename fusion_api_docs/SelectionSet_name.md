# SelectionSet.name Property

Parent Object: [SelectionSet](SelectionSet.md)  

## Description

Gets and sets the name of the SelectionSet object. If a name is assigned that is already used, Fusion will append a counter to the name to make it unique.

## Syntax

"selectionSet_var" is a variable referencing a SelectionSet object.  

```python
# Get the value of the property.
propertyValue = selectionSet_var.name

# Set the value of the property.
selectionSet_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version May 2022
