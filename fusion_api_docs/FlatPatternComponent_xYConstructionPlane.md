# FlatPatternComponent.xYConstructionPlane Property

Parent Object: [FlatPatternComponent](FlatPatternComponent.md)  

## Description

Returns the XY origin construction plane.

## Syntax

"flatPatternComponent_var" is a variable referencing a FlatPatternComponent object.  

```python
# Get the value of the property.
propertyValue = flatPatternComponent_var.xYConstructionPlane
```

## Property Value

This is a read only property whose value is a [ConstructionPlane](ConstructionPlane.md).

## Version

Introduced in version October 2022
