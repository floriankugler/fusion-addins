# RuledSurfaceFeature.name Property

Parent Object: [RuledSurfaceFeature](RuledSurfaceFeature.md)  

## Description

Returns the name of the feature as seen in the browser (non-parametric) or in the timeline (parametric).

## Syntax

"ruledSurfaceFeature_var" is a variable referencing a RuledSurfaceFeature object.  

```python
# Get the value of the property.
propertyValue = ruledSurfaceFeature_var.name

# Set the value of the property.
ruledSurfaceFeature_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version September 2020
