# StitchFeatures.count Property

Parent Object: [StitchFeatures](StitchFeatures.md)  

## Description

The number of Stitch features in the collection.

## Syntax

"stitchFeatures_var" is a variable referencing a StitchFeatures object.  

```python
# Get the value of the property.
propertyValue = stitchFeatures_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version June 2015
