# ExtendFeature.isSuppressed Property

Parent Object: [ExtendFeature](ExtendFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"extendFeature_var" is a variable referencing an ExtendFeature object.  

```python
# Get the value of the property.
propertyValue = extendFeature_var.isSuppressed

# Set the value of the property.
extendFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version June 2015
