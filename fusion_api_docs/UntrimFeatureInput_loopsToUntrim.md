# UntrimFeatureInput.loopsToUntrim Property

Parent Object: [UntrimFeatureInput](UntrimFeatureInput.md)  

## Description

Gets the loop objects to untrim. Returns null/None in the case where faces are specified instead of loops

## Syntax

"untrimFeatureInput_var" is a variable referencing a UntrimFeatureInput object.  

```python
# Get the value of the property.
propertyValue = untrimFeatureInput_var.loopsToUntrim
```

## Property Value

This is a read only property whose value is an array of type [BRepLoop](BRepLoop.md).

## Version

Introduced in version January 2021
