# SplitButtonControl.additionalDefinitions Property

Parent Object: [SplitButtonControl](SplitButtonControl.md)  

## Description

Gets or sets the command definitions used to define the buttons associated with the split button.

## Syntax

"splitButtonControl_var" is a variable referencing a SplitButtonControl object.  

```python
# Get the value of the property.
propertyValue = splitButtonControl_var.additionalDefinitions

# Set the value of the property.
splitButtonControl_var.additionalDefinitions = propertyValue
```

## Property Value

This is a read/write property whose value is an array of type [CommandDefinition](CommandDefinition.md).
