# Vector3D.setWithArray Method

Parent Object: [Vector3D](Vector3D.md)  

## Description

Reset this vector with the coordinate values in an array \[x, y, z\].

## Syntax

"vector3D_var" is a variable referencing a [Vector3D](Vector3D.md) object.

```python
returnValue = vector3D_var.setWithArray(coordinates)
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name        | Type       | Description                     |
|-------------|------------|---------------------------------|
| coordinates | double\[\] | The array of coordinate values. |

## Version

Introduced in version August 2014
