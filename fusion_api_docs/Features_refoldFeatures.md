# Features.refoldFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the existing refold features.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.refoldFeatures
```

## Property Value

This is a read only property whose value is a [RefoldFeatures](RefoldFeatures.md).

## Version

Introduced in version August 2020
