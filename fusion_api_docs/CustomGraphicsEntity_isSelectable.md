# CustomGraphicsEntity.isSelectable Property

Parent Object: [CustomGraphicsEntity](CustomGraphicsEntity.md)  

## Description

Gets and sets if the graphics entity is selectable within the graphics window. By default, when a new entity is created it is selectable.

## Syntax

"customGraphicsEntity_var" is a variable referencing a CustomGraphicsEntity object.  

```python
# Get the value of the property.
propertyValue = customGraphicsEntity_var.isSelectable

# Set the value of the property.
customGraphicsEntity_var.isSelectable = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version September 2017
