# TorusFeature.isSuppressed Property

Parent Object: [TorusFeature](TorusFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"torusFeature_var" is a variable referencing a TorusFeature object.  

```python
# Get the value of the property.
propertyValue = torusFeature_var.isSuppressed

# Set the value of the property.
torusFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version September 2015
