# SelectionEventHandler.notify Method

Parent Object: [SelectionEventHandler](SelectionEventHandler.md)  

## Description

This notify member is called when an event is triggered from any event that this handler has been added to.

## Syntax

"selectionEventHandler_var" is a variable referencing a [SelectionEventHandler](SelectionEventHandler.md) object.

```python
returnValue = selectionEventHandler_var.notify(eventArgs)
```

## Parameters

| Name | Type | Description |
|----|----|----|
| eventArgs | [SelectionEventArgs](SelectionEventArgs.md) | The arguments object with details about this event and the firing SelectionEvent. |

## Version

Introduced in version August 2014
