# RuleFilletSettings.ruleType Property

Parent Object: [RuleFilletSettings](RuleFilletSettings.md)  

## Description

Gets the rule type for the rule fillet.

## Syntax

"ruleFilletSettings_var" is a variable referencing a RuleFilletSettings object.  

```python
# Get the value of the property.
propertyValue = ruleFilletSettings_var.ruleType
```

## Property Value

This is a read only property whose value is a [RuleFilletRuleTypes](RuleFilletRuleTypes.md).

## Version

Introduced in version November 2025
