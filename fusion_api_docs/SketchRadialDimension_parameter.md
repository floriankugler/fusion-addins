# SketchRadialDimension.parameter Property

Parent Object: [SketchRadialDimension](SketchRadialDimension.md)  

## Description

Returns the associated parameter or null if there is no associated parameter.

## Syntax

"sketchRadialDimension_var" is a variable referencing a SketchRadialDimension object.  

```python
# Get the value of the property.
propertyValue = sketchRadialDimension_var.parameter
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version August 2014
