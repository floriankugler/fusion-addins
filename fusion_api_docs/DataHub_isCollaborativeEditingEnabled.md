# DataHub.isCollaborativeEditingEnabled Property

Parent Object: [DataHub](DataHub.md)  

## Description

Returns true if Concurrent Editing is enabled.

## Syntax

"dataHub_var" is a variable referencing a DataHub object.  

```python
# Get the value of the property.
propertyValue = dataHub_var.isCollaborativeEditingEnabled
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2025
