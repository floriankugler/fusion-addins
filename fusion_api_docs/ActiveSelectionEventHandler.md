# ActiveSelectionEventHandler Object

Derived from: [EventHandler](EventHandler.md) Object  

## Description

The ActiveSelectionEventHandler is a client implemented class that can be added as a handler to a ActiveSelectionEvent.

## Methods

| Name | Description |
|----|----|
| [notify](ActiveSelectionEventHandler_notify.md) | The function called by Fusion when the associated event is fired. |

## Version

Introduced in version August 2020
