# ConfigurationInsertStandardDesignColumn.index Property

Parent Object: [ConfigurationInsertStandardDesignColumn](ConfigurationInsertStandardDesignColumn.md)  

## Description

The index position of this column within the table. The first column is at index 0 and does not include the "Name" column.

## Syntax

"configurationInsertStandardDesignColumn_var" is a variable referencing a ConfigurationInsertStandardDesignColumn object.  

```python
# Get the value of the property.
propertyValue = configurationInsertStandardDesignColumn_var.index
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version September 2025
