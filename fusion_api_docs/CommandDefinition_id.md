# CommandDefinition.id Property

Parent Object: [CommandDefinition](CommandDefinition.md)  

## Description

Gets the unique id for this command definition. This is guaranteed to be unique with respect to all other command definitions.

## Syntax

"commandDefinition_var" is a variable referencing a CommandDefinition object.  

```python
# Get the value of the property.
propertyValue = commandDefinition_var.id
```

## Property Value

This is a read only property whose value is a string.
