# SketchText.definition Property

Parent Object: [SketchText](SketchText.md)  

## Description

Gets the definition that is currently used to specify how the sketch text is defined.

## Syntax

"sketchText_var" is a variable referencing a SketchText object.  

```python
# Get the value of the property.
propertyValue = sketchText_var.definition
```

## Property Value

This is a read only property whose value is a [SketchTextDefinition](SketchTextDefinition.md).

## Version

Introduced in version December 2020
