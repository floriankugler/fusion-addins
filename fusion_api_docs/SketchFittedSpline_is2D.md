# SketchFittedSpline.is2D Property

Parent Object: [SketchFittedSpline](SketchFittedSpline.md)  

## Description

Indicates if this curve lies entirely on the sketch x-y plane.

## Syntax

"sketchFittedSpline_var" is a variable referencing a SketchFittedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFittedSpline_var.is2D
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
