# BRepBody.wires Property

Parent Object: [BRepBody](BRepBody.md)  

## Description

Returns any wire bodies that exist within this body.

## Syntax

"bRepBody_var" is a variable referencing a BRepBody object.  

```python
# Get the value of the property.
propertyValue = bRepBody_var.wires
```

## Property Value

This is a read only property whose value is a [BRepWires](BRepWires.md).

## Version

Introduced in version December 2017
