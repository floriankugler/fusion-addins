# BRepShell.vertices Property

Parent Object: [BRepShell](BRepShell.md)  

## Description

Returns the BRepVertices owned by this shell

## Syntax

"bRepShell_var" is a variable referencing a BRepShell object.  

```python
# Get the value of the property.
propertyValue = bRepShell_var.vertices
```

## Property Value

This is a read only property whose value is a [BRepVertices](BRepVertices.md).

## Version

Introduced in version August 2014
