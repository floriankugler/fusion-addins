# DeriveFeature.isPlaceObjectsAtOrigin Property

Parent Object: [DeriveFeature](DeriveFeature.md)  

## Description

Gets or sets whether to place all derived objects at the origin in the destination design or not.

## Syntax

"deriveFeature_var" is a variable referencing a DeriveFeature object.  

```python
# Get the value of the property.
propertyValue = deriveFeature_var.isPlaceObjectsAtOrigin

# Set the value of the property.
deriveFeature_var.isPlaceObjectsAtOrigin = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2026
