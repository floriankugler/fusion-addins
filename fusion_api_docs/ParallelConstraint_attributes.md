# ParallelConstraint.attributes Property

Parent Object: [ParallelConstraint](ParallelConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"parallelConstraint_var" is a variable referencing a ParallelConstraint object.  

```python
# Get the value of the property.
propertyValue = parallelConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
