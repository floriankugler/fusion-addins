# GeneratedDataCollection.count Property

Parent Object: [GeneratedDataCollection](GeneratedDataCollection.md)  

## Description

The number of items in the collection.

## Syntax

"generatedDataCollection_var" is a variable referencing a GeneratedDataCollection object.  

```python
# Get the value of the property.
propertyValue = generatedDataCollection_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version July 2023
