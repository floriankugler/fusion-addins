# ConstructionAxisCircularFaceDefinition.circularFace Property

Parent Object: [ConstructionAxisCircularFaceDefinition](ConstructionAxisCircularFaceDefinition.md)  

## Description

Gets and sets the cylinder, cone, or torus this work axis is parametrically dependent on.

## Syntax

"constructionAxisCircularFaceDefinition_var" is a variable referencing a ConstructionAxisCircularFaceDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionAxisCircularFaceDefinition_var.circularFace

# Set the value of the property.
constructionAxisCircularFaceDefinition_var.circularFace = propertyValue
```

## Property Value

This is a read/write property whose value is a [BRepFace](BRepFace.md).

## Version

Introduced in version August 2014
