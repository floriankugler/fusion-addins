# ArrangeFeatureInput.isValid Property

Parent Object: [ArrangeFeatureInput](ArrangeFeatureInput.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"arrangeFeatureInput_var" is a variable referencing an ArrangeFeatureInput object.  

```python
# Get the value of the property.
propertyValue = arrangeFeatureInput_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2025
