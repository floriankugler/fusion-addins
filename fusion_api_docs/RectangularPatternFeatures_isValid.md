# RectangularPatternFeatures.isValid Property

Parent Object: [RectangularPatternFeatures](RectangularPatternFeatures.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"rectangularPatternFeatures_var" is a variable referencing a RectangularPatternFeatures object.  

```python
# Get the value of the property.
propertyValue = rectangularPatternFeatures_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version November 2014
