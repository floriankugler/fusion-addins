# Plane.copy Method

Parent Object: [Plane](Plane.md)  

## Description

Creates and returns an independent copy of this Plane object.

## Syntax

"plane_var" is a variable referencing a [Plane](Plane.md) object.

```python
returnValue = plane_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [Plane](Plane.md) | Returns a new Plane object that is a copy of this Plane object. |

## Version

Introduced in version August 2014
