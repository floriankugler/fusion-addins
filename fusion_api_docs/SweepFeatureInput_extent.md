# SweepFeatureInput.extent Property

Parent Object: [SweepFeatureInput](SweepFeatureInput.md)  

## Description

Gets and sets the sweep extent type. It defaults to PerpendicularToPathExtentType. This property is ignored when a guide rail has not been specified.

## Syntax

"sweepFeatureInput_var" is a variable referencing a SweepFeatureInput object.  

```python
# Get the value of the property.
propertyValue = sweepFeatureInput_var.extent

# Set the value of the property.
sweepFeatureInput_var.extent = propertyValue
```

## Property Value

This is a read/write property whose value is a [SweepExtentTypes](SweepExtentTypes.md).

## Version

Introduced in version December 2020
