# ConfigurationPlasticRuleTable.id Property

Parent Object: [ConfigurationPlasticRuleTable](ConfigurationPlasticRuleTable.md)  

## Description

Returns the unique ID of this table.

## Syntax

"configurationPlasticRuleTable_var" is a variable referencing a ConfigurationPlasticRuleTable object.  

```python
# Get the value of the property.
propertyValue = configurationPlasticRuleTable_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version
