# UserInterface.createFolderDialog Method

Parent Object: [UserInterface](UserInterface.md)  

## Description

Creates a new FolderDialog object which provides the ability to show a standard folder selection dialog to the user.

## Syntax

"userInterface_var" is a variable referencing a [UserInterface](UserInterface.md) object.

```python
returnValue = userInterface_var.createFolderDialog()
```

## Return Value

| Type | Description |
|----|----|
| [FolderDialog](FolderDialog.md) | Returns the created FolderDialog object that you can use to define the contents of and display a standard folder dialog. |
