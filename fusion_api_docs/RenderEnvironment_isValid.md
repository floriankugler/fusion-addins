# RenderEnvironment.isValid Property

Parent Object: [RenderEnvironment](RenderEnvironment.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"renderEnvironment_var" is a variable referencing a RenderEnvironment object.  

```python
# Get the value of the property.
propertyValue = renderEnvironment_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version May 2023
