# CircularPatternConstraint.deleteMe Method

Parent Object: [CircularPatternConstraint](CircularPatternConstraint.md)  

## Description

Deletes this constraint. The IsDeletable property can be used to determine if this constraint can be deleted.

## Syntax

"circularPatternConstraint_var" is a variable referencing a [CircularPatternConstraint](CircularPatternConstraint.md) object.

```python
returnValue = circularPatternConstraint_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version March 2016
