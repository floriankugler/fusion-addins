# ExtrudeFeature.attributes Property

Parent Object: [ExtrudeFeature](ExtrudeFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"extrudeFeature_var" is a variable referencing an ExtrudeFeature object.  

```python
# Get the value of the property.
propertyValue = extrudeFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
