# RipFeature.healthState Property

Parent Object: [RipFeature](RipFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"ripFeature_var" is a variable referencing a RipFeature object.  

```python
# Get the value of the property.
propertyValue = ripFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version September 2023
