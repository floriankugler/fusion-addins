# CAMFolder.isSelected Property

Parent Object: [CAMFolder](CAMFolder.md)  

## Description

Gets if this operation is selected in the 'Setups' browser.

## Syntax

"cAMFolder_var" is a variable referencing a CAMFolder object.  

```python
# Get the value of the property.
propertyValue = cAMFolder_var.isSelected
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version May 2020
