# SplitFaceFeature.attributes Property

Parent Object: [SplitFaceFeature](SplitFaceFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"splitFaceFeature_var" is a variable referencing a SplitFaceFeature object.  

```python
# Get the value of the property.
propertyValue = splitFaceFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
