# CAMHoleRecognition.duplicate Method

Parent Object: [CAMHoleRecognition](CAMHoleRecognition.md)  

## Description

Creates a duplicate of the operation in the tree after the itself.

## Syntax

"cAMHoleRecognition_var" is a variable referencing a [CAMHoleRecognition](CAMHoleRecognition.md) object.

```python
returnValue = cAMHoleRecognition_var.duplicate()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns if duplicate command was successful. |

## Version

Introduced in version May 2024
