# UntrimFeature.deleteMe Method

Parent Object: [UntrimFeature](UntrimFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"untrimFeature_var" is a variable referencing a [UntrimFeature](UntrimFeature.md) object.

```python
returnValue = untrimFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version January 2021
