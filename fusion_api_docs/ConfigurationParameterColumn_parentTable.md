# ConfigurationParameterColumn.parentTable Property

Parent Object: [ConfigurationParameterColumn](ConfigurationParameterColumn.md)  

## Description

Returns the parent table, either top or custom theme table, this column is in.

## Syntax

"configurationParameterColumn_var" is a variable referencing a ConfigurationParameterColumn object.  

```python
# Get the value of the property.
propertyValue = configurationParameterColumn_var.parentTable
```

## Property Value

This is a read only property whose value is a [ConfigurationTable](ConfigurationTable.md).

## Version

Introduced in version January 2024
