# SurfaceDeleteFaceFeature.attributes Property

Parent Object: [SurfaceDeleteFaceFeature](SurfaceDeleteFaceFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"surfaceDeleteFaceFeature_var" is a variable referencing a SurfaceDeleteFaceFeature object.  

```python
# Get the value of the property.
propertyValue = surfaceDeleteFaceFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version August 2016
