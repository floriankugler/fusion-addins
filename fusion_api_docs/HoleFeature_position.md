# HoleFeature.position Property

Parent Object: [HoleFeature](HoleFeature.md)  

## Description

Returns the position of the hole.

## Syntax

"holeFeature_var" is a variable referencing a HoleFeature object.  

```python
# Get the value of the property.
propertyValue = holeFeature_var.position
```

## Property Value

This is a read only property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
