# CAMHoleRecognition.error Property

Parent Object: [CAMHoleRecognition](CAMHoleRecognition.md)  

## Description

Returns a message corresponding to any active error associated with the value of this parameter.

## Syntax

"cAMHoleRecognition_var" is a variable referencing a CAMHoleRecognition object.  

```python
# Get the value of the property.
propertyValue = cAMHoleRecognition_var.error
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
