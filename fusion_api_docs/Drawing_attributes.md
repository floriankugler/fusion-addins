# Drawing.attributes Property

Parent Object: [Drawing](Drawing.md)  

## Description

Returns the collection of attributes associated with this product.

## Syntax

"drawing_var" is a variable referencing a Drawing object.  

```python
# Get the value of the property.
propertyValue = drawing_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version December 2020
