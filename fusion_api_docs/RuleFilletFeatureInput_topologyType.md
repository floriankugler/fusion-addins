# RuleFilletFeatureInput.topologyType Property

Parent Object: [RuleFilletFeatureInput](RuleFilletFeatureInput.md)  

## Description

Gets and sets the topology type of the rule fillet.

## Syntax

"ruleFilletFeatureInput_var" is a variable referencing a RuleFilletFeatureInput object.  

```python
# Get the value of the property.
propertyValue = ruleFilletFeatureInput_var.topologyType

# Set the value of the property.
ruleFilletFeatureInput_var.topologyType = propertyValue
```

## Property Value

This is a read/write property whose value is a [RuleFilletTopologyTypes](RuleFilletTopologyTypes.md).

## Version

Introduced in version November 2025
