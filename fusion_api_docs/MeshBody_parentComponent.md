# MeshBody.parentComponent Property

Parent Object: [MeshBody](MeshBody.md)  

## Description

Returns the parent Component.

## Syntax

"meshBody_var" is a variable referencing a MeshBody object.  

```python
# Get the value of the property.
propertyValue = meshBody_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version August 2014
