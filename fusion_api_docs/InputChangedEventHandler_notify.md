# InputChangedEventHandler.notify Method

Parent Object: [InputChangedEventHandler](InputChangedEventHandler.md)  

## Description

This notify member is called when an event is triggered from any event that this handler has been added to.

## Syntax

"inputChangedEventHandler_var" is a variable referencing an [InputChangedEventHandler](InputChangedEventHandler.md) object.

```python
returnValue = inputChangedEventHandler_var.notify(eventArgs)
```

## Parameters

| Name | Type | Description |
|----|----|----|
| eventArgs | [InputChangedEventArgs](InputChangedEventArgs.md) | The arguments object with details about this event and the firing InputChangedEvent. |

## Version

Introduced in version August 2014
