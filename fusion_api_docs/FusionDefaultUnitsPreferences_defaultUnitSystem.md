# FusionDefaultUnitsPreferences.defaultUnitSystem Property

Parent Object: [FusionDefaultUnitsPreferences](FusionDefaultUnitsPreferences.md)  

## Description

Gets and sets the default unit system when creating a new Fusion file.

## Syntax

"fusionDefaultUnitsPreferences_var" is a variable referencing a FusionDefaultUnitsPreferences object.  

```python
# Get the value of the property.
propertyValue = fusionDefaultUnitsPreferences_var.defaultUnitSystem

# Set the value of the property.
fusionDefaultUnitsPreferences_var.defaultUnitSystem = propertyValue
```

## Property Value

This is a read/write property whose value is a [UnitSystems](UnitSystems.md).

## Version

Introduced in version May 2025
