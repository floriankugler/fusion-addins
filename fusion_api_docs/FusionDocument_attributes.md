# FusionDocument.attributes Property

Parent Object: [FusionDocument](FusionDocument.md)  

## Description

Returns the collection of attributes associated with this document.

## Syntax

"fusionDocument_var" is a variable referencing a FusionDocument object.  

```python
# Get the value of the property.
propertyValue = fusionDocument_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
