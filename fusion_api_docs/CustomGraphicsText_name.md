# CustomGraphicsText.name Property

Parent Object: [CustomGraphicsText](CustomGraphicsText.md)  

## Description

Gets and sets the name displayed when this entity is selected. If no name has been set, "Custom Graphics" will be displayed.

## Syntax

"customGraphicsText_var" is a variable referencing a CustomGraphicsText object.  

```python
# Get the value of the property.
propertyValue = customGraphicsText_var.name

# Set the value of the property.
customGraphicsText_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version October 2022
