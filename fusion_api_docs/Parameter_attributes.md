# Parameter.attributes Property

Parent Object: [Parameter](Parameter.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"parameter_var" is a variable referencing a Parameter object.  

```python
# Get the value of the property.
propertyValue = parameter_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
