# RecognizedHoleSegment.axis Property

Parent Object: [RecognizedHoleSegment](RecognizedHoleSegment.md)  

## Description

Returns the unit vector that points straight up out of the segment in the global coordinate system.

## Syntax

"recognizedHoleSegment_var" is a variable referencing a RecognizedHoleSegment object.  

```python
# Get the value of the property.
propertyValue = recognizedHoleSegment_var.axis
```

## Property Value

This is a read only property whose value is a [Vector3D](Vector3D.md).

## Version

Introduced in version May 2023
