# Operation.attributes Property

Parent Object: [Operation](Operation.md)  

## Description

Returns the collection of attributes associated with this object.

## Syntax

"operation_var" is a variable referencing an Operation object.  

```python
# Get the value of the property.
propertyValue = operation_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version August 2021
