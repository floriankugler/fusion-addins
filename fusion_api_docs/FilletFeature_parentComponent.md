# FilletFeature.parentComponent Property

Parent Object: [FilletFeature](FilletFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"filletFeature_var" is a variable referencing a FilletFeature object.  

```python
# Get the value of the property.
propertyValue = filletFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version November 2014
