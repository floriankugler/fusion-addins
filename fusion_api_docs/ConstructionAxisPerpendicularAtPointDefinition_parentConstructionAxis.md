# ConstructionAxisPerpendicularAtPointDefinition.parentConstructionAxis Property

Parent Object: [ConstructionAxisPerpendicularAtPointDefinition](ConstructionAxisPerpendicularAtPointDefinition.md)  

## Description

Returns the ConstructionAxis object

## Syntax

"constructionAxisPerpendicularAtPointDefinition_var" is a variable referencing a ConstructionAxisPerpendicularAtPointDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionAxisPerpendicularAtPointDefinition_var.parentConstructionAxis
```

## Property Value

This is a read only property whose value is a [ConstructionAxis](ConstructionAxis.md).
