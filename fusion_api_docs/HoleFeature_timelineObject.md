# HoleFeature.timelineObject Property

Parent Object: [HoleFeature](HoleFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"holeFeature_var" is a variable referencing a HoleFeature object.  

```python
# Get the value of the property.
propertyValue = holeFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version August 2014
