# BRepBody.orientedMinimumBoundingBox Property

Parent Object: [BRepBody](BRepBody.md)  

## Description

Returns an oriented bounding box of the body that is best oriented to tightly fit the body.

## Syntax

"bRepBody_var" is a variable referencing a BRepBody object.  

```python
# Get the value of the property.
propertyValue = bRepBody_var.orientedMinimumBoundingBox
```

## Property Value

This is a read only property whose value is an [OrientedBoundingBox3D](OrientedBoundingBox3D.md).

## Version

Introduced in version January 2024
