# SheetMetalRule.bendRadius Property

Parent Object: [SheetMetalRule](SheetMetalRule.md)  

## Description

The interior radius of the bends. Use the returned SheetMetalRuleValue object to get and set the current value of the radius.

## Syntax

"sheetMetalRule_var" is a variable referencing a SheetMetalRule object.  

```python
# Get the value of the property.
propertyValue = sheetMetalRule_var.bendRadius
```

## Property Value

This is a read only property whose value is a [SheetMetalRuleValue](SheetMetalRuleValue.md).

## Version

Introduced in version November 2022
