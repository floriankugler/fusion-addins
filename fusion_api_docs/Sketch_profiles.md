# Sketch.profiles Property

Parent Object: [Sketch](Sketch.md)  

## Description

Returns the profiles currently computed for the sketch.

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.profiles
```

## Property Value

This is a read only property whose value is a [Profiles](Profiles.md).
