# BaseFeature.timelineObject Property

Parent Object: [BaseFeature](BaseFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"baseFeature_var" is a variable referencing a BaseFeature object.  

```python
# Get the value of the property.
propertyValue = baseFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version September 2015
