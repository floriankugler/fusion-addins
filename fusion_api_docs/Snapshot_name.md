# Snapshot.name Property

Parent Object: [Snapshot](Snapshot.md)  

## Description

Gets and sets the name of the snapshot as seen in the timeline.

## Syntax

"snapshot_var" is a variable referencing a Snapshot object.  

```python
# Get the value of the property.
propertyValue = snapshot_var.name

# Set the value of the property.
snapshot_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version August 2014
