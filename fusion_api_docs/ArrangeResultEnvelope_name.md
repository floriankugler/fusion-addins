# ArrangeResultEnvelope.name Property

Parent Object: [ArrangeResultEnvelope](ArrangeResultEnvelope.md)  

## Description

Gets and sets the name of the envelope as seen in the browser.

## Syntax

"arrangeResultEnvelope_var" is a variable referencing an ArrangeResultEnvelope object.  

```python
# Get the value of the property.
propertyValue = arrangeResultEnvelope_var.name

# Set the value of the property.
arrangeResultEnvelope_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version January 2025
