# ToolbarPanel.name Property

Parent Object: [ToolbarPanel](ToolbarPanel.md)  

## Description

Gets or sets the name of the panel as seen in the user interface.

## Syntax

"toolbarPanel_var" is a variable referencing a ToolbarPanel object.  

```python
# Get the value of the property.
propertyValue = toolbarPanel_var.name

# Set the value of the property.
toolbarPanel_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.
