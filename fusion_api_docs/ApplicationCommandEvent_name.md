# ApplicationCommandEvent.name Property

Parent Object: [ApplicationCommandEvent](ApplicationCommandEvent.md)  

## Description

The name of the event - e.g. "DocumentOpening"

## Syntax

"applicationCommandEvent_var" is a variable referencing an ApplicationCommandEvent object.  

```python
# Get the value of the property.
propertyValue = applicationCommandEvent_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version November 2015
