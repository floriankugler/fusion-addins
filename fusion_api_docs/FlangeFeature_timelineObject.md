# FlangeFeature.timelineObject Property

Parent Object: [FlangeFeature](FlangeFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"flangeFeature_var" is a variable referencing a FlangeFeature object.  

```python
# Get the value of the property.
propertyValue = flangeFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version August 2020
