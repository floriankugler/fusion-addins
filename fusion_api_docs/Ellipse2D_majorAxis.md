# Ellipse2D.majorAxis Property

Parent Object: [Ellipse2D](Ellipse2D.md)  

## Description

Gets and sets the major axis of the ellipse.

## Syntax

"ellipse2D_var" is a variable referencing an Ellipse2D object.  

```python
# Get the value of the property.
propertyValue = ellipse2D_var.majorAxis

# Set the value of the property.
ellipse2D_var.majorAxis = propertyValue
```

## Property Value

This is a read/write property whose value is a [Vector2D](Vector2D.md).

## Version

Introduced in version August 2014
