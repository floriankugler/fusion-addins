# JointOriginInput.angle Property

Parent Object: [JointOriginInput](JointOriginInput.md)  

## Description

Gets and sets the value that defines the angle for the joint origin. This defaults to zero if it's not specified. The value defines an angle and if the ValueInput is defined using the createByReal method the value is assumed to be radians.

## Syntax

"jointOriginInput_var" is a variable referencing a JointOriginInput object.  

```python
# Get the value of the property.
propertyValue = jointOriginInput_var.angle

# Set the value of the property.
jointOriginInput_var.angle = propertyValue
```

## Property Value

This is a read/write property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version September 2015
