# EllipticalArc3D.copy Method

Parent Object: [EllipticalArc3D](EllipticalArc3D.md)  

## Description

Creates and returns a copy of this EllipticalArc3D object.

## Syntax

"ellipticalArc3D_var" is a variable referencing an [EllipticalArc3D](EllipticalArc3D.md) object.

```python
returnValue = ellipticalArc3D_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [EllipticalArc3D](EllipticalArc3D.md) | Returns a new EllipticalArc3D object that is a copy of this Arc3D object. |

## Version

Introduced in version August 2014
