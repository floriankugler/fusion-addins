# SketchFittedSpline.parentSketch Property

Parent Object: [SketchFittedSpline](SketchFittedSpline.md)  

## Description

Returns the parent sketch.

## Syntax

"sketchFittedSpline_var" is a variable referencing a SketchFittedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFittedSpline_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
