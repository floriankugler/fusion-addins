# ConfigurationPlasticRuleColumn.parentPlasticRuleTable Property

Parent Object: [ConfigurationPlasticRuleColumn](ConfigurationPlasticRuleColumn.md)  

## Description

Returns the parent plastic rule table this column is in.

## Syntax

"configurationPlasticRuleColumn_var" is a variable referencing a ConfigurationPlasticRuleColumn object.  

```python
# Get the value of the property.
propertyValue = configurationPlasticRuleColumn_var.parentPlasticRuleTable
```

## Property Value

This is a read only property whose value is a [ConfigurationPlasticRuleTable](ConfigurationPlasticRuleTable.md).

## Version

Introduced in version January 2024
