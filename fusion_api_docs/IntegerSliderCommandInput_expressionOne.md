# IntegerSliderCommandInput.expressionOne Property

Parent Object: [IntegerSliderCommandInput](IntegerSliderCommandInput.md)  

## Description

Uses an expression to set the value in the first input field. This can contain equations and is evaluated using the specified unit type.

## Syntax

"integerSliderCommandInput_var" is a variable referencing an IntegerSliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = integerSliderCommandInput_var.expressionOne

# Set the value of the property.
integerSliderCommandInput_var.expressionOne = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version June 2015
