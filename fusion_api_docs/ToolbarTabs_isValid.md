# ToolbarTabs.isValid Property

Parent Object: [ToolbarTabs](ToolbarTabs.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"toolbarTabs_var" is a variable referencing a ToolbarTabs object.  

```python
# Get the value of the property.
propertyValue = toolbarTabs_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2019
