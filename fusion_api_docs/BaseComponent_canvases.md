# BaseComponent.canvases Property

Parent Object: [BaseComponent](BaseComponent.md)  

## Description

Returns the canvases collection associated with this component. This provides access to the existing canvases and supports the creation of new canvases.

## Syntax

"baseComponent_var" is a variable referencing a BaseComponent object.  

```python
# Get the value of the property.
propertyValue = baseComponent_var.canvases
```

## Property Value

This is a read only property whose value is a [Canvases](Canvases.md).

## Version

Introduced in version July 2023
