# CoilFeatureInput.coilType Property

Parent Object: [CoilFeatureInput](CoilFeatureInput.md)  

## Description

Gets the type of the coil.

## Syntax

"coilFeatureInput_var" is a variable referencing a CoilFeatureInput object.  

```python
# Get the value of the property.
propertyValue = coilFeatureInput_var.coilType
```

## Property Value

This is a read only property whose value is a [CoilFeatureTypes](CoilFeatureTypes.md).

## Version

Introduced in version March 2016
