# SplitBodyFeature.isSplittingToolExtended Property

Parent Object: [SplitBodyFeature](SplitBodyFeature.md)  

## Description

Gets whether or not the splitting tool is to be automatically extended (if possible) so as to completely intersect the bodyToSplit.

## Syntax

"splitBodyFeature_var" is a variable referencing a SplitBodyFeature object.  

```python
# Get the value of the property.
propertyValue = splitBodyFeature_var.isSplittingToolExtended
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version June 2015
