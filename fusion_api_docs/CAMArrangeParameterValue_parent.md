# CAMArrangeParameterValue.parent Property

Parent Object: [CAMArrangeParameterValue](CAMArrangeParameterValue.md)  

## Description

Get the parameter object that the value is associated with.

## Syntax

"cAMArrangeParameterValue_var" is a variable referencing a CAMArrangeParameterValue object.  

```python
# Get the value of the property.
propertyValue = cAMArrangeParameterValue_var.parent
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version March 2024
