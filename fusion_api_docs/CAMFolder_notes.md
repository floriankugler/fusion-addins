# CAMFolder.notes Property

Parent Object: [CAMFolder](CAMFolder.md)  

## Description

Gets and sets the notes of the operation.

## Syntax

"cAMFolder_var" is a variable referencing a CAMFolder object.  

```python
# Get the value of the property.
propertyValue = cAMFolder_var.notes

# Set the value of the property.
cAMFolder_var.notes = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version May 2020
