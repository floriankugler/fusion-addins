# Point3D.vectorTo Method

Parent Object: [Point3D](Point3D.md)  

## Description

Returns a vector from this point to another point.

## Syntax

"point3D_var" is a variable referencing a [Point3D](Point3D.md) object.

```python
returnValue = point3D_var.vectorTo(point)
```

## Return Value

| Type                     | Description                |
|--------------------------|----------------------------|
| [Vector3D](Vector3D.md) | Returns a Vector3D object. |

## Parameters

| Name  | Type                   | Description                                  |
|-------|------------------------|----------------------------------------------|
| point | [Point3D](Point3D.md) | The other point to use to create the vector. |

## Version

Introduced in version August 2014
