# CAMPattern.isSelected Property

Parent Object: [CAMPattern](CAMPattern.md)  

## Description

Gets if this operation is selected in the 'Setups' browser.

## Syntax

"cAMPattern_var" is a variable referencing a CAMPattern object.  

```python
# Get the value of the property.
propertyValue = cAMPattern_var.isSelected
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version May 2020
