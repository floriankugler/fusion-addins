# ImageCommandInput.deleteMe Method

Parent Object: [ImageCommandInput](ImageCommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"imageCommandInput_var" is a variable referencing an [ImageCommandInput](ImageCommandInput.md) object.

```python
returnValue = imageCommandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2020
