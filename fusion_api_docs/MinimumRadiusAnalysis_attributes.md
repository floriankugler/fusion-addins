# MinimumRadiusAnalysis.attributes Property

Parent Object: [MinimumRadiusAnalysis](MinimumRadiusAnalysis.md)  

## Description

A property that returns the collection of attributes associated with this Analysis.

## Syntax

"minimumRadiusAnalysis_var" is a variable referencing a MinimumRadiusAnalysis object.  

```python
# Get the value of the property.
propertyValue = minimumRadiusAnalysis_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version January 2023
