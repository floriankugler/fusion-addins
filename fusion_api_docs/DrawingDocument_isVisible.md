# DrawingDocument.isVisible Property

Parent Object: [DrawingDocument](DrawingDocument.md)  

## Description

Gets if a currently open document is open as visible.

## Syntax

"drawingDocument_var" is a variable referencing a DrawingDocument object.  

```python
# Get the value of the property.
propertyValue = drawingDocument_var.isVisible
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version December 2020
