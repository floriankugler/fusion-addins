# SketchLine.boundingBox Property

Parent Object: [SketchLine](SketchLine.md)  

## Description

Returns the bounding box of the entity in sketch space.

## Syntax

"sketchLine_var" is a variable referencing a SketchLine object.  

```python
# Get the value of the property.
propertyValue = sketchLine_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version August 2014
