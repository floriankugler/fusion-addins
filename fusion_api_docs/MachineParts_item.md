# MachineParts.item Method

Parent Object: [MachineParts](MachineParts.md)  

## Description

Get the part at index in this collection.

## Syntax

"machineParts_var" is a variable referencing a [MachineParts](MachineParts.md) object.

```python
returnValue = machineParts_var.item(index)
```

## Return Value

| Type                           | Description               |
|--------------------------------|---------------------------|
| [MachinePart](MachinePart.md) | The MachinePart at index. |

## Parameters

| Name  | Type    | Description            |
|-------|---------|------------------------|
| index | integer | The index of the part. |

## Version

Introduced in version April 2023
