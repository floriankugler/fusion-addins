# Toolbars.itemById Method

Parent Object: [Toolbars](Toolbars.md)  

## Description

Returns the Toolbar of the specified ID.

## Syntax

"toolbars_var" is a variable referencing a [Toolbars](Toolbars.md) object.

```python
returnValue = toolbars_var.itemById(id)
```

## Return Value

| Type | Description |
|----|----|
| [Toolbar](Toolbar.md) | Returns the toolbar with the specified ID or null if there's not a toolbar with the specified ID. |

## Parameters

| Name | Type   | Description                      |
|------|--------|----------------------------------|
| id   | string | The Id of the toolbar to return. |

## Version

Introduced in version August 2014
