# BaseFeature.parentComponent Property

Parent Object: [BaseFeature](BaseFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"baseFeature_var" is a variable referencing a BaseFeature object.  

```python
# Get the value of the property.
propertyValue = baseFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version September 2015
