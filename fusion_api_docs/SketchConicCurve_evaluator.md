# SketchConicCurve.evaluator Property

Parent Object: [SketchConicCurve](SketchConicCurve.md)  

## Description

Returns an evaluator object that lets you perform evaluations on the precise geometry of the curve.

## Syntax

"sketchConicCurve_var" is a variable referencing a SketchConicCurve object.  

```python
# Get the value of the property.
propertyValue = sketchConicCurve_var.evaluator
```

## Property Value

This is a read only property whose value is a [CurveEvaluator3D](CurveEvaluator3D.md).

## Version

Introduced in version January 2015
