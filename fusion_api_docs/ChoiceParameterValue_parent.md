# ChoiceParameterValue.parent Property

Parent Object: [ChoiceParameterValue](ChoiceParameterValue.md)  

## Description

Get the parameter object that the value is associated with.

## Syntax

"choiceParameterValue_var" is a variable referencing a ChoiceParameterValue object.  

```python
# Get the value of the property.
propertyValue = choiceParameterValue_var.parent
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version September 2021
