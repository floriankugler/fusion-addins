# DocumentToolLibrary.remove Method

Parent Object: [DocumentToolLibrary](DocumentToolLibrary.md)  

## Description

Remove Tool by index from ToolLibrary.

## Syntax

"documentToolLibrary_var" is a variable referencing a [DocumentToolLibrary](DocumentToolLibrary.md) object.

```python
returnValue = documentToolLibrary_var.remove(index)
```

## Return Value

| Type    | Description                                           |
|---------|-------------------------------------------------------|
| boolean | Returns true for successful deletion, false otherwise |

## Parameters

| Name | Type | Description |
|----|----|----|
| index | uinteger | Index of the Tool in the ToolLibrary that should be removed. |

## Version

Introduced in version April 2023
