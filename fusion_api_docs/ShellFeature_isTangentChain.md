# ShellFeature.isTangentChain Property

Parent Object: [ShellFeature](ShellFeature.md)  

## Description

Gets if any faces that are tangentially connected to any of the input faces will also be included in setting InputEntities.

## Syntax

"shellFeature_var" is a variable referencing a ShellFeature object.  

```python
# Get the value of the property.
propertyValue = shellFeature_var.isTangentChain
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version November 2014
