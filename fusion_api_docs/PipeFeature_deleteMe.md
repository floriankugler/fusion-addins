# PipeFeature.deleteMe Method

Parent Object: [PipeFeature](PipeFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"pipeFeature_var" is a variable referencing a [PipeFeature](PipeFeature.md) object.

```python
returnValue = pipeFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version September 2015
