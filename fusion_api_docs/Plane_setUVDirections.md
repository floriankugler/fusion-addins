# Plane.setUVDirections Method

Parent Object: [Plane](Plane.md)  

## Description

Sets the U and V directions of the plane.

## Syntax

"plane_var" is a variable referencing a [Plane](Plane.md) object.

```python
returnValue = plane_var.setUVDirections(uDirection, vDirection)
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name       | Type                     | Description                    |
|------------|--------------------------|--------------------------------|
| uDirection | [Vector3D](Vector3D.md) | The U direction for the plane. |
| vDirection | [Vector3D](Vector3D.md) | The V direction for the plane. |

## Version

Introduced in version August 2014
