# FaceContourSelection.hasWarning Property

Parent Object: [FaceContourSelection](FaceContourSelection.md)  

## Description

Gets if warnings were encountered when applying the selection to a parent.

## Syntax

"faceContourSelection_var" is a variable referencing a FaceContourSelection object.  

```python
# Get the value of the property.
propertyValue = faceContourSelection_var.hasWarning
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2023
