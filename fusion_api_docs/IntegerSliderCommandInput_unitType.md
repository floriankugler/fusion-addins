# IntegerSliderCommandInput.unitType Property

Parent Object: [IntegerSliderCommandInput](IntegerSliderCommandInput.md)  

## Description

Gets and sets the unit type that is used when evaluating the user's input.

## Syntax

"integerSliderCommandInput_var" is a variable referencing an IntegerSliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = integerSliderCommandInput_var.unitType

# Set the value of the property.
integerSliderCommandInput_var.unitType = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version June 2015
