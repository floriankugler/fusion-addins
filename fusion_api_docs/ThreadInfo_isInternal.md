# ThreadInfo.isInternal Property

Parent Object: [ThreadInfo](ThreadInfo.md)  

## Description

Returns and sets if the thread is an internal or external thread. A value of true indicates an internal thread. It defaults to true.

## Syntax

"threadInfo_var" is a variable referencing a ThreadInfo object.  

```python
# Get the value of the property.
propertyValue = threadInfo_var.isInternal

# Set the value of the property.
threadInfo_var.isInternal = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2015
