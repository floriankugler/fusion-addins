# ExtendFeature.parentComponent Property

Parent Object: [ExtendFeature](ExtendFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"extendFeature_var" is a variable referencing an ExtendFeature object.  

```python
# Get the value of the property.
propertyValue = extendFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version June 2015
