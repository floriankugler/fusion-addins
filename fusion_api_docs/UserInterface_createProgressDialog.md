# UserInterface.createProgressDialog Method

Parent Object: [UserInterface](UserInterface.md)  

## Description

Creates a new ProgressDialog object that you can use to display and control a progress dialog.

## Syntax

"userInterface_var" is a variable referencing a [UserInterface](UserInterface.md) object.

```python
returnValue = userInterface_var.createProgressDialog()
```

## Return Value

| Type | Description |
|----|----|
| [ProgressDialog](ProgressDialog.md) | Returns the created ProgressDialog object that you can use to define the contents of and display a progress dialog. |
