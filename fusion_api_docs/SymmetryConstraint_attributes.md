# SymmetryConstraint.attributes Property

Parent Object: [SymmetryConstraint](SymmetryConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"symmetryConstraint_var" is a variable referencing a SymmetryConstraint object.  

```python
# Get the value of the property.
propertyValue = symmetryConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
