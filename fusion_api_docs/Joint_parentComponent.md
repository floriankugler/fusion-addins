# Joint.parentComponent Property

Parent Object: [Joint](Joint.md)  

## Description

Returns the parent component that owns this joint.

## Syntax

"joint_var" is a variable referencing a Joint object.  

```python
# Get the value of the property.
propertyValue = joint_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version July 2015
