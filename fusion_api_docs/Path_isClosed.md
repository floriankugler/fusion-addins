# Path.isClosed Property

Parent Object: [Path](Path.md)  

## Description

Indicates if the path is closed or not. Returns True in the case of a closed path.

## Syntax

"path_var" is a variable referencing a Path object.  

```python
# Get the value of the property.
propertyValue = path_var.isClosed
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version November 2014
