# OperationBase.duplicate Method

Parent Object: [OperationBase](OperationBase.md)  

## Description

Creates a duplicate of the operation in the tree after the itself.

## Syntax

"operationBase_var" is a variable referencing an [OperationBase](OperationBase.md) object.

```python
returnValue = operationBase_var.duplicate()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns if duplicate command was successful. |

## Version

Introduced in version May 2024
