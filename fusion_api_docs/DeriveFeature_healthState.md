# DeriveFeature.healthState Property

Parent Object: [DeriveFeature](DeriveFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"deriveFeature_var" is a variable referencing a DeriveFeature object.  

```python
# Get the value of the property.
propertyValue = deriveFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version January 2026
