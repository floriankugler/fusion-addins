# ConfigurationPlasticRuleTable.rows Property

Parent Object: [ConfigurationPlasticRuleTable](ConfigurationPlasticRuleTable.md)  

## Description

Returns the rows (configurations) defined for this table and provides the functionality to create new rows.

## Syntax

"configurationPlasticRuleTable_var" is a variable referencing a ConfigurationPlasticRuleTable object.  

```python
# Get the value of the property.
propertyValue = configurationPlasticRuleTable_var.rows
```

## Property Value

This is a read only property whose value is a [ConfigurationRows](ConfigurationRows.md).

## Version

Introduced in version January 2024
