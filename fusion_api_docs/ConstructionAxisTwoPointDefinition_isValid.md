# ConstructionAxisTwoPointDefinition.isValid Property

Parent Object: [ConstructionAxisTwoPointDefinition](ConstructionAxisTwoPointDefinition.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"constructionAxisTwoPointDefinition_var" is a variable referencing a ConstructionAxisTwoPointDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionAxisTwoPointDefinition_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.
