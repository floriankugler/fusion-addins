# SketchOffsetDimension.parentSketch Property

Parent Object: [SketchOffsetDimension](SketchOffsetDimension.md)  

## Description

Returns the parent sketch object.

## Syntax

"sketchOffsetDimension_var" is a variable referencing a SketchOffsetDimension object.  

```python
# Get the value of the property.
propertyValue = sketchOffsetDimension_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
