# MidPointConstraint.point Property

Parent Object: [MidPointConstraint](MidPointConstraint.md)  

## Description

Returns the sketch point being constrained.

## Syntax

"midPointConstraint_var" is a variable referencing a MidPointConstraint object.  

```python
# Get the value of the property.
propertyValue = midPointConstraint_var.point
```

## Property Value

This is a read only property whose value is a [SketchPoint](SketchPoint.md).

## Version

Introduced in version August 2014
