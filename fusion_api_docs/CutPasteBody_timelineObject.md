# CutPasteBody.timelineObject Property

Parent Object: [CutPasteBody](CutPasteBody.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"cutPasteBody_var" is a variable referencing a CutPasteBody object.  

```python
# Get the value of the property.
propertyValue = cutPasteBody_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version June 2017
