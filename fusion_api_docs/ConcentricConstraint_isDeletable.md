# ConcentricConstraint.isDeletable Property

Parent Object: [ConcentricConstraint](ConcentricConstraint.md)  

## Description

Indicates if this constraint is deletable.

## Syntax

"concentricConstraint_var" is a variable referencing a ConcentricConstraint object.  

```python
# Get the value of the property.
propertyValue = concentricConstraint_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
