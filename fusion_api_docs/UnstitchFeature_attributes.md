# UnstitchFeature.attributes Property

Parent Object: [UnstitchFeature](UnstitchFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"unstitchFeature_var" is a variable referencing a UnstitchFeature object.  

```python
# Get the value of the property.
propertyValue = unstitchFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
