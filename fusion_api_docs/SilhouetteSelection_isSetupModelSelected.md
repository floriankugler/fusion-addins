# SilhouetteSelection.isSetupModelSelected Property

Parent Object: [SilhouetteSelection](SilhouetteSelection.md)  

## Description

Flag to include all B-Rep bodies set as the setup models.

## Syntax

"silhouetteSelection_var" is a variable referencing a SilhouetteSelection object.  

```python
# Get the value of the property.
propertyValue = silhouetteSelection_var.isSetupModelSelected

# Set the value of the property.
silhouetteSelection_var.isSetupModelSelected = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version April 2023
