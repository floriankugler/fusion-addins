# MotionLink.jointTwo Property

Parent Object: [MotionLink](MotionLink.md)  

## Description

Gets the second joint for this MotionLink. This can return null if the linked motions are from the same joint.

## Syntax

"motionLink_var" is a variable referencing a MotionLink object.  

```python
# Get the value of the property.
propertyValue = motionLink_var.jointTwo
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version November 2025
