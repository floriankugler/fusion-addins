# Script.isInternal Property

Parent Object: [Script](Script.md)  

## Description

Indicates if this is an internal script or add-in that is delivered with Fusion. Returns true if it is an internal script or add-in.

## Syntax

"script_var" is a variable referencing a Script object.  

```python
# Get the value of the property.
propertyValue = script_var.isInternal
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version October 2023
