# Point3D.asArray Method

Parent Object: [Point3D](Point3D.md)  

## Description

Get coordinate data of the point.

## Syntax

"point3D_var" is a variable referencing a [Point3D](Point3D.md) object.

```python
returnValue = point3D_var.asArray()
```

## Return Value

| Type | Description |
|----|----|
| double\[\] | Returns the coordinate data of the point as an array \[x, y, z\]. |

## Version

Introduced in version August 2014
