# FlatPattern.topFace Property

Parent Object: [FlatPattern](FlatPattern.md)  

## Description

Returns the "top" face of the flat pattern B-Rep body.

## Syntax

"flatPattern_var" is a variable referencing a FlatPattern object.  

```python
# Get the value of the property.
propertyValue = flatPattern_var.topFace
```

## Property Value

This is a read only property whose value is a [BRepFace](BRepFace.md).

## Version

Introduced in version October 2022
