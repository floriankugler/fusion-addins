# FlatPatternProduct.timeline Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Returns the timeline associated with this design.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.timeline
```

## Property Value

This is a read only property whose value is a [Timeline](Timeline.md).

## Version

Introduced in version October 2022
