# ScaleFeature.parentComponent Property

Parent Object: [ScaleFeature](ScaleFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"scaleFeature_var" is a variable referencing a ScaleFeature object.  

```python
# Get the value of the property.
propertyValue = scaleFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version January 2015
