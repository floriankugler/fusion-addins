# DataFolder.deleteMe Method

Parent Object: [DataFolder](DataFolder.md)  

## Description

Deletes this folder item.

## Syntax

"dataFolder_var" is a variable referencing a [DataFolder](DataFolder.md) object.

```python
returnValue = dataFolder_var.deleteMe()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns true if the deletion was successful. |

## Version

Introduced in version September 2016
