# AsBuiltJointInput.setAsRigidJointMotion Method

Parent Object: [AsBuiltJointInput](AsBuiltJointInput.md)  

## Description

Defines the relationship between the two joint geometries as a rigid joint.

## Syntax

"asBuiltJointInput_var" is a variable referencing an [AsBuiltJointInput](AsBuiltJointInput.md) object.

```python
returnValue = asBuiltJointInput_var.setAsRigidJointMotion()
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Version

Introduced in version September 2015
