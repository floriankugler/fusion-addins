# CAMPattern.activate Method

Parent Object: [CAMPattern](CAMPattern.md)  

## Description

Sets this object as the default container.

## Syntax

"cAMPattern_var" is a variable referencing a [CAMPattern](CAMPattern.md) object.

```python
returnValue = cAMPattern_var.activate()
```

## Return Value

| Type    | Description                                    |
|---------|------------------------------------------------|
| boolean | Returns true if the activation was successful. |

## Version

Introduced in version April 2023
