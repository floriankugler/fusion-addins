# SketchCircle.deleteMe Method

Parent Object: [SketchCircle](SketchCircle.md)  

## Description

Deletes the entity from the sketch.

## Syntax

"sketchCircle_var" is a variable referencing a [SketchCircle](SketchCircle.md) object.

```python
returnValue = sketchCircle_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true is the delete was successful. |

## Version

Introduced in version August 2014
