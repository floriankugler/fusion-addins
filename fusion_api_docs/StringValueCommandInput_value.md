# StringValueCommandInput.value Property

Parent Object: [StringValueCommandInput](StringValueCommandInput.md)  

## Description

Gets or sets the value of this input.

## Syntax

"stringValueCommandInput_var" is a variable referencing a StringValueCommandInput object.  

```python
# Get the value of the property.
propertyValue = stringValueCommandInput_var.value

# Set the value of the property.
stringValueCommandInput_var.value = propertyValue
```

## Property Value

This is a read/write property whose value is a string.
