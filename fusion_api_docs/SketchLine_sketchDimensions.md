# SketchLine.sketchDimensions Property

Parent Object: [SketchLine](SketchLine.md)  

## Description

Returns the sketch dimensions that are attached to this curve.

## Syntax

"sketchLine_var" is a variable referencing a SketchLine object.  

```python
# Get the value of the property.
propertyValue = sketchLine_var.sketchDimensions
```

## Property Value

This is a read only property whose value is a [SketchDimensionList](SketchDimensionList.md).

## Version

Introduced in version August 2014
