# Component.yZConstructionPlane Property

Parent Object: [Component](Component.md)  

## Description

Returns the YZ origin construction plane.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.yZConstructionPlane
```

## Property Value

This is a read only property whose value is a [ConstructionPlane](ConstructionPlane.md).

## Version

Introduced in version August 2014
