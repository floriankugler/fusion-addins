# ConstructionAxisTwoPointDefinition.parentConstructionAxis Property

Parent Object: [ConstructionAxisTwoPointDefinition](ConstructionAxisTwoPointDefinition.md)  

## Description

Returns the ConstructionAxis object

## Syntax

"constructionAxisTwoPointDefinition_var" is a variable referencing a ConstructionAxisTwoPointDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionAxisTwoPointDefinition_var.parentConstructionAxis
```

## Property Value

This is a read only property whose value is a [ConstructionAxis](ConstructionAxis.md).

## Version

Introduced in version August 2014
