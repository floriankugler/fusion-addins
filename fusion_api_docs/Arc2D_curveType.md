# Arc2D.curveType Property

Parent Object: [Arc2D](Arc2D.md)  

## Description

Returns the type of geometry this curve represents.

## Syntax

"arc2D_var" is a variable referencing an Arc2D object.  

```python
# Get the value of the property.
propertyValue = arc2D_var.curveType
```

## Property Value

This is a read only property whose value is a [Curve2DTypes](Curve2DTypes.md).

## Version

Introduced in version August 2014
