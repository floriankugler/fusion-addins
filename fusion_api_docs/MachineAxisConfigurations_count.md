# MachineAxisConfigurations.count Property

Parent Object: [MachineAxisConfigurations](MachineAxisConfigurations.md)  

## Description

Get the number of configurations in the collection.

## Syntax

"machineAxisConfigurations_var" is a variable referencing a MachineAxisConfigurations object.  

```python
# Get the value of the property.
propertyValue = machineAxisConfigurations_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version April 2023
