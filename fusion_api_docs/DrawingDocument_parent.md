# DrawingDocument.parent Property

Parent Object: [DrawingDocument](DrawingDocument.md)  

## Description

Returns the parent Application object.

## Syntax

"drawingDocument_var" is a variable referencing a DrawingDocument object.  

```python
# Get the value of the property.
propertyValue = drawingDocument_var.parent
```

## Property Value

This is a read only property whose value is an [Application](Application.md).

## Version

Introduced in version December 2020
