# RipFeature.parentComponent Property

Parent Object: [RipFeature](RipFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"ripFeature_var" is a variable referencing a RipFeature object.  

```python
# Get the value of the property.
propertyValue = ripFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version September 2023
