# ChoiceProperty.isReadOnly Property

Parent Object: [ChoiceProperty](ChoiceProperty.md)  

## Description

Indicates if this property is read-only. If True any attempted edits will fail.

## Syntax

"choiceProperty_var" is a variable referencing a ChoiceProperty object.  

```python
# Get the value of the property.
propertyValue = choiceProperty_var.isReadOnly
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
