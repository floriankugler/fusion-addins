# Arc2D.endAngle Property

Parent Object: [Arc2D](Arc2D.md)  

## Description

Gets and sets the end angle of the arc in radians, where 0 is along the x axis.

## Syntax

"arc2D_var" is a variable referencing an Arc2D object.  

```python
# Get the value of the property.
propertyValue = arc2D_var.endAngle

# Set the value of the property.
arc2D_var.endAngle = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version August 2014
