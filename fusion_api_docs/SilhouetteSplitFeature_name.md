# SilhouetteSplitFeature.name Property

Parent Object: [SilhouetteSplitFeature](SilhouetteSplitFeature.md)  

## Description

Returns the name of the feature as seen in the browser (non-parametric) or in the timeline (parametric).

## Syntax

"silhouetteSplitFeature_var" is a variable referencing a SilhouetteSplitFeature object.  

```python
# Get the value of the property.
propertyValue = silhouetteSplitFeature_var.name

# Set the value of the property.
silhouetteSplitFeature_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version June 2015
