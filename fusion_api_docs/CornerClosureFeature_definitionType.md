# CornerClosureFeature.definitionType Property

Parent Object: [CornerClosureFeature](CornerClosureFeature.md)  

## Description

Gets the type of corner closure defined.

## Syntax

"cornerClosureFeature_var" is a variable referencing a CornerClosureFeature object.  

```python
# Get the value of the property.
propertyValue = cornerClosureFeature_var.definitionType
```

## Property Value

This is a read only property whose value is a [CornerClosureFeatureDefinitionTypes](CornerClosureFeatureDefinitionTypes.md).

## Version

Introduced in version January 2026
