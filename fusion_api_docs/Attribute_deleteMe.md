# Attribute.deleteMe Method

Parent Object: [Attribute](Attribute.md)  

## Description

Deletes this attribute.

## Syntax

"attribute_var" is a variable referencing an [Attribute](Attribute.md) object.

```python
returnValue = attribute_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version May 2016
