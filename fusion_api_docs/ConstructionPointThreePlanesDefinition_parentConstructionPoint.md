# ConstructionPointThreePlanesDefinition.parentConstructionPoint Property

## Description

## Syntax

"constructionPointThreePlanesDefinition_var" is a variable referencing a ConstructionPointThreePlanesDefinition object.  

```python

propertyValue = constructionPointThreePlanesDefinition_var.parentConstructionPoint
```

## Property Value

This is a read only property whose value is a [ConstructionPoint](ConstructionPoint.md).
