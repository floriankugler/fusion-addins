# MotionLink.attributes Property

Parent Object: [MotionLink](MotionLink.md)  

## Description

Returns the collection of attributes associated with this MotionLink.

## Syntax

"motionLink_var" is a variable referencing a MotionLink object.  

```python
# Get the value of the property.
propertyValue = motionLink_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version November 2025
