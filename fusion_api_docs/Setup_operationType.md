# Setup.operationType Property

Parent Object: [Setup](Setup.md)  

## Description

Gets the Operation Type. It can be MillingOperation, TurningOperation, JetOperation or AdditiveOperation.

## Syntax

"setup_var" is a variable referencing a Setup object.  

```python
# Get the value of the property.
propertyValue = setup_var.operationType
```

## Property Value

This is a read only property whose value is an [OperationTypes](OperationTypes.md).

## Version

Introduced in version January 2016
