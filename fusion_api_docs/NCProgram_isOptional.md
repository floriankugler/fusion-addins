# NCProgram.isOptional Property

Parent Object: [NCProgram](NCProgram.md)  

## Description

Gets and sets the "Optional" property value of the operation. Gets/sets true if the operation is optional.

## Syntax

"nCProgram_var" is a variable referencing a NCProgram object.  

```python
# Get the value of the property.
propertyValue = nCProgram_var.isOptional

# Set the value of the property.
nCProgram_var.isOptional = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version April 2023
