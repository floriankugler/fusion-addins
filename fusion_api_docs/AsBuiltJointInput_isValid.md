# AsBuiltJointInput.isValid Property

Parent Object: [AsBuiltJointInput](AsBuiltJointInput.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"asBuiltJointInput_var" is a variable referencing an AsBuiltJointInput object.  

```python
# Get the value of the property.
propertyValue = asBuiltJointInput_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2015
