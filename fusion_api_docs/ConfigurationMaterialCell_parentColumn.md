# ConfigurationMaterialCell.parentColumn Property

Parent Object: [ConfigurationMaterialCell](ConfigurationMaterialCell.md)  

## Description

Returns the column this cell is in.

## Syntax

"configurationMaterialCell_var" is a variable referencing a ConfigurationMaterialCell object.  

```python
# Get the value of the property.
propertyValue = configurationMaterialCell_var.parentColumn
```

## Property Value

This is a read only property whose value is a [ConfigurationMaterialColumn](ConfigurationMaterialColumn.md).

## Version

Introduced in version January 2024
