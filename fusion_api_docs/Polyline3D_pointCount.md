# Polyline3D.pointCount Property

Parent Object: [Polyline3D](Polyline3D.md)  

## Description

Returns the number of points defining the polyline.

## Syntax

"polyline3D_var" is a variable referencing a Polyline3D object.  

```python
# Get the value of the property.
propertyValue = polyline3D_var.pointCount
```

## Property Value

This is a read only property whose value is an integer.

## Version

Introduced in version September 2024
