# SmoothConstraint.deleteMe Method

Parent Object: [SmoothConstraint](SmoothConstraint.md)  

## Description

Deletes this constraint. The IsDeletable property can be used to determine if this constraint can be deleted.

## Syntax

"smoothConstraint_var" is a variable referencing a [SmoothConstraint](SmoothConstraint.md) object.

```python
returnValue = smoothConstraint_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
