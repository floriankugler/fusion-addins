# CustomGraphicsLines.deleteMe Method

Parent Object: [CustomGraphicsLines](CustomGraphicsLines.md)  

## Description

Deletes the entity from the custom graphics group.

## Syntax

"customGraphicsLines_var" is a variable referencing a [CustomGraphicsLines](CustomGraphicsLines.md) object.

```python
returnValue = customGraphicsLines_var.deleteMe()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns true if the deletion was successful. |

## Version

Introduced in version September 2017
