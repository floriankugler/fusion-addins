# Preferences.generalPreferences Property

Parent Object: [Preferences](Preferences.md)  

## Description

Gets the GeneralPreferences object.

## Syntax

"preferences_var" is a variable referencing a Preferences object.  

```python
# Get the value of the property.
propertyValue = preferences_var.generalPreferences
```

## Property Value

This is a read only property whose value is a [GeneralPreferences](GeneralPreferences.md).

## Version

Introduced in version August 2014
