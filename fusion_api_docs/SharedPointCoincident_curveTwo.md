# SharedPointCoincident.curveTwo Property

Parent Object: [SharedPointCoincident](SharedPointCoincident.md)  

## Description

Returns the second sketch curve that is highlighted when the glyph was selected.

## Syntax

"sharedPointCoincident_var" is a variable referencing a SharedPointCoincident object.  

```python
# Get the value of the property.
propertyValue = sharedPointCoincident_var.curveTwo
```

## Property Value

This is a read only property whose value is a [SketchCurve](SketchCurve.md).

## Version

Introduced in version March 2025
