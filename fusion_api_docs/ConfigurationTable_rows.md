# ConfigurationTable.rows Property

Parent Object: [ConfigurationTable](ConfigurationTable.md)  

## Description

Returns the rows (configurations) defined for this table and provides the functionality to create new rows.

## Syntax

"configurationTable_var" is a variable referencing a ConfigurationTable object.  

```python
# Get the value of the property.
propertyValue = configurationTable_var.rows
```

## Property Value

This is a read only property whose value is a [ConfigurationRows](ConfigurationRows.md).

## Version

Introduced in version January 2024
