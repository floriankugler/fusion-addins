# BoolValueCommandInput.isVisible Property

Parent Object: [BoolValueCommandInput](BoolValueCommandInput.md)  

## Description

Gets or sets if this input will be visible to the user.  

Setting a SelectionCommandInput to be invisible will clear any selections it currently has.

## Syntax

"boolValueCommandInput_var" is a variable referencing a BoolValueCommandInput object.  

```python
# Get the value of the property.
propertyValue = boolValueCommandInput_var.isVisible

# Set the value of the property.
boolValueCommandInput_var.isVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
