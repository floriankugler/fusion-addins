# ConfigurationThemeColumn.id Property

Parent Object: [ConfigurationThemeColumn](ConfigurationThemeColumn.md)  

## Description

The id of the column. This is constant and cannot be changed by the user.

## Syntax

"configurationThemeColumn_var" is a variable referencing a ConfigurationThemeColumn object.  

```python
# Get the value of the property.
propertyValue = configurationThemeColumn_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version January 2024
