# OptimizedOrientationResults.isValid Property

Parent Object: [OptimizedOrientationResults](OptimizedOrientationResults.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"optimizedOrientationResults_var" is a variable referencing an OptimizedOrientationResults object.  

```python
# Get the value of the property.
propertyValue = optimizedOrientationResults_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2023
