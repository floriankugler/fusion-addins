# JointGeometry.geometryType Property

Parent Object: [JointGeometry](JointGeometry.md)  

## Description

Returns the type of geometry this JointGeometry object represents.

## Syntax

"jointGeometry_var" is a variable referencing a JointGeometry object.  

```python
# Get the value of the property.
propertyValue = jointGeometry_var.geometryType
```

## Property Value

This is a read only property whose value is a [JointGeometryTypes](JointGeometryTypes.md).

## Version

Introduced in version July 2015
