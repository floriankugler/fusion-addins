# Drawing.parentDocument Property

Parent Object: [Drawing](Drawing.md)  

## Description

Returns the parent Document object.

## Syntax

"drawing_var" is a variable referencing a Drawing object.  

```python
# Get the value of the property.
propertyValue = drawing_var.parentDocument
```

## Property Value

This is a read only property whose value is a [Document](Document.md).

## Version

Introduced in version December 2020
