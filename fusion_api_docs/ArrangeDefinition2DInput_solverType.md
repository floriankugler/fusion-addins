# ArrangeDefinition2DInput.solverType Property

Parent Object: [ArrangeDefinition2DInput](ArrangeDefinition2DInput.md)  

## Description

Gets the type of arrange feature defined by this definition.

## Syntax

"arrangeDefinition2DInput_var" is a variable referencing an ArrangeDefinition2DInput object.  

```python
# Get the value of the property.
propertyValue = arrangeDefinition2DInput_var.solverType
```

## Property Value

This is a read only property whose value is an [ArrangeSolverTypes](ArrangeSolverTypes.md).

## Version

Introduced in version January 2025
