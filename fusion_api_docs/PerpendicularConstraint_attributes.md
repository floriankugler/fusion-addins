# PerpendicularConstraint.attributes Property

Parent Object: [PerpendicularConstraint](PerpendicularConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"perpendicularConstraint_var" is a variable referencing a PerpendicularConstraint object.  

```python
# Get the value of the property.
propertyValue = perpendicularConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
