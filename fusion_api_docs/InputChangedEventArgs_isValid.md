# InputChangedEventArgs.isValid Property

Parent Object: [InputChangedEventArgs](InputChangedEventArgs.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"inputChangedEventArgs_var" is a variable referencing an InputChangedEventArgs object.  

```python
# Get the value of the property.
propertyValue = inputChangedEventArgs_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
