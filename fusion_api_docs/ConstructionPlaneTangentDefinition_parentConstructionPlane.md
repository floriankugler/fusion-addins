# ConstructionPlaneTangentDefinition.parentConstructionPlane Property

Parent Object: [ConstructionPlaneTangentDefinition](ConstructionPlaneTangentDefinition.md)  

## Description

Returns the ConstructionPlane object

## Syntax

"constructionPlaneTangentDefinition_var" is a variable referencing a ConstructionPlaneTangentDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionPlaneTangentDefinition_var.parentConstructionPlane
```

## Property Value

This is a read only property whose value is a [ConstructionPlane](ConstructionPlane.md).

## Version

Introduced in version August 2014
