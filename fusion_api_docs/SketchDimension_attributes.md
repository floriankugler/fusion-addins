# SketchDimension.attributes Property

Parent Object: [SketchDimension](SketchDimension.md)  

## Description

Returns the collection of attributes associated with this sketch dimension.

## Syntax

"sketchDimension_var" is a variable referencing a SketchDimension object.  

```python
# Get the value of the property.
propertyValue = sketchDimension_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
