# Line2D.asNurbsCurve Property

Parent Object: [Line2D](Line2D.md)  

## Description

Returns a NURBS curve that is geometrically identical to the line.

## Syntax

"line2D_var" is a variable referencing a Line2D object.  

```python
# Get the value of the property.
propertyValue = line2D_var.asNurbsCurve
```

## Property Value

This is a read only property whose value is a [NurbsCurve2D](NurbsCurve2D.md).

## Version

Introduced in version August 2014
