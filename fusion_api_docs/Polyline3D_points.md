# Polyline3D.points Property

Parent Object: [Polyline3D](Polyline3D.md)  

## Description

Gets and sets the points that define the coordinates of the polyline.

## Syntax

"polyline3D_var" is a variable referencing a Polyline3D object.  

```python
# Get the value of the property.
propertyValue = polyline3D_var.points

# Set the value of the property.
polyline3D_var.points = propertyValue
```

## Property Value

This is a read/write property whose value is an array of type [Point3D](Point3D.md).

## Version

Introduced in version September 2024
