# SketchDimensions.count Property

Parent Object: [SketchDimensions](SketchDimensions.md)  

## Description

Returns the number of sketch dimensions in the sketch.

## Syntax

"sketchDimensions_var" is a variable referencing a SketchDimensions object.  

```python
# Get the value of the property.
propertyValue = sketchDimensions_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version August 2014
