# CadContours2dParameterValue.isValid Property

Parent Object: [CadContours2dParameterValue](CadContours2dParameterValue.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"cadContours2dParameterValue_var" is a variable referencing a CadContours2dParameterValue object.  

```python
# Get the value of the property.
propertyValue = cadContours2dParameterValue_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version April 2023
