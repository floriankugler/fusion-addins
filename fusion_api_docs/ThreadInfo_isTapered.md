# ThreadInfo.isTapered Property

Parent Object: [ThreadInfo](ThreadInfo.md)  

## Description

Indicates if this ThreadInfo object defines a standard or tapered thread.

## Syntax

"threadInfo_var" is a variable referencing a ThreadInfo object.  

```python
# Get the value of the property.
propertyValue = threadInfo_var.isTapered
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2025
