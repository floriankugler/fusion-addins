# CadMachineAvoidGroupsParameterValue.parent Property

Parent Object: [CadMachineAvoidGroupsParameterValue](CadMachineAvoidGroupsParameterValue.md)  

## Description

Get the parameter object that the value is associated with.

## Syntax

"cadMachineAvoidGroupsParameterValue_var" is a variable referencing a CadMachineAvoidGroupsParameterValue object.  

```python
# Get the value of the property.
propertyValue = cadMachineAvoidGroupsParameterValue_var.parent
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version September 2024
