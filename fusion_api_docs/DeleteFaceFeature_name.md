# DeleteFaceFeature.name Property

Parent Object: [DeleteFaceFeature](DeleteFaceFeature.md)  

## Description

Returns the name of the feature as seen in the browser (non-parametric) or in the timeline (parametric).

## Syntax

"deleteFaceFeature_var" is a variable referencing a DeleteFaceFeature object.  

```python
# Get the value of the property.
propertyValue = deleteFaceFeature_var.name

# Set the value of the property.
deleteFaceFeature_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version August 2016
