# SphereFeature.attributes Property

Parent Object: [SphereFeature](SphereFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"sphereFeature_var" is a variable referencing a SphereFeature object.  

```python
# Get the value of the property.
propertyValue = sphereFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
