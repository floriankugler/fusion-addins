# Joint.attributes Property

Parent Object: [Joint](Joint.md)  

## Description

Returns the collection of attributes associated with this joint.

## Syntax

"joint_var" is a variable referencing a Joint object.  

```python
# Get the value of the property.
propertyValue = joint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
