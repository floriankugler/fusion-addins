# SketchOffsetCurvesDimension.parentSketch Property

Parent Object: [SketchOffsetCurvesDimension](SketchOffsetCurvesDimension.md)  

## Description

Returns the parent sketch object.

## Syntax

"sketchOffsetCurvesDimension_var" is a variable referencing a SketchOffsetCurvesDimension object.  

```python
# Get the value of the property.
propertyValue = sketchOffsetCurvesDimension_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version March 2016
