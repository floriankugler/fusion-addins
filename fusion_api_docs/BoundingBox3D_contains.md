# BoundingBox3D.contains Method

Parent Object: [BoundingBox3D](BoundingBox3D.md)  

## Description

Determines if the specified point is within the bound box.

## Syntax

"boundingBox3D_var" is a variable referencing a [BoundingBox3D](BoundingBox3D.md) object.

```python
returnValue = boundingBox3D_var.contains(point)
```

## Return Value

| Type    | Description                                           |
|---------|-------------------------------------------------------|
| boolean | Returns true if the point is within the bounding box. |

## Parameters

| Name | Type | Description |
|----|----|----|
| point | [Point3D](Point3D.md) | The point you want to check to see if it's in the bounding box. |

## Version

Introduced in version August 2014
