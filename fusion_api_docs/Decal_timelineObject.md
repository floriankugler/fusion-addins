# Decal.timelineObject Property

Parent Object: [Decal](Decal.md)  

## Description

Returns the timeline object associated with the creation of this decal.

## Syntax

"decal_var" is a variable referencing a Decal object.  

```python
# Get the value of the property.
propertyValue = decal_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version September 2024
