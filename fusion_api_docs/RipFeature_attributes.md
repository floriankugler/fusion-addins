# RipFeature.attributes Property

Parent Object: [RipFeature](RipFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"ripFeature_var" is a variable referencing a RipFeature object.  

```python
# Get the value of the property.
propertyValue = ripFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version September 2023
