# SketchCircle.isFullyConstrained Property

Parent Object: [SketchCircle](SketchCircle.md)  

## Description

Indicates if this sketch entity is fully constrained.

## Syntax

"sketchCircle_var" is a variable referencing a SketchCircle object.  

```python
# Get the value of the property.
propertyValue = sketchCircle_var.isFullyConstrained
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2016
