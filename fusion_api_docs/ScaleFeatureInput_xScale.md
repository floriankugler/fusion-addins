# ScaleFeatureInput.xScale Property

Parent Object: [ScaleFeatureInput](ScaleFeatureInput.md)  

## Description

Gets the scale in X direction.

## Syntax

"scaleFeatureInput_var" is a variable referencing a ScaleFeatureInput object.  

```python
# Get the value of the property.
propertyValue = scaleFeatureInput_var.xScale
```

## Property Value

This is a read only property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version January 2015
