# MultiLineTextDefinition.verticalAlignment Property

Parent Object: [MultiLineTextDefinition](MultiLineTextDefinition.md)  

## Description

Gets and sets the vertical alignment of the text with respect to the text rectangle.

## Syntax

"multiLineTextDefinition_var" is a variable referencing a MultiLineTextDefinition object.  

```python
# Get the value of the property.
propertyValue = multiLineTextDefinition_var.verticalAlignment

# Set the value of the property.
multiLineTextDefinition_var.verticalAlignment = propertyValue
```

## Property Value

This is a read/write property whose value is a [VerticalAlignments](VerticalAlignments.md).

## Version

Introduced in version December 2020
