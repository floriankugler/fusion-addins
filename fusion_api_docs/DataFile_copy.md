# DataFile.copy Method

Parent Object: [DataFile](DataFile.md)  

## Description

Copies this DataFile to the specified folder.

## Syntax

"dataFile_var" is a variable referencing a [DataFile](DataFile.md) object.

```python
returnValue = dataFile_var.copy(targetFolder)
```

## Return Value

| Type | Description |
|----|----|
| [DataFile](DataFile.md) | Returns the copied DataFile if the copy was successful. |

## Parameters

| Name | Type | Description |
|----|----|----|
| targetFolder | [DataFolder](DataFolder.md) | The folder to copy this DataFile to. |

## Version

Introduced in version September 2016
