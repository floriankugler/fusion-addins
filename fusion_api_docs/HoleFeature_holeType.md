# HoleFeature.holeType Property

Parent Object: [HoleFeature](HoleFeature.md)  

## Description

Returns the current type of hole this feature represents.

## Syntax

"holeFeature_var" is a variable referencing a HoleFeature object.  

```python
# Get the value of the property.
propertyValue = holeFeature_var.holeType
```

## Property Value

This is a read only property whose value is a [HoleTypes](HoleTypes.md).

## Version

Introduced in version August 2014
