# CAMFolder.strategy Property

Parent Object: [CAMFolder](CAMFolder.md)  

## Description

Gets the name of the strategy associated with this operation.

## Syntax

"cAMFolder_var" is a variable referencing a CAMFolder object.  

```python
# Get the value of the property.
propertyValue = cAMFolder_var.strategy
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version May 2022
