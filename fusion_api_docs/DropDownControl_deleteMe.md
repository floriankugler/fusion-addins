# DropDownControl.deleteMe Method

Parent Object: [DropDownControl](DropDownControl.md)  

## Description

Deletes the ToolbarControl

## Syntax

"dropDownControl_var" is a variable referencing a [DropDownControl](DropDownControl.md) object.

```python
returnValue = dropDownControl_var.deleteMe()
```

## Return Value

| Type    | Description                                                  |
|---------|--------------------------------------------------------------|
| boolean | Returns a boolean indicating if the deletion was successful. |

## Version

Introduced in version August 2014
