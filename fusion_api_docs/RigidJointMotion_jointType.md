# RigidJointMotion.jointType Property

Parent Object: [RigidJointMotion](RigidJointMotion.md)  

## Description

Returns an enum value indicating the type of joint this joint represents.

## Syntax

"rigidJointMotion_var" is a variable referencing a RigidJointMotion object.  

```python
# Get the value of the property.
propertyValue = rigidJointMotion_var.jointType
```

## Property Value

This is a read only property whose value is a [JointTypes](JointTypes.md).

## Version

Introduced in version July 2015
