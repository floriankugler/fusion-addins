# ReverseNormalFeature.attributes Property

Parent Object: [ReverseNormalFeature](ReverseNormalFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"reverseNormalFeature_var" is a variable referencing a ReverseNormalFeature object.  

```python
# Get the value of the property.
propertyValue = reverseNormalFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
