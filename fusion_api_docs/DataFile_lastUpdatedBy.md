# DataFile.lastUpdatedBy Property

Parent Object: [DataFile](DataFile.md)  

## Description

Returns the User that last updated this data file

## Syntax

"dataFile_var" is a variable referencing a DataFile object.  

```python
# Get the value of the property.
propertyValue = dataFile_var.lastUpdatedBy
```

## Property Value

This is a read only property whose value is a [User](User.md).

## Version

Introduced in version September 2016
