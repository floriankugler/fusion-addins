# Viewport.fit Method

Parent Object: [Viewport](Viewport.md)  

## Description

Forces a camera change so that all of the graphics are visible in the viewport.

## Syntax

"viewport_var" is a variable referencing a [Viewport](Viewport.md) object.

```python
returnValue = viewport_var.fit()
```

## Return Value

| Type    | Description                             |
|---------|-----------------------------------------|
| boolean | Returns true if the fit was successful. |

## Version

Introduced in version August 2014
