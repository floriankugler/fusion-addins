# UnitAndValuePreferences.degreeDisplayFormat Property

Parent Object: [UnitAndValuePreferences](UnitAndValuePreferences.md)  

## Description

Gets and sets the degree display format.

## Syntax

"unitAndValuePreferences_var" is a variable referencing a UnitAndValuePreferences object.  

```python
# Get the value of the property.
propertyValue = unitAndValuePreferences_var.degreeDisplayFormat

# Set the value of the property.
unitAndValuePreferences_var.degreeDisplayFormat = propertyValue
```

## Property Value

This is a read/write property whose value is a [DegreeDisplayFormats](DegreeDisplayFormats.md).

## Version

Introduced in version August 2014
