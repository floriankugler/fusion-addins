# IntegerSliderCommandInput.valueOne Property

Parent Object: [IntegerSliderCommandInput](IntegerSliderCommandInput.md)  

## Description

Gets or sets the value associated with the first slider.

## Syntax

"integerSliderCommandInput_var" is a variable referencing an IntegerSliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = integerSliderCommandInput_var.valueOne

# Set the value of the property.
integerSliderCommandInput_var.valueOne = propertyValue
```

## Property Value

This is a read/write property whose value is an integer.

## Version

Introduced in version June 2015
