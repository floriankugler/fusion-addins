# BRepFaceDefinition.surfaceGeometry Property

Parent Object: [BRepFaceDefinition](BRepFaceDefinition.md)  

## Description

Gets and sets the surface geometry associated with this face definition.

## Syntax

"bRepFaceDefinition_var" is a variable referencing a BRepFaceDefinition object.  

```python
# Get the value of the property.
propertyValue = bRepFaceDefinition_var.surfaceGeometry

# Set the value of the property.
bRepFaceDefinition_var.surfaceGeometry = propertyValue
```

## Property Value

This is a read/write property whose value is a [Surface](Surface.md).

## Version

Introduced in version September 2020
