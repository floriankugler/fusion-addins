# StringProperty.isReadOnly Property

Parent Object: [StringProperty](StringProperty.md)  

## Description

Indicates if this property is read-only. If True any attempted edits will fail.

## Syntax

"stringProperty_var" is a variable referencing a StringProperty object.  

```python
# Get the value of the property.
propertyValue = stringProperty_var.isReadOnly
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
