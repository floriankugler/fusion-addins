# FlatPatternProduct.appearances Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Returns the appearances contained in this document.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.appearances
```

## Property Value

This is a read only property whose value is an [Appearances](Appearances.md).

## Version

Introduced in version October 2022
