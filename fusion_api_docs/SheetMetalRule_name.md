# SheetMetalRule.name Property

Parent Object: [SheetMetalRule](SheetMetalRule.md)  

## Description

The name of the sheet metal rule. When setting the name, it should be unique with respect to other sheet metal rules in the design or library.

## Syntax

"sheetMetalRule_var" is a variable referencing a SheetMetalRule object.  

```python
# Get the value of the property.
propertyValue = sheetMetalRule_var.name

# Set the value of the property.
sheetMetalRule_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version November 2022
