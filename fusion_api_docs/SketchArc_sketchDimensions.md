# SketchArc.sketchDimensions Property

Parent Object: [SketchArc](SketchArc.md)  

## Description

Returns the sketch dimensions that are attached to this curve.

## Syntax

"sketchArc_var" is a variable referencing a SketchArc object.  

```python
# Get the value of the property.
propertyValue = sketchArc_var.sketchDimensions
```

## Property Value

This is a read only property whose value is a [SketchDimensionList](SketchDimensionList.md).

## Version

Introduced in version August 2014
