# BRepLumpDefinitions.add Method

Parent Object: [BRepLumpDefinitions](BRepLumpDefinitions.md)  

## Description

Creates a new empty BRepLumpDefinition associated with the parent BRepBodyDefinition object.

## Syntax

"bRepLumpDefinitions_var" is a variable referencing a [BRepLumpDefinitions](BRepLumpDefinitions.md) object.

```python
returnValue = bRepLumpDefinitions_var.add()
```

## Return Value

| Type | Description |
|----|----|
| [BRepLumpDefinition](BRepLumpDefinition.md) | Returns the newly created BRepLumpDefinition object. |
