# BRepEdgeDefinition.startVertex Property

Parent Object: [BRepEdgeDefinition](BRepEdgeDefinition.md)  

## Description

Gets and sets the start vertex of the edge definition.

## Syntax

"bRepEdgeDefinition_var" is a variable referencing a BRepEdgeDefinition object.  

```python
# Get the value of the property.
propertyValue = bRepEdgeDefinition_var.startVertex

# Set the value of the property.
bRepEdgeDefinition_var.startVertex = propertyValue
```

## Property Value

This is a read/write property whose value is a [BRepVertexDefinition](BRepVertexDefinition.md).

## Version

Introduced in version September 2020
