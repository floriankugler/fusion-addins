# RectangularPatternFeature.deleteMe Method

Parent Object: [RectangularPatternFeature](RectangularPatternFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"rectangularPatternFeature_var" is a variable referencing a [RectangularPatternFeature](RectangularPatternFeature.md) object.

```python
returnValue = rectangularPatternFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version November 2014
