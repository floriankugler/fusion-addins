# SeparatorControl.deleteMe Method

Parent Object: [SeparatorControl](SeparatorControl.md)  

## Description

Deletes the ToolbarControl

## Syntax

"separatorControl_var" is a variable referencing a [SeparatorControl](SeparatorControl.md) object.

```python
returnValue = separatorControl_var.deleteMe()
```

## Return Value

| Type    | Description                                                  |
|---------|--------------------------------------------------------------|
| boolean | Returns a boolean indicating if the deletion was successful. |

## Version

Introduced in version August 2014
