# SketchFixedSpline.deleteMe Method

Parent Object: [SketchFixedSpline](SketchFixedSpline.md)  

## Description

Deletes the entity from the sketch.

## Syntax

"sketchFixedSpline_var" is a variable referencing a [SketchFixedSpline](SketchFixedSpline.md) object.

```python
returnValue = sketchFixedSpline_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true is the delete was successful. |

## Version

Introduced in version August 2014
