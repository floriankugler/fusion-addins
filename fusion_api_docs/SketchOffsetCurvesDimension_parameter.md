# SketchOffsetCurvesDimension.parameter Property

Parent Object: [SketchOffsetCurvesDimension](SketchOffsetCurvesDimension.md)  

## Description

Returns the associated parameter or null if there is no associated parameter.

## Syntax

"sketchOffsetCurvesDimension_var" is a variable referencing a SketchOffsetCurvesDimension object.  

```python
# Get the value of the property.
propertyValue = sketchOffsetCurvesDimension_var.parameter
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version March 2016
