# NavigationEvent.name Property

Parent Object: [NavigationEvent](NavigationEvent.md)  

## Description

The name of the event - e.g. "DocumentOpening"

## Syntax

"navigationEvent_var" is a variable referencing a NavigationEvent object.  

```python
# Get the value of the property.
propertyValue = navigationEvent_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version March 2021
