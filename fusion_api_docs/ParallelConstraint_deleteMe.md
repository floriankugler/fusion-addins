# ParallelConstraint.deleteMe Method

Parent Object: [ParallelConstraint](ParallelConstraint.md)  

## Description

Deletes this constraint. The IsDeletable property can be used to determine if this constraint can be deleted.

## Syntax

"parallelConstraint_var" is a variable referencing a [ParallelConstraint](ParallelConstraint.md) object.

```python
returnValue = parallelConstraint_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
