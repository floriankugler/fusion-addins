# UnitAndValuePreferences.materialDisplayUnit Property

Parent Object: [UnitAndValuePreferences](UnitAndValuePreferences.md)  

## Description

Gets and sets the units types to use when displaying values.

## Syntax

"unitAndValuePreferences_var" is a variable referencing a UnitAndValuePreferences object.  

```python
# Get the value of the property.
propertyValue = unitAndValuePreferences_var.materialDisplayUnit

# Set the value of the property.
unitAndValuePreferences_var.materialDisplayUnit = propertyValue
```

## Property Value

This is a read/write property whose value is a [MaterialDisplayUnits](MaterialDisplayUnits.md).

## Version

Introduced in version August 2014
