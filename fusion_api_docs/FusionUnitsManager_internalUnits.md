# FusionUnitsManager.internalUnits Property

Parent Object: [FusionUnitsManager](FusionUnitsManager.md)  

## Description

Returns a string that represents internal units - i.e. "internalUnits". This can be used when performing conversions via Convert.

## Syntax

"fusionUnitsManager_var" is a variable referencing a FusionUnitsManager object.  

```python
# Get the value of the property.
propertyValue = fusionUnitsManager_var.internalUnits
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2014
