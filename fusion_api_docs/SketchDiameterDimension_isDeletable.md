# SketchDiameterDimension.isDeletable Property

Parent Object: [SketchDiameterDimension](SketchDiameterDimension.md)  

## Description

Indicates if this dimension is deletable.

## Syntax

"sketchDiameterDimension_var" is a variable referencing a SketchDiameterDimension object.  

```python
# Get the value of the property.
propertyValue = sketchDiameterDimension_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
