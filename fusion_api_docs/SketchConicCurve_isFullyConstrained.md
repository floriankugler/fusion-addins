# SketchConicCurve.isFullyConstrained Property

Parent Object: [SketchConicCurve](SketchConicCurve.md)  

## Description

Indicates if this sketch entity is fully constrained.

## Syntax

"sketchConicCurve_var" is a variable referencing a SketchConicCurve object.  

```python
# Get the value of the property.
propertyValue = sketchConicCurve_var.isFullyConstrained
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2016
