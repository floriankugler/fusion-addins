# BaseFeatures.add Method

Parent Object: [BaseFeatures](BaseFeatures.md)  

## Description

Creates a new empty base feature in the parent component.

## Syntax

"baseFeatures_var" is a variable referencing a [BaseFeatures](BaseFeatures.md) object.

```python
returnValue = baseFeatures_var.add()
```

## Return Value

| Type | Description |
|----|----|
| [BaseFeature](BaseFeature.md) | Returns the new BaseFeature or null in the case of an error. |
