# AlongPathTextDefinition.characterSpacing Property

Parent Object: [AlongPathTextDefinition](AlongPathTextDefinition.md)  

## Description

Gets and sets the spacing between the characters. This is an additional spacing to apply that is defined as a percentage of the default spacing. A spacing of 0 indicates no additional spacing. A spacing of 50 indicates to use the default plus 50% of the default.

## Syntax

"alongPathTextDefinition_var" is a variable referencing an AlongPathTextDefinition object.  

```python
# Get the value of the property.
propertyValue = alongPathTextDefinition_var.characterSpacing

# Set the value of the property.
alongPathTextDefinition_var.characterSpacing = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version January 2021
