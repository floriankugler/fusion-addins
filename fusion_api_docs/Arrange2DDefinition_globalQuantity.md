# Arrange2DDefinition.globalQuantity Property

Parent Object: [Arrange2DDefinition](Arrange2DDefinition.md)  

## Description

Returns the parameter that controls the global quantity. You can modify the value by using the properties on the returned ModelParameter object.

## Syntax

"arrange2DDefinition_var" is a variable referencing an Arrange2DDefinition object.  

```python
# Get the value of the property.
propertyValue = arrange2DDefinition_var.globalQuantity
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version January 2025
