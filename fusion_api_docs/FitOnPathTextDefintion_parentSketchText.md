# FitOnPathTextDefintion.parentSketchText Property

Parent Object: [FitOnPathTextDefintion](FitOnPathTextDefintion.md)  

## Description

Returns the SketchText object this definition is associated with. This property will return null in the case the definition object was obtained from a SketchTextInput object because the SketchText object does not yet exist.

## Syntax

"fitOnPathTextDefintion_var" is a variable referencing a FitOnPathTextDefintion object.  

```python
# Get the value of the property.
propertyValue = fitOnPathTextDefintion_var.parentSketchText
```

## Property Value

This is a read only property whose value is a [SketchText](SketchText.md).

## Version

Introduced in version July 2022
