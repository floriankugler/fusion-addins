# MachineAxisRange.createInfinite Method

Parent Object: [MachineAxisRange](MachineAxisRange.md)  

## Description

Creates a new unbounded range object.

## Syntax

This is a static method.  

```python

returnValue = adsk.cam.MachineAxisRange.createInfinite()
```

## Return Value

| Type                                     | Description         |
|------------------------------------------|---------------------|
| [MachineAxisRange](MachineAxisRange.md) | A new range object. |

## Version

Introduced in version April 2023
