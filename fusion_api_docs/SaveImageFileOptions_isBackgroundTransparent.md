# SaveImageFileOptions.isBackgroundTransparent Property

Parent Object: [SaveImageFileOptions](SaveImageFileOptions.md)  

## Description

Gets and sets if the background should be rendered as transparent. If false, the background will be the same as seen in Fusion.  

When the SaveImageFileOptions object is initially created, this is initialized to false.

## Syntax

"saveImageFileOptions_var" is a variable referencing a SaveImageFileOptions object.  

```python
# Get the value of the property.
propertyValue = saveImageFileOptions_var.isBackgroundTransparent

# Set the value of the property.
saveImageFileOptions_var.isBackgroundTransparent = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version May 2022
