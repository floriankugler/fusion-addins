# ChoiceProperty.id Property

Parent Object: [ChoiceProperty](ChoiceProperty.md)  

## Description

Returns the unique ID of this property.

## Syntax

"choiceProperty_var" is a variable referencing a ChoiceProperty object.  

```python
# Get the value of the property.
propertyValue = choiceProperty_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2014
