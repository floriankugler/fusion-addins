# SketchLine.isConstruction Property

Parent Object: [SketchLine](SketchLine.md)  

## Description

Gets and sets whether this curve is construction geometry.

## Syntax

"sketchLine_var" is a variable referencing a SketchLine object.  

```python
# Get the value of the property.
propertyValue = sketchLine_var.isConstruction

# Set the value of the property.
sketchLine_var.isConstruction = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
