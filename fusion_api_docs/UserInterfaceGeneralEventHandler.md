# UserInterfaceGeneralEventHandler Object

Derived from: [EventHandler](EventHandler.md) Object  

## Description

The UserInterfaceGeneralEventHandler is a client implemented class that can be added as a handler to a UserInterfaceGeneralEvent event and which is triggered when the event is fired.

## Methods

| Name | Description |
|----|----|
| [notify](UserInterfaceGeneralEventHandler_notify.md) | The function called by Fusion when the associated event is fired. |

## Version

Introduced in version March 2017
