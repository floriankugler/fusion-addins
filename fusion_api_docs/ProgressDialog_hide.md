# ProgressDialog.hide Method

Parent Object: [ProgressDialog](ProgressDialog.md)  

## Description

Hides the progress dialog. This should be used when the process has completed.

## Syntax

"progressDialog_var" is a variable referencing a [ProgressDialog](ProgressDialog.md) object.

```python
returnValue = progressDialog_var.hide()
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |
