# CustomGraphicsGroups.add Method

Parent Object: [CustomGraphicsGroups](CustomGraphicsGroups.md)  

## Description

Creates a new, empty CustomGraphicsGroup.

## Syntax

"customGraphicsGroups_var" is a variable referencing a [CustomGraphicsGroups](CustomGraphicsGroups.md) object.

```python
returnValue = customGraphicsGroups_var.add()
```

## Return Value

| Type | Description |
|----|----|
| [CustomGraphicsGroup](CustomGraphicsGroup.md) | Returns the new CustomGraphicsGroup object or null in the case of a failure. |
