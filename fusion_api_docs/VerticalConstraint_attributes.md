# VerticalConstraint.attributes Property

Parent Object: [VerticalConstraint](VerticalConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"verticalConstraint_var" is a variable referencing a VerticalConstraint object.  

```python
# Get the value of the property.
propertyValue = verticalConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
