# Features.shellFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the shell features within the component and supports the creation of new shell features.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.shellFeatures
```

## Property Value

This is a read only property whose value is a [ShellFeatures](ShellFeatures.md).
