# BossFeatureSideInput.clearRibExtent Method

Parent Object: [BossFeatureSideInput](BossFeatureSideInput.md)  

## Description

Clears rib extent types for all position points.

## Syntax

"bossFeatureSideInput_var" is a variable referencing a [BossFeatureSideInput](BossFeatureSideInput.md) object.

```python
returnValue = bossFeatureSideInput_var.clearRibExtent()
```

## Version

Introduced in version October 2022
