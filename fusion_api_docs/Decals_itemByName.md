# Decals.itemByName Method

Parent Object: [Decals](Decals.md)  

## Description

Returns the specified decal using the name of the decal.

## Syntax

"decals_var" is a variable referencing a [Decals](Decals.md) object.

```python
returnValue = decals_var.itemByName(name)
```

## Return Value

| Type | Description |
|----|----|
| [Decal](Decal.md) | Returns the specified Decal object, if it exists. Otherwise it returns null. |

## Parameters

| Name | Type   | Description                                                |
|------|--------|------------------------------------------------------------|
| name | string | The name of the decal as seen in the browser and timeline. |

## Version

Introduced in version September 2024
