# FormFeatures.add Method

Parent Object: [FormFeatures](FormFeatures.md)  

## Description

Creates a new empty form feature in the parent component.

## Syntax

"formFeatures_var" is a variable referencing a [FormFeatures](FormFeatures.md) object.

```python
returnValue = formFeatures_var.add()
```

## Return Value

| Type | Description |
|----|----|
| [FormFeature](FormFeature.md) | Returns the new FormFeature or null in the case of an error. |
