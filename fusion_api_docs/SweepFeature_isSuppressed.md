# SweepFeature.isSuppressed Property

Parent Object: [SweepFeature](SweepFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"sweepFeature_var" is a variable referencing a SweepFeature object.  

```python
# Get the value of the property.
propertyValue = sweepFeature_var.isSuppressed

# Set the value of the property.
sweepFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version November 2014
