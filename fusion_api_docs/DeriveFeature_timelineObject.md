# DeriveFeature.timelineObject Property

Parent Object: [DeriveFeature](DeriveFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"deriveFeature_var" is a variable referencing a DeriveFeature object.  

```python
# Get the value of the property.
propertyValue = deriveFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version January 2026
