# Preferences.gridPreferences Property

Parent Object: [Preferences](Preferences.md)  

## Description

Gets the GridPreferences object.

## Syntax

"preferences_var" is a variable referencing a Preferences object.  

```python
# Get the value of the property.
propertyValue = preferences_var.gridPreferences
```

## Property Value

This is a read only property whose value is a [GridPreferences](GridPreferences.md).

## Version

Introduced in version August 2014
