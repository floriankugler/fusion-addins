# ConstructionAxisDefinition.parentConstructionAxis Property

Parent Object: [ConstructionAxisDefinition](ConstructionAxisDefinition.md)  

## Description

Returns the ConstructionAxis object

## Syntax

"constructionAxisDefinition_var" is a variable referencing a ConstructionAxisDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionAxisDefinition_var.parentConstructionAxis
```

## Property Value

This is a read only property whose value is a [ConstructionAxis](ConstructionAxis.md).

## Version

Introduced in version August 2014
