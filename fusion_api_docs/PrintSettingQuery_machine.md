# PrintSettingQuery.machine Property

Parent Object: [PrintSettingQuery](PrintSettingQuery.md)  

## Description

The machine specifies which machine the found print setting are compatible with.

## Syntax

"printSettingQuery_var" is a variable referencing a PrintSettingQuery object.  

```python
# Get the value of the property.
propertyValue = printSettingQuery_var.machine

# Set the value of the property.
printSettingQuery_var.machine = propertyValue
```

## Property Value

This is a read/write property whose value is a [Machine](Machine.md).

## Version

Introduced in version January 2025
