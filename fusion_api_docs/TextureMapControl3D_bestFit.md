# TextureMapControl3D.bestFit Method

Parent Object: [TextureMapControl3D](TextureMapControl3D.md)  

## Description

Reorients the transform to best fit the geometry of the body.

## Syntax

"textureMapControl3D_var" is a variable referencing a [TextureMapControl3D](TextureMapControl3D.md) object.

```python
returnValue = textureMapControl3D_var.bestFit()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns true if the best fit was successful. |

## Version

Introduced in version March 2022
