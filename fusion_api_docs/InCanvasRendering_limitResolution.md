# InCanvasRendering.limitResolution Property

Parent Object: [InCanvasRendering](InCanvasRendering.md)  

## Description

Sets the percentage of the full resolution to render the image. Valid values are between 20 and 100 inclusive. 100 is full resolution (100%).

## Syntax

"inCanvasRendering_var" is a variable referencing an InCanvasRendering object.  

```python
# Get the value of the property.
propertyValue = inCanvasRendering_var.limitResolution

# Set the value of the property.
inCanvasRendering_var.limitResolution = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version September 2023
