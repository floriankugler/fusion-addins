# Sketch.healthState Property

Parent Object: [Sketch](Sketch.md)  

## Description

Returns the current health state of this sketch.

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).
