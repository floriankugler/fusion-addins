# EqualConstraint.deleteMe Method

Parent Object: [EqualConstraint](EqualConstraint.md)  

## Description

Deletes this constraint. The IsDeletable property can be used to determine if this constraint can be deleted.

## Syntax

"equalConstraint_var" is a variable referencing an [EqualConstraint](EqualConstraint.md) object.

```python
returnValue = equalConstraint_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
