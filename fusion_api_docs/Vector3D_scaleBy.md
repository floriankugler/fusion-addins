# Vector3D.scaleBy Method

Parent Object: [Vector3D](Vector3D.md)  

## Description

Scale this vector by the specified product.

## Syntax

"vector3D_var" is a variable referencing a [Vector3D](Vector3D.md) object.

```python
returnValue = vector3D_var.scaleBy(scale)
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name  | Type   | Description      |
|-------|--------|------------------|
| scale | double | The scale value. |

## Version

Introduced in version August 2014
