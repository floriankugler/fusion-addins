# MinimumRadiusAnalysis.deleteMe Method

Parent Object: [MinimumRadiusAnalysis](MinimumRadiusAnalysis.md)  

## Description

A method that deletes this Analysis.

## Syntax

"minimumRadiusAnalysis_var" is a variable referencing a [MinimumRadiusAnalysis](MinimumRadiusAnalysis.md) object.

```python
returnValue = minimumRadiusAnalysis_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version January 2023
