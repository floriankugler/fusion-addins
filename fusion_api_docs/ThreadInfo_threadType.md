# ThreadInfo.threadType Property

Parent Object: [ThreadInfo](ThreadInfo.md)  

## Description

Returns and sets the string that defines the thread type.

## Syntax

"threadInfo_var" is a variable referencing a ThreadInfo object.  

```python
# Get the value of the property.
propertyValue = threadInfo_var.threadType

# Set the value of the property.
threadInfo_var.threadType = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version January 2015
