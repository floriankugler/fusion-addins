# BossFeature.timelineObject Property

Parent Object: [BossFeature](BossFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"bossFeature_var" is a variable referencing a BossFeature object.  

```python
# Get the value of the property.
propertyValue = bossFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version October 2022
