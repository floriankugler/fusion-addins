# Profile.boundingBox Property

Parent Object: [Profile](Profile.md)  

## Description

Returns the 3D bounding box of the profile in sketch space.

## Syntax

"profile_var" is a variable referencing a Profile object.  

```python
# Get the value of the property.
propertyValue = profile_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version August 2014
