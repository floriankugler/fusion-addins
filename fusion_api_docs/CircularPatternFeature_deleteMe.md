# CircularPatternFeature.deleteMe Method

Parent Object: [CircularPatternFeature](CircularPatternFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"circularPatternFeature_var" is a variable referencing a [CircularPatternFeature](CircularPatternFeature.md) object.

```python
returnValue = circularPatternFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version November 2014
