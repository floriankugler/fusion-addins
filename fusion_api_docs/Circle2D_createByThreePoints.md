# Circle2D.createByThreePoints Method

Parent Object: [Circle2D](Circle2D.md)  

## Description

Creates a transient 2D circle through three points.

## Syntax

This is a static method.  

```python

returnValue = adsk.core.Circle2D.createByThreePoints(pointOne, pointTwo, pointThree)
```

## Return Value

| Type | Description |
|----|----|
| [Circle2D](Circle2D.md) | Returns the new Circle2D object or null if the creation failed. |

## Parameters

| Name       | Type                   | Description                     |
|------------|------------------------|---------------------------------|
| pointOne   | [Point2D](Point2D.md) | The first point on the circle.  |
| pointTwo   | [Point2D](Point2D.md) | The second point on the circle. |
| pointThree | [Point2D](Point2D.md) | The third point on the circle.  |

## Version

Introduced in version August 2014
