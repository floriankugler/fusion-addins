# BRepFace.attributes Property

Parent Object: [BRepFace](BRepFace.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"bRepFace_var" is a variable referencing a BRepFace object.  

```python
# Get the value of the property.
propertyValue = bRepFace_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
