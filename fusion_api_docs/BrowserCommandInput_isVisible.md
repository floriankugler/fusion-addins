# BrowserCommandInput.isVisible Property

Parent Object: [BrowserCommandInput](BrowserCommandInput.md)  

## Description

Gets or sets if this input will be visible to the user.  

Setting a SelectionCommandInput to be invisible will clear any selections it currently has.

## Syntax

"browserCommandInput_var" is a variable referencing a BrowserCommandInput object.  

```python
# Get the value of the property.
propertyValue = browserCommandInput_var.isVisible

# Set the value of the property.
browserCommandInput_var.isVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version July 2021
