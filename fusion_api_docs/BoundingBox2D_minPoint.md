# BoundingBox2D.minPoint Property

Parent Object: [BoundingBox2D](BoundingBox2D.md)  

## Description

Gets and sets the minimum point of the box.

## Syntax

"boundingBox2D_var" is a variable referencing a BoundingBox2D object.  

```python
# Get the value of the property.
propertyValue = boundingBox2D_var.minPoint

# Set the value of the property.
boundingBox2D_var.minPoint = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point2D](Point2D.md).

## Version

Introduced in version August 2014
