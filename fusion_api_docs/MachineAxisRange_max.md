# MachineAxisRange.max Property

Parent Object: [MachineAxisRange](MachineAxisRange.md)  

## Description

Maximum value of range Type depends on where this range is being used. Centimeters are used for distances and radians for angles. Returns infinity if this range is infinite.

## Syntax

"machineAxisRange_var" is a variable referencing a MachineAxisRange object.  

```python
# Get the value of the property.
propertyValue = machineAxisRange_var.max
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version April 2023
