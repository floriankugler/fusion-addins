# OperationBase.parentSetup Property

Parent Object: [OperationBase](OperationBase.md)  

## Description

Gets the Setup this operation belongs to.

## Syntax

"operationBase_var" is a variable referencing an OperationBase object.  

```python
# Get the value of the property.
propertyValue = operationBase_var.parentSetup
```

## Property Value

This is a read only property whose value is a [Setup](Setup.md).

## Version

Introduced in version January 2016
