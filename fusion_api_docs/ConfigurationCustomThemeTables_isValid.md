# ConfigurationCustomThemeTables.isValid Property

Parent Object: [ConfigurationCustomThemeTables](ConfigurationCustomThemeTables.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"configurationCustomThemeTables_var" is a variable referencing a ConfigurationCustomThemeTables object.  

```python
# Get the value of the property.
propertyValue = configurationCustomThemeTables_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2024
