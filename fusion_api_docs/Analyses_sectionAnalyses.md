# Analyses.sectionAnalyses Property

Parent Object: [Analyses](Analyses.md)  

## Description

Returns the SectionAnalyses object, which provides access to any existing SectionAnalysis objects in the design.

## Syntax

"analyses_var" is a variable referencing an Analyses object.  

```python
# Get the value of the property.
propertyValue = analyses_var.sectionAnalyses
```

## Property Value

This is a read only property whose value is a [SectionAnalyses](SectionAnalyses.md).

## Version

Introduced in version January 2023
