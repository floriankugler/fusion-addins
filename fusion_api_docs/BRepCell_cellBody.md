# BRepCell.cellBody Property

Parent Object: [BRepCell](BRepCell.md)  

## Description

Returns a BRepBody that represents this cell. This is a transient B-Rep body.

## Syntax

"bRepCell_var" is a variable referencing a BRepCell object.  

```python
# Get the value of the property.
propertyValue = bRepCell_var.cellBody
```

## Property Value

This is a read only property whose value is a [BRepBody](BRepBody.md).

## Version

Introduced in version June 2015
