# MachinePart.deleteMe Method

Parent Object: [MachinePart](MachinePart.md)  

## Description

Delete this part and its children from the kinematics tree.

## Syntax

"machinePart_var" is a variable referencing a [MachinePart](MachinePart.md) object.

```python
returnValue = machinePart_var.deleteMe()
```

## Version

Introduced in version April 2023
