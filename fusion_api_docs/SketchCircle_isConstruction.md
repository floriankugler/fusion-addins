# SketchCircle.isConstruction Property

Parent Object: [SketchCircle](SketchCircle.md)  

## Description

Gets and sets whether this curve is construction geometry.

## Syntax

"sketchCircle_var" is a variable referencing a SketchCircle object.  

```python
# Get the value of the property.
propertyValue = sketchCircle_var.isConstruction

# Set the value of the property.
sketchCircle_var.isConstruction = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
