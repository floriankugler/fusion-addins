# RemoveFeature.deleteMe Method

Parent Object: [RemoveFeature](RemoveFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"removeFeature_var" is a variable referencing a [RemoveFeature](RemoveFeature.md) object.

```python
returnValue = removeFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version September 2015
