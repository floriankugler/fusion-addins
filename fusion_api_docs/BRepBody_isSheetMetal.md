# BRepBody.isSheetMetal Property

Parent Object: [BRepBody](BRepBody.md)  

## Description

Indicates if this body represents a sheet metal folded part or not and if a flat pattern can be created.

## Syntax

"bRepBody_var" is a variable referencing a BRepBody object.  

```python
# Get the value of the property.
propertyValue = bRepBody_var.isSheetMetal
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version October 2022
