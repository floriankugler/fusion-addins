# PostLibrary.assetTypeName Property

Parent Object: [PostLibrary](PostLibrary.md)  

## Description

Get the name of the asset type which can be accessed by the library.

## Syntax

"postLibrary_var" is a variable referencing a PostLibrary object.  

```python
# Get the value of the property.
propertyValue = postLibrary_var.assetTypeName
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
