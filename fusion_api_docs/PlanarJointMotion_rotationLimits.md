# PlanarJointMotion.rotationLimits Property

Parent Object: [PlanarJointMotion](PlanarJointMotion.md)  

## Description

Returns a JointLimits object that defines the limits of rotation for this joint. Use the functionality of the returned JointLimits object to get, set, and modify the joint limits.

## Syntax

"planarJointMotion_var" is a variable referencing a PlanarJointMotion object.  

```python
# Get the value of the property.
propertyValue = planarJointMotion_var.rotationLimits
```

## Property Value

This is a read only property whose value is a [JointLimits](JointLimits.md).
