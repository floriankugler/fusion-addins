# NamedValues.create Method

Parent Object: [NamedValues](NamedValues.md)  

## Description

Creates a transient NamedValues object.

## Syntax

This is a static method.  

```python

returnValue = adsk.core.NamedValues.create()
```

## Return Value

| Type                           | Description                            |
|--------------------------------|----------------------------------------|
| [NamedValues](NamedValues.md) | Returns the created NamedValue object. |

## Version

Introduced in version August 2014
