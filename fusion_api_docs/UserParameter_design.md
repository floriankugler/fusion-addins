# UserParameter.design Property

Parent Object: [UserParameter](UserParameter.md)  

## Description

Returns the Design containing the UserParameter.

## Syntax

"userParameter_var" is a variable referencing a UserParameter object.  

```python
# Get the value of the property.
propertyValue = userParameter_var.design
```

## Property Value

This is a read only property whose value is a [Design](Design.md).

## Version

Introduced in version August 2014
