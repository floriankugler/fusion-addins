# ClearanceHoleInfo.standard Property

Parent Object: [ClearanceHoleInfo](ClearanceHoleInfo.md)  

## Description

Returns the string that defines the standard. This is typically obtained by using the ClearanceHoleDataQuery object.

## Syntax

"clearanceHoleInfo_var" is a variable referencing a ClearanceHoleInfo object.  

```python
# Get the value of the property.
propertyValue = clearanceHoleInfo_var.standard
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version September 2025
