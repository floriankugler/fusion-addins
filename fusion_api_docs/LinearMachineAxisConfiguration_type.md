# LinearMachineAxisConfiguration.type Property

Parent Object: [LinearMachineAxisConfiguration](LinearMachineAxisConfiguration.md)  

## Description

The type of this axis configuration. Use this to inform a cast to the derived types.

## Syntax

"linearMachineAxisConfiguration_var" is a variable referencing a LinearMachineAxisConfiguration object.  

```python
# Get the value of the property.
propertyValue = linearMachineAxisConfiguration_var.type
```

## Property Value

This is a read only property whose value is a [MachineAxisTypes](MachineAxisTypes.md).

## Version

Introduced in version April 2023
