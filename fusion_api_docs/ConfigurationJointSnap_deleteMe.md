# ConfigurationJointSnap.deleteMe Method

Parent Object: [ConfigurationJointSnap](ConfigurationJointSnap.md)  

## Description

Deletes this joint snap.

## Syntax

"configurationJointSnap_var" is a variable referencing a [ConfigurationJointSnap](ConfigurationJointSnap.md) object.

```python
returnValue = configurationJointSnap_var.deleteMe()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns true if the deletion was successful. |

## Version

Introduced in version September 2024
