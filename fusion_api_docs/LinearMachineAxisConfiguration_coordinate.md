# LinearMachineAxisConfiguration.coordinate Property

Parent Object: [LinearMachineAxisConfiguration](LinearMachineAxisConfiguration.md)  

## Description

Coordinate to use for post processing.

## Syntax

"linearMachineAxisConfiguration_var" is a variable referencing a LinearMachineAxisConfiguration object.  

```python
# Get the value of the property.
propertyValue = linearMachineAxisConfiguration_var.coordinate

# Set the value of the property.
linearMachineAxisConfiguration_var.coordinate = propertyValue
```

## Property Value

This is a read/write property whose value is a [MachineAxisCoordinates](MachineAxisCoordinates.md).

## Version

Introduced in version April 2023
