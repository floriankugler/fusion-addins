# CAM.parentDocument Property

Parent Object: [CAM](CAM.md)  

## Description

Returns the parent Document object.

## Syntax

"cAM_var" is a variable referencing a CAM object.  

```python
# Get the value of the property.
propertyValue = cAM_var.parentDocument
```

## Property Value

This is a read only property whose value is a [Document](Document.md).

## Version

Introduced in version January 2016
