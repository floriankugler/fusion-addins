# EllipticalArc3D.center Property

Parent Object: [EllipticalArc3D](EllipticalArc3D.md)  

## Description

Gets and sets the center point of the elliptical arc.

## Syntax

"ellipticalArc3D_var" is a variable referencing an EllipticalArc3D object.  

```python
# Get the value of the property.
propertyValue = ellipticalArc3D_var.center

# Set the value of the property.
ellipticalArc3D_var.center = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
