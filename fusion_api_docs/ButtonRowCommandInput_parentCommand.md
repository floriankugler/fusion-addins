# ButtonRowCommandInput.parentCommand Property

Parent Object: [ButtonRowCommandInput](ButtonRowCommandInput.md)  

## Description

Gets the parent Command.

## Syntax

"buttonRowCommandInput_var" is a variable referencing a ButtonRowCommandInput object.  

```python
# Get the value of the property.
propertyValue = buttonRowCommandInput_var.parentCommand
```

## Property Value

This is a read only property whose value is a [Command](Command.md).

## Version

Introduced in version June 2015
