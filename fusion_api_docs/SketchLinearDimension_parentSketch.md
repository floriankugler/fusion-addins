# SketchLinearDimension.parentSketch Property

Parent Object: [SketchLinearDimension](SketchLinearDimension.md)  

## Description

Returns the parent sketch object.

## Syntax

"sketchLinearDimension_var" is a variable referencing a SketchLinearDimension object.  

```python
# Get the value of the property.
propertyValue = sketchLinearDimension_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
