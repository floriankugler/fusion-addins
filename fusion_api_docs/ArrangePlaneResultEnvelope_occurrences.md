# ArrangePlaneResultEnvelope.occurrences Property

Parent Object: [ArrangePlaneResultEnvelope](ArrangePlaneResultEnvelope.md)  

## Description

Returns a collection object of the occurrences in this envelope.

## Syntax

"arrangePlaneResultEnvelope_var" is a variable referencing an ArrangePlaneResultEnvelope object.  

```python
# Get the value of the property.
propertyValue = arrangePlaneResultEnvelope_var.occurrences
```

## Property Value

This is a read only property whose value is an [ArrangeOccurrenceResults](ArrangeOccurrenceResults.md).

## Version

Introduced in version January 2025
