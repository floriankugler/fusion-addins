# SketchArc.boundingBox Property

Parent Object: [SketchArc](SketchArc.md)  

## Description

Returns the bounding box of the entity in sketch space.

## Syntax

"sketchArc_var" is a variable referencing a SketchArc object.  

```python
# Get the value of the property.
propertyValue = sketchArc_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version August 2014
