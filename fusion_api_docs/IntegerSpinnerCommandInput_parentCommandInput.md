# IntegerSpinnerCommandInput.parentCommandInput Property

Parent Object: [IntegerSpinnerCommandInput](IntegerSpinnerCommandInput.md)  

## Description

Gets the parent CommandInput if this commandInput is the child of a TabCommandInput or GroupCommandInput. Returns null if there is no parent.

## Syntax

"integerSpinnerCommandInput_var" is a variable referencing an IntegerSpinnerCommandInput object.  

```python
# Get the value of the property.
propertyValue = integerSpinnerCommandInput_var.parentCommandInput
```

## Property Value

This is a read only property whose value is a [CommandInput](CommandInput.md).

## Version

Introduced in version July 2015
