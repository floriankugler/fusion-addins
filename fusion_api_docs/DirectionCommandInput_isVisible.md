# DirectionCommandInput.isVisible Property

Parent Object: [DirectionCommandInput](DirectionCommandInput.md)  

## Description

Gets or sets if this input will be visible to the user.  

Setting a SelectionCommandInput to be invisible will clear any selections it currently has.

## Syntax

"directionCommandInput_var" is a variable referencing a DirectionCommandInput object.  

```python
# Get the value of the property.
propertyValue = directionCommandInput_var.isVisible

# Set the value of the property.
directionCommandInput_var.isVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2016
