# AccessibilityAnalysis.attributes Property

Parent Object: [AccessibilityAnalysis](AccessibilityAnalysis.md)  

## Description

A property that returns the collection of attributes associated with this Analysis.

## Syntax

"accessibilityAnalysis_var" is a variable referencing an AccessibilityAnalysis object.  

```python
# Get the value of the property.
propertyValue = accessibilityAnalysis_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version January 2023
