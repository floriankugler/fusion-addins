# StitchFeatureInput.isValid Property

Parent Object: [StitchFeatureInput](StitchFeatureInput.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"stitchFeatureInput_var" is a variable referencing a StitchFeatureInput object.  

```python
# Get the value of the property.
propertyValue = stitchFeatureInput_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version June 2015
