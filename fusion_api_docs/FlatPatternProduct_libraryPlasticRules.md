# FlatPatternProduct.libraryPlasticRules Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Gets the collection of plastic rules in the plastic rule library.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.libraryPlasticRules
```

## Property Value

This is a read only property whose value is a [PlasticRules](PlasticRules.md).

## Version

Introduced in version January 2024
