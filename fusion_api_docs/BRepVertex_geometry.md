# BRepVertex.geometry Property

Parent Object: [BRepVertex](BRepVertex.md)  

## Description

Returns the underlying geometry point

## Syntax

"bRepVertex_var" is a variable referencing a BRepVertex object.  

```python
# Get the value of the property.
propertyValue = bRepVertex_var.geometry
```

## Property Value

This is a read only property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
