# Sketch.sketchPoints Property

Parent Object: [Sketch](Sketch.md)  

## Description

Returns the sketch points collection associated with this sketch. This provides access to the existing sketch points and supports the creation of new sketch points.

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.sketchPoints
```

## Property Value

This is a read only property whose value is a [SketchPoints](SketchPoints.md).
