# Point3D.asVector Method

Parent Object: [Point3D](Point3D.md)  

## Description

Defines a vector using the coordinates of the point.

## Syntax

"point3D_var" is a variable referencing a [Point3D](Point3D.md) object.

```python
returnValue = point3D_var.asVector()
```

## Return Value

| Type                     | Description                |
|--------------------------|----------------------------|
| [Vector3D](Vector3D.md) | Returns a Vector2D object. |

## Version

Introduced in version August 2014
