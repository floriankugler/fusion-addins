# CAM3MFExportOptions.metadata Property

Parent Object: [CAM3MFExportOptions](CAM3MFExportOptions.md)  

## Description

Class for setting the meta data options with in the 3mf export

## Syntax

"cAM3MFExportOptions_var" is a variable referencing a CAM3MFExportOptions object.  

```python
# Get the value of the property.
propertyValue = cAM3MFExportOptions_var.metadata
```

## Property Value

This is a read only property whose value is a [CAM3MFExportMetadataOptions](CAM3MFExportMetadataOptions.md).

## Version

Introduced in version May 2023
