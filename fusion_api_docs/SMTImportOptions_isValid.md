# SMTImportOptions.isValid Property

Parent Object: [SMTImportOptions](SMTImportOptions.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"sMTImportOptions_var" is a variable referencing a SMTImportOptions object.  

```python
# Get the value of the property.
propertyValue = sMTImportOptions_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2015
