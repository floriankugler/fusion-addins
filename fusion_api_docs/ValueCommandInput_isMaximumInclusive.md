# ValueCommandInput.isMaximumInclusive Property

Parent Object: [ValueCommandInput](ValueCommandInput.md)  

## Description

Gets and sets if the maximum value can be equal to the value defined by the maximumValue property or if it must be less than the maximum.  

When a new ValueCommandInput is created, this defaults to true.

## Syntax

"valueCommandInput_var" is a variable referencing a ValueCommandInput object.  

```python
# Get the value of the property.
propertyValue = valueCommandInput_var.isMaximumInclusive

# Set the value of the property.
valueCommandInput_var.isMaximumInclusive = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version November 2022
