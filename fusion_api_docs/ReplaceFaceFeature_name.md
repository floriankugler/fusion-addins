# ReplaceFaceFeature.name Property

Parent Object: [ReplaceFaceFeature](ReplaceFaceFeature.md)  

## Description

Returns the name of the feature as seen in the browser (non-parametric) or in the timeline (parametric).

## Syntax

"replaceFaceFeature_var" is a variable referencing a ReplaceFaceFeature object.  

```python
# Get the value of the property.
propertyValue = replaceFaceFeature_var.name

# Set the value of the property.
replaceFaceFeature_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version March 2015
