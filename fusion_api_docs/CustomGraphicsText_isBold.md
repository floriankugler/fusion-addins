# CustomGraphicsText.isBold Property

Parent Object: [CustomGraphicsText](CustomGraphicsText.md)  

## Description

Specifies that the text displays using a bold style. This is the default bold style and applies to all of text unless there is a style override defined within the text.

## Syntax

"customGraphicsText_var" is a variable referencing a CustomGraphicsText object.  

```python
# Get the value of the property.
propertyValue = customGraphicsText_var.isBold

# Set the value of the property.
customGraphicsText_var.isBold = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version April 2019
