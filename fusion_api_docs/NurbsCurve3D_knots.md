# NurbsCurve3D.knots Property

Parent Object: [NurbsCurve3D](NurbsCurve3D.md)  

## Description

Returns an array of numbers that define the knot vector of the curve.

## Syntax

"nurbsCurve3D_var" is a variable referencing a NurbsCurve3D object.  

```python
# Get the value of the property.
propertyValue = nurbsCurve3D_var.knots
```

## Property Value

This is a read only property whose value is an array of type double.

## Version

Introduced in version August 2014
