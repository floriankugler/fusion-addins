# CylinderFeature.parentComponent Property

Parent Object: [CylinderFeature](CylinderFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"cylinderFeature_var" is a variable referencing a CylinderFeature object.  

```python
# Get the value of the property.
propertyValue = cylinderFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version September 2015
