# CommandInput.deleteMe Method

Parent Object: [CommandInput](CommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"commandInput_var" is a variable referencing a [CommandInput](CommandInput.md) object.

```python
returnValue = commandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2020
