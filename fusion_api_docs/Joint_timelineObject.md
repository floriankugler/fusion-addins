# Joint.timelineObject Property

Parent Object: [Joint](Joint.md)  

## Description

Returns the timeline object associated with this joint.

## Syntax

"joint_var" is a variable referencing a Joint object.  

```python
# Get the value of the property.
propertyValue = joint_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version July 2015
