# RecognizedHole.isThreaded Property

Parent Object: [RecognizedHole](RecognizedHole.md)  

## Description

Returns true if at least one segment of this hole is threaded, i.e. associated with a thread feature.

## Syntax

"recognizedHole_var" is a variable referencing a RecognizedHole object.  

```python
# Get the value of the property.
propertyValue = recognizedHole_var.isThreaded
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version May 2023
