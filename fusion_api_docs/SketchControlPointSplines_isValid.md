# SketchControlPointSplines.isValid Property

Parent Object: [SketchControlPointSplines](SketchControlPointSplines.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"sketchControlPointSplines_var" is a variable referencing a SketchControlPointSplines object.  

```python
# Get the value of the property.
propertyValue = sketchControlPointSplines_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2022
