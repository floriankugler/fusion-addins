# Arc3D.evaluator Property

Parent Object: [Arc3D](Arc3D.md)  

## Description

Returns an evaluator object that lets you perform additional evaluations on the curve.

## Syntax

"arc3D_var" is a variable referencing an Arc3D object.  

```python
# Get the value of the property.
propertyValue = arc3D_var.evaluator
```

## Property Value

This is a read only property whose value is a [CurveEvaluator3D](CurveEvaluator3D.md).
