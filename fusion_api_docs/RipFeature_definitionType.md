# RipFeature.definitionType Property

Parent Object: [RipFeature](RipFeature.md)  

## Description

Gets the type of rip defined.

## Syntax

"ripFeature_var" is a variable referencing a RipFeature object.  

```python
# Get the value of the property.
propertyValue = ripFeature_var.definitionType
```

## Property Value

This is a read only property whose value is a [RipFeatureDefinitionTypes](RipFeatureDefinitionTypes.md).

## Version

Introduced in version September 2023
