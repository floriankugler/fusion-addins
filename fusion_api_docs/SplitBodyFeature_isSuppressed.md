# SplitBodyFeature.isSuppressed Property

Parent Object: [SplitBodyFeature](SplitBodyFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"splitBodyFeature_var" is a variable referencing a SplitBodyFeature object.  

```python
# Get the value of the property.
propertyValue = splitBodyFeature_var.isSuppressed

# Set the value of the property.
splitBodyFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version June 2015
