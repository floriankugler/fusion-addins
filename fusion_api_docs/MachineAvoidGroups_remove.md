# MachineAvoidGroups.remove Method

Parent Object: [MachineAvoidGroups](MachineAvoidGroups.md)  

## Description

Function that removes the specified group object using an index in the collection.

## Syntax

"machineAvoidGroups_var" is a variable referencing a [MachineAvoidGroups](MachineAvoidGroups.md) object.

```python
returnValue = machineAvoidGroups_var.remove(index)
```

## Parameters

| Name | Type | Description |
|----|----|----|
| index | uinteger | The index of the item within the collection to remove. The first item in the collection has an index of 0. |

## Version

Introduced in version September 2024
