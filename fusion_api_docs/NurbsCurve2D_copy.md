# NurbsCurve2D.copy Method

Parent Object: [NurbsCurve2D](NurbsCurve2D.md)  

## Description

Creates and returns an independent copy of this NurbsCurve2D object.

## Syntax

"nurbsCurve2D_var" is a variable referencing a [NurbsCurve2D](NurbsCurve2D.md) object.

```python
returnValue = nurbsCurve2D_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [NurbsCurve2D](NurbsCurve2D.md) | Returns an independent copy of this NurbsCurve2D. |

## Version

Introduced in version August 2014
