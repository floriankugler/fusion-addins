# CoincidentConstraint.point Property

Parent Object: [CoincidentConstraint](CoincidentConstraint.md)  

## Description

Returns the sketch point that is constrained.

## Syntax

"coincidentConstraint_var" is a variable referencing a CoincidentConstraint object.  

```python
# Get the value of the property.
propertyValue = coincidentConstraint_var.point
```

## Property Value

This is a read only property whose value is a [SketchPoint](SketchPoint.md).

## Version

Introduced in version August 2014
