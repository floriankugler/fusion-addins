# ConstructionPlane.isLightBulbOn Property

Parent Object: [ConstructionPlane](ConstructionPlane.md)  

## Description

Indicates if the light bulb (as displayed in the browser) is on. A construction plane will only be visible if it's light bulb, and that of it's containing folder and parent component/s are also on.

## Syntax

"constructionPlane_var" is a variable referencing a ConstructionPlane object.  

```python
# Get the value of the property.
propertyValue = constructionPlane_var.isLightBulbOn

# Set the value of the property.
constructionPlane_var.isLightBulbOn = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
