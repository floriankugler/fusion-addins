# SetupChangeEventArgs.type Property

Parent Object: [SetupChangeEventArgs](SetupChangeEventArgs.md)  

## Description

The type specifies the actual change.

## Syntax

"setupChangeEventArgs_var" is a variable referencing a SetupChangeEventArgs object.  

```python
# Get the value of the property.
propertyValue = setupChangeEventArgs_var.type
```

## Property Value

This is a read only property whose value is a [SetupChangeEventType](SetupChangeEventType.md).

## Version

Introduced in version April 2023
