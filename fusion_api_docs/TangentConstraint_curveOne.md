# TangentConstraint.curveOne Property

Parent Object: [TangentConstraint](TangentConstraint.md)  

## Description

Returns the first curve.

## Syntax

"tangentConstraint_var" is a variable referencing a TangentConstraint object.  

```python
# Get the value of the property.
propertyValue = tangentConstraint_var.curveOne
```

## Property Value

This is a read only property whose value is a [SketchCurve](SketchCurve.md).

## Version

Introduced in version August 2014
