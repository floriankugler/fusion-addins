# SliderCommandInput.setText Method

Parent Object: [SliderCommandInput](SliderCommandInput.md)  

## Description

Sets the text of the slider. Both the left and the right text should be set.

## Syntax

"sliderCommandInput_var" is a variable referencing a [SliderCommandInput](SliderCommandInput.md) object.

```python
returnValue = sliderCommandInput_var.setText(left, right)
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name  | Type   | Description                                         |
|-------|--------|-----------------------------------------------------|
| left  | string | Indicates the text on the left side of the slider.  |
| right | string | Indicates the text on the right side of the slider. |

## Version

Introduced in version June 2015
