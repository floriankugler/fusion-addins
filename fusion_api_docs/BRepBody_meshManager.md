# BRepBody.meshManager Property

Parent Object: [BRepBody](BRepBody.md)  

## Description

Returns the mesh manager object for this body.

## Syntax

"bRepBody_var" is a variable referencing a BRepBody object.  

```python
# Get the value of the property.
propertyValue = bRepBody_var.meshManager
```

## Property Value

This is a read only property whose value is a [MeshManager](MeshManager.md).

## Version

Introduced in version August 2014
