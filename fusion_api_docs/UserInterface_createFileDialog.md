# UserInterface.createFileDialog Method

Parent Object: [UserInterface](UserInterface.md)  

## Description

Creates a new FileDialog object which provides the ability to show a standard file selection dialog to the user.

## Syntax

"userInterface_var" is a variable referencing a [UserInterface](UserInterface.md) object.

```python
returnValue = userInterface_var.createFileDialog()
```

## Return Value

| Type | Description |
|----|----|
| [FileDialog](FileDialog.md) | Returns the created FileDialog object that you can use to define the contents of and display a standard file dialog. |
