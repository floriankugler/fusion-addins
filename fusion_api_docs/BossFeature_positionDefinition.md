# BossFeature.positionDefinition Property

Parent Object: [BossFeature](BossFeature.md)  

## Description

Returns a BossPositionDefinition object that provides access to the information used to define the position of the boss feature.

## Syntax

"bossFeature_var" is a variable referencing a BossFeature object.  

```python
# Get the value of the property.
propertyValue = bossFeature_var.positionDefinition
```

## Property Value

This is a read only property whose value is a [BossPositionDefinition](BossPositionDefinition.md).

## Version

Introduced in version October 2022
