# ConfigurationInsertStandardDesignColumn.parentTable Property

Parent Object: [ConfigurationInsertStandardDesignColumn](ConfigurationInsertStandardDesignColumn.md)  

## Description

Returns the parent table this column is in.

## Syntax

"configurationInsertStandardDesignColumn_var" is a variable referencing a ConfigurationInsertStandardDesignColumn object.  

```python
# Get the value of the property.
propertyValue = configurationInsertStandardDesignColumn_var.parentTable
```

## Property Value

This is a read only property whose value is a [ConfigurationTable](ConfigurationTable.md).

## Version

Introduced in version September 2025
