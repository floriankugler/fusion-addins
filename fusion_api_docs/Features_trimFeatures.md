# Features.trimFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the Trim features within the component and supports the creation of new Trim features.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.trimFeatures
```

## Property Value

This is a read only property whose value is a [TrimFeatures](TrimFeatures.md).
