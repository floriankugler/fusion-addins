# ReplaceFaceFeature.attributes Property

Parent Object: [ReplaceFaceFeature](ReplaceFaceFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"replaceFaceFeature_var" is a variable referencing a ReplaceFaceFeature object.  

```python
# Get the value of the property.
propertyValue = replaceFaceFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
