# BoundingBox2D.intersects Method

Parent Object: [BoundingBox2D](BoundingBox2D.md)  

## Description

Test if this bounding box intersects with the specified bounding box.

## Syntax

"boundingBox2D_var" is a variable referencing a [BoundingBox2D](BoundingBox2D.md) object.

```python
returnValue = boundingBox2D_var.intersects(boundingBox)
```

## Return Value

| Type    | Description                                   |
|---------|-----------------------------------------------|
| boolean | Returns true if the bounding boxes intersect. |

## Parameters

| Name | Type | Description |
|----|----|----|
| boundingBox | [BoundingBox2D](BoundingBox2D.md) | The bounding box to test intersection with. |

## Version

Introduced in version August 2014
