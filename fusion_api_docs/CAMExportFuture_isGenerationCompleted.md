# CAMExportFuture.isGenerationCompleted Property

Parent Object: [CAMExportFuture](CAMExportFuture.md)  

## Description

Returns true if the export has finished generating.

## Syntax

"cAMExportFuture_var" is a variable referencing a CAMExportFuture object.  

```python
# Get the value of the property.
propertyValue = cAMExportFuture_var.isGenerationCompleted
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2024
