# NCProgram.hasError Property

Parent Object: [NCProgram](NCProgram.md)  

## Description

Gets if errors were encountered when generating the operation.

## Syntax

"nCProgram_var" is a variable referencing a NCProgram object.  

```python
# Get the value of the property.
propertyValue = nCProgram_var.hasError
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version April 2023
