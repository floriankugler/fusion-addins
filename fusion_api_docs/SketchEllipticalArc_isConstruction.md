# SketchEllipticalArc.isConstruction Property

Parent Object: [SketchEllipticalArc](SketchEllipticalArc.md)  

## Description

Gets and sets whether this curve is construction geometry.

## Syntax

"sketchEllipticalArc_var" is a variable referencing a SketchEllipticalArc object.  

```python
# Get the value of the property.
propertyValue = sketchEllipticalArc_var.isConstruction

# Set the value of the property.
sketchEllipticalArc_var.isConstruction = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
