# PipeFeatureInput.path Property

Parent Object: [PipeFeatureInput](PipeFeatureInput.md)  

## Description

Gets and sets the path to create the Pipe.

## Syntax

"pipeFeatureInput_var" is a variable referencing a PipeFeatureInput object.  

```python
# Get the value of the property.
propertyValue = pipeFeatureInput_var.path

# Set the value of the property.
pipeFeatureInput_var.path = propertyValue
```

## Property Value

This is a read/write property whose value is a [Path](Path.md).

## Version

Introduced in version October 2023
