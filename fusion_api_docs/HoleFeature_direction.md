# HoleFeature.direction Property

Parent Object: [HoleFeature](HoleFeature.md)  

## Description

Returns the direction of the hole.

## Syntax

"holeFeature_var" is a variable referencing a HoleFeature object.  

```python
# Get the value of the property.
propertyValue = holeFeature_var.direction
```

## Property Value

This is a read only property whose value is a [Vector3D](Vector3D.md).

## Version

Introduced in version August 2014
