# BoolValueCommandInput.parentCommand Property

Parent Object: [BoolValueCommandInput](BoolValueCommandInput.md)  

## Description

Gets the parent Command.

## Syntax

"boolValueCommandInput_var" is a variable referencing a BoolValueCommandInput object.  

```python
# Get the value of the property.
propertyValue = boolValueCommandInput_var.parentCommand
```

## Property Value

This is a read only property whose value is a [Command](Command.md).

## Version

Introduced in version June 2015
