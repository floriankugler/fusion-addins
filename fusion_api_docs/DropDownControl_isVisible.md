# DropDownControl.isVisible Property

Parent Object: [DropDownControl](DropDownControl.md)  

## Description

Gets or sets if this control is currently visible.

## Syntax

"dropDownControl_var" is a variable referencing a DropDownControl object.  

```python
# Get the value of the property.
propertyValue = dropDownControl_var.isVisible

# Set the value of the property.
dropDownControl_var.isVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
