# SelectionCommandInput.name Property

Parent Object: [SelectionCommandInput](SelectionCommandInput.md)  

## Description

Gets the user visible name of this input.

## Syntax

"selectionCommandInput_var" is a variable referencing a SelectionCommandInput object.  

```python
# Get the value of the property.
propertyValue = selectionCommandInput_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2014
