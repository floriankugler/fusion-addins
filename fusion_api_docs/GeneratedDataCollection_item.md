# GeneratedDataCollection.item Method

Parent Object: [GeneratedDataCollection](GeneratedDataCollection.md)  

## Description

Gets the desired generated data at the given index.

## Syntax

"generatedDataCollection_var" is a variable referencing a [GeneratedDataCollection](GeneratedDataCollection.md) object.

```python
returnValue = generatedDataCollection_var.item(index)
```

## Parameters

| Name  | Type    | Description |
|-------|---------|-------------|
| index | integer |             |

## Version

Introduced in version July 2023
