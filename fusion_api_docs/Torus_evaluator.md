# Torus.evaluator Property

Parent Object: [Torus](Torus.md)  

## Description

Returns the surface evaluator.

## Syntax

"torus_var" is a variable referencing a Torus object.  

```python
# Get the value of the property.
propertyValue = torus_var.evaluator
```

## Property Value

This is a read only property whose value is a [SurfaceEvaluator](SurfaceEvaluator.md).

## Version

Introduced in version August 2014
