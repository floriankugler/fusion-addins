# FlatPattern.isSuppressed Property

Parent Object: [FlatPattern](FlatPattern.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"flatPattern_var" is a variable referencing a FlatPattern object.  

```python
# Get the value of the property.
propertyValue = flatPattern_var.isSuppressed

# Set the value of the property.
flatPattern_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version October 2022
