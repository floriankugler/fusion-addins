# ChamferFeature.deleteMe Method

Parent Object: [ChamferFeature](ChamferFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"chamferFeature_var" is a variable referencing a [ChamferFeature](ChamferFeature.md) object.

```python
returnValue = chamferFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version November 2014
