# FormFeature.attributes Property

Parent Object: [FormFeature](FormFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"formFeature_var" is a variable referencing a FormFeature object.  

```python
# Get the value of the property.
propertyValue = formFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
