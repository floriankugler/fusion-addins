# Application.get Method

Parent Object: [Application](Application.md)  

## Description

Access to the root Application object.

## Syntax

This is a static method.  

```python

returnValue = adsk.core.Application.get()
```

## Return Value

| Type | Description |
|----|----|
| [Application](Application.md) | Return the root Application object or null if it failed. |

## Version

Introduced in version August 2014
