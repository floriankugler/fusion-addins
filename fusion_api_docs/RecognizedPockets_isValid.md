# RecognizedPockets.isValid Property

Parent Object: [RecognizedPockets](RecognizedPockets.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"recognizedPockets_var" is a variable referencing a RecognizedPockets object.  

```python
# Get the value of the property.
propertyValue = recognizedPockets_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2023
