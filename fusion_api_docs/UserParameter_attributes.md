# UserParameter.attributes Property

Parent Object: [UserParameter](UserParameter.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"userParameter_var" is a variable referencing a UserParameter object.  

```python
# Get the value of the property.
propertyValue = userParameter_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
