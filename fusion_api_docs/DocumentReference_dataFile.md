# DocumentReference.dataFile Property

Parent Object: [DocumentReference](DocumentReference.md)  

## Description

The dataFile on A360 that this object references.

## Syntax

"documentReference_var" is a variable referencing a DocumentReference object.  

```python
# Get the value of the property.
propertyValue = documentReference_var.dataFile
```

## Property Value

This is a read only property whose value is a [DataFile](DataFile.md).

## Version

Introduced in version June 2017
