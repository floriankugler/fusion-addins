# CustomGraphicsText.isStrikeThrough Property

Parent Object: [CustomGraphicsText](CustomGraphicsText.md)  

## Description

Specifies that the text displays using a strike through style. This is the default strike through style and applies to all of text unless there is a style override defined within the text.

## Syntax

"customGraphicsText_var" is a variable referencing a CustomGraphicsText object.  

```python
# Get the value of the property.
propertyValue = customGraphicsText_var.isStrikeThrough

# Set the value of the property.
customGraphicsText_var.isStrikeThrough = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version April 2019
