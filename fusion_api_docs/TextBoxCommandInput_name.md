# TextBoxCommandInput.name Property

Parent Object: [TextBoxCommandInput](TextBoxCommandInput.md)  

## Description

Gets the user visible name of this input.

## Syntax

"textBoxCommandInput_var" is a variable referencing a TextBoxCommandInput object.  

```python
# Get the value of the property.
propertyValue = textBoxCommandInput_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version June 2015
