# RadioButtonGroupCommandInput.deleteMe Method

Parent Object: [RadioButtonGroupCommandInput](RadioButtonGroupCommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"radioButtonGroupCommandInput_var" is a variable referencing a [RadioButtonGroupCommandInput](RadioButtonGroupCommandInput.md) object.

```python
returnValue = radioButtonGroupCommandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2020
