# FlatPatternComponent.partNumber Property

Parent Object: [FlatPatternComponent](FlatPatternComponent.md)  

## Description

Gets and sets the part number associated with this component. Setting this to an empty string will reset it to be the same as the component name.

## Syntax

"flatPatternComponent_var" is a variable referencing a FlatPatternComponent object.  

```python
# Get the value of the property.
propertyValue = flatPatternComponent_var.partNumber

# Set the value of the property.
flatPatternComponent_var.partNumber = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version October 2022
