# ArrangeEnvelopeDefinition.parentArrange Property

Parent Object: [ArrangeEnvelopeDefinition](ArrangeEnvelopeDefinition.md)  

## Description

Returns the parent ArrangeFeature this envelope is associated with.

## Syntax

"arrangeEnvelopeDefinition_var" is a variable referencing an ArrangeEnvelopeDefinition object.  

```python
# Get the value of the property.
propertyValue = arrangeEnvelopeDefinition_var.parentArrange
```

## Property Value

This is a read only property whose value is an [ArrangeFeature](ArrangeFeature.md).

## Version

Introduced in version January 2025
