# SketchConicCurve.attributes Property

Parent Object: [SketchConicCurve](SketchConicCurve.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"sketchConicCurve_var" is a variable referencing a SketchConicCurve object.  

```python
# Get the value of the property.
propertyValue = sketchConicCurve_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
