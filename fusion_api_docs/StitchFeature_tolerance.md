# StitchFeature.tolerance Property

Parent Object: [StitchFeature](StitchFeature.md)  

## Description

Returns the parameter controlling the tolerance. You can edit the tolerance by editing the value of the parameter object.

## Syntax

"stitchFeature_var" is a variable referencing a StitchFeature object.  

```python
# Get the value of the property.
propertyValue = stitchFeature_var.tolerance
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version June 2015
