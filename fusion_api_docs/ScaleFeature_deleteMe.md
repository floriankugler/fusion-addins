# ScaleFeature.deleteMe Method

Parent Object: [ScaleFeature](ScaleFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"scaleFeature_var" is a variable referencing a [ScaleFeature](ScaleFeature.md) object.

```python
returnValue = scaleFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version January 2015
