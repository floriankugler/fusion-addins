# TextCommandPalette.isVisible Property

Parent Object: [TextCommandPalette](TextCommandPalette.md)  

## Description

Gets and sets whether this palette is currently being displayed in the user interface.

## Syntax

"textCommandPalette_var" is a variable referencing a TextCommandPalette object.  

```python
# Get the value of the property.
propertyValue = textCommandPalette_var.isVisible

# Set the value of the property.
textCommandPalette_var.isVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.
