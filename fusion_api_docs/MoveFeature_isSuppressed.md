# MoveFeature.isSuppressed Property

Parent Object: [MoveFeature](MoveFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"moveFeature_var" is a variable referencing a MoveFeature object.  

```python
# Get the value of the property.
propertyValue = moveFeature_var.isSuppressed

# Set the value of the property.
moveFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version March 2015
