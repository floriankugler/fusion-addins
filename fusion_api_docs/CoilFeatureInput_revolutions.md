# CoilFeatureInput.revolutions Property

Parent Object: [CoilFeatureInput](CoilFeatureInput.md)  

## Description

Gets the revolutions number. Returns null in the case where the coilType property returns HeightAndPitchCoilType.

## Syntax

"coilFeatureInput_var" is a variable referencing a CoilFeatureInput object.  

```python
# Get the value of the property.
propertyValue = coilFeatureInput_var.revolutions
```

## Property Value

This is a read only property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version March 2016
