# HoleFeature.tappedHoleInfo Property

Parent Object: [HoleFeature](HoleFeature.md)  

## Description

This property returns the information used to define a tapped hole. Otherwise, this property returns null.

## Syntax

"holeFeature_var" is a variable referencing a HoleFeature object.  

```python
# Get the value of the property.
propertyValue = holeFeature_var.tappedHoleInfo
```

## Property Value

This is a read only property whose value is a [ThreadInfo](ThreadInfo.md).

## Version

Introduced in version September 2025
