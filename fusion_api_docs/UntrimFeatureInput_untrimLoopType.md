# UntrimFeatureInput.untrimLoopType Property

Parent Object: [UntrimFeatureInput](UntrimFeatureInput.md)  

## Description

Gets the loop type to be untrimmed. This is only used when faces are being untrimmed and is ignored for loops.

## Syntax

"untrimFeatureInput_var" is a variable referencing a UntrimFeatureInput object.  

```python
# Get the value of the property.
propertyValue = untrimFeatureInput_var.untrimLoopType
```

## Property Value

This is a read only property whose value is a [UntrimLoopTypes](UntrimLoopTypes.md).

## Version

Introduced in version January 2021
