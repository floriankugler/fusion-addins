# Matrix3D.isEqualTo Method

Parent Object: [Matrix3D](Matrix3D.md)  

## Description

Compares this matrix with another matrix and returns True if they're identical.

## Syntax

"matrix3D_var" is a variable referencing a [Matrix3D](Matrix3D.md) object.

```python
returnValue = matrix3D_var.isEqualTo(matrix)
```

## Return Value

| Type    | Description                             |
|---------|-----------------------------------------|
| boolean | Returns true if the matrices are equal. |

## Parameters

| Name   | Type                     | Description                           |
|--------|--------------------------|---------------------------------------|
| matrix | [Matrix3D](Matrix3D.md) | The matrix to compare this matrix to. |

## Version

Introduced in version August 2014
