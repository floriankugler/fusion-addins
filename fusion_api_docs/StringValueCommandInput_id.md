# StringValueCommandInput.id Property

Parent Object: [StringValueCommandInput](StringValueCommandInput.md)  

## Description

Gets the unique identifier for this input in the command's CommandInputs.

## Syntax

"stringValueCommandInput_var" is a variable referencing a StringValueCommandInput object.  

```python
# Get the value of the property.
propertyValue = stringValueCommandInput_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2014
