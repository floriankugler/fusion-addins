# Circle3D.asNurbsCurve Property

Parent Object: [Circle3D](Circle3D.md)  

## Description

Returns a NURBS curve that is geometrically identical to the circle.

## Syntax

"circle3D_var" is a variable referencing a Circle3D object.  

```python
# Get the value of the property.
propertyValue = circle3D_var.asNurbsCurve
```

## Property Value

This is a read only property whose value is a [NurbsCurve3D](NurbsCurve3D.md).

## Version

Introduced in version August 2014
