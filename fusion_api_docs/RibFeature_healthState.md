# RibFeature.healthState Property

Parent Object: [RibFeature](RibFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"ribFeature_var" is a variable referencing a RibFeature object.  

```python
# Get the value of the property.
propertyValue = ribFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
