# TorusFeature.attributes Property

Parent Object: [TorusFeature](TorusFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"torusFeature_var" is a variable referencing a TorusFeature object.  

```python
# Get the value of the property.
propertyValue = torusFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
