# RigidGroup.occurrences Property

Parent Object: [RigidGroup](RigidGroup.md)  

## Description

Returns the list of occurrences that are part of the rigid group.

## Syntax

"rigidGroup_var" is a variable referencing a RigidGroup object.  

```python
# Get the value of the property.
propertyValue = rigidGroup_var.occurrences
```

## Property Value

This is a read only property whose value is an [OccurrenceList](OccurrenceList.md).

## Version

Introduced in version September 2015
