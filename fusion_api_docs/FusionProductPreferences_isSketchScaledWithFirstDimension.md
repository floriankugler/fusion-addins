# FusionProductPreferences.isSketchScaledWithFirstDimension Property

Parent Object: [FusionProductPreferences](FusionProductPreferences.md)  

## Description

Gets and sets if the sketch geometry is automatically scaled when the first dimension is added.

## Syntax

"fusionProductPreferences_var" is a variable referencing a FusionProductPreferences object.  

```python
# Get the value of the property.
propertyValue = fusionProductPreferences_var.isSketchScaledWithFirstDimension

# Set the value of the property.
fusionProductPreferences_var.isSketchScaledWithFirstDimension = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version May 2022
