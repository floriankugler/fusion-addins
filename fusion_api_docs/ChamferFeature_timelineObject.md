# ChamferFeature.timelineObject Property

Parent Object: [ChamferFeature](ChamferFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"chamferFeature_var" is a variable referencing a ChamferFeature object.  

```python
# Get the value of the property.
propertyValue = chamferFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version November 2014
