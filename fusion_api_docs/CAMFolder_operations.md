# CAMFolder.operations Property

Parent Object: [CAMFolder](CAMFolder.md)  

## Description

Returns the Operations collection that provides access to existing individual operations in this folder.

## Syntax

"cAMFolder_var" is a variable referencing a CAMFolder object.  

```python
# Get the value of the property.
propertyValue = cAMFolder_var.operations
```

## Property Value

This is a read only property whose value is an [Operations](Operations.md).

## Version

Introduced in version January 2016
