# SketchLine.parentSketch Property

Parent Object: [SketchLine](SketchLine.md)  

## Description

Returns the parent sketch.

## Syntax

"sketchLine_var" is a variable referencing a SketchLine object.  

```python
# Get the value of the property.
propertyValue = sketchLine_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
