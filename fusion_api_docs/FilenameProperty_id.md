# FilenameProperty.id Property

Parent Object: [FilenameProperty](FilenameProperty.md)  

## Description

Returns the unique ID of this property.

## Syntax

"filenameProperty_var" is a variable referencing a FilenameProperty object.  

```python
# Get the value of the property.
propertyValue = filenameProperty_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2014
