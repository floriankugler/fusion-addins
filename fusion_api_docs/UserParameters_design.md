# UserParameters.design Property

Parent Object: [UserParameters](UserParameters.md)  

## Description

Returns the design that owns the user parameters collection.

## Syntax

"userParameters_var" is a variable referencing a UserParameters object.  

```python
# Get the value of the property.
propertyValue = userParameters_var.design
```

## Property Value

This is a read only property whose value is a [Design](Design.md).

## Version

Introduced in version August 2014
