# NurbsCurve2D.controlPointCount Property

Parent Object: [NurbsCurve2D](NurbsCurve2D.md)  

## Description

Gets the number of control points that define the curve

## Syntax

"nurbsCurve2D_var" is a variable referencing a NurbsCurve2D object.  

```python
# Get the value of the property.
propertyValue = nurbsCurve2D_var.controlPointCount
```

## Property Value

This is a read only property whose value is an integer.

## Version

Introduced in version August 2014
