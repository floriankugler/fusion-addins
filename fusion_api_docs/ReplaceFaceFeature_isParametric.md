# ReplaceFaceFeature.isParametric Property

Parent Object: [ReplaceFaceFeature](ReplaceFaceFeature.md)  

## Description

Indicates if this feature is parametric or not.

## Syntax

"replaceFaceFeature_var" is a variable referencing a ReplaceFaceFeature object.  

```python
# Get the value of the property.
propertyValue = replaceFaceFeature_var.isParametric
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version March 2015
