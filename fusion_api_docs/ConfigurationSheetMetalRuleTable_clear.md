# ConfigurationSheetMetalRuleTable.clear Method

Parent Object: [ConfigurationSheetMetalRuleTable](ConfigurationSheetMetalRuleTable.md)  

## Description

Clears the content of the sheet metal rule table, removes the reference from the top table, and hides it in the user interface.

## Syntax

"configurationSheetMetalRuleTable_var" is a variable referencing a [ConfigurationSheetMetalRuleTable](ConfigurationSheetMetalRuleTable.md) object.

```python
returnValue = configurationSheetMetalRuleTable_var.clear()
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Version

Introduced in version March 2024
