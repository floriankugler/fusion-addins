# Preferences.graphicsPreferences Property

Parent Object: [Preferences](Preferences.md)  

## Description

Gets the GraphicsPreferences object.

## Syntax

"preferences_var" is a variable referencing a Preferences object.  

```python
# Get the value of the property.
propertyValue = preferences_var.graphicsPreferences
```

## Property Value

This is a read only property whose value is a [GraphicsPreferences](GraphicsPreferences.md).

## Version

Introduced in version August 2014
