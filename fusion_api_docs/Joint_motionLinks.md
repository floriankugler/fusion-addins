# Joint.motionLinks Property

Parent Object: [Joint](Joint.md)  

## Description

Returns the MotionLink objects that this joint is involved in.

## Syntax

"joint_var" is a variable referencing a Joint object.  

```python
# Get the value of the property.
propertyValue = joint_var.motionLinks
```

## Property Value

This is a read only property whose value is an array of type [MotionLink](MotionLink.md).

## Version

Introduced in version November 2025
