# Workspace.productType Property

Parent Object: [Workspace](Workspace.md)  

## Description

Returns the name of the product this workspace is associated with.

## Syntax

"workspace_var" is a variable referencing a Workspace object.  

```python
# Get the value of the property.
propertyValue = workspace_var.productType
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version June 2015
