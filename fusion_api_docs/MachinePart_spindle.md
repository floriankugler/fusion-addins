# MachinePart.spindle Property

Parent Object: [MachinePart](MachinePart.md)  

## Description

Get the spindle object for this part used to reference this part for other operations. Will return null if the part has no spindle assigned.

## Syntax

"machinePart_var" is a variable referencing a MachinePart object.  

```python
# Get the value of the property.
propertyValue = machinePart_var.spindle
```

## Property Value

This is a read only property whose value is a [MachineSpindle](MachineSpindle.md).

## Version

Introduced in version April 2023
