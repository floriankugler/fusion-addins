# Vector3D.asPoint Method

Parent Object: [Vector3D](Vector3D.md)  

## Description

Returns a new point with the same coordinate values as this vector.

## Syntax

"vector3D_var" is a variable referencing a [Vector3D](Vector3D.md) object.

```python
returnValue = vector3D_var.asPoint()
```

## Return Value

| Type                   | Description           |
|------------------------|-----------------------|
| [Point3D](Point3D.md) | Return the new point. |

## Version

Introduced in version August 2014
