# BaseComponent.propertyGroups Property

Parent Object: [BaseComponent](BaseComponent.md)  

## Description

Returns the PropertyGroups object associated with this component.

## Syntax

"baseComponent_var" is a variable referencing a BaseComponent object.  

```python
# Get the value of the property.
propertyValue = baseComponent_var.propertyGroups
```

## Property Value

This is a read only property whose value is a [PropertyGroups](PropertyGroups.md).

## Version

Introduced in version January 2024
