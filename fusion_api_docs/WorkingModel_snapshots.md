# WorkingModel.snapshots Property

Parent Object: [WorkingModel](WorkingModel.md)  

## Description

Returns the Snapshots object associated with this design which provides access to the existing snapshots and the creation of new snapshots.

## Syntax

"workingModel_var" is a variable referencing a WorkingModel object.  

```python
# Get the value of the property.
propertyValue = workingModel_var.snapshots
```

## Property Value

This is a read only property whose value is a [Snapshots](Snapshots.md).

## Version

Introduced in version January 2024
