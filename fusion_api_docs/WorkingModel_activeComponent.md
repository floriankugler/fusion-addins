# WorkingModel.activeComponent Property

Parent Object: [WorkingModel](WorkingModel.md)  

## Description

Returns the component that is current being edited. This can return the root component or another component within the design.

## Syntax

"workingModel_var" is a variable referencing a WorkingModel object.  

```python
# Get the value of the property.
propertyValue = workingModel_var.activeComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version January 2024
