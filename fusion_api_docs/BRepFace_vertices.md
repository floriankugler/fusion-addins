# BRepFace.vertices Property

Parent Object: [BRepFace](BRepFace.md)  

## Description

Returns the BRepVertices used by this face

## Syntax

"bRepFace_var" is a variable referencing a BRepFace object.  

```python
# Get the value of the property.
propertyValue = bRepFace_var.vertices
```

## Property Value

This is a read only property whose value is a [BRepVertices](BRepVertices.md).

## Version

Introduced in version August 2014
