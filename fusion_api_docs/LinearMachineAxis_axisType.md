# LinearMachineAxis.axisType Property

Parent Object: [LinearMachineAxis](LinearMachineAxis.md)  

## Description

The type of axis.

## Syntax

"linearMachineAxis_var" is a variable referencing a LinearMachineAxis object.  

```python
# Get the value of the property.
propertyValue = linearMachineAxis_var.axisType
```

## Property Value

This is a read only property whose value is a [MachineAxisTypes](MachineAxisTypes.md).

## Version

Introduced in version April 2023
