# SketchAngularDimension.lineTwo Property

Parent Object: [SketchAngularDimension](SketchAngularDimension.md)  

## Description

The second line being constrained.

## Syntax

"sketchAngularDimension_var" is a variable referencing a SketchAngularDimension object.  

```python
# Get the value of the property.
propertyValue = sketchAngularDimension_var.lineTwo
```

## Property Value

This is a read only property whose value is a [SketchLine](SketchLine.md).

## Version

Introduced in version August 2014
