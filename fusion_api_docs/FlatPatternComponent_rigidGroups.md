# FlatPatternComponent.rigidGroups Property

Parent Object: [FlatPatternComponent](FlatPatternComponent.md)  

## Description

Returns the collection of rigid groups associated with this component.

## Syntax

"flatPatternComponent_var" is a variable referencing a FlatPatternComponent object.  

```python
# Get the value of the property.
propertyValue = flatPatternComponent_var.rigidGroups
```

## Property Value

This is a read only property whose value is a [RigidGroups](RigidGroups.md).

## Version

Introduced in version October 2022
