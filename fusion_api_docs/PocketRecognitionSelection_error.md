# PocketRecognitionSelection.error Property

Parent Object: [PocketRecognitionSelection](PocketRecognitionSelection.md)  

## Description

Gets the last warning string encountered after the selection was applied to a parent.

## Syntax

"pocketRecognitionSelection_var" is a variable referencing a PocketRecognitionSelection object.  

```python
# Get the value of the property.
propertyValue = pocketRecognitionSelection_var.error
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version July 2023
