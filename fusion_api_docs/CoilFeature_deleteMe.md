# CoilFeature.deleteMe Method

Parent Object: [CoilFeature](CoilFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"coilFeature_var" is a variable referencing a [CoilFeature](CoilFeature.md) object.

```python
returnValue = coilFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version March 2015
