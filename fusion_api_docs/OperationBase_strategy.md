# OperationBase.strategy Property

Parent Object: [OperationBase](OperationBase.md)  

## Description

Gets the name of the strategy associated with this operation.

## Syntax

"operationBase_var" is a variable referencing an OperationBase object.  

```python
# Get the value of the property.
propertyValue = operationBase_var.strategy
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version May 2022
