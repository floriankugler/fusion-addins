# UntrimFeature.loopsToUntrim Property

Parent Object: [UntrimFeature](UntrimFeature.md)  

## Description

Gets the loop objects to untrim. Returns null/None in the case where faces are specified instead of loops

## Syntax

"untrimFeature_var" is a variable referencing a UntrimFeature object.  

```python
# Get the value of the property.
propertyValue = untrimFeature_var.loopsToUntrim
```

## Property Value

This is a read only property whose value is an array of type [BRepLoop](BRepLoop.md).

## Version

Introduced in version January 2021
