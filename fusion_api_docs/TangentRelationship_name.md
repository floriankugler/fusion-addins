# TangentRelationship.name Property

Parent Object: [TangentRelationship](TangentRelationship.md)  

## Description

Gets and sets the name of the tangent relationship.

## Syntax

"tangentRelationship_var" is a variable referencing a TangentRelationship object.  

```python
# Get the value of the property.
propertyValue = tangentRelationship_var.name

# Set the value of the property.
tangentRelationship_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version May 2022
