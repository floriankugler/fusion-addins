# ChordLengthFilletEdgeSet.isTangentChain Property

Parent Object: [ChordLengthFilletEdgeSet](ChordLengthFilletEdgeSet.md)  

## Description

Gets and sets the Tangent chain for fillet. This enables tangent chain option for fillet.

## Syntax

"chordLengthFilletEdgeSet_var" is a variable referencing a ChordLengthFilletEdgeSet object.  

```python
# Get the value of the property.
propertyValue = chordLengthFilletEdgeSet_var.isTangentChain

# Set the value of the property.
chordLengthFilletEdgeSet_var.isTangentChain = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version March 2021
