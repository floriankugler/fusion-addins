# CombineFeatures.count Property

Parent Object: [CombineFeatures](CombineFeatures.md)  

## Description

The number of combine features in the collection. This property returns nothing in the case where the feature is non-parametric.

## Syntax

"combineFeatures_var" is a variable referencing a CombineFeatures object.  

```python
# Get the value of the property.
propertyValue = combineFeatures_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version November 2014
