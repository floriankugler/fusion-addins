# BRepEdge.body Property

Parent Object: [BRepEdge](BRepEdge.md)  

## Description

Returns the parent body of the edge.

## Syntax

"bRepEdge_var" is a variable referencing a BRepEdge object.  

```python
# Get the value of the property.
propertyValue = bRepEdge_var.body
```

## Property Value

This is a read only property whose value is a [BRepBody](BRepBody.md).

## Version

Introduced in version August 2014
