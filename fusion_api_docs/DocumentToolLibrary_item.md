# DocumentToolLibrary.item Method

Parent Object: [DocumentToolLibrary](DocumentToolLibrary.md)  

## Description

Get Tool by index in ToolLibrary.

## Syntax

"documentToolLibrary_var" is a variable referencing a [DocumentToolLibrary](DocumentToolLibrary.md) object.

```python
returnValue = documentToolLibrary_var.item(index)
```

## Return Value

| Type             | Description                                         |
|------------------|-----------------------------------------------------|
| [Tool](Tool.md) | Returns the Tool in the ToolLibrary by given index. |

## Parameters

| Name  | Type     | Description                           |
|-------|----------|---------------------------------------|
| index | uinteger | Index of the Tool in the ToolLibrary. |

## Version

Introduced in version April 2023
