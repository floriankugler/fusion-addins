# ObjectCollection.count Property

Parent Object: [ObjectCollection](ObjectCollection.md)  

## Description

Returns the number of occurrences in the collection.

## Syntax

"objectCollection_var" is a variable referencing an ObjectCollection object.  

```python
# Get the value of the property.
propertyValue = objectCollection_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version August 2014
