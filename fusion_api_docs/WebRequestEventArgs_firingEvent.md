# WebRequestEventArgs.firingEvent Property

Parent Object: [WebRequestEventArgs](WebRequestEventArgs.md)  

## Description

The event that the firing is in response to.

## Syntax

"webRequestEventArgs_var" is a variable referencing a WebRequestEventArgs object.  

```python
# Get the value of the property.
propertyValue = webRequestEventArgs_var.firingEvent
```

## Property Value

This is a read only property whose value is an [Event](Event.md).

## Version

Introduced in version May 2016
