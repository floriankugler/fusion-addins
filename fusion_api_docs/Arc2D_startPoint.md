# Arc2D.startPoint Property

Parent Object: [Arc2D](Arc2D.md)  

## Description

Gets the position of the start point of the arc.

## Syntax

"arc2D_var" is a variable referencing an Arc2D object.  

```python
# Get the value of the property.
propertyValue = arc2D_var.startPoint
```

## Property Value

This is a read only property whose value is a [Point2D](Point2D.md).

## Version

Introduced in version August 2014
