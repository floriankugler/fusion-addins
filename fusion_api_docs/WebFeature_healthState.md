# WebFeature.healthState Property

Parent Object: [WebFeature](WebFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"webFeature_var" is a variable referencing a WebFeature object.  

```python
# Get the value of the property.
propertyValue = webFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
