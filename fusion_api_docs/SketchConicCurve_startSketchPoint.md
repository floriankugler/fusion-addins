# SketchConicCurve.startSketchPoint Property

Parent Object: [SketchConicCurve](SketchConicCurve.md)  

## Description

The sketch point at the start of the curve.

## Syntax

"sketchConicCurve_var" is a variable referencing a SketchConicCurve object.  

```python
# Get the value of the property.
propertyValue = sketchConicCurve_var.startSketchPoint
```

## Property Value

This is a read only property whose value is a [SketchPoint](SketchPoint.md).

## Version

Introduced in version January 2015
