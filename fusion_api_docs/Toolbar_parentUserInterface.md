# Toolbar.parentUserInterface Property

Parent Object: [Toolbar](Toolbar.md)  

## Description

Gets the owning UserInterface object.

## Syntax

"toolbar_var" is a variable referencing a Toolbar object.  

```python
# Get the value of the property.
propertyValue = toolbar_var.parentUserInterface
```

## Property Value

This is a read only property whose value is a [UserInterface](UserInterface.md).

## Version

Introduced in version August 2014
