# PatchFeature.deleteMe Method

Parent Object: [PatchFeature](PatchFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"patchFeature_var" is a variable referencing a [PatchFeature](PatchFeature.md) object.

```python
returnValue = patchFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version September 2015
