# ConstructionAxis.geometry Property

Parent Object: [ConstructionAxis](ConstructionAxis.md)  

## Description

Returns an infinite line that represents the position and orientation of the construction axis. This geometry is defined in the AssemblyContext of this ConstructionAxis.

## Syntax

"constructionAxis_var" is a variable referencing a ConstructionAxis object.  

```python
# Get the value of the property.
propertyValue = constructionAxis_var.geometry
```

## Property Value

This is a read only property whose value is an [InfiniteLine3D](InfiniteLine3D.md).

## Version

Introduced in version August 2014
