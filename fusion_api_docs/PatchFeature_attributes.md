# PatchFeature.attributes Property

Parent Object: [PatchFeature](PatchFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"patchFeature_var" is a variable referencing a PatchFeature object.  

```python
# Get the value of the property.
propertyValue = patchFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
