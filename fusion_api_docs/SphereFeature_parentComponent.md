# SphereFeature.parentComponent Property

Parent Object: [SphereFeature](SphereFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"sphereFeature_var" is a variable referencing a SphereFeature object.  

```python
# Get the value of the property.
propertyValue = sphereFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version September 2015
