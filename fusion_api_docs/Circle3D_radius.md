# Circle3D.radius Property

Parent Object: [Circle3D](Circle3D.md)  

## Description

Gets and sets the radius of the circle.

## Syntax

"circle3D_var" is a variable referencing a Circle3D object.  

```python
# Get the value of the property.
propertyValue = circle3D_var.radius

# Set the value of the property.
circle3D_var.radius = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version August 2014
