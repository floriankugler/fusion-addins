# Drawing.workspaces Property

Parent Object: [Drawing](Drawing.md)  

## Description

Returns the workspaces associated with this product.

## Syntax

"drawing_var" is a variable referencing a Drawing object.  

```python
# Get the value of the property.
propertyValue = drawing_var.workspaces
```

## Property Value

This is a read only property whose value is a [WorkspaceList](WorkspaceList.md).

## Version

Introduced in version December 2020
