# DataFile.parentFolder Property

Parent Object: [DataFile](DataFile.md)  

## Description

Returns the parent folder this item is contained within.

## Syntax

"dataFile_var" is a variable referencing a DataFile object.  

```python
# Get the value of the property.
propertyValue = dataFile_var.parentFolder
```

## Property Value

This is a read only property whose value is a [DataFolder](DataFolder.md).

## Version

Introduced in version January 2015
