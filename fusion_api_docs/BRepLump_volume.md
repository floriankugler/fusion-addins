# BRepLump.volume Property

Parent Object: [BRepLump](BRepLump.md)  

## Description

Returns the volume in cm ^ 3. Returns 0 in the case the lump is not solid.

## Syntax

"bRepLump_var" is a variable referencing a BRepLump object.  

```python
# Get the value of the property.
propertyValue = bRepLump_var.volume
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version August 2014
