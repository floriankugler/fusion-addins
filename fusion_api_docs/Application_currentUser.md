# Application.currentUser Property

Parent Object: [Application](Application.md)  

## Description

Returns the User that is currently logged in.

## Syntax

"application_var" is a variable referencing an Application object.  

```python
# Get the value of the property.
propertyValue = application_var.currentUser
```

## Property Value

This is a read only property whose value is a [User](User.md).

## Version

Introduced in version January 2016
