# Drawing.exportManager Property

Parent Object: [Drawing](Drawing.md)  

## Description

Returns the DrawingExportManager for this drawing.Â You use the ExportManager to export the drawing in various formats.

## Syntax


## Property Value

This is a read only property whose value is a [DrawingExportManager](DrawingExportManager.md).
