# Vector2D.dotProduct Method

Parent Object: [Vector2D](Vector2D.md)  

## Description

Calculates the Dot Product of this vector and an input vector.

## Syntax

"vector2D_var" is a variable referencing a [Vector2D](Vector2D.md) object.

```python
returnValue = vector2D_var.dotProduct(vector)
```

## Return Value

| Type   | Description                                 |
|--------|---------------------------------------------|
| double | Returns the dot product of the two vectors. |

## Parameters

| Name | Type | Description |
|----|----|----|
| vector | [Vector2D](Vector2D.md) | The vector to use in the dot product calculation. |

## Version

Introduced in version August 2014
