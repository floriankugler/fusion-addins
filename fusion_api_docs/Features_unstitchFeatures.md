# Features.unstitchFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the Unstitch features within the component and supports the creation of new Unstitch features.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.unstitchFeatures
```

## Property Value

This is a read only property whose value is a [UnstitchFeatures](UnstitchFeatures.md).

## Version

Introduced in version July 2015
