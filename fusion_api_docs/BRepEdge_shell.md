# BRepEdge.shell Property

Parent Object: [BRepEdge](BRepEdge.md)  

## Description

Returns the parent shell of the edge.

## Syntax

"bRepEdge_var" is a variable referencing a BRepEdge object.  

```python
# Get the value of the property.
propertyValue = bRepEdge_var.shell
```

## Property Value

This is a read only property whose value is a [BRepShell](BRepShell.md).

## Version

Introduced in version August 2014
