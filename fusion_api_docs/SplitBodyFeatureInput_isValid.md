# SplitBodyFeatureInput.isValid Property

Parent Object: [SplitBodyFeatureInput](SplitBodyFeatureInput.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"splitBodyFeatureInput_var" is a variable referencing a SplitBodyFeatureInput object.  

```python
# Get the value of the property.
propertyValue = splitBodyFeatureInput_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version June 2015
