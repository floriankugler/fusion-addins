# IntegerSpinnerCommandInput.id Property

Parent Object: [IntegerSpinnerCommandInput](IntegerSpinnerCommandInput.md)  

## Description

Gets the unique identifier for this input in the command's CommandInputs.

## Syntax

"integerSpinnerCommandInput_var" is a variable referencing an IntegerSpinnerCommandInput object.  

```python
# Get the value of the property.
propertyValue = integerSpinnerCommandInput_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version July 2015
