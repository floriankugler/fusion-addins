# TangentRelationship.parentComponent Property

Parent Object: [TangentRelationship](TangentRelationship.md)  

## Description

Returns the parent component that owns this tangent relationship.

## Syntax

"tangentRelationship_var" is a variable referencing a TangentRelationship object.  

```python
# Get the value of the property.
propertyValue = tangentRelationship_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version May 2022
