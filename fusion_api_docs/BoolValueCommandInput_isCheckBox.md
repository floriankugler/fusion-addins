# BoolValueCommandInput.isCheckBox Property

Parent Object: [BoolValueCommandInput](BoolValueCommandInput.md)  

## Description

Indicates if this is being shown as a button or check box.

## Syntax

"boolValueCommandInput_var" is a variable referencing a BoolValueCommandInput object.  

```python
# Get the value of the property.
propertyValue = boolValueCommandInput_var.isCheckBox
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version June 2015
