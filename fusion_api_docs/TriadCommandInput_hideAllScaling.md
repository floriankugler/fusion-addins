# TriadCommandInput.hideAllScaling Method

Parent Object: [TriadCommandInput](TriadCommandInput.md)  

## Description

Sets all scaling related controls to be invisible. This is useful if you are only using translations or rotations.

## Syntax

"triadCommandInput_var" is a variable referencing a [TriadCommandInput](TriadCommandInput.md) object.

```python
returnValue = triadCommandInput_var.hideAllScaling()
```

## Return Value

| Type    | Description                                         |
|---------|-----------------------------------------------------|
| boolean | Returns true if hiding the controls was successful. |

## Version

Introduced in version May 2022
