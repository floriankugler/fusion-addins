# ScaleFeatureInput.scaleFactor Property

Parent Object: [ScaleFeatureInput](ScaleFeatureInput.md)  

## Description

Gets and sets the scale factor used for a uniform scale. Setting this value will cause the isUniform property to be set to true.

## Syntax

"scaleFeatureInput_var" is a variable referencing a ScaleFeatureInput object.  

```python
# Get the value of the property.
propertyValue = scaleFeatureInput_var.scaleFactor

# Set the value of the property.
scaleFeatureInput_var.scaleFactor = propertyValue
```

## Property Value

This is a read/write property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version January 2015
