# PipeFeature.sectionType Property

Parent Object: [PipeFeature](PipeFeature.md)  

## Description

Gets and sets the section type of the Pipe. The type can be: Circular, Square, Triangular.

## Syntax

"pipeFeature_var" is a variable referencing a PipeFeature object.  

```python
# Get the value of the property.
propertyValue = pipeFeature_var.sectionType

# Set the value of the property.
pipeFeature_var.sectionType = propertyValue
```

## Property Value

This is a read/write property whose value is a [PipeSectionTypes](PipeSectionTypes.md).

## Version

Introduced in version October 2023
