# Component.jointOrigins Property

Parent Object: [Component](Component.md)  

## Description

Returns the collection of joint origins associated with this component.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.jointOrigins
```

## Property Value

This is a read only property whose value is a [JointOrigins](JointOrigins.md).

## Version

Introduced in version September 2017
