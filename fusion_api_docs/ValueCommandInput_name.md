# ValueCommandInput.name Property

Parent Object: [ValueCommandInput](ValueCommandInput.md)  

## Description

Gets the user visible name of this input.

## Syntax

"valueCommandInput_var" is a variable referencing a ValueCommandInput object.  

```python
# Get the value of the property.
propertyValue = valueCommandInput_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2014
