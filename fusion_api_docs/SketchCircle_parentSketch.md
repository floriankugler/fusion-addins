# SketchCircle.parentSketch Property

Parent Object: [SketchCircle](SketchCircle.md)  

## Description

Returns the parent sketch.

## Syntax

"sketchCircle_var" is a variable referencing a SketchCircle object.  

```python
# Get the value of the property.
propertyValue = sketchCircle_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
