# RadioButtonGroupCommandInput.commandInputs Property

Parent Object: [RadioButtonGroupCommandInput](RadioButtonGroupCommandInput.md)  

## Description

Gets the CommandInputs class of the parent, which can be a Command, GroupCommandInput or TabCommandInput.

## Syntax

"radioButtonGroupCommandInput_var" is a variable referencing a RadioButtonGroupCommandInput object.  

```python
# Get the value of the property.
propertyValue = radioButtonGroupCommandInput_var.commandInputs
```

## Property Value

This is a read only property whose value is a [CommandInputs](CommandInputs.md).

## Version

Introduced in version July 2015
