# SketchEllipticalArc.geometry Property

Parent Object: [SketchEllipticalArc](SketchEllipticalArc.md)  

## Description

Returns the transient geometry of the elliptical arc which provides geometric information about the elliptical arc. The returned geometry is always in sketch space.

## Syntax

"sketchEllipticalArc_var" is a variable referencing a SketchEllipticalArc object.  

```python
# Get the value of the property.
propertyValue = sketchEllipticalArc_var.geometry
```

## Property Value

This is a read only property whose value is an [EllipticalArc3D](EllipticalArc3D.md).

## Version

Introduced in version August 2014
