# ConfigurationSheetMetalRuleTable.name Property

Parent Object: [ConfigurationSheetMetalRuleTable](ConfigurationSheetMetalRuleTable.md)  

## Description

Returns the name of the table as seen in the user interface.

## Syntax

"configurationSheetMetalRuleTable_var" is a variable referencing a ConfigurationSheetMetalRuleTable object.  

```python
# Get the value of the property.
propertyValue = configurationSheetMetalRuleTable_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version March 2024
