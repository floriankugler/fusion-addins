# MachineAxisConfigurations.item Method

Parent Object: [MachineAxisConfigurations](MachineAxisConfigurations.md)  

## Description

Get the configuration at index in this collection

## Syntax

"machineAxisConfigurations_var" is a variable referencing a [MachineAxisConfigurations](MachineAxisConfigurations.md) object.

```python
returnValue = machineAxisConfigurations_var.item(index)
```

## Return Value

| Type | Description |
|----|----|
| [MachineAxisConfiguration](MachineAxisConfiguration.md) | Returns the MachineAxisConfiguration at index. |

## Parameters

| Name  | Type    | Description             |
|-------|---------|-------------------------|
| index | integer | Index of configuration. |

## Version

Introduced in version April 2023
