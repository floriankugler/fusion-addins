# Operation.isProtected Property

Parent Object: [Operation](Operation.md)  

## Description

Gets and sets the "protected" property value of the operation. Gets/sets true if the operation is protected.

## Syntax

"operation_var" is a variable referencing an Operation object.  

```python
# Get the value of the property.
propertyValue = operation_var.isProtected

# Set the value of the property.
operation_var.isProtected = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2016
