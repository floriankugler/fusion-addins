# OrientedBoundingBox3D.heightDirection Property

Parent Object: [OrientedBoundingBox3D](OrientedBoundingBox3D.md)  

## Description

Gets the direction of the height of the oriented bounding box. A unit vector is always returned.

## Syntax

"orientedBoundingBox3D_var" is a variable referencing an OrientedBoundingBox3D object.  

```python
# Get the value of the property.
propertyValue = orientedBoundingBox3D_var.heightDirection
```

## Property Value

This is a read only property whose value is a [Vector3D](Vector3D.md).
