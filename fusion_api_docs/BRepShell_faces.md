# BRepShell.faces Property

Parent Object: [BRepShell](BRepShell.md)  

## Description

Returns the BRepFaces directly owned by this shell

## Syntax

"bRepShell_var" is a variable referencing a BRepShell object.  

```python
# Get the value of the property.
propertyValue = bRepShell_var.faces
```

## Property Value

This is a read only property whose value is a [BRepFaces](BRepFaces.md).

## Version

Introduced in version August 2014
