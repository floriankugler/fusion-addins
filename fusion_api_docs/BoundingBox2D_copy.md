# BoundingBox2D.copy Method

Parent Object: [BoundingBox2D](BoundingBox2D.md)  

## Description

Create a copy of this bounding box.

## Syntax

"boundingBox2D_var" is a variable referencing a [BoundingBox2D](BoundingBox2D.md) object.

```python
returnValue = boundingBox2D_var.copy()
```

## Return Value

| Type                               | Description                        |
|------------------------------------|------------------------------------|
| [BoundingBox2D](BoundingBox2D.md) | Returns the new bounding box copy. |

## Version

Introduced in version August 2014
