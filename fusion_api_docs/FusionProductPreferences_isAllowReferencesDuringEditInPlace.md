# FusionProductPreferences.isAllowReferencesDuringEditInPlace Property

Parent Object: [FusionProductPreferences](FusionProductPreferences.md)  

## Description

Gets and sets if you can create associative references while editing external components in context.

## Syntax

"fusionProductPreferences_var" is a variable referencing a FusionProductPreferences object.  

```python
# Get the value of the property.
propertyValue = fusionProductPreferences_var.isAllowReferencesDuringEditInPlace

# Set the value of the property.
fusionProductPreferences_var.isAllowReferencesDuringEditInPlace = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version May 2022
