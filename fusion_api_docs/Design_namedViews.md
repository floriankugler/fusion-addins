# Design.namedViews Property

Parent Object: [Design](Design.md)  

## Description

Returns the NamedViews object associated with this product. The NamedViews collection provides access to the named views defined in this product and supports the creation of new named views.

## Syntax

"design_var" is a variable referencing a Design object.  

```python
# Get the value of the property.
propertyValue = design_var.namedViews
```

## Property Value

This is a read only property whose value is a [NamedViews](NamedViews.md).

## Version

Introduced in version September 2023
