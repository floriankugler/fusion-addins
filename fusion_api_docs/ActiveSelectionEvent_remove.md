# ActiveSelectionEvent.remove Method

Parent Object: [ActiveSelectionEvent](ActiveSelectionEvent.md)  

## Description

Removes a handler from the event.

## Syntax

"activeSelectionEvent_var" is a variable referencing an [ActiveSelectionEvent](ActiveSelectionEvent.md) object.

```python
returnValue = activeSelectionEvent_var.remove(handler)
```

## Return Value

| Type    | Description                                            |
|---------|--------------------------------------------------------|
| boolean | Returns true if removal of the handler was successful. |

## Parameters

| Name | Type | Description |
|----|----|----|
| handler | [ActiveSelectionEventHandler](ActiveSelectionEventHandler.md) | The handler object to be removed from the event. |

## Version

Introduced in version August 2020
