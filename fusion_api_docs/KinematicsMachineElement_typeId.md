# KinematicsMachineElement.typeId Property

Parent Object: [KinematicsMachineElement](KinematicsMachineElement.md)  

## Description

Identifier for this type of machine element.

## Syntax

"kinematicsMachineElement_var" is a variable referencing a KinematicsMachineElement object.  

```python
# Get the value of the property.
propertyValue = kinematicsMachineElement_var.typeId
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
