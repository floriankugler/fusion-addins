# ProfileCurve.geometry Property

Parent Object: [ProfileCurve](ProfileCurve.md)  

## Description

Returns the geometric entity of this portion of the profile.

## Syntax

"profileCurve_var" is a variable referencing a ProfileCurve object.  

```python
# Get the value of the property.
propertyValue = profileCurve_var.geometry
```

## Property Value

This is a read only property whose value is a [Curve3D](Curve3D.md).

## Version

Introduced in version August 2014
