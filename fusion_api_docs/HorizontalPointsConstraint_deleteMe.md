# HorizontalPointsConstraint.deleteMe Method

Parent Object: [HorizontalPointsConstraint](HorizontalPointsConstraint.md)  

## Description

Deletes this constraint. The IsDeletable property can be used to determine if this constraint can be deleted.

## Syntax

"horizontalPointsConstraint_var" is a variable referencing a [HorizontalPointsConstraint](HorizontalPointsConstraint.md) object.

```python
returnValue = horizontalPointsConstraint_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
