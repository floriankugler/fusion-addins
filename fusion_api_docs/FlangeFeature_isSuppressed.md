# FlangeFeature.isSuppressed Property

Parent Object: [FlangeFeature](FlangeFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"flangeFeature_var" is a variable referencing a FlangeFeature object.  

```python
# Get the value of the property.
propertyValue = flangeFeature_var.isSuppressed

# Set the value of the property.
flangeFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2020
