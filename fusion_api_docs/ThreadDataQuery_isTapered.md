# ThreadDataQuery.isTapered Property

Parent Object: [ThreadDataQuery](ThreadDataQuery.md)  

## Description

Returns if this ThreadDataQuery was created to query for standard or tapered threads.

## Syntax

"threadDataQuery_var" is a variable referencing a ThreadDataQuery object.  

```python
# Get the value of the property.
propertyValue = threadDataQuery_var.isTapered
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2025
