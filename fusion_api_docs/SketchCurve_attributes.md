# SketchCurve.attributes Property

Parent Object: [SketchCurve](SketchCurve.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"sketchCurve_var" is a variable referencing a SketchCurve object.  

```python
# Get the value of the property.
propertyValue = sketchCurve_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
