# Operation.parent Property

Parent Object: [Operation](Operation.md)  

## Description

Returns the parent Setup, Folder or Pattern for this operation.

## Syntax

"operation_var" is a variable referencing an Operation object.  

```python
# Get the value of the property.
propertyValue = operation_var.parent
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version January 2016
