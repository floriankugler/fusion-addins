# JointOrigin.deleteMe Method

Parent Object: [JointOrigin](JointOrigin.md)  

## Description

Deletes this joint origin.

## Syntax

"jointOrigin_var" is a variable referencing a [JointOrigin](JointOrigin.md) object.

```python
returnValue = jointOrigin_var.deleteMe()
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Version

Introduced in version September 2015
