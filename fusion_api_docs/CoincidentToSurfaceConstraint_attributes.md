# CoincidentToSurfaceConstraint.attributes Property

Parent Object: [CoincidentToSurfaceConstraint](CoincidentToSurfaceConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"coincidentToSurfaceConstraint_var" is a variable referencing a CoincidentToSurfaceConstraint object.  

```python
# Get the value of the property.
propertyValue = coincidentToSurfaceConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version September 2023
