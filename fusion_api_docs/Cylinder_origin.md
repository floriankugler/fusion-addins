# Cylinder.origin Property

Parent Object: [Cylinder](Cylinder.md)  

## Description

The origin point (center) of the base of the cylinder.

## Syntax

"cylinder_var" is a variable referencing a Cylinder object.  

```python
# Get the value of the property.
propertyValue = cylinder_var.origin

# Set the value of the property.
cylinder_var.origin = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
