# HoleFeature.healthState Property

Parent Object: [HoleFeature](HoleFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"holeFeature_var" is a variable referencing a HoleFeature object.  

```python
# Get the value of the property.
propertyValue = holeFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
