# ChamferFeatures.count Property

Parent Object: [ChamferFeatures](ChamferFeatures.md)  

## Description

The number of chamfer features in the collection.

## Syntax

"chamferFeatures_var" is a variable referencing a ChamferFeatures object.  

```python
# Get the value of the property.
propertyValue = chamferFeatures_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version November 2014
