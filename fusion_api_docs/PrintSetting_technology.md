# PrintSetting.technology Property

Parent Object: [PrintSetting](PrintSetting.md)  

## Description

Gets the technology of the PrintSetting.

## Syntax

"printSetting_var" is a variable referencing a PrintSetting object.  

```python
# Get the value of the property.
propertyValue = printSetting_var.technology
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
