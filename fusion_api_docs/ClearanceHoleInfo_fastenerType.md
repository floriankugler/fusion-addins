# ClearanceHoleInfo.fastenerType Property

Parent Object: [ClearanceHoleInfo](ClearanceHoleInfo.md)  

## Description

Returns the string that defines the fastener type.

## Syntax

"clearanceHoleInfo_var" is a variable referencing a ClearanceHoleInfo object.  

```python
# Get the value of the property.
propertyValue = clearanceHoleInfo_var.fastenerType
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version September 2025
