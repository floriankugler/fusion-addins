# OperationBase.hasWarning Property

Parent Object: [OperationBase](OperationBase.md)  

## Description

Gets if problems were encountered when generating the operation.

## Syntax

"operationBase_var" is a variable referencing an OperationBase object.  

```python
# Get the value of the property.
propertyValue = operationBase_var.hasWarning
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version April 2023
