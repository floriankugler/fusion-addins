# DocumentToolLibrary.createQuery Method

Parent Object: [DocumentToolLibrary](DocumentToolLibrary.md)  

## Description

Creates a new ToolQuery that is used to query the library for tools matching the query.

## Syntax

"documentToolLibrary_var" is a variable referencing a [DocumentToolLibrary](DocumentToolLibrary.md) object.

```python
returnValue = documentToolLibrary_var.createQuery()
```

## Return Value

| Type | Description |
|----|----|
| [ToolQuery](ToolQuery.md) | Returns a new ToolQuery. The query is predefined by given parameter. |

## Version

Introduced in version April 2023
