# SplitBodyFeature.timelineObject Property

Parent Object: [SplitBodyFeature](SplitBodyFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"splitBodyFeature_var" is a variable referencing a SplitBodyFeature object.  

```python
# Get the value of the property.
propertyValue = splitBodyFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version June 2015
