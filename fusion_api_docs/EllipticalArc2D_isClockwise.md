# EllipticalArc2D.isClockwise Property

Parent Object: [EllipticalArc2D](EllipticalArc2D.md)  

## Description

Gets if the sweep direction of the elliptical arc is clockwise or counterclockwise.

## Syntax

"ellipticalArc2D_var" is a variable referencing an EllipticalArc2D object.  

```python
# Get the value of the property.
propertyValue = ellipticalArc2D_var.isClockwise
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
