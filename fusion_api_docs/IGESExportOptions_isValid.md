# IGESExportOptions.isValid Property

Parent Object: [IGESExportOptions](IGESExportOptions.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"iGESExportOptions_var" is a variable referencing an IGESExportOptions object.  

```python
# Get the value of the property.
propertyValue = iGESExportOptions_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2015
