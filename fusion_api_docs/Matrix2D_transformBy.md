# Matrix2D.transformBy Method

Parent Object: [Matrix2D](Matrix2D.md)  

## Description

Transforms this matrix using the input matrix.

## Syntax

"matrix2D_var" is a variable referencing a [Matrix2D](Matrix2D.md) object.

```python
returnValue = matrix2D_var.transformBy(matrix)
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name   | Type                     | Description                |
|--------|--------------------------|----------------------------|
| matrix | [Matrix2D](Matrix2D.md) | The transformation matrix. |

## Version

Introduced in version August 2014
