# Operation.generatingProgress Property

Parent Object: [Operation](Operation.md)  

## Description

Gets the generation progress value for this operation.

## Syntax

"operation_var" is a variable referencing an Operation object.  

```python
# Get the value of the property.
propertyValue = operation_var.generatingProgress
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version January 2016
