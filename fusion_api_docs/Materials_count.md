# Materials.count Property

Parent Object: [Materials](Materials.md)  

## Description

The number of Materials in the collection.

## Syntax

"materials_var" is a variable referencing a Materials object.  

```python
# Get the value of the property.
propertyValue = materials_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version August 2014
