# Component.material Property

Parent Object: [Component](Component.md)  

## Description

Gets and sets the physical material assigned to this component.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.material

# Set the value of the property.
component_var.material = propertyValue
```

## Property Value

This is a read/write property whose value is a [Material](Material.md).

## Version

Introduced in version September 2015
