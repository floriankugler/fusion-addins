# LineOnPlanarSurfaceConstraint.planarSurface Property

Parent Object: [LineOnPlanarSurfaceConstraint](LineOnPlanarSurfaceConstraint.md)  

## Description

Returns the planar BRepFace or ConstructionPlane the line is constrained to lie on.

## Syntax

"lineOnPlanarSurfaceConstraint_var" is a variable referencing a LineOnPlanarSurfaceConstraint object.  

```python
# Get the value of the property.
propertyValue = lineOnPlanarSurfaceConstraint_var.planarSurface
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version September 2023
