# ConfigurationRow.deleteMe Method

Parent Object: [ConfigurationRow](ConfigurationRow.md)  

## Description

Deletes this row from the table. The first row of the top table cannot be deleted, and this method will fail.

## Syntax

"configurationRow_var" is a variable referencing a [ConfigurationRow](ConfigurationRow.md) object.

```python
returnValue = configurationRow_var.deleteMe()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns true if the deletion was successful. |

## Version

Introduced in version March 2024
