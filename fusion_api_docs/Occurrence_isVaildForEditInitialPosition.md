# Occurrence.isVaildForEditInitialPosition Property

Parent Object: [Occurrence](Occurrence.md)  

## Description

Gets whether this occurrence is valid for edit initial position.

## Syntax

"occurrence_var" is a variable referencing an Occurrence object.  

```python
# Get the value of the property.
propertyValue = occurrence_var.isVaildForEditInitialPosition
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version November 2024
