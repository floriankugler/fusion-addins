# FusionDocument.parent Property

Parent Object: [FusionDocument](FusionDocument.md)  

## Description

Returns the parent Application object.

## Syntax

"fusionDocument_var" is a variable referencing a FusionDocument object.  

```python
# Get the value of the property.
propertyValue = fusionDocument_var.parent
```

## Property Value

This is a read only property whose value is an [Application](Application.md).

## Version

Introduced in version August 2014
