# DataEventHandler Object

Derived from: [EventHandler](EventHandler.md) Object  

## Description

The DataEventHandler is a client implemented class that can be added as a handler to a DataEvent.

## Methods

| Name | Description |
|----|----|
| [notify](DataEventHandler_notify.md) | The function called by Fusion when the associated event is fired. |

## Version

Introduced in version January 2022
