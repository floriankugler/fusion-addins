# MirrorFeature.attributes Property

Parent Object: [MirrorFeature](MirrorFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"mirrorFeature_var" is a variable referencing a MirrorFeature object.  

```python
# Get the value of the property.
propertyValue = mirrorFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
