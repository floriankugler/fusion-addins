# StringProperty.id Property

Parent Object: [StringProperty](StringProperty.md)  

## Description

Returns the unique ID of this property.

## Syntax

"stringProperty_var" is a variable referencing a StringProperty object.  

```python
# Get the value of the property.
propertyValue = stringProperty_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2014
