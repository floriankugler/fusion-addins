# AppearanceTextureProperty.name Property

Parent Object: [AppearanceTextureProperty](AppearanceTextureProperty.md)  

## Description

Returns the name of this property as seen in the user interface. This name is localized and can change based on the current language

## Syntax

"appearanceTextureProperty_var" is a variable referencing an AppearanceTextureProperty object.  

```python
# Get the value of the property.
propertyValue = appearanceTextureProperty_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2014
