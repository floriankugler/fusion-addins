# GenerateToolpathFuture.numberOfTasks Property

Parent Object: [GenerateToolpathFuture](GenerateToolpathFuture.md)  

## Description

Returns the number of tasks of operations that need to be generated.

## Syntax

"generateToolpathFuture_var" is a variable referencing a GenerateToolpathFuture object.  

```python
# Get the value of the property.
propertyValue = generateToolpathFuture_var.numberOfTasks
```

## Property Value

This is a read only property whose value is an integer.

## Version

Introduced in version July 2023
