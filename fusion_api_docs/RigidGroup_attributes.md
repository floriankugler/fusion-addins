# RigidGroup.attributes Property

Parent Object: [RigidGroup](RigidGroup.md)  

## Description

Returns the collection of attributes associated with this rigid group.

## Syntax

"rigidGroup_var" is a variable referencing a RigidGroup object.  

```python
# Get the value of the property.
propertyValue = rigidGroup_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
