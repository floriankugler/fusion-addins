# Components.itemByName Method

Parent Object: [Components](Components.md)  

## Description

Function that returns the specified component by name.

## Syntax

"components_var" is a variable referencing a [Components](Components.md) object.

```python
returnValue = components_var.itemByName(name)
```

## Return Value

| Type | Description |
|----|----|
| [Component](Component.md) | Returns the specified component or null if the name is not found. |

## Parameters

| Name | Type   | Description                                                |
|------|--------|------------------------------------------------------------|
| name | string | The name of the component within the collection to return. |

## Version

Introduced in version January 2016
