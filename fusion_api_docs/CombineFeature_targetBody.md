# CombineFeature.targetBody Property

Parent Object: [CombineFeature](CombineFeature.md)  

## Description

## Syntax

"combineFeature_var" is a variable referencing a CombineFeature object.  

```python
# Get the value of the property.
propertyValue = combineFeature_var.targetBody

# Set the value of the property.
combineFeature_var.targetBody = propertyValue
```

## Property Value

This is a read/write property whose value is a [BRepBody](BRepBody.md).

## Version

Introduced in version November 2014
