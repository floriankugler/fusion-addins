# FlatPatternProduct.isConfiguredDesign Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Gets if this design is a configured design. A configured design contains a configuration table. Use the configurationTable property to get the associated table.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.isConfiguredDesign
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2024
