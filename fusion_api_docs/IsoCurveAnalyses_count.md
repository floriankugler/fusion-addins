# IsoCurveAnalyses.count Property

Parent Object: [IsoCurveAnalyses](IsoCurveAnalyses.md)  

## Description

Returns the number of CurvatureCombAnalysis objects in the collection.

## Syntax

"isoCurveAnalyses_var" is a variable referencing an IsoCurveAnalyses object.  

```python
# Get the value of the property.
propertyValue = isoCurveAnalyses_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version January 2023
