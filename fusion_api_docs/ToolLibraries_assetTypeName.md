# ToolLibraries.assetTypeName Property

Parent Object: [ToolLibraries](ToolLibraries.md)  

## Description

Get the name of the asset type which can be accessed by the library.

## Syntax

"toolLibraries_var" is a variable referencing a ToolLibraries object.  

```python
# Get the value of the property.
propertyValue = toolLibraries_var.assetTypeName
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
