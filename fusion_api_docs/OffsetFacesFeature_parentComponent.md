# OffsetFacesFeature.parentComponent Property

Parent Object: [OffsetFacesFeature](OffsetFacesFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"offsetFacesFeature_var" is a variable referencing an OffsetFacesFeature object.  

```python
# Get the value of the property.
propertyValue = offsetFacesFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version June 2017
