# Features.pipeFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the existing pipe features.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.pipeFeatures
```

## Property Value

This is a read only property whose value is a [PipeFeatures](PipeFeatures.md).

## Version

Introduced in version January 2022
