# BoolValueCommandInput.commandInputs Property

Parent Object: [BoolValueCommandInput](BoolValueCommandInput.md)  

## Description

Gets the CommandInputs class of the parent, which can be a Command, GroupCommandInput or TabCommandInput.

## Syntax

"boolValueCommandInput_var" is a variable referencing a BoolValueCommandInput object.  

```python
# Get the value of the property.
propertyValue = boolValueCommandInput_var.commandInputs
```

## Property Value

This is a read only property whose value is a [CommandInputs](CommandInputs.md).

## Version

Introduced in version August 2014
