# CutPasteBody.attributes Property

Parent Object: [CutPasteBody](CutPasteBody.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"cutPasteBody_var" is a variable referencing a CutPasteBody object.  

```python
# Get the value of the property.
propertyValue = cutPasteBody_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version June 2017
