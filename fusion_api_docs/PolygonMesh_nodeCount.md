# PolygonMesh.nodeCount Property

Parent Object: [PolygonMesh](PolygonMesh.md)  

## Description

Returns the number of nodes in the mesh.

## Syntax

"polygonMesh_var" is a variable referencing a PolygonMesh object.  

```python
# Get the value of the property.
propertyValue = polygonMesh_var.nodeCount
```

## Property Value

This is a read only property whose value is an integer.

## Version

Introduced in version August 2014
