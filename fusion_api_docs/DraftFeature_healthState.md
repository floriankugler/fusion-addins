# DraftFeature.healthState Property

Parent Object: [DraftFeature](DraftFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"draftFeature_var" is a variable referencing a DraftFeature object.  

```python
# Get the value of the property.
propertyValue = draftFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
