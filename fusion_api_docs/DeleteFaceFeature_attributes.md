# DeleteFaceFeature.attributes Property

Parent Object: [DeleteFaceFeature](DeleteFaceFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"deleteFaceFeature_var" is a variable referencing a DeleteFaceFeature object.  

```python
# Get the value of the property.
propertyValue = deleteFaceFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version August 2016
