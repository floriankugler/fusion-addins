# TabCommandInput.deleteMe Method

Parent Object: [TabCommandInput](TabCommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"tabCommandInput_var" is a variable referencing a [TabCommandInput](TabCommandInput.md) object.

```python
returnValue = tabCommandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2020
