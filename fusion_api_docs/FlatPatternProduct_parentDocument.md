# FlatPatternProduct.parentDocument Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Returns the parent Document object.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.parentDocument
```

## Property Value

This is a read only property whose value is a [Document](Document.md).

## Version

Introduced in version October 2022
