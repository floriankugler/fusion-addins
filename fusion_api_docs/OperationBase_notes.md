# OperationBase.notes Property

Parent Object: [OperationBase](OperationBase.md)  

## Description

Gets and sets the notes of the operation.

## Syntax

"operationBase_var" is a variable referencing an OperationBase object.  

```python
# Get the value of the property.
propertyValue = operationBase_var.notes

# Set the value of the property.
operationBase_var.notes = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version May 2020
