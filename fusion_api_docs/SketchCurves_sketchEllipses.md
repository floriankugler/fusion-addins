# SketchCurves.sketchEllipses Property

Parent Object: [SketchCurves](SketchCurves.md)  

## Description

Returns the sketch ellipses collection associated with this sketch. This provides access to the existing sketch ellipses and supports the creation of new sketch ellipses.

## Syntax

"sketchCurves_var" is a variable referencing a SketchCurves object.  

```python
# Get the value of the property.
propertyValue = sketchCurves_var.sketchEllipses
```

## Property Value

This is a read only property whose value is a [SketchEllipses](SketchEllipses.md).
