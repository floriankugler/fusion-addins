# SweepFeature.attributes Property

Parent Object: [SweepFeature](SweepFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"sweepFeature_var" is a variable referencing a SweepFeature object.  

```python
# Get the value of the property.
propertyValue = sweepFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
