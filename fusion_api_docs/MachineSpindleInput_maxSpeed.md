# MachineSpindleInput.maxSpeed Property

Parent Object: [MachineSpindleInput](MachineSpindleInput.md)  

## Description

Specifies the maximum speed (rpm) for this spindle.

## Syntax

"machineSpindleInput_var" is a variable referencing a MachineSpindleInput object.  

```python
# Get the value of the property.
propertyValue = machineSpindleInput_var.maxSpeed

# Set the value of the property.
machineSpindleInput_var.maxSpeed = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version April 2023
