# HttpResponse.statusCode Property

Parent Object: [HttpResponse](HttpResponse.md)  

## Description

Gets the status code of the response.

## Syntax

"httpResponse_var" is a variable referencing a HttpResponse object.  

```python
# Get the value of the property.
propertyValue = httpResponse_var.statusCode
```

## Property Value

This is a read only property whose value is an integer.
