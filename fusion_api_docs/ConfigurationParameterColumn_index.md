# ConfigurationParameterColumn.index Property

Parent Object: [ConfigurationParameterColumn](ConfigurationParameterColumn.md)  

## Description

The index position of this column within the table. The first column is at index 0 and does not include the "Name" column.

## Syntax

"configurationParameterColumn_var" is a variable referencing a ConfigurationParameterColumn object.  

```python
# Get the value of the property.
propertyValue = configurationParameterColumn_var.index
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version January 2024
