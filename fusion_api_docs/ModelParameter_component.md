# ModelParameter.component Property

Parent Object: [ModelParameter](ModelParameter.md)  

## Description

Returns the Component containing the ModelParameter.

## Syntax

"modelParameter_var" is a variable referencing a ModelParameter object.  

```python
# Get the value of the property.
propertyValue = modelParameter_var.component
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version August 2014
