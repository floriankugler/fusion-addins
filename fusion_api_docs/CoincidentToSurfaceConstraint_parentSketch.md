# CoincidentToSurfaceConstraint.parentSketch Property

Parent Object: [CoincidentToSurfaceConstraint](CoincidentToSurfaceConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"coincidentToSurfaceConstraint_var" is a variable referencing a CoincidentToSurfaceConstraint object.  

```python
# Get the value of the property.
propertyValue = coincidentToSurfaceConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version September 2023
