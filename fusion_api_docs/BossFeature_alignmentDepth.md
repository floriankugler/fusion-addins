# BossFeature.alignmentDepth Property

Parent Object: [BossFeature](BossFeature.md)  

## Description

Returns the model parameter controlling the step depth used for alignment of its counterparts.

## Syntax

"bossFeature_var" is a variable referencing a BossFeature object.  

```python
# Get the value of the property.
propertyValue = bossFeature_var.alignmentDepth
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version October 2022
