# DataEventHandler.notify Method

Parent Object: [DataEventHandler](DataEventHandler.md)  

## Description

The function called by Fusion when the associated event is fired.

## Syntax

"dataEventHandler_var" is a variable referencing a [DataEventHandler](DataEventHandler.md) object.

```python
returnValue = dataEventHandler_var.notify(eventArgs)
```

## Parameters

| Name | Type | Description |
|----|----|----|
| eventArgs | [DataEventArgs](DataEventArgs.md) | Returns an object that provides access to additional information associated with the event. |

## Version

Introduced in version January 2022
