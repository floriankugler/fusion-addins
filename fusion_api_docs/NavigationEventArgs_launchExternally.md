# NavigationEventArgs.launchExternally Property

Parent Object: [NavigationEventArgs](NavigationEventArgs.md)  

## Description

If True, the URL will be navigated to in an external browser by the operating system. If False, the default value, the URL will be navigated to in the palette's browser.

## Syntax

"navigationEventArgs_var" is a variable referencing a NavigationEventArgs object.  

```python
# Get the value of the property.
propertyValue = navigationEventArgs_var.launchExternally

# Set the value of the property.
navigationEventArgs_var.launchExternally = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version March 2021
