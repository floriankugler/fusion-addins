# SketchEllipseMinorRadiusDimension.ellipse Property

Parent Object: [SketchEllipseMinorRadiusDimension](SketchEllipseMinorRadiusDimension.md)  

## Description

Returns the ellipse or elliptical arc being constrained.

## Syntax

"sketchEllipseMinorRadiusDimension_var" is a variable referencing a SketchEllipseMinorRadiusDimension object.  

```python
# Get the value of the property.
propertyValue = sketchEllipseMinorRadiusDimension_var.ellipse
```

## Property Value

This is a read only property whose value is a [SketchCurve](SketchCurve.md).

## Version

Introduced in version August 2014
