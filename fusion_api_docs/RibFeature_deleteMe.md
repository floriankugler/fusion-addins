# RibFeature.deleteMe Method

Parent Object: [RibFeature](RibFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"ribFeature_var" is a variable referencing a [RibFeature](RibFeature.md) object.

```python
returnValue = ribFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version September 2015
