# Circle2D.copy Method

Parent Object: [Circle2D](Circle2D.md)  

## Description

Creates and returns an independent copy of this Circle2D object.

## Syntax

"circle2D_var" is a variable referencing a [Circle2D](Circle2D.md) object.

```python
returnValue = circle2D_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [Circle2D](Circle2D.md) | Returns an independent copy of this Circle2D object. |

## Version

Introduced in version August 2014
