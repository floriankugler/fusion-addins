# ConfigurationAppearanceColumn.id Property

Parent Object: [ConfigurationAppearanceColumn](ConfigurationAppearanceColumn.md)  

## Description

The id of the column. This is constant and cannot be changed by the user.

## Syntax

"configurationAppearanceColumn_var" is a variable referencing a ConfigurationAppearanceColumn object.  

```python
# Get the value of the property.
propertyValue = configurationAppearanceColumn_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version January 2024
