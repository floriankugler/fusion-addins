# ChamferFeature.edgeSets Property

Parent Object: [ChamferFeature](ChamferFeature.md)  

## Description

Returns the edge sets associated with this chamfer.

## Syntax

"chamferFeature_var" is a variable referencing a ChamferFeature object.  

```python
# Get the value of the property.
propertyValue = chamferFeature_var.edgeSets
```

## Property Value

This is a read only property whose value is a [ChamferEdgeSets](ChamferEdgeSets.md).

## Version

Introduced in version December 2020
