# ConfigurationVisibilityColumn.parentTable Property

Parent Object: [ConfigurationVisibilityColumn](ConfigurationVisibilityColumn.md)  

## Description

Returns the parent table, either top or custom theme table, this column is in.

## Syntax

"configurationVisibilityColumn_var" is a variable referencing a ConfigurationVisibilityColumn object.  

```python
# Get the value of the property.
propertyValue = configurationVisibilityColumn_var.parentTable
```

## Property Value

This is a read only property whose value is a [ConfigurationTable](ConfigurationTable.md).

## Version

Introduced in version January 2024
