# DeleteFaceFeature.timelineObject Property

Parent Object: [DeleteFaceFeature](DeleteFaceFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"deleteFaceFeature_var" is a variable referencing a DeleteFaceFeature object.  

```python
# Get the value of the property.
propertyValue = deleteFaceFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version August 2016
