# NurbsCurve3D.controlPoints Property

Parent Object: [NurbsCurve3D](NurbsCurve3D.md)  

## Description

Returns an array of Point3D objects that define the control points of the curve.

## Syntax

"nurbsCurve3D_var" is a variable referencing a NurbsCurve3D object.  

```python
# Get the value of the property.
propertyValue = nurbsCurve3D_var.controlPoints
```

## Property Value

This is a read only property whose value is an array of type [Point3D](Point3D.md).

## Version

Introduced in version August 2014
