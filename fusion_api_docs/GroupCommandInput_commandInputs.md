# GroupCommandInput.commandInputs Property

Parent Object: [GroupCommandInput](GroupCommandInput.md)  

## Description

Gets the CommandInputs class of the parent, which can be a Command, GroupCommandInput or TabCommandInput.

## Syntax

"groupCommandInput_var" is a variable referencing a GroupCommandInput object.  

```python
# Get the value of the property.
propertyValue = groupCommandInput_var.commandInputs
```

## Property Value

This is a read only property whose value is a [CommandInputs](CommandInputs.md).

## Version

Introduced in version July 2015
