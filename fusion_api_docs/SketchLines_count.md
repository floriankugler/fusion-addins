# SketchLines.count Property

Parent Object: [SketchLines](SketchLines.md)  

## Description

Returns the number of lines in the sketch.

## Syntax

"sketchLines_var" is a variable referencing a SketchLines object.  

```python
# Get the value of the property.
propertyValue = sketchLines_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version August 2014
