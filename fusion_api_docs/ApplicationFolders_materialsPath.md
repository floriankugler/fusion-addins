# ApplicationFolders.materialsPath Property

Parent Object: [ApplicationFolders](ApplicationFolders.md)  

## Description

Returns the path where user-created material and appearance libraries are saved.

## Syntax

"applicationFolders_var" is a variable referencing an ApplicationFolders object.  

```python
# Get the value of the property.
propertyValue = applicationFolders_var.materialsPath
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version September 2024
