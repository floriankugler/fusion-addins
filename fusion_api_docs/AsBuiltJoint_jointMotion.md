# AsBuiltJoint.jointMotion Property

Parent Object: [AsBuiltJoint](AsBuiltJoint.md)  

## Description

Returns a JointMotion object that defines the motion relationship between the two geometries.

## Syntax

"asBuiltJoint_var" is a variable referencing an AsBuiltJoint object.  

```python
# Get the value of the property.
propertyValue = asBuiltJoint_var.jointMotion
```

## Property Value

This is a read only property whose value is a [JointMotion](JointMotion.md).

## Version

Introduced in version September 2015
