# AlongEdgeRipFeatureDefinition.gapDistance Property

Parent Object: [AlongEdgeRipFeatureDefinition](AlongEdgeRipFeatureDefinition.md)  

## Description

Gets the ModelParameter that defines the gap distance of the rip. The value can be edited by using the properties of the returned ModelParameter object.

## Syntax

"alongEdgeRipFeatureDefinition_var" is a variable referencing an AlongEdgeRipFeatureDefinition object.  

```python
# Get the value of the property.
propertyValue = alongEdgeRipFeatureDefinition_var.gapDistance
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version September 2023
