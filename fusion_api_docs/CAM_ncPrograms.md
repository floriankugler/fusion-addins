# CAM.ncPrograms Property

Parent Object: [CAM](CAM.md)  

## Description

Returns the collection of NC programs in the document.

## Syntax

"cAM_var" is a variable referencing a CAM object.  

```python
# Get the value of the property.
propertyValue = cAM_var.ncPrograms
```

## Property Value

This is a read only property whose value is a [NCPrograms](NCPrograms.md).

## Version

Introduced in version April 2023
