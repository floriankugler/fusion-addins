# GeneralPreferences.isGestureBasedViewNavigationUsed Property

Parent Object: [GeneralPreferences](GeneralPreferences.md)  

## Description

Gets and sets if gesture based view navigation is used.

## Syntax

"generalPreferences_var" is a variable referencing a GeneralPreferences object.  

```python
# Get the value of the property.
propertyValue = generalPreferences_var.isGestureBasedViewNavigationUsed

# Set the value of the property.
generalPreferences_var.isGestureBasedViewNavigationUsed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
