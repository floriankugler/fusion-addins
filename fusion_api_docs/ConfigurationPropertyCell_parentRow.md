# ConfigurationPropertyCell.parentRow Property

Parent Object: [ConfigurationPropertyCell](ConfigurationPropertyCell.md)  

## Description

Returns the row this cell is in.

## Syntax

"configurationPropertyCell_var" is a variable referencing a ConfigurationPropertyCell object.  

```python
# Get the value of the property.
propertyValue = configurationPropertyCell_var.parentRow
```

## Property Value

This is a read only property whose value is a [ConfigurationRow](ConfigurationRow.md).

## Version

Introduced in version January 2024
