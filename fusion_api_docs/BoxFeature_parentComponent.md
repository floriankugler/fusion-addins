# BoxFeature.parentComponent Property

Parent Object: [BoxFeature](BoxFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"boxFeature_var" is a variable referencing a BoxFeature object.  

```python
# Get the value of the property.
propertyValue = boxFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version September 2015
