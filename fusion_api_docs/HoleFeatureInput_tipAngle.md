# HoleFeatureInput.tipAngle Property

Parent Object: [HoleFeatureInput](HoleFeatureInput.md)  

## Description

Gets the ValueInput object that defines the angle of the tip of the hole. The default is "118.0 deg" but can be modified by setting it using another Value object.

## Syntax

"holeFeatureInput_var" is a variable referencing a HoleFeatureInput object.  

```python
# Get the value of the property.
propertyValue = holeFeatureInput_var.tipAngle

# Set the value of the property.
holeFeatureInput_var.tipAngle = propertyValue
```

## Property Value

This is a read/write property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version August 2014
