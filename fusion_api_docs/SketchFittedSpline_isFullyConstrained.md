# SketchFittedSpline.isFullyConstrained Property

Parent Object: [SketchFittedSpline](SketchFittedSpline.md)  

## Description

Indicates if this sketch entity is fully constrained.

## Syntax

"sketchFittedSpline_var" is a variable referencing a SketchFittedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFittedSpline_var.isFullyConstrained
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2016
