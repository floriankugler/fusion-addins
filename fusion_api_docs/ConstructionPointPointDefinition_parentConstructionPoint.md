# ConstructionPointPointDefinition.parentConstructionPoint Property

Parent Object: [ConstructionPointPointDefinition](ConstructionPointPointDefinition.md)  

## Description

Returns the ConstructionPoint object

## Syntax

"constructionPointPointDefinition_var" is a variable referencing a ConstructionPointPointDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionPointPointDefinition_var.parentConstructionPoint
```

## Property Value

This is a read only property whose value is a [ConstructionPoint](ConstructionPoint.md).

## Version

Introduced in version August 2014
