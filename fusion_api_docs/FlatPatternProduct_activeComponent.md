# FlatPatternProduct.activeComponent Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Returns the component that is current being edited. This can return the root component or another component within the design.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.activeComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version October 2022
