# Material.parent Property

Parent Object: [Material](Material.md)  

## Description

Returns the Parent object (a Library or a Design).

## Syntax

"material_var" is a variable referencing a Material object.  

```python
# Get the value of the property.
propertyValue = material_var.parent
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version August 2014
