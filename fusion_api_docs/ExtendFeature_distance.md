# ExtendFeature.distance Property

Parent Object: [ExtendFeature](ExtendFeature.md)  

## Description

Returns the parameter controlling the extend distance. You can edit the distance by editing the value of the parameter object.

## Syntax

"extendFeature_var" is a variable referencing an ExtendFeature object.  

```python
# Get the value of the property.
propertyValue = extendFeature_var.distance
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version June 2015
