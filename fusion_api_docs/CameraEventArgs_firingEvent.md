# CameraEventArgs.firingEvent Property

Parent Object: [CameraEventArgs](CameraEventArgs.md)  

## Description

The event that the firing is in response to.

## Syntax

"cameraEventArgs_var" is a variable referencing a CameraEventArgs object.  

```python
# Get the value of the property.
propertyValue = cameraEventArgs_var.firingEvent
```

## Property Value

This is a read only property whose value is an [Event](Event.md).

## Version

Introduced in version December 2017
