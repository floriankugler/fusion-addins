# Polyline2D.transformBy Method

Parent Object: [Polyline2D](Polyline2D.md)  

## Description

Transforms this curve in 2D space.

## Syntax

"polyline2D_var" is a variable referencing a [Polyline2D](Polyline2D.md) object.

```python
returnValue = polyline2D_var.transformBy(matrix)
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Return true if the transform was successful. |

## Parameters

| Name | Type | Description |
|----|----|----|
| matrix | [Matrix2D](Matrix2D.md) | A 2D matrix that defines the transform to apply to the curve. |

## Version

Introduced in version September 2024
