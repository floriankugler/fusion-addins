# DataFile.latestVersion Property

Parent Object: [DataFile](DataFile.md)  

## Description

Returns the latest version of the DataFile. It can return a reference to the same DataFile is this DataFile is the latest version.

## Syntax

"dataFile_var" is a variable referencing a DataFile object.  

```python
# Get the value of the property.
propertyValue = dataFile_var.latestVersion
```

## Property Value

This is a read only property whose value is a [DataFile](DataFile.md).

## Version

Introduced in version September 2016
