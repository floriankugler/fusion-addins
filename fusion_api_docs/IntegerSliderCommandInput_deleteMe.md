# IntegerSliderCommandInput.deleteMe Method

Parent Object: [IntegerSliderCommandInput](IntegerSliderCommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"integerSliderCommandInput_var" is a variable referencing an [IntegerSliderCommandInput](IntegerSliderCommandInput.md) object.

```python
returnValue = integerSliderCommandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2020
