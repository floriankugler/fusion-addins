# Joint.isSuppressed Property

Parent Object: [Joint](Joint.md)  

## Description

Gets and sets if this joint is suppressed.

## Syntax

"joint_var" is a variable referencing a Joint object.  

```python
# Get the value of the property.
propertyValue = joint_var.isSuppressed

# Set the value of the property.
joint_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2016
