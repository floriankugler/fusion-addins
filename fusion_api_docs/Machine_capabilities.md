# Machine.capabilities Property

Parent Object: [Machine](Machine.md)  

## Description

Gets the capabilities of the machine.

## Syntax

"machine_var" is a variable referencing a Machine object.  

```python
# Get the value of the property.
propertyValue = machine_var.capabilities
```

## Property Value

This is a read only property whose value is a [MachineCapabilities](MachineCapabilities.md).

## Version

Introduced in version April 2023
