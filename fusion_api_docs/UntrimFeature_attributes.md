# UntrimFeature.attributes Property

Parent Object: [UntrimFeature](UntrimFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"untrimFeature_var" is a variable referencing a UntrimFeature object.  

```python
# Get the value of the property.
propertyValue = untrimFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version January 2021
