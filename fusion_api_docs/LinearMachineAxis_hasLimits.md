# LinearMachineAxis.hasLimits Property

Parent Object: [LinearMachineAxis](LinearMachineAxis.md)  

## Description

Does this axis have a limited range of motion.

## Syntax

"linearMachineAxis_var" is a variable referencing a LinearMachineAxis object.  

```python
# Get the value of the property.
propertyValue = linearMachineAxis_var.hasLimits
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version April 2023
