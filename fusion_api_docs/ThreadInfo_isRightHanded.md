# ThreadInfo.isRightHanded Property

Parent Object: [ThreadInfo](ThreadInfo.md)  

## Description

Gets and sets if the thread is right or left-handed thread. A value of true indicates a right-handed thread. It defaults to true.

## Syntax

"threadInfo_var" is a variable referencing a ThreadInfo object.  

```python
# Get the value of the property.
propertyValue = threadInfo_var.isRightHanded

# Set the value of the property.
threadInfo_var.isRightHanded = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version September 2025
