# EqualConstraint.parentSketch Property

Parent Object: [EqualConstraint](EqualConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"equalConstraint_var" is a variable referencing an EqualConstraint object.  

```python
# Get the value of the property.
propertyValue = equalConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
