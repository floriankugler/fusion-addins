# GeometricConstraint.deleteMe Method

Parent Object: [GeometricConstraint](GeometricConstraint.md)  

## Description

Deletes this constraint. The IsDeletable property can be used to determine if this constraint can be deleted.

## Syntax

"geometricConstraint_var" is a variable referencing a [GeometricConstraint](GeometricConstraint.md) object.

```python
returnValue = geometricConstraint_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
