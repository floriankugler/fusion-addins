# Feature.parentComponent Property

Parent Object: [Feature](Feature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"feature_var" is a variable referencing a Feature object.  

```python
# Get the value of the property.
propertyValue = feature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version August 2014
