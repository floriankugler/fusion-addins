# FlatPatternProduct.attributes Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Returns the collection of attributes associated with this product.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version October 2022
