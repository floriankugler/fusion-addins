# Document.parent Property

Parent Object: [Document](Document.md)  

## Description

Returns the parent Application object.

## Syntax

"document_var" is a variable referencing a Document object.  

```python
# Get the value of the property.
propertyValue = document_var.parent
```

## Property Value

This is a read only property whose value is an [Application](Application.md).

## Version

Introduced in version August 2014
