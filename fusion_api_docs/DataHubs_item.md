# DataHubs.item Method

Parent Object: [DataHubs](DataHubs.md)  

## Description

Returns the specified hub.

## Syntax

"dataHubs_var" is a variable referencing a [DataHubs](DataHubs.md) object.

```python
returnValue = dataHubs_var.item(index)
```

## Return Value

| Type | Description |
|----|----|
| [DataHub](DataHub.md) | Returns the specified item or null if an invalid index was specified. |

## Parameters

| Name | Type | Description |
|----|----|----|
| index | uinteger | The index of the hub to return. The first hub in the list has an index of 0. |

## Version

Introduced in version September 2016
