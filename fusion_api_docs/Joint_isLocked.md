# Joint.isLocked Property

Parent Object: [Joint](Joint.md)  

## Description

Gets and sets if the joint is locked.

## Syntax

"joint_var" is a variable referencing a Joint object.  

```python
# Get the value of the property.
propertyValue = joint_var.isLocked

# Set the value of the property.
joint_var.isLocked = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.
