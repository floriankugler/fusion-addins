# MoveFeatureTranslateXYZDefinition.xDistance Property

Parent Object: [MoveFeatureTranslateXYZDefinition](MoveFeatureTranslateXYZDefinition.md)  

## Description

Gets the model parameter that controls the X distance of the translation. You can use properties on the returned ModelParameter object to edit the offset distance.

## Syntax

"moveFeatureTranslateXYZDefinition_var" is a variable referencing a MoveFeatureTranslateXYZDefinition object.  

```python
# Get the value of the property.
propertyValue = moveFeatureTranslateXYZDefinition_var.xDistance
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version January 2023
