# GeneralPreferences.userInterfaceTheme Property

Parent Object: [GeneralPreferences](GeneralPreferences.md)  

## Description

Gets and sets which color theme is used by the user interface.

## Syntax

"generalPreferences_var" is a variable referencing a GeneralPreferences object.  

```python
# Get the value of the property.
propertyValue = generalPreferences_var.userInterfaceTheme

# Set the value of the property.
generalPreferences_var.userInterfaceTheme = propertyValue
```

## Property Value

This is a read/write property whose value is a [UserInterfaceThemes](UserInterfaceThemes.md).

## Version

Introduced in version January 2025
