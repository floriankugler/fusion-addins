# TimelineObject.healthState Property

Parent Object: [TimelineObject](TimelineObject.md)  

## Description

Returns the current health state of the object associated with this TimelineObject.

## Syntax

"timelineObject_var" is a variable referencing a TimelineObject object.  

```python
# Get the value of the property.
propertyValue = timelineObject_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).
