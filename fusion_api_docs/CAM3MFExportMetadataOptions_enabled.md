# CAM3MFExportMetadataOptions.enabled Property

Parent Object: [CAM3MFExportMetadataOptions](CAM3MFExportMetadataOptions.md)  

## Description

Enable or disable the integration of Metadata in the 3mf. This is true by default.

## Syntax

"cAM3MFExportMetadataOptions_var" is a variable referencing a CAM3MFExportMetadataOptions object.  

```python
# Get the value of the property.
propertyValue = cAM3MFExportMetadataOptions_var.enabled

# Set the value of the property.
cAM3MFExportMetadataOptions_var.enabled = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version May 2023
