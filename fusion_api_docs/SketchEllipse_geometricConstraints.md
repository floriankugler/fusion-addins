# SketchEllipse.geometricConstraints Property

Parent Object: [SketchEllipse](SketchEllipse.md)  

## Description

Returns the sketch constraints that are attached to this curve.

## Syntax

"sketchEllipse_var" is a variable referencing a SketchEllipse object.  

```python
# Get the value of the property.
propertyValue = sketchEllipse_var.geometricConstraints
```

## Property Value

This is a read only property whose value is a [GeometricConstraintList](GeometricConstraintList.md).

## Version

Introduced in version January 2015
