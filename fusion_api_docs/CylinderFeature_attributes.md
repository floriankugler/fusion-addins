# CylinderFeature.attributes Property

Parent Object: [CylinderFeature](CylinderFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"cylinderFeature_var" is a variable referencing a CylinderFeature object.  

```python
# Get the value of the property.
propertyValue = cylinderFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
