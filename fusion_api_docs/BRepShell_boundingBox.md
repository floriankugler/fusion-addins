# BRepShell.boundingBox Property

Parent Object: [BRepShell](BRepShell.md)  

## Description

Returns the bounding box of this shell

## Syntax

"bRepShell_var" is a variable referencing a BRepShell object.  

```python
# Get the value of the property.
propertyValue = bRepShell_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version August 2014
