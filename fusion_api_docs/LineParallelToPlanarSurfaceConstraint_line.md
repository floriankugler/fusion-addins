# LineParallelToPlanarSurfaceConstraint.line Property

Parent Object: [LineParallelToPlanarSurfaceConstraint](LineParallelToPlanarSurfaceConstraint.md)  

## Description

Returns the SketchLine object that is constrained.

## Syntax

"lineParallelToPlanarSurfaceConstraint_var" is a variable referencing a LineParallelToPlanarSurfaceConstraint object.  

```python
# Get the value of the property.
propertyValue = lineParallelToPlanarSurfaceConstraint_var.line
```

## Property Value

This is a read only property whose value is a [SketchLine](SketchLine.md).

## Version

Introduced in version September 2023
