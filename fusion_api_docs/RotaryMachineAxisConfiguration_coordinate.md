# RotaryMachineAxisConfiguration.coordinate Property

Parent Object: [RotaryMachineAxisConfiguration](RotaryMachineAxisConfiguration.md)  

## Description

Coordinate to use for post processing.

## Syntax

"rotaryMachineAxisConfiguration_var" is a variable referencing a RotaryMachineAxisConfiguration object.  

```python
# Get the value of the property.
propertyValue = rotaryMachineAxisConfiguration_var.coordinate

# Set the value of the property.
rotaryMachineAxisConfiguration_var.coordinate = propertyValue
```

## Property Value

This is a read/write property whose value is a [MachineAxisCoordinates](MachineAxisCoordinates.md).

## Version

Introduced in version April 2023
