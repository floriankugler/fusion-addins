# LineParallelToPlanarSurfaceConstraint.attributes Property

Parent Object: [LineParallelToPlanarSurfaceConstraint](LineParallelToPlanarSurfaceConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"lineParallelToPlanarSurfaceConstraint_var" is a variable referencing a LineParallelToPlanarSurfaceConstraint object.  

```python
# Get the value of the property.
propertyValue = lineParallelToPlanarSurfaceConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version September 2023
