# CustomEventHandler Object

Derived from: [EventHandler](EventHandler.md) Object  

## Description

The ApplicationEventHandler is a client implemented class that can be added as a handler to an ApplicationEvent.

## Methods

| Name | Description |
|----|----|
| [notify](CustomEventHandler_notify.md) | The function called by Fusion when the associated event is fired. |

## Version

Introduced in version January 2017
