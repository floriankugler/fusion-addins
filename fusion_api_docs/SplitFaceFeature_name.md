# SplitFaceFeature.name Property

Parent Object: [SplitFaceFeature](SplitFaceFeature.md)  

## Description

Returns the name of the feature as seen in the browser (non-parametric) or in the timeline (parametric).

## Syntax

"splitFaceFeature_var" is a variable referencing a SplitFaceFeature object.  

```python
# Get the value of the property.
propertyValue = splitFaceFeature_var.name

# Set the value of the property.
splitFaceFeature_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version June 2015
