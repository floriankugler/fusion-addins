# Appearance.name Property

Parent Object: [Appearance](Appearance.md)  

## Description

Returns the name of this Appearance. This is the localized name shown in the UI.

## Syntax

"appearance_var" is a variable referencing an Appearance object.  

```python
# Get the value of the property.
propertyValue = appearance_var.name

# Set the value of the property.
appearance_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.
