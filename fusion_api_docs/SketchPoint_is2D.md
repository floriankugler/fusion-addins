# SketchPoint.is2D Property

Parent Object: [SketchPoint](SketchPoint.md)  

## Description

Indicates if this curve lies entirely on the sketch x-y plane.

## Syntax

"sketchPoint_var" is a variable referencing a SketchPoint object.  

```python
# Get the value of the property.
propertyValue = sketchPoint_var.is2D
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
