# CurvatureMapAnalysis.deleteMe Method

Parent Object: [CurvatureMapAnalysis](CurvatureMapAnalysis.md)  

## Description

A method that deletes this Analysis.

## Syntax

"curvatureMapAnalysis_var" is a variable referencing a [CurvatureMapAnalysis](CurvatureMapAnalysis.md) object.

```python
returnValue = curvatureMapAnalysis_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version January 2023
