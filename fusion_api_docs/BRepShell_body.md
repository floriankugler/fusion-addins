# BRepShell.body Property

Parent Object: [BRepShell](BRepShell.md)  

## Description

Returns the parent body of the shell.

## Syntax

"bRepShell_var" is a variable referencing a BRepShell object.  

```python
# Get the value of the property.
propertyValue = bRepShell_var.body
```

## Property Value

This is a read only property whose value is a [BRepBody](BRepBody.md).

## Version

Introduced in version August 2014
