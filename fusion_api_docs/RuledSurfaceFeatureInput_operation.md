# RuledSurfaceFeatureInput.operation Property

Parent Object: [RuledSurfaceFeatureInput](RuledSurfaceFeatureInput.md)  

## Description

Gets and sets the type of operation performed by the ruled surface.

## Syntax

"ruledSurfaceFeatureInput_var" is a variable referencing a RuledSurfaceFeatureInput object.  

```python
# Get the value of the property.
propertyValue = ruledSurfaceFeatureInput_var.operation

# Set the value of the property.
ruledSurfaceFeatureInput_var.operation = propertyValue
```

## Property Value

This is a read/write property whose value is a [FeatureOperations](FeatureOperations.md).

## Version

Introduced in version July 2021
