# ThroughAllExtentDefinition.create Method

Parent Object: [ThroughAllExtentDefinition](ThroughAllExtentDefinition.md)  

## Description

Statically creates a new ThroughAllExtentDefinition object. This is used as input when defining the extents of a feature to be through all.

## Syntax

This is a static method.  

```python

returnValue = adsk.fusion.ThroughAllExtentDefinition.create()
```

## Return Value

| Type | Description |
|----|----|
| [ThroughAllExtentDefinition](ThroughAllExtentDefinition.md) | Returns the newly created ThroughAllExtentDefinition or null in the case of a failure. |
