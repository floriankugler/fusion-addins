# TwoDistancesChamferEdgeSet.distanceOne Property

Parent Object: [TwoDistancesChamferEdgeSet](TwoDistancesChamferEdgeSet.md)  

## Description

Returns the model parameter that controls the first offset distance of the chamfer. You can edit the distance by using the properties on the returned ModelParameter object.

## Syntax

"twoDistancesChamferEdgeSet_var" is a variable referencing a TwoDistancesChamferEdgeSet object.  

```python
# Get the value of the property.
propertyValue = twoDistancesChamferEdgeSet_var.distanceOne
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version December 2020
