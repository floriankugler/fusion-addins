# Material.materialProperties Property

Parent Object: [Material](Material.md)  

## Description

Returns the collection of material properties associated with this material.

## Syntax

"material_var" is a variable referencing a Material object.  

```python
# Get the value of the property.
propertyValue = material_var.materialProperties
```

## Property Value

This is a read only property whose value is a [Properties](Properties.md).

## Version

Introduced in version August 2014
