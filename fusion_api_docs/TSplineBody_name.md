# TSplineBody.name Property

Parent Object: [TSplineBody](TSplineBody.md)  

## Description

Gets and sets the name of the body. If setting this property, there is the side-effect that the B-Rep body created from this T-Spline body is also renamed.

## Syntax

"tSplineBody_var" is a variable referencing a TSplineBody object.  

```python
# Get the value of the property.
propertyValue = tSplineBody_var.name

# Set the value of the property.
tSplineBody_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version April 2019
