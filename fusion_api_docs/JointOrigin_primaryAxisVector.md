# JointOrigin.primaryAxisVector Property

Parent Object: [JointOrigin](JointOrigin.md)  

## Description

Returns the direction of the primary axis that's been calculated for this joint origin. This is conceptually the Z axis as shown by the triad representing the joint origin.

## Syntax

"jointOrigin_var" is a variable referencing a JointOrigin object.  

```python
# Get the value of the property.
propertyValue = jointOrigin_var.primaryAxisVector
```

## Property Value

This is a read only property whose value is a [Vector3D](Vector3D.md).

## Version

Introduced in version September 2015
