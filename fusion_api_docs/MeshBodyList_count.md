# MeshBodyList.count Property

Parent Object: [MeshBodyList](MeshBodyList.md)  

## Description

Returns the number of mesh bodies in the collection.

## Syntax

"meshBodyList_var" is a variable referencing a MeshBodyList object.  

```python
# Get the value of the property.
propertyValue = meshBodyList_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version August 2014
