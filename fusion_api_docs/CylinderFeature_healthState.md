# CylinderFeature.healthState Property

Parent Object: [CylinderFeature](CylinderFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"cylinderFeature_var" is a variable referencing a CylinderFeature object.  

```python
# Get the value of the property.
propertyValue = cylinderFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
