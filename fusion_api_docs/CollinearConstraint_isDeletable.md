# CollinearConstraint.isDeletable Property

Parent Object: [CollinearConstraint](CollinearConstraint.md)  

## Description

Indicates if this constraint is deletable.

## Syntax

"collinearConstraint_var" is a variable referencing a CollinearConstraint object.  

```python
# Get the value of the property.
propertyValue = collinearConstraint_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
