# BRepFace.shell Property

Parent Object: [BRepFace](BRepFace.md)  

## Description

Returns the parent shell of the face.

## Syntax

"bRepFace_var" is a variable referencing a BRepFace object.  

```python
# Get the value of the property.
propertyValue = bRepFace_var.shell
```

## Property Value

This is a read only property whose value is a [BRepShell](BRepShell.md).

## Version

Introduced in version August 2014
