# AdditiveFFFLimitsMachineElement.maximumBedTemperature Property

Parent Object: [AdditiveFFFLimitsMachineElement](AdditiveFFFLimitsMachineElement.md)  

## Description

Maximum bed temperature in degrees C.

## Syntax

"additiveFFFLimitsMachineElement_var" is a variable referencing an AdditiveFFFLimitsMachineElement object.  

```python
# Get the value of the property.
propertyValue = additiveFFFLimitsMachineElement_var.maximumBedTemperature

# Set the value of the property.
additiveFFFLimitsMachineElement_var.maximumBedTemperature = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version July 2023
