# DataFile.parentProject Property

Parent Object: [DataFile](DataFile.md)  

## Description

Returns the parent project that this item is in.

## Syntax

"dataFile_var" is a variable referencing a DataFile object.  

```python
# Get the value of the property.
propertyValue = dataFile_var.parentProject
```

## Property Value

This is a read only property whose value is a [DataProject](DataProject.md).

## Version

Introduced in version January 2015
