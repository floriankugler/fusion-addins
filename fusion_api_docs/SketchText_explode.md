# SketchText.explode Method

Parent Object: [SketchText](SketchText.md)  

## Description

Explodes the SketchText into a set of curves. The original SketchText is deleted as a result of calling this.

## Syntax

"sketchText_var" is a variable referencing a [SketchText](SketchText.md) object.

```python
returnValue = sketchText_var.explode()
```

## Return Value

| Type | Description |
|----|----|
| [SketchCurve](SketchCurve.md)\[\] | Returns an array of the sketch curves that were created that represent the text. |

## Version

Introduced in version August 2016
