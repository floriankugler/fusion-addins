# ToolbarTabList.isValid Property

Parent Object: [ToolbarTabList](ToolbarTabList.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"toolbarTabList_var" is a variable referencing a ToolbarTabList object.  

```python
# Get the value of the property.
propertyValue = toolbarTabList_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2019
