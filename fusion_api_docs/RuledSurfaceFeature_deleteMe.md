# RuledSurfaceFeature.deleteMe Method

Parent Object: [RuledSurfaceFeature](RuledSurfaceFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"ruledSurfaceFeature_var" is a variable referencing a [RuledSurfaceFeature](RuledSurfaceFeature.md) object.

```python
returnValue = ruledSurfaceFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version September 2020
