# HorizontalConstraint.line Property

Parent Object: [HorizontalConstraint](HorizontalConstraint.md)  

## Description

Returns the line being constrained.

## Syntax

"horizontalConstraint_var" is a variable referencing a HorizontalConstraint object.  

```python
# Get the value of the property.
propertyValue = horizontalConstraint_var.line
```

## Property Value

This is a read only property whose value is a [SketchLine](SketchLine.md).

## Version

Introduced in version August 2014
