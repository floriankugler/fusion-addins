# DeriveFeature.parentComponent Property

Parent Object: [DeriveFeature](DeriveFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"deriveFeature_var" is a variable referencing a DeriveFeature object.  

```python
# Get the value of the property.
propertyValue = deriveFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version January 2026
