# SketchLinearDimension.entityOne Property

Parent Object: [SketchLinearDimension](SketchLinearDimension.md)  

## Description

The first entity being constrained.

## Syntax

"sketchLinearDimension_var" is a variable referencing a SketchLinearDimension object.  

```python
# Get the value of the property.
propertyValue = sketchLinearDimension_var.entityOne
```

## Property Value

This is a read only property whose value is a [SketchEntity](SketchEntity.md).

## Version

Introduced in version August 2014
