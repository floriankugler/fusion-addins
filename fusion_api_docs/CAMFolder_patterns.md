# CAMFolder.patterns Property

Parent Object: [CAMFolder](CAMFolder.md)  

## Description

Returns the Patterns collection that provides access to existing patterns in this folder.

## Syntax

"cAMFolder_var" is a variable referencing a CAMFolder object.  

```python
# Get the value of the property.
propertyValue = cAMFolder_var.patterns
```

## Property Value

This is a read only property whose value is a [CAMPatterns](CAMPatterns.md).

## Version

Introduced in version January 2016
