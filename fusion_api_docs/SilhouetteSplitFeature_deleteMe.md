# SilhouetteSplitFeature.deleteMe Method

Parent Object: [SilhouetteSplitFeature](SilhouetteSplitFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"silhouetteSplitFeature_var" is a variable referencing a [SilhouetteSplitFeature](SilhouetteSplitFeature.md) object.

```python
returnValue = silhouetteSplitFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version June 2015
