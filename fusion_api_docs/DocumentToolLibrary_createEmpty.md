# DocumentToolLibrary.createEmpty Method

Parent Object: [DocumentToolLibrary](DocumentToolLibrary.md)  

## Description

Creates an empty ToolLibrary.

## Syntax

This is a static method.  

```python

returnValue = adsk.cam.DocumentToolLibrary.createEmpty()
```

## Return Value

| Type                           | Description                            |
|--------------------------------|----------------------------------------|
| [ToolLibrary](ToolLibrary.md) | Returns the newly created ToolLibrary. |

## Version

Introduced in version April 2023
