# AdditiveFFFLimitsMachineElement.isValid Property

Parent Object: [AdditiveFFFLimitsMachineElement](AdditiveFFFLimitsMachineElement.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"additiveFFFLimitsMachineElement_var" is a variable referencing an AdditiveFFFLimitsMachineElement object.  

```python
# Get the value of the property.
propertyValue = additiveFFFLimitsMachineElement_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2023
