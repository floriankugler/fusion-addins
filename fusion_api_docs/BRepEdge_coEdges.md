# BRepEdge.coEdges Property

Parent Object: [BRepEdge](BRepEdge.md)  

## Description

Returns the BRepCoEdges on the edge.

## Syntax

"bRepEdge_var" is a variable referencing a BRepEdge object.  

```python
# Get the value of the property.
propertyValue = bRepEdge_var.coEdges
```

## Property Value

This is a read only property whose value is a [BRepCoEdges](BRepCoEdges.md).

## Version

Introduced in version August 2014
