# MirrorFeatureInput.mirrorPlane Property

Parent Object: [MirrorFeatureInput](MirrorFeatureInput.md)  

## Description

Gets and sets the mirror plane. This can be either a planar face or construction plane.

## Syntax

"mirrorFeatureInput_var" is a variable referencing a MirrorFeatureInput object.  

```python
# Get the value of the property.
propertyValue = mirrorFeatureInput_var.mirrorPlane

# Set the value of the property.
mirrorFeatureInput_var.mirrorPlane = propertyValue
```

## Property Value

This is a read/write property whose value is a [Base](Base.md).

## Version

Introduced in version November 2014
