# RectangularPatternConstraintInput.distanceOne Property

Parent Object: [RectangularPatternConstraintInput](RectangularPatternConstraintInput.md)  

## Description

Gets and sets the distance in the first direction.

## Syntax

"rectangularPatternConstraintInput_var" is a variable referencing a RectangularPatternConstraintInput object.  

```python
# Get the value of the property.
propertyValue = rectangularPatternConstraintInput_var.distanceOne

# Set the value of the property.
rectangularPatternConstraintInput_var.distanceOne = propertyValue
```

## Property Value

This is a read/write property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version September 2022
