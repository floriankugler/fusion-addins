# LoftFeature.parentComponent Property

Parent Object: [LoftFeature](LoftFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"loftFeature_var" is a variable referencing a LoftFeature object.  

```python
# Get the value of the property.
propertyValue = loftFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version September 2015
