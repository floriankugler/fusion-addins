# AtCenterHolePositionDefinition.centerEdge Property

Parent Object: [AtCenterHolePositionDefinition](AtCenterHolePositionDefinition.md)  

## Description

Returns the circular or elliptical edge the hole is centered at.

## Syntax

"atCenterHolePositionDefinition_var" is a variable referencing an AtCenterHolePositionDefinition object.  

```python
# Get the value of the property.
propertyValue = atCenterHolePositionDefinition_var.centerEdge
```

## Property Value

This is a read only property whose value is a [BRepEdge](BRepEdge.md).

## Version

Introduced in version August 2014
