# WorkspaceEventArgs.workspace Property

Parent Object: [WorkspaceEventArgs](WorkspaceEventArgs.md)  

## Description

Provides access to the workspace.

## Syntax

"workspaceEventArgs_var" is a variable referencing a WorkspaceEventArgs object.  

```python
# Get the value of the property.
propertyValue = workspaceEventArgs_var.workspace
```

## Property Value

This is a read only property whose value is a [Workspace](Workspace.md).

## Version

Introduced in version March 2015
