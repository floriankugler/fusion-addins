# SketchConicCurve.is2D Property

Parent Object: [SketchConicCurve](SketchConicCurve.md)  

## Description

Indicates if this curve lies entirely on the sketch x-y plane.

## Syntax

"sketchConicCurve_var" is a variable referencing a SketchConicCurve object.  

```python
# Get the value of the property.
propertyValue = sketchConicCurve_var.is2D
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2015
