# IGESImportOptions.isValid Property

Parent Object: [IGESImportOptions](IGESImportOptions.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"iGESImportOptions_var" is a variable referencing an IGESImportOptions object.  

```python
# Get the value of the property.
propertyValue = iGESImportOptions_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2015
