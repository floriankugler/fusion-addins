# CoincidentConstraint.isDeletable Property

Parent Object: [CoincidentConstraint](CoincidentConstraint.md)  

## Description

Indicates if this constraint is deletable.

## Syntax

"coincidentConstraint_var" is a variable referencing a CoincidentConstraint object.  

```python
# Get the value of the property.
propertyValue = coincidentConstraint_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
