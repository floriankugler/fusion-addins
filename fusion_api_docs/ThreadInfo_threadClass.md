# ThreadInfo.threadClass Property

Parent Object: [ThreadInfo](ThreadInfo.md)  

## Description

Returns and sets the string that defines the thread class.

## Syntax

"threadInfo_var" is a variable referencing a ThreadInfo object.  

```python
# Get the value of the property.
propertyValue = threadInfo_var.threadClass

# Set the value of the property.
threadInfo_var.threadClass = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version January 2015
