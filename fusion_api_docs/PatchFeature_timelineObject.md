# PatchFeature.timelineObject Property

Parent Object: [PatchFeature](PatchFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"patchFeature_var" is a variable referencing a PatchFeature object.  

```python
# Get the value of the property.
propertyValue = patchFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version September 2015
