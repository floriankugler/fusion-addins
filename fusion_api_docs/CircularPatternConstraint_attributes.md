# CircularPatternConstraint.attributes Property

Parent Object: [CircularPatternConstraint](CircularPatternConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"circularPatternConstraint_var" is a variable referencing a CircularPatternConstraint object.  

```python
# Get the value of the property.
propertyValue = circularPatternConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
