# ParameterValue.parent Property

Parent Object: [ParameterValue](ParameterValue.md)  

## Description

Get the parameter object that the value is associated with.

## Syntax

"parameterValue_var" is a variable referencing a ParameterValue object.  

```python
# Get the value of the property.
propertyValue = parameterValue_var.parent
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version September 2021
