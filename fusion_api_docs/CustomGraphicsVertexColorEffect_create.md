# CustomGraphicsVertexColorEffect.create Method

Parent Object: [CustomGraphicsVertexColorEffect](CustomGraphicsVertexColorEffect.md)  

## Description

Statically creates a new CustomGraphicsVertexColorEffect object.

## Syntax

This is a static method.  

```python

returnValue = adsk.fusion.CustomGraphicsVertexColorEffect.create()
```

## Return Value

| Type | Description |
|----|----|
| [CustomGraphicsVertexColorEffect](CustomGraphicsVertexColorEffect.md) | Returns the created CustomGraphicsVertexColorEffect or null in case of a failure. |

## Version

Introduced in version September 2017
