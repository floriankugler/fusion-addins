# HttpEventHandler Object

Derived from: [EventHandler](EventHandler.md) Object  

## Description

The HttpEventHandler is a client implemented class that can be added as a handler to an HttpEvent.

## Methods

| Name | Description |
|----|----|
| [notify](HttpEventHandler_notify.md) | The function called by Fusion when the associated event is fired. |

## Version

Introduced in version January 2024
