# CustomGraphicsBasicMaterialColorEffect.emissiveColor Property

Parent Object: [CustomGraphicsBasicMaterialColorEffect](CustomGraphicsBasicMaterialColorEffect.md)  

## Description

Gets and sets the emissiveColor associated with this CustomGraphicsBasicMaterialColorEffect object. The emissive color is the primary color of the entity

## Syntax

"customGraphicsBasicMaterialColorEffect_var" is a variable referencing a CustomGraphicsBasicMaterialColorEffect object.  

```python
# Get the value of the property.
propertyValue = customGraphicsBasicMaterialColorEffect_var.emissiveColor

# Set the value of the property.
customGraphicsBasicMaterialColorEffect_var.emissiveColor = propertyValue
```

## Property Value

This is a read/write property whose value is a [Color](Color.md).

## Version

Introduced in version September 2017
