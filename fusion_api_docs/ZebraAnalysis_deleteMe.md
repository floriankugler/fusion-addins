# ZebraAnalysis.deleteMe Method

Parent Object: [ZebraAnalysis](ZebraAnalysis.md)  

## Description

A method that deletes this Analysis.

## Syntax

"zebraAnalysis_var" is a variable referencing a [ZebraAnalysis](ZebraAnalysis.md) object.

```python
returnValue = zebraAnalysis_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version January 2023
