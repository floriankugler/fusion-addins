# DataFile.fileExtension Property

Parent Object: [DataFile](DataFile.md)  

## Description

Gets the file extension for this data file. The file type can be inferred from this.

## Syntax

"dataFile_var" is a variable referencing a DataFile object.  

```python
# Get the value of the property.
propertyValue = dataFile_var.fileExtension
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version September 2016
