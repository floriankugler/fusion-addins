# BossFeature.attributes Property

Parent Object: [BossFeature](BossFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"bossFeature_var" is a variable referencing a BossFeature object.  

```python
# Get the value of the property.
propertyValue = bossFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version October 2022
