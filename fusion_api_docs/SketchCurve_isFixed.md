# SketchCurve.isFixed Property

Parent Object: [SketchCurve](SketchCurve.md)  

## Description

Indicates if this geometry is "fixed".

## Syntax

"sketchCurve_var" is a variable referencing a SketchCurve object.  

```python
# Get the value of the property.
propertyValue = sketchCurve_var.isFixed

# Set the value of the property.
sketchCurve_var.isFixed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
