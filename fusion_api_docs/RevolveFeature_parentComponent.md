# RevolveFeature.parentComponent Property

Parent Object: [RevolveFeature](RevolveFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"revolveFeature_var" is a variable referencing a RevolveFeature object.  

```python
# Get the value of the property.
propertyValue = revolveFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version August 2014
