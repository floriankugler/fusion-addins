# DeriveFeature.sourceEntities Property

Parent Object: [DeriveFeature](DeriveFeature.md)  

## Description

The array of entities that are derived.

## Syntax

"deriveFeature_var" is a variable referencing a DeriveFeature object.  

```python
# Get the value of the property.
propertyValue = deriveFeature_var.sourceEntities
```

## Property Value

This is a read only property whose value is an array of type [Base](Base.md).

## Version

Introduced in version January 2026
