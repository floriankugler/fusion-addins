# Vector3D.crossProduct Method

Parent Object: [Vector3D](Vector3D.md)  

## Description

Returns the cross product between this vector and the specified vector.

## Syntax

"vector3D_var" is a variable referencing a [Vector3D](Vector3D.md) object.

```python
returnValue = vector3D_var.crossProduct(vector)
```

## Return Value

| Type                     | Description                       |
|--------------------------|-----------------------------------|
| [Vector3D](Vector3D.md) | Returns the vector cross product. |

## Parameters

| Name   | Type                     | Description                              |
|--------|--------------------------|------------------------------------------|
| vector | [Vector3D](Vector3D.md) | The vector to take the cross product to. |

## Version

Introduced in version August 2014
