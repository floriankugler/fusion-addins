# RuledSurfaceFeature.attributes Property

Parent Object: [RuledSurfaceFeature](RuledSurfaceFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"ruledSurfaceFeature_var" is a variable referencing a RuledSurfaceFeature object.  

```python
# Get the value of the property.
propertyValue = ruledSurfaceFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version September 2020
