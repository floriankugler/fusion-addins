# TangentRelationship.timelineObject Property

Parent Object: [TangentRelationship](TangentRelationship.md)  

## Description

Returns the timeline object associated with this tangent relationship.

## Syntax

"tangentRelationship_var" is a variable referencing a TangentRelationship object.  

```python
# Get the value of the property.
propertyValue = tangentRelationship_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version May 2022
