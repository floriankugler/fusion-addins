# DefaultUnitsPreferences.name Property

Parent Object: [DefaultUnitsPreferences](DefaultUnitsPreferences.md)  

## Description

Returns the name of this DefaultUnitPreferences object.

## Syntax

"defaultUnitsPreferences_var" is a variable referencing a DefaultUnitsPreferences object.  

```python
# Get the value of the property.
propertyValue = defaultUnitsPreferences_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2014
