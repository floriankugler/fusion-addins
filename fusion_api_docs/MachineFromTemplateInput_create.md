# MachineFromTemplateInput.create Method

Parent Object: [MachineFromTemplateInput](MachineFromTemplateInput.md)  

## Description

Create a "MachineFromTemplateInput" object to be used as input for "Machine.create(MachineInput)" method.

## Syntax

This is a static method.  

```python

returnValue = adsk.cam.MachineFromTemplateInput.create(machineTemplate)
```

## Return Value

| Type | Description |
|----|----|
| [MachineFromTemplateInput](MachineFromTemplateInput.md) | The newly created "MachineFromTemplateInput" object in a valid state. |

## Parameters

| Name | Type | Description |
|----|----|----|
| machineTemplate | [MachineTemplate](MachineTemplate.md) | The template to act as a base for creating a machine from. |

## Version

Introduced in version April 2023
