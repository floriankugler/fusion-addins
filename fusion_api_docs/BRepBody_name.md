# BRepBody.name Property

Parent Object: [BRepBody](BRepBody.md)  

## Description

Gets and sets the name of the body.  

This property is only valid if the IsTransient property is false.

## Syntax

"bRepBody_var" is a variable referencing a BRepBody object.  

```python
# Get the value of the property.
propertyValue = bRepBody_var.name

# Set the value of the property.
bRepBody_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version August 2014
