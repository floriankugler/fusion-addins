# InfiniteLine3D.isColinearTo Method

Parent Object: [InfiniteLine3D](InfiniteLine3D.md)  

## Description

Compare this line with another to check for collinearity.

## Syntax

"infiniteLine3D_var" is a variable referencing an [InfiniteLine3D](InfiniteLine3D.md) object.

```python
returnValue = infiniteLine3D_var.isColinearTo(line)
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns true if the two lines are collinear. |

## Parameters

| Name | Type | Description |
|----|----|----|
| line | [InfiniteLine3D](InfiniteLine3D.md) | The line to compare with for collinearity. |

## Version

Introduced in version August 2014
