# ConfigurationPlasticRuleTable.clear Method

Parent Object: [ConfigurationPlasticRuleTable](ConfigurationPlasticRuleTable.md)  

## Description

Clears the content of the plastic rule table, removes the reference from the top table, and hides it in the user interface.

## Syntax

"configurationPlasticRuleTable_var" is a variable referencing a [ConfigurationPlasticRuleTable](ConfigurationPlasticRuleTable.md) object.

```python
returnValue = configurationPlasticRuleTable_var.clear()
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Version

Introduced in version March 2024
