# Features.baseFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the existing base features and supports the creation of new base features. A base feature represents a body that is non-parametric.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.baseFeatures
```

## Property Value

This is a read only property whose value is a [BaseFeatures](BaseFeatures.md).
