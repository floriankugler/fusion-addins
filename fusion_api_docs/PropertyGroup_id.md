# PropertyGroup.id Property

Parent Object: [PropertyGroup](PropertyGroup.md)  

## Description

Returns the unique ID of this property.

## Syntax

"propertyGroup_var" is a variable referencing a PropertyGroup object.  

```python
# Get the value of the property.
propertyValue = propertyGroup_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version January 2024
