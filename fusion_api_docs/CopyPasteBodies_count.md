# CopyPasteBodies.count Property

Parent Object: [CopyPasteBodies](CopyPasteBodies.md)  

## Description

The number of Copy/Paste Body features in the collection.

## Syntax

"copyPasteBodies_var" is a variable referencing a CopyPasteBodies object.  

```python
# Get the value of the property.
propertyValue = copyPasteBodies_var.count
```

## Property Value

This is a read only property whose value is a uinteger.
