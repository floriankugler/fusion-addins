# ScriptInput.author Property

Parent Object: [ScriptInput](ScriptInput.md)  

## Description

The author of the add-in that is displayed in the "Scripts and Add-Ins" dialog. This defaults to an empty string.

## Syntax

"scriptInput_var" is a variable referencing a ScriptInput object.  

```python
# Get the value of the property.
propertyValue = scriptInput_var.author

# Set the value of the property.
scriptInput_var.author = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version October 2023
