# SketchEllipse.deleteMe Method

Parent Object: [SketchEllipse](SketchEllipse.md)  

## Description

Deletes the entity from the sketch.

## Syntax

"sketchEllipse_var" is a variable referencing a [SketchEllipse](SketchEllipse.md) object.

```python
returnValue = sketchEllipse_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true is the delete was successful. |

## Version

Introduced in version August 2014
