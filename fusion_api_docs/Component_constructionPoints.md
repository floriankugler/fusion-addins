# Component.constructionPoints Property

Parent Object: [Component](Component.md)  

## Description

Returns the construction points collection associated with this component. This provides access to the existing construction points and supports the creation of new construction points.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.constructionPoints
```

## Property Value

This is a read only property whose value is a [ConstructionPoints](ConstructionPoints.md).

## Version

Introduced in version August 2014
