# SketchDiameterDimension.parameter Property

Parent Object: [SketchDiameterDimension](SketchDiameterDimension.md)  

## Description

Returns the associated parameter or null if there is no associated parameter.

## Syntax

"sketchDiameterDimension_var" is a variable referencing a SketchDiameterDimension object.  

```python
# Get the value of the property.
propertyValue = sketchDiameterDimension_var.parameter
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version August 2014
