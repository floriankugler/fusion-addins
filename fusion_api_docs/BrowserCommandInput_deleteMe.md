# BrowserCommandInput.deleteMe Method

Parent Object: [BrowserCommandInput](BrowserCommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"browserCommandInput_var" is a variable referencing a [BrowserCommandInput](BrowserCommandInput.md) object.

```python
returnValue = browserCommandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version July 2021
