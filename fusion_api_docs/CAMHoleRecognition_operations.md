# CAMHoleRecognition.operations Property

Parent Object: [CAMHoleRecognition](CAMHoleRecognition.md)  

## Description

Returns the Operations collection that provides access to existing individual operations in this hole recognition.

## Syntax

"cAMHoleRecognition_var" is a variable referencing a CAMHoleRecognition object.  

```python
# Get the value of the property.
propertyValue = cAMHoleRecognition_var.operations
```

## Property Value

This is a read only property whose value is an [Operations](Operations.md).

## Version

Introduced in version April 2023
