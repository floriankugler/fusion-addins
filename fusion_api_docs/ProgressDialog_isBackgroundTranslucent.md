# ProgressDialog.isBackgroundTranslucent Property

Parent Object: [ProgressDialog](ProgressDialog.md)  

## Description

Gets and sets if the dialog background is translucent. This is false by default

## Syntax

"progressDialog_var" is a variable referencing a ProgressDialog object.  

```python
# Get the value of the property.
propertyValue = progressDialog_var.isBackgroundTranslucent

# Set the value of the property.
progressDialog_var.isBackgroundTranslucent = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.
