# CustomGraphicsMesh.isSelectable Property

Parent Object: [CustomGraphicsMesh](CustomGraphicsMesh.md)  

## Description

Gets and sets if the graphics entity is selectable within the graphics window. By default, when a new entity is created it is selectable.

## Syntax

"customGraphicsMesh_var" is a variable referencing a CustomGraphicsMesh object.  

```python
# Get the value of the property.
propertyValue = customGraphicsMesh_var.isSelectable

# Set the value of the property.
customGraphicsMesh_var.isSelectable = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version September 2017
