# MeshBodyDisplayOverrides.isSuppressTriangleEdges Property

Parent Object: [MeshBodyDisplayOverrides](MeshBodyDisplayOverrides.md)  

## Description

Controls whether the edges of the triangles of the mesh body are shown. If set to true, individual triangles will not be visible, edges of face groups (if any) will be shown instead.

## Syntax

"meshBodyDisplayOverrides_var" is a variable referencing a MeshBodyDisplayOverrides object.  

```python
# Get the value of the property.
propertyValue = meshBodyDisplayOverrides_var.isSuppressTriangleEdges

# Set the value of the property.
meshBodyDisplayOverrides_var.isSuppressTriangleEdges = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version May 2025
