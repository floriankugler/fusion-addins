# ConstructionAxisCircularFaceDefinition.parentConstructionAxis Property

## Description

## Syntax

"constructionAxisCircularFaceDefinition_var" is a variable referencing a ConstructionAxisCircularFaceDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionAxisCircularFaceDefinition_var.parentConstructionAxis
```

## Property Value

This is a read only property whose value is a [ConstructionAxis](ConstructionAxis.md).
