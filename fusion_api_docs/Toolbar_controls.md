# Toolbar.controls Property

Parent Object: [Toolbar](Toolbar.md)  

## Description

Gets the controls in this toolbar.

## Syntax

"toolbar_var" is a variable referencing a Toolbar object.  

```python
# Get the value of the property.
propertyValue = toolbar_var.controls
```

## Property Value

This is a read only property whose value is a [ToolbarControls](ToolbarControls.md).
