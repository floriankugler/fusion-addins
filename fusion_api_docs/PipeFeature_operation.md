# PipeFeature.operation Property

Parent Object: [PipeFeature](PipeFeature.md)  

## Description

Gets and sets the type of operation performed by the Pipe.

## Syntax

"pipeFeature_var" is a variable referencing a PipeFeature object.  

```python
# Get the value of the property.
propertyValue = pipeFeature_var.operation

# Set the value of the property.
pipeFeature_var.operation = propertyValue
```

## Property Value

This is a read/write property whose value is a [FeatureOperations](FeatureOperations.md).

## Version

Introduced in version October 2023
