# ArrangePlaneEnvelopeDefinition.isPartialArrangeAllowed Property

Parent Object: [ArrangePlaneEnvelopeDefinition](ArrangePlaneEnvelopeDefinition.md)  

## Description

Gets and sets if a partial arrange is allowed for this envelope.

## Syntax

"arrangePlaneEnvelopeDefinition_var" is a variable referencing an ArrangePlaneEnvelopeDefinition object.  

```python
# Get the value of the property.
propertyValue = arrangePlaneEnvelopeDefinition_var.isPartialArrangeAllowed

# Set the value of the property.
arrangePlaneEnvelopeDefinition_var.isPartialArrangeAllowed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2025
