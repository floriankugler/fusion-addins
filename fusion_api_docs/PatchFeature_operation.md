# PatchFeature.operation Property

Parent Object: [PatchFeature](PatchFeature.md)  

## Description

Gets the type of operation performed by the patch feature.

## Syntax

"patchFeature_var" is a variable referencing a PatchFeature object.  

```python
# Get the value of the property.
propertyValue = patchFeature_var.operation
```

## Property Value

This is a read only property whose value is a [FeatureOperations](FeatureOperations.md).

## Version

Introduced in version July 2016
