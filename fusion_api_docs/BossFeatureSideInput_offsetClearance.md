# BossFeatureSideInput.offsetClearance Property

Parent Object: [BossFeatureSideInput](BossFeatureSideInput.md)  

## Description

Get or set offset clearance as additional small offset from the selected parting plane and position point.

## Syntax

"bossFeatureSideInput_var" is a variable referencing a BossFeatureSideInput object.  

```python
# Get the value of the property.
propertyValue = bossFeatureSideInput_var.offsetClearance

# Set the value of the property.
bossFeatureSideInput_var.offsetClearance = propertyValue
```

## Property Value

This is a read/write property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version October 2022
