# PerpendicularToSurfaceConstraint.attributes Property

Parent Object: [PerpendicularToSurfaceConstraint](PerpendicularToSurfaceConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"perpendicularToSurfaceConstraint_var" is a variable referencing a PerpendicularToSurfaceConstraint object.  

```python
# Get the value of the property.
propertyValue = perpendicularToSurfaceConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version September 2023
