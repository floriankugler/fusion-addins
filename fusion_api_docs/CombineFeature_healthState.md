# CombineFeature.healthState Property

Parent Object: [CombineFeature](CombineFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"combineFeature_var" is a variable referencing a CombineFeature object.  

```python
# Get the value of the property.
propertyValue = combineFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
