# CommandInput.commandInputs Property

Parent Object: [CommandInput](CommandInput.md)  

## Description

Gets the CommandInputs class of the parent, which can be a Command, GroupCommandInput or TabCommandInput.

## Syntax

"commandInput_var" is a variable referencing a CommandInput object.  

```python
# Get the value of the property.
propertyValue = commandInput_var.commandInputs
```

## Property Value

This is a read only property whose value is a [CommandInputs](CommandInputs.md).

## Version

Introduced in version August 2014
