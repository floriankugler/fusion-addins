# StringValueCommandInput.parentCommand Property

Parent Object: [StringValueCommandInput](StringValueCommandInput.md)  

## Description

Gets the parent Command.

## Syntax

"stringValueCommandInput_var" is a variable referencing a StringValueCommandInput object.  

```python
# Get the value of the property.
propertyValue = stringValueCommandInput_var.parentCommand
```

## Property Value

This is a read only property whose value is a [Command](Command.md).

## Version

Introduced in version June 2015
