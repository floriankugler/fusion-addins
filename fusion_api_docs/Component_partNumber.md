# Component.partNumber Property

Parent Object: [Component](Component.md)  

## Description

Gets and sets the part number associated with this component. Setting this to an empty string will reset it to be the same as the component name.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.partNumber

# Set the value of the property.
component_var.partNumber = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version June 2015
