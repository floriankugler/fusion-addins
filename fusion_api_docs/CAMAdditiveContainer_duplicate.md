# CAMAdditiveContainer.duplicate Method

Parent Object: [CAMAdditiveContainer](CAMAdditiveContainer.md)  

## Description

Creates a duplicate of the operation in the tree after the itself.

## Syntax

"cAMAdditiveContainer_var" is a variable referencing a [CAMAdditiveContainer](CAMAdditiveContainer.md) object.

```python
returnValue = cAMAdditiveContainer_var.duplicate()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns if duplicate command was successful. |

## Version

Introduced in version July 2024
