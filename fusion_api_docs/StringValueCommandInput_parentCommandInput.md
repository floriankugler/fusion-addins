# StringValueCommandInput.parentCommandInput Property

Parent Object: [StringValueCommandInput](StringValueCommandInput.md)  

## Description

Gets the parent CommandInput if this commandInput is the child of a TabCommandInput or GroupCommandInput. Returns null if there is no parent.

## Syntax

"stringValueCommandInput_var" is a variable referencing a StringValueCommandInput object.  

```python
# Get the value of the property.
propertyValue = stringValueCommandInput_var.parentCommandInput
```

## Property Value

This is a read only property whose value is a [CommandInput](CommandInput.md).

## Version

Introduced in version July 2015
