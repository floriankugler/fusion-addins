# TableCommandInput.deleteMe Method

Parent Object: [TableCommandInput](TableCommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"tableCommandInput_var" is a variable referencing a [TableCommandInput](TableCommandInput.md) object.

```python
returnValue = tableCommandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2020
