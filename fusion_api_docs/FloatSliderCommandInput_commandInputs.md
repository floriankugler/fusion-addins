# FloatSliderCommandInput.commandInputs Property

Parent Object: [FloatSliderCommandInput](FloatSliderCommandInput.md)  

## Description

Gets the CommandInputs class of the parent, which can be a Command, GroupCommandInput or TabCommandInput.

## Syntax

"floatSliderCommandInput_var" is a variable referencing a FloatSliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = floatSliderCommandInput_var.commandInputs
```

## Property Value

This is a read only property whose value is a [CommandInputs](CommandInputs.md).

## Version

Introduced in version June 2015
