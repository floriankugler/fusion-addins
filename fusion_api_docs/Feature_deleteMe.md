# Feature.deleteMe Method

Parent Object: [Feature](Feature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"feature_var" is a variable referencing a [Feature](Feature.md) object.

```python
returnValue = feature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version August 2014
