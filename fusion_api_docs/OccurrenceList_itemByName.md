# OccurrenceList.itemByName Method

Parent Object: [OccurrenceList](OccurrenceList.md)  

## Description

Returns the specified occurrence using the name of the occurrence.

## Syntax

"occurrenceList_var" is a variable referencing an [OccurrenceList](OccurrenceList.md) object.

```python
returnValue = occurrenceList_var.itemByName(name)
```

## Return Value

| Type | Description |
|----|----|
| [Occurrence](Occurrence.md) | Returns the occurrence or null if an invalid name was specified |

## Parameters

| Name | Type   | Description                           |
|------|--------|---------------------------------------|
| name | string | The name of the occurrence to return. |

## Version

Introduced in version September 2016
