# Workspace.isNative Property

Parent Object: [Workspace](Workspace.md)  

## Description

Gets if this workspace is native to Fusion or was created via the API.

## Syntax

"workspace_var" is a variable referencing a Workspace object.  

```python
# Get the value of the property.
propertyValue = workspace_var.isNative
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
