# FusionDocument.allDocumentReferences Property

Parent Object: [FusionDocument](FusionDocument.md)  

## Description

Returns a collection containing all of the documents referenced directly by this document and those referenced by all sub-assemblies.

## Syntax

"fusionDocument_var" is a variable referencing a FusionDocument object.  

```python
# Get the value of the property.
propertyValue = fusionDocument_var.allDocumentReferences
```

## Property Value

This is a read only property whose value is a [DocumentReferences](DocumentReferences.md).

## Version

Introduced in version June 2017
