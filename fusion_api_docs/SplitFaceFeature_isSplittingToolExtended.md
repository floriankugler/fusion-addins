# SplitFaceFeature.isSplittingToolExtended Property

Parent Object: [SplitFaceFeature](SplitFaceFeature.md)  

## Description

Gets whether or not the setting to automatically extend the splittingTool was set when the feature was created.  

This property is valid only when the splitType property returns surfaceIntersectionSplitType.

## Syntax

"splitFaceFeature_var" is a variable referencing a SplitFaceFeature object.  

```python
# Get the value of the property.
propertyValue = splitFaceFeature_var.isSplittingToolExtended
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version June 2015
