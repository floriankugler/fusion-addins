# CylinderFeatures.count Property

Parent Object: [CylinderFeatures](CylinderFeatures.md)  

## Description

The number of cylinder features in the collection.

## Syntax

"cylinderFeatures_var" is a variable referencing a CylinderFeatures object.  

```python
# Get the value of the property.
propertyValue = cylinderFeatures_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version September 2015
