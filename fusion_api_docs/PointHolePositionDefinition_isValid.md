# PointHolePositionDefinition.isValid Property

Parent Object: [PointHolePositionDefinition](PointHolePositionDefinition.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"pointHolePositionDefinition_var" is a variable referencing a PointHolePositionDefinition object.  

```python
# Get the value of the property.
propertyValue = pointHolePositionDefinition_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
