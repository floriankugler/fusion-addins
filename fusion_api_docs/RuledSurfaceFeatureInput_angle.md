# RuledSurfaceFeatureInput.angle Property

Parent Object: [RuledSurfaceFeatureInput](RuledSurfaceFeatureInput.md)  

## Description

Gets and sets the ValueInput object that defines the Ruled Surface angle. If the input is a real value, the units are radians.

## Syntax

"ruledSurfaceFeatureInput_var" is a variable referencing a RuledSurfaceFeatureInput object.  

```python
# Get the value of the property.
propertyValue = ruledSurfaceFeatureInput_var.angle

# Set the value of the property.
ruledSurfaceFeatureInput_var.angle = propertyValue
```

## Property Value

This is a read/write property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version September 2020
