# RotaryMachineAxisConfiguration.partId Property

Parent Object: [RotaryMachineAxisConfiguration](RotaryMachineAxisConfiguration.md)  

## Description

ID of the part in the KinematicsMachineElement that this axis configuration modifies.

## Syntax

"rotaryMachineAxisConfiguration_var" is a variable referencing a RotaryMachineAxisConfiguration object.  

```python
# Get the value of the property.
propertyValue = rotaryMachineAxisConfiguration_var.partId
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
