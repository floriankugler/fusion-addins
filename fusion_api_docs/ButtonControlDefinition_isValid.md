# ButtonControlDefinition.isValid Property

Parent Object: [ButtonControlDefinition](ButtonControlDefinition.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"buttonControlDefinition_var" is a variable referencing a ButtonControlDefinition object.  

```python
# Get the value of the property.
propertyValue = buttonControlDefinition_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
