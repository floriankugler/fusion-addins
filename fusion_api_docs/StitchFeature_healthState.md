# StitchFeature.healthState Property

Parent Object: [StitchFeature](StitchFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"stitchFeature_var" is a variable referencing a StitchFeature object.  

```python
# Get the value of the property.
propertyValue = stitchFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
