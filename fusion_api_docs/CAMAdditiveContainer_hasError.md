# CAMAdditiveContainer.hasError Property

Parent Object: [CAMAdditiveContainer](CAMAdditiveContainer.md)  

## Description

Gets if errors were encountered when generating the operation.

## Syntax

"cAMAdditiveContainer_var" is a variable referencing a CAMAdditiveContainer object.  

```python
# Get the value of the property.
propertyValue = cAMAdditiveContainer_var.hasError
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2024
