# EllipticalArc2D.asNurbsCurve Property

Parent Object: [EllipticalArc2D](EllipticalArc2D.md)  

## Description

Returns a NURBS curve that is geometrically identical to the elliptical arc.

## Syntax

"ellipticalArc2D_var" is a variable referencing an EllipticalArc2D object.  

```python
# Get the value of the property.
propertyValue = ellipticalArc2D_var.asNurbsCurve
```

## Property Value

This is a read only property whose value is a [NurbsCurve2D](NurbsCurve2D.md).

## Version

Introduced in version August 2014
