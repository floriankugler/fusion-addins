# EllipticalArc2D.majorAxis Property

Parent Object: [EllipticalArc2D](EllipticalArc2D.md)  

## Description

Gets and sets the major axis of the elliptical arc.

## Syntax

"ellipticalArc2D_var" is a variable referencing an EllipticalArc2D object.  

```python
# Get the value of the property.
propertyValue = ellipticalArc2D_var.majorAxis

# Set the value of the property.
ellipticalArc2D_var.majorAxis = propertyValue
```

## Property Value

This is a read/write property whose value is a [Vector2D](Vector2D.md).

## Version

Introduced in version August 2014
