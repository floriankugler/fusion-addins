# CoilFeature.timelineObject Property

Parent Object: [CoilFeature](CoilFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"coilFeature_var" is a variable referencing a CoilFeature object.  

```python
# Get the value of the property.
propertyValue = coilFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version March 2015
