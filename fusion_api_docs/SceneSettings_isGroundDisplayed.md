# SceneSettings.isGroundDisplayed Property

Parent Object: [SceneSettings](SceneSettings.md)  

## Description

Gets and sets if the ground plane is displayed. The plane allows shadows on the ground and reflections if the isGroundReflections property is true.

## Syntax

"sceneSettings_var" is a variable referencing a SceneSettings object.  

```python
# Get the value of the property.
propertyValue = sceneSettings_var.isGroundDisplayed

# Set the value of the property.
sceneSettings_var.isGroundDisplayed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version May 2023
