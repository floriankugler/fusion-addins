# TriangleMeshCalculator.maxSideLength Property

Parent Object: [TriangleMeshCalculator](TriangleMeshCalculator.md)  

## Description

Specifies the maximum side of any triangle in the mesh. A value of 0 (the default) indicates that no maximum length is specified. The value is specified in centimeters.

## Syntax

"triangleMeshCalculator_var" is a variable referencing a TriangleMeshCalculator object.  

```python
# Get the value of the property.
propertyValue = triangleMeshCalculator_var.maxSideLength

# Set the value of the property.
triangleMeshCalculator_var.maxSideLength = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version August 2014
