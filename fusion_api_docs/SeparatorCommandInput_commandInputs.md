# SeparatorCommandInput.commandInputs Property

Parent Object: [SeparatorCommandInput](SeparatorCommandInput.md)  

## Description

Gets the CommandInputs class of the parent, which can be a Command, GroupCommandInput or TabCommandInput.

## Syntax

"separatorCommandInput_var" is a variable referencing a SeparatorCommandInput object.  

```python
# Get the value of the property.
propertyValue = separatorCommandInput_var.commandInputs
```

## Property Value

This is a read only property whose value is a [CommandInputs](CommandInputs.md).

## Version

Introduced in version May 2024
