# CustomGraphicsMesh.coordinates Property

Parent Object: [CustomGraphicsMesh](CustomGraphicsMesh.md)  

## Description

Gets and sets the coordinates associated with this CustomGraphicsMesh.

## Syntax

"customGraphicsMesh_var" is a variable referencing a CustomGraphicsMesh object.  

```python
# Get the value of the property.
propertyValue = customGraphicsMesh_var.coordinates

# Set the value of the property.
customGraphicsMesh_var.coordinates = propertyValue
```

## Property Value

This is a read/write property whose value is a [CustomGraphicsCoordinates](CustomGraphicsCoordinates.md).

## Version

Introduced in version September 2017
