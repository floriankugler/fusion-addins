# LinearMarkingMenu.isValid Property

Parent Object: [LinearMarkingMenu](LinearMarkingMenu.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"linearMarkingMenu_var" is a variable referencing a LinearMarkingMenu object.  

```python
# Get the value of the property.
propertyValue = linearMarkingMenu_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2017
