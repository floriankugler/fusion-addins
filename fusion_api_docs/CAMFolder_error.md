# CAMFolder.error Property

Parent Object: [CAMFolder](CAMFolder.md)  

## Description

Returns a message corresponding to any active error associated with the value of this parameter.

## Syntax

"cAMFolder_var" is a variable referencing a CAMFolder object.  

```python
# Get the value of the property.
propertyValue = cAMFolder_var.error
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
