# UserInterfaceGeneralEventHandler.notify Method

Parent Object: [UserInterfaceGeneralEventHandler](UserInterfaceGeneralEventHandler.md)  

## Description

The function called by Fusion when the associated event is fired.

## Syntax

"userInterfaceGeneralEventHandler_var" is a variable referencing a [UserInterfaceGeneralEventHandler](UserInterfaceGeneralEventHandler.md) object.

```python
returnValue = userInterfaceGeneralEventHandler_var.notify(eventArgs)
```

## Parameters

| Name | Type | Description |
|----|----|----|
| eventArgs | [UserInterfaceGeneralEventArgs](UserInterfaceGeneralEventArgs.md) | Returns an object that provides access to additional information associated with the event. |

## Version

Introduced in version March 2017
