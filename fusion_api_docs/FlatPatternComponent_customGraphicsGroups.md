# FlatPatternComponent.customGraphicsGroups Property

Parent Object: [FlatPatternComponent](FlatPatternComponent.md)  

## Description

Returns the customGraphicsGroups object in this component.

## Syntax

"flatPatternComponent_var" is a variable referencing a FlatPatternComponent object.  

```python
# Get the value of the property.
propertyValue = flatPatternComponent_var.customGraphicsGroups
```

## Property Value

This is a read only property whose value is a [CustomGraphicsGroups](CustomGraphicsGroups.md).

## Version

Introduced in version October 2022
