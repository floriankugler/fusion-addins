# WorkingModel.exportManager Property

Parent Object: [WorkingModel](WorkingModel.md)  

## Description

Returns the ExportManager for this design. You use the ExportManager to export the current design in various formats.

## Syntax

"workingModel_var" is a variable referencing a WorkingModel object.  

```python
# Get the value of the property.
propertyValue = workingModel_var.exportManager
```

## Property Value

This is a read only property whose value is an [ExportManager](ExportManager.md).

## Version

Introduced in version January 2024
