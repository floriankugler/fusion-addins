# SketchPoint.sketchDimensions Property

Parent Object: [SketchPoint](SketchPoint.md)  

## Description

Returns the sketch dimensions that are attached to this curve.

## Syntax

"sketchPoint_var" is a variable referencing a SketchPoint object.  

```python
# Get the value of the property.
propertyValue = sketchPoint_var.sketchDimensions
```

## Property Value

This is a read only property whose value is a [SketchDimensionList](SketchDimensionList.md).

## Version

Introduced in version August 2014
