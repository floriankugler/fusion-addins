# ProfileCurve.sketchEntity Property

Parent Object: [ProfileCurve](ProfileCurve.md)  

## Description

Returns the associated sketch entity that defines this curve.

## Syntax

"profileCurve_var" is a variable referencing a ProfileCurve object.  

```python
# Get the value of the property.
propertyValue = profileCurve_var.sketchEntity
```

## Property Value

This is a read only property whose value is a [SketchEntity](SketchEntity.md).

## Version

Introduced in version August 2014
