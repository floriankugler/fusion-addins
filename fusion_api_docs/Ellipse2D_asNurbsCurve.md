# Ellipse2D.asNurbsCurve Property

Parent Object: [Ellipse2D](Ellipse2D.md)  

## Description

Returns a NURBS curve that is geometrically identical to the ellipse.

## Syntax

"ellipse2D_var" is a variable referencing an Ellipse2D object.  

```python
# Get the value of the property.
propertyValue = ellipse2D_var.asNurbsCurve
```

## Property Value

This is a read only property whose value is a [NurbsCurve2D](NurbsCurve2D.md).

## Version

Introduced in version August 2014
