# ShellFeature.timelineObject Property

Parent Object: [ShellFeature](ShellFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"shellFeature_var" is a variable referencing a ShellFeature object.  

```python
# Get the value of the property.
propertyValue = shellFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version November 2014
