# JointMotion.jointType Property

Parent Object: [JointMotion](JointMotion.md)  

## Description

Returns an enum value indicating the type of joint this joint represents.

## Syntax

"jointMotion_var" is a variable referencing a JointMotion object.  

```python
# Get the value of the property.
propertyValue = jointMotion_var.jointType
```

## Property Value

This is a read only property whose value is a [JointTypes](JointTypes.md).

## Version

Introduced in version July 2015
