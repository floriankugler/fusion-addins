# Occurrence.isReferencedComponent Property

Parent Object: [Occurrence](Occurrence.md)  

## Description

Gets if this occurrence is referencing an external component.

## Syntax

"occurrence_var" is a variable referencing an Occurrence object.  

```python
# Get the value of the property.
propertyValue = occurrence_var.isReferencedComponent
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2015
