# CAMHoleRecognition.isSelected Property

Parent Object: [CAMHoleRecognition](CAMHoleRecognition.md)  

## Description

Gets if this operation is selected in the 'Setups' browser.

## Syntax

"cAMHoleRecognition_var" is a variable referencing a CAMHoleRecognition object.  

```python
# Get the value of the property.
propertyValue = cAMHoleRecognition_var.isSelected
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version April 2023
