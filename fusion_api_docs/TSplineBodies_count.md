# TSplineBodies.count Property

Parent Object: [TSplineBodies](TSplineBodies.md)  

## Description

The number of bodies in the collection.

## Syntax

"tSplineBodies_var" is a variable referencing a TSplineBodies object.  

```python
# Get the value of the property.
propertyValue = tSplineBodies_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version April 2019
