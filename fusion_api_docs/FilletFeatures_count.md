# FilletFeatures.count Property

Parent Object: [FilletFeatures](FilletFeatures.md)  

## Description

The number of fillet features in the collection.

## Syntax

"filletFeatures_var" is a variable referencing a FilletFeatures object.  

```python
# Get the value of the property.
propertyValue = filletFeatures_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version November 2014
