# Features.splitBodyFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the SplitBody features within the component and supports the creation of new SplitBody features

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.splitBodyFeatures
```

## Property Value

This is a read only property whose value is a [SplitBodyFeatures](SplitBodyFeatures.md).
