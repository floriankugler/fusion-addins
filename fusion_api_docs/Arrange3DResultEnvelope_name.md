# Arrange3DResultEnvelope.name Property

Parent Object: [Arrange3DResultEnvelope](Arrange3DResultEnvelope.md)  

## Description

Gets and sets the name of the envelope as seen in the browser.

## Syntax

"arrange3DResultEnvelope_var" is a variable referencing an Arrange3DResultEnvelope object.  

```python
# Get the value of the property.
propertyValue = arrange3DResultEnvelope_var.name

# Set the value of the property.
arrange3DResultEnvelope_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version January 2025
