# SketchLinearDimension.parameter Property

Parent Object: [SketchLinearDimension](SketchLinearDimension.md)  

## Description

Returns the associated parameter or null if there is no associated parameter.

## Syntax

"sketchLinearDimension_var" is a variable referencing a SketchLinearDimension object.  

```python
# Get the value of the property.
propertyValue = sketchLinearDimension_var.parameter
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version August 2014
