# UntrimFeature.parentComponent Property

Parent Object: [UntrimFeature](UntrimFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"untrimFeature_var" is a variable referencing a UntrimFeature object.  

```python
# Get the value of the property.
propertyValue = untrimFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version January 2021
