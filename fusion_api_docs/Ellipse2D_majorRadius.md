# Ellipse2D.majorRadius Property

Parent Object: [Ellipse2D](Ellipse2D.md)  

## Description

Gets and sets the major radius of the ellipse.

## Syntax

"ellipse2D_var" is a variable referencing an Ellipse2D object.  

```python
# Get the value of the property.
propertyValue = ellipse2D_var.majorRadius

# Set the value of the property.
ellipse2D_var.majorRadius = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version August 2014
