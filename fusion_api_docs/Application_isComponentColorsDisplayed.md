# Application.isComponentColorsDisplayed Property

Parent Object: [Application](Application.md)  

## Description

Get and sets if component colors are used when displaying the components within a design. This is the API equivalent of the "Display Component Colors" command.

## Syntax

"application_var" is a variable referencing an Application object.  

```python
# Get the value of the property.
propertyValue = application_var.isComponentColorsDisplayed

# Set the value of the property.
application_var.isComponentColorsDisplayed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version May 2024
