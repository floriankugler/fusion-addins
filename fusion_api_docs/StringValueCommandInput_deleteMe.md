# StringValueCommandInput.deleteMe Method

Parent Object: [StringValueCommandInput](StringValueCommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"stringValueCommandInput_var" is a variable referencing a [StringValueCommandInput](StringValueCommandInput.md) object.

```python
returnValue = stringValueCommandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2020
