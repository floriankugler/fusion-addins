# SketchEntity.deleteMe Method

Parent Object: [SketchEntity](SketchEntity.md)  

## Description

Deletes the entity from the sketch.

## Syntax

"sketchEntity_var" is a variable referencing a [SketchEntity](SketchEntity.md) object.

```python
returnValue = sketchEntity_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true is the delete was successful. |

## Version

Introduced in version August 2014
