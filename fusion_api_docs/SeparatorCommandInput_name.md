# SeparatorCommandInput.name Property

Parent Object: [SeparatorCommandInput](SeparatorCommandInput.md)  

## Description

Gets the user visible name of this input.

## Syntax

"separatorCommandInput_var" is a variable referencing a SeparatorCommandInput object.  

```python
# Get the value of the property.
propertyValue = separatorCommandInput_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version May 2024
