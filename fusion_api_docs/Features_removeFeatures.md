# Features.removeFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the Remove features within the component and supports the creation of new Remove features.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.removeFeatures
```

## Property Value

This is a read only property whose value is a [RemoveFeatures](RemoveFeatures.md).
