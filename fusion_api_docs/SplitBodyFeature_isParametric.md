# SplitBodyFeature.isParametric Property

Parent Object: [SplitBodyFeature](SplitBodyFeature.md)  

## Description

Indicates if this feature is parametric or not.

## Syntax

"splitBodyFeature_var" is a variable referencing a SplitBodyFeature object.  

```python
# Get the value of the property.
propertyValue = splitBodyFeature_var.isParametric
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version June 2015
