# CAM.exportManager Property

Parent Object: [CAM](CAM.md)  

## Description

Returns the Export Manager which provides access to the functionality to export in various formats.

## Syntax

"cAM_var" is a variable referencing a CAM object.  

```python
# Get the value of the property.
propertyValue = cAM_var.exportManager
```

## Property Value

This is a read only property whose value is a [CAMExportManager](CAMExportManager.md).

## Version

Introduced in version November 2021
