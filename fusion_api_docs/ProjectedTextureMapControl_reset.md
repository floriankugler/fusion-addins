# ProjectedTextureMapControl.reset Method

Parent Object: [ProjectedTextureMapControl](ProjectedTextureMapControl.md)  

## Description

Resets the texture map back to its original default settings.

## Syntax

"projectedTextureMapControl_var" is a variable referencing a [ProjectedTextureMapControl](ProjectedTextureMapControl.md) object.

```python
returnValue = projectedTextureMapControl_var.reset()
```

## Return Value

| Type    | Description                               |
|---------|-------------------------------------------|
| boolean | Returns true if the reset was successful. |

## Version

Introduced in version March 2022
