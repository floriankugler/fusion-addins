# BRepLump.vertices Property

Parent Object: [BRepLump](BRepLump.md)  

## Description

Returns the BRepVertices owned by the lump

## Syntax

"bRepLump_var" is a variable referencing a BRepLump object.  

```python
# Get the value of the property.
propertyValue = bRepLump_var.vertices
```

## Property Value

This is a read only property whose value is a [BRepVertices](BRepVertices.md).

## Version

Introduced in version August 2014
