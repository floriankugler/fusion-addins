# SketchFixedSpline.isFullyConstrained Property

Parent Object: [SketchFixedSpline](SketchFixedSpline.md)  

## Description

Indicates if this sketch entity is fully constrained.

## Syntax

"sketchFixedSpline_var" is a variable referencing a SketchFixedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFixedSpline_var.isFullyConstrained
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2016
