# FusionProductPreferences.isAutoHideSketchOnFeatureCreation Property

Parent Object: [FusionProductPreferences](FusionProductPreferences.md)  

## Description

Gets and sets if the sketch should be automatically hidden whenever a feature is created from it.

## Syntax

"fusionProductPreferences_var" is a variable referencing a FusionProductPreferences object.  

```python
# Get the value of the property.
propertyValue = fusionProductPreferences_var.isAutoHideSketchOnFeatureCreation

# Set the value of the property.
fusionProductPreferences_var.isAutoHideSketchOnFeatureCreation = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version May 2022
