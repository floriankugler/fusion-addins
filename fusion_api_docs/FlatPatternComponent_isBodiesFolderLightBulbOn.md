# FlatPatternComponent.isBodiesFolderLightBulbOn Property

Parent Object: [FlatPatternComponent](FlatPatternComponent.md)  

## Description

Gets and sets if the light bulb of the bodies folder as seen in the browser is on or off. This controls the visibility of the solid/surface bodies and the mesh bodies in this component.

## Syntax

"flatPatternComponent_var" is a variable referencing a FlatPatternComponent object.  

```python
# Get the value of the property.
propertyValue = flatPatternComponent_var.isBodiesFolderLightBulbOn

# Set the value of the property.
flatPatternComponent_var.isBodiesFolderLightBulbOn = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version October 2022
