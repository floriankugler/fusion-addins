# FaceContourSelection.warning Property

Parent Object: [FaceContourSelection](FaceContourSelection.md)  

## Description

Gets the last warning string encountered after the selection was applied to a parent.

## Syntax

"faceContourSelection_var" is a variable referencing a FaceContourSelection object.  

```python
# Get the value of the property.
propertyValue = faceContourSelection_var.warning
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version July 2023
