# ImageCommandInput.scaleFactor Property

Parent Object: [ImageCommandInput](ImageCommandInput.md)  

## Description

Gets and sets the scale of the image. This defaults to 1.0, which is full-scale.

## Syntax

"imageCommandInput_var" is a variable referencing an ImageCommandInput object.  

```python
# Get the value of the property.
propertyValue = imageCommandInput_var.scaleFactor

# Set the value of the property.
imageCommandInput_var.scaleFactor = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version March 2024
