# ArrangeFeature.resultEnvelopes Property

Parent Object: [ArrangeFeature](ArrangeFeature.md)  

## Description

Returns the resulting envelopes and their contents.

## Syntax

"arrangeFeature_var" is a variable referencing an ArrangeFeature object.  

```python
# Get the value of the property.
propertyValue = arrangeFeature_var.resultEnvelopes
```

## Property Value

This is a read only property whose value is an [ArrangeResultEnvelopes](ArrangeResultEnvelopes.md).

## Version

Introduced in version January 2025
