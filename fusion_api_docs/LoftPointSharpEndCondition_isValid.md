# LoftPointSharpEndCondition.isValid Property

Parent Object: [LoftPointSharpEndCondition](LoftPointSharpEndCondition.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"loftPointSharpEndCondition_var" is a variable referencing a LoftPointSharpEndCondition object.  

```python
# Get the value of the property.
propertyValue = loftPointSharpEndCondition_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2016
