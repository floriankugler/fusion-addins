# SketchOffsetCurvesDimension.textPosition Property

Parent Object: [SketchOffsetCurvesDimension](SketchOffsetCurvesDimension.md)  

## Description

Gets and sets position of the dimension text.

## Syntax

"sketchOffsetCurvesDimension_var" is a variable referencing a SketchOffsetCurvesDimension object.  

```python
# Get the value of the property.
propertyValue = sketchOffsetCurvesDimension_var.textPosition

# Set the value of the property.
sketchOffsetCurvesDimension_var.textPosition = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version March 2016
