# Operation.hasToolpath Property

Parent Object: [Operation](Operation.md)  

## Description

Gets if a toolpath currently exists (has been generated) for this operation.

## Syntax

"operation_var" is a variable referencing an Operation object.  

```python
# Get the value of the property.
propertyValue = operation_var.hasToolpath
```

## Property Value

This is a read only property whose value is a boolean.
