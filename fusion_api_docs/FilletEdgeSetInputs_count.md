# FilletEdgeSetInputs.count Property

Parent Object: [FilletEdgeSetInputs](FilletEdgeSetInputs.md)  

## Description

The number of fillet edge set input objects in the collection.

## Syntax

"filletEdgeSetInputs_var" is a variable referencing a FilletEdgeSetInputs object.  

```python
# Get the value of the property.
propertyValue = filletEdgeSetInputs_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version November 2022
