# ExtrudeFeature.parentComponent Property

Parent Object: [ExtrudeFeature](ExtrudeFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"extrudeFeature_var" is a variable referencing an ExtrudeFeature object.  

```python
# Get the value of the property.
propertyValue = extrudeFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version August 2014
