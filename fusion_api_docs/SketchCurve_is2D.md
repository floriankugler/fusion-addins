# SketchCurve.is2D Property

Parent Object: [SketchCurve](SketchCurve.md)  

## Description

Indicates if this curve lies entirely on the sketch x-y plane.

## Syntax

"sketchCurve_var" is a variable referencing a SketchCurve object.  

```python
# Get the value of the property.
propertyValue = sketchCurve_var.is2D
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
