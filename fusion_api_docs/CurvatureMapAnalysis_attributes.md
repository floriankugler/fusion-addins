# CurvatureMapAnalysis.attributes Property

Parent Object: [CurvatureMapAnalysis](CurvatureMapAnalysis.md)  

## Description

A property that returns the collection of attributes associated with this Analysis.

## Syntax

"curvatureMapAnalysis_var" is a variable referencing a CurvatureMapAnalysis object.  

```python
# Get the value of the property.
propertyValue = curvatureMapAnalysis_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version January 2023
