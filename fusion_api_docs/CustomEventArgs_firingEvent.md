# CustomEventArgs.firingEvent Property

Parent Object: [CustomEventArgs](CustomEventArgs.md)  

## Description

The event that the firing is in response to.

## Syntax

"customEventArgs_var" is a variable referencing a CustomEventArgs object.  

```python
# Get the value of the property.
propertyValue = customEventArgs_var.firingEvent
```

## Property Value

This is a read only property whose value is an [Event](Event.md).

## Version

Introduced in version January 2017
