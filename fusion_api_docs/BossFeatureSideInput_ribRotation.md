# BossFeatureSideInput.ribRotation Property

Parent Object: [BossFeatureSideInput](BossFeatureSideInput.md)  

## Description

Get or set rotation angle of the first rib from the reference vector. Reference vector is X-axis of the parent sketch from selected sketch point(s).

## Syntax

"bossFeatureSideInput_var" is a variable referencing a BossFeatureSideInput object.  

```python
# Get the value of the property.
propertyValue = bossFeatureSideInput_var.ribRotation

# Set the value of the property.
bossFeatureSideInput_var.ribRotation = propertyValue
```

## Property Value

This is a read/write property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version October 2022
