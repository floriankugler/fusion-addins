# FlatPatternProduct.fusionUnitsManager Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Returns a specialized UnitsManager that can set the default length units and work with parameters.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.fusionUnitsManager
```

## Property Value

This is a read only property whose value is a [FusionUnitsManager](FusionUnitsManager.md).

## Version

Introduced in version October 2022
