# Cone.copy Method

Parent Object: [Cone](Cone.md)  

## Description

Creates and returns an independent copy of this Cone object.

## Syntax

"cone_var" is a variable referencing a [Cone](Cone.md) object.

```python
returnValue = cone_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [Cone](Cone.md) | Returns a new Cone object that is a copy of this Cone object. |

## Version

Introduced in version August 2014
