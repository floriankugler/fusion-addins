# ExtrudeFeature.timelineObject Property

Parent Object: [ExtrudeFeature](ExtrudeFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"extrudeFeature_var" is a variable referencing an ExtrudeFeature object.  

```python
# Get the value of the property.
propertyValue = extrudeFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version August 2014
