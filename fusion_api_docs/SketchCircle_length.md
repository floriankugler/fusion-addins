# SketchCircle.length Property

Parent Object: [SketchCircle](SketchCircle.md)  

## Description

Returns the length of the curve in centimeters.

## Syntax

"sketchCircle_var" is a variable referencing a SketchCircle object.  

```python
# Get the value of the property.
propertyValue = sketchCircle_var.length
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version August 2014
