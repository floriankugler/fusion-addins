# SketchRadialDimension.nativeObject Property

Parent Object: [SketchRadialDimension](SketchRadialDimension.md)  

## Description

The NativeObject is the object outside the context of an assembly and in the context of it's parent component. Returns null in the case where this object is not in the context of an assembly but is already the native object.

## Syntax

"sketchRadialDimension_var" is a variable referencing a SketchRadialDimension object.  

```python
# Get the value of the property.
propertyValue = sketchRadialDimension_var.nativeObject
```

## Property Value

This is a read only property whose value is a [SketchRadialDimension](SketchRadialDimension.md).

## Version

Introduced in version August 2014
