# Arc3D.transformBy Method

Parent Object: [Arc3D](Arc3D.md)  

## Description

Transforms this curve in 3D space.

## Syntax

"arc3D_var" is a variable referencing an [Arc3D](Arc3D.md) object.

```python
returnValue = arc3D_var.transformBy(matrix)
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Return true if the transform was successful. |

## Parameters

| Name | Type | Description |
|----|----|----|
| matrix | [Matrix3D](Matrix3D.md) | A 3D matrix that defines the transform to apply to the curve. |

## Version

Introduced in version August 2014
