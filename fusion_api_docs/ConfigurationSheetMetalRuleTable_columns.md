# ConfigurationSheetMetalRuleTable.columns Property

Parent Object: [ConfigurationSheetMetalRuleTable](ConfigurationSheetMetalRuleTable.md)  

## Description

Returns the collection that provides access to the columns in this table.

## Syntax

"configurationSheetMetalRuleTable_var" is a variable referencing a ConfigurationSheetMetalRuleTable object.  

```python
# Get the value of the property.
propertyValue = configurationSheetMetalRuleTable_var.columns
```

## Property Value

This is a read only property whose value is a [ConfigurationSheetMetalRuleColumns](ConfigurationSheetMetalRuleColumns.md).

## Version

Introduced in version January 2024
