# HorizontalConstraint.isDeletable Property

Parent Object: [HorizontalConstraint](HorizontalConstraint.md)  

## Description

Indicates if this constraint is deletable.

## Syntax

"horizontalConstraint_var" is a variable referencing a HorizontalConstraint object.  

```python
# Get the value of the property.
propertyValue = horizontalConstraint_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
