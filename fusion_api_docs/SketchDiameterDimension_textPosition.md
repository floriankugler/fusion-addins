# SketchDiameterDimension.textPosition Property

Parent Object: [SketchDiameterDimension](SketchDiameterDimension.md)  

## Description

Gets and sets position of the dimension text.

## Syntax

"sketchDiameterDimension_var" is a variable referencing a SketchDiameterDimension object.  

```python
# Get the value of the property.
propertyValue = sketchDiameterDimension_var.textPosition

# Set the value of the property.
sketchDiameterDimension_var.textPosition = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
