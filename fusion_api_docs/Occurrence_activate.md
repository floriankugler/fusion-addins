# Occurrence.activate Method

Parent Object: [Occurrence](Occurrence.md)  

## Description

Makes the occurrence the active edit target in the user interface. This is the same as enabling the radio button next to the occurrence in the browser.

## Syntax

"occurrence_var" is a variable referencing an [Occurrence](Occurrence.md) object.

```python
returnValue = occurrence_var.activate()
```

## Return Value

| Type    | Description                                    |
|---------|------------------------------------------------|
| boolean | Returns true if the activation was successful. |

## Version

Introduced in version June 2015
