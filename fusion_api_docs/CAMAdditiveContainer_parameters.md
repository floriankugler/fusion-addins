# CAMAdditiveContainer.parameters Property

Parent Object: [CAMAdditiveContainer](CAMAdditiveContainer.md)  

## Description

Gets the CAMParameters collection for this operation.

## Syntax

"cAMAdditiveContainer_var" is a variable referencing a CAMAdditiveContainer object.  

```python
# Get the value of the property.
propertyValue = cAMAdditiveContainer_var.parameters
```

## Property Value

This is a read only property whose value is a [CAMParameters](CAMParameters.md).

## Version

Introduced in version July 2024
