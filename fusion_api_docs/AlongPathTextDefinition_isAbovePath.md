# AlongPathTextDefinition.isAbovePath Property

Parent Object: [AlongPathTextDefinition](AlongPathTextDefinition.md)  

## Description

Gets and sets if the text should be positioned above or below the path entity.

## Syntax

"alongPathTextDefinition_var" is a variable referencing an AlongPathTextDefinition object.  

```python
# Get the value of the property.
propertyValue = alongPathTextDefinition_var.isAbovePath

# Set the value of the property.
alongPathTextDefinition_var.isAbovePath = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version December 2020
