# MFGDMDataEvent.remove Method

Parent Object: [MFGDMDataEvent](MFGDMDataEvent.md)  

## Description

Removes a handler from the event.

## Syntax

"mFGDMDataEvent_var" is a variable referencing a [MFGDMDataEvent](MFGDMDataEvent.md) object.

```python
returnValue = mFGDMDataEvent_var.remove(handler)
```

## Return Value

| Type    | Description                                            |
|---------|--------------------------------------------------------|
| boolean | Returns true if removal of the handler was successful. |

## Parameters

| Name | Type | Description |
|----|----|----|
| handler | [MFGDMDataEventHandler](MFGDMDataEventHandler.md) | The handler object to be removed from the event. |

## Version

Introduced in version July 2025
