# SetupEvent.sender Property

Parent Object: [SetupEvent](SetupEvent.md)  

## Description

The object that is firing the event. For example, in the case of a command input event this will return the command.

## Syntax

"setupEvent_var" is a variable referencing a SetupEvent object.  

```python
# Get the value of the property.
propertyValue = setupEvent_var.sender
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version April 2023
