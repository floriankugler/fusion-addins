# ObjectVisibility.isJointOriginAxesVisible Property

Parent Object: [ObjectVisibility](ObjectVisibility.md)  

## Description

Gets and sets if the axes lines shown in the glyphs for joint origins in all components, local or referenced by this design, are visible in the graphics window.

## Syntax

"objectVisibility_var" is a variable referencing an ObjectVisibility object.  

```python
# Get the value of the property.
propertyValue = objectVisibility_var.isJointOriginAxesVisible

# Set the value of the property.
objectVisibility_var.isJointOriginAxesVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version November 2025
