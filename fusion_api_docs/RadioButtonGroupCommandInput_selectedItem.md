# RadioButtonGroupCommandInput.selectedItem Property

Parent Object: [RadioButtonGroupCommandInput](RadioButtonGroupCommandInput.md)  

## Description

Gets and sets the item in the radio button list that is currently selected.

## Syntax

"radioButtonGroupCommandInput_var" is a variable referencing a RadioButtonGroupCommandInput object.  

```python
# Get the value of the property.
propertyValue = radioButtonGroupCommandInput_var.selectedItem
```

## Property Value

This is a read only property whose value is a [ListItem](ListItem.md).

## Version

Introduced in version July 2015
