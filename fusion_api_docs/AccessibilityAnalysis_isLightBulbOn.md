# AccessibilityAnalysis.isLightBulbOn Property

Parent Object: [AccessibilityAnalysis](AccessibilityAnalysis.md)  

## Description

A property that gets and sets if the display is enabled for this Analysis object. If false, this analysis will be hidden. If true and the IsLightBulbOn property of the Analyses object is True the Analysis will be visible.

## Syntax

"accessibilityAnalysis_var" is a variable referencing an AccessibilityAnalysis object.  

```python
# Get the value of the property.
propertyValue = accessibilityAnalysis_var.isLightBulbOn

# Set the value of the property.
accessibilityAnalysis_var.isLightBulbOn = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2023
