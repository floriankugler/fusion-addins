# ObjectCollection.clear Method

Parent Object: [ObjectCollection](ObjectCollection.md)  

## Description

Clears the entire contents of the collection.

## Syntax

"objectCollection_var" is a variable referencing an [ObjectCollection](ObjectCollection.md) object.

```python
returnValue = objectCollection_var.clear()
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Version

Introduced in version August 2014
