# CurveSelections.createNewPocketSelection Method

Parent Object: [CurveSelections](CurveSelections.md)  

## Description

Creates a new pocket selection and adds it to the end of the collection.

## Syntax

"curveSelections_var" is a variable referencing a [CurveSelections](CurveSelections.md) object.

```python
returnValue = curveSelections_var.createNewPocketSelection()
```

## Return Value

| Type | Description |
|----|----|
| [PocketSelection](PocketSelection.md) | Returns newly created PocketSelection. |

## Version

Introduced in version April 2023
