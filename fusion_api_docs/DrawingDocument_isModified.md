# DrawingDocument.isModified Property

Parent Object: [DrawingDocument](DrawingDocument.md)  

## Description

Property that indicates if the document has been modified since it was last saved.

## Syntax

"drawingDocument_var" is a variable referencing a DrawingDocument object.  

```python
# Get the value of the property.
propertyValue = drawingDocument_var.isModified
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version December 2020
