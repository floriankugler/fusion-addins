# SketchEllipseMinorRadiusDimension.parentSketch Property

Parent Object: [SketchEllipseMinorRadiusDimension](SketchEllipseMinorRadiusDimension.md)  

## Description

Returns the parent sketch object.

## Syntax

"sketchEllipseMinorRadiusDimension_var" is a variable referencing a SketchEllipseMinorRadiusDimension object.  

```python
# Get the value of the property.
propertyValue = sketchEllipseMinorRadiusDimension_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).
