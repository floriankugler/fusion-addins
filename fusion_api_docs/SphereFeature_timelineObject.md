# SphereFeature.timelineObject Property

Parent Object: [SphereFeature](SphereFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"sphereFeature_var" is a variable referencing a SphereFeature object.  

```python
# Get the value of the property.
propertyValue = sphereFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version September 2015
