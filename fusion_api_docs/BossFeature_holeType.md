# BossFeature.holeType Property

Parent Object: [BossFeature](BossFeature.md)  

## Description

Returns the current type of hole this feature represents.

## Syntax

"bossFeature_var" is a variable referencing a BossFeature object.  

```python
# Get the value of the property.
propertyValue = bossFeature_var.holeType
```

## Property Value

This is a read only property whose value is a [HoleTypes](HoleTypes.md).

## Version

Introduced in version October 2022
