# VerticalPointsConstraint.attributes Property

Parent Object: [VerticalPointsConstraint](VerticalPointsConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"verticalPointsConstraint_var" is a variable referencing a VerticalPointsConstraint object.  

```python
# Get the value of the property.
propertyValue = verticalPointsConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
