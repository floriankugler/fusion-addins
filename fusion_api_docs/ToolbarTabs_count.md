# ToolbarTabs.count Property

Parent Object: [ToolbarTabs](ToolbarTabs.md)  

## Description

Gets the number of ToolbarTabs.

## Syntax

"toolbarTabs_var" is a variable referencing a ToolbarTabs object.  

```python
# Get the value of the property.
propertyValue = toolbarTabs_var.count
```

## Property Value

This is a read only property whose value is a uinteger.
