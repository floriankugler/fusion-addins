# CAMAdditiveContainer.name Property

Parent Object: [CAMAdditiveContainer](CAMAdditiveContainer.md)  

## Description

Gets and sets the name of the operation as seen in the browser.Â This name is unique as compared to the names of all other operations in the document.

## Syntax

"cAMAdditiveContainer_var" is a variable referencing a CAMAdditiveContainer object.  

```python
# Get the value of the property.
propertyValue = cAMAdditiveContainer_var.name

# Set the value of the property.
cAMAdditiveContainer_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version July 2024
