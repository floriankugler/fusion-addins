# SketchLine.isFixed Property

Parent Object: [SketchLine](SketchLine.md)  

## Description

Indicates if this geometry is "fixed".

## Syntax

"sketchLine_var" is a variable referencing a SketchLine object.  

```python
# Get the value of the property.
propertyValue = sketchLine_var.isFixed

# Set the value of the property.
sketchLine_var.isFixed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
