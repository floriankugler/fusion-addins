# MarkingMenuEventArgs.firingEvent Property

Parent Object: [MarkingMenuEventArgs](MarkingMenuEventArgs.md)  

## Description

The event that the firing is in response to.

## Syntax

"markingMenuEventArgs_var" is a variable referencing a MarkingMenuEventArgs object.  

```python
# Get the value of the property.
propertyValue = markingMenuEventArgs_var.firingEvent
```

## Property Value

This is a read only property whose value is an [Event](Event.md).

## Version

Introduced in version January 2017
