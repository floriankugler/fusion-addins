# MotionLink.timelineObject Property

Parent Object: [MotionLink](MotionLink.md)  

## Description

Returns the timeline object associated with this MotionLink.

## Syntax

"motionLink_var" is a variable referencing a MotionLink object.  

```python
# Get the value of the property.
propertyValue = motionLink_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version November 2025
