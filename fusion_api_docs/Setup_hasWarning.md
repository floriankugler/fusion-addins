# Setup.hasWarning Property

Parent Object: [Setup](Setup.md)  

## Description

Gets if problems were encountered when generating the operation.

## Syntax

"setup_var" is a variable referencing a Setup object.  

```python
# Get the value of the property.
propertyValue = setup_var.hasWarning
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version April 2023
