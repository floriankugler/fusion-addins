# SketchCircle.isFixed Property

Parent Object: [SketchCircle](SketchCircle.md)  

## Description

Indicates if this geometry is "fixed".

## Syntax

"sketchCircle_var" is a variable referencing a SketchCircle object.  

```python
# Get the value of the property.
propertyValue = sketchCircle_var.isFixed

# Set the value of the property.
sketchCircle_var.isFixed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
