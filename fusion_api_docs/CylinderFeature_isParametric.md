# CylinderFeature.isParametric Property

Parent Object: [CylinderFeature](CylinderFeature.md)  

## Description

Indicates if this feature is parametric or not.

## Syntax

"cylinderFeature_var" is a variable referencing a CylinderFeature object.  

```python
# Get the value of the property.
propertyValue = cylinderFeature_var.isParametric
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2015
