# FlatPatternProduct.flatPattern Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Gets the flat pattern associated with this FlatPatternProduct.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.flatPattern
```

## Property Value

This is a read only property whose value is a [FlatPattern](FlatPattern.md).

## Version

Introduced in version October 2022
