# CombineFeature.isNewComponent Property

Parent Object: [CombineFeature](CombineFeature.md)  

## Description

Gets a boolean value for whether or not a new component was created when this feature was created.

## Syntax

"combineFeature_var" is a variable referencing a CombineFeature object.  

```python
# Get the value of the property.
propertyValue = combineFeature_var.isNewComponent
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version November 2014
