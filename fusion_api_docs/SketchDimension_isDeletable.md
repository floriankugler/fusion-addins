# SketchDimension.isDeletable Property

Parent Object: [SketchDimension](SketchDimension.md)  

## Description

Indicates if this dimension is deletable.

## Syntax

"sketchDimension_var" is a variable referencing a SketchDimension object.  

```python
# Get the value of the property.
propertyValue = sketchDimension_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
