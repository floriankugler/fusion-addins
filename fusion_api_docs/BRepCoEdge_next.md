# BRepCoEdge.next Property

Parent Object: [BRepCoEdge](BRepCoEdge.md)  

## Description

Returns the next co-edge in the loop.

## Syntax

"bRepCoEdge_var" is a variable referencing a BRepCoEdge object.  

```python
# Get the value of the property.
propertyValue = bRepCoEdge_var.next
```

## Property Value

This is a read only property whose value is a [BRepCoEdge](BRepCoEdge.md).

## Version

Introduced in version August 2014
