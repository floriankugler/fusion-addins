# CommandControl.id Property

Parent Object: [CommandControl](CommandControl.md)  

## Description

Gets the unique ID of this control. The ID is unique with respect to the other controls within the same panel, toolbar, or drop-down control.

## Syntax

"commandControl_var" is a variable referencing a CommandControl object.  

```python
# Get the value of the property.
propertyValue = commandControl_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2014
