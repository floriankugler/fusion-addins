# KinematicsMachineElement.parts Property

Parent Object: [KinematicsMachineElement](KinematicsMachineElement.md)  

## Description

Get the root parts collection.

## Syntax

"kinematicsMachineElement_var" is a variable referencing a KinematicsMachineElement object.  

```python
# Get the value of the property.
propertyValue = kinematicsMachineElement_var.parts
```

## Property Value

This is a read only property whose value is a [MachineParts](MachineParts.md).

## Version

Introduced in version April 2023
