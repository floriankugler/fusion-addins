# SeparatorCommandInput.parentCommand Property

Parent Object: [SeparatorCommandInput](SeparatorCommandInput.md)  

## Description

Gets the parent Command.

## Syntax

"separatorCommandInput_var" is a variable referencing a SeparatorCommandInput object.  

```python
# Get the value of the property.
propertyValue = separatorCommandInput_var.parentCommand
```

## Property Value

This is a read only property whose value is a [Command](Command.md).

## Version

Introduced in version May 2024
