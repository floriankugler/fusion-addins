# HTMLEvent.sender Property

Parent Object: [HTMLEvent](HTMLEvent.md)  

## Description

The object that is firing the event. For example, in the case of a command input event this will return the command.

## Syntax

"hTMLEvent_var" is a variable referencing a HTMLEvent object.  

```python
# Get the value of the property.
propertyValue = hTMLEvent_var.sender
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version March 2017
