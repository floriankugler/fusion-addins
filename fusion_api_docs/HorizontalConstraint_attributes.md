# HorizontalConstraint.attributes Property

Parent Object: [HorizontalConstraint](HorizontalConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"horizontalConstraint_var" is a variable referencing a HorizontalConstraint object.  

```python
# Get the value of the property.
propertyValue = horizontalConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
