# GeometrySelection.error Property

Parent Object: [GeometrySelection](GeometrySelection.md)  

## Description

Gets the last warning string encountered after the selection was applied to a parent.

## Syntax

"geometrySelection_var" is a variable referencing a GeometrySelection object.  

```python
# Get the value of the property.
propertyValue = geometrySelection_var.error
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version July 2023
