# ShellFeature.healthState Property

Parent Object: [ShellFeature](ShellFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"shellFeature_var" is a variable referencing a ShellFeature object.  

```python
# Get the value of the property.
propertyValue = shellFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
