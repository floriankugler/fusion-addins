# BRepBody.vertices Property

Parent Object: [BRepBody](BRepBody.md)  

## Description

Returns a collection of all of the vertices in the body.

## Syntax

"bRepBody_var" is a variable referencing a BRepBody object.  

```python
# Get the value of the property.
propertyValue = bRepBody_var.vertices
```

## Property Value

This is a read only property whose value is a [BRepVertices](BRepVertices.md).

## Version

Introduced in version August 2014
