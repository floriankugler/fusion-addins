# EllipticalArc2D.evaluator Property

Parent Object: [EllipticalArc2D](EllipticalArc2D.md)  

## Description

Returns an evaluator object that lets you perform additional evaluations on the curve.

## Syntax

"ellipticalArc2D_var" is a variable referencing an EllipticalArc2D object.  

```python
# Get the value of the property.
propertyValue = ellipticalArc2D_var.evaluator
```

## Property Value

This is a read only property whose value is a [CurveEvaluator2D](CurveEvaluator2D.md).

## Version

Introduced in version August 2014
