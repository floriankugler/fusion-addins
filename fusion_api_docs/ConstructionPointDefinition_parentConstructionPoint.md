# ConstructionPointDefinition.parentConstructionPoint Property

Parent Object: [ConstructionPointDefinition](ConstructionPointDefinition.md)  

## Description

Returns the ConstructionPoint object

## Syntax

"constructionPointDefinition_var" is a variable referencing a ConstructionPointDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionPointDefinition_var.parentConstructionPoint
```

## Property Value

This is a read only property whose value is a [ConstructionPoint](ConstructionPoint.md).

## Version

Introduced in version August 2014
