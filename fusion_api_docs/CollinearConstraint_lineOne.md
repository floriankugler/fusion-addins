# CollinearConstraint.lineOne Property

Parent Object: [CollinearConstraint](CollinearConstraint.md)  

## Description

Returns the first line.

## Syntax

"collinearConstraint_var" is a variable referencing a CollinearConstraint object.  

```python
# Get the value of the property.
propertyValue = collinearConstraint_var.lineOne
```

## Property Value

This is a read only property whose value is a [SketchLine](SketchLine.md).

## Version

Introduced in version August 2014
