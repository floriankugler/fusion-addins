# SketchEllipseMajorRadiusDimension.attributes Property

Parent Object: [SketchEllipseMajorRadiusDimension](SketchEllipseMajorRadiusDimension.md)  

## Description

Returns the collection of attributes associated with this sketch dimension.

## Syntax

"sketchEllipseMajorRadiusDimension_var" is a variable referencing a SketchEllipseMajorRadiusDimension object.  

```python
# Get the value of the property.
propertyValue = sketchEllipseMajorRadiusDimension_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
