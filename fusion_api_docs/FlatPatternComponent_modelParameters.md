# FlatPatternComponent.modelParameters Property

Parent Object: [FlatPatternComponent](FlatPatternComponent.md)  

## Description

Returns the collection of model parameters in the Component.

## Syntax

"flatPatternComponent_var" is a variable referencing a FlatPatternComponent object.  

```python
# Get the value of the property.
propertyValue = flatPatternComponent_var.modelParameters
```

## Property Value

This is a read only property whose value is a [ModelParameters](ModelParameters.md).

## Version

Introduced in version October 2022
