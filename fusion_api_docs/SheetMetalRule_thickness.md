# SheetMetalRule.thickness Property

Parent Object: [SheetMetalRule](SheetMetalRule.md)  

## Description

The thickness of the part. Use the returned SheetMetalRuleValue object to get and set the current value of the thickness.

## Syntax

"sheetMetalRule_var" is a variable referencing a SheetMetalRule object.  

```python
# Get the value of the property.
propertyValue = sheetMetalRule_var.thickness
```

## Property Value

This is a read only property whose value is a [SheetMetalRuleValue](SheetMetalRuleValue.md).

## Version

Introduced in version November 2022
