# StitchFeature.attributes Property

Parent Object: [StitchFeature](StitchFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"stitchFeature_var" is a variable referencing a StitchFeature object.  

```python
# Get the value of the property.
propertyValue = stitchFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
