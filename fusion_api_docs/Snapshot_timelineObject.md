# Snapshot.timelineObject Property

Parent Object: [Snapshot](Snapshot.md)  

## Description

Returns the timeline object associated with this snapshot.

## Syntax

"snapshot_var" is a variable referencing a Snapshot object.  

```python
# Get the value of the property.
propertyValue = snapshot_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version August 2014
