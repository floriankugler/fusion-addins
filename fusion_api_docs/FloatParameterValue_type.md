# FloatParameterValue.type Property

Parent Object: [FloatParameterValue](FloatParameterValue.md)  

## Description

Get the type of the float parameter.

## Syntax

"floatParameterValue_var" is a variable referencing a FloatParameterValue object.  

```python
# Get the value of the property.
propertyValue = floatParameterValue_var.type
```

## Property Value

This is a read only property whose value is a [FloatParameterValueTypes](FloatParameterValueTypes.md).

## Version

Introduced in version May 2023
