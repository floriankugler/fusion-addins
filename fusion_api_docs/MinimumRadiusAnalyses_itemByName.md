# MinimumRadiusAnalyses.itemByName Method

Parent Object: [MinimumRadiusAnalyses](MinimumRadiusAnalyses.md)  

## Description

A method that returns the specified MinimumRadiusAnalysis object using the name of the analysis as displayed in the browser.

## Syntax

"minimumRadiusAnalyses_var" is a variable referencing a [MinimumRadiusAnalyses](MinimumRadiusAnalyses.md) object.

```python
returnValue = minimumRadiusAnalyses_var.itemByName(name)
```

## Return Value

| Type | Description |
|----|----|
| [MinimumRadiusAnalysis](MinimumRadiusAnalysis.md) | Returns the specified item or null if an invalid name was specified. |

## Parameters

| Name | Type | Description |
|----|----|----|
| name | string | The name of the MinimumRadiusAnalysis object as displayed in the browser. |

## Version

Introduced in version January 2023
