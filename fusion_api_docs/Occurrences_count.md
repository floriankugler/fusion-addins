# Occurrences.count Property

Parent Object: [Occurrences](Occurrences.md)  

## Description

Returns the number of occurrences in the collection.

## Syntax

"occurrences_var" is a variable referencing an Occurrences object.  

```python
# Get the value of the property.
propertyValue = occurrences_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version August 2014
