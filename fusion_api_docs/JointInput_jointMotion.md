# JointInput.jointMotion Property

Parent Object: [JointInput](JointInput.md)  

## Description

Returns an object derived from JointMotion that defines how the motion between the two joint geometries is defined.

## Syntax

"jointInput_var" is a variable referencing a JointInput object.  

```python
# Get the value of the property.
propertyValue = jointInput_var.jointMotion
```

## Property Value

This is a read only property whose value is a [JointMotion](JointMotion.md).

## Version

Introduced in version January 2016
