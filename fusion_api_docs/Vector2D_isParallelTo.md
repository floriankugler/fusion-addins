# Vector2D.isParallelTo Method

Parent Object: [Vector2D](Vector2D.md)  

## Description

Compare this vector with another to check for parallelism.

## Syntax

"vector2D_var" is a variable referencing a [Vector2D](Vector2D.md) object.

```python
returnValue = vector2D_var.isParallelTo(vector)
```

## Return Value

| Type    | Description                               |
|---------|-------------------------------------------|
| boolean | Returns true if the vectors are parallel. |

## Parameters

| Name | Type | Description |
|----|----|----|
| vector | [Vector2D](Vector2D.md) | The vector to compare with for parallelism. |

## Version

Introduced in version August 2014
