# Point2D.copy Method

Parent Object: [Point2D](Point2D.md)  

## Description

Creates and returns a copy of this point object.

## Syntax

"point2D_var" is a variable referencing a [Point2D](Point2D.md) object.

```python
returnValue = point2D_var.copy()
```

## Return Value

| Type                   | Description                                       |
|------------------------|---------------------------------------------------|
| [Point2D](Point2D.md) | Returns an independent copy of this point object. |

## Version

Introduced in version August 2014
