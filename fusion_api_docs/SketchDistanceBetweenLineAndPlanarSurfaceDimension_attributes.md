# SketchDistanceBetweenLineAndPlanarSurfaceDimension.attributes Property

Parent Object: [SketchDistanceBetweenLineAndPlanarSurfaceDimension](SketchDistanceBetweenLineAndPlanarSurfaceDimension.md)  

## Description

Returns the collection of attributes associated with this sketch dimension.

## Syntax

"sketchDistanceBetweenLineAndPlanarSurfaceDimension_var" is a variable referencing a SketchDistanceBetweenLineAndPlanarSurfaceDimension object.  

```python
# Get the value of the property.
propertyValue = sketchDistanceBetweenLineAndPlanarSurfaceDimension_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).
