# CAMHoleRecognition.strategy Property

Parent Object: [CAMHoleRecognition](CAMHoleRecognition.md)  

## Description

Gets the name of the strategy associated with this operation.

## Syntax

"cAMHoleRecognition_var" is a variable referencing a CAMHoleRecognition object.  

```python
# Get the value of the property.
propertyValue = cAMHoleRecognition_var.strategy
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
