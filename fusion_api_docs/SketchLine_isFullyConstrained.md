# SketchLine.isFullyConstrained Property

Parent Object: [SketchLine](SketchLine.md)  

## Description

Indicates if this sketch entity is fully constrained.

## Syntax

"sketchLine_var" is a variable referencing a SketchLine object.  

```python
# Get the value of the property.
propertyValue = sketchLine_var.isFullyConstrained
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2016
