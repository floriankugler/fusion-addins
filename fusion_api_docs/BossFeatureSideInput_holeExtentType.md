# BossFeatureSideInput.holeExtentType Property

Parent Object: [BossFeatureSideInput](BossFeatureSideInput.md)  

## Description

Get or set hole extent this feature represents. For top side only through hole extent is accepted.

## Syntax

"bossFeatureSideInput_var" is a variable referencing a BossFeatureSideInput object.  

```python
# Get the value of the property.
propertyValue = bossFeatureSideInput_var.holeExtentType

# Set the value of the property.
bossFeatureSideInput_var.holeExtentType = propertyValue
```

## Property Value

This is a read/write property whose value is a [BossHoleExtentTypes](BossHoleExtentTypes.md).
