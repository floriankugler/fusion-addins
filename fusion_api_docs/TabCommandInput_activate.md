# TabCommandInput.activate Method

Parent Object: [TabCommandInput](TabCommandInput.md)  

## Description

Sets this to be the currently activated (selected) tab.

## Syntax

"tabCommandInput_var" is a variable referencing a [TabCommandInput](TabCommandInput.md) object.

```python
returnValue = tabCommandInput_var.activate()
```

## Return Value

| Type    | Description                                                     |
|---------|-----------------------------------------------------------------|
| boolean | Returns true if the TabCommandInput was successfully activated. |

## Version

Introduced in version July 2015
