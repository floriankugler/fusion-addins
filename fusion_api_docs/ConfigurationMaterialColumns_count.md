# ConfigurationMaterialColumns.count Property

Parent Object: [ConfigurationMaterialColumns](ConfigurationMaterialColumns.md)  

## Description

Returns the number of columns in the table where the name column is not included.

## Syntax

"configurationMaterialColumns_var" is a variable referencing a ConfigurationMaterialColumns object.  

```python
# Get the value of the property.
propertyValue = configurationMaterialColumns_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version January 2024
