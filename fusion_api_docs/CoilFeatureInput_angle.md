# CoilFeatureInput.angle Property

Parent Object: [CoilFeatureInput](CoilFeatureInput.md)  

## Description

Gets the angle. Returns null in the case where the coilType property returns SpiralCoilType.

## Syntax

"coilFeatureInput_var" is a variable referencing a CoilFeatureInput object.  

```python
# Get the value of the property.
propertyValue = coilFeatureInput_var.angle
```

## Property Value

This is a read only property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version March 2016
