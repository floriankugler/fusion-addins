# NurbsCurve2D.controlPoints Property

Parent Object: [NurbsCurve2D](NurbsCurve2D.md)  

## Description

Returns an array of Point2D objects that define the control points of the curve.

## Syntax

"nurbsCurve2D_var" is a variable referencing a NurbsCurve2D object.  

```python
# Get the value of the property.
propertyValue = nurbsCurve2D_var.controlPoints
```

## Property Value

This is a read only property whose value is an array of type [Point2D](Point2D.md).

## Version

Introduced in version August 2014
