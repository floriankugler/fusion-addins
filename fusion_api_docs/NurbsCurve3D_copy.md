# NurbsCurve3D.copy Method

Parent Object: [NurbsCurve3D](NurbsCurve3D.md)  

## Description

Creates and returns an independent copy of this NurbsCurve3D object.

## Syntax

"nurbsCurve3D_var" is a variable referencing a [NurbsCurve3D](NurbsCurve3D.md) object.

```python
returnValue = nurbsCurve3D_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [NurbsCurve3D](NurbsCurve3D.md) | Returns an independent copy of this NurbsCurve3D. |

## Version

Introduced in version August 2014
