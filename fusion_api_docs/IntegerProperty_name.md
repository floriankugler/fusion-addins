# IntegerProperty.name Property

Parent Object: [IntegerProperty](IntegerProperty.md)  

## Description

Returns the name of this property as seen in the user interface. This name is localized and can change based on the current language

## Syntax

"integerProperty_var" is a variable referencing an IntegerProperty object.  

```python
# Get the value of the property.
propertyValue = integerProperty_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2014
