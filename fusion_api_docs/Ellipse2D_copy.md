# Ellipse2D.copy Method

Parent Object: [Ellipse2D](Ellipse2D.md)  

## Description

Creates and returns a copy of this Ellipse2D object.

## Syntax

"ellipse2D_var" is a variable referencing an [Ellipse2D](Ellipse2D.md) object.

```python
returnValue = ellipse2D_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [Ellipse2D](Ellipse2D.md) | Returns a new Ellipse2D object that is a copy of this Ellipse2D object. |

## Version

Introduced in version August 2014
