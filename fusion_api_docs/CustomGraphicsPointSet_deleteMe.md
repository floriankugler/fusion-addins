# CustomGraphicsPointSet.deleteMe Method

Parent Object: [CustomGraphicsPointSet](CustomGraphicsPointSet.md)  

## Description

Deletes the entity from the custom graphics group.

## Syntax

"customGraphicsPointSet_var" is a variable referencing a [CustomGraphicsPointSet](CustomGraphicsPointSet.md) object.

```python
returnValue = customGraphicsPointSet_var.deleteMe()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns true if the deletion was successful. |

## Version

Introduced in version September 2017
