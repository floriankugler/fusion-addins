# SweepFeature.sideFaces Property

Parent Object: [SweepFeature](SweepFeature.md)  

## Description

Property that returns an object that provides access to all of the faces created around the perimeter of the feature.

## Syntax

"sweepFeature_var" is a variable referencing a SweepFeature object.  

```python
# Get the value of the property.
propertyValue = sweepFeature_var.sideFaces
```

## Property Value

This is a read only property whose value is a [BRepFaces](BRepFaces.md).

## Version

Introduced in version November 2014
