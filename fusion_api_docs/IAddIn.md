# IAddIn Object

## Description

The interface implemented by an add-in.

## Methods

| Name | Description |
|----|----|
| [onActivate](IAddIn_onActivate.md) | Lets the application do any initialization when it is safe to do so. |
| [onDeactivate](IAddIn_onDeactivate.md) | Lets the application do any termination when it is safe to do so. In general, if the add-in is closing, termination should be minimized. |

## Version

Introduced in version March 2022
