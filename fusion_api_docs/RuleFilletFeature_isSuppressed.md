# RuleFilletFeature.isSuppressed Property

Parent Object: [RuleFilletFeature](RuleFilletFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"ruleFilletFeature_var" is a variable referencing a RuleFilletFeature object.  

```python
# Get the value of the property.
propertyValue = ruleFilletFeature_var.isSuppressed

# Set the value of the property.
ruleFilletFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version September 2015
