# Plane.surfaceType Property

Parent Object: [Plane](Plane.md)  

## Description

Returns the surface type.

## Syntax

"plane_var" is a variable referencing a Plane object.  

```python
# Get the value of the property.
propertyValue = plane_var.surfaceType
```

## Property Value

This is a read only property whose value is a [SurfaceTypes](SurfaceTypes.md).

## Version

Introduced in version August 2014
