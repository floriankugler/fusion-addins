# Milestones.itemByName Method

Parent Object: [Milestones](Milestones.md)  

## Description

Returns the milestone specified using its name..

## Syntax

"milestones_var" is a variable referencing a [Milestones](Milestones.md) object.

```python
returnValue = milestones_var.itemByName(name)
```

## Return Value

| Type | Description |
|----|----|
| [Milestone](Milestone.md) | Returns the Milestone object or null if a milestone with the specified name is not found. |

## Parameters

| Name | Type   | Description                          |
|------|--------|--------------------------------------|
| name | string | The name of the milestone to return. |

## Version

Introduced in version March 2024
