# ParallelConstraint.parentSketch Property

Parent Object: [ParallelConstraint](ParallelConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"parallelConstraint_var" is a variable referencing a ParallelConstraint object.  

```python
# Get the value of the property.
propertyValue = parallelConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
