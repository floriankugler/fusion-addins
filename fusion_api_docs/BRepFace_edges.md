# BRepFace.edges Property

Parent Object: [BRepFace](BRepFace.md)  

## Description

Returns the BRepEdges used by this face

## Syntax

"bRepFace_var" is a variable referencing a BRepFace object.  

```python
# Get the value of the property.
propertyValue = bRepFace_var.edges
```

## Property Value

This is a read only property whose value is a [BRepEdges](BRepEdges.md).

## Version

Introduced in version August 2014
