# ShellFeatures.count Property

Parent Object: [ShellFeatures](ShellFeatures.md)  

## Description

The number of shell features in the collection.

## Syntax

"shellFeatures_var" is a variable referencing a ShellFeatures object.  

```python
# Get the value of the property.
propertyValue = shellFeatures_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version November 2014
