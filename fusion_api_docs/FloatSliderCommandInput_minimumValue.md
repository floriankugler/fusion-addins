# FloatSliderCommandInput.minimumValue Property

Parent Object: [FloatSliderCommandInput](FloatSliderCommandInput.md)  

## Description

Gets and sets minimum value of the slider in database units. Gets a failure when set if the value of this command input was added by value list.

## Syntax

"floatSliderCommandInput_var" is a variable referencing a FloatSliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = floatSliderCommandInput_var.minimumValue

# Set the value of the property.
floatSliderCommandInput_var.minimumValue = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version June 2015
