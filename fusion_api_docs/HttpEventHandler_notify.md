# HttpEventHandler.notify Method

Parent Object: [HttpEventHandler](HttpEventHandler.md)  

## Description

The function called by Fusion when the associated event is fired.

## Syntax

"httpEventHandler_var" is a variable referencing a [HttpEventHandler](HttpEventHandler.md) object.

```python
returnValue = httpEventHandler_var.notify(eventArgs)
```

## Parameters

| Name | Type | Description |
|----|----|----|
| eventArgs | [HttpEventArgs](HttpEventArgs.md) | Returns an object that provides access to additional information associated with the event. |

## Version

Introduced in version January 2024
