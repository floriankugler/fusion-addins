# ThreadFeature.parentComponent Property

Parent Object: [ThreadFeature](ThreadFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"threadFeature_var" is a variable referencing a ThreadFeature object.  

```python
# Get the value of the property.
propertyValue = threadFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version January 2015
