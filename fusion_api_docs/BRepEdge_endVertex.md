# BRepEdge.endVertex Property

Parent Object: [BRepEdge](BRepEdge.md)  

## Description

Returns the BRepVertex that bounds its high parameter end.

## Syntax

"bRepEdge_var" is a variable referencing a BRepEdge object.  

```python
# Get the value of the property.
propertyValue = bRepEdge_var.endVertex
```

## Property Value

This is a read only property whose value is a [BRepVertex](BRepVertex.md).

## Version

Introduced in version August 2014
