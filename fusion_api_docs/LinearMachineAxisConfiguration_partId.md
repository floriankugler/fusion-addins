# LinearMachineAxisConfiguration.partId Property

Parent Object: [LinearMachineAxisConfiguration](LinearMachineAxisConfiguration.md)  

## Description

ID of the part in the KinematicsMachineElement that this axis configuration modifies.

## Syntax

"linearMachineAxisConfiguration_var" is a variable referencing a LinearMachineAxisConfiguration object.  

```python
# Get the value of the property.
propertyValue = linearMachineAxisConfiguration_var.partId
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version April 2023
