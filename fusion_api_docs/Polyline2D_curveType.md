# Polyline2D.curveType Property

Parent Object: [Polyline2D](Polyline2D.md)  

## Description

Returns the type of geometry this curve represents.

## Syntax

"polyline2D_var" is a variable referencing a Polyline2D object.  

```python
# Get the value of the property.
propertyValue = polyline2D_var.curveType
```

## Property Value

This is a read only property whose value is a [Curve2DTypes](Curve2DTypes.md).

## Version

Introduced in version September 2024
