# LoftFeature.endLoftEdgeAlignment Property

Parent Object: [LoftFeature](LoftFeature.md)  

## Description

Specifies the end edge alignment option for the loft feature. The default is Free Edges.

## Syntax

"loftFeature_var" is a variable referencing a LoftFeature object.  

```python
# Get the value of the property.
propertyValue = loftFeature_var.endLoftEdgeAlignment

# Set the value of the property.
loftFeature_var.endLoftEdgeAlignment = propertyValue
```

## Property Value

This is a read/write property whose value is a [LoftEdgeAlignments](LoftEdgeAlignments.md).

## Version

Introduced in version September 2024
