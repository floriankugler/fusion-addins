# CAMFolder.parameters Property

Parent Object: [CAMFolder](CAMFolder.md)  

## Description

Gets the CAMParameters collection for this operation.

## Syntax

"cAMFolder_var" is a variable referencing a CAMFolder object.  

```python
# Get the value of the property.
propertyValue = cAMFolder_var.parameters
```

## Property Value

This is a read only property whose value is a [CAMParameters](CAMParameters.md).

## Version

Introduced in version September 2020
