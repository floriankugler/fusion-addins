# DataFiles.asArray Method

Parent Object: [DataFiles](DataFiles.md)  

## Description

Get the current list of all files.

## Syntax

"dataFiles_var" is a variable referencing a [DataFiles](DataFiles.md) object.

```python
returnValue = dataFiles_var.asArray()
```

## Return Value

| Type                         | Description                            |
|------------------------------|----------------------------------------|
| [DataFile](DataFile.md)\[\] | Returns the current list of all files. |

## Version

Introduced in version April 2021
