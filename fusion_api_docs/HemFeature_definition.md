# HemFeature.definition Property

Parent Object: [HemFeature](HemFeature.md)  

## Description

Returns the HemFeatureDefinition object which provides access to the information defining this HemFeature and the ability to edit it.

## Syntax

"hemFeature_var" is a variable referencing a HemFeature object.  

```python
# Get the value of the property.
propertyValue = hemFeature_var.definition
```

## Property Value

This is a read only property whose value is a [HemFeatureDefinition](HemFeatureDefinition.md).

## Version

Introduced in version September 2025
