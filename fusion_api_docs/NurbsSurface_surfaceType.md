# NurbsSurface.surfaceType Property

Parent Object: [NurbsSurface](NurbsSurface.md)  

## Description

Returns the surface type.

## Syntax

"nurbsSurface_var" is a variable referencing a NurbsSurface object.  

```python
# Get the value of the property.
propertyValue = nurbsSurface_var.surfaceType
```

## Property Value

This is a read only property whose value is a [SurfaceTypes](SurfaceTypes.md).

## Version

Introduced in version August 2014
