# HemFeature.timelineObject Property

Parent Object: [HemFeature](HemFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"hemFeature_var" is a variable referencing a HemFeature object.  

```python
# Get the value of the property.
propertyValue = hemFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version September 2025
