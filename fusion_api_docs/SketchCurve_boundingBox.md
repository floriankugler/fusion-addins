# SketchCurve.boundingBox Property

Parent Object: [SketchCurve](SketchCurve.md)  

## Description

Returns the bounding box of the entity in sketch space.

## Syntax

"sketchCurve_var" is a variable referencing a SketchCurve object.  

```python
# Get the value of the property.
propertyValue = sketchCurve_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version August 2014
