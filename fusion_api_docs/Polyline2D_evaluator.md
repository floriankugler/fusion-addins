# Polyline2D.evaluator Property

Parent Object: [Polyline2D](Polyline2D.md)  

## Description

Returns an evaluator object that lets you perform additional evaluations on the curve.

## Syntax

"polyline2D_var" is a variable referencing a Polyline2D object.  

```python
# Get the value of the property.
propertyValue = polyline2D_var.evaluator
```

## Property Value

This is a read only property whose value is a [CurveEvaluator2D](CurveEvaluator2D.md).

## Version

Introduced in version September 2024
