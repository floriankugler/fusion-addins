# ConfigurationFeatureAspectBooleanCell.parentColumn Property

Parent Object: [ConfigurationFeatureAspectBooleanCell](ConfigurationFeatureAspectBooleanCell.md)  

## Description

Returns the column this cell is in.

## Syntax

"configurationFeatureAspectBooleanCell_var" is a variable referencing a ConfigurationFeatureAspectBooleanCell object.  

```python
# Get the value of the property.
propertyValue = configurationFeatureAspectBooleanCell_var.parentColumn
```

## Property Value

This is a read only property whose value is a [ConfigurationFeatureAspectColumn](ConfigurationFeatureAspectColumn.md).

## Version

Introduced in version September 2024
