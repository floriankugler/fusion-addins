# DeleteFaceFeature.parentComponent Property

Parent Object: [DeleteFaceFeature](DeleteFaceFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"deleteFaceFeature_var" is a variable referencing a DeleteFaceFeature object.  

```python
# Get the value of the property.
propertyValue = deleteFaceFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version August 2016
