# FusionDefaultUnitsPreferences.name Property

Parent Object: [FusionDefaultUnitsPreferences](FusionDefaultUnitsPreferences.md)  

## Description

Returns the name of this DefaultUnitPreferences object.

## Syntax

"fusionDefaultUnitsPreferences_var" is a variable referencing a FusionDefaultUnitsPreferences object.  

```python
# Get the value of the property.
propertyValue = fusionDefaultUnitsPreferences_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2014
