# BaseComponent.constructionPlanes Property

Parent Object: [BaseComponent](BaseComponent.md)  

## Description

Returns the construction planes collection associated with this component. This provides access to the existing construction planes and supports the creation of new construction planes.

## Syntax

"baseComponent_var" is a variable referencing a BaseComponent object.  

```python
# Get the value of the property.
propertyValue = baseComponent_var.constructionPlanes
```

## Property Value

This is a read only property whose value is a [ConstructionPlanes](ConstructionPlanes.md).

## Version

Introduced in version August 2014
