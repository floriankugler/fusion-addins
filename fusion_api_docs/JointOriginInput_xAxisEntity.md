# JointOriginInput.xAxisEntity Property

Parent Object: [JointOriginInput](JointOriginInput.md)  

## Description

Gets and sets the entity that defines the X axis direction. This defaults to null meaning the X axis is inferred from the input geometry.

## Syntax

"jointOriginInput_var" is a variable referencing a JointOriginInput object.  

```python
# Get the value of the property.
propertyValue = jointOriginInput_var.xAxisEntity

# Set the value of the property.
jointOriginInput_var.xAxisEntity = propertyValue
```

## Property Value

This is a read/write property whose value is a [Base](Base.md).

## Version

Introduced in version September 2015
