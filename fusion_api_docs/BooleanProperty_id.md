# BooleanProperty.id Property

Parent Object: [BooleanProperty](BooleanProperty.md)  

## Description

Returns the unique ID of this property.

## Syntax

"booleanProperty_var" is a variable referencing a BooleanProperty object.  

```python
# Get the value of the property.
propertyValue = booleanProperty_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2014
