# MirrorFeature.patternElements Property

Parent Object: [MirrorFeature](MirrorFeature.md)  

## Description

Gets the PatternElements collection that contains the elements created by this pattern.

## Syntax

"mirrorFeature_var" is a variable referencing a MirrorFeature object.  

```python
# Get the value of the property.
propertyValue = mirrorFeature_var.patternElements
```

## Property Value

This is a read only property whose value is a [PatternElements](PatternElements.md).

## Version

Introduced in version November 2014
