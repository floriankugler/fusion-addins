# Polyline3D.curveType Property

Parent Object: [Polyline3D](Polyline3D.md)  

## Description

Returns the type of geometry this curve represents.

## Syntax

"polyline3D_var" is a variable referencing a Polyline3D object.  

```python
# Get the value of the property.
propertyValue = polyline3D_var.curveType
```

## Property Value

This is a read only property whose value is a [Curve3DTypes](Curve3DTypes.md).

## Version

Introduced in version September 2024
