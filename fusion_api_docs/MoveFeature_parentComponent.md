# MoveFeature.parentComponent Property

Parent Object: [MoveFeature](MoveFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"moveFeature_var" is a variable referencing a MoveFeature object.  

```python
# Get the value of the property.
propertyValue = moveFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version March 2015
