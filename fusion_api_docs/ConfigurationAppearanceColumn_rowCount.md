# ConfigurationAppearanceColumn.rowCount Property

Parent Object: [ConfigurationAppearanceColumn](ConfigurationAppearanceColumn.md)  

## Description

Returns the number of rows in this column.

## Syntax

"configurationAppearanceColumn_var" is a variable referencing a ConfigurationAppearanceColumn object.  

```python
# Get the value of the property.
propertyValue = configurationAppearanceColumn_var.rowCount
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version March 2024
