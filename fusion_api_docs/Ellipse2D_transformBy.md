# Ellipse2D.transformBy Method

Parent Object: [Ellipse2D](Ellipse2D.md)  

## Description

Transforms this curve in 2D space.

## Syntax

"ellipse2D_var" is a variable referencing an [Ellipse2D](Ellipse2D.md) object.

```python
returnValue = ellipse2D_var.transformBy(matrix)
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Return true if the transform was successful. |

## Parameters

| Name | Type | Description |
|----|----|----|
| matrix | [Matrix2D](Matrix2D.md) | A 2D matrix that defines the transform to apply to the curve. |

## Version

Introduced in version August 2014
