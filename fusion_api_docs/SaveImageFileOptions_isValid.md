# SaveImageFileOptions.isValid Property

Parent Object: [SaveImageFileOptions](SaveImageFileOptions.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"saveImageFileOptions_var" is a variable referencing a SaveImageFileOptions object.  

```python
# Get the value of the property.
propertyValue = saveImageFileOptions_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version May 2022
