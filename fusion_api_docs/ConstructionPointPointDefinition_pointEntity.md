# ConstructionPointPointDefinition.pointEntity Property

Parent Object: [ConstructionPointPointDefinition](ConstructionPointPointDefinition.md)  

## Description

Gets and sets the position of the point using a construction point, sketch point or vertex. Non-parametric points will always return a Point3D object

## Syntax

"constructionPointPointDefinition_var" is a variable referencing a ConstructionPointPointDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionPointPointDefinition_var.pointEntity

# Set the value of the property.
constructionPointPointDefinition_var.pointEntity = propertyValue
```

## Property Value

This is a read/write property whose value is a [Base](Base.md).

## Version

Introduced in version August 2014
