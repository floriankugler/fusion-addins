# Features.loftFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the existing loft features and supports the creation of new loft features.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.loftFeatures
```

## Property Value

This is a read only property whose value is a [LoftFeatures](LoftFeatures.md).
