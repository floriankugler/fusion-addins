# LoftFeatureInput.operation Property

Parent Object: [LoftFeatureInput](LoftFeatureInput.md)  

## Description

Gets and sets the type of operation performed by the loft.

## Syntax

"loftFeatureInput_var" is a variable referencing a LoftFeatureInput object.  

```python
# Get the value of the property.
propertyValue = loftFeatureInput_var.operation

# Set the value of the property.
loftFeatureInput_var.operation = propertyValue
```

## Property Value

This is a read/write property whose value is a [FeatureOperations](FeatureOperations.md).

## Version

Introduced in version August 2016
