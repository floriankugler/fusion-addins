# ExtendFeature.timelineObject Property

Parent Object: [ExtendFeature](ExtendFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"extendFeature_var" is a variable referencing an ExtendFeature object.  

```python
# Get the value of the property.
propertyValue = extendFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version June 2015
