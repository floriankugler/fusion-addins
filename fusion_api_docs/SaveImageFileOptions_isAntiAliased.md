# SaveImageFileOptions.isAntiAliased Property

Parent Object: [SaveImageFileOptions](SaveImageFileOptions.md)  

## Description

Gets and sets if the rendered image should be anti-aliased or not. If false, there is no anti-aliasing.  

When the SaveImageFileOptions object is initially created, this is initialized to true.

## Syntax

"saveImageFileOptions_var" is a variable referencing a SaveImageFileOptions object.  

```python
# Get the value of the property.
propertyValue = saveImageFileOptions_var.isAntiAliased

# Set the value of the property.
saveImageFileOptions_var.isAntiAliased = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version May 2022
