# PathPatternFeatureInput.isOrientationAlongPath Property

Parent Object: [PathPatternFeatureInput](PathPatternFeatureInput.md)  

## Description

Gets and sets if the orientation is along path. If false, the orientation is identical.

## Syntax

"pathPatternFeatureInput_var" is a variable referencing a PathPatternFeatureInput object.  

```python
# Get the value of the property.
propertyValue = pathPatternFeatureInput_var.isOrientationAlongPath

# Set the value of the property.
pathPatternFeatureInput_var.isOrientationAlongPath = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version November 2014
