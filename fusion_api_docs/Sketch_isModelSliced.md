# Sketch.isModelSliced Property

Parent Object: [Sketch](Sketch.md)  

## Description

Gets and sets whether the model is sliced along the sketch plane when this sketch is active. This provides access to the "Slice" setting in the "SKETCH PALETTE".

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.isModelSliced

# Set the value of the property.
sketch_var.isModelSliced = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version March 2024
