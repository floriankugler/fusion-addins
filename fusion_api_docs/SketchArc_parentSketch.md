# SketchArc.parentSketch Property

Parent Object: [SketchArc](SketchArc.md)  

## Description

Returns the parent sketch.

## Syntax

"sketchArc_var" is a variable referencing a SketchArc object.  

```python
# Get the value of the property.
propertyValue = sketchArc_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
