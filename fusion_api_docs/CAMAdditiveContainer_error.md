# CAMAdditiveContainer.error Property

Parent Object: [CAMAdditiveContainer](CAMAdditiveContainer.md)  

## Description

Returns a message corresponding to any active error associated with the value of this parameter.

## Syntax

"cAMAdditiveContainer_var" is a variable referencing a CAMAdditiveContainer object.  

```python
# Get the value of the property.
propertyValue = cAMAdditiveContainer_var.error
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version July 2024
