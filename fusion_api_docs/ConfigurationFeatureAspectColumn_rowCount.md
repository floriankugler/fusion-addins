# ConfigurationFeatureAspectColumn.rowCount Property

Parent Object: [ConfigurationFeatureAspectColumn](ConfigurationFeatureAspectColumn.md)  

## Description

Returns the number of rows in this column.

## Syntax

"configurationFeatureAspectColumn_var" is a variable referencing a ConfigurationFeatureAspectColumn object.  

```python
# Get the value of the property.
propertyValue = configurationFeatureAspectColumn_var.rowCount
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version September 2024
