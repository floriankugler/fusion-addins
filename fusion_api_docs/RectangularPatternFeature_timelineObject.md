# RectangularPatternFeature.timelineObject Property

Parent Object: [RectangularPatternFeature](RectangularPatternFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"rectangularPatternFeature_var" is a variable referencing a RectangularPatternFeature object.  

```python
# Get the value of the property.
propertyValue = rectangularPatternFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version November 2014
