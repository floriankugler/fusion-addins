# ConfigurationInsertCell.parentRow Property

Parent Object: [ConfigurationInsertCell](ConfigurationInsertCell.md)  

## Description

Returns the row this cell is in.

## Syntax

"configurationInsertCell_var" is a variable referencing a ConfigurationInsertCell object.  

```python
# Get the value of the property.
propertyValue = configurationInsertCell_var.parentRow
```

## Property Value

This is a read only property whose value is a [ConfigurationRow](ConfigurationRow.md).

## Version

Introduced in version January 2024
