# RevolveFeature.attributes Property

Parent Object: [RevolveFeature](RevolveFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"revolveFeature_var" is a variable referencing a RevolveFeature object.  

```python
# Get the value of the property.
propertyValue = revolveFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
