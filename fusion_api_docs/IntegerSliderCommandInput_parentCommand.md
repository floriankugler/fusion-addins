# IntegerSliderCommandInput.parentCommand Property

Parent Object: [IntegerSliderCommandInput](IntegerSliderCommandInput.md)  

## Description

Gets the parent Command.

## Syntax

"integerSliderCommandInput_var" is a variable referencing an IntegerSliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = integerSliderCommandInput_var.parentCommand
```

## Property Value

This is a read only property whose value is a [Command](Command.md).

## Version

Introduced in version June 2015
