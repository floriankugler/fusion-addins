# NamedView.apply Method

Parent Object: [NamedView](NamedView.md)  

## Description

This updates the active viewport to use the camera associated with this named view.

## Syntax

"namedView_var" is a variable referencing a [NamedView](NamedView.md) object.

```python
returnValue = namedView_var.apply()
```

## Return Value

| Type    | Description                                   |
|---------|-----------------------------------------------|
| boolean | Returns true if the operation was successful. |

## Version

Introduced in version September 2023
