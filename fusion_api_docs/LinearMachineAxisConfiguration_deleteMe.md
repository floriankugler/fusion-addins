# LinearMachineAxisConfiguration.deleteMe Method

Parent Object: [LinearMachineAxisConfiguration](LinearMachineAxisConfiguration.md)  

## Description

Delete this axis configuration from the controller configuration.

## Syntax

"linearMachineAxisConfiguration_var" is a variable referencing a [LinearMachineAxisConfiguration](LinearMachineAxisConfiguration.md) object.

```python
returnValue = linearMachineAxisConfiguration_var.deleteMe()
```

## Version

Introduced in version April 2023
