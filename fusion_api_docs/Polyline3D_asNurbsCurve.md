# Polyline3D.asNurbsCurve Property

Parent Object: [Polyline3D](Polyline3D.md)  

## Description

Returns a NURBS curve that is geometrically identical to the polyline.

## Syntax

"polyline3D_var" is a variable referencing a Polyline3D object.  

```python
# Get the value of the property.
propertyValue = polyline3D_var.asNurbsCurve
```

## Property Value

This is a read only property whose value is a [NurbsCurve3D](NurbsCurve3D.md).

## Version

Introduced in version September 2024
