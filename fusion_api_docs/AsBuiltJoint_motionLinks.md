# AsBuiltJoint.motionLinks Property

Parent Object: [AsBuiltJoint](AsBuiltJoint.md)  

## Description

Returns the MotionLink objects that this joint is involved in.

## Syntax

"asBuiltJoint_var" is a variable referencing an AsBuiltJoint object.  

```python
# Get the value of the property.
propertyValue = asBuiltJoint_var.motionLinks
```

## Property Value

This is a read only property whose value is an array of type [MotionLink](MotionLink.md).

## Version

Introduced in version November 2025
