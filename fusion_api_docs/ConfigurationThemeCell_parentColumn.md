# ConfigurationThemeCell.parentColumn Property

Parent Object: [ConfigurationThemeCell](ConfigurationThemeCell.md)  

## Description

Returns the column this cell is in.

## Syntax

"configurationThemeCell_var" is a variable referencing a ConfigurationThemeCell object.  

```python
# Get the value of the property.
propertyValue = configurationThemeCell_var.parentColumn
```

## Property Value

This is a read only property whose value is a [ConfigurationThemeColumn](ConfigurationThemeColumn.md).

## Version

Introduced in version January 2024
