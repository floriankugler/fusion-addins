# GeometricConstraint.isDeletable Property

Parent Object: [GeometricConstraint](GeometricConstraint.md)  

## Description

Indicates if this constraint is deletable.

## Syntax

"geometricConstraint_var" is a variable referencing a GeometricConstraint object.  

```python
# Get the value of the property.
propertyValue = geometricConstraint_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
