# Operation.isSelected Property

Parent Object: [Operation](Operation.md)  

## Description

Gets if this operation is selected in the 'Setups' browser.

## Syntax

"operation_var" is a variable referencing an Operation object.  

```python
# Get the value of the property.
propertyValue = operation_var.isSelected
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version May 2020
