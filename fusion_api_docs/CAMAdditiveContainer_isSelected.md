# CAMAdditiveContainer.isSelected Property

Parent Object: [CAMAdditiveContainer](CAMAdditiveContainer.md)  

## Description

Gets if this operation is selected in the 'Setups' browser.

## Syntax

"cAMAdditiveContainer_var" is a variable referencing a CAMAdditiveContainer object.  

```python
# Get the value of the property.
propertyValue = cAMAdditiveContainer_var.isSelected
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2024
