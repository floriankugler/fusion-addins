# SketchAngularDimension.deleteMe Method

Parent Object: [SketchAngularDimension](SketchAngularDimension.md)  

## Description

Deletes this dimension. The IsDeletable property indicates if this dimension can be deleted.

## Syntax

"sketchAngularDimension_var" is a variable referencing a [SketchAngularDimension](SketchAngularDimension.md) object.

```python
returnValue = sketchAngularDimension_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
