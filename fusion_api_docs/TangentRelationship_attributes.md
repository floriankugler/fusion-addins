# TangentRelationship.attributes Property

Parent Object: [TangentRelationship](TangentRelationship.md)  

## Description

Returns the collection of attributes associated with this tangent relationship.

## Syntax

"tangentRelationship_var" is a variable referencing a TangentRelationship object.  

```python
# Get the value of the property.
propertyValue = tangentRelationship_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2022
