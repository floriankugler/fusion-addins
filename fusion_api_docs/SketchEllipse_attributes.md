# SketchEllipse.attributes Property

Parent Object: [SketchEllipse](SketchEllipse.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"sketchEllipse_var" is a variable referencing a SketchEllipse object.  

```python
# Get the value of the property.
propertyValue = sketchEllipse_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
