# IntegerProperty.id Property

Parent Object: [IntegerProperty](IntegerProperty.md)  

## Description

Returns the unique ID of this property.

## Syntax

"integerProperty_var" is a variable referencing an IntegerProperty object.  

```python
# Get the value of the property.
propertyValue = integerProperty_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version August 2014
