# SketchFixedSpline.boundingBox Property

Parent Object: [SketchFixedSpline](SketchFixedSpline.md)  

## Description

Returns the bounding box of the entity in sketch space.

## Syntax

"sketchFixedSpline_var" is a variable referencing a SketchFixedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFixedSpline_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version August 2014
