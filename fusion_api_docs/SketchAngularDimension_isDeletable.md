# SketchAngularDimension.isDeletable Property

Parent Object: [SketchAngularDimension](SketchAngularDimension.md)  

## Description

Indicates if this dimension is deletable.

## Syntax

"sketchAngularDimension_var" is a variable referencing a SketchAngularDimension object.  

```python
# Get the value of the property.
propertyValue = sketchAngularDimension_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
