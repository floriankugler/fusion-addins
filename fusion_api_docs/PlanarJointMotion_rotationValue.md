# PlanarJointMotion.rotationValue Property

Parent Object: [PlanarJointMotion](PlanarJointMotion.md)  

## Description

Gets and sets the rotation value. This is in radians. Setting this value is the equivalent of using the Drive Joints command.

## Syntax

"planarJointMotion_var" is a variable referencing a PlanarJointMotion object.  

```python
# Get the value of the property.
propertyValue = planarJointMotion_var.rotationValue

# Set the value of the property.
planarJointMotion_var.rotationValue = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version July 2015
