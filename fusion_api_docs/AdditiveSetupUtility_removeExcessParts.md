# AdditiveSetupUtility.removeExcessParts Method

Parent Object: [AdditiveSetupUtility](AdditiveSetupUtility.md)  

## Description

Remove components from setups, that are outside of the buildroom.

## Syntax

"additiveSetupUtility_var" is a variable referencing an [AdditiveSetupUtility](AdditiveSetupUtility.md) object.

```python
returnValue = additiveSetupUtility_var.removeExcessParts()
```

## Return Value

| Type    | Description                                                  |
|---------|--------------------------------------------------------------|
| integer | Returns count of removed parts on success and -1 on failure. |

## Version

Introduced in version January 2024
