# ScaleFeature.timelineObject Property

Parent Object: [ScaleFeature](ScaleFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"scaleFeature_var" is a variable referencing a ScaleFeature object.  

```python
# Get the value of the property.
propertyValue = scaleFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version January 2015
