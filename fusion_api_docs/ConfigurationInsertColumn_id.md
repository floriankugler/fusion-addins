# ConfigurationInsertColumn.id Property

Parent Object: [ConfigurationInsertColumn](ConfigurationInsertColumn.md)  

## Description

The id of the column. This is constant and cannot be changed by the user.

## Syntax

"configurationInsertColumn_var" is a variable referencing a ConfigurationInsertColumn object.  

```python
# Get the value of the property.
propertyValue = configurationInsertColumn_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version January 2024
