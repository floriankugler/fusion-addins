# ConfigurationColumn.isValid Property

Parent Object: [ConfigurationColumn](ConfigurationColumn.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"configurationColumn_var" is a variable referencing a ConfigurationColumn object.  

```python
# Get the value of the property.
propertyValue = configurationColumn_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2024
