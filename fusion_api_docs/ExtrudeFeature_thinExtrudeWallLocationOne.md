# ExtrudeFeature.thinExtrudeWallLocationOne Property

Parent Object: [ExtrudeFeature](ExtrudeFeature.md)  

## Description

Gets and sets the wall location for a one sided thin extrude or side one of a two sided thin extrude

## Syntax

"extrudeFeature_var" is a variable referencing an ExtrudeFeature object.  

```python
# Get the value of the property.
propertyValue = extrudeFeature_var.thinExtrudeWallLocationOne

# Set the value of the property.
extrudeFeature_var.thinExtrudeWallLocationOne = propertyValue
```

## Property Value

This is a read/write property whose value is a [ThinExtrudeWallLocation](ThinExtrudeWallLocation.md).

## Version

Introduced in version April 2021
