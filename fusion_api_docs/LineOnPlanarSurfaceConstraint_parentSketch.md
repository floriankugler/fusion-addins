# LineOnPlanarSurfaceConstraint.parentSketch Property

Parent Object: [LineOnPlanarSurfaceConstraint](LineOnPlanarSurfaceConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"lineOnPlanarSurfaceConstraint_var" is a variable referencing a LineOnPlanarSurfaceConstraint object.  

```python
# Get the value of the property.
propertyValue = lineOnPlanarSurfaceConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version September 2023
