# CustomGraphicsBRepBody.deleteMe Method

Parent Object: [CustomGraphicsBRepBody](CustomGraphicsBRepBody.md)  

## Description

Deletes the entity from the custom graphics group.

## Syntax

"customGraphicsBRepBody_var" is a variable referencing a [CustomGraphicsBRepBody](CustomGraphicsBRepBody.md) object.

```python
returnValue = customGraphicsBRepBody_var.deleteMe()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns true if the deletion was successful. |

## Version

Introduced in version September 2017
