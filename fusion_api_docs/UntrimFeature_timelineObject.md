# UntrimFeature.timelineObject Property

Parent Object: [UntrimFeature](UntrimFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"untrimFeature_var" is a variable referencing a UntrimFeature object.  

```python
# Get the value of the property.
propertyValue = untrimFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version January 2021
