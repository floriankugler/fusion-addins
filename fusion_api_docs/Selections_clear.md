# Selections.clear Method

Parent Object: [Selections](Selections.md)  

## Description

Clears the selection set so no entities are currently selected.

## Syntax

"selections_var" is a variable referencing a [Selections](Selections.md) object.

```python
returnValue = selections_var.clear()
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Version

Introduced in version August 2014
