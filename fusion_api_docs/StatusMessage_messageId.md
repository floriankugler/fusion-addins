# StatusMessage.messageId Property

Parent Object: [StatusMessage](StatusMessage.md)  

## Description

Gets and sets the ID of the message being used. This is a predefined ID within the Fusion message string table.

## Syntax

"statusMessage_var" is a variable referencing a StatusMessage object.  

```python
# Get the value of the property.
propertyValue = statusMessage_var.messageId

# Set the value of the property.
statusMessage_var.messageId = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version July 2021
