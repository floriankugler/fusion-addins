# LoftFeature.attributes Property

Parent Object: [LoftFeature](LoftFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"loftFeature_var" is a variable referencing a LoftFeature object.  

```python
# Get the value of the property.
propertyValue = loftFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
