# Point3D.distanceTo Method

Parent Object: [Point3D](Point3D.md)  

## Description

Returns the distance from this point to another point.

## Syntax

"point3D_var" is a variable referencing a [Point3D](Point3D.md) object.

```python
returnValue = point3D_var.distanceTo(point)
```

## Return Value

| Type   | Description                        |
|--------|------------------------------------|
| double | Returns the distance to the point. |

## Parameters

| Name  | Type                   | Description                           |
|-------|------------------------|---------------------------------------|
| point | [Point3D](Point3D.md) | The point to measure the distance to. |

## Version

Introduced in version August 2014
