# DataFile.inUseBy Property

Parent Object: [DataFile](DataFile.md)  

## Description

Returns the array of users that are currently using (have open for edit) this data file.

## Syntax

"dataFile_var" is a variable referencing a DataFile object.  

```python
# Get the value of the property.
propertyValue = dataFile_var.inUseBy
```

## Property Value

This is a read only property whose value is an array of type [User](User.md).

## Version

Introduced in version September 2016
