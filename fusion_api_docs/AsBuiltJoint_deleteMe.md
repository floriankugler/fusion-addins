# AsBuiltJoint.deleteMe Method

Parent Object: [AsBuiltJoint](AsBuiltJoint.md)  

## Description

Deletes this as-built joint.

## Syntax

"asBuiltJoint_var" is a variable referencing an [AsBuiltJoint](AsBuiltJoint.md) object.

```python
returnValue = asBuiltJoint_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version September 2015
