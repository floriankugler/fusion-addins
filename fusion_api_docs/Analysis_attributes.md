# Analysis.attributes Property

Parent Object: [Analysis](Analysis.md)  

## Description

A property that returns the collection of attributes associated with this Analysis.

## Syntax

"analysis_var" is a variable referencing an Analysis object.  

```python
# Get the value of the property.
propertyValue = analysis_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version January 2023
