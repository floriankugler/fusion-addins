# UserParameter.comment Property

Parent Object: [UserParameter](UserParameter.md)  

## Description

The comment associated with this parameter

## Syntax

"userParameter_var" is a variable referencing a UserParameter object.  

```python
# Get the value of the property.
propertyValue = userParameter_var.comment

# Set the value of the property.
userParameter_var.comment = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version August 2014
