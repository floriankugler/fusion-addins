# ModelParameter.createdBy Property

Parent Object: [ModelParameter](ModelParameter.md)  

## Description

Returns the object that created this parameter. For example, a feature, a sketch dimension, or a construction plane.

## Syntax

"modelParameter_var" is a variable referencing a ModelParameter object.  

```python
# Get the value of the property.
propertyValue = modelParameter_var.createdBy
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version August 2014
