# BoundingBox2D.expand Method

Parent Object: [BoundingBox2D](BoundingBox2D.md)  

## Description

Expand this bounding box to contain the specified point.

## Syntax

"boundingBox2D_var" is a variable referencing a [BoundingBox2D](BoundingBox2D.md) object.

```python
returnValue = boundingBox2D_var.expand(point)
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name  | Type                   | Description                     |
|-------|------------------------|---------------------------------|
| point | [Point2D](Point2D.md) | The point to expand the box to. |

## Version

Introduced in version August 2014
