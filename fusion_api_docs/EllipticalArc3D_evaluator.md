# EllipticalArc3D.evaluator Property

Parent Object: [EllipticalArc3D](EllipticalArc3D.md)  

## Description

Returns an evaluator object that lets you perform additional evaluations on the curve.

## Syntax

"ellipticalArc3D_var" is a variable referencing an EllipticalArc3D object.  

```python
# Get the value of the property.
propertyValue = ellipticalArc3D_var.evaluator
```

## Property Value

This is a read only property whose value is a [CurveEvaluator3D](CurveEvaluator3D.md).

## Version

Introduced in version August 2014
