# CustomGraphicsGroup.deleteMe Method

Parent Object: [CustomGraphicsGroup](CustomGraphicsGroup.md)  

## Description

Deletes the entity from the custom graphics group.

## Syntax

"customGraphicsGroup_var" is a variable referencing a [CustomGraphicsGroup](CustomGraphicsGroup.md) object.

```python
returnValue = customGraphicsGroup_var.deleteMe()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns true if the deletion was successful. |

## Version

Introduced in version September 2017
