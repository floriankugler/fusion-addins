# SmoothConstraint.attributes Property

Parent Object: [SmoothConstraint](SmoothConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"smoothConstraint_var" is a variable referencing a SmoothConstraint object.  

```python
# Get the value of the property.
propertyValue = smoothConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
