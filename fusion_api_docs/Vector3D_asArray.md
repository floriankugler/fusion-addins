# Vector3D.asArray Method

Parent Object: [Vector3D](Vector3D.md)  

## Description

Returns the vector coordinates as an array \[x, y, z\].

## Syntax

"vector3D_var" is a variable referencing a [Vector3D](Vector3D.md) object.

```python
returnValue = vector3D_var.asArray()
```

## Return Value

| Type       | Description                                          |
|------------|------------------------------------------------------|
| double\[\] | Returns the array of vector coordinates \[x, y, z\]. |

## Version

Introduced in version August 2014
