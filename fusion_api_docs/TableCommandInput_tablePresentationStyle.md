# TableCommandInput.tablePresentationStyle Property

Parent Object: [TableCommandInput](TableCommandInput.md)  

## Description

Gets and sets the presentation style the table is currently using for its display.

## Syntax

"tableCommandInput_var" is a variable referencing a TableCommandInput object.  

```python
# Get the value of the property.
propertyValue = tableCommandInput_var.tablePresentationStyle

# Set the value of the property.
tableCommandInput_var.tablePresentationStyle = propertyValue
```

## Property Value

This is a read/write property whose value is a [TablePresentationStyles](TablePresentationStyles.md).

## Version

Introduced in version September 2016
