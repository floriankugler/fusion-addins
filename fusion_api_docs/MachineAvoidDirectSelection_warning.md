# MachineAvoidDirectSelection.warning Property

Parent Object: [MachineAvoidDirectSelection](MachineAvoidDirectSelection.md)  

## Description

Gets the last warning string encountered after the selection was applied to a parent.

## Syntax

"machineAvoidDirectSelection_var" is a variable referencing a MachineAvoidDirectSelection object.  

```python
# Get the value of the property.
propertyValue = machineAvoidDirectSelection_var.warning
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version September 2024
