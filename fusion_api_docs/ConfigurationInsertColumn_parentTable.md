# ConfigurationInsertColumn.parentTable Property

Parent Object: [ConfigurationInsertColumn](ConfigurationInsertColumn.md)  

## Description

Returns the parent table, either top or custom theme table, this column is in.

## Syntax

"configurationInsertColumn_var" is a variable referencing a ConfigurationInsertColumn object.  

```python
# Get the value of the property.
propertyValue = configurationInsertColumn_var.parentTable
```

## Property Value

This is a read only property whose value is a [ConfigurationTable](ConfigurationTable.md).

## Version

Introduced in version March 2024
