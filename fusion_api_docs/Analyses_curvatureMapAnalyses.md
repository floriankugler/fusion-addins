# Analyses.curvatureMapAnalyses Property

Parent Object: [Analyses](Analyses.md)  

## Description

Returns the CurvatureMapAnalyses object, which provides access to any existing CurvatureMapAnalysis objects in the design.

## Syntax

"analyses_var" is a variable referencing an Analyses object.  

```python
# Get the value of the property.
propertyValue = analyses_var.curvatureMapAnalyses
```

## Property Value

This is a read only property whose value is a [CurvatureMapAnalyses](CurvatureMapAnalyses.md).

## Version

Introduced in version January 2023
