# LoftFeatures.count Property

Parent Object: [LoftFeatures](LoftFeatures.md)  

## Description

The number of loft features in the collection.

## Syntax

"loftFeatures_var" is a variable referencing a LoftFeatures object.  

```python
# Get the value of the property.
propertyValue = loftFeatures_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version September 2015
