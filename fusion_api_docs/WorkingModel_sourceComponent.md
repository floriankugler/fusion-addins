# WorkingModel.sourceComponent Property

Parent Object: [WorkingModel](WorkingModel.md)  

## Description

Returns the component being referenced by this working model.

## Syntax

"workingModel_var" is a variable referencing a WorkingModel object.  

```python
# Get the value of the property.
propertyValue = workingModel_var.sourceComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version January 2024
