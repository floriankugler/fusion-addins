# FilletFeature.attributes Property

Parent Object: [FilletFeature](FilletFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"filletFeature_var" is a variable referencing a FilletFeature object.  

```python
# Get the value of the property.
propertyValue = filletFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
