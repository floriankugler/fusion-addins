# CAMHoleRecognition.isOptional Property

Parent Object: [CAMHoleRecognition](CAMHoleRecognition.md)  

## Description

Gets and sets the "Optional" property value of the operation. Gets/sets true if the operation is optional.

## Syntax

"cAMHoleRecognition_var" is a variable referencing a CAMHoleRecognition object.  

```python
# Get the value of the property.
propertyValue = cAMHoleRecognition_var.isOptional

# Set the value of the property.
cAMHoleRecognition_var.isOptional = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version April 2023
