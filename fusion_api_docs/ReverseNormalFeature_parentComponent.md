# ReverseNormalFeature.parentComponent Property

Parent Object: [ReverseNormalFeature](ReverseNormalFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"reverseNormalFeature_var" is a variable referencing a ReverseNormalFeature object.  

```python
# Get the value of the property.
propertyValue = reverseNormalFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version September 2015
