# RotaryMachineAxisInput.physicalRange Property

Parent Object: [RotaryMachineAxisInput](RotaryMachineAxisInput.md)  

## Description

Range of possible values for this axis. Units are cm for linear axes or radians for rotary axes.

## Syntax

"rotaryMachineAxisInput_var" is a variable referencing a RotaryMachineAxisInput object.  

```python
# Get the value of the property.
propertyValue = rotaryMachineAxisInput_var.physicalRange

# Set the value of the property.
rotaryMachineAxisInput_var.physicalRange = propertyValue
```

## Property Value

This is a read/write property whose value is a [MachineAxisRange](MachineAxisRange.md).

## Version

Introduced in version April 2023
