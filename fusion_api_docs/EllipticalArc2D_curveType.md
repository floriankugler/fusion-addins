# EllipticalArc2D.curveType Property

Parent Object: [EllipticalArc2D](EllipticalArc2D.md)  

## Description

Returns the type of geometry this curve represents.

## Syntax

"ellipticalArc2D_var" is a variable referencing an EllipticalArc2D object.  

```python
# Get the value of the property.
propertyValue = ellipticalArc2D_var.curveType
```

## Property Value

This is a read only property whose value is a [Curve2DTypes](Curve2DTypes.md).

## Version

Introduced in version August 2014
