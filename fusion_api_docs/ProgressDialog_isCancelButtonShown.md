# ProgressDialog.isCancelButtonShown Property

Parent Object: [ProgressDialog](ProgressDialog.md)  

## Description

Gets and sets if the cancel button is included in the dialog. This is false by default.

## Syntax

"progressDialog_var" is a variable referencing a ProgressDialog object.  

```python
# Get the value of the property.
propertyValue = progressDialog_var.isCancelButtonShown

# Set the value of the property.
progressDialog_var.isCancelButtonShown = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.
