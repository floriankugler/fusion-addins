# TwoSidesToExtentDefinition.toEntityOne Property

Parent Object: [TwoSidesToExtentDefinition](TwoSidesToExtentDefinition.md)  

## Description

Gets and sets the entity that defines the extent on side one. The valid types of entities can vary depending on the type of feature this is being used with.

## Syntax

"twoSidesToExtentDefinition_var" is a variable referencing a TwoSidesToExtentDefinition object.  

```python
# Get the value of the property.
propertyValue = twoSidesToExtentDefinition_var.toEntityOne

# Set the value of the property.
twoSidesToExtentDefinition_var.toEntityOne = propertyValue
```

## Property Value

This is a read/write property whose value is a [Base](Base.md).

## Version

Introduced in version March 2022
