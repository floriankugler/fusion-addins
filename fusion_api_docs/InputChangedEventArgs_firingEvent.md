# InputChangedEventArgs.firingEvent Property

Parent Object: [InputChangedEventArgs](InputChangedEventArgs.md)  

## Description

The event that the firing is in response to.

## Syntax

"inputChangedEventArgs_var" is a variable referencing an InputChangedEventArgs object.  

```python
# Get the value of the property.
propertyValue = inputChangedEventArgs_var.firingEvent
```

## Property Value

This is a read only property whose value is an [Event](Event.md).

## Version

Introduced in version August 2014
