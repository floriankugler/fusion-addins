# DerivedParameter.attributes Property

Parent Object: [DerivedParameter](DerivedParameter.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"derivedParameter_var" is a variable referencing a DerivedParameter object.  

```python
# Get the value of the property.
propertyValue = derivedParameter_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version January 2026
