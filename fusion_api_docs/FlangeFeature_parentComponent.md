# FlangeFeature.parentComponent Property

Parent Object: [FlangeFeature](FlangeFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"flangeFeature_var" is a variable referencing a FlangeFeature object.  

```python
# Get the value of the property.
propertyValue = flangeFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version August 2020
