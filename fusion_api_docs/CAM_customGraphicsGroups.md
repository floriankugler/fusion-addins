# CAM.customGraphicsGroups Property

Parent Object: [CAM](CAM.md)  

## Description

Returns the customGraphicsGroups object associated with the CAM product.

## Syntax

"cAM_var" is a variable referencing a CAM object.  

```python
# Get the value of the property.
propertyValue = cAM_var.customGraphicsGroups
```

## Property Value

This is a read only property whose value is a [CustomGraphicsGroups](CustomGraphicsGroups.md).

## Version

Introduced in version May 2020
