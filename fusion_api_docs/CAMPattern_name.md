# CAMPattern.name Property

Parent Object: [CAMPattern](CAMPattern.md)  

## Description

Gets and sets the name of the operation as seen in the browser.Â This name is unique as compared to the names of all other operations in the document.

## Syntax

"cAMPattern_var" is a variable referencing a CAMPattern object.  

```python
# Get the value of the property.
propertyValue = cAMPattern_var.name

# Set the value of the property.
cAMPattern_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version January 2016
