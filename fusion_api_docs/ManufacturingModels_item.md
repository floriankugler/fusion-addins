# ManufacturingModels.item Method

Parent Object: [ManufacturingModels](ManufacturingModels.md)  

## Description

Get ManufacturingModel by index in collection.

## Syntax

"manufacturingModels_var" is a variable referencing a [ManufacturingModels](ManufacturingModels.md) object.

```python
returnValue = manufacturingModels_var.item(index)
```

## Return Value

| Type | Description |
|----|----|
| [ManufacturingModel](ManufacturingModel.md) | Returns ManufacturingModel at given index. |

## Parameters

| Name  | Type     | Description |
|-------|----------|-------------|
| index | uinteger |             |

## Version

Introduced in version April 2023
