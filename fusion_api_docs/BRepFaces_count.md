# BRepFaces.count Property

Parent Object: [BRepFaces](BRepFaces.md)  

## Description

The number of faces in the collection.

## Syntax

"bRepFaces_var" is a variable referencing a BRepFaces object.  

```python
# Get the value of the property.
propertyValue = bRepFaces_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version August 2014
