# Component.modelParameters Property

Parent Object: [Component](Component.md)  

## Description

Returns the collection of model parameters in the Component.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.modelParameters
```

## Property Value

This is a read only property whose value is a [ModelParameters](ModelParameters.md).

## Version

Introduced in version August 2014
