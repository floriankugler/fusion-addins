# SketchControlPointSpline.degree Property

Parent Object: [SketchControlPointSpline](SketchControlPointSpline.md)  

## Description

Gets and sets the degree of the spline.

## Syntax

"sketchControlPointSpline_var" is a variable referencing a SketchControlPointSpline object.  

```python
# Get the value of the property.
propertyValue = sketchControlPointSpline_var.degree

# Set the value of the property.
sketchControlPointSpline_var.degree = propertyValue
```

## Property Value

This is a read/write property whose value is a [SplineDegrees](SplineDegrees.md).

## Version

Introduced in version July 2022
