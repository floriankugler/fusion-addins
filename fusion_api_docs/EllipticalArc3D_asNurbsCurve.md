# EllipticalArc3D.asNurbsCurve Property

Parent Object: [EllipticalArc3D](EllipticalArc3D.md)  

## Description

Returns a NURBS curve that is geometrically identical to the elliptical arc.

## Syntax

"ellipticalArc3D_var" is a variable referencing an EllipticalArc3D object.  

```python
# Get the value of the property.
propertyValue = ellipticalArc3D_var.asNurbsCurve
```

## Property Value

This is a read only property whose value is a [NurbsCurve3D](NurbsCurve3D.md).

## Version

Introduced in version August 2014
