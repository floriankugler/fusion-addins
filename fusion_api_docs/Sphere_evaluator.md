# Sphere.evaluator Property

Parent Object: [Sphere](Sphere.md)  

## Description

Returns the surface evaluator.

## Syntax

"sphere_var" is a variable referencing a Sphere object.  

```python
# Get the value of the property.
propertyValue = sphere_var.evaluator
```

## Property Value

This is a read only property whose value is a [SurfaceEvaluator](SurfaceEvaluator.md).

## Version

Introduced in version August 2014
