# LineParallelToPlanarSurfaceConstraint.planarSurface Property

Parent Object: [LineParallelToPlanarSurfaceConstraint](LineParallelToPlanarSurfaceConstraint.md)  

## Description

Returns the planar BRepFace or ConstructionPlane the line is parallel to.

## Syntax

"lineParallelToPlanarSurfaceConstraint_var" is a variable referencing a LineParallelToPlanarSurfaceConstraint object.  

```python
# Get the value of the property.
propertyValue = lineParallelToPlanarSurfaceConstraint_var.planarSurface
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version September 2023
