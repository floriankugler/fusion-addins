# ConcentricConstraint.parentSketch Property

Parent Object: [ConcentricConstraint](ConcentricConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"concentricConstraint_var" is a variable referencing a ConcentricConstraint object.  

```python
# Get the value of the property.
propertyValue = concentricConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
