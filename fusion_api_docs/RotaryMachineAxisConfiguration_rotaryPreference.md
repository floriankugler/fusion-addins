# RotaryMachineAxisConfiguration.rotaryPreference Property

Parent Object: [RotaryMachineAxisConfiguration](RotaryMachineAxisConfiguration.md)  

## Description

Specify the preferred angle direction at the beginning of an operation.

## Syntax

"rotaryMachineAxisConfiguration_var" is a variable referencing a RotaryMachineAxisConfiguration object.  

```python
# Get the value of the property.
propertyValue = rotaryMachineAxisConfiguration_var.rotaryPreference

# Set the value of the property.
rotaryMachineAxisConfiguration_var.rotaryPreference = propertyValue
```

## Property Value

This is a read/write property whose value is a [MachineAnglePreferences](MachineAnglePreferences.md).

## Version

Introduced in version April 2023
