# ConfigurationVisibilityColumn.id Property

Parent Object: [ConfigurationVisibilityColumn](ConfigurationVisibilityColumn.md)  

## Description

The id of the column. This is constant and cannot be changed by the user.

## Syntax

"configurationVisibilityColumn_var" is a variable referencing a ConfigurationVisibilityColumn object.  

```python
# Get the value of the property.
propertyValue = configurationVisibilityColumn_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version January 2024
