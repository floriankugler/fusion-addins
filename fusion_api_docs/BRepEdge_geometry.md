# BRepEdge.geometry Property

Parent Object: [BRepEdge](BRepEdge.md)  

## Description

Returns the underlying curve geometry of the edge.

## Syntax

"bRepEdge_var" is a variable referencing a BRepEdge object.  

```python
# Get the value of the property.
propertyValue = bRepEdge_var.geometry
```

## Property Value

This is a read only property whose value is a [Curve3D](Curve3D.md).

## Version

Introduced in version August 2014
