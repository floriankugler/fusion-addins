# TextCommandPalette.top Property

Parent Object: [TextCommandPalette](TextCommandPalette.md)  

## Description

Gets and sets the top of the palette relative to screen space and in pixels. Because palettes can be positioned outside of the Fusion window, a value of zero indicates the top of the screen and not the Fusion window.

## Syntax

"textCommandPalette_var" is a variable referencing a TextCommandPalette object.  

```python
# Get the value of the property.
propertyValue = textCommandPalette_var.top

# Set the value of the property.
textCommandPalette_var.top = propertyValue
```

## Property Value

This is a read/write property whose value is an integer.

## Version

Introduced in version August 2017
