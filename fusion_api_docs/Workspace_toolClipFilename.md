# Workspace.toolClipFilename Property

Parent Object: [Workspace](Workspace.md)  

## Description

Gets or sets the full filename of the image file (PNG) used for the tool clip. the tool clip is the image shown when the user hovers the mouse over the workspace name in the workspace drop-down.

## Syntax

"workspace_var" is a variable referencing a Workspace object.  

```python
# Get the value of the property.
propertyValue = workspace_var.toolClipFilename

# Set the value of the property.
workspace_var.toolClipFilename = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version September 2017
