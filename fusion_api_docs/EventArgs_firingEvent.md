# EventArgs.firingEvent Property

Parent Object: [EventArgs](EventArgs.md)  

## Description

The event that the firing is in response to.

## Syntax

"eventArgs_var" is a variable referencing an EventArgs object.  

```python
# Get the value of the property.
propertyValue = eventArgs_var.firingEvent
```

## Property Value

This is a read only property whose value is an [Event](Event.md).

## Version

Introduced in version August 2014
