# TableCommandInput.clear Method

Parent Object: [TableCommandInput](TableCommandInput.md)  

## Description

Removes all rows in the table and the toolbar.

## Syntax

"tableCommandInput_var" is a variable referencing a [TableCommandInput](TableCommandInput.md) object.

```python
returnValue = tableCommandInput_var.clear()
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Version

Introduced in version September 2016
