# SeparatorControl.id Property

Parent Object: [SeparatorControl](SeparatorControl.md)  

## Description

Gets the unique ID of this control. The ID is unique with respect to the other controls within the same panel, toolbar, or drop-down control.

## Syntax

"separatorControl_var" is a variable referencing a SeparatorControl object.  

```python
# Get the value of the property.
propertyValue = separatorControl_var.id
```

## Property Value

This is a read only property whose value is a string.
