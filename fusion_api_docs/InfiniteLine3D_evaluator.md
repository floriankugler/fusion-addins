# InfiniteLine3D.evaluator Property

Parent Object: [InfiniteLine3D](InfiniteLine3D.md)  

## Description

Returns an evaluator object that lets you perform additional evaluations on the curve.

## Syntax

"infiniteLine3D_var" is a variable referencing an InfiniteLine3D object.  

```python
# Get the value of the property.
propertyValue = infiniteLine3D_var.evaluator
```

## Property Value

This is a read only property whose value is a [CurveEvaluator3D](CurveEvaluator3D.md).

## Version

Introduced in version August 2014
