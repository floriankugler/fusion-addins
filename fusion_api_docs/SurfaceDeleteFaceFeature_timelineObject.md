# SurfaceDeleteFaceFeature.timelineObject Property

Parent Object: [SurfaceDeleteFaceFeature](SurfaceDeleteFaceFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"surfaceDeleteFaceFeature_var" is a variable referencing a SurfaceDeleteFaceFeature object.  

```python
# Get the value of the property.
propertyValue = surfaceDeleteFaceFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version August 2016
