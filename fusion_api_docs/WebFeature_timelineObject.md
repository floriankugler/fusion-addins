# WebFeature.timelineObject Property

Parent Object: [WebFeature](WebFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"webFeature_var" is a variable referencing a WebFeature object.  

```python
# Get the value of the property.
propertyValue = webFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version September 2015
