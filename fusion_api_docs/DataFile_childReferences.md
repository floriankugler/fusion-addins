# DataFile.childReferences Property

Parent Object: [DataFile](DataFile.md)  

## Description

Returns a collection of DataFiles that are the children (referenced designs) this datafile references.

## Syntax

"dataFile_var" is a variable referencing a DataFile object.  

```python
# Get the value of the property.
propertyValue = dataFile_var.childReferences
```

## Property Value

This is a read only property whose value is a [DataFiles](DataFiles.md).

## Version

Introduced in version September 2016
