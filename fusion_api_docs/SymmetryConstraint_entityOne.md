# SymmetryConstraint.entityOne Property

Parent Object: [SymmetryConstraint](SymmetryConstraint.md)  

## Description

Returns the first curve.

## Syntax

"symmetryConstraint_var" is a variable referencing a SymmetryConstraint object.  

```python
# Get the value of the property.
propertyValue = symmetryConstraint_var.entityOne
```

## Property Value

This is a read only property whose value is a [SketchEntity](SketchEntity.md).

## Version

Introduced in version August 2014
