# BossFeature.holeMajorDraftAngle Property

Parent Object: [BossFeature](BossFeature.md)  

## Description

Returns the model parameter controlling major hole draft angle for counterbore and countersink hole. If hole type is set to simple hole or boss shape is to BossBlank this parameter is unused.

## Syntax

"bossFeature_var" is a variable referencing a BossFeature object.  

```python
# Get the value of the property.
propertyValue = bossFeature_var.holeMajorDraftAngle
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version October 2022
