# WorkingModel.workspaces Property

Parent Object: [WorkingModel](WorkingModel.md)  

## Description

Returns the workspaces associated with this product.

## Syntax

"workingModel_var" is a variable referencing a WorkingModel object.  

```python
# Get the value of the property.
propertyValue = workingModel_var.workspaces
```

## Property Value

This is a read only property whose value is a [WorkspaceList](WorkspaceList.md).

## Version

Introduced in version January 2024
