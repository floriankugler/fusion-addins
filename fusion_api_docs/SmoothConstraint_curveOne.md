# SmoothConstraint.curveOne Property

Parent Object: [SmoothConstraint](SmoothConstraint.md)  

## Description

Returns the first curve.

## Syntax

"smoothConstraint_var" is a variable referencing a SmoothConstraint object.  

```python
# Get the value of the property.
propertyValue = smoothConstraint_var.curveOne
```

## Property Value

This is a read only property whose value is a [SketchCurve](SketchCurve.md).

## Version

Introduced in version August 2014
