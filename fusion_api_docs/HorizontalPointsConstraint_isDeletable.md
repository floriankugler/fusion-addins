# HorizontalPointsConstraint.isDeletable Property

Parent Object: [HorizontalPointsConstraint](HorizontalPointsConstraint.md)  

## Description

Indicates if this constraint is deletable.

## Syntax

"horizontalPointsConstraint_var" is a variable referencing a HorizontalPointsConstraint object.  

```python
# Get the value of the property.
propertyValue = horizontalPointsConstraint_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
