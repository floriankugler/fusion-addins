# CAM.workspaces Property

Parent Object: [CAM](CAM.md)  

## Description

Returns the workspaces associated with this product.

## Syntax

"cAM_var" is a variable referencing a CAM object.  

```python
# Get the value of the property.
propertyValue = cAM_var.workspaces
```

## Property Value

This is a read only property whose value is a [WorkspaceList](WorkspaceList.md).

## Version

Introduced in version January 2016
