# Torus.surfaceType Property

Parent Object: [Torus](Torus.md)  

## Description

Returns the surface type.

## Syntax

"torus_var" is a variable referencing a Torus object.  

```python
# Get the value of the property.
propertyValue = torus_var.surfaceType
```

## Property Value

This is a read only property whose value is a [SurfaceTypes](SurfaceTypes.md).

## Version

Introduced in version August 2014
