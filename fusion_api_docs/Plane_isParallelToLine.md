# Plane.isParallelToLine Method

Parent Object: [Plane](Plane.md)  

## Description

Checks if this plane is parallel to a line.

## Syntax

"plane_var" is a variable referencing a [Plane](Plane.md) object.

```python
returnValue = plane_var.isParallelToLine(line)
```

## Return Value

| Type    | Description                                      |
|---------|--------------------------------------------------|
| boolean | Returns true if the plane and line are parallel. |

## Parameters

| Name | Type                 | Description                               |
|------|----------------------|-------------------------------------------|
| line | [Line3D](Line3D.md) | The line to compare with for parallelism. |

## Version

Introduced in version August 2014
