# SurfaceEvaluator.isClosedInV Property

Parent Object: [SurfaceEvaluator](SurfaceEvaluator.md)  

## Description

Returns if the surface is closed (forms a loop) in the V direction

## Syntax

"surfaceEvaluator_var" is a variable referencing a SurfaceEvaluator object.  

```python
# Get the value of the property.
propertyValue = surfaceEvaluator_var.isClosedInV
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
