# SymmetryConstraint.isDeletable Property

Parent Object: [SymmetryConstraint](SymmetryConstraint.md)  

## Description

Indicates if this constraint is deletable.

## Syntax

"symmetryConstraint_var" is a variable referencing a SymmetryConstraint object.  

```python
# Get the value of the property.
propertyValue = symmetryConstraint_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
