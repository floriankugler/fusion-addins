# Preferences.apiPreferences Property

Parent Object: [Preferences](Preferences.md)  

## Description

Gets the APIPreferences object, which provides access to the various preferences associated with the API.

## Syntax

"preferences_var" is a variable referencing a Preferences object.  

```python
# Get the value of the property.
propertyValue = preferences_var.apiPreferences
```

## Property Value

This is a read only property whose value is an [APIPreferences](APIPreferences.md).

## Version

Introduced in version October 2023
