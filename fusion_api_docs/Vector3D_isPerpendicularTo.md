# Vector3D.isPerpendicularTo Method

Parent Object: [Vector3D](Vector3D.md)  

## Description

Determines if the input vector is perpendicular to this vector.

## Syntax

"vector3D_var" is a variable referencing a [Vector3D](Vector3D.md) object.

```python
returnValue = vector3D_var.isPerpendicularTo(vector)
```

## Return Value

| Type    | Description                                    |
|---------|------------------------------------------------|
| boolean | Returns true if the vectors are perpendicular. |

## Parameters

| Name   | Type                     | Description                             |
|--------|--------------------------|-----------------------------------------|
| vector | [Vector3D](Vector3D.md) | The vector to test perpendicularity to. |

## Version

Introduced in version August 2014
