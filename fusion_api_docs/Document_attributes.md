# Document.attributes Property

Parent Object: [Document](Document.md)  

## Description

Returns the collection of attributes associated with this document.

## Syntax

"document_var" is a variable referencing a Document object.  

```python
# Get the value of the property.
propertyValue = document_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
