# ContactSet.deleteMe Method

Parent Object: [ContactSet](ContactSet.md)  

## Description

Deletes this contact set from the design.

## Syntax

"contactSet_var" is a variable referencing a [ContactSet](ContactSet.md) object.

```python
returnValue = contactSet_var.deleteMe()
```

## Return Value

| Type    | Description                                 |
|---------|---------------------------------------------|
| boolean | Returns true if the delete was succcessful. |

## Version

Introduced in version September 2016
