# BossFeatureSideInput.alignmentType Property

Parent Object: [BossFeatureSideInput](BossFeatureSideInput.md)  

## Description

Get or set boss alignment shape. This usually corresponds to the alignment shape of the boss counterpart.

## Syntax

"bossFeatureSideInput_var" is a variable referencing a BossFeatureSideInput object.  

```python
# Get the value of the property.
propertyValue = bossFeatureSideInput_var.alignmentType

# Set the value of the property.
bossFeatureSideInput_var.alignmentType = propertyValue
```

## Property Value

This is a read/write property whose value is a [BossAlignmentTypes](BossAlignmentTypes.md).
