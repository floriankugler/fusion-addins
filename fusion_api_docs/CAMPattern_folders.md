# CAMPattern.folders Property

Parent Object: [CAMPattern](CAMPattern.md)  

## Description

Returns the Folders collection that provides access to existing folders in this folder.

## Syntax

"cAMPattern_var" is a variable referencing a CAMPattern object.  

```python
# Get the value of the property.
propertyValue = cAMPattern_var.folders
```

## Property Value

This is a read only property whose value is a [CAMFolders](CAMFolders.md).

## Version

Introduced in version January 2016
