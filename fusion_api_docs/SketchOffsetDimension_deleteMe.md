# SketchOffsetDimension.deleteMe Method

Parent Object: [SketchOffsetDimension](SketchOffsetDimension.md)  

## Description

Deletes this dimension. The IsDeletable property indicates if this dimension can be deleted.

## Syntax

"sketchOffsetDimension_var" is a variable referencing a [SketchOffsetDimension](SketchOffsetDimension.md) object.

```python
returnValue = sketchOffsetDimension_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
