# Arc2D.copy Method

Parent Object: [Arc2D](Arc2D.md)  

## Description

Creates and returns an independent copy of this Arc2D object.

## Syntax

"arc2D_var" is a variable referencing an [Arc2D](Arc2D.md) object.

```python
returnValue = arc2D_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [Arc2D](Arc2D.md) | Returns a new Arc2D object that is a copy of this Arc2D object. |

## Version

Introduced in version August 2014
