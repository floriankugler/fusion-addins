# ExtrudeFeature.deleteMe Method

Parent Object: [ExtrudeFeature](ExtrudeFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"extrudeFeature_var" is a variable referencing an [ExtrudeFeature](ExtrudeFeature.md) object.

```python
returnValue = extrudeFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version August 2014
