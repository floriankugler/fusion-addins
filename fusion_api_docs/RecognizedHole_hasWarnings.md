# RecognizedHole.hasWarnings Property

Parent Object: [RecognizedHole](RecognizedHole.md)  

## Description

Returns true if there are any warnings associated with this hole.

## Syntax

"recognizedHole_var" is a variable referencing a RecognizedHole object.  

```python
# Get the value of the property.
propertyValue = recognizedHole_var.hasWarnings
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version May 2023
