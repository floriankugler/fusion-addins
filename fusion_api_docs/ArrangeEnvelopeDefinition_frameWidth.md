# ArrangeEnvelopeDefinition.frameWidth Property

Parent Object: [ArrangeEnvelopeDefinition](ArrangeEnvelopeDefinition.md)  

## Description

Returns the parameter that controls the width of the envelope frame. This defines the offset distance of the objects from the edge of the frame. You can modify the value by using the properties on the returned ModelParameter object.

## Syntax

"arrangeEnvelopeDefinition_var" is a variable referencing an ArrangeEnvelopeDefinition object.  

```python
# Get the value of the property.
propertyValue = arrangeEnvelopeDefinition_var.frameWidth
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version January 2025
