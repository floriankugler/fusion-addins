# SketchLinearDimension.textPosition Property

Parent Object: [SketchLinearDimension](SketchLinearDimension.md)  

## Description

Gets and sets position of the dimension text.

## Syntax

"sketchLinearDimension_var" is a variable referencing a SketchLinearDimension object.  

```python
# Get the value of the property.
propertyValue = sketchLinearDimension_var.textPosition

# Set the value of the property.
sketchLinearDimension_var.textPosition = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
