# CustomGraphicsGroup.name Property

Parent Object: [CustomGraphicsGroup](CustomGraphicsGroup.md)  

## Description

Gets and sets the name displayed when this entity is selected. If no name has been set, "Custom Graphics" will be displayed.

## Syntax

"customGraphicsGroup_var" is a variable referencing a CustomGraphicsGroup object.  

```python
# Get the value of the property.
propertyValue = customGraphicsGroup_var.name

# Set the value of the property.
customGraphicsGroup_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version October 2022
