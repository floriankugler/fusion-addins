# CAMArrangeParameterValue.isValid Property

Parent Object: [CAMArrangeParameterValue](CAMArrangeParameterValue.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"cAMArrangeParameterValue_var" is a variable referencing a CAMArrangeParameterValue object.  

```python
# Get the value of the property.
propertyValue = cAMArrangeParameterValue_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version March 2024
