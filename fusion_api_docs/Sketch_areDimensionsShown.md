# Sketch.areDimensionsShown Property

Parent Object: [Sketch](Sketch.md)  

## Description

Indicates if the dimensions of the sketch are displayed when the sketch is not active (in sketch edit mode)

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.areDimensionsShown

# Set the value of the property.
sketch_var.areDimensionsShown = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
