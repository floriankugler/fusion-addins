# ScaleFeatureInput.point Property

Parent Object: [ScaleFeatureInput](ScaleFeatureInput.md)  

## Description

Gets and sets the origin point of the scale. This can be a BRepVertex, a SketchPoint or a ConstructionPoint.

## Syntax

"scaleFeatureInput_var" is a variable referencing a ScaleFeatureInput object.  

```python
# Get the value of the property.
propertyValue = scaleFeatureInput_var.point

# Set the value of the property.
scaleFeatureInput_var.point = propertyValue
```

## Property Value

This is a read/write property whose value is a [Base](Base.md).

## Version

Introduced in version January 2015
