# BaseComponent.constructionPoints Property

Parent Object: [BaseComponent](BaseComponent.md)  

## Description

Returns the construction points collection associated with this component. This provides access to the existing construction points and supports the creation of new construction points.

## Syntax

"baseComponent_var" is a variable referencing a BaseComponent object.  

```python
# Get the value of the property.
propertyValue = baseComponent_var.constructionPoints
```

## Property Value

This is a read only property whose value is a [ConstructionPoints](ConstructionPoints.md).

## Version

Introduced in version August 2014
