# SymmetryConstraint.parentSketch Property

Parent Object: [SymmetryConstraint](SymmetryConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"symmetryConstraint_var" is a variable referencing a SymmetryConstraint object.  

```python
# Get the value of the property.
propertyValue = symmetryConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
