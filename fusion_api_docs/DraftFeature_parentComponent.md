# DraftFeature.parentComponent Property

Parent Object: [DraftFeature](DraftFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"draftFeature_var" is a variable referencing a DraftFeature object.  

```python
# Get the value of the property.
propertyValue = draftFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version January 2015
