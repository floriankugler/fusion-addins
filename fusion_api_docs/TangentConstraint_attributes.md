# TangentConstraint.attributes Property

Parent Object: [TangentConstraint](TangentConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"tangentConstraint_var" is a variable referencing a TangentConstraint object.  

```python
# Get the value of the property.
propertyValue = tangentConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
