# ValueInput.createByObject Method

Parent Object: [ValueInput](ValueInput.md)  

## Description

Creates a new ValueInput object that contains a reference to any Fusion object.

## Syntax

This is a static method.  

```python

returnValue = adsk.core.ValueInput.createByObject(objectReference)
```

## Return Value

| Type | Description |
|----|----|
| [ValueInput](ValueInput.md) | Returns the newly created ValueInput object or null if the creation failed. |

## Parameters

| Name | Type | Description |
|----|----|----|
| objectReference | [Base](Base.md) | The Fusion object that you want to create the ValueInput for. |

## Version

Introduced in version August 2016
