# Path.assemblyContext Property

Parent Object: [Path](Path.md)  

## Description

This property is not supported for the Path object.

## Syntax

"path_var" is a variable referencing a Path object.  

```python
# Get the value of the property.
propertyValue = path_var.assemblyContext
```

## Property Value

This is a read only property whose value is an [Occurrence](Occurrence.md).

## Version

Introduced in version November 2014
