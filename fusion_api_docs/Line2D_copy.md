# Line2D.copy Method

Parent Object: [Line2D](Line2D.md)  

## Description

Creates and returns a copy of this line object.

## Syntax

"line2D_var" is a variable referencing a [Line2D](Line2D.md) object.

```python
returnValue = line2D_var.copy()
```

## Return Value

| Type                 | Description                                      |
|----------------------|--------------------------------------------------|
| [Line2D](Line2D.md) | Returns an independent copy of this line object. |

## Version

Introduced in version August 2014
