# Component.motionLinks Property

Parent Object: [Component](Component.md)  

## Description

Returns the collection of MotionLinks associated with this component.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.motionLinks
```

## Property Value

This is a read only property whose value is a [MotionLinks](MotionLinks.md).

## Version

Introduced in version November 2025
