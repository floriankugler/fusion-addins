# ObjectCollection Object

Derived from: [Base](Base.md) Object  

## Description

Generic collection used to handle lists of any object type.

## Methods

| Name | Description |
|----|----|
| [add](ObjectCollection_add.md) | Adds an object to the end of the collection. Duplicates can be added to the collection. |
| [asArray](ObjectCollection_asArray.md) | Returns the content of the ObjectCollection as an array. |
| [classType](ObjectCollection_classType.md) | Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType(). |
| [clear](ObjectCollection_clear.md) | Clears the entire contents of the collection. |
| [contains](ObjectCollection_contains.md) | Returns whether the specified object exists within the collection. |
| [create](ObjectCollection_create.md) | Creates a new ObjectCollection object. |
| [createWithArray](ObjectCollection_createWithArray.md) | Creates a new ObjectCollection that is initialized with the content of the provided array. |
| [find](ObjectCollection_find.md) | Finds the specified component in the collection. |
| [item](ObjectCollection_item.md) | Function that returns the specified object using an index into the collection. |
| [removeByIndex](ObjectCollection_removeByIndex.md) | Function that removes an item from the list. Will fail if the list is read only. |
| [removeByItem](ObjectCollection_removeByItem.md) | Function that removes an item from the collection. |

## Properties

| Name | Description |
| --- | --- |
| [count](ObjectCollection_count.md) | Returns the number of occurrences in the collection. |
| [isValid](ObjectCollection_isValid.md) | Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference. |
| [objectType](ObjectCollection_objectType.md) | This property is supported by all objects in the API and returns a string that contains the full name (namespace::objecttype) describing the type of the object. It's often useful to use this in combination with the classType method to see if an object is a certain type. For example: if obj.objectType == adsk.core.Point3D.classType(): |
