# UnstitchFeature.healthState Property

Parent Object: [UnstitchFeature](UnstitchFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"unstitchFeature_var" is a variable referencing a UnstitchFeature object.  

```python
# Get the value of the property.
propertyValue = unstitchFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
