# Vector3D.isEqualTo Method

Parent Object: [Vector3D](Vector3D.md)  

## Description

Determines if this vector is equal to the specified vector.

## Syntax

"vector3D_var" is a variable referencing a [Vector3D](Vector3D.md) object.

```python
returnValue = vector3D_var.isEqualTo(vector)
```

## Return Value

| Type    | Description                            |
|---------|----------------------------------------|
| boolean | Returns true if the vectors are equal. |

## Parameters

| Name   | Type                     | Description                     |
|--------|--------------------------|---------------------------------|
| vector | [Vector3D](Vector3D.md) | The vector to test equality to. |

## Version

Introduced in version August 2014
