# CoilFeatureInput.operation Property

Parent Object: [CoilFeatureInput](CoilFeatureInput.md)  

## Description

Gets and sets the type of operation performed by the coil.

## Syntax

"coilFeatureInput_var" is a variable referencing a CoilFeatureInput object.  

```python
# Get the value of the property.
propertyValue = coilFeatureInput_var.operation

# Set the value of the property.
coilFeatureInput_var.operation = propertyValue
```

## Property Value

This is a read/write property whose value is a [FeatureOperations](FeatureOperations.md).

## Version

Introduced in version March 2016
