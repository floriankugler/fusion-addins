# Timeline.markerPosition Property

Parent Object: [Timeline](Timeline.md)  

## Description

Gets and sets the current position of the marker where 0 is at the beginning of the timeline and the value of Timeline.count is the end of the timeline.

## Syntax

"timeline_var" is a variable referencing a Timeline object.  

```python
# Get the value of the property.
propertyValue = timeline_var.markerPosition

# Set the value of the property.
timeline_var.markerPosition = propertyValue
```

## Property Value

This is a read/write property whose value is an integer.

## Version

Introduced in version August 2014
