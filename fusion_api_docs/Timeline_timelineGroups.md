# Timeline.timelineGroups Property

Parent Object: [Timeline](Timeline.md)  

## Description

Returns the collection of groups within the timeline.

## Syntax

"timeline_var" is a variable referencing a Timeline object.  

```python
# Get the value of the property.
propertyValue = timeline_var.timelineGroups
```

## Property Value

This is a read only property whose value is a [TimelineGroups](TimelineGroups.md).

## Version

Introduced in version August 2014
