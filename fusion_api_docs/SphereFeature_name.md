# SphereFeature.name Property

Parent Object: [SphereFeature](SphereFeature.md)  

## Description

Returns the name of the feature as seen in the browser (non-parametric) or in the timeline (parametric).

## Syntax

"sphereFeature_var" is a variable referencing a SphereFeature object.  

```python
# Get the value of the property.
propertyValue = sphereFeature_var.name

# Set the value of the property.
sphereFeature_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version September 2015
