# Line3D.create Method

Parent Object: [Line3D](Line3D.md)  

## Description

Creates a transient line.

## Syntax

This is a static method.  

```python

returnValue = adsk.core.Line3D.create(startPoint, endPoint)
```

## Return Value

| Type | Description |
|----|----|
| [Line3D](Line3D.md) | Returns the new Line3D object or null if the creation failed. |

## Parameters

| Name       | Type                   | Description                  |
|------------|------------------------|------------------------------|
| startPoint | [Point3D](Point3D.md) | The start point of the line. |
| endPoint   | [Point3D](Point3D.md) | The end point of the line.   |

## Version

Introduced in version August 2014
