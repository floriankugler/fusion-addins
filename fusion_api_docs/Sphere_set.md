# Sphere.set Method

Parent Object: [Sphere](Sphere.md)  

## Description

Sets all of the data defining the sphere.

## Syntax

"sphere_var" is a variable referencing a [Sphere](Sphere.md) object.

```python
returnValue = sphere_var.set(origin, radius)
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name   | Type                   | Description                              |
|--------|------------------------|------------------------------------------|
| origin | [Point3D](Point3D.md) | The origin point (center) of the sphere. |
| radius | double                 | The radius of the sphere.                |

## Version

Introduced in version August 2014
