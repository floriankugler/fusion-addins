# TrimFeature.attributes Property

Parent Object: [TrimFeature](TrimFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"trimFeature_var" is a variable referencing a TrimFeature object.  

```python
# Get the value of the property.
propertyValue = trimFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
