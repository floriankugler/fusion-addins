# TableCommandInput.name Property

Parent Object: [TableCommandInput](TableCommandInput.md)  

## Description

Gets the user visible name of this input.

## Syntax

"tableCommandInput_var" is a variable referencing a TableCommandInput object.  

```python
# Get the value of the property.
propertyValue = tableCommandInput_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version September 2016
