# SketchText.boundingBox Property

Parent Object: [SketchText](SketchText.md)  

## Description

Returns the bounding box of the entity in sketch space.

## Syntax

"sketchText_var" is a variable referencing a SketchText object.  

```python
# Get the value of the property.
propertyValue = sketchText_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version March 2015
