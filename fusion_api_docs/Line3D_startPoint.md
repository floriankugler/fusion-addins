# Line3D.startPoint Property

Parent Object: [Line3D](Line3D.md)  

## Description

Gets and sets the start point of the line.

## Syntax

"line3D_var" is a variable referencing a Line3D object.  

```python
# Get the value of the property.
propertyValue = line3D_var.startPoint

# Set the value of the property.
line3D_var.startPoint = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
