# PlasticRule.isDefault Property

Parent Object: [PlasticRule](PlasticRule.md)  

## Description

This gets and sets which rule in a library is the default rule. This is only valid for rules in a library and will fail for rules in a design.

## Syntax

"plasticRule_var" is a variable referencing a PlasticRule object.  

```python
# Get the value of the property.
propertyValue = plasticRule_var.isDefault

# Set the value of the property.
plasticRule_var.isDefault = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2024
