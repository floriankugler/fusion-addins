# SketchCurves.sketchEllipticalArcs Property

Parent Object: [SketchCurves](SketchCurves.md)  

## Description

Returns the sketch elliptical arcs collection associated with this sketch. This provides access to the existing sketch elliptical arcs and supports the creation of new sketch elliptical arcs.

## Syntax

"sketchCurves_var" is a variable referencing a SketchCurves object.  

```python
# Get the value of the property.
propertyValue = sketchCurves_var.sketchEllipticalArcs
```

## Property Value

This is a read only property whose value is a [SketchEllipticalArcs](SketchEllipticalArcs.md).

## Version

Introduced in version August 2014
