# BossFeature.offsetClearance Property

Parent Object: [BossFeature](BossFeature.md)  

## Description

Returns the model parameter controlling the offset clearance from the selected parting plane and offset.

## Syntax

"bossFeature_var" is a variable referencing a BossFeature object.  

```python
# Get the value of the property.
propertyValue = bossFeature_var.offsetClearance
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version October 2022
