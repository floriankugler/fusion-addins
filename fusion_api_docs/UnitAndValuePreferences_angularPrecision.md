# UnitAndValuePreferences.angularPrecision Property

Parent Object: [UnitAndValuePreferences](UnitAndValuePreferences.md)  

## Description

Gets and sets the angular precision. This value specifies the number of decimals to display.

## Syntax

"unitAndValuePreferences_var" is a variable referencing a UnitAndValuePreferences object.  

```python
# Get the value of the property.
propertyValue = unitAndValuePreferences_var.angularPrecision

# Set the value of the property.
unitAndValuePreferences_var.angularPrecision = propertyValue
```

## Property Value

This is a read/write property whose value is an integer.

## Version

Introduced in version August 2014
