# PostConfiguration.capability Property

Parent Object: [PostConfiguration](PostConfiguration.md)  

## Description

Gets the capabilities supported by the post. Capabilities define what types of operations can be post processed using this configuration.

## Syntax

"postConfiguration_var" is a variable referencing a PostConfiguration object.  

```python
# Get the value of the property.
propertyValue = postConfiguration_var.capability
```

## Property Value

This is a read only property whose value is a [PostCapabilities](PostCapabilities.md).

## Version

Introduced in version April 2023
