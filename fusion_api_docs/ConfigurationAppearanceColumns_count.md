# ConfigurationAppearanceColumns.count Property

Parent Object: [ConfigurationAppearanceColumns](ConfigurationAppearanceColumns.md)  

## Description

Returns the number of columns in the table where the name column is not included.

## Syntax

"configurationAppearanceColumns_var" is a variable referencing a ConfigurationAppearanceColumns object.  

```python
# Get the value of the property.
propertyValue = configurationAppearanceColumns_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version January 2024
