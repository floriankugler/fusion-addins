# FlangeFeature.name Property

Parent Object: [FlangeFeature](FlangeFeature.md)  

## Description

Returns the name of the feature as seen in the browser (non-parametric) or in the timeline (parametric).

## Syntax

"flangeFeature_var" is a variable referencing a FlangeFeature object.  

```python
# Get the value of the property.
propertyValue = flangeFeature_var.name

# Set the value of the property.
flangeFeature_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version August 2020
