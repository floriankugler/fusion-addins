# CoilFeatureInput.basePlane Property

Parent Object: [CoilFeatureInput](CoilFeatureInput.md)  

## Description

Gets and sets the base plane.

## Syntax

"coilFeatureInput_var" is a variable referencing a CoilFeatureInput object.  

```python
# Get the value of the property.
propertyValue = coilFeatureInput_var.basePlane

# Set the value of the property.
coilFeatureInput_var.basePlane = propertyValue
```

## Property Value

This is a read/write property whose value is a [Base](Base.md).

## Version

Introduced in version March 2016
