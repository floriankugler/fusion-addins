# AppearanceTextureProperty.isReadOnly Property

Parent Object: [AppearanceTextureProperty](AppearanceTextureProperty.md)  

## Description

Indicates if this property is read-only. If True any attempted edits will fail.

## Syntax

"appearanceTextureProperty_var" is a variable referencing an AppearanceTextureProperty object.  

```python
# Get the value of the property.
propertyValue = appearanceTextureProperty_var.isReadOnly
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
