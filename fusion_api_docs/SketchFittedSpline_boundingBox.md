# SketchFittedSpline.boundingBox Property

Parent Object: [SketchFittedSpline](SketchFittedSpline.md)  

## Description

Returns the bounding box of the entity in sketch space.

## Syntax

"sketchFittedSpline_var" is a variable referencing a SketchFittedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFittedSpline_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version August 2014
