# PocketRecognitionSelection.hasError Property

Parent Object: [PocketRecognitionSelection](PocketRecognitionSelection.md)  

## Description

Gets if errors were encountered when applying the selection to a a parent.

## Syntax

"pocketRecognitionSelection_var" is a variable referencing a PocketRecognitionSelection object.  

```python
# Get the value of the property.
propertyValue = pocketRecognitionSelection_var.hasError
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2023
