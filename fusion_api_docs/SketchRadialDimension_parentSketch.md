# SketchRadialDimension.parentSketch Property

Parent Object: [SketchRadialDimension](SketchRadialDimension.md)  

## Description

Returns the parent sketch object.

## Syntax

"sketchRadialDimension_var" is a variable referencing a SketchRadialDimension object.  

```python
# Get the value of the property.
propertyValue = sketchRadialDimension_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
