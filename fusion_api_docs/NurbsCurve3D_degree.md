# NurbsCurve3D.degree Property

Parent Object: [NurbsCurve3D](NurbsCurve3D.md)  

## Description

Returns the degree of the curve.

## Syntax

"nurbsCurve3D_var" is a variable referencing a NurbsCurve3D object.  

```python
# Get the value of the property.
propertyValue = nurbsCurve3D_var.degree
```

## Property Value

This is a read only property whose value is an integer.

## Version

Introduced in version August 2014
