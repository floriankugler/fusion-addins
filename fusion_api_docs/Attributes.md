# Attributes Object

Derived from: [Base](Base.md) Object  

## Description

Provides access to attributes associated with a specific entity, Product, or Document. Also supports the creation of new attributes.

## Methods

| Name | Description |
|----|----|
| [add](Attributes_add.md) | Adds a new attribute to the parent entity. If an attribute already exists on the entity with the same groupName and name already exists, this will update the existing attribute with the new value. |
| [classType](Attributes_classType.md) | Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType(). |
| [item](Attributes_item.md) | Returns the specified attribute using an index into the collection. |
| [itemByName](Attributes_itemByName.md) | Returns the specified attribute using the name of the attribute. |
| [itemsByGroup](Attributes_itemsByGroup.md) | Returns an array of all of the attributes that belong to the specified group. |

## Properties

| Name | Description |
| --- | --- |
| [count](Attributes_count.md) | Returns the number of attributes in the collection. |
| [groupNames](Attributes_groupNames.md) | Returns an array of strings that are all of the name of attribute groups that exist on this entity. An empty array can be returns if there are no attributes on the entity. |
| [isValid](Attributes_isValid.md) | Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference. |
| [objectType](Attributes_objectType.md) | This property is supported by all objects in the API and returns a string that contains the full name (namespace::objecttype) describing the type of the object. It's often useful to use this in combination with the classType method to see if an object is a certain type. For example: if obj.objectType == adsk.core.Point3D.classType(): |
