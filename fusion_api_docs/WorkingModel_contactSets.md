# WorkingModel.contactSets Property

Parent Object: [WorkingModel](WorkingModel.md)  

## Description

Returns the contact sets associated with this design.

## Syntax

"workingModel_var" is a variable referencing a WorkingModel object.  

```python
# Get the value of the property.
propertyValue = workingModel_var.contactSets
```

## Property Value

This is a read only property whose value is a [ContactSets](ContactSets.md).

## Version

Introduced in version January 2024
