# Sphere.surfaceType Property

Parent Object: [Sphere](Sphere.md)  

## Description

Returns the surface type.

## Syntax

"sphere_var" is a variable referencing a Sphere object.  

```python
# Get the value of the property.
propertyValue = sphere_var.surfaceType
```

## Property Value

This is a read only property whose value is a [SurfaceTypes](SurfaceTypes.md).

## Version

Introduced in version August 2014
