# GeneralPreferences.isDefaultMeasureShown Property

Parent Object: [GeneralPreferences](GeneralPreferences.md)  

## Description

Gets and sets if the default measure is shown.

## Syntax

"generalPreferences_var" is a variable referencing a GeneralPreferences object.  

```python
# Get the value of the property.
propertyValue = generalPreferences_var.isDefaultMeasureShown

# Set the value of the property.
generalPreferences_var.isDefaultMeasureShown = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version May 2022
