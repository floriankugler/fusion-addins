# SilhouetteSplitFeatures.isValid Property

Parent Object: [SilhouetteSplitFeatures](SilhouetteSplitFeatures.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"silhouetteSplitFeatures_var" is a variable referencing a SilhouetteSplitFeatures object.  

```python
# Get the value of the property.
propertyValue = silhouetteSplitFeatures_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version June 2015
