# Torus.origin Property

Parent Object: [Torus](Torus.md)  

## Description

Gets and sets the origin point (center) of the torus.

## Syntax

"torus_var" is a variable referencing a Torus object.  

```python
# Get the value of the property.
propertyValue = torus_var.origin

# Set the value of the property.
torus_var.origin = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
