# TangentRelationship.healthState Property

Parent Object: [TangentRelationship](TangentRelationship.md)  

## Description

Returns the current health state of the tangent relationship.

## Syntax

"tangentRelationship_var" is a variable referencing a TangentRelationship object.  

```python
# Get the value of the property.
propertyValue = tangentRelationship_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version May 2022
