# ReplaceFaceFeature.parentComponent Property

Parent Object: [ReplaceFaceFeature](ReplaceFaceFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"replaceFaceFeature_var" is a variable referencing a ReplaceFaceFeature object.  

```python
# Get the value of the property.
propertyValue = replaceFaceFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version March 2015
