# CameraEventHandler Object

Derived from: [EventHandler](EventHandler.md) Object  

## Description

The CameraEventHandler is a client implemented class that can be added as a handler to an CameraEvent.

## Methods

| Name | Description |
|----|----|
| [notify](CameraEventHandler_notify.md) | The function called by Fusion when the associated event is fired. |

## Version

Introduced in version December 2017
