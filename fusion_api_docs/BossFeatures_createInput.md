# BossFeatures.createInput Method

Parent Object: [BossFeatures](BossFeatures.md)  

## Description

Creates a new BossFeatureInput object that is used to specify the input needed to create a new boss feature(s).

## Syntax

"bossFeatures_var" is a variable referencing a [BossFeatures](BossFeatures.md) object.

```python
returnValue = bossFeatures_var.createInput()
```

## Return Value

| Type | Description |
|----|----|
| [BossFeatureInput](BossFeatureInput.md) | Returns the newly created BossFeatureInput object or null if the creation failed. |
