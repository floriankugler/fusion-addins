# ParallelConstraint.lineOne Property

Parent Object: [ParallelConstraint](ParallelConstraint.md)  

## Description

Returns the first line.

## Syntax

"parallelConstraint_var" is a variable referencing a ParallelConstraint object.  

```python
# Get the value of the property.
propertyValue = parallelConstraint_var.lineOne
```

## Property Value

This is a read only property whose value is a [SketchLine](SketchLine.md).

## Version

Introduced in version August 2014
