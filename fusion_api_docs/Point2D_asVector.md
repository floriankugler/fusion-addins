# Point2D.asVector Method

Parent Object: [Point2D](Point2D.md)  

## Description

Defines a vector using the coordinates of the point.

## Syntax

"point2D_var" is a variable referencing a [Point2D](Point2D.md) object.

```python
returnValue = point2D_var.asVector()
```

## Return Value

| Type                     | Description               |
|--------------------------|---------------------------|
| [Vector2D](Vector2D.md) | Returns a Vector2D object |

## Version

Introduced in version August 2014
