# ChildOperationList.isValid Property

Parent Object: [ChildOperationList](ChildOperationList.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"childOperationList_var" is a variable referencing a ChildOperationList object.  

```python
# Get the value of the property.
propertyValue = childOperationList_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2016
