# ButtonRowCommandInput.isVisible Property

Parent Object: [ButtonRowCommandInput](ButtonRowCommandInput.md)  

## Description

Gets or sets if this input will be visible to the user.  

Setting a SelectionCommandInput to be invisible will clear any selections it currently has.

## Syntax

"buttonRowCommandInput_var" is a variable referencing a ButtonRowCommandInput object.  

```python
# Get the value of the property.
propertyValue = buttonRowCommandInput_var.isVisible

# Set the value of the property.
buttonRowCommandInput_var.isVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version June 2015
