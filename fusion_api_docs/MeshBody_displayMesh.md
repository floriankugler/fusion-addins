# MeshBody.displayMesh Property

Parent Object: [MeshBody](MeshBody.md)  

## Description

Returns the associated mesh that is used for the display. This will always be triangles and includes any textures.

## Syntax

"meshBody_var" is a variable referencing a MeshBody object.  

```python
# Get the value of the property.
propertyValue = meshBody_var.displayMesh
```

## Property Value

This is a read only property whose value is a [TriangleMesh](TriangleMesh.md).

## Version

Introduced in version August 2014
