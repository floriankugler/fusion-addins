# DraftFeature.attributes Property

Parent Object: [DraftFeature](DraftFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"draftFeature_var" is a variable referencing a DraftFeature object.  

```python
# Get the value of the property.
propertyValue = draftFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
