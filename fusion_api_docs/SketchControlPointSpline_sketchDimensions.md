# SketchControlPointSpline.sketchDimensions Property

Parent Object: [SketchControlPointSpline](SketchControlPointSpline.md)  

## Description

Returns the sketch dimensions that are attached to this curve.

## Syntax

"sketchControlPointSpline_var" is a variable referencing a SketchControlPointSpline object.  

```python
# Get the value of the property.
propertyValue = sketchControlPointSpline_var.sketchDimensions
```

## Property Value

This is a read only property whose value is a [SketchDimensionList](SketchDimensionList.md).

## Version

Introduced in version July 2022
