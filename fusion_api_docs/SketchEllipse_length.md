# SketchEllipse.length Property

Parent Object: [SketchEllipse](SketchEllipse.md)  

## Description

Returns the length of the curve in centimeters.

## Syntax

"sketchEllipse_var" is a variable referencing a SketchEllipse object.  

```python
# Get the value of the property.
propertyValue = sketchEllipse_var.length
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version August 2014
