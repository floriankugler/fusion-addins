# SketchEllipse.geometry Property

Parent Object: [SketchEllipse](SketchEllipse.md)  

## Description

Returns the transient geometry of the ellipse which provides geometric information about the ellipse. The returned geometry is always in sketch space.

## Syntax

"sketchEllipse_var" is a variable referencing a SketchEllipse object.  

```python
# Get the value of the property.
propertyValue = sketchEllipse_var.geometry
```

## Property Value

This is a read only property whose value is an [Ellipse3D](Ellipse3D.md).

## Version

Introduced in version August 2014
