# CylinderFeature.isSuppressed Property

Parent Object: [CylinderFeature](CylinderFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"cylinderFeature_var" is a variable referencing a CylinderFeature object.  

```python
# Get the value of the property.
propertyValue = cylinderFeature_var.isSuppressed

# Set the value of the property.
cylinderFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version September 2015
