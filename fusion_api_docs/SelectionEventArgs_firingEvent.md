# SelectionEventArgs.firingEvent Property

Parent Object: [SelectionEventArgs](SelectionEventArgs.md)  

## Description

The event that the firing is in response to.

## Syntax

"selectionEventArgs_var" is a variable referencing a SelectionEventArgs object.  

```python
# Get the value of the property.
propertyValue = selectionEventArgs_var.firingEvent
```

## Property Value

This is a read only property whose value is an [Event](Event.md).

## Version

Introduced in version August 2014
