# NurbsSurface.copy Method

Parent Object: [NurbsSurface](NurbsSurface.md)  

## Description

Creates and returns an independent copy of this NurbsSurface object.

## Syntax

"nurbsSurface_var" is a variable referencing a [NurbsSurface](NurbsSurface.md) object.

```python
returnValue = nurbsSurface_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [NurbsSurface](NurbsSurface.md) | Returns a new NurbsSurface object that is a copy of this NurbsSurface object. |

## Version

Introduced in version August 2014
