# SketchEllipseMajorRadiusDimension.ellipse Property

Parent Object: [SketchEllipseMajorRadiusDimension](SketchEllipseMajorRadiusDimension.md)  

## Description

Returns the ellipse or elliptical arc being constrained.

## Syntax

"sketchEllipseMajorRadiusDimension_var" is a variable referencing a SketchEllipseMajorRadiusDimension object.  

```python
# Get the value of the property.
propertyValue = sketchEllipseMajorRadiusDimension_var.ellipse
```

## Property Value

This is a read only property whose value is a [SketchCurve](SketchCurve.md).

## Version

Introduced in version August 2014
