# RefoldFeature.attributes Property

Parent Object: [RefoldFeature](RefoldFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"refoldFeature_var" is a variable referencing a RefoldFeature object.  

```python
# Get the value of the property.
propertyValue = refoldFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version August 2020
