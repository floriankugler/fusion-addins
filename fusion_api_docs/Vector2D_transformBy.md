# Vector2D.transformBy Method

Parent Object: [Vector2D](Vector2D.md)  

## Description

Transforms the vector by specifying a 2D transformation matrix.

## Syntax

"vector2D_var" is a variable referencing a [Vector2D](Vector2D.md) object.

```python
returnValue = vector2D_var.transformBy(matrix)
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name | Type | Description |
|----|----|----|
| matrix | [Matrix2D](Matrix2D.md) | The Matrix2D object that defines the transformation. |

## Version

Introduced in version August 2014
