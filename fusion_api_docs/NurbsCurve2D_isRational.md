# NurbsCurve2D.isRational Property

Parent Object: [NurbsCurve2D](NurbsCurve2D.md)  

## Description

Returns if the curve is rational or non-rational type

## Syntax

"nurbsCurve2D_var" is a variable referencing a NurbsCurve2D object.  

```python
# Get the value of the property.
propertyValue = nurbsCurve2D_var.isRational
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
