# Setup.printSetting Property

Parent Object: [Setup](Setup.md)  

## Description

Gets and sets the PrintSetting associated with the setup.

## Syntax

"setup_var" is a variable referencing a Setup object.  

```python
# Get the value of the property.
propertyValue = setup_var.printSetting

# Set the value of the property.
setup_var.printSetting = propertyValue
```

## Property Value

This is a read/write property whose value is a [PrintSetting](PrintSetting.md).

## Version

Introduced in version April 2023
