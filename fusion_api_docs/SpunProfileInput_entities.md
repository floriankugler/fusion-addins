# SpunProfileInput.entities Property

Parent Object: [SpunProfileInput](SpunProfileInput.md)  

## Description

Gets and sets the entities (BRepBody or BRepFace) used to define the shape of the spun profile.

## Syntax

"spunProfileInput_var" is a variable referencing a SpunProfileInput object.  

```python
# Get the value of the property.
propertyValue = spunProfileInput_var.entities

# Set the value of the property.
spunProfileInput_var.entities = propertyValue
```

## Property Value

This is a read/write property whose value is an array of type [Base](Base.md).

## Version

Introduced in version November 2024
