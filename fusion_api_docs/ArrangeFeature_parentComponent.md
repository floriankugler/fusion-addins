# ArrangeFeature.parentComponent Property

Parent Object: [ArrangeFeature](ArrangeFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"arrangeFeature_var" is a variable referencing an ArrangeFeature object.  

```python
# Get the value of the property.
propertyValue = arrangeFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version January 2025
