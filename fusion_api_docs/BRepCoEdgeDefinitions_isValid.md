# BRepCoEdgeDefinitions.isValid Property

Parent Object: [BRepCoEdgeDefinitions](BRepCoEdgeDefinitions.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"bRepCoEdgeDefinitions_var" is a variable referencing a BRepCoEdgeDefinitions object.  

```python
# Get the value of the property.
propertyValue = bRepCoEdgeDefinitions_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2020
