# BrowserCommandInput.htmlFileURL Property

Parent Object: [BrowserCommandInput](BrowserCommandInput.md)  

## Description

Gets and sets the URL to the HTML file currently being displayed. This can be local or on the web.

## Syntax

"browserCommandInput_var" is a variable referencing a BrowserCommandInput object.  

```python
# Get the value of the property.
propertyValue = browserCommandInput_var.htmlFileURL

# Set the value of the property.
browserCommandInput_var.htmlFileURL = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version July 2021
