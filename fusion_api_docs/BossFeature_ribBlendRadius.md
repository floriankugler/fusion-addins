# BossFeature.ribBlendRadius Property

Parent Object: [BossFeature](BossFeature.md)  

## Description

Returns the model parameter controlling size of rib root blend radius.

## Syntax

"bossFeature_var" is a variable referencing a BossFeature object.  

```python
# Get the value of the property.
propertyValue = bossFeature_var.ribBlendRadius
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version October 2022
