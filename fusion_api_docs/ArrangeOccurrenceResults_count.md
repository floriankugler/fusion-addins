# ArrangeOccurrenceResults.count Property

Parent Object: [ArrangeOccurrenceResults](ArrangeOccurrenceResults.md)  

## Description

Returns the number of Arrange occurrences in the collection.

## Syntax

"arrangeOccurrenceResults_var" is a variable referencing an ArrangeOccurrenceResults object.  

```python
# Get the value of the property.
propertyValue = arrangeOccurrenceResults_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version January 2025
