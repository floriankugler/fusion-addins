# CombineFeature.parentComponent Property

Parent Object: [CombineFeature](CombineFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"combineFeature_var" is a variable referencing a CombineFeature object.  

```python
# Get the value of the property.
propertyValue = combineFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version November 2014
