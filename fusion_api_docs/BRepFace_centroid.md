# BRepFace.centroid Property

Parent Object: [BRepFace](BRepFace.md)  

## Description

Returns a point at the centroid (aka, geometric center) of the face.

## Syntax

"bRepFace_var" is a variable referencing a BRepFace object.  

```python
# Get the value of the property.
propertyValue = bRepFace_var.centroid
```

## Property Value

This is a read only property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version March 2016
