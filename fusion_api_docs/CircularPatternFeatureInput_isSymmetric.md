# CircularPatternFeatureInput.isSymmetric Property

Parent Object: [CircularPatternFeatureInput](CircularPatternFeatureInput.md)  

## Description

Gets and sets if the angle extent is in one direction or symmetric.

## Syntax

"circularPatternFeatureInput_var" is a variable referencing a CircularPatternFeatureInput object.  

```python
# Get the value of the property.
propertyValue = circularPatternFeatureInput_var.isSymmetric

# Set the value of the property.
circularPatternFeatureInput_var.isSymmetric = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.
