# BRepVertex.body Property

Parent Object: [BRepVertex](BRepVertex.md)  

## Description

Returns the parent body.

## Syntax

"bRepVertex_var" is a variable referencing a BRepVertex object.  

```python
# Get the value of the property.
propertyValue = bRepVertex_var.body
```

## Property Value

This is a read only property whose value is a [BRepBody](BRepBody.md).

## Version

Introduced in version August 2014
