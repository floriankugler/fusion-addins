# DrawingDocument.version Property

Parent Object: [DrawingDocument](DrawingDocument.md)  

## Description

Returns the Fusion version this document was last saved with.

## Syntax

"drawingDocument_var" is a variable referencing a DrawingDocument object.  

```python
# Get the value of the property.
propertyValue = drawingDocument_var.version
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version December 2020
