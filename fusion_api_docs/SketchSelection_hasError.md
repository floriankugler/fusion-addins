# SketchSelection.hasError Property

Parent Object: [SketchSelection](SketchSelection.md)  

## Description

Gets if errors were encountered when applying the selection to a a parent.

## Syntax

"sketchSelection_var" is a variable referencing a SketchSelection object.  

```python
# Get the value of the property.
propertyValue = sketchSelection_var.hasError
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2023
