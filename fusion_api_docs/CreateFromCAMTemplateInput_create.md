# CreateFromCAMTemplateInput.create Method

Parent Object: [CreateFromCAMTemplateInput](CreateFromCAMTemplateInput.md)  

## Description

Creates an empty input object to be passed into createFromCAMTemplate

## Syntax

This is a static method.  

```python

returnValue = adsk.cam.CreateFromCAMTemplateInput.create()
```

## Return Value

| Type | Description |
|----|----|
| [CreateFromCAMTemplateInput](CreateFromCAMTemplateInput.md) | The newly created input object. |

## Version

Introduced in version October 2023
