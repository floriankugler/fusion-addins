# DropDownCommandInput.deleteMe Method

Parent Object: [DropDownCommandInput](DropDownCommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"dropDownCommandInput_var" is a variable referencing a [DropDownCommandInput](DropDownCommandInput.md) object.

```python
returnValue = dropDownCommandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2020
