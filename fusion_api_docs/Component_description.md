# Component.description Property

Parent Object: [Component](Component.md)  

## Description

Gets and sets the description associated with this component.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.description

# Set the value of the property.
component_var.description = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version June 2015
