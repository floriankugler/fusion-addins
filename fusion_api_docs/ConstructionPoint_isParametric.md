# ConstructionPoint.isParametric Property

Parent Object: [ConstructionPoint](ConstructionPoint.md)  

## Description

Indicates if the construction point is parametric.

## Syntax

"constructionPoint_var" is a variable referencing a ConstructionPoint object.  

```python
# Get the value of the property.
propertyValue = constructionPoint_var.isParametric
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
