# PolygonMesh.normalVectors Property

Parent Object: [PolygonMesh](PolygonMesh.md)  

## Description

Returns the normal vectors as an array of Vector 3D objects. There is one normal vector for each index.

## Syntax

"polygonMesh_var" is a variable referencing a PolygonMesh object.  

```python
# Get the value of the property.
propertyValue = polygonMesh_var.normalVectors
```

## Property Value

This is a read only property whose value is an array of type [Vector3D](Vector3D.md).

## Version

Introduced in version August 2014
