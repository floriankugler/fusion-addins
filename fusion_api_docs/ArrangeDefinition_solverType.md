# ArrangeDefinition.solverType Property

Parent Object: [ArrangeDefinition](ArrangeDefinition.md)  

## Description

Gets the type of arrange feature defined by this definition.

## Syntax

"arrangeDefinition_var" is a variable referencing an ArrangeDefinition object.  

```python
# Get the value of the property.
propertyValue = arrangeDefinition_var.solverType
```

## Property Value

This is a read only property whose value is an [ArrangeSolverTypes](ArrangeSolverTypes.md).

## Version

Introduced in version January 2025
