# ConfigurationSheetMetalRuleCell.parentRow Property

Parent Object: [ConfigurationSheetMetalRuleCell](ConfigurationSheetMetalRuleCell.md)  

## Description

Returns the row this cell is in.

## Syntax

"configurationSheetMetalRuleCell_var" is a variable referencing a ConfigurationSheetMetalRuleCell object.  

```python
# Get the value of the property.
propertyValue = configurationSheetMetalRuleCell_var.parentRow
```

## Property Value

This is a read only property whose value is a [ConfigurationRow](ConfigurationRow.md).

## Version

Introduced in version January 2024
