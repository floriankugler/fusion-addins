# SliderCommandInput.unitType Property

Parent Object: [SliderCommandInput](SliderCommandInput.md)  

## Description

Gets and sets the unit type that is used when evaluating the user's input.

## Syntax

"sliderCommandInput_var" is a variable referencing a SliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = sliderCommandInput_var.unitType

# Set the value of the property.
sliderCommandInput_var.unitType = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version June 2015
