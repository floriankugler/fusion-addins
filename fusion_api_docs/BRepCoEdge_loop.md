# BRepCoEdge.loop Property

Parent Object: [BRepCoEdge](BRepCoEdge.md)  

## Description

Returns the loop this co-edge is part of.

## Syntax

"bRepCoEdge_var" is a variable referencing a BRepCoEdge object.  

```python
# Get the value of the property.
propertyValue = bRepCoEdge_var.loop
```

## Property Value

This is a read only property whose value is a [BRepLoop](BRepLoop.md).

## Version

Introduced in version August 2014
