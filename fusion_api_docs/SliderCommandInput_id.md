# SliderCommandInput.id Property

Parent Object: [SliderCommandInput](SliderCommandInput.md)  

## Description

Gets the unique identifier for this input in the command's CommandInputs.

## Syntax

"sliderCommandInput_var" is a variable referencing a SliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = sliderCommandInput_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version June 2015
