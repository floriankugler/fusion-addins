# ConfigurationSuppressColumn.id Property

Parent Object: [ConfigurationSuppressColumn](ConfigurationSuppressColumn.md)  

## Description

The id of the column. This is constant and cannot be changed by the user.

## Syntax

"configurationSuppressColumn_var" is a variable referencing a ConfigurationSuppressColumn object.  

```python
# Get the value of the property.
propertyValue = configurationSuppressColumn_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version January 2024
