# Preferences.materialPreferences Property

Parent Object: [Preferences](Preferences.md)  

## Description

Gets the MaterialPreferences object.

## Syntax

"preferences_var" is a variable referencing a Preferences object.  

```python
# Get the value of the property.
propertyValue = preferences_var.materialPreferences
```

## Property Value

This is a read only property whose value is a [MaterialPreferences](MaterialPreferences.md).

## Version

Introduced in version November 2014
