# PlasticRule.parentDesign Property

Parent Object: [PlasticRule](PlasticRule.md)  

## Description

Returns the parent design for a plastic rule in a design or it returns null if the plastic rule is in the library.

## Syntax

"plasticRule_var" is a variable referencing a PlasticRule object.  

```python
# Get the value of the property.
propertyValue = plasticRule_var.parentDesign
```

## Property Value

This is a read only property whose value is a [Design](Design.md).

## Version

Introduced in version January 2024
