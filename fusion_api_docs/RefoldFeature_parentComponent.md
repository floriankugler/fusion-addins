# RefoldFeature.parentComponent Property

Parent Object: [RefoldFeature](RefoldFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"refoldFeature_var" is a variable referencing a RefoldFeature object.  

```python
# Get the value of the property.
propertyValue = refoldFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version August 2020
