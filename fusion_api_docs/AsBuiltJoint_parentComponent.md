# AsBuiltJoint.parentComponent Property

Parent Object: [AsBuiltJoint](AsBuiltJoint.md)  

## Description

Returns the parent component that owns this AsBuiltJoint.

## Syntax

"asBuiltJoint_var" is a variable referencing an AsBuiltJoint object.  

```python
# Get the value of the property.
propertyValue = asBuiltJoint_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version September 2015
