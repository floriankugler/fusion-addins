# MachineAvoidDefaultSelection.error Property

Parent Object: [MachineAvoidDefaultSelection](MachineAvoidDefaultSelection.md)  

## Description

Gets the last warning string encountered after the selection was applied to a parent.

## Syntax

"machineAvoidDefaultSelection_var" is a variable referencing a MachineAvoidDefaultSelection object.  

```python
# Get the value of the property.
propertyValue = machineAvoidDefaultSelection_var.error
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version September 2024
