# HttpResponse.request Property

Parent Object: [HttpResponse](HttpResponse.md)  

## Description

Gets the request that generated this response.

## Syntax

"httpResponse_var" is a variable referencing a HttpResponse object.  

```python
# Get the value of the property.
propertyValue = httpResponse_var.request
```

## Property Value

This is a read only property whose value is a [HttpRequest](HttpRequest.md).

## Version

Introduced in version January 2024
