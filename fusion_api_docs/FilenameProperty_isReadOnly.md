# FilenameProperty.isReadOnly Property

Parent Object: [FilenameProperty](FilenameProperty.md)  

## Description

Indicates if this property is read-only. If True any attempted edits will fail.

## Syntax

"filenameProperty_var" is a variable referencing a FilenameProperty object.  

```python
# Get the value of the property.
propertyValue = filenameProperty_var.isReadOnly
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
