# Arc3D.curveType Property

Parent Object: [Arc3D](Arc3D.md)  

## Description

Returns the type of geometry this curve represents.

## Syntax

"arc3D_var" is a variable referencing an Arc3D object.  

```python
# Get the value of the property.
propertyValue = arc3D_var.curveType
```

## Property Value

This is a read only property whose value is a [Curve3DTypes](Curve3DTypes.md).

## Version

Introduced in version August 2014
