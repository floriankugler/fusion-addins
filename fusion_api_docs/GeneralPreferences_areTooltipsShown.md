# GeneralPreferences.areTooltipsShown Property

Parent Object: [GeneralPreferences](GeneralPreferences.md)  

## Description

Gets and sets if tooltips are shown.

## Syntax

"generalPreferences_var" is a variable referencing a GeneralPreferences object.  

```python
# Get the value of the property.
propertyValue = generalPreferences_var.areTooltipsShown

# Set the value of the property.
generalPreferences_var.areTooltipsShown = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
