# WorkingModel.attributes Property

Parent Object: [WorkingModel](WorkingModel.md)  

## Description

Returns the collection of attributes associated with this product.

## Syntax

"workingModel_var" is a variable referencing a WorkingModel object.  

```python
# Get the value of the property.
propertyValue = workingModel_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version January 2024
