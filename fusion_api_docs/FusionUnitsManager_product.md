# FusionUnitsManager.product Property

Parent Object: [FusionUnitsManager](FusionUnitsManager.md)  

## Description

Returns the parent Product.

## Syntax

"fusionUnitsManager_var" is a variable referencing a FusionUnitsManager object.  

```python
# Get the value of the property.
propertyValue = fusionUnitsManager_var.product
```

## Property Value

This is a read only property whose value is a [Product](Product.md).

## Version

Introduced in version August 2014
