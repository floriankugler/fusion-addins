# NavigationEventHandler Object

Derived from: [EventHandler](EventHandler.md) Object  

## Description

The NavigationEventHandler is a client implemented class that can be added as a handler to a Navigation event which is triggered by the browser within a palette.

## Methods

| Name | Description |
|----|----|
| [notify](NavigationEventHandler_notify.md) | The function called by Fusion when the associated event is fired. |

## Version

Introduced in version March 2021
