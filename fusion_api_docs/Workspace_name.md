# Workspace.name Property

Parent Object: [Workspace](Workspace.md)  

## Description

Gets the visible name of the workspace as seen in the user interface. This is the localized name.

## Syntax

"workspace_var" is a variable referencing a Workspace object.  

```python
# Get the value of the property.
propertyValue = workspace_var.name
```

## Property Value

This is a read only property whose value is a string.
