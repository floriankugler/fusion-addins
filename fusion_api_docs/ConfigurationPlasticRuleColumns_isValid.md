# ConfigurationPlasticRuleColumns.isValid Property

Parent Object: [ConfigurationPlasticRuleColumns](ConfigurationPlasticRuleColumns.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"configurationPlasticRuleColumns_var" is a variable referencing a ConfigurationPlasticRuleColumns object.  

```python
# Get the value of the property.
propertyValue = configurationPlasticRuleColumns_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2024
