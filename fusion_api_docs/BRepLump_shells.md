# BRepLump.shells Property

Parent Object: [BRepLump](BRepLump.md)  

## Description

Returns the BRepShells owned by the lump

## Syntax

"bRepLump_var" is a variable referencing a BRepLump object.  

```python
# Get the value of the property.
propertyValue = bRepLump_var.shells
```

## Property Value

This is a read only property whose value is a [BRepShells](BRepShells.md).

## Version

Introduced in version August 2014
