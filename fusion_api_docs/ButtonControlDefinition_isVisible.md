# ButtonControlDefinition.isVisible Property

Parent Object: [ButtonControlDefinition](ButtonControlDefinition.md)  

## Description

Gets or sets if this definition is visible or not. This has the effect of making any associated controls visible or invisible in the user interface.

## Syntax

"buttonControlDefinition_var" is a variable referencing a ButtonControlDefinition object.  

```python
# Get the value of the property.
propertyValue = buttonControlDefinition_var.isVisible

# Set the value of the property.
buttonControlDefinition_var.isVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.
