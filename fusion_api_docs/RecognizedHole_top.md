# RecognizedHole.top Property

Parent Object: [RecognizedHole](RecognizedHole.md)  

## Description

Returns a point at the center of the hole top.

## Syntax

"recognizedHole_var" is a variable referencing a RecognizedHole object.  

```python
# Get the value of the property.
propertyValue = recognizedHole_var.top
```

## Property Value

This is a read only property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version May 2023
