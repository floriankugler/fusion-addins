# BaseFeature.isSuppressed Property

Parent Object: [BaseFeature](BaseFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"baseFeature_var" is a variable referencing a BaseFeature object.  

```python
# Get the value of the property.
propertyValue = baseFeature_var.isSuppressed

# Set the value of the property.
baseFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version September 2015
