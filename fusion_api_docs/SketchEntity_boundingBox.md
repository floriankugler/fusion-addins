# SketchEntity.boundingBox Property

Parent Object: [SketchEntity](SketchEntity.md)  

## Description

Returns the bounding box of the entity in sketch space.

## Syntax

"sketchEntity_var" is a variable referencing a SketchEntity object.  

```python
# Get the value of the property.
propertyValue = sketchEntity_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version August 2014
