# SketchDiameterDimension.deleteMe Method

Parent Object: [SketchDiameterDimension](SketchDiameterDimension.md)  

## Description

Deletes this dimension. The IsDeletable property indicates if this dimension can be deleted.

## Syntax

"sketchDiameterDimension_var" is a variable referencing a [SketchDiameterDimension](SketchDiameterDimension.md) object.

```python
returnValue = sketchDiameterDimension_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
