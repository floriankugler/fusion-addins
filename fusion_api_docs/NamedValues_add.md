# NamedValues.add Method

Parent Object: [NamedValues](NamedValues.md)  

## Description

Adds a name value pair to the NamedValues object

## Syntax

"namedValues_var" is a variable referencing a [NamedValues](NamedValues.md) object.

```python
returnValue = namedValues_var.add(name, value)
```

## Return Value

| Type    | Description                                                |
|---------|------------------------------------------------------------|
| boolean | Returns true if the name value pair is added successfully. |

## Parameters

| Name | Type | Description |
|----|----|----|
| name | string | A name for the name value pair |
| value | [ValueInput](ValueInput.md) | A ValueInput object that defines the value of the name value pair |

## Version

Introduced in version August 2014
