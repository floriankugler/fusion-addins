# MouseEventArgs.wheelDelta Property

Parent Object: [MouseEventArgs](MouseEventArgs.md)  

## Description

Gets a signed count of the number of detents the mouse wheel has rotated.

## Syntax

"mouseEventArgs_var" is a variable referencing a MouseEventArgs object.  

```python
# Get the value of the property.
propertyValue = mouseEventArgs_var.wheelDelta
```

## Property Value

This is a read only property whose value is an integer.

## Version

Introduced in version August 2014
