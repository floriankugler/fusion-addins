# WebFeatures.count Property

Parent Object: [WebFeatures](WebFeatures.md)  

## Description

The number of web features in the collection.

## Syntax

"webFeatures_var" is a variable referencing a WebFeatures object.  

```python
# Get the value of the property.
propertyValue = webFeatures_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version September 2015
