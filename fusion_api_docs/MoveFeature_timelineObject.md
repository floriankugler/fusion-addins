# MoveFeature.timelineObject Property

Parent Object: [MoveFeature](MoveFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"moveFeature_var" is a variable referencing a MoveFeature object.  

```python
# Get the value of the property.
propertyValue = moveFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version March 2015
