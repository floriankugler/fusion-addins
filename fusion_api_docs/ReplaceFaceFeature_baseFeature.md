# ReplaceFaceFeature.baseFeature Property

Parent Object: [ReplaceFaceFeature](ReplaceFaceFeature.md)  

## Description

If this feature is associated with a base feature, this property will return that base feature. If it's not associated with a base feature, this property will return null.

## Syntax

"replaceFaceFeature_var" is a variable referencing a ReplaceFaceFeature object.  

```python
# Get the value of the property.
propertyValue = replaceFaceFeature_var.baseFeature
```

## Property Value

This is a read only property whose value is a [BaseFeature](BaseFeature.md).

## Version

Introduced in version May 2016
