# StitchFeature.parentComponent Property

Parent Object: [StitchFeature](StitchFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"stitchFeature_var" is a variable referencing a StitchFeature object.  

```python
# Get the value of the property.
propertyValue = stitchFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version June 2015
