# LoftEndCondition.parentLoftSection Property

Parent Object: [LoftEndCondition](LoftEndCondition.md)  

## Description

Returns the parent loft section.

## Syntax

"loftEndCondition_var" is a variable referencing a LoftEndCondition object.  

```python
# Get the value of the property.
propertyValue = loftEndCondition_var.parentLoftSection
```

## Property Value

This is a read only property whose value is a [LoftSection](LoftSection.md).

## Version

Introduced in version August 2016
