# RemoveFeatures.count Property

Parent Object: [RemoveFeatures](RemoveFeatures.md)  

## Description

The number of Remove features in the collection.

## Syntax

"removeFeatures_var" is a variable referencing a RemoveFeatures object.  

```python
# Get the value of the property.
propertyValue = removeFeatures_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version September 2015
