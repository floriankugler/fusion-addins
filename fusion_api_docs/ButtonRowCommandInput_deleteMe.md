# ButtonRowCommandInput.deleteMe Method

Parent Object: [ButtonRowCommandInput](ButtonRowCommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"buttonRowCommandInput_var" is a variable referencing a [ButtonRowCommandInput](ButtonRowCommandInput.md) object.

```python
returnValue = buttonRowCommandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2020
