# CAMPattern.duplicate Method

Parent Object: [CAMPattern](CAMPattern.md)  

## Description

Creates a duplicate of the operation in the tree after the itself.

## Syntax

"cAMPattern_var" is a variable referencing a [CAMPattern](CAMPattern.md) object.

```python
returnValue = cAMPattern_var.duplicate()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns if duplicate command was successful. |

## Version

Introduced in version May 2024
