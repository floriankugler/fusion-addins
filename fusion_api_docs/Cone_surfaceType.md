# Cone.surfaceType Property

Parent Object: [Cone](Cone.md)  

## Description

Returns the surface type.

## Syntax

"cone_var" is a variable referencing a Cone object.  

```python
# Get the value of the property.
propertyValue = cone_var.surfaceType
```

## Property Value

This is a read only property whose value is a [SurfaceTypes](SurfaceTypes.md).

## Version

Introduced in version August 2014
