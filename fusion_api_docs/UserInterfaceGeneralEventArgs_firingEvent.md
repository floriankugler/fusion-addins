# UserInterfaceGeneralEventArgs.firingEvent Property

Parent Object: [UserInterfaceGeneralEventArgs](UserInterfaceGeneralEventArgs.md)  

## Description

The event that the firing is in response to.

## Syntax

"userInterfaceGeneralEventArgs_var" is a variable referencing a UserInterfaceGeneralEventArgs object.  

```python
# Get the value of the property.
propertyValue = userInterfaceGeneralEventArgs_var.firingEvent
```

## Property Value

This is a read only property whose value is an [Event](Event.md).

## Version

Introduced in version March 2017
