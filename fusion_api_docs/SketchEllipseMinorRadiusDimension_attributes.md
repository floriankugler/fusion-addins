# SketchEllipseMinorRadiusDimension.attributes Property

Parent Object: [SketchEllipseMinorRadiusDimension](SketchEllipseMinorRadiusDimension.md)  

## Description

Returns the collection of attributes associated with this sketch dimension.

## Syntax

"sketchEllipseMinorRadiusDimension_var" is a variable referencing a SketchEllipseMinorRadiusDimension object.  

```python
# Get the value of the property.
propertyValue = sketchEllipseMinorRadiusDimension_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
