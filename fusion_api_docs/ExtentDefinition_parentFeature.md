# ExtentDefinition.parentFeature Property

Parent Object: [ExtentDefinition](ExtentDefinition.md)  

## Description

Returns the parent feature that this definition is associated with. If this definition has been created statically and is not associated with a feature this property will return null.

## Syntax

"extentDefinition_var" is a variable referencing an ExtentDefinition object.  

```python
# Get the value of the property.
propertyValue = extentDefinition_var.parentFeature
```

## Property Value

This is a read only property whose value is a [Feature](Feature.md).

## Version

Introduced in version August 2014
