# DataFile.hasChildReferences Property

Parent Object: [DataFile](DataFile.md)  

## Description

Gets if this datafile has children, (i.e. a Fusion Design containing referenced components).

## Syntax

"dataFile_var" is a variable referencing a DataFile object.  

```python
# Get the value of the property.
propertyValue = dataFile_var.hasChildReferences
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2016
