# PropertyGroup.name Property

Parent Object: [PropertyGroup](PropertyGroup.md)  

## Description

Returns the name of this group as seen in the user interface. This name is localized and can change based on the current language

## Syntax

"propertyGroup_var" is a variable referencing a PropertyGroup object.  

```python
# Get the value of the property.
propertyValue = propertyGroup_var.name

# Set the value of the property.
propertyGroup_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version January 2024
