# SketchFixedSpline.length Property

Parent Object: [SketchFixedSpline](SketchFixedSpline.md)  

## Description

Returns the length of the curve in centimeters.

## Syntax

"sketchFixedSpline_var" is a variable referencing a SketchFixedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFixedSpline_var.length
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version August 2014
