# VerticalPointsConstraint.isDeletable Property

Parent Object: [VerticalPointsConstraint](VerticalPointsConstraint.md)  

## Description

Indicates if this constraint is deletable.

## Syntax

"verticalPointsConstraint_var" is a variable referencing a VerticalPointsConstraint object.  

```python
# Get the value of the property.
propertyValue = verticalPointsConstraint_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
