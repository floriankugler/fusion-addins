# HorizontalConstraint.parentSketch Property

Parent Object: [HorizontalConstraint](HorizontalConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"horizontalConstraint_var" is a variable referencing a HorizontalConstraint object.  

```python
# Get the value of the property.
propertyValue = horizontalConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
