# OptimizedOrientationResult.supportVolume Property

Parent Object: [OptimizedOrientationResult](OptimizedOrientationResult.md)  

## Description

The volume of the support hull below the oriented part. The value is given in cubic centimeters

## Syntax

"optimizedOrientationResult_var" is a variable referencing an OptimizedOrientationResult object.  

```python
# Get the value of the property.
propertyValue = optimizedOrientationResult_var.supportVolume
```

## Property Value

This is a read only property whose value is a float.

## Version

Introduced in version July 2023
