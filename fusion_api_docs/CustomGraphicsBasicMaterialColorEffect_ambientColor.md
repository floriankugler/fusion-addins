# CustomGraphicsBasicMaterialColorEffect.ambientColor Property

Parent Object: [CustomGraphicsBasicMaterialColorEffect](CustomGraphicsBasicMaterialColorEffect.md)  

## Description

Gets and sets the ambientColor associated with this CustomGraphicsBasicMaterialColorEffect object. The ambient color is the color of the light anywhere there's not a specific light source.

## Syntax

"customGraphicsBasicMaterialColorEffect_var" is a variable referencing a CustomGraphicsBasicMaterialColorEffect object.  

```python
# Get the value of the property.
propertyValue = customGraphicsBasicMaterialColorEffect_var.ambientColor

# Set the value of the property.
customGraphicsBasicMaterialColorEffect_var.ambientColor = propertyValue
```

## Property Value

This is a read/write property whose value is a [Color](Color.md).

## Version

Introduced in version September 2017
