# ConfigurationSuppressColumn.parentTable Property

Parent Object: [ConfigurationSuppressColumn](ConfigurationSuppressColumn.md)  

## Description

Returns the parent table, either top or custom theme table, this column is in.

## Syntax

"configurationSuppressColumn_var" is a variable referencing a ConfigurationSuppressColumn object.  

```python
# Get the value of the property.
propertyValue = configurationSuppressColumn_var.parentTable
```

## Property Value

This is a read only property whose value is a [ConfigurationTable](ConfigurationTable.md).

## Version

Introduced in version January 2024
