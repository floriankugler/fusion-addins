# IntegerProperty.hasMultipleValues Property

Parent Object: [IntegerProperty](IntegerProperty.md)  

## Description

Gets the boolean flag that indicates if this property has multiple values or not.  

This is most commonly used for properties associated with materials and appearances.

## Syntax

"integerProperty_var" is a variable referencing an IntegerProperty object.  

```python
# Get the value of the property.
propertyValue = integerProperty_var.hasMultipleValues
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
