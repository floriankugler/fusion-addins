# CAMPattern.isProtected Property

Parent Object: [CAMPattern](CAMPattern.md)  

## Description

Gets and sets the "protected" property value of the operation. Gets/sets true if the operation is protected.

## Syntax

"cAMPattern_var" is a variable referencing a CAMPattern object.  

```python
# Get the value of the property.
propertyValue = cAMPattern_var.isProtected

# Set the value of the property.
cAMPattern_var.isProtected = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2016
