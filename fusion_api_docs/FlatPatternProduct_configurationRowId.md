# FlatPatternProduct.configurationRowId Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Returns the ID of the row that defines this configuration. Use the isConfiguration property to determine if this Design is a configuration or not. If this is not a configuration, this property returns an empty string.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.configurationRowId
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version January 2024
