# PostProcessingMachineElement.typeId Property

Parent Object: [PostProcessingMachineElement](PostProcessingMachineElement.md)  

## Description

Identifier for this type of machine element.

## Syntax

"postProcessingMachineElement_var" is a variable referencing a PostProcessingMachineElement object.  

```python
# Get the value of the property.
propertyValue = postProcessingMachineElement_var.typeId
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version July 2025
