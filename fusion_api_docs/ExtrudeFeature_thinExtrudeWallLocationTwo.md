# ExtrudeFeature.thinExtrudeWallLocationTwo Property

Parent Object: [ExtrudeFeature](ExtrudeFeature.md)  

## Description

Gets and sets the wall location for side two of a two sided thin extrude

## Syntax

"extrudeFeature_var" is a variable referencing an ExtrudeFeature object.  

```python
# Get the value of the property.
propertyValue = extrudeFeature_var.thinExtrudeWallLocationTwo

# Set the value of the property.
extrudeFeature_var.thinExtrudeWallLocationTwo = propertyValue
```

## Property Value

This is a read/write property whose value is a [ThinExtrudeWallLocation](ThinExtrudeWallLocation.md).

## Version

Introduced in version April 2021
