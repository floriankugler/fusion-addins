# SketchRadialDimension.attributes Property

Parent Object: [SketchRadialDimension](SketchRadialDimension.md)  

## Description

Returns the collection of attributes associated with this sketch dimension.

## Syntax

"sketchRadialDimension_var" is a variable referencing a SketchRadialDimension object.  

```python
# Get the value of the property.
propertyValue = sketchRadialDimension_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
