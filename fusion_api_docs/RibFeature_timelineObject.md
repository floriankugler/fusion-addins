# RibFeature.timelineObject Property

Parent Object: [RibFeature](RibFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"ribFeature_var" is a variable referencing a RibFeature object.  

```python
# Get the value of the property.
propertyValue = ribFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version September 2015
