# ConfigurationAppearanceTable.parentTableColumn Property

Parent Object: [ConfigurationAppearanceTable](ConfigurationAppearanceTable.md)  

## Description

Returns the column in the top table that references this appearance table.

## Syntax

"configurationAppearanceTable_var" is a variable referencing a ConfigurationAppearanceTable object.  

```python
# Get the value of the property.
propertyValue = configurationAppearanceTable_var.parentTableColumn
```

## Property Value

This is a read only property whose value is a [ConfigurationThemeColumn](ConfigurationThemeColumn.md).

## Version

Introduced in version March 2024
