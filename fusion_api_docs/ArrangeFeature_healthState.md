# ArrangeFeature.healthState Property

Parent Object: [ArrangeFeature](ArrangeFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"arrangeFeature_var" is a variable referencing an ArrangeFeature object.  

```python
# Get the value of the property.
propertyValue = arrangeFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version January 2025
