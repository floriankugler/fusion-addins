# Document.allDocumentReferences Property

Parent Object: [Document](Document.md)  

## Description

Returns a collection containing all of the documents referenced directly by this document and those referenced by all sub-assemblies.

## Syntax

"document_var" is a variable referencing a Document object.  

```python
# Get the value of the property.
propertyValue = document_var.allDocumentReferences
```

## Property Value

This is a read only property whose value is a [DocumentReferences](DocumentReferences.md).

## Version

Introduced in version June 2017
