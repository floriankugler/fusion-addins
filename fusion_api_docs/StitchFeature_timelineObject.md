# StitchFeature.timelineObject Property

Parent Object: [StitchFeature](StitchFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"stitchFeature_var" is a variable referencing a StitchFeature object.  

```python
# Get the value of the property.
propertyValue = stitchFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version June 2015
