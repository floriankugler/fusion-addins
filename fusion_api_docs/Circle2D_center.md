# Circle2D.center Property

Parent Object: [Circle2D](Circle2D.md)  

## Description

Gets and sets the center position of the circle.

## Syntax

"circle2D_var" is a variable referencing a Circle2D object.  

```python
# Get the value of the property.
propertyValue = circle2D_var.center

# Set the value of the property.
circle2D_var.center = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point2D](Point2D.md).

## Version

Introduced in version August 2014
