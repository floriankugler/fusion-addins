# ConstructionPoint.isVisible Property

Parent Object: [ConstructionPoint](ConstructionPoint.md)  

## Description

Gets if the construction point is visible. This property is affected by the AssemblyContext of the construction point.

## Syntax

"constructionPoint_var" is a variable referencing a ConstructionPoint object.  

```python
# Get the value of the property.
propertyValue = constructionPoint_var.isVisible
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
