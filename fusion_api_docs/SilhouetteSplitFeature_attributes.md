# SilhouetteSplitFeature.attributes Property

Parent Object: [SilhouetteSplitFeature](SilhouetteSplitFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"silhouetteSplitFeature_var" is a variable referencing a SilhouetteSplitFeature object.  

```python
# Get the value of the property.
propertyValue = silhouetteSplitFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
