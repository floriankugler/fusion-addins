# ConfigurationReplaceDesign.deleteMe Method

Parent Object: [ConfigurationReplaceDesign](ConfigurationReplaceDesign.md)  

## Description

Deletes this ConfigurationReplaceDesign.

## Syntax

"configurationReplaceDesign_var" is a variable referencing a [ConfigurationReplaceDesign](ConfigurationReplaceDesign.md) object.

```python
returnValue = configurationReplaceDesign_var.deleteMe()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns true if the deletion was successful. |

## Version

Introduced in version September 2025
