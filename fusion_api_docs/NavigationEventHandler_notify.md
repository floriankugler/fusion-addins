# NavigationEventHandler.notify Method

Parent Object: [NavigationEventHandler](NavigationEventHandler.md)  

## Description

The function called by Fusion when the associated event is fired.

## Syntax

"navigationEventHandler_var" is a variable referencing a [NavigationEventHandler](NavigationEventHandler.md) object.

```python
returnValue = navigationEventHandler_var.notify(eventArgs)
```

## Parameters

| Name | Type | Description |
|----|----|----|
| eventArgs | [NavigationEventArgs](NavigationEventArgs.md) | Returns an object that provides access to additional information associated with the event. |

## Version

Introduced in version March 2021
