# Setup.folders Property

Parent Object: [Setup](Setup.md)  

## Description

Returns the Folders collection that provides access to existing folders in this setup.

## Syntax

"setup_var" is a variable referencing a Setup object.  

```python
# Get the value of the property.
propertyValue = setup_var.folders
```

## Property Value

This is a read only property whose value is a [CAMFolders](CAMFolders.md).

## Version

Introduced in version January 2016
