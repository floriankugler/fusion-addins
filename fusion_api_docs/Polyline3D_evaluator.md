# Polyline3D.evaluator Property

Parent Object: [Polyline3D](Polyline3D.md)  

## Description

Returns an evaluator object that lets you perform additional evaluations on the curve.

## Syntax

"polyline3D_var" is a variable referencing a Polyline3D object.  

```python
# Get the value of the property.
propertyValue = polyline3D_var.evaluator
```

## Property Value

This is a read only property whose value is a [CurveEvaluator3D](CurveEvaluator3D.md).

## Version

Introduced in version September 2024
