# Vector2D.asPoint Method

Parent Object: [Vector2D](Vector2D.md)  

## Description

Return a point with the same x and y values as this vector.

## Syntax

"vector2D_var" is a variable referencing a [Vector2D](Vector2D.md) object.

```python
returnValue = vector2D_var.asPoint()
```

## Return Value

| Type                   | Description            |
|------------------------|------------------------|
| [Point2D](Point2D.md) | Returns the new point. |

## Version

Introduced in version August 2014
