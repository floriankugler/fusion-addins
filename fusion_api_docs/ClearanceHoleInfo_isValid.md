# ClearanceHoleInfo.isValid Property

Parent Object: [ClearanceHoleInfo](ClearanceHoleInfo.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"clearanceHoleInfo_var" is a variable referencing a ClearanceHoleInfo object.  

```python
# Get the value of the property.
propertyValue = clearanceHoleInfo_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2025
