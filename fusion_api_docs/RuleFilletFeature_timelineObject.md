# RuleFilletFeature.timelineObject Property

Parent Object: [RuleFilletFeature](RuleFilletFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"ruleFilletFeature_var" is a variable referencing a RuleFilletFeature object.  

```python
# Get the value of the property.
propertyValue = ruleFilletFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version September 2015
