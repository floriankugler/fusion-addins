# ProfileCurve.parentProfile Property

Parent Object: [ProfileCurve](ProfileCurve.md)  

## Description

Returns the parent Profile object.

## Syntax

"profileCurve_var" is a variable referencing a ProfileCurve object.  

```python
# Get the value of the property.
propertyValue = profileCurve_var.parentProfile
```

## Property Value

This is a read only property whose value is a [Profile](Profile.md).

## Version

Introduced in version August 2014
