# SketchPoint.attributes Property

Parent Object: [SketchPoint](SketchPoint.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"sketchPoint_var" is a variable referencing a SketchPoint object.  

```python
# Get the value of the property.
propertyValue = sketchPoint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
