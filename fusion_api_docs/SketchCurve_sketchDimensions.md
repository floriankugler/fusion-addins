# SketchCurve.sketchDimensions Property

Parent Object: [SketchCurve](SketchCurve.md)  

## Description

Returns the sketch dimensions that are attached to this curve.

## Syntax

"sketchCurve_var" is a variable referencing a SketchCurve object.  

```python
# Get the value of the property.
propertyValue = sketchCurve_var.sketchDimensions
```

## Property Value

This is a read only property whose value is a [SketchDimensionList](SketchDimensionList.md).

## Version

Introduced in version August 2014
