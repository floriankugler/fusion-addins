# BRepFace.boundingBox Property

Parent Object: [BRepFace](BRepFace.md)  

## Description

Returns the bounding box of this face

## Syntax

"bRepFace_var" is a variable referencing a BRepFace object.  

```python
# Get the value of the property.
propertyValue = bRepFace_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version March 2016
