# MachineAxis.axisType Property

Parent Object: [MachineAxis](MachineAxis.md)  

## Description

The type of axis.

## Syntax

"machineAxis_var" is a variable referencing a MachineAxis object.  

```python
# Get the value of the property.
propertyValue = machineAxis_var.axisType
```

## Property Value

This is a read only property whose value is a [MachineAxisTypes](MachineAxisTypes.md).

## Version

Introduced in version April 2023
