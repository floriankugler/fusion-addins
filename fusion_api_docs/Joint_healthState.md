# Joint.healthState Property

Parent Object: [Joint](Joint.md)  

## Description

Returns the current health state of the joint.

## Syntax

"joint_var" is a variable referencing a Joint object.  

```python
# Get the value of the property.
propertyValue = joint_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).
