# WebFeature.attributes Property

Parent Object: [WebFeature](WebFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"webFeature_var" is a variable referencing a WebFeature object.  

```python
# Get the value of the property.
propertyValue = webFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
