# Occurrence.timelineObject Property

Parent Object: [Occurrence](Occurrence.md)  

## Description

Returns the timeline object associated with the creation of this occurrence.

## Syntax

"occurrence_var" is a variable referencing an Occurrence object.  

```python
# Get the value of the property.
propertyValue = occurrence_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version August 2014
