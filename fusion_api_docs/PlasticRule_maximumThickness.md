# PlasticRule.maximumThickness Property

Parent Object: [PlasticRule](PlasticRule.md)  

## Description

The maximum thickness of the part in centimeters.Â This is used by the Design Advice command when analyzing the part for manufacturability.

## Syntax

"plasticRule_var" is a variable referencing a PlasticRule object.  

```python
# Get the value of the property.
propertyValue = plasticRule_var.maximumThickness

# Set the value of the property.
plasticRule_var.maximumThickness = propertyValue
```

## Property Value

This is a read/write property whose value is a double.
