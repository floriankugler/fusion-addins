# LineOnPlanarSurfaceConstraint.attributes Property

Parent Object: [LineOnPlanarSurfaceConstraint](LineOnPlanarSurfaceConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"lineOnPlanarSurfaceConstraint_var" is a variable referencing a LineOnPlanarSurfaceConstraint object.  

```python
# Get the value of the property.
propertyValue = lineOnPlanarSurfaceConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version September 2023
