# SketchEllipse.parentSketch Property

Parent Object: [SketchEllipse](SketchEllipse.md)  

## Description

Returns the parent sketch.

## Syntax

"sketchEllipse_var" is a variable referencing a SketchEllipse object.  

```python
# Get the value of the property.
propertyValue = sketchEllipse_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
