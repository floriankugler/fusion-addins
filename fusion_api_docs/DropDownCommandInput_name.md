# DropDownCommandInput.name Property

Parent Object: [DropDownCommandInput](DropDownCommandInput.md)  

## Description

Gets the user visible name of this input.

## Syntax

"dropDownCommandInput_var" is a variable referencing a DropDownCommandInput object.  

```python
# Get the value of the property.
propertyValue = dropDownCommandInput_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version January 2015
