# Sketch.areConstraintsShown Property

Parent Object: [Sketch](Sketch.md)  

## Description

Indicates if the constraints of the sketch are displayed when the sketch is active.

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.areConstraintsShown

# Set the value of the property.
sketch_var.areConstraintsShown = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version April 2019
