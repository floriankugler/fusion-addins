# SMTImportOptions.filename Property

Parent Object: [SMTImportOptions](SMTImportOptions.md)  

## Description

Gets and sets the filename or URL of the file to be imported.

## Syntax

"sMTImportOptions_var" is a variable referencing a SMTImportOptions object.  

```python
# Get the value of the property.
propertyValue = sMTImportOptions_var.filename

# Set the value of the property.
sMTImportOptions_var.filename = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version September 2015
