# Parameter.valueType Property

Parent Object: [Parameter](Parameter.md)  

## Description

Returns the type of value this parameter is. For a numeric parameter, you can get the value using the value property. For a text parameter, you can get the value using the textValue property.

## Syntax

"parameter_var" is a variable referencing a Parameter object.  

```python
# Get the value of the property.
propertyValue = parameter_var.valueType
```

## Property Value

This is a read only property whose value is a [ParameterValueTypes](ParameterValueTypes.md).

## Version

Introduced in version September 2025
