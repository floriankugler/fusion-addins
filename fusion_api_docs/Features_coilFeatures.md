# Features.coilFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the Coil Primitive features within the component.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.coilFeatures
```

## Property Value

This is a read only property whose value is a [CoilFeatures](CoilFeatures.md).

## Version

Introduced in version September 2015
