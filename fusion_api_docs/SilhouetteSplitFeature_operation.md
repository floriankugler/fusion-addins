# SilhouetteSplitFeature.operation Property

Parent Object: [SilhouetteSplitFeature](SilhouetteSplitFeature.md)  

## Description

Gets the type of silhouette split operation.

## Syntax

"silhouetteSplitFeature_var" is a variable referencing a SilhouetteSplitFeature object.  

```python
# Get the value of the property.
propertyValue = silhouetteSplitFeature_var.operation
```

## Property Value

This is a read only property whose value is a [SilhouetteSplitOperations](SilhouetteSplitOperations.md).

## Version

Introduced in version June 2015
