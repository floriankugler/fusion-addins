# Torus.axis Property

Parent Object: [Torus](Torus.md)  

## Description

Gets and sets the center axis of the torus.

## Syntax

"torus_var" is a variable referencing a Torus object.  

```python
# Get the value of the property.
propertyValue = torus_var.axis

# Set the value of the property.
torus_var.axis = propertyValue
```

## Property Value

This is a read/write property whose value is a [Vector3D](Vector3D.md).

## Version

Introduced in version August 2014
