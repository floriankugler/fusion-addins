# PostProcessingMachineElement.isValid Property

Parent Object: [PostProcessingMachineElement](PostProcessingMachineElement.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"postProcessingMachineElement_var" is a variable referencing a PostProcessingMachineElement object.  

```python
# Get the value of the property.
propertyValue = postProcessingMachineElement_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2025
