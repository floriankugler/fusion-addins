# SketchPoints.count Property

Parent Object: [SketchPoints](SketchPoints.md)  

## Description

Returns the number of sketch points in the sketch.

## Syntax

"sketchPoints_var" is a variable referencing a SketchPoints object.  

```python
# Get the value of the property.
propertyValue = sketchPoints_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version August 2014
