# AsBuiltJoint.attributes Property

Parent Object: [AsBuiltJoint](AsBuiltJoint.md)  

## Description

Returns the collection of attributes associated with this as-built joint.

## Syntax

"asBuiltJoint_var" is a variable referencing an AsBuiltJoint object.  

```python
# Get the value of the property.
propertyValue = asBuiltJoint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
