# FlatPattern.foldedBody Property

Parent Object: [FlatPattern](FlatPattern.md)  

## Description

Returns the folded B-Rep body in the design that this flat pattern was created from.

## Syntax

"flatPattern_var" is a variable referencing a FlatPattern object.  

```python
# Get the value of the property.
propertyValue = flatPattern_var.foldedBody
```

## Property Value

This is a read only property whose value is a [BRepBody](BRepBody.md).

## Version

Introduced in version October 2022
