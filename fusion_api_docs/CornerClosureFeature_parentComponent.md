# CornerClosureFeature.parentComponent Property

Parent Object: [CornerClosureFeature](CornerClosureFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"cornerClosureFeature_var" is a variable referencing a CornerClosureFeature object.  

```python
# Get the value of the property.
propertyValue = cornerClosureFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version January 2026
