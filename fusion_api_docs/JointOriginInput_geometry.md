# JointOriginInput.geometry Property

Parent Object: [JointOriginInput](JointOriginInput.md)  

## Description

Gets and sets the joint geometry for this joint origin input. This defines the location of the joint origin.

## Syntax

"jointOriginInput_var" is a variable referencing a JointOriginInput object.  

```python
# Get the value of the property.
propertyValue = jointOriginInput_var.geometry

# Set the value of the property.
jointOriginInput_var.geometry = propertyValue
```

## Property Value

This is a read/write property whose value is a [JointGeometry](JointGeometry.md).

## Version

Introduced in version September 2015
