# DataProjects.item Method

Parent Object: [DataProjects](DataProjects.md)  

## Description

Returns the specified project.

## Syntax

"dataProjects_var" is a variable referencing a [DataProjects](DataProjects.md) object.

```python
returnValue = dataProjects_var.item(index)
```

## Return Value

| Type | Description |
|----|----|
| [DataProject](DataProject.md) | Returns the specified item or null if an invalid index was specified. |

## Parameters

| Name | Type | Description |
|----|----|----|
| index | uinteger | The index of the project to return. The first project in the list has an index of 0. |

## Version

Introduced in version January 2015
