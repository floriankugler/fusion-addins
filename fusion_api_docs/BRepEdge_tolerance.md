# BRepEdge.tolerance Property

Parent Object: [BRepEdge](BRepEdge.md)  

## Description

Returns the tolerance used by a tolerant edge. This value is only useful when isTolerant is true.

## Syntax

"bRepEdge_var" is a variable referencing a BRepEdge object.  

```python
# Get the value of the property.
propertyValue = bRepEdge_var.tolerance
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version August 2014
