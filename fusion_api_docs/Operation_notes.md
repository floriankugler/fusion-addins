# Operation.notes Property

Parent Object: [Operation](Operation.md)  

## Description

Gets and sets the notes of the operation.

## Syntax

"operation_var" is a variable referencing an Operation object.  

```python
# Get the value of the property.
propertyValue = operation_var.notes

# Set the value of the property.
operation_var.notes = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version May 2020
