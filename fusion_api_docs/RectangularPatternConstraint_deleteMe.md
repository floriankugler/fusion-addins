# RectangularPatternConstraint.deleteMe Method

Parent Object: [RectangularPatternConstraint](RectangularPatternConstraint.md)  

## Description

Deletes this constraint. The IsDeletable property can be used to determine if this constraint can be deleted.

## Syntax

"rectangularPatternConstraint_var" is a variable referencing a [RectangularPatternConstraint](RectangularPatternConstraint.md) object.

```python
returnValue = rectangularPatternConstraint_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version March 2016
