# FlangeFeature.attributes Property

Parent Object: [FlangeFeature](FlangeFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"flangeFeature_var" is a variable referencing a FlangeFeature object.  

```python
# Get the value of the property.
propertyValue = flangeFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version August 2020
