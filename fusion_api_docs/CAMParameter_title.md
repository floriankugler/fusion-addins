# CAMParameter.title Property

Parent Object: [CAMParameter](CAMParameter.md)  

## Description

Returns the title of this parameter as seen in the user interface. This title is localized and can change based on the current language

## Syntax

"cAMParameter_var" is a variable referencing a CAMParameter object.  

```python
# Get the value of the property.
propertyValue = cAMParameter_var.title
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version September 2020
