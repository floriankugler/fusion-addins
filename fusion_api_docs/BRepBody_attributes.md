# BRepBody.attributes Property

Parent Object: [BRepBody](BRepBody.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"bRepBody_var" is a variable referencing a BRepBody object.  

```python
# Get the value of the property.
propertyValue = bRepBody_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
