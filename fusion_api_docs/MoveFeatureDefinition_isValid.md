# MoveFeatureDefinition.isValid Property

Parent Object: [MoveFeatureDefinition](MoveFeatureDefinition.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"moveFeatureDefinition_var" is a variable referencing a MoveFeatureDefinition object.  

```python
# Get the value of the property.
propertyValue = moveFeatureDefinition_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2023
