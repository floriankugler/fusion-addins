# SketchLine.startSketchPoint Property

Parent Object: [SketchLine](SketchLine.md)  

## Description

The sketch point at the start of the line. The line is dependent on this point and moving the point will cause the line to adjust.

## Syntax

"sketchLine_var" is a variable referencing a SketchLine object.  

```python
# Get the value of the property.
propertyValue = sketchLine_var.startSketchPoint
```

## Property Value

This is a read only property whose value is a [SketchPoint](SketchPoint.md).

## Version

Introduced in version August 2014
