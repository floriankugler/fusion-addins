# ShellFeature.parentComponent Property

Parent Object: [ShellFeature](ShellFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"shellFeature_var" is a variable referencing a ShellFeature object.  

```python
# Get the value of the property.
propertyValue = shellFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version November 2014
