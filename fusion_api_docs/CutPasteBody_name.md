# CutPasteBody.name Property

Parent Object: [CutPasteBody](CutPasteBody.md)  

## Description

Returns the name of the feature as seen in the browser (non-parametric) or in the timeline (parametric).

## Syntax

"cutPasteBody_var" is a variable referencing a CutPasteBody object.  

```python
# Get the value of the property.
propertyValue = cutPasteBody_var.name

# Set the value of the property.
cutPasteBody_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.
