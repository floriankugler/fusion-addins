# ConstructionPointEdgePlaneDefinition.isValid Property

Parent Object: [ConstructionPointEdgePlaneDefinition](ConstructionPointEdgePlaneDefinition.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"constructionPointEdgePlaneDefinition_var" is a variable referencing a ConstructionPointEdgePlaneDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionPointEdgePlaneDefinition_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.
