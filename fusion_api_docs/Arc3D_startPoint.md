# Arc3D.startPoint Property

Parent Object: [Arc3D](Arc3D.md)  

## Description

Gets the start point of the arc.

## Syntax

"arc3D_var" is a variable referencing an Arc3D object.  

```python
# Get the value of the property.
propertyValue = arc3D_var.startPoint
```

## Property Value

This is a read only property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
