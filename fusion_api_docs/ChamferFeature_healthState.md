# ChamferFeature.healthState Property

Parent Object: [ChamferFeature](ChamferFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"chamferFeature_var" is a variable referencing a ChamferFeature object.  

```python
# Get the value of the property.
propertyValue = chamferFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
