# MFGDMDataEventHandler.notify Method

Parent Object: [MFGDMDataEventHandler](MFGDMDataEventHandler.md)  

## Description

The function called by Fusion when the associated event is fired.

## Syntax

"mFGDMDataEventHandler_var" is a variable referencing a [MFGDMDataEventHandler](MFGDMDataEventHandler.md) object.

```python
returnValue = mFGDMDataEventHandler_var.notify(eventArgs)
```

## Parameters

| Name | Type | Description |
|----|----|----|
| eventArgs | [MFGDMDataEventArgs](MFGDMDataEventArgs.md) | Returns an object that provides access to additional information associated with the event. |

## Version

Introduced in version July 2025
