# Line3D.isColinearTo Method

Parent Object: [Line3D](Line3D.md)  

## Description

Compare this line with another to check for collinearity

## Syntax

"line3D_var" is a variable referencing a [Line3D](Line3D.md) object.

```python
returnValue = line3D_var.isColinearTo(line)
```

## Return Value

| Type    | Description                                 |
|---------|---------------------------------------------|
| boolean | Returns true if the two lines are collinear |

## Parameters

| Name | Type                 | Description                               |
|------|----------------------|-------------------------------------------|
| line | [Line3D](Line3D.md) | The line to compare with for collinearity |

## Version

Introduced in version August 2014
