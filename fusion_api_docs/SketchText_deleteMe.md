# SketchText.deleteMe Method

Parent Object: [SketchText](SketchText.md)  

## Description

Deletes the entity from the sketch.

## Syntax

"sketchText_var" is a variable referencing a [SketchText](SketchText.md) object.

```python
returnValue = sketchText_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true is the delete was successful. |

## Version

Introduced in version March 2015
