# MachineAvoidDirectSelection.hasWarning Property

Parent Object: [MachineAvoidDirectSelection](MachineAvoidDirectSelection.md)  

## Description

Gets if warnings were encountered when applying the selection to a parent.

## Syntax

"machineAvoidDirectSelection_var" is a variable referencing a MachineAvoidDirectSelection object.  

```python
# Get the value of the property.
propertyValue = machineAvoidDirectSelection_var.hasWarning
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2024
