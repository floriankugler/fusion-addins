# MeshBody.name Property

Parent Object: [MeshBody](MeshBody.md)  

## Description

Gets and sets the name of the mesh body as displayed in the browser.

## Syntax

"meshBody_var" is a variable referencing a MeshBody object.  

```python
# Get the value of the property.
propertyValue = meshBody_var.name

# Set the value of the property.
meshBody_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.
