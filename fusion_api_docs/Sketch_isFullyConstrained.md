# Sketch.isFullyConstrained Property

Parent Object: [Sketch](Sketch.md)  

## Description

Indicates if this sketch is fully constrained.

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.isFullyConstrained
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2016
