# User.displayName Property

Parent Object: [User](User.md)  

## Description

Returns display name of the user. (i.e. the name that shows up in the Fusion UI)

## Syntax

"user_var" is a variable referencing a User object.  

```python
# Get the value of the property.
propertyValue = user_var.displayName
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version January 2016
