# DeleteFaceFeature.healthState Property

Parent Object: [DeleteFaceFeature](DeleteFaceFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"deleteFaceFeature_var" is a variable referencing a DeleteFaceFeature object.  

```python
# Get the value of the property.
propertyValue = deleteFaceFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version August 2016
