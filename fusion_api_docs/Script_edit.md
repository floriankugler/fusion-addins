# Script.edit Method

Parent Object: [Script](Script.md)  

## Description

Invokes the default edit behavior for this script or add-in.

## Syntax

"script_var" is a variable referencing a [Script](Script.md) object.

```python
returnValue = script_var.edit()
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Version

Introduced in version October 2023
