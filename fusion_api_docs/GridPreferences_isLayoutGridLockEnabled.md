# GridPreferences.isLayoutGridLockEnabled Property

Parent Object: [GridPreferences](GridPreferences.md)  

## Description

Gets and sets if the layout grid lock is enabled.

## Syntax

"gridPreferences_var" is a variable referencing a GridPreferences object.  

```python
# Get the value of the property.
propertyValue = gridPreferences_var.isLayoutGridLockEnabled

# Set the value of the property.
gridPreferences_var.isLayoutGridLockEnabled = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
