# Ellipse3D.copy Method

Parent Object: [Ellipse3D](Ellipse3D.md)  

## Description

Creates a copy of this Ellipse3D object.

## Syntax

"ellipse3D_var" is a variable referencing an [Ellipse3D](Ellipse3D.md) object.

```python
returnValue = ellipse3D_var.copy()
```

## Return Value

| Type                       | Description                                  |
|----------------------------|----------------------------------------------|
| [Ellipse3D](Ellipse3D.md) | Returns the independent copy of the ellipse. |

## Version

Introduced in version August 2014
