# GeneralPreferences.graphicsDriver Property

Parent Object: [GeneralPreferences](GeneralPreferences.md)  

## Description

Gets and sets the graphics driver used to display the graphics.

## Syntax

"generalPreferences_var" is a variable referencing a GeneralPreferences object.  

```python
# Get the value of the property.
propertyValue = generalPreferences_var.graphicsDriver

# Set the value of the property.
generalPreferences_var.graphicsDriver = propertyValue
```

## Property Value

This is a read/write property whose value is a [GraphicsDrivers](GraphicsDrivers.md).

## Version

Introduced in version August 2014
