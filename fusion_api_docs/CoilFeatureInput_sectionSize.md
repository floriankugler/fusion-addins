# CoilFeatureInput.sectionSize Property

Parent Object: [CoilFeatureInput](CoilFeatureInput.md)  

## Description

Gets and sets the section size.

## Syntax

"coilFeatureInput_var" is a variable referencing a CoilFeatureInput object.  

```python
# Get the value of the property.
propertyValue = coilFeatureInput_var.sectionSize

# Set the value of the property.
coilFeatureInput_var.sectionSize = propertyValue
```

## Property Value

This is a read/write property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version March 2016
