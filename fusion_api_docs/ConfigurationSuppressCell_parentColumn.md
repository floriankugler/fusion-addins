# ConfigurationSuppressCell.parentColumn Property

Parent Object: [ConfigurationSuppressCell](ConfigurationSuppressCell.md)  

## Description

Returns the column this cell is in.

## Syntax

"configurationSuppressCell_var" is a variable referencing a ConfigurationSuppressCell object.  

```python
# Get the value of the property.
propertyValue = configurationSuppressCell_var.parentColumn
```

## Property Value

This is a read only property whose value is a [ConfigurationSuppressColumn](ConfigurationSuppressColumn.md).

## Version

Introduced in version January 2024
