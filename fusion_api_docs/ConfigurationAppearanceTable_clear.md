# ConfigurationAppearanceTable.clear Method

Parent Object: [ConfigurationAppearanceTable](ConfigurationAppearanceTable.md)  

## Description

Clears the content of the appearance table, removes the reference from the top table, and hides it in the user interface.

## Syntax

"configurationAppearanceTable_var" is a variable referencing a [ConfigurationAppearanceTable](ConfigurationAppearanceTable.md) object.

```python
returnValue = configurationAppearanceTable_var.clear()
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Version

Introduced in version March 2024
