# RotaryMachineAxis.axisType Property

Parent Object: [RotaryMachineAxis](RotaryMachineAxis.md)  

## Description

The type of axis.

## Syntax

"rotaryMachineAxis_var" is a variable referencing a RotaryMachineAxis object.  

```python
# Get the value of the property.
propertyValue = rotaryMachineAxis_var.axisType
```

## Property Value

This is a read only property whose value is a [MachineAxisTypes](MachineAxisTypes.md).

## Version

Introduced in version April 2023
