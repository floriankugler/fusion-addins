# SketchEllipticalArc.isFixed Property

Parent Object: [SketchEllipticalArc](SketchEllipticalArc.md)  

## Description

Indicates if this geometry is "fixed".

## Syntax

"sketchEllipticalArc_var" is a variable referencing a SketchEllipticalArc object.  

```python
# Get the value of the property.
propertyValue = sketchEllipticalArc_var.isFixed

# Set the value of the property.
sketchEllipticalArc_var.isFixed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
