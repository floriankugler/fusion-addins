# Sketch.timelineObject Property

Parent Object: [Sketch](Sketch.md)  

## Description

Returns the timeline object associated with this sketch.

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version August 2014
