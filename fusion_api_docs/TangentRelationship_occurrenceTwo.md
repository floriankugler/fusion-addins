# TangentRelationship.occurrenceTwo Property

Parent Object: [TangentRelationship](TangentRelationship.md)  

## Description

Returns the second of two occurrences that this tangent relationship defines a relationship between.

## Syntax

"tangentRelationship_var" is a variable referencing a TangentRelationship object.  

```python
# Get the value of the property.
propertyValue = tangentRelationship_var.occurrenceTwo
```

## Property Value

This is a read only property whose value is an [Occurrence](Occurrence.md).

## Version

Introduced in version May 2022
