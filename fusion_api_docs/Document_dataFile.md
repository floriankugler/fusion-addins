# Document.dataFile Property

Parent Object: [Document](Document.md)  

## Description

Gets the DataFile that represents this document in A360.

## Syntax

"document_var" is a variable referencing a Document object.  

```python
# Get the value of the property.
propertyValue = document_var.dataFile
```

## Property Value

This is a read only property whose value is a [DataFile](DataFile.md).

## Version

Introduced in version July 2016
