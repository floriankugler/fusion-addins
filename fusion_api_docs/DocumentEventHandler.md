# DocumentEventHandler Object

Derived from: [EventHandler](EventHandler.md) Object  

## Description

The DocumentEventHandler is a client implemented class that can be added as a handler to a DocumentEvent.

## Methods

| Name | Description |
|----|----|
| [notify](DocumentEventHandler_notify.md) | The function called by Fusion when the associated event is fired. |

## Version

Introduced in version August 2014
