# URL.toString Method

Parent Object: [URL](URL.md)  

## Description

Get the entire URL as string.

## Syntax

"uRL_var" is a variable referencing a [URL](URL.md) object.

```python
returnValue = uRL_var.toString()
```

## Return Value

| Type   | Description                       |
|--------|-----------------------------------|
| string | Returns the entire URL as string. |

## Version

Introduced in version April 2023
