# ReverseNormalFeature.name Property

Parent Object: [ReverseNormalFeature](ReverseNormalFeature.md)  

## Description

Returns the name of the feature as seen in the browser (non-parametric) or in the timeline (parametric).

## Syntax

"reverseNormalFeature_var" is a variable referencing a ReverseNormalFeature object.  

```python
# Get the value of the property.
propertyValue = reverseNormalFeature_var.name

# Set the value of the property.
reverseNormalFeature_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version September 2015
