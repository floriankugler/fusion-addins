# BRepShell.wire Property

Parent Object: [BRepShell](BRepShell.md)  

## Description

Returns the wire body, if any, that exists in this shell. Returns null if the shell doesn't have a wire body.

## Syntax

"bRepShell_var" is a variable referencing a BRepShell object.  

```python
# Get the value of the property.
propertyValue = bRepShell_var.wire
```

## Property Value

This is a read only property whose value is a [BRepWire](BRepWire.md).

## Version

Introduced in version December 2017
