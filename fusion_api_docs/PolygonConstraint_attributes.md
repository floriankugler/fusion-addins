# PolygonConstraint.attributes Property

Parent Object: [PolygonConstraint](PolygonConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"polygonConstraint_var" is a variable referencing a PolygonConstraint object.  

```python
# Get the value of the property.
propertyValue = polygonConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
