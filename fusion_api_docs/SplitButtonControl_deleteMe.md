# SplitButtonControl.deleteMe Method

Parent Object: [SplitButtonControl](SplitButtonControl.md)  

## Description

Deletes the ToolbarControl

## Syntax

"splitButtonControl_var" is a variable referencing a [SplitButtonControl](SplitButtonControl.md) object.

```python
returnValue = splitButtonControl_var.deleteMe()
```

## Return Value

| Type    | Description                                                  |
|---------|--------------------------------------------------------------|
| boolean | Returns a boolean indicating if the deletion was successful. |

## Version

Introduced in version August 2014
