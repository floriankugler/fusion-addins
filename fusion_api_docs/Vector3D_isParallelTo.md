# Vector3D.isParallelTo Method

Parent Object: [Vector3D](Vector3D.md)  

## Description

Determines if the input vector is parallel with this vector.

## Syntax

"vector3D_var" is a variable referencing a [Vector3D](Vector3D.md) object.

```python
returnValue = vector3D_var.isParallelTo(vector)
```

## Return Value

| Type    | Description                               |
|---------|-------------------------------------------|
| boolean | Returns true if the vectors are parallel. |

## Parameters

| Name   | Type                     | Description                        |
|--------|--------------------------|------------------------------------|
| vector | [Vector3D](Vector3D.md) | The vector to test parallelism to. |

## Version

Introduced in version August 2014
