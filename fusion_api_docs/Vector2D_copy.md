# Vector2D.copy Method

Parent Object: [Vector2D](Vector2D.md)  

## Description

Creates and returns an independent copy of this Vector2D object.

## Syntax

"vector2D_var" is a variable referencing a [Vector2D](Vector2D.md) object.

```python
returnValue = vector2D_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [Vector2D](Vector2D.md) | Returns a new Vector2D object that is a copy of this Vector2D object. |

## Version

Introduced in version August 2014
