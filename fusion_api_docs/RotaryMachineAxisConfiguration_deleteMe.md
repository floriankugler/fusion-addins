# RotaryMachineAxisConfiguration.deleteMe Method

Parent Object: [RotaryMachineAxisConfiguration](RotaryMachineAxisConfiguration.md)  

## Description

Delete this axis configuration from the controller configuration.

## Syntax

"rotaryMachineAxisConfiguration_var" is a variable referencing a [RotaryMachineAxisConfiguration](RotaryMachineAxisConfiguration.md) object.

```python
returnValue = rotaryMachineAxisConfiguration_var.deleteMe()
```

## Version

Introduced in version April 2023
