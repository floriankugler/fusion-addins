# CustomGraphicsText.size Property

Parent Object: [CustomGraphicsText](CustomGraphicsText.md)  

## Description

Gets and sets the size of the text in centimeters. This is the default size and applies to all of text unless there is a size override defined within the text.

## Syntax

"customGraphicsText_var" is a variable referencing a CustomGraphicsText object.  

```python
# Get the value of the property.
propertyValue = customGraphicsText_var.size

# Set the value of the property.
customGraphicsText_var.size = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version April 2019
