# RibFeature.isSuppressed Property

Parent Object: [RibFeature](RibFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"ribFeature_var" is a variable referencing a RibFeature object.  

```python
# Get the value of the property.
propertyValue = ribFeature_var.isSuppressed

# Set the value of the property.
ribFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version September 2015
