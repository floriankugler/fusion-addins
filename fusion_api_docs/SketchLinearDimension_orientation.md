# SketchLinearDimension.orientation Property

Parent Object: [SketchLinearDimension](SketchLinearDimension.md)  

## Description

The orientation of this dimension.

## Syntax

"sketchLinearDimension_var" is a variable referencing a SketchLinearDimension object.  

```python
# Get the value of the property.
propertyValue = sketchLinearDimension_var.orientation
```

## Property Value

This is a read only property whose value is a [DimensionOrientations](DimensionOrientations.md).

## Version

Introduced in version August 2014
