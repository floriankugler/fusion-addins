# ActiveSelectionEventHandler.notify Method

Parent Object: [ActiveSelectionEventHandler](ActiveSelectionEventHandler.md)  

## Description

The function called by Fusion when the associated event is fired.

## Syntax

"activeSelectionEventHandler_var" is a variable referencing an [ActiveSelectionEventHandler](ActiveSelectionEventHandler.md) object.

```python
returnValue = activeSelectionEventHandler_var.notify(eventArgs)
```

## Parameters

| Name | Type | Description |
|----|----|----|
| eventArgs | [ActiveSelectionEventArgs](ActiveSelectionEventArgs.md) | Returns an object that provides access to additional information associated with the event. |

## Version

Introduced in version August 2020
