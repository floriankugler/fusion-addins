# Arc3D.copy Method

Parent Object: [Arc3D](Arc3D.md)  

## Description

Creates and returns an independent copy of this Arc3D object.

## Syntax

"arc3D_var" is a variable referencing an [Arc3D](Arc3D.md) object.

```python
returnValue = arc3D_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [Arc3D](Arc3D.md) | Returns a new Arc3D object that is a copy of this Arc3D object. |

## Version

Introduced in version August 2014
