# ConfigurationSheetMetalRuleColumn.rowCount Property

Parent Object: [ConfigurationSheetMetalRuleColumn](ConfigurationSheetMetalRuleColumn.md)  

## Description

Returns the number of rows in this column.

## Syntax

"configurationSheetMetalRuleColumn_var" is a variable referencing a ConfigurationSheetMetalRuleColumn object.  

```python
# Get the value of the property.
propertyValue = configurationSheetMetalRuleColumn_var.rowCount
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version March 2024
