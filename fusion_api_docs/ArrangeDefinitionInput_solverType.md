# ArrangeDefinitionInput.solverType Property

Parent Object: [ArrangeDefinitionInput](ArrangeDefinitionInput.md)  

## Description

Gets the type of arrange feature defined by this definition.

## Syntax

"arrangeDefinitionInput_var" is a variable referencing an ArrangeDefinitionInput object.  

```python
# Get the value of the property.
propertyValue = arrangeDefinitionInput_var.solverType
```

## Property Value

This is a read only property whose value is an [ArrangeSolverTypes](ArrangeSolverTypes.md).

## Version

Introduced in version January 2025
