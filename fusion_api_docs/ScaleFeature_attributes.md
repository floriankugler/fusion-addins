# ScaleFeature.attributes Property

Parent Object: [ScaleFeature](ScaleFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"scaleFeature_var" is a variable referencing a ScaleFeature object.  

```python
# Get the value of the property.
propertyValue = scaleFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
