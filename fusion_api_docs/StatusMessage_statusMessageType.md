# StatusMessage.statusMessageType Property

Parent Object: [StatusMessage](StatusMessage.md)  

## Description

Returns the type of message this StatusMessage represents.

## Syntax

"statusMessage_var" is a variable referencing a StatusMessage object.  

```python
# Get the value of the property.
propertyValue = statusMessage_var.statusMessageType
```

## Property Value

This is a read only property whose value is a [StatusMessageTypes](StatusMessageTypes.md).

## Version

Introduced in version July 2021
