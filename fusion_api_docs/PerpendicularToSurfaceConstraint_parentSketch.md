# PerpendicularToSurfaceConstraint.parentSketch Property

Parent Object: [PerpendicularToSurfaceConstraint](PerpendicularToSurfaceConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"perpendicularToSurfaceConstraint_var" is a variable referencing a PerpendicularToSurfaceConstraint object.  

```python
# Get the value of the property.
propertyValue = perpendicularToSurfaceConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version September 2023
