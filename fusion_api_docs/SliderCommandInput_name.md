# SliderCommandInput.name Property

Parent Object: [SliderCommandInput](SliderCommandInput.md)  

## Description

Gets the user visible name of this input.

## Syntax

"sliderCommandInput_var" is a variable referencing a SliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = sliderCommandInput_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version June 2015
