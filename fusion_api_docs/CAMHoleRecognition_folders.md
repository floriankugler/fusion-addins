# CAMHoleRecognition.folders Property

Parent Object: [CAMHoleRecognition](CAMHoleRecognition.md)  

## Description

Returns the Folders collection that provides access to existing folders in this hole recognition.

## Syntax

"cAMHoleRecognition_var" is a variable referencing a CAMHoleRecognition object.  

```python
# Get the value of the property.
propertyValue = cAMHoleRecognition_var.folders
```

## Property Value

This is a read only property whose value is a [CAMFolders](CAMFolders.md).

## Version

Introduced in version April 2023
