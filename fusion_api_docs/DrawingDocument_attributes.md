# DrawingDocument.attributes Property

Parent Object: [DrawingDocument](DrawingDocument.md)  

## Description

Returns the collection of attributes associated with this document.

## Syntax

"drawingDocument_var" is a variable referencing a DrawingDocument object.  

```python
# Get the value of the property.
propertyValue = drawingDocument_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version December 2020
