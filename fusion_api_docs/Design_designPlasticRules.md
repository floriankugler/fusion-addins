# Design.designPlasticRules Property

Parent Object: [Design](Design.md)  

## Description

Gets the collection of plastic rules in the design.

## Syntax

"design_var" is a variable referencing a Design object.  

```python
# Get the value of the property.
propertyValue = design_var.designPlasticRules
```

## Property Value

This is a read only property whose value is a [PlasticRules](PlasticRules.md).

## Version

Introduced in version January 2024
