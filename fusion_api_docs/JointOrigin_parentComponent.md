# JointOrigin.parentComponent Property

Parent Object: [JointOrigin](JointOrigin.md)  

## Description

Returns the parent component that owns this joint origin.

## Syntax

"jointOrigin_var" is a variable referencing a JointOrigin object.  

```python
# Get the value of the property.
propertyValue = jointOrigin_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version September 2015
