# PipeFeature.isHollow Property

Parent Object: [PipeFeature](PipeFeature.md)  

## Description

Specifies if the Pipe is hollow or not.  

Setting this to true will default the sectionThickness to 0.1 cm.

## Syntax

"pipeFeature_var" is a variable referencing a PipeFeature object.  

```python
# Get the value of the property.
propertyValue = pipeFeature_var.isHollow

# Set the value of the property.
pipeFeature_var.isHollow = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version October 2023
