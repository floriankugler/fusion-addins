# SketchFixedSpline.attributes Property

Parent Object: [SketchFixedSpline](SketchFixedSpline.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"sketchFixedSpline_var" is a variable referencing a SketchFixedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFixedSpline_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
