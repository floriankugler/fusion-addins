# SketchLine.isCenterLine Property

Parent Object: [SketchLine](SketchLine.md)  

## Description

Gets and sets whether this line is defined as a centerline.

## Syntax

"sketchLine_var" is a variable referencing a SketchLine object.  

```python
# Get the value of the property.
propertyValue = sketchLine_var.isCenterLine

# Set the value of the property.
sketchLine_var.isCenterLine = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2021
