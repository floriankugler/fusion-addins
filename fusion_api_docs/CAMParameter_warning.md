# CAMParameter.warning Property

Parent Object: [CAMParameter](CAMParameter.md)  

## Description

Returns a message corresponding to any active warning associated with the value of this parameter.

## Syntax

"cAMParameter_var" is a variable referencing a CAMParameter object.  

```python
# Get the value of the property.
propertyValue = cAMParameter_var.warning
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version September 2020
