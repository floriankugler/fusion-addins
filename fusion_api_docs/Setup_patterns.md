# Setup.patterns Property

Parent Object: [Setup](Setup.md)  

## Description

Returns the Patterns collection that provides access to existing patterns in this setup.

## Syntax

"setup_var" is a variable referencing a Setup object.  

```python
# Get the value of the property.
propertyValue = setup_var.patterns
```

## Property Value

This is a read only property whose value is a [CAMPatterns](CAMPatterns.md).

## Version

Introduced in version January 2016
