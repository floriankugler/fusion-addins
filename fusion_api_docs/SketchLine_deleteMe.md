# SketchLine.deleteMe Method

Parent Object: [SketchLine](SketchLine.md)  

## Description

Deletes the entity from the sketch.

## Syntax

"sketchLine_var" is a variable referencing a [SketchLine](SketchLine.md) object.

```python
returnValue = sketchLine_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true is the delete was successful. |

## Version

Introduced in version August 2014
