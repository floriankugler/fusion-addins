# SceneSettings.restoreDefaults Method

Parent Object: [SceneSettings](SceneSettings.md)  

## Description

Changes all of the scene settings to the default values.

## Syntax

"sceneSettings_var" is a variable referencing a [SceneSettings](SceneSettings.md) object.

```python
returnValue = sceneSettings_var.restoreDefaults()
```

## Return Value

| Type    | Description                                                     |
|---------|-----------------------------------------------------------------|
| boolean | Returns true if setting to the default settings was successful. |

## Version

Introduced in version May 2023
