# SketchPoint.isFullyConstrained Property

Parent Object: [SketchPoint](SketchPoint.md)  

## Description

Indicates if this sketch entity is fully constrained.

## Syntax

"sketchPoint_var" is a variable referencing a SketchPoint object.  

```python
# Get the value of the property.
propertyValue = sketchPoint_var.isFullyConstrained
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2016
