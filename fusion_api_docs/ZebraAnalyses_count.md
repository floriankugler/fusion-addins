# ZebraAnalyses.count Property

Parent Object: [ZebraAnalyses](ZebraAnalyses.md)  

## Description

Returns the number of ZebraAnalysis objects in the collection.

## Syntax

"zebraAnalyses_var" is a variable referencing a ZebraAnalyses object.  

```python
# Get the value of the property.
propertyValue = zebraAnalyses_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version January 2023
