# FusionDefaultUnitsPreferences.distanceDisplayUnits Property

Parent Object: [FusionDefaultUnitsPreferences](FusionDefaultUnitsPreferences.md)  

## Description

Gets and sets the default design units for length when creating a new Fusion file. Setting this property will have the side effect of changing the defaultUnitSystem property to custom.

## Syntax

"fusionDefaultUnitsPreferences_var" is a variable referencing a FusionDefaultUnitsPreferences object.  

```python
# Get the value of the property.
propertyValue = fusionDefaultUnitsPreferences_var.distanceDisplayUnits

# Set the value of the property.
fusionDefaultUnitsPreferences_var.distanceDisplayUnits = propertyValue
```

## Property Value

This is a read/write property whose value is a [DistanceUnits](DistanceUnits.md).

## Version

Introduced in version August 2014
