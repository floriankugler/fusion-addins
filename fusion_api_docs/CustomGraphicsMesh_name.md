# CustomGraphicsMesh.name Property

Parent Object: [CustomGraphicsMesh](CustomGraphicsMesh.md)  

## Description

Gets and sets the name displayed when this entity is selected. If no name has been set, "Custom Graphics" will be displayed.

## Syntax

"customGraphicsMesh_var" is a variable referencing a CustomGraphicsMesh object.  

```python
# Get the value of the property.
propertyValue = customGraphicsMesh_var.name

# Set the value of the property.
customGraphicsMesh_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version October 2022
