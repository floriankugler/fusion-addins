# BRepFace.loops Property

Parent Object: [BRepFace](BRepFace.md)  

## Description

Returns the BRepLoops owned by this face

## Syntax

"bRepFace_var" is a variable referencing a BRepFace object.  

```python
# Get the value of the property.
propertyValue = bRepFace_var.loops
```

## Property Value

This is a read only property whose value is a [BRepLoops](BRepLoops.md).
