# MotionLink.parentComponent Property

Parent Object: [MotionLink](MotionLink.md)  

## Description

Returns the parent component that owns this MotionLink.

## Syntax

"motionLink_var" is a variable referencing a MotionLink object.  

```python
# Get the value of the property.
propertyValue = motionLink_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version November 2025
