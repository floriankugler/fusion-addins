# RectangularPatternFeature.patternElements Property

Parent Object: [RectangularPatternFeature](RectangularPatternFeature.md)  

## Description

Gets the PatternElements collection that contains the elements created by this pattern.

## Syntax

"rectangularPatternFeature_var" is a variable referencing a RectangularPatternFeature object.  

```python
# Get the value of the property.
propertyValue = rectangularPatternFeature_var.patternElements
```

## Property Value

This is a read only property whose value is a [PatternElements](PatternElements.md).

## Version

Introduced in version November 2014
