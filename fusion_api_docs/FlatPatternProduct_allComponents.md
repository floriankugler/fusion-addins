# FlatPatternProduct.allComponents Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Returns the Components collection that provides access to existing components in a design.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.allComponents
```

## Property Value

This is a read only property whose value is a [Components](Components.md).

## Version

Introduced in version October 2022
