# SketchLinearDiameterDimension.attributes Property

Parent Object: [SketchLinearDiameterDimension](SketchLinearDiameterDimension.md)  

## Description

Returns the collection of attributes associated with this sketch dimension.

## Syntax

"sketchLinearDiameterDimension_var" is a variable referencing a SketchLinearDiameterDimension object.  

```python
# Get the value of the property.
propertyValue = sketchLinearDiameterDimension_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version September 2022
