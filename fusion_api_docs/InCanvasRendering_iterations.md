# InCanvasRendering.iterations Property

Parent Object: [InCanvasRendering](InCanvasRendering.md)  

## Description

Returns the current number of iterations the renderer has completed.

## Syntax

"inCanvasRendering_var" is a variable referencing an InCanvasRendering object.  

```python
# Get the value of the property.
propertyValue = inCanvasRendering_var.iterations
```

## Property Value

This is a read only property whose value is an integer.

## Version

Introduced in version September 2023
