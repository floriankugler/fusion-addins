# UnitAndValuePreferences.areAbbreviationsForUnitDisplayed Property

Parent Object: [UnitAndValuePreferences](UnitAndValuePreferences.md)  

## Description

Gets and sets if abbreviations are used for units display.

## Syntax

"unitAndValuePreferences_var" is a variable referencing a UnitAndValuePreferences object.  

```python
# Get the value of the property.
propertyValue = unitAndValuePreferences_var.areAbbreviationsForUnitDisplayed

# Set the value of the property.
unitAndValuePreferences_var.areAbbreviationsForUnitDisplayed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
