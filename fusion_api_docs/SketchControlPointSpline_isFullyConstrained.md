# SketchControlPointSpline.isFullyConstrained Property

Parent Object: [SketchControlPointSpline](SketchControlPointSpline.md)  

## Description

Indicates if this sketch entity is fully constrained.

## Syntax

"sketchControlPointSpline_var" is a variable referencing a SketchControlPointSpline object.  

```python
# Get the value of the property.
propertyValue = sketchControlPointSpline_var.isFullyConstrained
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2022
