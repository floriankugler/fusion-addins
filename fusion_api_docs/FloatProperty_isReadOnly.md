# FloatProperty.isReadOnly Property

Parent Object: [FloatProperty](FloatProperty.md)  

## Description

Indicates if this property is read-only. If True any attempted edits will fail.

## Syntax

"floatProperty_var" is a variable referencing a FloatProperty object.  

```python
# Get the value of the property.
propertyValue = floatProperty_var.isReadOnly
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
