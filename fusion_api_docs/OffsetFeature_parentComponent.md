# OffsetFeature.parentComponent Property

Parent Object: [OffsetFeature](OffsetFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"offsetFeature_var" is a variable referencing an OffsetFeature object.  

```python
# Get the value of the property.
propertyValue = offsetFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version June 2015
