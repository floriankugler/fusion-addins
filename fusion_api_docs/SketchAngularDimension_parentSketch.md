# SketchAngularDimension.parentSketch Property

Parent Object: [SketchAngularDimension](SketchAngularDimension.md)  

## Description

Returns the parent sketch object.

## Syntax

"sketchAngularDimension_var" is a variable referencing a SketchAngularDimension object.  

```python
# Get the value of the property.
propertyValue = sketchAngularDimension_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
