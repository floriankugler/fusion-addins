# CopyPasteBody.timelineObject Property

Parent Object: [CopyPasteBody](CopyPasteBody.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"copyPasteBody_var" is a variable referencing a CopyPasteBody object.  

```python
# Get the value of the property.
propertyValue = copyPasteBody_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version June 2017
