# DeriveFeature.attributes Property

Parent Object: [DeriveFeature](DeriveFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"deriveFeature_var" is a variable referencing a DeriveFeature object.  

```python
# Get the value of the property.
propertyValue = deriveFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version January 2026
