# OperationBase.isSelected Property

Parent Object: [OperationBase](OperationBase.md)  

## Description

Gets if this operation is selected in the 'Setups' browser.

## Syntax

"operationBase_var" is a variable referencing an OperationBase object.  

```python
# Get the value of the property.
propertyValue = operationBase_var.isSelected
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version May 2020
