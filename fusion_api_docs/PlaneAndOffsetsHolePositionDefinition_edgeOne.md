# PlaneAndOffsetsHolePositionDefinition.edgeOne Property

Parent Object: [PlaneAndOffsetsHolePositionDefinition](PlaneAndOffsetsHolePositionDefinition.md)  

## Description

The first of 2 edges the hole position is measured from. OffsetOne provides access to the model parameter controlling the offset distance.

## Syntax

"planeAndOffsetsHolePositionDefinition_var" is a variable referencing a PlaneAndOffsetsHolePositionDefinition object.  

```python
# Get the value of the property.
propertyValue = planeAndOffsetsHolePositionDefinition_var.edgeOne
```

## Property Value

This is a read only property whose value is a [BRepEdge](BRepEdge.md).

## Version

Introduced in version August 2014
