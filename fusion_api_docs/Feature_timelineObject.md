# Feature.timelineObject Property

Parent Object: [Feature](Feature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"feature_var" is a variable referencing a Feature object.  

```python
# Get the value of the property.
propertyValue = feature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version August 2014
