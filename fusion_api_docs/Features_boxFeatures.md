# Features.boxFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the existing box features.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.boxFeatures
```

## Property Value

This is a read only property whose value is a [BoxFeatures](BoxFeatures.md).

## Version

Introduced in version September 2015
