# Sphere.radius Property

Parent Object: [Sphere](Sphere.md)  

## Description

Gets and sets the radius of the sphere.

## Syntax

"sphere_var" is a variable referencing a Sphere object.  

```python
# Get the value of the property.
propertyValue = sphere_var.radius

# Set the value of the property.
sphere_var.radius = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version August 2014
