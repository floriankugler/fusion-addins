# SketchOffsetDimension.isDeletable Property

Parent Object: [SketchOffsetDimension](SketchOffsetDimension.md)  

## Description

Indicates if this dimension is deletable.

## Syntax

"sketchOffsetDimension_var" is a variable referencing a SketchOffsetDimension object.  

```python
# Get the value of the property.
propertyValue = sketchOffsetDimension_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
