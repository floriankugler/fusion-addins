# Component.parentDesign Property

Parent Object: [Component](Component.md)  

## Description

Returns the parent product this component is owned by.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.parentDesign
```

## Property Value

This is a read only property whose value is a [Design](Design.md).

## Version

Introduced in version August 2014
