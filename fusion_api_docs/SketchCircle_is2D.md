# SketchCircle.is2D Property

Parent Object: [SketchCircle](SketchCircle.md)  

## Description

Indicates if this curve lies entirely on the sketch x-y plane.

## Syntax

"sketchCircle_var" is a variable referencing a SketchCircle object.  

```python
# Get the value of the property.
propertyValue = sketchCircle_var.is2D
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
