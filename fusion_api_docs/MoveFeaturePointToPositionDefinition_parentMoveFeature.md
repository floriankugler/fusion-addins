# MoveFeaturePointToPositionDefinition.parentMoveFeature Property

Parent Object: [MoveFeaturePointToPositionDefinition](MoveFeaturePointToPositionDefinition.md)  

## Description

## Syntax

"moveFeaturePointToPositionDefinition_var" is a variable referencing a MoveFeaturePointToPositionDefinition object.  

```python
# Get the value of the property.
propertyValue = moveFeaturePointToPositionDefinition_var.parentMoveFeature
```

## Property Value

This is a read only property whose value is a [MoveFeature](MoveFeature.md).
