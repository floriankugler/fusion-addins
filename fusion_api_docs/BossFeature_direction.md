# BossFeature.direction Property

Parent Object: [BossFeature](BossFeature.md)  

## Description

Returns the direction of the boss feature with respect to selected position point. For selected sketch point this direction represents a Z-axis of the sketch.

## Syntax

"bossFeature_var" is a variable referencing a BossFeature object.  

```python
# Get the value of the property.
propertyValue = bossFeature_var.direction
```

## Property Value

This is a read only property whose value is a [Vector3D](Vector3D.md).

## Version

Introduced in version October 2022
