# UnfoldFeature.parentComponent Property

Parent Object: [UnfoldFeature](UnfoldFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"unfoldFeature_var" is a variable referencing a UnfoldFeature object.  

```python
# Get the value of the property.
propertyValue = unfoldFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version August 2020
