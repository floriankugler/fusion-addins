# ControlDefinition.isVisible Property

Parent Object: [ControlDefinition](ControlDefinition.md)  

## Description

Gets or sets if this definition is visible or not. This has the effect of making any associated controls visible or invisible in the user interface.

## Syntax

"controlDefinition_var" is a variable referencing a ControlDefinition object.  

```python
# Get the value of the property.
propertyValue = controlDefinition_var.isVisible

# Set the value of the property.
controlDefinition_var.isVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
