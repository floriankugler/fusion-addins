# Vector3D.dotProduct Method

Parent Object: [Vector3D](Vector3D.md)  

## Description

Returns the dot product between this vector and the specified vector.

## Syntax

"vector3D_var" is a variable referencing a [Vector3D](Vector3D.md) object.

```python
returnValue = vector3D_var.dotProduct(vector)
```

## Return Value

| Type   | Description                    |
|--------|--------------------------------|
| double | Returns the dot product value. |

## Parameters

| Name   | Type                     | Description                            |
|--------|--------------------------|----------------------------------------|
| vector | [Vector3D](Vector3D.md) | The vector to take the dot product to. |

## Version

Introduced in version August 2014
