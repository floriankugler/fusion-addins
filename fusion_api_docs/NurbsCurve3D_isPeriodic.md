# NurbsCurve3D.isPeriodic Property

Parent Object: [NurbsCurve3D](NurbsCurve3D.md)  

## Description

Gets if the curve is periodic.

## Syntax

"nurbsCurve3D_var" is a variable referencing a NurbsCurve3D object.  

```python
# Get the value of the property.
propertyValue = nurbsCurve3D_var.isPeriodic
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
