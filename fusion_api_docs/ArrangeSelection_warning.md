# ArrangeSelection.warning Property

Parent Object: [ArrangeSelection](ArrangeSelection.md)  

## Description

Gets the last warning string encountered after the selection was applied to a parent.

## Syntax

"arrangeSelection_var" is a variable referencing an ArrangeSelection object.  

```python
# Get the value of the property.
propertyValue = arrangeSelection_var.warning
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version March 2024
