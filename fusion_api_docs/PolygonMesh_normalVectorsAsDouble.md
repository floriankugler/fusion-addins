# PolygonMesh.normalVectorsAsDouble Property

Parent Object: [PolygonMesh](PolygonMesh.md)  

## Description

Returns the normal vectors as an array of doubles where they are the x, y, z components of each vector. There is one normal vector for each index.

## Syntax

"polygonMesh_var" is a variable referencing a PolygonMesh object.  

```python
# Get the value of the property.
propertyValue = polygonMesh_var.normalVectorsAsDouble
```

## Property Value

This is a read only property whose value is an array of type double.

## Version

Introduced in version August 2014
