# SketchText.isFullyConstrained Property

Parent Object: [SketchText](SketchText.md)  

## Description

Indicates if this sketch entity is fully constrained.

## Syntax

"sketchText_var" is a variable referencing a SketchText object.  

```python
# Get the value of the property.
propertyValue = sketchText_var.isFullyConstrained
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2016
