# PolygonConstraint.isValid Property

Parent Object: [PolygonConstraint](PolygonConstraint.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"polygonConstraint_var" is a variable referencing a PolygonConstraint object.  

```python
# Get the value of the property.
propertyValue = polygonConstraint_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version March 2016
