# EllipticalCone.origin Property

Parent Object: [EllipticalCone](EllipticalCone.md)  

## Description

Gets and sets the origin point (center) of the base of the cone.

## Syntax

"ellipticalCone_var" is a variable referencing an EllipticalCone object.  

```python
# Get the value of the property.
propertyValue = ellipticalCone_var.origin

# Set the value of the property.
ellipticalCone_var.origin = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
