# DataFolder.id Property

Parent Object: [DataFolder](DataFolder.md)  

## Description

Returns the unique ID for this folder. This is the same id used in the APS Data Management API.

## Syntax

"dataFolder_var" is a variable referencing a DataFolder object.  

```python
# Get the value of the property.
propertyValue = dataFolder_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version September 2016
