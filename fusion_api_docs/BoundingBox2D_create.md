# BoundingBox2D.create Method

Parent Object: [BoundingBox2D](BoundingBox2D.md)  

## Description

Creates a transient bounding box object.

## Syntax

This is a static method.  

```python

returnValue = adsk.core.BoundingBox2D.create(minPoint, maxPoint)
```

## Return Value

| Type                               | Description                   |
|------------------------------------|-------------------------------|
| [BoundingBox2D](BoundingBox2D.md) | Returns the new bounding box. |

## Parameters

| Name     | Type                   | Description                   |
|----------|------------------------|-------------------------------|
| minPoint | [Point2D](Point2D.md) | The minimum point of the box. |
| maxPoint | [Point2D](Point2D.md) | The maximum point of the box. |

## Version

Introduced in version August 2014
