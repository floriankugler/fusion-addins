# UnfoldFeature.timelineObject Property

Parent Object: [UnfoldFeature](UnfoldFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"unfoldFeature_var" is a variable referencing a UnfoldFeature object.  

```python
# Get the value of the property.
propertyValue = unfoldFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version August 2020
