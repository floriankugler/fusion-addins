# ConfigurationSheetMetalRuleColumn.id Property

Parent Object: [ConfigurationSheetMetalRuleColumn](ConfigurationSheetMetalRuleColumn.md)  

## Description

The id of the column. This is constant and cannot be changed by the user.

## Syntax

"configurationSheetMetalRuleColumn_var" is a variable referencing a ConfigurationSheetMetalRuleColumn object.  

```python
# Get the value of the property.
propertyValue = configurationSheetMetalRuleColumn_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version January 2024
