# GroupCommandInput.deleteMe Method

Parent Object: [GroupCommandInput](GroupCommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"groupCommandInput_var" is a variable referencing a [GroupCommandInput](GroupCommandInput.md) object.

```python
returnValue = groupCommandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2020
