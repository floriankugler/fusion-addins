# ToolbarPanels.count Property

Parent Object: [ToolbarPanels](ToolbarPanels.md)  

## Description

Gets the number of ToolbarPanels.

## Syntax

"toolbarPanels_var" is a variable referencing a ToolbarPanels object.  

```python
# Get the value of the property.
propertyValue = toolbarPanels_var.count
```

## Property Value

This is a read only property whose value is a uinteger.
