# CircularPatternConstraint.parentSketch Property

Parent Object: [CircularPatternConstraint](CircularPatternConstraint.md)  

## Description

Returns the parent sketch object.

## Syntax

"circularPatternConstraint_var" is a variable referencing a CircularPatternConstraint object.  

```python
# Get the value of the property.
propertyValue = circularPatternConstraint_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version March 2016
