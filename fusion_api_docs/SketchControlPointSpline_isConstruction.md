# SketchControlPointSpline.isConstruction Property

Parent Object: [SketchControlPointSpline](SketchControlPointSpline.md)  

## Description

Gets and sets whether this curve is construction geometry.

## Syntax

"sketchControlPointSpline_var" is a variable referencing a SketchControlPointSpline object.  

```python
# Get the value of the property.
propertyValue = sketchControlPointSpline_var.isConstruction

# Set the value of the property.
sketchControlPointSpline_var.isConstruction = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version July 2022
