# CAMFolder.attributes Property

Parent Object: [CAMFolder](CAMFolder.md)  

## Description

Returns the collection of attributes associated with this object.

## Syntax

"cAMFolder_var" is a variable referencing a CAMFolder object.  

```python
# Get the value of the property.
propertyValue = cAMFolder_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version August 2021
