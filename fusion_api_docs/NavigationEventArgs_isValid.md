# NavigationEventArgs.isValid Property

Parent Object: [NavigationEventArgs](NavigationEventArgs.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"navigationEventArgs_var" is a variable referencing a NavigationEventArgs object.  

```python
# Get the value of the property.
propertyValue = navigationEventArgs_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version March 2021
