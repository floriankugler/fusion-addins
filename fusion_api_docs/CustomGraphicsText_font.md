# CustomGraphicsText.font Property

Parent Object: [CustomGraphicsText](CustomGraphicsText.md)  

## Description

Gets and sets the font used to display the text. This is the default font and applies to all of text unless there is a font override defined within the text.

## Syntax

"customGraphicsText_var" is a variable referencing a CustomGraphicsText object.  

```python
# Get the value of the property.
propertyValue = customGraphicsText_var.font

# Set the value of the property.
customGraphicsText_var.font = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version April 2019
