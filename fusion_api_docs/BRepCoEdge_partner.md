# BRepCoEdge.partner Property

Parent Object: [BRepCoEdge](BRepCoEdge.md)  

## Description

Returns the co-edge on the adjacent face

## Syntax

"bRepCoEdge_var" is a variable referencing a BRepCoEdge object.  

```python
# Get the value of the property.
propertyValue = bRepCoEdge_var.partner
```

## Property Value

This is a read only property whose value is a [BRepCoEdge](BRepCoEdge.md).

## Version

Introduced in version August 2014
