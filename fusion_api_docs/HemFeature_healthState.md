# HemFeature.healthState Property

Parent Object: [HemFeature](HemFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"hemFeature_var" is a variable referencing a HemFeature object.  

```python
# Get the value of the property.
propertyValue = hemFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version September 2025
