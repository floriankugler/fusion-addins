# Circle3D.copy Method

Parent Object: [Circle3D](Circle3D.md)  

## Description

Creates and returns an independent copy of this Circle3D object.

## Syntax

"circle3D_var" is a variable referencing a [Circle3D](Circle3D.md) object.

```python
returnValue = circle3D_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [Circle3D](Circle3D.md) | Returns an independent copy of this Circle3D object. |

## Version

Introduced in version August 2014
