# RotaryMachineAxis.hasLimits Property

Parent Object: [RotaryMachineAxis](RotaryMachineAxis.md)  

## Description

Does this axis have a limited range of motion.

## Syntax

"rotaryMachineAxis_var" is a variable referencing a RotaryMachineAxis object.  

```python
# Get the value of the property.
propertyValue = rotaryMachineAxis_var.hasLimits
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version April 2023
