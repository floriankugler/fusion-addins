# RotaryMachineAxisInput.axisType Property

Parent Object: [RotaryMachineAxisInput](RotaryMachineAxisInput.md)  

## Description

The type of axis. This axis type determines which parameters of this object are valid to be accessed or modified.

## Syntax

"rotaryMachineAxisInput_var" is a variable referencing a RotaryMachineAxisInput object.  

```python
# Get the value of the property.
propertyValue = rotaryMachineAxisInput_var.axisType
```

## Property Value

This is a read only property whose value is a [MachineAxisTypes](MachineAxisTypes.md).

## Version

Introduced in version April 2023
