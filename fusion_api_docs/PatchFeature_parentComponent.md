# PatchFeature.parentComponent Property

Parent Object: [PatchFeature](PatchFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"patchFeature_var" is a variable referencing a PatchFeature object.  

```python
# Get the value of the property.
propertyValue = patchFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version September 2015
