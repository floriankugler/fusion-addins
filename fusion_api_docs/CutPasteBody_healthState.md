# CutPasteBody.healthState Property

Parent Object: [CutPasteBody](CutPasteBody.md)  

## Description

Returns the current health state of the feature.

## Syntax

"cutPasteBody_var" is a variable referencing a CutPasteBody object.  

```python
# Get the value of the property.
propertyValue = cutPasteBody_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version June 2017
