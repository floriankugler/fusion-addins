# SplitBodyFeatureInput.splitBodies Property

Parent Object: [SplitBodyFeatureInput](SplitBodyFeatureInput.md)  

## Description

Gets and sets the input solid or open bodies to be split. This can be a single BRepBody or an ObjectCollection if multiple bodies are to be split.

## Syntax

"splitBodyFeatureInput_var" is a variable referencing a SplitBodyFeatureInput object.  

```python
# Get the value of the property.
propertyValue = splitBodyFeatureInput_var.splitBodies

# Set the value of the property.
splitBodyFeatureInput_var.splitBodies = propertyValue
```

## Property Value

This is a read/write property whose value is a [Base](Base.md).

## Version

Introduced in version June 2015
