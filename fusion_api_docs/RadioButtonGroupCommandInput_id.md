# RadioButtonGroupCommandInput.id Property

Parent Object: [RadioButtonGroupCommandInput](RadioButtonGroupCommandInput.md)  

## Description

Gets the unique identifier for this input in the command's CommandInputs.

## Syntax

"radioButtonGroupCommandInput_var" is a variable referencing a RadioButtonGroupCommandInput object.  

```python
# Get the value of the property.
propertyValue = radioButtonGroupCommandInput_var.id
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version July 2015
