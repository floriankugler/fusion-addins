# UnstitchFeature.isSuppressed Property

Parent Object: [UnstitchFeature](UnstitchFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"unstitchFeature_var" is a variable referencing a UnstitchFeature object.  

```python
# Get the value of the property.
propertyValue = unstitchFeature_var.isSuppressed

# Set the value of the property.
unstitchFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version July 2015
