# Document.documentReferences Property

Parent Object: [Document](Document.md)  

## Description

Returns a collection containing the documents directly referenced by this document.

## Syntax

"document_var" is a variable referencing a Document object.  

```python
# Get the value of the property.
propertyValue = document_var.documentReferences
```

## Property Value

This is a read only property whose value is a [DocumentReferences](DocumentReferences.md).

## Version

Introduced in version June 2017
