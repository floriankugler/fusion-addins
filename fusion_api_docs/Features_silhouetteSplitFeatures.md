# Features.silhouetteSplitFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the Parting Line Split features within the component and supports the creation of new Parting Line Split features

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.silhouetteSplitFeatures
```

## Property Value

This is a read only property whose value is a [SilhouetteSplitFeatures](SilhouetteSplitFeatures.md).
