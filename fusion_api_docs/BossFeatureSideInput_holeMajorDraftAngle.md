# BossFeatureSideInput.holeMajorDraftAngle Property

Parent Object: [BossFeatureSideInput](BossFeatureSideInput.md)  

## Description

Get or set major hole draft angle for counterbore and countersink hole. This input is ignored for blank boss or boss with simple hole.

## Syntax

"bossFeatureSideInput_var" is a variable referencing a BossFeatureSideInput object.  

```python
# Get the value of the property.
propertyValue = bossFeatureSideInput_var.holeMajorDraftAngle

# Set the value of the property.
bossFeatureSideInput_var.holeMajorDraftAngle = propertyValue
```

## Property Value

This is a read/write property whose value is a [ValueInput](ValueInput.md).
