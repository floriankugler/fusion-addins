# ToolLibrary.add Method

Parent Object: [ToolLibrary](ToolLibrary.md)  

## Description

Inserts a Tool at the end of the ToolLibrary.

## Syntax

"toolLibrary_var" is a variable referencing a [ToolLibrary](ToolLibrary.md) object.

```python
returnValue = toolLibrary_var.add(tool)
```

## Return Value

| Type    | Description                                            |
|---------|--------------------------------------------------------|
| boolean | Returns true for successful insertion, false otherwise |

## Parameters

| Name | Type             | Description                    |
|------|------------------|--------------------------------|
| tool | [Tool](Tool.md) | The Tool that should be added. |

## Version

Introduced in version April 2023
