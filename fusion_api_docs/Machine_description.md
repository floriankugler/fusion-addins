# Machine.description Property

Parent Object: [Machine](Machine.md)  

## Description

Gets and sets the description of the machine.

## Syntax

"machine_var" is a variable referencing a Machine object.  

```python
# Get the value of the property.
propertyValue = machine_var.description

# Set the value of the property.
machine_var.description = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version April 2023
