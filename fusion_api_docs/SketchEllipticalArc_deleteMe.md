# SketchEllipticalArc.deleteMe Method

Parent Object: [SketchEllipticalArc](SketchEllipticalArc.md)  

## Description

Deletes the entity from the sketch.

## Syntax

"sketchEllipticalArc_var" is a variable referencing a [SketchEllipticalArc](SketchEllipticalArc.md) object.

```python
returnValue = sketchEllipticalArc_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true is the delete was successful. |

## Version

Introduced in version August 2014
