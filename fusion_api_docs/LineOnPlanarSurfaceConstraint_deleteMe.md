# LineOnPlanarSurfaceConstraint.deleteMe Method

Parent Object: [LineOnPlanarSurfaceConstraint](LineOnPlanarSurfaceConstraint.md)  

## Description

Deletes this constraint. The IsDeletable property can be used to determine if this constraint can be deleted.

## Syntax

"lineOnPlanarSurfaceConstraint_var" is a variable referencing a [LineOnPlanarSurfaceConstraint](LineOnPlanarSurfaceConstraint.md) object.

```python
returnValue = lineOnPlanarSurfaceConstraint_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version September 2023
