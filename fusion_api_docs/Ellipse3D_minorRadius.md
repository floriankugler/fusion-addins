# Ellipse3D.minorRadius Property

Parent Object: [Ellipse3D](Ellipse3D.md)  

## Description

Gets and sets the minor radius of the ellipse.

## Syntax

"ellipse3D_var" is a variable referencing an Ellipse3D object.  

```python
# Get the value of the property.
propertyValue = ellipse3D_var.minorRadius

# Set the value of the property.
ellipse3D_var.minorRadius = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version August 2014
