# CAMAdditiveContainer.strategy Property

Parent Object: [CAMAdditiveContainer](CAMAdditiveContainer.md)  

## Description

Gets the name of the strategy associated with this operation.

## Syntax

"cAMAdditiveContainer_var" is a variable referencing a CAMAdditiveContainer object.  

```python
# Get the value of the property.
propertyValue = cAMAdditiveContainer_var.strategy
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version July 2024
