# ArrangeProfileOrFaceResultEnvelope.occurrences Property

Parent Object: [ArrangeProfileOrFaceResultEnvelope](ArrangeProfileOrFaceResultEnvelope.md)  

## Description

Returns a collection object of the occurrences in this envelope.

## Syntax

"arrangeProfileOrFaceResultEnvelope_var" is a variable referencing an ArrangeProfileOrFaceResultEnvelope object.  

```python
# Get the value of the property.
propertyValue = arrangeProfileOrFaceResultEnvelope_var.occurrences
```

## Property Value

This is a read only property whose value is an [ArrangeOccurrenceResults](ArrangeOccurrenceResults.md).

## Version

Introduced in version January 2025
