# ArrangePlaneEnvelopeDefinition.envelopeSpacing Property

Parent Object: [ArrangePlaneEnvelopeDefinition](ArrangePlaneEnvelopeDefinition.md)  

## Description

Returns the parameter that controls the offset distance between envelopes when there is more than one. You can modify the value by using the properties on the returned ModelParameter object.

## Syntax

"arrangePlaneEnvelopeDefinition_var" is a variable referencing an ArrangePlaneEnvelopeDefinition object.  

```python
# Get the value of the property.
propertyValue = arrangePlaneEnvelopeDefinition_var.envelopeSpacing
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version January 2025
