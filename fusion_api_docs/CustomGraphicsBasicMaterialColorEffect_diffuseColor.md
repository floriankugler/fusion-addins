# CustomGraphicsBasicMaterialColorEffect.diffuseColor Property

Parent Object: [CustomGraphicsBasicMaterialColorEffect](CustomGraphicsBasicMaterialColorEffect.md)  

## Description

Gets and sets the diffuseColor associated with this CustomGraphicsBasicMaterialColorEffect object. The diffuse color is the color of reflected light as it scatters off of a rough surface.

## Syntax

"customGraphicsBasicMaterialColorEffect_var" is a variable referencing a CustomGraphicsBasicMaterialColorEffect object.  

```python
# Get the value of the property.
propertyValue = customGraphicsBasicMaterialColorEffect_var.diffuseColor

# Set the value of the property.
customGraphicsBasicMaterialColorEffect_var.diffuseColor = propertyValue
```

## Property Value

This is a read/write property whose value is a [Color](Color.md).

## Version

Introduced in version September 2017
