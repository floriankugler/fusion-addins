# Sketch.origin Property

Parent Object: [Sketch](Sketch.md)  

## Description

Returns the origin point of the sketch in model space.

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.origin
```

## Property Value

This is a read only property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
