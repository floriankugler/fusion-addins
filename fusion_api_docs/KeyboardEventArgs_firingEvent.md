# KeyboardEventArgs.firingEvent Property

Parent Object: [KeyboardEventArgs](KeyboardEventArgs.md)  

## Description

The event that the firing is in response to.

## Syntax

"keyboardEventArgs_var" is a variable referencing a KeyboardEventArgs object.  

```python
# Get the value of the property.
propertyValue = keyboardEventArgs_var.firingEvent
```

## Property Value

This is a read only property whose value is an [Event](Event.md).

## Version

Introduced in version August 2014
