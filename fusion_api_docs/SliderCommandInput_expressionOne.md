# SliderCommandInput.expressionOne Property

Parent Object: [SliderCommandInput](SliderCommandInput.md)  

## Description

Uses an expression to set the value in the first input field. This can contain equations and is evaluated using the specified unit type.

## Syntax

"sliderCommandInput_var" is a variable referencing a SliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = sliderCommandInput_var.expressionOne

# Set the value of the property.
sliderCommandInput_var.expressionOne = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version June 2015
