# RemoveFeature.parentComponent Property

Parent Object: [RemoveFeature](RemoveFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"removeFeature_var" is a variable referencing a RemoveFeature object.  

```python
# Get the value of the property.
propertyValue = removeFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version September 2015
