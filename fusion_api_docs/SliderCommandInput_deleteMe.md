# SliderCommandInput.deleteMe Method

Parent Object: [SliderCommandInput](SliderCommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"sliderCommandInput_var" is a variable referencing a [SliderCommandInput](SliderCommandInput.md) object.

```python
returnValue = sliderCommandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2020
