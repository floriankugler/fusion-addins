# UserInterface.activeToolbar Property

Parent Object: [UserInterface](UserInterface.md)  

## Description

Retrieve the active Toolbar being displayed in the user interface.

## Syntax

"userInterface_var" is a variable referencing a UserInterface object.  

```python
# Get the value of the property.
propertyValue = userInterface_var.activeToolbar
```

## Property Value

This is a read only property whose value is a [Toolbar](Toolbar.md).

## Version

Introduced in version May 2024
