# UnitAndValuePreferences.isScientificNotationUsed Property

Parent Object: [UnitAndValuePreferences](UnitAndValuePreferences.md)  

## Description

Gets and sets if scientific notation is used when displaying numbers.

## Syntax

"unitAndValuePreferences_var" is a variable referencing a UnitAndValuePreferences object.  

```python
# Get the value of the property.
propertyValue = unitAndValuePreferences_var.isScientificNotationUsed

# Set the value of the property.
unitAndValuePreferences_var.isScientificNotationUsed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
