# BRepLoop.face Property

Parent Object: [BRepLoop](BRepLoop.md)  

## Description

Returns the parent face of the loop.

## Syntax

"bRepLoop_var" is a variable referencing a BRepLoop object.  

```python
# Get the value of the property.
propertyValue = bRepLoop_var.face
```

## Property Value

This is a read only property whose value is a [BRepFace](BRepFace.md).

## Version

Introduced in version August 2014
