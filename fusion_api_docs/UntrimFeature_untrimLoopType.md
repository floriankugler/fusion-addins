# UntrimFeature.untrimLoopType Property

Parent Object: [UntrimFeature](UntrimFeature.md)  

## Description

Gets the loop type that was untrimmed. To change the trim type, use one of the redefine methods.

## Syntax

"untrimFeature_var" is a variable referencing a UntrimFeature object.  

```python
# Get the value of the property.
propertyValue = untrimFeature_var.untrimLoopType
```

## Property Value

This is a read only property whose value is a [UntrimLoopTypes](UntrimLoopTypes.md).

## Version

Introduced in version January 2021
