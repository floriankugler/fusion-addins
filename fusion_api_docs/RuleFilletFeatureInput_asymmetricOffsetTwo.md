# RuleFilletFeatureInput.asymmetricOffsetTwo Property

Parent Object: [RuleFilletFeatureInput](RuleFilletFeatureInput.md)  

## Description

Returns the current second offset for an asymmetric fillet. Use setAsymmetricOffsets to set the offsets. Returns null in the case where the fillet is a constant radius fillet.

## Syntax

"ruleFilletFeatureInput_var" is a variable referencing a RuleFilletFeatureInput object.  

```python
# Get the value of the property.
propertyValue = ruleFilletFeatureInput_var.asymmetricOffsetTwo
```

## Property Value

This is a read only property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version November 2025
