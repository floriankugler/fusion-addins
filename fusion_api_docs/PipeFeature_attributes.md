# PipeFeature.attributes Property

Parent Object: [PipeFeature](PipeFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"pipeFeature_var" is a variable referencing a PipeFeature object.  

```python
# Get the value of the property.
propertyValue = pipeFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
