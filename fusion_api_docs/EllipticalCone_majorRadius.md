# EllipticalCone.majorRadius Property

Parent Object: [EllipticalCone](EllipticalCone.md)  

## Description

Gets and sets the major radius of the ellipse that defines the cone.

## Syntax

"ellipticalCone_var" is a variable referencing an EllipticalCone object.  

```python
# Get the value of the property.
propertyValue = ellipticalCone_var.majorRadius

# Set the value of the property.
ellipticalCone_var.majorRadius = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version August 2014
