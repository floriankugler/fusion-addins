# SketchPoint.deleteMe Method

Parent Object: [SketchPoint](SketchPoint.md)  

## Description

Deletes the entity from the sketch.

## Syntax

"sketchPoint_var" is a variable referencing a [SketchPoint](SketchPoint.md) object.

```python
returnValue = sketchPoint_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true is the delete was successful. |

## Version

Introduced in version August 2014
