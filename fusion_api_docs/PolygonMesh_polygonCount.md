# PolygonMesh.polygonCount Property

Parent Object: [PolygonMesh](PolygonMesh.md)  

## Description

Returns the number of polygons (more than 4 sides) in the mesh.

## Syntax

"polygonMesh_var" is a variable referencing a PolygonMesh object.  

```python
# Get the value of the property.
propertyValue = polygonMesh_var.polygonCount
```

## Property Value

This is a read only property whose value is an integer.

## Version

Introduced in version August 2014
