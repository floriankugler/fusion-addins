# LinearMachineAxisInput.name Property

Parent Object: [LinearMachineAxisInput](LinearMachineAxisInput.md)  

## Description

The user facing name of this axis.

## Syntax

"linearMachineAxisInput_var" is a variable referencing a LinearMachineAxisInput object.  

```python
# Get the value of the property.
propertyValue = linearMachineAxisInput_var.name

# Set the value of the property.
linearMachineAxisInput_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version April 2023
