# FlatPatternProduct.contactSets Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

Returns the contact sets associated with this design.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.contactSets
```

## Property Value

This is a read only property whose value is a [ContactSets](ContactSets.md).

## Version

Introduced in version October 2022
