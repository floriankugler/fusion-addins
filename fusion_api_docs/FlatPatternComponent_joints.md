# FlatPatternComponent.joints Property

Parent Object: [FlatPatternComponent](FlatPatternComponent.md)  

## Description

Returns the collection of joints associated with this component.

## Syntax

"flatPatternComponent_var" is a variable referencing a FlatPatternComponent object.  

```python
# Get the value of the property.
propertyValue = flatPatternComponent_var.joints
```

## Property Value

This is a read only property whose value is a [Joints](Joints.md).

## Version

Introduced in version October 2022
