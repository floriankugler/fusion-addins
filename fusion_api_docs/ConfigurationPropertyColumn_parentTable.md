# ConfigurationPropertyColumn.parentTable Property

Parent Object: [ConfigurationPropertyColumn](ConfigurationPropertyColumn.md)  

## Description

Returns the parent table this column is in.

## Syntax

"configurationPropertyColumn_var" is a variable referencing a ConfigurationPropertyColumn object.  

```python
# Get the value of the property.
propertyValue = configurationPropertyColumn_var.parentTable
```

## Property Value

This is a read only property whose value is a [ConfigurationTable](ConfigurationTable.md).

## Version

Introduced in version March 2024
