# FlatPatternProduct.configurationTopTable Property

Parent Object: [FlatPatternProduct](FlatPatternProduct.md)  

## Description

If this design is a configured design or a configuration, this property returns the associated ConfigurationTopTable object. If this is not a configured design or configuration, this property returns null.

## Syntax

"flatPatternProduct_var" is a variable referencing a FlatPatternProduct object.  

```python
# Get the value of the property.
propertyValue = flatPatternProduct_var.configurationTopTable
```

## Property Value

This is a read only property whose value is a [ConfigurationTopTable](ConfigurationTopTable.md).

## Version

Introduced in version January 2024
