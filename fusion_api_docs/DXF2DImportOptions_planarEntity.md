# DXF2DImportOptions.planarEntity Property

Parent Object: [DXF2DImportOptions](DXF2DImportOptions.md)  

## Description

Gets and sets the construction plane or planar face that defines the plane that the resulting sketches will be created on.

## Syntax

"dXF2DImportOptions_var" is a variable referencing a DXF2DImportOptions object.  

```python
# Get the value of the property.
propertyValue = dXF2DImportOptions_var.planarEntity

# Set the value of the property.
dXF2DImportOptions_var.planarEntity = propertyValue
```

## Property Value

This is a read/write property whose value is a [Base](Base.md).

## Version

Introduced in version November 2015
