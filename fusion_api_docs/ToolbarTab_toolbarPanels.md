# ToolbarTab.toolbarPanels Property

Parent Object: [ToolbarTab](ToolbarTab.md)  

## Description

Gets the collection containing the panels associated with this tab. It's through this collection that you can add new toolbar panels.

## Syntax

"toolbarTab_var" is a variable referencing a ToolbarTab object.  

```python
# Get the value of the property.
propertyValue = toolbarTab_var.toolbarPanels
```

## Property Value

This is a read only property whose value is a [ToolbarPanels](ToolbarPanels.md).
