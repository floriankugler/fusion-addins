# ConstructionPlaneDefinition.parentConstructionPlane Property

Parent Object: [ConstructionPlaneDefinition](ConstructionPlaneDefinition.md)  

## Description

Returns the ConstructionPlane object

## Syntax

"constructionPlaneDefinition_var" is a variable referencing a ConstructionPlaneDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionPlaneDefinition_var.parentConstructionPlane
```

## Property Value

This is a read only property whose value is a [ConstructionPlane](ConstructionPlane.md).

## Version

Introduced in version August 2014
