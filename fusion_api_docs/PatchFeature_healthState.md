# PatchFeature.healthState Property

Parent Object: [PatchFeature](PatchFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"patchFeature_var" is a variable referencing a PatchFeature object.  

```python
# Get the value of the property.
propertyValue = patchFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
