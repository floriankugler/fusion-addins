# ConcentricConstraint.entityTwo Property

Parent Object: [ConcentricConstraint](ConcentricConstraint.md)  

## Description

Returns the second curve.

## Syntax

"concentricConstraint_var" is a variable referencing a ConcentricConstraint object.  

```python
# Get the value of the property.
propertyValue = concentricConstraint_var.entityTwo
```

## Property Value

This is a read only property whose value is a [SketchCurve](SketchCurve.md).

## Version

Introduced in version August 2014
