# EllipticalCone.copy Method

Parent Object: [EllipticalCone](EllipticalCone.md)  

## Description

Creates and returns an independent copy of this EllipticalCone object.

## Syntax

"ellipticalCone_var" is a variable referencing an [EllipticalCone](EllipticalCone.md) object.

```python
returnValue = ellipticalCone_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [EllipticalCone](EllipticalCone.md) | Returns a new EllipticalCone object that is a copy of this EllipticalCone object. |

## Version

Introduced in version August 2014
