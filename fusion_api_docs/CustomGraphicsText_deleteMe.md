# CustomGraphicsText.deleteMe Method

Parent Object: [CustomGraphicsText](CustomGraphicsText.md)  

## Description

Deletes the entity from the custom graphics group.

## Syntax

"customGraphicsText_var" is a variable referencing a [CustomGraphicsText](CustomGraphicsText.md) object.

```python
returnValue = customGraphicsText_var.deleteMe()
```

## Return Value

| Type    | Description                                  |
|---------|----------------------------------------------|
| boolean | Returns true if the deletion was successful. |

## Version

Introduced in version September 2017
