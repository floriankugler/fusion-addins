# MinimumRadiusAnalyses.item Method

Parent Object: [MinimumRadiusAnalyses](MinimumRadiusAnalyses.md)  

## Description

A method that returns the specified MinimumRadiusAnalysis object using an index into the collection.

## Syntax

"minimumRadiusAnalyses_var" is a variable referencing a [MinimumRadiusAnalyses](MinimumRadiusAnalyses.md) object.

```python
returnValue = minimumRadiusAnalyses_var.item(index)
```

## Return Value

| Type | Description |
|----|----|
| [MinimumRadiusAnalysis](MinimumRadiusAnalysis.md) | Returns the specified item or null if an invalid index was specified. |

## Parameters

| Name | Type | Description |
|----|----|----|
| index | uinteger | The index of the item within the collection to return. The first item in the collection has an index of 0. |

## Version

Introduced in version January 2023
