# CAMExportFuture.error Property

Parent Object: [CAMExportFuture](CAMExportFuture.md)  

## Description

Gets the last encountered error message generated on the export thread.

## Syntax

"cAMExportFuture_var" is a variable referencing a CAMExportFuture object.  

```python
# Get the value of the property.
propertyValue = cAMExportFuture_var.error
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version September 2024
