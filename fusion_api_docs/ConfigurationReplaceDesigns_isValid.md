# ConfigurationReplaceDesigns.isValid Property

Parent Object: [ConfigurationReplaceDesigns](ConfigurationReplaceDesigns.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"configurationReplaceDesigns_var" is a variable referencing a ConfigurationReplaceDesigns object.  

```python
# Get the value of the property.
propertyValue = configurationReplaceDesigns_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version September 2025
