# Arrange2DPlaneEnvelopeInput.isFlipped Property

Parent Object: [Arrange2DPlaneEnvelopeInput](Arrange2DPlaneEnvelopeInput.md)  

## Description

Specifies if the arrangement of objects is so they are above or X-Y plane of the envelope. Defaults to false so the objects are above the construction plane, profile or face.

## Syntax

"arrange2DPlaneEnvelopeInput_var" is a variable referencing an Arrange2DPlaneEnvelopeInput object.  

```python
# Get the value of the property.
propertyValue = arrange2DPlaneEnvelopeInput_var.isFlipped

# Set the value of the property.
arrange2DPlaneEnvelopeInput_var.isFlipped = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2025
