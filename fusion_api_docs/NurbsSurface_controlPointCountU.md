# NurbsSurface.controlPointCountU Property

Parent Object: [NurbsSurface](NurbsSurface.md)  

## Description

Gets the number of control points in the U direction.

## Syntax

"nurbsSurface_var" is a variable referencing a NurbsSurface object.  

```python
# Get the value of the property.
propertyValue = nurbsSurface_var.controlPointCountU
```

## Property Value

This is a read only property whose value is an integer.

## Version

Introduced in version August 2014
