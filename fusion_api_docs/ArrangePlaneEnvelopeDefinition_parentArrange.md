# ArrangePlaneEnvelopeDefinition.parentArrange Property

Parent Object: [ArrangePlaneEnvelopeDefinition](ArrangePlaneEnvelopeDefinition.md)  

## Description

Returns the parent ArrangeFeature this envelope is associated with.

## Syntax

"arrangePlaneEnvelopeDefinition_var" is a variable referencing an ArrangePlaneEnvelopeDefinition object.  

```python
# Get the value of the property.
propertyValue = arrangePlaneEnvelopeDefinition_var.parentArrange
```

## Property Value

This is a read only property whose value is an [ArrangeFeature](ArrangeFeature.md).

## Version

Introduced in version January 2025
