# BRepEdge.boundingBox Property

Parent Object: [BRepEdge](BRepEdge.md)  

## Description

Returns the bounding box of this edge.

## Syntax

"bRepEdge_var" is a variable referencing a BRepEdge object.  

```python
# Get the value of the property.
propertyValue = bRepEdge_var.boundingBox
```

## Property Value

This is a read only property whose value is a [BoundingBox3D](BoundingBox3D.md).

## Version

Introduced in version September 2017
