# Plane.origin Property

Parent Object: [Plane](Plane.md)  

## Description

Gets and sets the origin point of the plane.

## Syntax

"plane_var" is a variable referencing a Plane object.  

```python
# Get the value of the property.
propertyValue = plane_var.origin

# Set the value of the property.
plane_var.origin = propertyValue
```

## Property Value

This is a read/write property whose value is a [Point3D](Point3D.md).

## Version

Introduced in version August 2014
