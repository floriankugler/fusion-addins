# OperationBase.parameters Property

Parent Object: [OperationBase](OperationBase.md)  

## Description

Gets the CAMParameters collection for this operation.

## Syntax

"operationBase_var" is a variable referencing an OperationBase object.  

```python
# Get the value of the property.
propertyValue = operationBase_var.parameters
```

## Property Value

This is a read only property whose value is a [CAMParameters](CAMParameters.md).

## Version

Introduced in version September 2020
