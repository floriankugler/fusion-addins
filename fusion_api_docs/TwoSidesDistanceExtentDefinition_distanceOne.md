# TwoSidesDistanceExtentDefinition.distanceOne Property

Parent Object: [TwoSidesDistanceExtentDefinition](TwoSidesDistanceExtentDefinition.md)  

## Description

Gets the ModelParameter that defines the first distance

## Syntax

"twoSidesDistanceExtentDefinition_var" is a variable referencing a TwoSidesDistanceExtentDefinition object.  

```python
# Get the value of the property.
propertyValue = twoSidesDistanceExtentDefinition_var.distanceOne
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version March 2022
