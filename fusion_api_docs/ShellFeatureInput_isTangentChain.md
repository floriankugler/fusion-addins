# ShellFeatureInput.isTangentChain Property

Parent Object: [ShellFeatureInput](ShellFeatureInput.md)  

## Description

Gets and sets if any faces that are tangentially connected to any of the input faces will also be included in setting InputEntities. It defaults to true.

## Syntax

"shellFeatureInput_var" is a variable referencing a ShellFeatureInput object.  

```python
# Get the value of the property.
propertyValue = shellFeatureInput_var.isTangentChain

# Set the value of the property.
shellFeatureInput_var.isTangentChain = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version November 2014
