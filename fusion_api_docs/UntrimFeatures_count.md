# UntrimFeatures.count Property

Parent Object: [UntrimFeatures](UntrimFeatures.md)  

## Description

The number of Untrim features in the collection.

## Syntax

"untrimFeatures_var" is a variable referencing a UntrimFeatures object.  

```python
# Get the value of the property.
propertyValue = untrimFeatures_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version January 2021
