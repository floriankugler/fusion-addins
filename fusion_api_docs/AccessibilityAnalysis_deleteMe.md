# AccessibilityAnalysis.deleteMe Method

Parent Object: [AccessibilityAnalysis](AccessibilityAnalysis.md)  

## Description

A method that deletes this Analysis.

## Syntax

"accessibilityAnalysis_var" is a variable referencing an [AccessibilityAnalysis](AccessibilityAnalysis.md) object.

```python
returnValue = accessibilityAnalysis_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version January 2023
