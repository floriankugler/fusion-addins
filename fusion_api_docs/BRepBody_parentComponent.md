# BRepBody.parentComponent Property

Parent Object: [BRepBody](BRepBody.md)  

## Description

Returns the component this body is owned by.

## Syntax

"bRepBody_var" is a variable referencing a BRepBody object.  

```python
# Get the value of the property.
propertyValue = bRepBody_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version August 2014
