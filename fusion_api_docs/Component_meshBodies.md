# Component.meshBodies Property

Parent Object: [Component](Component.md)  

## Description

Returns the mesh bodies collection associated with this component.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.meshBodies
```

## Property Value

This is a read only property whose value is a [MeshBodies](MeshBodies.md).

## Version

Introduced in version August 2014
