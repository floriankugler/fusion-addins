# Point3D.copy Method

Parent Object: [Point3D](Point3D.md)  

## Description

Creates and returns a copy of this point object.

## Syntax

"point3D_var" is a variable referencing a [Point3D](Point3D.md) object.

```python
returnValue = point3D_var.copy()
```

## Return Value

| Type                   | Description                                       |
|------------------------|---------------------------------------------------|
| [Point3D](Point3D.md) | Returns an independent copy of this point object. |

## Version

Introduced in version August 2014
