# SymmetryConstraint.symmetryLine Property

Parent Object: [SymmetryConstraint](SymmetryConstraint.md)  

## Description

Returns the axis (SketchLine) that defines the symmetry.

## Syntax

"symmetryConstraint_var" is a variable referencing a SymmetryConstraint object.  

```python
# Get the value of the property.
propertyValue = symmetryConstraint_var.symmetryLine
```

## Property Value

This is a read only property whose value is a [SketchLine](SketchLine.md).

## Version

Introduced in version August 2014
