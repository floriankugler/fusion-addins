# MachineQuery.execute Method

Parent Object: [MachineQuery](MachineQuery.md)  

## Description

Executes the query for specific machines based on the query's properties.

## Syntax

"machineQuery_var" is a variable referencing a [MachineQuery](MachineQuery.md) object.

```python
returnValue = machineQuery_var.execute()
```

## Return Value

| Type | Description |
|----|----|
| [Machine](Machine.md)\[\] | Returns a list of \`Machine\`. Each returned machine matches this query. |

## Version

Introduced in version May 2023
