# SketchEllipticalArc.length Property

Parent Object: [SketchEllipticalArc](SketchEllipticalArc.md)  

## Description

Returns the length of the curve in centimeters.

## Syntax

"sketchEllipticalArc_var" is a variable referencing a SketchEllipticalArc object.  

```python
# Get the value of the property.
propertyValue = sketchEllipticalArc_var.length
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version August 2014
