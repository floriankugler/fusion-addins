# ScaleFeature.healthState Property

Parent Object: [ScaleFeature](ScaleFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"scaleFeature_var" is a variable referencing a ScaleFeature object.  

```python
# Get the value of the property.
propertyValue = scaleFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version July 2016
