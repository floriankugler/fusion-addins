# SplitButtonControl.defaultCommandDefinition Property

Parent Object: [SplitButtonControl](SplitButtonControl.md)  

## Description

Gets the command definition that is used as the default command on the main portion of the split button.

## Syntax

"splitButtonControl_var" is a variable referencing a SplitButtonControl object.  

```python
# Get the value of the property.
propertyValue = splitButtonControl_var.defaultCommandDefinition
```

## Property Value

This is a read only property whose value is a [CommandDefinition](CommandDefinition.md).
