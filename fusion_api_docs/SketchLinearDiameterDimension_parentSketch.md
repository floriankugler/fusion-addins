# SketchLinearDiameterDimension.parentSketch Property

Parent Object: [SketchLinearDiameterDimension](SketchLinearDiameterDimension.md)  

## Description

Returns the parent sketch object.

## Syntax

"sketchLinearDiameterDimension_var" is a variable referencing a SketchLinearDiameterDimension object.  

```python
# Get the value of the property.
propertyValue = sketchLinearDiameterDimension_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version September 2022
