# RipFeature.definition Property

Parent Object: [RipFeature](RipFeature.md)  

## Description

Returns the RipFeatureDefinition object which provides access to the information defining this RipFeature and the ability to edit it.

## Syntax

"ripFeature_var" is a variable referencing a RipFeature object.  

```python
# Get the value of the property.
propertyValue = ripFeature_var.definition
```

## Property Value

This is a read only property whose value is a [RipFeatureDefinition](RipFeatureDefinition.md).

## Version

Introduced in version September 2023
