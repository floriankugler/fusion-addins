# Application.userName Property

Parent Object: [Application](Application.md)  

## Description

Returns the user name of the Autodesk account currently logged in.

## Syntax

"application_var" is a variable referencing an Application object.  

```python
# Get the value of the property.
propertyValue = application_var.userName
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version September 2015
