# Features.ripFeatures Property

Parent Object: [Features](Features.md)  

## Description

Returns the collection that provides access to the existing Rip features.

## Syntax

"features_var" is a variable referencing a Features object.  

```python
# Get the value of the property.
propertyValue = features_var.ripFeatures
```

## Property Value

This is a read only property whose value is a [RipFeatures](RipFeatures.md).
