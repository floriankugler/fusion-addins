# BooleanProperty.isReadOnly Property

Parent Object: [BooleanProperty](BooleanProperty.md)  

## Description

Indicates if this property is read-only. If True any attempted edits will fail.

## Syntax

"booleanProperty_var" is a variable referencing a BooleanProperty object.  

```python
# Get the value of the property.
propertyValue = booleanProperty_var.isReadOnly
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
