# CurvatureMapAnalysis.isValid Property

Parent Object: [CurvatureMapAnalysis](CurvatureMapAnalysis.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"curvatureMapAnalysis_var" is a variable referencing a CurvatureMapAnalysis object.  

```python
# Get the value of the property.
propertyValue = curvatureMapAnalysis_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2023
