# Analysis.deleteMe Method

Parent Object: [Analysis](Analysis.md)  

## Description

A method that deletes this Analysis.

## Syntax

"analysis_var" is a variable referencing an [Analysis](Analysis.md) object.

```python
returnValue = analysis_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version January 2023
