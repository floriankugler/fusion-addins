# SketchText.fontName Property

Parent Object: [SketchText](SketchText.md)  

## Description

Gets and sets the name of the font to use.

## Syntax

"sketchText_var" is a variable referencing a SketchText object.  

```python
# Get the value of the property.
propertyValue = sketchText_var.fontName

# Set the value of the property.
sketchText_var.fontName = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version March 2015
