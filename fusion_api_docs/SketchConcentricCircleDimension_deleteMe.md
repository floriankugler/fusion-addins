# SketchConcentricCircleDimension.deleteMe Method

Parent Object: [SketchConcentricCircleDimension](SketchConcentricCircleDimension.md)  

## Description

Deletes this dimension. The IsDeletable property indicates if this dimension can be deleted.

## Syntax

"sketchConcentricCircleDimension_var" is a variable referencing a [SketchConcentricCircleDimension](SketchConcentricCircleDimension.md) object.

```python
returnValue = sketchConcentricCircleDimension_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
