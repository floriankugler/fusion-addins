# SliderJointMotion.jointType Property

Parent Object: [SliderJointMotion](SliderJointMotion.md)  

## Description

Returns an enum value indicating the type of joint this joint represents.

## Syntax

"sliderJointMotion_var" is a variable referencing a SliderJointMotion object.  

```python
# Get the value of the property.
propertyValue = sliderJointMotion_var.jointType
```

## Property Value

This is a read only property whose value is a [JointTypes](JointTypes.md).

## Version

Introduced in version July 2015
