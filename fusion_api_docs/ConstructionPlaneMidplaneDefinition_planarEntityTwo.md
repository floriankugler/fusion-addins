# ConstructionPlaneMidplaneDefinition.planarEntityTwo Property

Parent Object: [ConstructionPlaneMidplaneDefinition](ConstructionPlaneMidplaneDefinition.md)  

## Description

Gets the second planar face or construction plane that defines this ConstructionPlane.

## Syntax

"constructionPlaneMidplaneDefinition_var" is a variable referencing a ConstructionPlaneMidplaneDefinition object.  

```python
# Get the value of the property.
propertyValue = constructionPlaneMidplaneDefinition_var.planarEntityTwo
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version August 2014
