# BallJointMotion.pitchDirectionVector Property

Parent Object: [BallJointMotion](BallJointMotion.md)  

## Description

Returns the direction that the pitch angle is measured from. This property will return null in the case where the BallJointMotion object was obtained from a JointInput object.

## Syntax

"ballJointMotion_var" is a variable referencing a BallJointMotion object.  

```python
# Get the value of the property.
propertyValue = ballJointMotion_var.pitchDirectionVector
```

## Property Value

This is a read only property whose value is a [Vector3D](Vector3D.md).

## Version

Introduced in version July 2015
