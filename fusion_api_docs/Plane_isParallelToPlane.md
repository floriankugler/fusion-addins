# Plane.isParallelToPlane Method

Parent Object: [Plane](Plane.md)  

## Description

Checks if this plane is parallel to another plane.

## Syntax

"plane_var" is a variable referencing a [Plane](Plane.md) object.

```python
returnValue = plane_var.isParallelToPlane(plane)
```

## Return Value

| Type    | Description                              |
|---------|------------------------------------------|
| boolean | Returns true if the planes are parallel. |

## Parameters

| Name  | Type               | Description                                |
|-------|--------------------|--------------------------------------------|
| plane | [Plane](Plane.md) | The plane to compare with for parallelism. |

## Version

Introduced in version August 2014
