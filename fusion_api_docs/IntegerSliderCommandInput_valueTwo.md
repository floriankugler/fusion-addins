# IntegerSliderCommandInput.valueTwo Property

Parent Object: [IntegerSliderCommandInput](IntegerSliderCommandInput.md)  

## Description

If the hasTwoSliders property is true, this returns the value associated with the second slider.

## Syntax

"integerSliderCommandInput_var" is a variable referencing an IntegerSliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = integerSliderCommandInput_var.valueTwo

# Set the value of the property.
integerSliderCommandInput_var.valueTwo = propertyValue
```

## Property Value

This is a read/write property whose value is an integer.

## Version

Introduced in version June 2015
