# OperationBase.isSuppressed Property

Parent Object: [OperationBase](OperationBase.md)  

## Description

Gets and sets the "Suppressed" property value of the operation. Gets/sets true if the operation is suppressed.

## Syntax

"operationBase_var" is a variable referencing an OperationBase object.  

```python
# Get the value of the property.
propertyValue = operationBase_var.isSuppressed

# Set the value of the property.
operationBase_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2016
