# LoftFeature.timelineObject Property

Parent Object: [LoftFeature](LoftFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"loftFeature_var" is a variable referencing a LoftFeature object.  

```python
# Get the value of the property.
propertyValue = loftFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version September 2015
