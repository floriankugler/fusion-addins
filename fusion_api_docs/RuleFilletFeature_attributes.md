# RuleFilletFeature.attributes Property

Parent Object: [RuleFilletFeature](RuleFilletFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"ruleFilletFeature_var" is a variable referencing a RuleFilletFeature object.  

```python
# Get the value of the property.
propertyValue = ruleFilletFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
