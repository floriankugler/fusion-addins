# SketchCircle.attributes Property

Parent Object: [SketchCircle](SketchCircle.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"sketchCircle_var" is a variable referencing a SketchCircle object.  

```python
# Get the value of the property.
propertyValue = sketchCircle_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
