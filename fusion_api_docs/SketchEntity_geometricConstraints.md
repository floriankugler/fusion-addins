# SketchEntity.geometricConstraints Property

Parent Object: [SketchEntity](SketchEntity.md)  

## Description

Returns the sketch constraints that are attached to this curve.

## Syntax

"sketchEntity_var" is a variable referencing a SketchEntity object.  

```python
# Get the value of the property.
propertyValue = sketchEntity_var.geometricConstraints
```

## Property Value

This is a read only property whose value is a [GeometricConstraintList](GeometricConstraintList.md).

## Version

Introduced in version January 2015
