# BoundingBox2D.contains Method

Parent Object: [BoundingBox2D](BoundingBox2D.md)  

## Description

Determines if the specified point lies within the bounding box.

## Syntax

"boundingBox2D_var" is a variable referencing a [BoundingBox2D](BoundingBox2D.md) object.

```python
returnValue = boundingBox2D_var.contains(point)
```

## Return Value

| Type    | Description                                             |
|---------|---------------------------------------------------------|
| boolean | Returns true if the point lies within the bounding box. |

## Parameters

| Name  | Type                   | Description                         |
|-------|------------------------|-------------------------------------|
| point | [Point2D](Point2D.md) | The point to test containment with. |

## Version

Introduced in version August 2014
