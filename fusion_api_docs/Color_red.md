# Color.red Property

Parent Object: [Color](Color.md)  

## Description

Gets and sets the red component of the color. The value can be 0 to 255.

## Syntax

"color_var" is a variable referencing a Color object.  

```python
# Get the value of the property.
propertyValue = color_var.red

# Set the value of the property.
color_var.red = propertyValue
```

## Property Value

This is a read/write property whose value is a short.

## Version

Introduced in version August 2014
