# RectangularPatternConstraint.attributes Property

Parent Object: [RectangularPatternConstraint](RectangularPatternConstraint.md)  

## Description

Returns the collection of attributes associated with this geometric constraint.

## Syntax

"rectangularPatternConstraint_var" is a variable referencing a RectangularPatternConstraint object.  

```python
# Get the value of the property.
propertyValue = rectangularPatternConstraint_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
