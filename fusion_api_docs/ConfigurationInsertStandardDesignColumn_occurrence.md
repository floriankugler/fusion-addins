# ConfigurationInsertStandardDesignColumn.occurrence Property

Parent Object: [ConfigurationInsertStandardDesignColumn](ConfigurationInsertStandardDesignColumn.md)  

## Description

Returns the occurrence being controlled by this column.  

This property returns null when the table being queried was obtained from a DataFile object.

## Syntax

"configurationInsertStandardDesignColumn_var" is a variable referencing a ConfigurationInsertStandardDesignColumn object.  

```python
# Get the value of the property.
propertyValue = configurationInsertStandardDesignColumn_var.occurrence
```

## Property Value

This is a read only property whose value is an [Occurrence](Occurrence.md).

## Version

Introduced in version September 2025
