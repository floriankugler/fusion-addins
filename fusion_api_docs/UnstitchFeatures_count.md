# UnstitchFeatures.count Property

Parent Object: [UnstitchFeatures](UnstitchFeatures.md)  

## Description

The number of Unstitch features in the collection.

## Syntax

"unstitchFeatures_var" is a variable referencing a UnstitchFeatures object.  

```python
# Get the value of the property.
propertyValue = unstitchFeatures_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version July 2015
