# ConfigurationThemeColumn.referencedTable Property

Parent Object: [ConfigurationThemeColumn](ConfigurationThemeColumn.md)  

## Description

Returns the theme table that this column references.

## Syntax

"configurationThemeColumn_var" is a variable referencing a ConfigurationThemeColumn object.  

```python
# Get the value of the property.
propertyValue = configurationThemeColumn_var.referencedTable
```

## Property Value

This is a read only property whose value is a [ConfigurationTable](ConfigurationTable.md).

## Version

Introduced in version January 2024
