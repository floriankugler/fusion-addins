# GroupCommandInput.parentCommand Property

Parent Object: [GroupCommandInput](GroupCommandInput.md)  

## Description

Gets the parent Command.

## Syntax

"groupCommandInput_var" is a variable referencing a GroupCommandInput object.  

```python
# Get the value of the property.
propertyValue = groupCommandInput_var.parentCommand
```

## Property Value

This is a read only property whose value is a [Command](Command.md).

## Version

Introduced in version July 2015
