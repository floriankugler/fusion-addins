# IGESImportOptions.filename Property

Parent Object: [IGESImportOptions](IGESImportOptions.md)  

## Description

Gets and sets the filename or URL of the file to be imported.

## Syntax

"iGESImportOptions_var" is a variable referencing an IGESImportOptions object.  

```python
# Get the value of the property.
propertyValue = iGESImportOptions_var.filename

# Set the value of the property.
iGESImportOptions_var.filename = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version September 2015
