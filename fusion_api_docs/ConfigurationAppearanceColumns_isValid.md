# ConfigurationAppearanceColumns.isValid Property

Parent Object: [ConfigurationAppearanceColumns](ConfigurationAppearanceColumns.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"configurationAppearanceColumns_var" is a variable referencing a ConfigurationAppearanceColumns object.  

```python
# Get the value of the property.
propertyValue = configurationAppearanceColumns_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2024
