# CameraEventHandler.notify Method

Parent Object: [CameraEventHandler](CameraEventHandler.md)  

## Description

The function called by Fusion when the associated event is fired.

## Syntax

"cameraEventHandler_var" is a variable referencing a [CameraEventHandler](CameraEventHandler.md) object.

```python
returnValue = cameraEventHandler_var.notify(eventArgs)
```

## Parameters

| Name | Type | Description |
|----|----|----|
| eventArgs | [CameraEventArgs](CameraEventArgs.md) | Returns an object that provides access to additional information associated with the event. |

## Version

Introduced in version December 2017
