# CommandEventArgs.firingEvent Property

Parent Object: [CommandEventArgs](CommandEventArgs.md)  

## Description

The event that the firing is in response to.

## Syntax

"commandEventArgs_var" is a variable referencing a CommandEventArgs object.  

```python
# Get the value of the property.
propertyValue = commandEventArgs_var.firingEvent
```

## Property Value

This is a read only property whose value is an [Event](Event.md).

## Version

Introduced in version August 2014
