# DeriveFeature.isIncludeComponentParameters Property

Parent Object: [DeriveFeature](DeriveFeature.md)  

## Description

Gets or sets whether all feature parameters from all derived components from the source design are derived or not.

## Syntax

"deriveFeature_var" is a variable referencing a DeriveFeature object.  

```python
# Get the value of the property.
propertyValue = deriveFeature_var.isIncludeComponentParameters

# Set the value of the property.
deriveFeature_var.isIncludeComponentParameters = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version January 2026
