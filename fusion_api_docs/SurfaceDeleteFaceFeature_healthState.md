# SurfaceDeleteFaceFeature.healthState Property

Parent Object: [SurfaceDeleteFaceFeature](SurfaceDeleteFaceFeature.md)  

## Description

Returns the current health state of the feature.

## Syntax

"surfaceDeleteFaceFeature_var" is a variable referencing a SurfaceDeleteFaceFeature object.  

```python
# Get the value of the property.
propertyValue = surfaceDeleteFaceFeature_var.healthState
```

## Property Value

This is a read only property whose value is a [FeatureHealthStates](FeatureHealthStates.md).

## Version

Introduced in version August 2016
