# SplitFaceFeature.splitType Property

Parent Object: [SplitFaceFeature](SplitFaceFeature.md)  

## Description

Returns the type of split type currently defined. To change the split type, use one of the set methods.

## Syntax

"splitFaceFeature_var" is a variable referencing a SplitFaceFeature object.  

```python
# Get the value of the property.
propertyValue = splitFaceFeature_var.splitType
```

## Property Value

This is a read only property whose value is a [SplitFaceSplitTypes](SplitFaceSplitTypes.md).

## Version

Introduced in version May 2017
