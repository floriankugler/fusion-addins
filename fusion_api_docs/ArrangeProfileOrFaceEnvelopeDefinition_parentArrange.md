# ArrangeProfileOrFaceEnvelopeDefinition.parentArrange Property

Parent Object: [ArrangeProfileOrFaceEnvelopeDefinition](ArrangeProfileOrFaceEnvelopeDefinition.md)  

## Description

Returns the parent ArrangeFeature this envelope is associated with.

## Syntax

"arrangeProfileOrFaceEnvelopeDefinition_var" is a variable referencing an ArrangeProfileOrFaceEnvelopeDefinition object.  

```python
# Get the value of the property.
propertyValue = arrangeProfileOrFaceEnvelopeDefinition_var.parentArrange
```

## Property Value

This is a read only property whose value is an [ArrangeFeature](ArrangeFeature.md).

## Version

Introduced in version January 2025
