# Component.isCanvasFolderLightBulbOn Property

Parent Object: [Component](Component.md)  

## Description

Gets and sets if the light bulb of the canvas folder as seen in the browser is on or off. This controls the visibility of all the canvases in the component.

## Syntax

"component_var" is a variable referencing a Component object.  

```python
# Get the value of the property.
propertyValue = component_var.isCanvasFolderLightBulbOn

# Set the value of the property.
component_var.isCanvasFolderLightBulbOn = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version July 2023
