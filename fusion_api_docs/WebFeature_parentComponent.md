# WebFeature.parentComponent Property

Parent Object: [WebFeature](WebFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"webFeature_var" is a variable referencing a WebFeature object.  

```python
# Get the value of the property.
propertyValue = webFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version September 2015
