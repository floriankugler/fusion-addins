# BRepEdge.evaluator Property

Parent Object: [BRepEdge](BRepEdge.md)  

## Description

Returns CurveEvaluator3D for evaluation.

## Syntax

"bRepEdge_var" is a variable referencing a BRepEdge object.  

```python
# Get the value of the property.
propertyValue = bRepEdge_var.evaluator
```

## Property Value

This is a read only property whose value is a [CurveEvaluator3D](CurveEvaluator3D.md).

## Version

Introduced in version August 2014
