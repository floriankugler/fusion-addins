# Arc2D.isClockwise Property

Parent Object: [Arc2D](Arc2D.md)  

## Description

Gets if the sweep direction of the arc is clockwise or counterclockwise.

## Syntax

"arc2D_var" is a variable referencing an Arc2D object.  

```python
# Get the value of the property.
propertyValue = arc2D_var.isClockwise
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
