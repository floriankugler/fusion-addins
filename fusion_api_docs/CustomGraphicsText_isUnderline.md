# CustomGraphicsText.isUnderline Property

Parent Object: [CustomGraphicsText](CustomGraphicsText.md)  

## Description

Specifies that the text displays using an underline style. This is the default underline style and applies to all of text unless there is a style override defined within the text.

## Syntax

"customGraphicsText_var" is a variable referencing a CustomGraphicsText object.  

```python
# Get the value of the property.
propertyValue = customGraphicsText_var.isUnderline

# Set the value of the property.
customGraphicsText_var.isUnderline = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version April 2019
