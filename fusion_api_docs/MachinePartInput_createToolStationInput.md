# MachinePartInput.createToolStationInput Method

Parent Object: [MachinePartInput](MachinePartInput.md)  

## Description

Creates a new MachineToolStationInput object to be used to create a new MachineToolStation.

## Syntax

"machinePartInput_var" is a variable referencing a [MachinePartInput](MachinePartInput.md) object.

```python
returnValue = machinePartInput_var.createToolStationInput()
```

## Return Value

| Type | Description |
|----|----|
| [MachineToolStationInput](MachineToolStationInput.md) | Returns new MachineToolStationInput object. |

## Version

Introduced in version July 2025
