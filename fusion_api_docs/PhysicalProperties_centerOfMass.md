# PhysicalProperties.centerOfMass Property

Parent Object: [PhysicalProperties](PhysicalProperties.md)  

## Description

Returns the center of mass position

## Syntax

"physicalProperties_var" is a variable referencing a PhysicalProperties object.  

```python
# Get the value of the property.
propertyValue = physicalProperties_var.centerOfMass
```

## Property Value

This is a read only property whose value is a [Point3D](Point3D.md).
