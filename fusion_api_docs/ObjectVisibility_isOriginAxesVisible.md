# ObjectVisibility.isOriginAxesVisible Property

Parent Object: [ObjectVisibility](ObjectVisibility.md)  

## Description

Gets and sets if the origin construction axes of all components, local or referenced by this design, are visible in the graphics window.

## Syntax

"objectVisibility_var" is a variable referencing an ObjectVisibility object.  

```python
# Get the value of the property.
propertyValue = objectVisibility_var.isOriginAxesVisible

# Set the value of the property.
objectVisibility_var.isOriginAxesVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version November 2025
