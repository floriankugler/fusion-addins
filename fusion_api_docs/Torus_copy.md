# Torus.copy Method

Parent Object: [Torus](Torus.md)  

## Description

Creates and returns an independent copy of this Torus object.

## Syntax

"torus_var" is a variable referencing a [Torus](Torus.md) object.

```python
returnValue = torus_var.copy()
```

## Return Value

| Type | Description |
|----|----|
| [Torus](Torus.md) | Returns a new Torus object that is a copy of this Torus object. |

## Version

Introduced in version August 2014
