# CommandCreatedEventHandler Object

Derived from: [EventHandler](EventHandler.md) Object  

## Description

Class that contains the call back function that is called when the CommandCreated event is fired.

## Methods

| Name | Description |
|----|----|
| [notify](CommandCreatedEventHandler_notify.md) | This notify member is called when an event is triggered from any event that this handler has been added to. |
