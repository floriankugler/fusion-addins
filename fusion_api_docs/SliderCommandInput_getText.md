# SliderCommandInput.getText Method

Parent Object: [SliderCommandInput](SliderCommandInput.md)  

## Description

Gets the texts of the slider if text has been defined.

## Syntax

"sliderCommandInput_var" is a variable referencing a [SliderCommandInput](SliderCommandInput.md) object.

```python
returnValue = sliderCommandInput_var.getText(isLeft)
```

## Return Value

| Type   | Description                                   |
|--------|-----------------------------------------------|
| string | Returns the left or right text of the slider. |

## Parameters

| Name   | Type    | Description                              |
|--------|---------|------------------------------------------|
| isLeft | boolean | Indicates to get the left or right text. |

## Version

Introduced in version June 2015
