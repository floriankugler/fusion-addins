# CurvatureMapAnalyses.itemByName Method

Parent Object: [CurvatureMapAnalyses](CurvatureMapAnalyses.md)  

## Description

A method that returns the specified CurvatureMapAnalysis object using the name of the analysis as displayed in the browser.

## Syntax

"curvatureMapAnalyses_var" is a variable referencing a [CurvatureMapAnalyses](CurvatureMapAnalyses.md) object.

```python
returnValue = curvatureMapAnalyses_var.itemByName(name)
```

## Return Value

| Type | Description |
|----|----|
| [CurvatureMapAnalysis](CurvatureMapAnalysis.md) | Returns the specified item or null if an invalid name was specified. |

## Parameters

| Name | Type | Description |
|----|----|----|
| name | string | The name of the CurvatureMapAnalysis object as displayed in the browser. |

## Version

Introduced in version January 2023
