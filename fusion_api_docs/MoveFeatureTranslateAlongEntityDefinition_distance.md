# MoveFeatureTranslateAlongEntityDefinition.distance Property

Parent Object: [MoveFeatureTranslateAlongEntityDefinition](MoveFeatureTranslateAlongEntityDefinition.md)  

## Description

Gets the model parameter that controls the offset distance. You can use properties on the returned ModelParameter object to edit the offset distance.

## Syntax

"moveFeatureTranslateAlongEntityDefinition_var" is a variable referencing a MoveFeatureTranslateAlongEntityDefinition object.  

```python
# Get the value of the property.
propertyValue = moveFeatureTranslateAlongEntityDefinition_var.distance
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version January 2023
