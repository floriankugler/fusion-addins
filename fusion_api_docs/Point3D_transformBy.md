# Point3D.transformBy Method

Parent Object: [Point3D](Point3D.md)  

## Description

Transforms the point using the provided matrix.

## Syntax

"point3D_var" is a variable referencing a [Point3D](Point3D.md) object.

```python
returnValue = point3D_var.transformBy(matrix)
```

## Return Value

| Type    | Description                 |
|---------|-----------------------------|
| boolean | Returns true if successful. |

## Parameters

| Name | Type | Description |
|----|----|----|
| matrix | [Matrix3D](Matrix3D.md) | The Matrix3D object that defines the transformation. |

## Version

Introduced in version August 2014
