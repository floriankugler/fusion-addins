# SeparatorCommandInput.deleteMe Method

Parent Object: [SeparatorCommandInput](SeparatorCommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"separatorCommandInput_var" is a variable referencing a [SeparatorCommandInput](SeparatorCommandInput.md) object.

```python
returnValue = separatorCommandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version May 2024
