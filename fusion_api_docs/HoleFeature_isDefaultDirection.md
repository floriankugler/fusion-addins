# HoleFeature.isDefaultDirection Property

Parent Object: [HoleFeature](HoleFeature.md)  

## Description

Gets and sets if the hole is in the default direction or not.

## Syntax

"holeFeature_var" is a variable referencing a HoleFeature object.  

```python
# Get the value of the property.
propertyValue = holeFeature_var.isDefaultDirection

# Set the value of the property.
holeFeature_var.isDefaultDirection = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
