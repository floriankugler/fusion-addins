# DrawingDocument.allDocumentReferences Property

Parent Object: [DrawingDocument](DrawingDocument.md)  

## Description

Returns a collection containing all of the documents referenced directly by this document and those referenced by all sub-assemblies.

## Syntax

"drawingDocument_var" is a variable referencing a DrawingDocument object.  

```python
# Get the value of the property.
propertyValue = drawingDocument_var.allDocumentReferences
```

## Property Value

This is a read only property whose value is a [DocumentReferences](DocumentReferences.md).

## Version

Introduced in version December 2020
