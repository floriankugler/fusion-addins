# RotaryMachineAxisConfiguration.reset Property

Parent Object: [RotaryMachineAxisConfiguration](RotaryMachineAxisConfiguration.md)  

## Description

Specify when to reset the initial axis position.

## Syntax

"rotaryMachineAxisConfiguration_var" is a variable referencing a RotaryMachineAxisConfiguration object.  

```python
# Get the value of the property.
propertyValue = rotaryMachineAxisConfiguration_var.reset

# Set the value of the property.
rotaryMachineAxisConfiguration_var.reset = propertyValue
```

## Property Value

This is a read/write property whose value is a [MachineResetOptions](MachineResetOptions.md).

## Version

Introduced in version April 2023
