# FlatPatternComponent.isDecalFolderLightBulbOn Property

Parent Object: [FlatPatternComponent](FlatPatternComponent.md)  

## Description

Gets and sets whether the light bulb of the decal folder, as seen in the browser, is on or off. This controls the visibility of all the decals in the component.

## Syntax

"flatPatternComponent_var" is a variable referencing a FlatPatternComponent object.  

```python
# Get the value of the property.
propertyValue = flatPatternComponent_var.isDecalFolderLightBulbOn

# Set the value of the property.
flatPatternComponent_var.isDecalFolderLightBulbOn = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version September 2024
