# EqualConstraint.isDeletable Property

Parent Object: [EqualConstraint](EqualConstraint.md)  

## Description

Indicates if this constraint is deletable.

## Syntax

"equalConstraint_var" is a variable referencing an EqualConstraint object.  

```python
# Get the value of the property.
propertyValue = equalConstraint_var.isDeletable
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version August 2014
