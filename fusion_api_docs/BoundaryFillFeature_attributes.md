# BoundaryFillFeature.attributes Property

Parent Object: [BoundaryFillFeature](BoundaryFillFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"boundaryFillFeature_var" is a variable referencing a BoundaryFillFeature object.  

```python
# Get the value of the property.
propertyValue = boundaryFillFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
