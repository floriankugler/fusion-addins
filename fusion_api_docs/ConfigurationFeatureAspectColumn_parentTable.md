# ConfigurationFeatureAspectColumn.parentTable Property

Parent Object: [ConfigurationFeatureAspectColumn](ConfigurationFeatureAspectColumn.md)  

## Description

This property returns the parent table, either the top or custom theme table this column is in.

## Syntax

"configurationFeatureAspectColumn_var" is a variable referencing a ConfigurationFeatureAspectColumn object.  

```python
# Get the value of the property.
propertyValue = configurationFeatureAspectColumn_var.parentTable
```

## Property Value

This is a read only property whose value is a [ConfigurationTable](ConfigurationTable.md).

## Version

Introduced in version September 2024
