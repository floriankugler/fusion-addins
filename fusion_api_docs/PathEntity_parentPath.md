# PathEntity.parentPath Property

Parent Object: [PathEntity](PathEntity.md)  

## Description

Property that returns the parent Path of the entity.

## Syntax

"pathEntity_var" is a variable referencing a PathEntity object.  

```python
# Get the value of the property.
propertyValue = pathEntity_var.parentPath
```

## Property Value

This is a read only property whose value is a [Path](Path.md).

## Version

Introduced in version November 2014
