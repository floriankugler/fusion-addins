# DropDownCommandInput.tooltip Property

Parent Object: [DropDownCommandInput](DropDownCommandInput.md)  

## Description

Gets or sets the base tooltip string. This is always shown for commands. If the tooltip description and/or tool clip are also specified then the tooltip will progressively display more information as the user hovers the mouse over the control.

## Syntax

"dropDownCommandInput_var" is a variable referencing a DropDownCommandInput object.  

```python
# Get the value of the property.
propertyValue = dropDownCommandInput_var.tooltip

# Set the value of the property.
dropDownCommandInput_var.tooltip = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version July 2015
