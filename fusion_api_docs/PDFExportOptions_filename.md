# PDFExportOptions.filename Property

Parent Object: [PDFExportOptions](PDFExportOptions.md)  

## Description

Gets and sets the filename that the exported file will be written to.

## Syntax

"pDFExportOptions_var" is a variable referencing a PDFExportOptions object.  

```python
# Get the value of the property.
propertyValue = pDFExportOptions_var.filename

# Set the value of the property.
pDFExportOptions_var.filename = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version December 2020
