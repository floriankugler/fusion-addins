# BoolValueCommandInput.deleteMe Method

Parent Object: [BoolValueCommandInput](BoolValueCommandInput.md)  

## Description

Deletes this Command input.

## Syntax

"boolValueCommandInput_var" is a variable referencing a [BoolValueCommandInput](BoolValueCommandInput.md) object.

```python
returnValue = boolValueCommandInput_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2020
