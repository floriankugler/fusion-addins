# OffsetFacesFeature.name Property

Parent Object: [OffsetFacesFeature](OffsetFacesFeature.md)  

## Description

Returns the name of the feature as seen in the browser (non-parametric) or in the timeline (parametric).

## Syntax

"offsetFacesFeature_var" is a variable referencing an OffsetFacesFeature object.  

```python
# Get the value of the property.
propertyValue = offsetFacesFeature_var.name

# Set the value of the property.
offsetFacesFeature_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version June 2017
