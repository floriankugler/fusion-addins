# GroupCommandInput.isVisible Property

Parent Object: [GroupCommandInput](GroupCommandInput.md)  

## Description

Gets or sets if this input will be visible to the user.  

Setting a SelectionCommandInput to be invisible will clear any selections it currently has.

## Syntax

"groupCommandInput_var" is a variable referencing a GroupCommandInput object.  

```python
# Get the value of the property.
propertyValue = groupCommandInput_var.isVisible

# Set the value of the property.
groupCommandInput_var.isVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version July 2015
