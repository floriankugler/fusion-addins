# DerivedParameter.comment Property

Parent Object: [DerivedParameter](DerivedParameter.md)  

## Description

The comment associated with this parameter

## Syntax

"derivedParameter_var" is a variable referencing a DerivedParameter object.  

```python
# Get the value of the property.
propertyValue = derivedParameter_var.comment

# Set the value of the property.
derivedParameter_var.comment = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version January 2026
