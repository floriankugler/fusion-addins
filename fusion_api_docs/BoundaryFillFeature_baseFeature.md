# BoundaryFillFeature.baseFeature Property

Parent Object: [BoundaryFillFeature](BoundaryFillFeature.md)  

## Description

If this feature is associated with a base feature, this property will return that base feature. If it's not associated with a base feature, this property will return null.

## Syntax

"boundaryFillFeature_var" is a variable referencing a BoundaryFillFeature object.  

```python
# Get the value of the property.
propertyValue = boundaryFillFeature_var.baseFeature
```

## Property Value

This is a read only property whose value is a [BaseFeature](BaseFeature.md).

## Version

Introduced in version May 2016
