# TrimFeatures.count Property

Parent Object: [TrimFeatures](TrimFeatures.md)  

## Description

The number of trim features in the collection.

## Syntax

"trimFeatures_var" is a variable referencing a TrimFeatures object.  

```python
# Get the value of the property.
propertyValue = trimFeatures_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version July 2015
