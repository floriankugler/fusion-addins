# SketchLine.length Property

Parent Object: [SketchLine](SketchLine.md)  

## Description

Returns the length of the curve in centimeters.

## Syntax

"sketchLine_var" is a variable referencing a SketchLine object.  

```python
# Get the value of the property.
propertyValue = sketchLine_var.length
```

## Property Value

This is a read only property whose value is a double.

## Version

Introduced in version August 2014
