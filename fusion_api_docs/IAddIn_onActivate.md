# IAddIn.onActivate Method

Parent Object: [IAddIn](IAddIn.md)  

## Description

Lets the application do any initialization when it is safe to do so.

## Syntax

"iAddIn_var" is a variable referencing an [IAddIn](IAddIn.md) object.

```python
returnValue = iAddIn_var.onActivate()
```

## Version

Introduced in version March 2022
