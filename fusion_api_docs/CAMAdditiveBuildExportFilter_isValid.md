# CAMAdditiveBuildExportFilter.isValid Property

Parent Object: [CAMAdditiveBuildExportFilter](CAMAdditiveBuildExportFilter.md)  

## Description

Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.

## Syntax

"cAMAdditiveBuildExportFilter_var" is a variable referencing a CAMAdditiveBuildExportFilter object.  

```python
# Get the value of the property.
propertyValue = cAMAdditiveBuildExportFilter_var.isValid
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version October 2023
