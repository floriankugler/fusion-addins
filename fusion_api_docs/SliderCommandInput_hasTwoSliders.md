# SliderCommandInput.hasTwoSliders Property

Parent Object: [SliderCommandInput](SliderCommandInput.md)  

## Description

Gets if the command input has two sliders.

## Syntax

"sliderCommandInput_var" is a variable referencing a SliderCommandInput object.  

```python
# Get the value of the property.
propertyValue = sliderCommandInput_var.hasTwoSliders
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version June 2015
