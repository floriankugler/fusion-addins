# ConfigurationPlasticRuleColumns.count Property

Parent Object: [ConfigurationPlasticRuleColumns](ConfigurationPlasticRuleColumns.md)  

## Description

Returns the number of columns in the table where the name column is not included.

## Syntax

"configurationPlasticRuleColumns_var" is a variable referencing a ConfigurationPlasticRuleColumns object.  

```python
# Get the value of the property.
propertyValue = configurationPlasticRuleColumns_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version January 2024
