# Product.parentDocument Property

Parent Object: [Product](Product.md)  

## Description

Returns the parent Document object.

## Syntax

"product_var" is a variable referencing a Product object.  

```python
# Get the value of the property.
propertyValue = product_var.parentDocument
```

## Property Value

This is a read only property whose value is a [Document](Document.md).

## Version

Introduced in version August 2014
