# PathPatternFeature.deleteMe Method

Parent Object: [PathPatternFeature](PathPatternFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"pathPatternFeature_var" is a variable referencing a [PathPatternFeature](PathPatternFeature.md) object.

```python
returnValue = pathPatternFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version November 2014
