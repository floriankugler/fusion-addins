# SplitBodyFeature.deleteMe Method

Parent Object: [SplitBodyFeature](SplitBodyFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"splitBodyFeature_var" is a variable referencing a [SplitBodyFeature](SplitBodyFeature.md) object.

```python
returnValue = splitBodyFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version June 2015
