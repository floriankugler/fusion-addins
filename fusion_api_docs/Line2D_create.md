# Line2D.create Method

Parent Object: [Line2D](Line2D.md)  

## Description

Creates a transient line.

## Syntax

This is a static method.  

```python

returnValue = adsk.core.Line2D.create(startPoint, endPoint)
```

## Return Value

| Type | Description |
|----|----|
| [Line2D](Line2D.md) | Returns the new Line2D object or null if the creation failed. |

## Parameters

| Name       | Type                   | Description                 |
|------------|------------------------|-----------------------------|
| startPoint | [Point2D](Point2D.md) | The start point of the line |
| endPoint   | [Point2D](Point2D.md) | The end point of the line   |

## Version

Introduced in version August 2014
