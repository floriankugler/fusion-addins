# BoundaryFillFeature.parentComponent Property

Parent Object: [BoundaryFillFeature](BoundaryFillFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"boundaryFillFeature_var" is a variable referencing a BoundaryFillFeature object.  

```python
# Get the value of the property.
propertyValue = boundaryFillFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version June 2015
