# CAMHoleRecognition.activate Method

Parent Object: [CAMHoleRecognition](CAMHoleRecognition.md)  

## Description

Set this object as the default container.

## Syntax

"cAMHoleRecognition_var" is a variable referencing a [CAMHoleRecognition](CAMHoleRecognition.md) object.

```python
returnValue = cAMHoleRecognition_var.activate()
```

## Return Value

| Type    | Description                                    |
|---------|------------------------------------------------|
| boolean | Returns true if the activation was successful. |

## Version

Introduced in version May 2023
