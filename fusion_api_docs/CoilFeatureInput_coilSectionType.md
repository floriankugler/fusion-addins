# CoilFeatureInput.coilSectionType Property

Parent Object: [CoilFeatureInput](CoilFeatureInput.md)  

## Description

Gets the section type of the coil. It defaults to CircularCoilSectionType.

## Syntax

"coilFeatureInput_var" is a variable referencing a CoilFeatureInput object.  

```python
# Get the value of the property.
propertyValue = coilFeatureInput_var.coilSectionType

# Set the value of the property.
coilFeatureInput_var.coilSectionType = propertyValue
```

## Property Value

This is a read/write property whose value is a [CoilFeatureSectionTypes](CoilFeatureSectionTypes.md).

## Version

Introduced in version March 2016
