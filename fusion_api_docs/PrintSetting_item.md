# PrintSetting.item Method

Parent Object: [PrintSetting](PrintSetting.md)  

## Description

Get the PrintSettingItem at index in PrintSetting.

## Syntax

"printSetting_var" is a variable referencing a [PrintSetting](PrintSetting.md) object.

```python
returnValue = printSetting_var.item(index)
```

## Return Value

| Type                                     | Description                    |
|------------------------------------------|--------------------------------|
| [PrintSettingItem](PrintSettingItem.md) | The PrintSettingItem at index. |

## Parameters

| Name  | Type    | Description                        |
|-------|---------|------------------------------------|
| index | integer | The index of the PrintSettingItem. |

## Version

Introduced in version April 2023
