# ConstructionPlane.timelineObject Property

Parent Object: [ConstructionPlane](ConstructionPlane.md)  

## Description

Returns the timeline object associated with this construction plane.

## Syntax

"constructionPlane_var" is a variable referencing a ConstructionPlane object.  

```python
# Get the value of the property.
propertyValue = constructionPlane_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version August 2014
