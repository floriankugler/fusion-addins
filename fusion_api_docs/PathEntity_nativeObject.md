# PathEntity.nativeObject Property

Parent Object: [PathEntity](PathEntity.md)  

## Description

This property is not supported for a PathEntity object.

## Syntax

"pathEntity_var" is a variable referencing a PathEntity object.  

```python
# Get the value of the property.
propertyValue = pathEntity_var.nativeObject
```

## Property Value

This is a read only property whose value is a [PathEntity](PathEntity.md).

## Version

Introduced in version November 2014
