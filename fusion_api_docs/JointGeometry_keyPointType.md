# JointGeometry.keyPointType Property

Parent Object: [JointGeometry](JointGeometry.md)  

## Description

Returns the keypoint type this JointGeometry is using.

## Syntax

"jointGeometry_var" is a variable referencing a JointGeometry object.  

```python
# Get the value of the property.
propertyValue = jointGeometry_var.keyPointType
```

## Property Value

This is a read only property whose value is a [JointKeyPointTypes](JointKeyPointTypes.md).

## Version

Introduced in version July 2015
