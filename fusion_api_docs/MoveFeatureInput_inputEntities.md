# MoveFeatureInput.inputEntities Property

Parent Object: [MoveFeatureInput](MoveFeatureInput.md)  

## Description

An ObjectCollection containing the objects to move. The collection can contain BRepBody or BRepFace objects but not a mixture of the two types.

## Syntax

"moveFeatureInput_var" is a variable referencing a MoveFeatureInput object.  

```python
# Get the value of the property.
propertyValue = moveFeatureInput_var.inputEntities

# Set the value of the property.
moveFeatureInput_var.inputEntities = propertyValue
```

## Property Value

This is a read/write property whose value is an [ObjectCollection](ObjectCollection.md).

## Version

Introduced in version March 2015
