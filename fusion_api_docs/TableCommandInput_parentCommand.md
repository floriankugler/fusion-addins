# TableCommandInput.parentCommand Property

Parent Object: [TableCommandInput](TableCommandInput.md)  

## Description

Gets the parent Command.

## Syntax

"tableCommandInput_var" is a variable referencing a TableCommandInput object.  

```python
# Get the value of the property.
propertyValue = tableCommandInput_var.parentCommand
```

## Property Value

This is a read only property whose value is a [Command](Command.md).

## Version

Introduced in version September 2016
