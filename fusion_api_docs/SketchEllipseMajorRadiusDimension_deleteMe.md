# SketchEllipseMajorRadiusDimension.deleteMe Method

Parent Object: [SketchEllipseMajorRadiusDimension](SketchEllipseMajorRadiusDimension.md)  

## Description

Deletes this dimension. The IsDeletable property indicates if this dimension can be deleted.

## Syntax

"sketchEllipseMajorRadiusDimension_var" is a variable referencing a [SketchEllipseMajorRadiusDimension](SketchEllipseMajorRadiusDimension.md) object.

```python
returnValue = sketchEllipseMajorRadiusDimension_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version August 2014
