# RevolveFeature.timelineObject Property

Parent Object: [RevolveFeature](RevolveFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"revolveFeature_var" is a variable referencing a RevolveFeature object.  

```python
# Get the value of the property.
propertyValue = revolveFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version August 2014
