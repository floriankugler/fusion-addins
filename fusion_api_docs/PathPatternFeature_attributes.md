# PathPatternFeature.attributes Property

Parent Object: [PathPatternFeature](PathPatternFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"pathPatternFeature_var" is a variable referencing a PathPatternFeature object.  

```python
# Get the value of the property.
propertyValue = pathPatternFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
