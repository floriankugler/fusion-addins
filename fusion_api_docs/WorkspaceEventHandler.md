# WorkspaceEventHandler Object

Derived from: [EventHandler](EventHandler.md) Object  

## Description

The WorkspaceEventHandler is a client implemented class that can be added as a handler to a WorkspaceEvent.

## Methods

| Name | Description |
|----|----|
| [notify](WorkspaceEventHandler_notify.md) | The function called by Fusion when the associated event is fired. |

## Version

Introduced in version March 2015
