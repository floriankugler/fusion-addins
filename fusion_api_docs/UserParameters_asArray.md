# UserParameters.asArray Method

Parent Object: [UserParameters](UserParameters.md)  

## Description

Returns the user parameters in the design as an array.

## Syntax

"userParameters_var" is a variable referencing a [UserParameters](UserParameters.md) object.

```python
returnValue = userParameters_var.asArray()
```

## Return Value

| Type | Description |
|----|----|
| [UserParameter](UserParameter.md)\[\] | Returns an array of the user parameters in the design. |

## Version

Introduced in version September 2024
