# FilenameProperty.value Property

Parent Object: [FilenameProperty](FilenameProperty.md)  

## Description

Gets and sets the value of this property.

## Syntax

"filenameProperty_var" is a variable referencing a FilenameProperty object.  

```python
# Get the value of the property.
propertyValue = filenameProperty_var.value

# Set the value of the property.
filenameProperty_var.value = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version August 2014
