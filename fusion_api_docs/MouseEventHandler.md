# MouseEventHandler Object

Derived from: [EventHandler](EventHandler.md) Object  

## Description

An event handler base class that a client derives from to handle events triggered by a MouseEvent. A client implemented instance of this class can be added to a MouseEvent to receive these event notifications.

## Methods

| Name | Description |
|----|----|
| [notify](MouseEventHandler_notify.md) | This notify member is called when an event is triggered from any event that this handler has been added to. |

## Version

Introduced in version August 2014
