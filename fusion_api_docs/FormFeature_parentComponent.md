# FormFeature.parentComponent Property

Parent Object: [FormFeature](FormFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"formFeature_var" is a variable referencing a FormFeature object.  

```python
# Get the value of the property.
propertyValue = formFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version September 2015
