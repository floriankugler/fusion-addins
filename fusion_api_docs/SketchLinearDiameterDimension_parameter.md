# SketchLinearDiameterDimension.parameter Property

Parent Object: [SketchLinearDiameterDimension](SketchLinearDiameterDimension.md)  

## Description

Returns the associated parameter or null if there is no associated parameter.

## Syntax

"sketchLinearDiameterDimension_var" is a variable referencing a SketchLinearDiameterDimension object.  

```python
# Get the value of the property.
propertyValue = sketchLinearDiameterDimension_var.parameter
```

## Property Value

This is a read only property whose value is a [ModelParameter](ModelParameter.md).

## Version

Introduced in version September 2022
