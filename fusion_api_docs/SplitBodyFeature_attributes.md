# SplitBodyFeature.attributes Property

Parent Object: [SplitBodyFeature](SplitBodyFeature.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"splitBodyFeature_var" is a variable referencing a SplitBodyFeature object.  

```python
# Get the value of the property.
propertyValue = splitBodyFeature_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version May 2016
