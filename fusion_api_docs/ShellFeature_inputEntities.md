# ShellFeature.inputEntities Property

Parent Object: [ShellFeature](ShellFeature.md)  

## Description

Gets the input faces/bodies.

## Syntax

"shellFeature_var" is a variable referencing a ShellFeature object.  

```python
# Get the value of the property.
propertyValue = shellFeature_var.inputEntities
```

## Property Value

This is a read only property whose value is an [ObjectCollection](ObjectCollection.md).

## Version

Introduced in version November 2014
