# MachineSpindleInput.description Property

Parent Object: [MachineSpindleInput](MachineSpindleInput.md)  

## Description

The description of this spindle.

## Syntax

"machineSpindleInput_var" is a variable referencing a MachineSpindleInput object.  

```python
# Get the value of the property.
propertyValue = machineSpindleInput_var.description

# Set the value of the property.
machineSpindleInput_var.description = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version April 2023
