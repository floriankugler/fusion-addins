# Occurrence.isConfiguration Property

Parent Object: [Occurrence](Occurrence.md)  

## Description

Returns if this occurrence represents a configuration or not. If true, you can access the full configuration from the parent component.

## Syntax

"occurrence_var" is a variable referencing an Occurrence object.  

```python
# Get the value of the property.
propertyValue = occurrence_var.isConfiguration
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version January 2024
