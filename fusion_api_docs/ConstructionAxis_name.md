# ConstructionAxis.name Property

Parent Object: [ConstructionAxis](ConstructionAxis.md)  

## Description

The name of the construction axis as it is shown in the browser.

## Syntax

"constructionAxis_var" is a variable referencing a ConstructionAxis object.  

```python
# Get the value of the property.
propertyValue = constructionAxis_var.name

# Set the value of the property.
constructionAxis_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version August 2014
