# PatternElement.name Property

Parent Object: [PatternElement](PatternElement.md)  

## Description

Get the name of the pattern element.

## Syntax

"patternElement_var" is a variable referencing a PatternElement object.  

```python
# Get the value of the property.
propertyValue = patternElement_var.name
```

## Property Value

This is a read only property whose value is a string.

## Version

Introduced in version November 2014
