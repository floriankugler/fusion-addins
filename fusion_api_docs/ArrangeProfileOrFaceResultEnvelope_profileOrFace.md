# ArrangeProfileOrFaceResultEnvelope.profileOrFace Property

Parent Object: [ArrangeProfileOrFaceResultEnvelope](ArrangeProfileOrFaceResultEnvelope.md)  

## Description

Returns the Profile or BRepFace object that defines the shape of this result envelope.

"arrangeProfileOrFaceResultEnvelope_var" is a variable referencing an ArrangeProfileOrFaceResultEnvelope object.  

```python
# Get the value of the property.
propertyValue = arrangeProfileOrFaceResultEnvelope_var.profileOrFace
```

This is a read only property whose value is a [Base](Base.md).
