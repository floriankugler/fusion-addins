# ExtrudeFeature.name Property

Parent Object: [ExtrudeFeature](ExtrudeFeature.md)  

## Description

Returns the name of the feature as seen in the browser (non-parametric) or in the timeline (parametric).

## Syntax

"extrudeFeature_var" is a variable referencing an ExtrudeFeature object.  

```python
# Get the value of the property.
propertyValue = extrudeFeature_var.name

# Set the value of the property.
extrudeFeature_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version August 2014
