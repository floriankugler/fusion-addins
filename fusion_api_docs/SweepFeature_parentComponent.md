# SweepFeature.parentComponent Property

Parent Object: [SweepFeature](SweepFeature.md)  

## Description

Returns the parent component that owns this feature.

## Syntax

"sweepFeature_var" is a variable referencing a SweepFeature object.  

```python
# Get the value of the property.
propertyValue = sweepFeature_var.parentComponent
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version November 2014
