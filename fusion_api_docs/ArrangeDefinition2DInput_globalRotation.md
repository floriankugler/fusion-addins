# ArrangeDefinition2DInput.globalRotation Property

Parent Object: [ArrangeDefinition2DInput](ArrangeDefinition2DInput.md)  

## Description

Gets and sets the global rotation type. This defaults to AllRotationsArrangeRotationType.

## Syntax

"arrangeDefinition2DInput_var" is a variable referencing an ArrangeDefinition2DInput object.  

```python
# Get the value of the property.
propertyValue = arrangeDefinition2DInput_var.globalRotation

# Set the value of the property.
arrangeDefinition2DInput_var.globalRotation = propertyValue
```

## Property Value

This is a read/write property whose value is an [ArrangeRotationTypes](ArrangeRotationTypes.md).

## Version

Introduced in version January 2025
