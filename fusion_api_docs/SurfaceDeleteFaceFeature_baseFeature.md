# SurfaceDeleteFaceFeature.baseFeature Property

Parent Object: [SurfaceDeleteFaceFeature](SurfaceDeleteFaceFeature.md)  

## Description

If this feature is associated with a base feature, this property will return that base feature. If it's not associated with a base feature, this property will return null.

## Syntax

"surfaceDeleteFaceFeature_var" is a variable referencing a SurfaceDeleteFaceFeature object.  

```python
# Get the value of the property.
propertyValue = surfaceDeleteFaceFeature_var.baseFeature
```

## Property Value

This is a read only property whose value is a [BaseFeature](BaseFeature.md).

## Version

Introduced in version August 2016
