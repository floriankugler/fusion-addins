# UserInterface.toolbars Property

Parent Object: [UserInterface](UserInterface.md)  

## Description

Gets a collection that provides access to the toolbars. This includes the left and right QAT, and the Navbar.

## Syntax

"userInterface_var" is a variable referencing a UserInterface object.  

```python
# Get the value of the property.
propertyValue = userInterface_var.toolbars
```

## Property Value

This is a read only property whose value is a [Toolbars](Toolbars.md).
