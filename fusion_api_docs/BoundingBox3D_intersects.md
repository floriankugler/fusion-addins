# BoundingBox3D.intersects Method

Parent Object: [BoundingBox3D](BoundingBox3D.md)  

## Description

Determines if the two bounding boxes intersect.

## Syntax

"boundingBox3D_var" is a variable referencing a [BoundingBox3D](BoundingBox3D.md) object.

```python
returnValue = boundingBox3D_var.intersects(boundingBox)
```

## Return Value

| Type    | Description                              |
|---------|------------------------------------------|
| boolean | Returns true if the two boxes intersect. |

## Parameters

| Name | Type | Description |
|----|----|----|
| boundingBox | [BoundingBox3D](BoundingBox3D.md) | The other bounding box to check for intersection with. |

## Version

Introduced in version August 2014
