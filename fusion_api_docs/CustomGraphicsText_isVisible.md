# CustomGraphicsText.isVisible Property

Parent Object: [CustomGraphicsText](CustomGraphicsText.md)  

## Description

Gets and sets if the graphics entity is visible in the graphics window. By default, when a new entity is created it is visible.

## Syntax

"customGraphicsText_var" is a variable referencing a CustomGraphicsText object.  

```python
# Get the value of the property.
propertyValue = customGraphicsText_var.isVisible

# Set the value of the property.
customGraphicsText_var.isVisible = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version September 2017
