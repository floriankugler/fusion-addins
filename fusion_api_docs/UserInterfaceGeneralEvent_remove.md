# UserInterfaceGeneralEvent.remove Method

Parent Object: [UserInterfaceGeneralEvent](UserInterfaceGeneralEvent.md)  

## Description

Removes a handler from the event.

## Syntax

"userInterfaceGeneralEvent_var" is a variable referencing a [UserInterfaceGeneralEvent](UserInterfaceGeneralEvent.md) object.

```python
returnValue = userInterfaceGeneralEvent_var.remove(handler)
```

## Return Value

| Type    | Description                                            |
|---------|--------------------------------------------------------|
| boolean | Returns true if removal of the handler was successful. |

## Parameters

| Name | Type | Description |
|----|----|----|
| handler | [UserInterfaceGeneralEventHandler](UserInterfaceGeneralEventHandler.md) | The handler object to be removed from the event. |

## Version

Introduced in version March 2017
