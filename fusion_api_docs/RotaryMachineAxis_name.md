# RotaryMachineAxis.name Property

Parent Object: [RotaryMachineAxis](RotaryMachineAxis.md)  

## Description

The name of this axis.

## Syntax

"rotaryMachineAxis_var" is a variable referencing a RotaryMachineAxis object.  

```python
# Get the value of the property.
propertyValue = rotaryMachineAxis_var.name

# Set the value of the property.
rotaryMachineAxis_var.name = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version April 2023
