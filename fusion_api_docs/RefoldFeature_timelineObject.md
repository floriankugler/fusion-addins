# RefoldFeature.timelineObject Property

Parent Object: [RefoldFeature](RefoldFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"refoldFeature_var" is a variable referencing a RefoldFeature object.  

```python
# Get the value of the property.
propertyValue = refoldFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version August 2020
