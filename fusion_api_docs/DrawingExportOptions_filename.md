# DrawingExportOptions.filename Property

Parent Object: [DrawingExportOptions](DrawingExportOptions.md)  

## Description

Gets and sets the filename that the exported file will be written to.

## Syntax

"drawingExportOptions_var" is a variable referencing a DrawingExportOptions object.  

```python
# Get the value of the property.
propertyValue = drawingExportOptions_var.filename

# Set the value of the property.
drawingExportOptions_var.filename = propertyValue
```

## Property Value

This is a read/write property whose value is a string.

## Version

Introduced in version December 2020
