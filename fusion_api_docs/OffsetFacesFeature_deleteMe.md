# OffsetFacesFeature.deleteMe Method

Parent Object: [OffsetFacesFeature](OffsetFacesFeature.md)  

## Description

Deletes the feature. This works for both parametric and non-parametric features.

## Syntax

"offsetFacesFeature_var" is a variable referencing an [OffsetFacesFeature](OffsetFacesFeature.md) object.

```python
returnValue = offsetFacesFeature_var.deleteMe()
```

## Return Value

| Type    | Description                                                    |
|---------|----------------------------------------------------------------|
| boolean | Returns a bool indicating if the delete was successful or not. |

## Version

Introduced in version June 2017
