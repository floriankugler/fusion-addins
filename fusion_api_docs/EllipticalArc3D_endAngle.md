# EllipticalArc3D.endAngle Property

Parent Object: [EllipticalArc3D](EllipticalArc3D.md)  

## Description

Gets and sets the end angle of the elliptical arc.

## Syntax

"ellipticalArc3D_var" is a variable referencing an EllipticalArc3D object.  

```python
# Get the value of the property.
propertyValue = ellipticalArc3D_var.endAngle

# Set the value of the property.
ellipticalArc3D_var.endAngle = propertyValue
```

## Property Value

This is a read/write property whose value is a double.

## Version

Introduced in version August 2014
