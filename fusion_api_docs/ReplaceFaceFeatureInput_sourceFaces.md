# ReplaceFaceFeatureInput.sourceFaces Property

Parent Object: [ReplaceFaceFeatureInput](ReplaceFaceFeatureInput.md)  

## Description

Gets and sets the entities that define the source faces to perform replace. The collection can contain the faces from a solid and/or from features. All the faces must be on the same body.

## Syntax

"replaceFaceFeatureInput_var" is a variable referencing a ReplaceFaceFeatureInput object.  

```python
# Get the value of the property.
propertyValue = replaceFaceFeatureInput_var.sourceFaces

# Set the value of the property.
replaceFaceFeatureInput_var.sourceFaces = propertyValue
```

## Property Value

This is a read/write property whose value is an [ObjectCollection](ObjectCollection.md).

## Version

Introduced in version March 2015
