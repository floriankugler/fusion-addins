# SketchFittedSpline.deleteMe Method

Parent Object: [SketchFittedSpline](SketchFittedSpline.md)  

## Description

Deletes the entity from the sketch.

## Syntax

"sketchFittedSpline_var" is a variable referencing a [SketchFittedSpline](SketchFittedSpline.md) object.

```python
returnValue = sketchFittedSpline_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true is the delete was successful. |

## Version

Introduced in version August 2014
