# CopyPasteBody.attributes Property

Parent Object: [CopyPasteBody](CopyPasteBody.md)  

## Description

Returns the collection of attributes associated with this face.

## Syntax

"copyPasteBody_var" is a variable referencing a CopyPasteBody object.  

```python
# Get the value of the property.
propertyValue = copyPasteBody_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version June 2017
