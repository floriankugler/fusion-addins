# BoxFeature.isSuppressed Property

Parent Object: [BoxFeature](BoxFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"boxFeature_var" is a variable referencing a BoxFeature object.  

```python
# Get the value of the property.
propertyValue = boxFeature_var.isSuppressed

# Set the value of the property.
boxFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version September 2015
