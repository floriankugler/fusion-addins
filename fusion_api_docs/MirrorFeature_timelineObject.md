# MirrorFeature.timelineObject Property

Parent Object: [MirrorFeature](MirrorFeature.md)  

## Description

Returns the timeline object associated with this feature.

## Syntax

"mirrorFeature_var" is a variable referencing a MirrorFeature object.  

```python
# Get the value of the property.
propertyValue = mirrorFeature_var.timelineObject
```

## Property Value

This is a read only property whose value is a [TimelineObject](TimelineObject.md).

## Version

Introduced in version November 2014
