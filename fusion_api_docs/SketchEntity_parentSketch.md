# SketchEntity.parentSketch Property

Parent Object: [SketchEntity](SketchEntity.md)  

## Description

Returns the parent sketch.

## Syntax

"sketchEntity_var" is a variable referencing a SketchEntity object.  

```python
# Get the value of the property.
propertyValue = sketchEntity_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).

## Version

Introduced in version August 2014
