# BaseComponent.bRepBodies Property

Parent Object: [BaseComponent](BaseComponent.md)  

## Description

Returns the B-Rep bodies collection associated with this component.

## Syntax

"baseComponent_var" is a variable referencing a BaseComponent object.  

```python
# Get the value of the property.
propertyValue = baseComponent_var.bRepBodies
```

## Property Value

This is a read only property whose value is a [BRepBodies](BRepBodies.md).

## Version

Introduced in version August 2014
