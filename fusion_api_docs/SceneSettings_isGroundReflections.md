# SceneSettings.isGroundReflections Property

Parent Object: [SceneSettings](SceneSettings.md)  

## Description

Gets and sets if objects are reflected on the ground plane.

## Syntax

"sceneSettings_var" is a variable referencing a SceneSettings object.  

```python
# Get the value of the property.
propertyValue = sceneSettings_var.isGroundReflections

# Set the value of the property.
sceneSettings_var.isGroundReflections = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version May 2023
