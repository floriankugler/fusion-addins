# OperationBase.attributes Property

Parent Object: [OperationBase](OperationBase.md)  

## Description

Returns the collection of attributes associated with this object.

## Syntax

"operationBase_var" is a variable referencing an OperationBase object.  

```python
# Get the value of the property.
propertyValue = operationBase_var.attributes
```

## Property Value

This is a read only property whose value is an [Attributes](Attributes.md).

## Version

Introduced in version August 2021
