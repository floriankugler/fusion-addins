# Sketch.isConstructionGeometryShown Property

Parent Object: [Sketch](Sketch.md)  

## Description

Gets and sets whether construction geometry in the sketch is displayed. This provides access to the "Construction Geometries" setting in the "SKETCH PALETTE".

## Syntax

"sketch_var" is a variable referencing a Sketch object.  

```python
# Get the value of the property.
propertyValue = sketch_var.isConstructionGeometryShown

# Set the value of the property.
sketch_var.isConstructionGeometryShown = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version March 2024
