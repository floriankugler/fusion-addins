# SphereFeature.isSuppressed Property

Parent Object: [SphereFeature](SphereFeature.md)  

## Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

## Syntax

"sphereFeature_var" is a variable referencing a SphereFeature object.  

```python
# Get the value of the property.
propertyValue = sphereFeature_var.isSuppressed

# Set the value of the property.
sphereFeature_var.isSuppressed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version September 2015
