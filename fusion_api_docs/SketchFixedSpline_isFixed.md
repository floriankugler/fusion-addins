# SketchFixedSpline.isFixed Property

Parent Object: [SketchFixedSpline](SketchFixedSpline.md)  

## Description

Indicates if this geometry is "fixed".

## Syntax

"sketchFixedSpline_var" is a variable referencing a SketchFixedSpline object.  

```python
# Get the value of the property.
propertyValue = sketchFixedSpline_var.isFixed

# Set the value of the property.
sketchFixedSpline_var.isFixed = propertyValue
```

## Property Value

This is a read/write property whose value is a boolean.

## Version

Introduced in version August 2014
