# Surface.evaluator Property

Parent Object: [Surface](Surface.md)  

## Description

Returns the surface evaluator.

## Syntax

"surface_var" is a variable referencing a Surface object.  

```python
# Get the value of the property.
propertyValue = surface_var.evaluator
```

## Property Value

This is a read only property whose value is a [SurfaceEvaluator](SurfaceEvaluator.md).

## Version

Introduced in version August 2014
