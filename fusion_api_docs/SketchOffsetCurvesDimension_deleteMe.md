# SketchOffsetCurvesDimension.deleteMe Method

Parent Object: [SketchOffsetCurvesDimension](SketchOffsetCurvesDimension.md)  

## Description

Deletes this dimension. The IsDeletable property indicates if this dimension can be deleted.

## Syntax

"sketchOffsetCurvesDimension_var" is a variable referencing a [SketchOffsetCurvesDimension](SketchOffsetCurvesDimension.md) object.

```python
returnValue = sketchOffsetCurvesDimension_var.deleteMe()
```

## Return Value

| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | Returns true if the delete was successful. |

## Version

Introduced in version March 2016
