# ConstructionPlane.component Property

Parent Object: [ConstructionPlane](ConstructionPlane.md)  

## Description

Returns the component this construction plane belongs to.

## Syntax

"constructionPlane_var" is a variable referencing a ConstructionPlane object.  

```python
# Get the value of the property.
propertyValue = constructionPlane_var.component
```

## Property Value

This is a read only property whose value is a [Component](Component.md).

## Version

Introduced in version August 2014
