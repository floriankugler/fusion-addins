# UntrimFeatureInput.extensionDistance Property

Parent Object: [UntrimFeatureInput](UntrimFeatureInput.md)  

## Description

Gets and sets the ValueInput object that defines the extension distance applied to faces when an external boundary is removed.

## Syntax

"untrimFeatureInput_var" is a variable referencing a UntrimFeatureInput object.  

```python
# Get the value of the property.
propertyValue = untrimFeatureInput_var.extensionDistance

# Set the value of the property.
untrimFeatureInput_var.extensionDistance = propertyValue
```

## Property Value

This is a read/write property whose value is a [ValueInput](ValueInput.md).

## Version

Introduced in version January 2021
