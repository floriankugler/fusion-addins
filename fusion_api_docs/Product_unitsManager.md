# Product.unitsManager Property

Parent Object: [Product](Product.md)  

## Description

Returns the UnitsManager object associated with this product.

## Syntax

"product_var" is a variable referencing a Product object.  

```python
# Get the value of the property.
propertyValue = product_var.unitsManager
```

## Property Value

This is a read only property whose value is a [UnitsManager](UnitsManager.md).

## Version

Introduced in version August 2014
