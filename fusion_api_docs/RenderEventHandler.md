# RenderEventHandler Object

Derived from: [EventHandler](EventHandler.md) Object  

## Description

The RenderEventHandler is a client implemented class that can be added as a handler to a RenderEvent.

## Methods

| Name | Description |
|----|----|
| [notify](RenderEventHandler_notify.md) | The function called by Fusion when the associated event is fired. |

## Version

Introduced in version September 2023
