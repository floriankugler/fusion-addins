# ConfigurationVisibilityCell.parentColumn Property

Parent Object: [ConfigurationVisibilityCell](ConfigurationVisibilityCell.md)  

## Description

Returns the column this cell is in.

## Syntax

"configurationVisibilityCell_var" is a variable referencing a ConfigurationVisibilityCell object.  

```python
# Get the value of the property.
propertyValue = configurationVisibilityCell_var.parentColumn
```

## Property Value

This is a read only property whose value is a [ConfigurationVisibilityColumn](ConfigurationVisibilityColumn.md).

## Version

Introduced in version January 2024
