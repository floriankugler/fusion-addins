# CAMPattern.parent Property

Parent Object: [CAMPattern](CAMPattern.md)  

## Description

Returns the parent Setup, Folder or Pattern for this Folder.

## Syntax

"cAMPattern_var" is a variable referencing a CAMPattern object.  

```python
# Get the value of the property.
propertyValue = cAMPattern_var.parent
```

## Property Value

This is a read only property whose value is a [Base](Base.md).

## Version

Introduced in version January 2016
