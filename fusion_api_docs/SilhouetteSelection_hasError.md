# SilhouetteSelection.hasError Property

Parent Object: [SilhouetteSelection](SilhouetteSelection.md)  

## Description

Gets if errors were encountered when applying the selection to a a parent.

## Syntax

"silhouetteSelection_var" is a variable referencing a SilhouetteSelection object.  

```python
# Get the value of the property.
propertyValue = silhouetteSelection_var.hasError
```

## Property Value

This is a read only property whose value is a boolean.

## Version

Introduced in version July 2023
