# ArrangeResultEnvelopes.count Property

Parent Object: [ArrangeResultEnvelopes](ArrangeResultEnvelopes.md)  

## Description

Returns the number of Arrange envelope results for this Arrange feature.

## Syntax

"arrangeResultEnvelopes_var" is a variable referencing an ArrangeResultEnvelopes object.  

```python
# Get the value of the property.
propertyValue = arrangeResultEnvelopes_var.count
```

## Property Value

This is a read only property whose value is a uinteger.

## Version

Introduced in version January 2025
