# OperationBase.generatedDataCollection Property

Parent Object: [OperationBase](OperationBase.md)  

## Description

Get the generated data associated with a given operation base instance. The type of data depends on the strategy type and might not be available for all strategy types. The available types can be found in GeneratedData.cs

## Syntax

"operationBase_var" is a variable referencing an OperationBase object.  

```python
# Get the value of the property.
propertyValue = operationBase_var.generatedDataCollection
```

## Property Value

This is a read only property whose value is a [GeneratedDataCollection](GeneratedDataCollection.md).

## Version

Introduced in version July 2023
