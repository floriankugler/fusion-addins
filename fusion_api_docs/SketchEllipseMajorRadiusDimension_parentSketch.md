# SketchEllipseMajorRadiusDimension.parentSketch Property

Parent Object: [SketchEllipseMajorRadiusDimension](SketchEllipseMajorRadiusDimension.md)  

## Description

Returns the parent sketch object.

## Syntax

"sketchEllipseMajorRadiusDimension_var" is a variable referencing a SketchEllipseMajorRadiusDimension object.  

```python
# Get the value of the property.
propertyValue = sketchEllipseMajorRadiusDimension_var.parentSketch
```

## Property Value

This is a read only property whose value is a [Sketch](Sketch.md).
