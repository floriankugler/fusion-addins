# PrintSettingQuery.execute Method

Parent Object: [PrintSettingQuery](PrintSettingQuery.md)  

## Description

Query for specific PrintSettings. This PrintSettingQuery query.

## Syntax

"printSettingQuery_var" is a variable referencing a [PrintSettingQuery](PrintSettingQuery.md) object.

```python
returnValue = printSettingQuery_var.execute()
```

## Return Value

| Type | Description |
|----|----|
| [PrintSetting](PrintSetting.md)\[\] | Returns a list of PrintSetting. Each returned PrintSetting matches this query. |

## Version

Introduced in version May 2023
