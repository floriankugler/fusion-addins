# DataFolders.itemByName Method

Parent Object: [DataFolders](DataFolders.md)  

## Description

Returns the folder specified using the name of the folder.

## Syntax

"dataFolders_var" is a variable referencing a [DataFolders](DataFolders.md) object.

```python
returnValue = dataFolders_var.itemByName(name)
```

## Return Value

| Type | Description |
|----|----|
| [DataFolder](DataFolder.md) | Returns the folder or null if a folder of the specified name is not found. |

## Parameters

| Name | Type   | Description                       |
|------|--------|-----------------------------------|
| name | string | The name of the folder to return. |

## Version

Introduced in version January 2015
